Schlüsselwörter

Die Analyse Schlüsselwörter durchsucht ein oder mehrere Felder nach bestimmten Schlüsselwörtern. Diese Schlüsselwörter können manuell eingegeben, aus einer vordefinierten Liste an Schlüsselwörtern ausgewählt oder aus einer Liste mit Begriffen, die aus den ausgewählten Feldern generiert wird, ausgewählt werden.

Dieser Test enthält auch einen Parameter, mit dem Sie bei der Suche nach Schlüsselwörtern zwischen exakten Übereinstimmungen und Teilübereinstimmungen wählen können.

Diese Analyse kann zur Identifizierung verwendet werden:

  • Transaktionen, die signifikante Schätzungen enthalten

  • Fiktive Transaktionen

  • Transaktionen, die von komplexer oder ungewöhnlicher Art sind

  • Unternehmensinterne Transaktion

  • Transaktionen, die zur Abstimmung von Nebenbüchern verwendet werden. Sie können zum Beispiel nach den Stichworten „Nebenbuch abstimmen“ oder „Nebenbuch ausgleichen“ suchen.

Für die Analyse verwendete Felder

Die folgenden Felder sind für diese Analyse erforderlich:

  • Referenzfeld(er) - Eindeutige(s) Feld(er), das/die zur Erstellung einer eindeutigen Transaktions-ID verwendet wird/werden, wie z. B. das Feld Eintrags-ID für den Hauptbuch-Datensatz. Diese Spalten sind nicht Teil des Ergebnisses, sondern dienen der Identifizierung der Transaktionen, die Teil des Ergebnisses sind. Dieses Feld wurde bereits im Test definiert und kann nicht geändert werden.

  • Schlüsselwortfeld(er) - Die Zeichenfelder, die nach den ausgewählten Schlüsselwörtern durchsucht werden.

Parameter

Zur Durchführung dieses Tests müssen die folgenden Parameter eingestellt werden:

  • Schlüsselwortliste - Eine Liste von Schlüsselwörtern, die als Textdatei gespeichert wird.

  • Genaue Übereinstimmung - Wählen Sie aus, ob die Werte exakt mit dem Schlüsselwort übereinstimmen müssen. Wenn Sie nicht auswählen, dass nur exakte Übereinstimmungen ermittelt werden, werden auch Teilübereinstimmungen ermittelt.

  • Groß- und Kleinschreibung ignorieren - Wählen Sie aus, ob bei den Suchbegriffen zwischen Groß- und Kleinschreibung unterschieden werden soll oder nicht.

  • Spalte für übereinstimmende Wörter erstellen - Erstellen Sie eine zusätzliche Spalte, in der die Wörter aufgelistet werden, für die in dieser Konfiguration Übereinstimmungen gefunden wurden.

Test-Konfigurationen

Die einzige Konfiguration, die für diesen Test zur Verfügung steht, ist die Schlüsselwortsuche.

Technische Spezifikationen

Wenn Sie die Analysefunktion Schlüsselwörter ausführen, werden die folgenden Schritte zur Durchführung des Tests durchgeführt:

  1. Überprüfen Sie, ob die erforderlichen Referenzfelder ausgewählt wurden. Wenn keine Felder ausgewählt wurden, erstellen Sie ein eindeutiges Referenzfeld. Dieser Schritt wird nur ausgeführt, wenn bestimmte Felder ausgewählt wurden. Wenn alle Felder vorhanden sind, ist dieser Schritt nicht erforderlich.

  2. Bestätigen Sie, dass ein oder mehrere Zeichenfelder für die Schlüsselwortsuche ausgewählt wurden.

  3. Ermitteln Sie die Wortliste, die je nach Schnittstelle aus mehreren Quellen oder einer Mischung aus einer oder mehreren Quellen stammen kann:

    • Eine vorprogrammierte Liste von Schlüsselwörtern.

    • Eine vom Benutzer hochgeladene Liste der Schlüsselwörter.

    • Der Benutzer gibt die Schlüsselwörter direkt in die Schnittstelle ein.

    • Eine Kombination aus einem oder mehreren der oben genannten Punkte.

  4. Erlauben Sie dem Benutzer, Schlüsselwörter aus der Liste hinzuzufügen oder zu löschen, insbesondere wenn sie aus 3a oder 3b stammen.

  5. Ermitteln Sie, ob die Groß- und Kleinschreibung beachtet werden muss oder nicht.

  6. Je nach Auswahl von exakten oder teilweisen Übereinstimmungen führen Sie die Suche nach den Stichwörtern aus Schritt 3 durch.

  7. Wenn eine Spalte mit einem passenden Schlüsselwort ausgewählt wurde, wird nur eine Spalte hinzugefügt, auch wenn mehrere Spalten für die Suche ausgewählt wurden.
