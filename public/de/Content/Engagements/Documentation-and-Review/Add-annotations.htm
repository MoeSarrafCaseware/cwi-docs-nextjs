Referenzen hinzufügen

Referenzen können in Hauptabschlussübersicht, Checklisten, Abschlussbuchungen oder PDF-Dokumenten hinzugefügt werden. Sie können einem bestimmten Eintrag einen Status, einen Kommentar oder eine Verknüpfung zu anderen relevanten Dokumenten hinzufügen.

Sobald Sie eine Referenz hinzugefügt haben, erscheint eines der folgenden Symbole auf der Seite, um den Zweck der Referenz darzustellen:

  • Agrees to ()

  • Comes from ()

  • Goes to ()

  • Calculation Verified ()

  • Addition Checked ()

  • Link ()

  • Note ()

  • Variance ()

Eine Referenz zu einer Checkliste hinzufügen

In einem Checklistendokument können Sie eine Referenz zu Prüfungshandlungen und untergeordneten Verfahren hinzufügen.

Referenzen zu einer Checkliste hinzufügen:

  1. Öffnen Sie die Engagementdatei und wählen Sie Dokumente

  2. Wählen Sie ein Checklistendokument aus, um es zu öffnen.

  3. Gehen Sie zu der Prozedur oder Unterprozedur, die Sie mit einer Referenz versehen möchten, und wählen Sie Referenz hinzufügen Symbol ().

  4. Geben Sie die Informationen für die Referenz ein.

    1. Wählen Sie den Zweck der Referenz aus.

      Bitte beachten Sie: Sie können einen Vorschlag auswählen, um die vorgeschlagene Referenz automatisch zuzuweisen.

    2. Verknüpfen Sie die Referenz mit einem Dokument oder einer Buchung, falls zutreffend. Wenn Sie die Referenz mit einer Checkliste verknüpfen, können Sie ein bestimmtes Verfahren zur Verknüpfung auswählen.

    3. Geben Sie Notizen oder Kommentare ein.

  5. Klicken Sie auf Speichern.

Das entsprechende Symbol für jede Referenz wird neben dem Dokument angezeigt. Wenn Sie das Symbol auswählen erscheint ein Pop-up mit einer Zusammenfassung der Referenz, dem Erstellungsdatum und dem Ersteller.

Eine Referenz zu einer Hauptabschlussübersicht hinzufügen (Seite Daten)

Sie können Konten und Gruppenzeilen in der Hauptabschlussübersicht mit einer Referenz versehen.

Referenzen in der Hauptabschlussübersicht hinzufügen:

  1. Navigieren Sie auf der Seite Daten zu der Zeile, in der Sie eine Referenz hinzufügen möchten.

    2. Hinweis: Stellen Sie sicher, dass das Kontrollkästchen Referenzen in der Drop-down-Liste Indikatoren im oberen Bereich der Seite aktiviert ist.

  2. Wählen Sie Referenz hinzufügen ().

  3. Geben Sie die Details der Referenz ein.

    1. Wählen Sie bei Referenz hinzufügen den Zweck der Referenz aus.

    2. Wählen Sie ggf. bei Verknüpfung ein Dokument oder eine Buchung aus, um es mit der Referenz zu verknüpfen. Wenn Sie die Referenz mit einer Checkliste verknüpfen, können Sie ein bestimmtes Verfahren zur Verknüpfung auswählen.

    3. Geben Sie bei Anhangdokument Notizen oder Kommentare ein.

  4. Klicken Sie auf Speichern.

    5. Das entsprechende Symbol für die Referenz wird neben der kommentierten Zeile angezeigt. Wenn Sie auf die Seite zurückkehren und das Symbol auswählen oder den Mauszeiger auf das Symbol stellen, erscheint ein Pop-up mit einer Zusammenfassung der Referenz, dem Erstellungsdatum und dem Ersteller.

  3. Geben Sie die Details der Referenz ein.

Eine Referenz zu einer Buchung hinzufügen

Sie können eine Referenz für die gesamte Abschlussbuchung oder für jede Zeile des Eintrags hinzufügen.

Referenzen zu Buchungen hinzufügen:

  1. Gehen Sie auf die Seite Buchungen.

  2. Klicken Sie auf Bearbeiten () neben der Buchung, die Sie mit einer Referenz versehen möchten.

    Der Dialog mit Buchungsdetails wird geöffnet.

  3. Um den gesamten Buchungseintrag mit Referenzen zu versehen, wählen Sie oben rechts im Dialog die Option Referenzen hinzufügen () und geben Sie die Buchungsdetails ein.

  4. Um eine Zeile im Buchungseintrag mit einer Referenz zu versehen, gehen Sie in den Bereich Details und wählen Referenzen hinzufügen () neben der Zeile.

  5. Geben Sie die Details der Referenz ein.

    1. Wählen Sie den Zweck der Referenz aus.

      Bitte beachten Sie: Sie können einen Vorschlag auswählen, um den vorgeschlagenen Zweck der Referenz automatisch zuzuweisen.

    2. Verknüpfen Sie die Referenz mit einem Dokument oder einer Buchung, falls zutreffend. Wenn Sie die Referenz mit einer Checkliste verknüpfen, können Sie ein bestimmtes Verfahren zur Verknüpfung auswählen.

    3. Geben Sie Notizen oder Kommentare ein.

  6. Klicken Sie auf Speichern.

    Das entsprechende Symbol für die Referenz wird neben der kommentierten Zeile angezeigt. Wenn Sie das Symbol auswählen erscheint ein Pop-up mit einer Zusammenfassung der Referenz, dem Erstellungsdatum und dem Ersteller.
