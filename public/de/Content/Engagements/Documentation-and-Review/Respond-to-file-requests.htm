Auf Dateianforderungen antworten

Hier sehen Sie als Kontakt, wie Sie auf Dateianforderungen reagieren, die Ihnen von Firmenmitarbeitern zugewiesen wurden. Dateianforderungen sind durch das Symbol Dateianforderung () in der Liste Meine Arbeit gekennzeichnet.

Auf eine Dateianforderung antworten:

  1. Wählen Sie Antworten, um die Dateianforderung zu öffnen.

  2. Klicken Sie auf die Schaltfläche Wählen Sie Dateien aus oder ziehen Sie die Datei per Drag-and-Drop in den dafür vorgesehenen Bereich.

  3. Geben Sie Kommentare in das Textfeld ein.

  4. Wählen Sie Senden (Antworten), um die Dateianforderung fertigzustellen.
