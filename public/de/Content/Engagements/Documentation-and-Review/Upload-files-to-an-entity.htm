Dateien in eine Entität hochladen

Sie können Dateien und Dokumente in Cloud hinzufügen, so dass das relevante Material für Ihr Engagement an einem zugänglichen Ort gespeichert wird. Bei relevantem Material kann es sich um Dateien handeln, dazu können auch Kopien der Korrespondenz mit dem Mandanten gehören oder Dokumente im Zusammenhang mit der Engagementarbeit.

Wenn Sie steuerrelevante Arbeiten in TaxPrep ausführen, können Sie Ihre TaxPrep Dateien in Cloud hochladen, um die Arbeit an diesen Dateien nachzuverfolgen und zuzuweisen, wie bei jedem anderen Dateityp.

Das Verfahren zum Hinzufügen von einer TaxPrep Datei entspricht dem Hinzufügen jeder anderen Datei. Nachdem die Datei hochgeladen wurde, erstellt Cloud automatisch ein Anderes Engagement und speichert die TaxPrep Datei dort.

Bevor Sie beginnen, stellen Sie bitte sicher, dass Sie über die Rolle Editor oder gleichwertige Berechtigungen verfügen. Weitere Informationen zu Vorgegebene Rollen.

Eine Datei hochladen:

  1. Dateien.

  2. Wählen Sie die betreffende Entität aus der Drop-down-Liste der Entitäten aus.

  3. Wählen Sie Neu und anschließend Hochladen. Der Dialog Datei hochladen wird angezeigt.

  4. Klicken Sie auf die Schaltfläche Wählen Sie Dateien aus oder ziehen Sie die Datei per Drag-and-Drop in den dafür vorgesehenen Bereich.

  5. Wählen Sie im Dialog Freigabeeinstellungen die Freigabeoptionen für die Datei und klicken Sie auf OK.
