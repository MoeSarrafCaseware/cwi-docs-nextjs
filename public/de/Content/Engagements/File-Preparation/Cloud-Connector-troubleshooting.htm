Troubleshooting for Caseware Cloud Connector

This section outlines possible solutions for common issues users may encounter while using Caseware Cloud Connector.

Cloud Connector pane is blank

There are several reasons why the contents of the Cloud Connector pane may not display:

  • Page loading time

    You may occasionally need to wait longer than usual for the page to load.

  • When new updates are released for Cloud Connector, the add-in does not always immediately update. Complete the following steps to resolve the issue:

    1. From the Home toolbar of your Excel document, select Add-ins | More Add-ins | My Add-ins.

    2. In the dialog, select Refresh.

    3. Close and reopen Excel.

  • A known Excel bug that prevents the pane from loading

    If the pane still does not load, select the Cloud Connector drop-down and select Resize.

  • A runtime error that prevents the pane from loading

    If the previous troubleshooting tips don’t resolve the issue, complete the following steps before contacting our support team.

    1. Right-click on the pane and select Inspect. The developer tools window opens.
    2. Select the Console tab at the top of the window.
    3. Take a screenshot of the Console tab contents and send it to our support team to help identify and resolve the issue.

Invalid firm message

If you receive an invalid firm message, you may have the wrong version of the Caseware Cloud Connector add-in installed.

To confirm which version of the add-in is installed:

  1. Select the drop-down at the top of the Cloud Connector pane and select Security Info.

  2. If the URL contains “cwcloudpartner”, you have the incorrect version installed.

To uninstall the add-in:

  1. On the Excel ribbon, select My Add-ins from the Insert tab.

  2. Select Caseware Cloud Connector, then Remove.

To re-install the add-in, see Get started with Caseware Cloud Connector.

Multiple versions of Cloud Connector installed

If you have multiple versions of Caseware Cloud Connector installed, Cloud Connector will not work as expected.

To confirm how many versions of Cloud Connector you have installed, select My Add-ins from the Insert tab.

If you have multiple Cloud Connector add-ins installed, uninstall the add-ins.

To uninstall the add-in:

  1. Select My Add-ins from the Insert tab.

  2. Select Caseware Cloud Connector, then Remove.

After you uninstall the add-ins, copy your sheets to a new Excel workbook and reupload it to your engagement file to replace the old file. You can then re-install the correct version of Cloud Connector. For more information, see Get started with Caseware Cloud Connector.

General troubleshooting

For general issues that are not covered in the previous sections, try clearing your cache and re-installing the add-in.

To uninstall the add-in:

  1. On the Excel ribbon, select My Add-ins from the Insert tab.

  2. Select Caseware Cloud Connector, then Remove.

  3. Clear your Excel cache. For more information, see Clear the Office cache.

You can then re-install the correct version of Cloud Connector. For more information, see Get started with Caseware Cloud Connector.
