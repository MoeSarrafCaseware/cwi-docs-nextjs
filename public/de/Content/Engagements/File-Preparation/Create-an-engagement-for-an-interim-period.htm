Ein Engagement für eine Zwischenperiode erstellen

Sie können Engagementdateien für monatliche, vierteljährliche und halbjährliche Zwischenzeiträume erstellen.

Für Interimsengagements können Sie nur Excel- oder CSV-Dateien in Ihre Bilanz importieren. Bitte beachten Sie: Die Verfügbarkeit der nachfolgend beschriebenen Funktionen hängt von den Apps ab, die Sie im Einsatz haben.

Eine vorläufige Engagementdatei erstellen:

  1. Erstellen Sie die ursprüngliche Einstellungsdatei.

    Weitere Informationen erhalten Sie in Eine Engagementdatei erstellen.

  2. Wählen Sie in den Eigenschaften des Engagements die Berichtsperiode und das Jahresabschlussdatum.

  3. Wählen Sie die Registerkarte Hauptabschlussübersicht und anschließend Excel oder CSV, um die Hauptabschlussübersicht zu importieren.

    Hauptabschlussübersichten müssen für ein Jahr kumulierte Salden und nicht Periodensalden beinhalten. Nachdem Sie eine Hauptabschlussübersicht importiert haben, können Sie den Berichtszeitraum für das Engagement nicht mehr ändern.

So erstellen Sie die Engagementdatei für den nächsten Zeitraum:

  1. Wählen Sie Datei der nächsten Periode erstellen unter Weitere Aktionen ().

  2. Füllen Sie den Dialog Übertrag nach Bedarf aus.

  3. Wählen Sie im Feld Daten übertragen von die Engagementdatei mit den Daten aus, die Sie übertragen möchten.

    Die Mandantenstammdaten im neuen Engagement werden auf Aktuelle Periode gesetzt.

  4. Akzeptieren oder verwerfen Sie Antworten, geänderte Abfragen oder neue Dokumente aus dem vorherigen Zeitraum. Weitere Informationen erhalten Sie in Ein Engagement übertragen.

  5. Wählen Sie auf der Registerkarte Hauptabschlussübersicht die Option Weitere Aktionen () | Erneut importieren und Excel oder CSV, um die Hauptabschlussübersicht für die neue Periode zu importieren. Die Spalten in der Hauptabschlussübersicht spiegeln nun die kumulierten Jahressalden für die neue Periode wider.

    Hauptabschlussübersichten müssen kumulierte Jahressalden und nicht Periodensalden beinhalten. Nachdem Sie eine Hauptabschlussübersicht importiert haben, können Sie den Berichtszeitraum für das Engagement nicht mehr ändern.

  6. Wählen Sie auf der Registerkarte Dokumente den Jahresabschluss und blättern Sie nach unten zur Gewinn- und Verlustrechnung.

    Beachten Sie die Spalten für den periodenspezifischen Saldo und kumulierte Jahressalden für das laufende und das Vorjahr.

  7. Führen Sie das Engagement genauso aus wie ein jährliches Engagement.
