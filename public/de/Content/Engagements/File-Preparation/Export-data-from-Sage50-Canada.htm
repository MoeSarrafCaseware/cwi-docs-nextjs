Export Transactions by Account report

To export the Transactions by Account report from Sage 50 Canada:

  1. From the Reports menu, select Report Centre.
  2. On the Report Centre dialog, in Select a report type, select Financials | Transactions by account.
  3. Click Create. A preview of the report that will be exported is displayed.

Note: The Report Centre exports data for the current financial year by default. If your engagement dates are from the previous financial year, you can click the Modify this report option, and from the Report on Fiscal Year dropdown, select Previous Year (YYYY).

  1. From the File menu, select Export...
  2. In File name, specify a name for the report that you will recognize when importing it into Caseware Cloud and click Save.

Export the Customer Ledger record

To export the Customer Ledger record from Sage 50 Canada:

  1. From the File menu, select Import/Export | Export records…
  2. On the Export Records Wizard dialog, in Select a report type, select Export records to a Sage 50 Import file format. Klicken Sie auf Next.
  3. In Select the type of records to export, select Customers.
  4. Specify a name and location for the export file and click Next.

  1. In the Select the customers you want to export box, select the customers whose ledger records you want to export or click Select All to export all customer ledger records.
  2. Klicken Sie auf Next.
  3. View the Export Records Summary and click OK.
  4. Click Finish.

Export the Customer Aged Detail report

To export the Customer Aged Detail report from Sage 50 Canada:

  1. From the Reports menu, select Report Centre.
  2. On the Report Centre dialog, in Select a report type, select Customers & Sales | Customer Aged Detail.
  3. Click Create. A preview of the report that will be exported is displayed.

Note: The Report Centre exports all invoices paid in the last 999 days as of the closing date of the current fiscal year by default. You can modify the As at date by clicking the Modify this report option. Make sure that the selected date is later than or equal to the End Date of your Caseware engagement.

 

 

  1. From the File menu, select Export…
  2. In File name, specify a name for the report that you will recognize when importing it into Caseware Cloud and click Save.

Export the Vendor Ledger record

To export the Vendor Ledger record from Sage 50 Canada:

  1. From the File menu, select Import/Export | Export records…
  2. On the Export Records Wizard dialog, in Select a report type, select Export records to a Sage 50 Import file format. Click Next.
  3. In Select the type of records to export, select Vendors.
  4. Specify a name and location for the export file and click Next.

  1. In the Select the vendors you want to export box, select the customers whose ledger records you want to export or click Select All to export all vendor ledger records.
  2. Klicken Sie auf Next.
  3. View the Export Records Summary and click OK.
  4. Click Finish.
