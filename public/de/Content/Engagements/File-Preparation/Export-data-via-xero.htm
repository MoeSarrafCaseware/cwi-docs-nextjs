Export data from Xero to Caseware Cloud

See the instructions below to export your transaction data from Xero for import into Caseware Cloud.

The process for importing and exporting datasets varies slightly depending on the type of data you are exporting, for example, trial balance, general ledger, various types of subledger, or other data.

For information on importing transaction files to Caseware Cloud, see instructions to do so via CSV or Excel and through connected accounting software packages.

Export the trial balance

To export the trial balance:

  1. In the Accounting drop-down menu, select Reports.

  2. Navigate to the Taxes and Balances section and select Trial Balance.
  3. Select your preferences regarding date range, columns, comparisons and grouping and select Update.
  4. At the bottom-right of the page, click Export and select the Excel option.

TIP: To save files as CSV for more flexible use, open the Excel file, select Save As…, and under the Save as type, select All Files. Type .csv as the file extension.

Export the general ledger

To export the general ledger:

  1. In the Accounting drop-down menu, select Advanced.
  2. From the Advanced Features menu, select Export accounting data.
  3. From the Select a product drop-down menu, under the General Ledger section, click Caseware.
  4. Select the date range for your General Ledger activity. For example, if you're auditing the year ended 31 December 2020, your file should be dated from 1 January 2020 to 31 December 2020.
  5. Select Download to save your general ledger file.

  6. Change the file extension from .txt (default) to .csv.

Export the subledger – accounts payable (AP)

See the instructions below to export the various types of subledger — accounts payable data.

Export AP invoices/payments made

To export accounts payable – invoices/payments made data:

  1. In the Accounting drop-down menu, select Reports.
  2. In the Payables and Receivables section, select either Payable Invoice Summary or Payable Invoice Detail depending on your requirements.
  3. Select your preferences regarding date range, columns, comparisons and grouping and select Update.
  4. At the bottom-right of the page, click Export and select the Excel option.

Export AP suppliers

To export accounts payable – suppliers data:

  1. From the Contacts drop-down menu, select Suppliers.

  2. Select the three dots in the top right corner and click Export to save the file as .csv.

Export AP open balances

To export accounts payable – open balances data:

  1. In the Accounting drop-down, select Reports.
  2. In the Payables and Receivables section, select either Aged Payables Summary or Detail depending on your requirements.
  3. Select your preferences regarding date range, columns, comparisons and grouping and select Update.
  4. At the bottom-right of the page, click Export and select the Excel option.

Export the subledger – accounts receivable (AR)

To export accounts receivable – invoices/cash received data:

  2. In the Payables and Receivables section, select either Receivable Invoice Summary or Detail depending on your requirements.
  3. Select your preferences regarding date range, columns, comparisons and grouping and select Update.

Export AR customers

To export accounts receivable – customers data:

  1. Select Customers from the Contacts drop-down menu.
  2. Select the three dots in the top right corner and click Export to save the file as .csv.

  2. In the Payables and Receivables section, select either Aged Receivable Summary or Detail depending on your requirements.

Export the subledger – inventory

See the instructions below to export the various types of subledger — accounts receivable data.

Export inventory transactions

To export subledger – inventory transactions data:

  1. In the Accounting drop-down menu, select Reports.
  2. In the Transactions section, select either Inventory Item Summary or Detail depending on your requirements.
  3. Select your preferences regarding date range, columns, and comparisons. In the Grouping/Summarizing drop-down menu, select Inventory Account.

  4. Select Update, and in the bottom right, click Export and select the Excel option.

Export inventory on hand

To export subledger – inventory on hand data:

  1. In the Accounting drop-down menu, select Reports.
  2. In the Reconciliations section, select Inventory Item List.
  3. Select your preferences regarding date range, columns, and comparisons. In the Grouping/Summarizing drop-down menu, select Inventory Account.
  4. Select Update, and in the bottom right, click Export and select the Excel option.
