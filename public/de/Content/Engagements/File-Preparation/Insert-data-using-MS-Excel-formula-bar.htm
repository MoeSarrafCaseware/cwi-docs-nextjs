Daten mit Hilfe der MS Excel-Formelleiste in Caseware Cloud Connector einfügen

Sie können die Microsoft Excel-Formelleiste verwenden, um Daten aus der Engagementdatei abzurufen und in die Excel-Datei einzufügen.

So fügen Sie Daten über die Formelleiste ein:

  1. Wählen Sie die Formelleiste und geben Sie =cwi ein, um eine Liste aller verfügbaren Cloud Connector Funktionen anzuzeigen. Sie können auch die Liste der Cloud Connector Funktionen einsehen, um alle verfügbaren Cloud Connector Funktionen und ihre Parameter zu sehen.

  2. Wählen Sie die gewünschte Funktion und drücken Sie Tab.

    Tipp: Wenn Sie den Namen der Funktion kennen, geben Sie ihn in die Formelleiste ein und drücken Sie Tab.

  3. Wenn mit der Funktion Parameter verbunden sind, geben Sie die Parameter wie auf dem Bildschirm angezeigt ein.

    Verwenden Sie die Formelleiste, wenn Sie Hilfe zu den Parametern benötigen (was sie bedeuten oder was das erwartete Format ist). Er bietet Anweisungen und Beschreibungen auf dem Bildschirm.
