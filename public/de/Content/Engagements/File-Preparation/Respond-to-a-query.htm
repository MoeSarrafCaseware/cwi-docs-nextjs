Eine Abfrage beantworten

Sobald alle Fragen als beantwortet markiert sind, wird eine Benachrichtigung mit der Schaltfläche Alle als beantwortet kennzeichnen angezeigt, so dass der Kontakt einfach zum nächsten Schritt im Arbeitsablauf übergehen kann.

Sobald der Status der Abfrage auf Beantwortet wechselt, wird das Symbol Beantwortet () neben der Abfrage auf der Seite Dokumente angezeigt.

Öffnen Sie die Abfrage, um mit der Überprüfung der Antworten zu beginnen. Beachten Sie, wie die Zeit der Antwort und der Name des Kontakts, der auf die jeweilige Frage geantwortet hat, angezeigt werden.
