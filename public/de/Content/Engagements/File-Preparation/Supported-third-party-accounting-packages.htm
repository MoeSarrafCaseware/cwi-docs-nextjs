Unterstützte Desktop-Buchhaltungssoftware von Drittanbietern

You can import client data from most third-party desktop accounting software for use with a variety of Caseware Cloud apps or IDEA. To learn more about importing client data, see Import data from a desktop accounting software.

Note: If the supported import data for your accounting software includes both the trial balance and transactions, the Cloud Import Utility will import both accounts and transactions. If only the trial balance data is supported, the Cloud Import Utility will import only the accounts.
