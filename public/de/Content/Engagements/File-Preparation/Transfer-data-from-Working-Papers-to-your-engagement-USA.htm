Transfer data from Working Papers to your engagement

After you add CLBG document to a Working Papers engagement, you can transfer your trial balance and general ledger tax data to your Cloud engagement.

Hinweis: OnPoint CloudBridge does not support transferring adjustments posted to groups. Only account balances are transferred.

To export your engagement data to a Cloud engagement using OnPoint CloudBridge:

  1. In the Document Manager, double-click CLBG. The CLBG document displays.

  2. In the Step 1 section, choose an appropriate entity and engagement.

  3. In the Step 2 section, choose the appropriate trial balance data to export.

  4. In the Step 2 section, select the Use remap file check box.

    The remap file is used to transfer the current account mapping structure in Working Papers to your Cloud engagement. If your engagement uses the default Working Papers mapping, use the remap file located in the folder where you installed OnPoint CloudBridge (C:\Program Files (x86)\Caseware\Scripts\cloudbridge). If your engagement uses a custom mapping structure specific to your firm, you'll need to edit the default remap file manually in a text editor (e.g. Notepad).

  5. Select Choose file to browse and add the remap file.

  6. Once you're done, select Transfer Data.
