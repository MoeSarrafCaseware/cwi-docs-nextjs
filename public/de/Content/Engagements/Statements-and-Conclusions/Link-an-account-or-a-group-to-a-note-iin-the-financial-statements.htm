Ein Konto oder eine Kontengruppe mit einer Anmerkung im Jahresabschluss verknüpfen

Eine Nummerierungsreferenz ist eine Verknüpfung zu einer vorhandenen Anmerkung/Erläuterung im Jahresabschluss. Sie können in Tabellenzeilen des Jahresabschlusses Nummerierungsreferenzen hinzufügen. Zum Beispiel, um ein Konto oder eine Gruppe mit einer bestehenden Erläuterung zu verknüpfen, die entsprechende Details oder Informationen enthält.

So fügen Sie eine Nummerierungsreferenz in einer Zeile hinzu:

  1. Gehen Sie zu der Zeile, in der Sie eine Nummerierungsreferenz hinzufügen möchten.

  2. Wählen Sie die Schaltfläche Hinzufügen () oberhalb der Zeile.

  3. Wählen Sie Kommentarzeile.

    Eine Nummerierungsreferenz wird direkt über der Zeile hinzugefügt.

  4. Wählen Sie (Notiz#), um eine Liste der vorhandenen Anmerkungen im Jahresabschluss anzuzeigen.

  5. Wählen Sie eine Erläuterung aus der Liste aus, um sie in der Zeile zu referenzieren.

    Sie können auch Verborgene Notizen anzeigen wählen, um eine verborgene Notiz zu referenzieren.

  6. Wählen Sie SPEICHERN oder SPEICHERN & SCHLIEßEN.

Die Nummer der referenzierten Erläuterung/Notiz wird neben dem Zeilennamen angezeigt.

Sie können auch in Spalten auf Erläuterungen verweisen.

So fügen Sie eine Nummerierungsreferenz in einer Spalte hinzu:

  1. Fügen Sie der Tabelle eine Spalte hinzu.

    1. Wählen Sie die Tabelle aus.

      Symbole für Tabellenoptionen werden rechts angezeigt.

    2. Wählen Sie Weitere Aktionen () und anschließend Dynamische Tabelle bearbeiten.

      3. Die Tabellenoptionen im Bearbeitungsmodus.

    3. Gehen Sie zu der Stelle, an der Sie eine Spalte hinzufügen möchten.

    4. Wählen Sie Hinzufügen () am oberen Tabellenrand.

      5. Der Tabelle wird eine Spalte hinzugefügt und die Spalteneinstellungen werden auf der Seite angezeigt.

  2. Markieren Sie die Überschrift der neuen Spalte und ändern Sie den Spaltentyp in Nummerierungsreferenz.

  3. Geben Sie eine Überschrift ein und legen Sie die gewünschte Breite fest.

  4. Klicken Sie auf Fertig.

    5. Die neue Spalte wird in der Tabelle mit leeren Verweisen (#) hinzugefügt.

  5. Geben Sie die Nummerierungsreferenz für jede Zeile an.

    1. Wählen Sie einen leeren Verweis (#), um eine Liste der im Jahresabschlussdokument vorhandenen Anmerkungen anzuzeigen.

    2. Wählen Sie eine oder mehrere Anmerkungen aus der Liste aus.

      3. Sie können auch Ausgeschlossene Notizen anzeigen wählen, um eine ausgeschlossene Notiz zu referenzieren.

  6. Wählen Sie SPEICHERN oder SPEICHERN & SCHLIEßEN.

Bitte beachten Sie: Sie können direkt zu der referenzierten Notiz im Bereich Anhänge zum Jahresabschlussnavigieren, indem Sie die soeben erstellte Nummerierungsreferenz verwenden.

Sie können auch die oben genannten Methoden verwenden, um eine Nummerierungsreferenz zu ändern, oder Sie können sie direkt in der Tabelle ändern.

Um eine Nummerierungsreferenz direkt in der Tabelle zu ändern:

  1. Wählen Sie die Nummerierungsreferenz aus, die Sie ändern möchten.

    Eine Liste der vorhandenen Anmerkungen im Jahresabschluss wird angezeigt

  2. Wählen Sie eine oder mehrere Erläuterungen aus der Liste aus, auf die Sie in der Zeile oder Spalte verweisen möchten.

    Sie können auch Verborgene Notizen anzeigen wählen, um verborgene Notizen zu referenzieren.
