Legen Sie eine Standardanzeigeoption für Konten und Kontengruppen im Jahresabschluss fest

Sie können vordefinieren, wie Gruppen und Konten in jeder Tabelle im Jahresabschluss angezeigt werden. Sie können zum Beispiel eine Tabelle so einstellen, dass Gruppen standardmäßig zugeklappt angezeigt werden.

So legen Sie die Anzeigeoption für Gruppen und Konten in einer Tabelle fest:

  1. Wählen Sie die gewünschte Tabelle aus.

  2. Wählen Sie Weitere Aktionen () | Dynamische Tabelle bearbeiten.

  3. Klicken Sie auf Einstellungen ().

  4. Wählen Sie die gewünschte Option für Konten.

    5. Verfügbare Optionen:

    • Ausschließen - Gruppen nur in der Tabelle anzeigen.

    • Eingeschlossen (Zugeklappt) - Konten und Gruppen in der Tabelle anzeigen, wobei die Gruppen standardmäßig zugeklappt sind.

    • Eingeschlossen (Aufgeklappt) - Konten und Gruppen in der Tabelle anzeigen, wobei Gruppen standardmäßig aufgeklappt sind.

  5. Wählen Sie SPEICHERN oder SPEICHERN & SCHLIEßEN.
