Nicht-konsolidierte Salden im Jahresabschluss verwenden

Sobald Sie ein konsolidiertes Engagement erstellt haben, werden die konsolidierten Salden im gesamten System verwendet. Sie haben jedoch drei Möglichkeiten, nicht konsolidierte Salden anzuzeigen:

  • Legen Sie für das gesamte Dokument die Verwendung nicht konsolidierter Salden fest.

  • Legen Sie für eine oder mehrere Tabellen fest, nicht konsolidierte Salden anzuzeigen.

  • Legen Sie für eine oder mehrere Tabellenspalten fest, nicht konsolidierte Salden anzuzeigen.

Sie können das Dokument zum Beispiel so einstellen, dass nicht konsolidierte Daten verwendet werden, wenn Sie getrennte Jahresabschlüsse für die Muttergesellschaft und ihre Tochtergesellschaften erstellen müssen.

Legen Sie für das Dokument die Verwendung nicht konsolidierter Salden fest

Zur Verwendung nicht konsolidierter Daten in Ihrem Jahresabschluss:

  1. Öffnen Sie den Jahresabschluss und gehen Sie zur Dokumentennavigation.

  2. Wählen Sie Einstellungen () und aktivieren Sie dann das Kontrollkästchen Nicht konsolidierte Daten verwenden.

    Die konsolidierten Daten sind im Finanzbogen nicht mehr verfügbar.

Um die konsolidierten Daten wieder in die Jahresabschlüsse aufzunehmen, deaktivieren Sie das Kontrollkästchen Nicht konsolidierte Daten verwenden.

Legen Sie für eine Tabelle fest, nicht konsolidierte Salden anzuzeigen.

So zeigen Sie nicht konsolidierte Daten in einer Tabelle an:

  1. Wählen Sie die gewünschte Tabelle aus.

    2. Symbole für Tabellenoptionen werden rechts angezeigt.

  2. Wählen Sie Weitere Aktionen () | Dynamische Tabelle bearbeiten.

    3. Die Tabellenoptionen im Bearbeitungsmodus.

  3. Wählen Sie Einstellungen () | Konsolidierung und aktivieren Sie dann das Kontrollkästchen Nicht konsolidierte Daten verwenden.

Um die konsolidierten Daten wieder in die Tabelle aufzunehmen, deaktivieren Sie das Kontrollkästchen Nicht konsolidierte Daten verwenden.

Legen Sie für eine Tabellenspalte fest, nicht konsolidierte Salden anzuzeigen

So zeigen Sie nicht konsolidierte Daten in einer Tabellenspalte an:

  1. Wählen Sie die gewünschte Tabelle aus.

    2. Symbole für Tabellenoptionen werden rechts angezeigt.

  2. Wählen Sie Weitere Aktionen () | Dynamische Tabelle bearbeiten.

    3. Die Tabellenoptionen im Bearbeitungsmodus.

  3. Wählen Sie die gewünschte Spalte.

  4. Aktivieren Sie das Kontrollkästchen Nicht konsolidierte Daten verwenden.

    Um die konsolidierten Daten wieder in die Spalte aufzunehmen, deaktivieren Sie das Kontrollkästchen Nicht konsolidierte Daten verwenden.
