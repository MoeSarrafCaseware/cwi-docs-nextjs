Konten- und Kontengruppendetails im Jahresabschluss anzeigen

Sie können in jeder Jahresabschlusstabelle ein bestimmtes Konto auswählen, um die zugehörigen Daten der Hauptabschlussübersicht anzuzeigen

Um die Kontodetails anzuzeigen, wählen Sie das Konto aus, um das Popup-Fenster Kontodetails“ anzuzeigen, in dem die Daten der Hauptabschlussübersicht für dieses Konto aufgeführt sind. Bitte beachten Sie:

  • Beachten Sie, dass bei Auswahl einer Zelle mit überschriebenem Saldo im Popup-Fenster Kontodetails eine hervorgehobene Zeile mit dem ursprünglichen Saldo für diese Zelle (dem HÜ-Betrag) angezeigt wird Um zum ursprünglichen Betrag zurückzukehren, wählen Sie Zurücksetzen (). Weitere Informationen finden Sie in Salden im Jahresabschluss überschreiben.

  • Wenn eine Rundungsdifferenz auf das Konto angewendet wird, zeigt das Popup-Fenster Kontodetails auch eine hervorgehobene Zeile mit den Rundungsdetails an. Weitere Informationen finden Sie in Rundungsdetails im Jahresabschluss anzeigen.

  • In konsolidierten Engagements wird im Popup Kontodetails eine zusätzliche separate Spalte für Konsolidierungsbuchungen angezeigt.

    Hinweis: Diese Spalte wird nicht angezeigt, wenn Sie die Anzeige von nicht konsolidierten Salden für den Jahresabschluss ausgewählt haben. Weitere Informationen finden Sie in Nicht konsolidierte Salden im Jahresabschluss verwenden.

Sie können Gruppen auf- oder zuklappen, um entweder Daten für jedes Konto in der Gruppe oder den Gesamtbetrag für die Konten in einer Gruppe anzuzeigen.

Bitte beachten Sie: Sie können Gruppen nur dann zuklappen oder aufklappen, wenn die Einstellung für Konten in der Tabelle entweder Included (Zugeklappt) oder Included (Aufgeklappt) ist. Weitere Informationen finden Sie in Legen Sie eine Standardanzeigeoption für Konten und Kontengruppen im Jahresabschluss fest.
