Eine Währung hinzufügen

Sie können eine Währung zu Ihrem Produkt hinzufügen, um zusätzliche Währungen zu unterstützen:

  1. Wählen Sie Format | Währung und anschließend Währung hinzufügen.

  2. Geben Sie den Namen der Währung in das Textfeld ein, um die Liste der Währungen zu filtern.

    3. Sie können die Liste auch nach dem dreibuchstabigen Währungscode aus ISO 4217 filtern.

  3. Wählen Sie die gewünschte Währung, um sie zu Ihrem Produkt hinzuzufügen.

    4. Sie können auch eine Währung aus Ihrem Produkt entfernen, indem Sie () wählen.
