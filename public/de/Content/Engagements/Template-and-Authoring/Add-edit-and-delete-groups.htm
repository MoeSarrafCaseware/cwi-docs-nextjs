Customize financial groups

In the Data page, you can manually add trial balance groups to a grouping structure in the firm template, if needed. You can also edit or delete any groups you create. Note that you cannot edit or delete any pre-existing groups in the template.

Note: To allow staff to add financial groups, select the Enable user-defined financial groups checkbox on the Data tab of the product settings. Alternatively, deselect the checkbox to restrict this setting.

Eine Gruppe hinzufügen

You can manually add groups to a grouping structure.

Eine Gruppe hinzufügen:

  1. Go to the Data page, then select the grouping structure you want to add a group to from the left pane:

    • Financials groupings

    • Tax groupings

    • Prescribed groupings.

  2. Select Add () | Add group.

    The Group Properties dialog displays.

  3. Set the properties for the new group.

    Refer to Group properties for a description of each group property.

  4. Select Save to add the group.

Edit a group

You can edit the properties of a group you have added to the firm template.

To edit a group:

  1. Go to the Data page, then select the grouping structure from the left pane:

    • Financials groupings

    • Tax groupings

    • Prescribed groupings.

  2. Select the group you want to edit, then select Edit () to open the Group Properties dialog.

    Tipp: You can search groups using the Search field on the top right corner of the page.

  3. Edit the group properties as needed.

    Refer to Group properties for a description of each group property.

  4. Once you're done, select Save.

Reorder a group

You can drag and drop groups in the Data page to reorder them. Navigate to the grouping structure in the Data page and drag the group you want to reorder to the desired location.

Note that you can only use drag and drop to reorder groups under the following conditions:

  • The groups you are reordering are all sibling groups in the grouping structure. You cannot drag and drop a child group to be the same level as or higher level than the parent group.

  • All the sibling groups are collapsed or do not have any child groups.

Delete a group

You can delete a group you have added to the firm template.

To delete a group:

  1. Go to the Data page, then select the grouping structure from the left pane:

    • Financials groupings

    • Tax groupings

    • Prescribed groupings.

  2. Select the group you want to edit, then select Edit () to open the Group Properties dialog.

    Tipp: You can search groups using the Search field on the top right corner of the page.

  3. Select Delete.
