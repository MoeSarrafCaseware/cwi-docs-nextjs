Analysis graphs in financial statements

A visual presentation of your client's account information is available in the financial statements to help you to help you present the financial results to your client.

You can also include the Financial Statements Analysis Graphs section in the financial statements to populate a visual representation of your client's data in a designated section.

From the 109 Financial statement optimiser, respond Yes to the Would you like to include analysis graphs in your financial statements? procedure. For Select the type of analysis graphs you would like to add, select Ratio and trend, Note disclosures or Both.

As you review the information, you can:

  • Choose which analyses figures to view using the Select Graph drop-down.

    • Tipp: You can also close a figure () to have it removed from the display.

  • Display the figure in a full size across the section select using the Full size icon ().

  • Select the Edit icon () in the figure's title bar, then select the Add icon () to add an item to the figure or select Remove () to remove an item from the figure.

    Hinweis: The Add icon () is only available if one or more of the predefined items in the graph are not included by default. Otherwise, the Remove icon () is only available.

  • Add comments or notes at the Edit documentation comments… text field.

  • Choose which table columns to view using the COLUMNS button.

  • Select the Reset to default icon () to reset the figure to its original state.
