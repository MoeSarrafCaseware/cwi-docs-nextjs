Controls

Complete the documents in the Controls folder in the following sequence:

  1. 535 Identifying control risk factors by accounting cycle
    Use this document to identify control risk factors by accounting cycle.

  2. 575 Internal control deficiencies identified - worksheet
    Use this document to record control deficiencies identified during the audit (on forms such as 520, 522, 530 and 540 to 560) and those identified from the results of audit procedures performed.

If required, add a Control as you progress through your engagement.

To add a control:

  1. Select the Add icon (), then New | Control.

  2. In the Create New Control dialog, add the control name and description.

    Note that the Risks drop down menu will only appear in the controls dialog if there is already an existing risk in place. To learn more, see Add a risk.

  3. You can also select the Add icon () to add an IT dependent report or IT system affected.

    Once the dialog is complete, select SAVE. Your control appears on the bottom right side of the screen.

  4. Select OPEN to expand your control.
