Import the client's data

If you're preparing financial statements as part of your tax engagement, you can import your client's trial balance and transactions data.

You can request the client's financial data using the 1-100Q - Financial data and general entity information queries and responses query document. To learn more, see Client correspondence (Queries).

Caseware Taxflow allows you to import trial balance and transactions data from a variety of sources:

  • Transfer individual engagements with CloudBridge

  • Import trial balance from an engagement

    Tipp: Caseware Taxflow uses the same standard taxonomy structure as OnPoint apps. When you import the trial balance from an OnPoint Audit or OnPoint PCR engagement, the accounts will be automatically assigned to the same Financial Groupings as the original source.

  • Import trial balance from CSV or Excel file

  • Import transactions from CSV or Excel file

    Note: You must import the client’s trial balance data before you import transactions.

  • Import from Accounting Software or Structured Files:

    • Import from a desktop accounting software using the Cloud Import Utility

    • Import from online accounting software (Quickbooks and Xero)

When you import data using any of the sources listed above, the financial accounts must be assigned to financial groupings. For more information on assigning accounts to groups, see Assign Accounts to groups in the Data page.
