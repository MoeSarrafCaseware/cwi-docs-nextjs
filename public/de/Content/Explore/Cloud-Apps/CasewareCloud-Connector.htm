Was ist Caseware Cloud Connector?

Caseware Cloud Connector is a Microsoft Office Add-in that provides real-time data linkage between your engagement and MS Excel. Mit dieser Verknüpfung können Sie auf die Daten der Summenbilanz, die Eigenschaften des Engagements und die Firmeneigenschaften zugreifen und sie abrufen und dann in eine MS Excel-Datei einfügen. Wenn sich ein verknüpfter Wert oder eine Information im Engagement ändert, wird sie automatisch in der Excel-Datei aktualisiert.

Sie können den Cloud Connector verwenden, um Jahresabschlüsse, Arbeitsblätter und Zeitpläne zu erstellen oder um Berechnungen zu überprüfen.

Beachten Sie, dass Cloud Connector die folgenden Informationen nicht abrufen kann:

  • Salden für Tochtergesellschaften in konsolidierten Engagements

  • Dimensionensalden

  • Vorläufige Salden

  • Buchungen

  • Wesentlichkeitswerte

  • Salden aus verschiedenen Kontengruppenstrukturen

  • Salden aus Vorperiode 2-4. Nur die Salden des laufenden Jahres und der Vorjahre werden unterstützt.

To learn more, see Get started with Caseware Cloud Connector.
