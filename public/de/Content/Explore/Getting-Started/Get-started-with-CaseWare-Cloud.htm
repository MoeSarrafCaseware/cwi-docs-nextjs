Erste Schritte in Caseware Cloud - Mitarbeiter

Bevor Sie in der Caseware Cloud starten, sollten Sie sicherstellen, dass Ihr Firmenadministrator die notwendigen Voraussetzungen geschaffen hat.

  1. Persönliche Einstellungen ändern

  2. Dateien in eine Entität hochladen

  3. Review-Anmerkungen zu einer Kundenentität hinzufügen

  4. Diskussionen posten

  5. Mandantendaten oder Dokumente anfordern

  6. Überblick Engagementstatus anzeigen
