Systemvoraussetzungen

Wir unterstützen den Einsatz von Cloud-Apps auf Rechnern mit einem Minimum von 8GB Arbeitsspeicher, empfehlen aber Geräte zu verwenden, die für eine bessere Leistung über mindestens 16GB Arbeitsspeicher verfügen.

Unterstützte Browser

Sie können sich von jedem Browser, der TLS v1.2 oder aktueller verwendet, bei einem Caseware Cloud Produkt anmelden. Wir unterstützen und empfehlen aber, eine stabile Version (keine Beta) der folgenden Produkte zu verwenden:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

Wir empfehlen eine Bildschirmauflösung von mindestens 1280 x 800 Pixeln und eine Mindestbreite des Browserfensters von 1024 Pixeln.

Unterstützte MS Office-Versionen

Caseware Cloud-Produktsupport:

  • Microsoft Office 2016 oder aktueller

  • Microsoft Office 365 (Desktop-Version)

Unterstützte Working Papers(AuditAgent)-Versionen

Integrieren Sie Working Papers (AuditAgent) in Caseware Cloud, um über einen Webbrowser auf die Daten in Ihrer Mandantendatei zugreifen zu können. Wir unterstützen die letzten beiden Versionen von Working Papers (AuditAgent); die Cloud-Integration steht jedoch ab Working Papers 2017 zur Verfügung. Bitte beachten Sie, dass für einige Funktionen die aktuelle Version erforderlich ist.

Weitere Informationen erhalten Sie in Caseware Support Policy.
