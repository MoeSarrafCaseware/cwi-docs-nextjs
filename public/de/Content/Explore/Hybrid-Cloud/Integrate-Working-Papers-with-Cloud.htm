Caseware Working Papers (AuditAgent) in Caseware Cloud integrieren

Wenn Sie Caseware Working Papers in eine Cloud-Instanz integrieren, erhalten Sie Zugang zu den Cloud-Tools für Zusammenarbeit, Dateimanagement und Nachverfolgung des Auftrags. Außerdem können Sie von jedem beliebigen Ort mit Internetverbindung auf die Daten Ihres Unternehmens zugreifen, was die Arbeit an einem externen Standort erleichtert.

Voraussetzungen:

  • Ihr Unternehmen muss über Arbeitspapiere mit SmartSync-Technologie verfügen. Wenn Sie nicht über SmartSync verfügen, lesen Sie den Abschnitt Unterschiede zwischen SmartSync und nicht-SmartSync und wenden Sie sich für weitere Informationen an Ihren lokalen Distributor.

  • Ihr Unternehmen muss über eine Caseware Cloud-Instanz verfügen. Wenn Sie noch keine Cloud-Instanz haben, können Sie eine Einladung anfordern.

  • Sie müssen einen Internet-Browser verwenden, der TLS 1.2 oder höher unterstützt. Weitere Informationen finden Sie unter Zugriff auf Caseware Cloud für ältere und nicht unterstützte Browser deaktivieren.

Caseware Working Papers (AuditAgent) in Caseware Cloud integrieren:

  1. Stellen Sie sicher, dass Sie Ihre aktuelle Mandantendatei schließen (Datei | Schließen).

  2. Wählen Sie im Menüband Datei | Optionen | Caseware Cloud.

  3. Klicken Sie auf In Caseware Cloud integrieren.

  4. Klicken Sie in der Tabelle auf das Feld Server und geben Sie die URL der Cloud-Instanz Ihres Unternehmens ein. Sie brauchen das Webprotokoll (http:// or https://) nicht anzugeben. Klicken Sie auf OK.

  5. Öffnen Sie eine Working Papers Datei. Der Anmeldebildschirm wird angezeigt, in den Sie Ihre Anmeldedaten eingeben können. Klicken Sie auf Anmelden.

Ihre Kopie von Working Papers wird in die Cloud-Instanz integriert. Sie können durch Wiederholen dieses Vorgangs eine Integration mit weiteren Cloud-Instanzen durchführen. Es kann jedoch immer nur eine Cloud-Instanz aktiv sein. Sie finden Ihre Mandantendateien, indem Sie auf Datei | Öffnen | Caseware Cloud klicken und nach der Entität suchen, in der sich die Dateien befinden.
