Kleine Datensätze (unter 50 Dateien) migrieren

Sie können einen Großteil Ihrer Cloud-Umgebung einrichten, indem Sie vorhandene Engagementdaten migrieren oder manuell neue Daten direkt über die Cloud-Schnittstelle erstellen. Diese Migrationsmethode wird für kleinere oder neue Datensätze empfohlen.

Voraussetzungen:

  • Ihr Unternehmen muss über Arbeitspapiere mit SmartSync-Technologie verfügen. Wenn Sie nicht über SmartSync verfügen, lesen Sie den Abschnitt Unterschiede zwischen SmartSync und nicht-SmartSync und wenden Sie sich für weitere Informationen an Ihren lokalen Distributor.

  • Ihr Unternehmen muss über eine Caseware Cloud-Instanz verfügen. Wenn Sie noch keine Cloud-Instanz haben, können Sie eine Einladung anfordern.

  • Sie müssen einen Internet-Browser verwenden, der TLS 1.2 oder höher unterstützt. Weitere Informationen finden Sie unter Zugriff auf Caseware Cloud für ältere und nicht unterstützte Browser deaktivieren.
