Globale Benachrichtigungen

Unter Meine Einstellungen | Benachrichtigungen | Globale Benachrichtigungen können Sie konfigurieren, welche Ereignisse dazu führen, dass Sie eine E-Mail-Benachrichtigung erhalten.

Name Beschreibung
Optionen für Schnellabonnement Aktivieren Sie dieses Kontrollkästchen, um Benachrichtigungen für alle Aktionen zu erhalten, die Kontakte an von Ihnen abonnierte Entitäten, Aktivitäten oder Dateien durchführen.
Systemweite Abonnementeinstellungen

Aktivieren Sie ein Häkchen, um alle Aktionen dieses Typs zu abonnieren, die in Cloud vorgenommen wurden.

Wenn Sie zum Beispiel das Häkchen Entitäten in der Spalte Bearbeiten aktivieren, erhalten Sie eine E-Mail-Benachrichtigung, sobald ein Benutzer eine Entität bearbeitet.
Individuelle Abonnementeinstellungen

Aktivieren Sie ein Häkchen, um alle Aktionen dieses Typs zu abonnieren, die an Aktivitäten, Dateien, Entitäten oder Inhalten von Entitäten vorgenommen wurden.

Wenn Sie zum Beispiel Aktivitäten unter Inhalte von abonnierten Entitäten in der Spalte Erstellen aktivieren, erhalten Sie eine E-Mail-Benachrichtigung, wenn eine neue Aktivität in einer von Ihnen abonnierten Entität erstellt wird.

Wird das Häkchen Entitäten (oberste Ebene) aktiviert, erhalten Sie eine E-Mail-Benachrichtigung für alle Cloud-Inhalte, die mit von Ihnen abonnierten Entitäten verbunden sind.
