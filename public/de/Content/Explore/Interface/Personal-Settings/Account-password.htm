Kontoeinstellungen

Unter Meine Einstellungen | Andere | Kontoeinstellungen können Sie Ihr bestehendes Kennwort ändern und die Zwei-Faktor-Authentifizierung aktivieren.

Sie können auch Ihr Kennwort ändern, bevor Sie sich auf der Anmeldeseite anmelden.

Name Beschreibung

Neues Kennwort bestätigen

 Geben Sie das neue Kennwort erneut ein.
Land Wählen Sie das Land Ihres Wohnsitzes.
Aktuelles Kennwort Geben Sie das aktuelle Kennwort für die Anmeldung am Caseware Cloud-Konto an.
Mobiltelefon Geben Sie Ihre Telefonnummer ein.

Neues Kennwort

 Geben Sie das neue Kennwort ein, das Sie für das Cloud-Konto verwenden möchten. Dieses Kennwort darf nicht eines der letzten zehn verwendeten Kennwörter sein.
Anforderungen an Kennwortqualität Zeigt die Standardanforderungen an Kennwortqualität an.
Telefonnummer aktualisieren Wählen Sie diese Option, um die mit Ihrem Konto verknüpfte Telefonnummer zu ändern.
Verifizierungscode Geben Sie den Verifizierungscode ein, der an Ihre Telefonnummer gesendet wurde.
