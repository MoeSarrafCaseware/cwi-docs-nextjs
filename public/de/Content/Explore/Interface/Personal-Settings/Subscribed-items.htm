Abonnierte Elemente

Unter Meine Einstellungen | Benachrichtigungen | Abonnierte Elemente können Sie den Status des Abonnements für den individuellen Inhalt anzeigen und aktualisieren. Abonnierter Inhalt wird mit Hyperlinks zum Inhalt angezeigt.

Sie sind automatisch für alle Inhalte abonniert, die Sie in Cloud hinzugefügt haben. Eine E-Mail-Benachrichtigung wird immer dann gesendet, wenn Änderungen am Inhalt vorgenommen werden. Wenn z. B. ein Benutzer eine Aktivität kommentiert, die Sie gepostet haben, werden Sie per E-Mail benachrichtigt.

Sie erhalten keine E-Mail-Benachrichtigungen für von Ihnen vorgenommene Aktualisierungen.
