VoIP-Einstellungen

Unter Meine Einstellungen | Andere | VoIP-Einstellungen können Sie die VoIP-Integration in Cloud aktivieren. Nach der Aktivierung werden alle Telefonnummern in Links konvertiert, die von Ihrem Computer gewählt werden können. Das Klicken auf diese Links startet Ihre Standard-VoIP-Anwendung. Eine VoIP-Anwendung muss auf Ihrem Computer installiert sein, damit dieses Feature funktioniert.

Zu den unterstützten Telefonclients gehören:

  • Skype
  • Google Voice
  • Microsoft NetMeeting

Ist keine Landesvorwahl in einer Telefonnummer angegeben, wird die Standard-Landesvorwahl der Telefonnummer vorangestellt. Fragen Sie Ihren Cloud-Administrator, ob diese Einstellungen in Einstellungen festgelegt ist.
