Deaktivieren des Zugriffs auf Caseware Cloud für ältere und nicht unterstützte Browser

Veröffentlicht: 18. Juni 2020

Kurz nach der Veröffentlichung von Caseware Cloud 31 wird Caseware den Zugriff auf Caseware Cloud für ältere und nicht unterstützte Webbrowser deaktivieren. Diese älteren und nicht unterstützten Browser verwenden veraltete Verschlüsselungsprotokolle – TLS Version 1.0 und v1.1.

Der PCI Security Standards Council hat allen Organisationen, die seine „Best Practices“ befolgen, empfohlen, von SSL/TLS auf TLS v1.2 oder höher zu migrieren Diese Änderung wird in der gesamten Branche durchgesetzt, um sichere Verbindungen aufrechtzuerhalten, die Ihre sensiblen Daten verschlüsseln und vor böswilligen Zugriffen schützen. Caseware wird dieser Empfehlung bei allen eingehenden Verbindungen folgen.

Was ist TLS?

TLS steht für "Transport Layer Security.“ Es handelt sich um ein Protokoll, das den Datenschutz und die Datenintegrität zwischen zwei kommunizierenden Anwendungen gewährleistet. Es ist das heute am weitesten verbreitete Sicherheitsprotokoll und wird für Webbrowser und andere Anwendungen verwendet, bei denen Daten sicher über ein Netzwerk ausgetauscht werden müssen. TLS stellt durch Verschlüsselung und Überprüfung der Identität des Endpunkts sicher, dass eine Verbindung zu einem entfernten Endpunkt der beabsichtigte Endpunkt ist. Die bisherigen Versionen von TLS sind TLS 1.0, 1.1, 1.2 und 1.3.

Wer ist betroffen?

Caseware Cloud Kunden (Mitarbeiter und Kontakte) und Hybrid Cloud Kunden, die ältere und nicht unterstützte Webbrowser für den Zugriff auf Caseware Produkte verwenden, sind davon betroffen. Die vollständige Liste der betroffenen Webbrowser finden Sie im Abschnitt Kompatibilität mit Internet-Browsern weiter unten.

So überprüfen Sie, ob Sie einen nicht unterstützten Browser verwenden:

  1. Verwenden Sie dieses Tool, um die SSL/TLS-Fähigkeiten Ihres Browsers zu überprüfen.
  2. Wenn der Browser aktualisiert werden muss (d. h. er zeigt nicht 1.2 oder höher an), finden Sie auf dieser Website Schritte, um TLS 1.2 oder höher zu aktivieren: Enabling TLS 1.1 and TLS 1.2 on web browsers (in Englisch).

Benutzererfahrung, wenn keine Maßnahmen ergriffen werden

Nutzer können sich nicht mit Caseware Cloud verbinden, wenn ihr Browser oder ihre Browsereinstellungen nicht gemäß den unten genannten Kompatibilitätsrichtlinien aktualisiert werden.

  1. Caseware Cloud Benutzer

    Benutzer, die keinen kompatiblen Browser verwenden, erhalten eine Fehlermeldung auf der Anmeldeseite der Caseware Cloud und können sich nicht mit dem Server verbinden. Sobald der Browser oder die Browsereinstellungen aktualisiert sind, können die Benutzer die Seite aktualisieren, um auf den Anmeldebildschirm zuzugreifen.

  2. Benutzer von Working Papers mit Cloud-Integration für SmartSync

    Bei Benutzern, die keinen kompatiblen Browser verwenden, werden einige oder alle der folgenden Probleme auftreten:

    Working Papers 2018 oder aktueller

    • Sie erhalten eine Fehlermeldung, wenn Sie sich von Working Papers aus mit dem Anmeldebildschirm der Caseware Cloud verbinden.
    • Sie können keine Dateien auf der Registerkarte SmartSync Server unter Datei | Öffnen sehen oder darauf zugreifen.
    • Sie können keine Verbindung zum Server herstellen (d. h. online gehen), um Änderungen aus Ihrer lokalen Synchronisierungskopie zu synchronisieren. Eine Sprechblase in der unteren rechten Ecke zeigt an, dass die Datei offline ist.

    Working Papers 2017 oder älter

    • Sie können keine Dateien auf der Registerkarte SmartSync Server unter Datei | Öffnen sehen oder darauf zugreifen.
    • Sie können keine Verbindung zum Server herstellen (d. h. online gehen), um Änderungen aus Ihrer lokalen Synchronisierungskopie zu synchronisieren. Eine Sprechblase in der unteren rechten Ecke zeigt an, dass die Datei offline ist.

Sobald der Browser oder die Browser-Einstellungen aktualisiert wurden, kehren die Working Papers-Dateien in ihren Online-Status zurück, wenn sie mit dem Internet verbunden sind. Sie müssen sich möglicherweise bei Caseware Cloud anmelden, wenn Ihre Firmeneinstellungen so konfiguriert sind, dass Sie sich täglich bei Caseware Cloud anmelden müssen.

Kompatibilität mit Internet-Browsern

Gemäß unseren Systemanforderungen unterstützt Caseware Cloud die folgenden Browser:

  • Google Chrome (empfohlen)
  • Mozilla Firefox
  • Microsoft Edge

Prüfen Sie mit diesem Tool, ob Ihr Browser TLS 1.2 oder höher unterstützt. Wenn der Browser aktualisiert werden muss (d. h. er zeigt nicht 1.2 oder höher an), finden Sie auf dieser Website Schritte, um TLS 1.2 oder höher zu aktivieren.
