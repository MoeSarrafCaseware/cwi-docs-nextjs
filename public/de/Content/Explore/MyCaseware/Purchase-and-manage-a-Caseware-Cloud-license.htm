Eine Caseware Cloud Lizenz kaufen und verwalten

Um eine Caseware Cloud Lizenz zu erwerben, besuchen Sie https://www.caseware.com/contact/ und füllen Sie das Informationsformular aus. Sie werden von einem Caseware Mitarbeiter kontaktiert, um Ihren Kauf abzuschließen.

Zusätzliche Lizenzen erwerben

Zusätzliche Lizenzen für die Caseware Cloud App können direkt über ein Selbstbedierungssystem in MyCaseware erworben werden. Für jede neue Lizenz, die Sie erwerben, können Sie ein zusätzliches Mitarbeiterkonto anlegen. Für jede erworbene App-Lizenz kann ein weiterer Benutzer auf diese App zugreifen. Weitere Informationen zu den Preisen erhalten Sie, wenn Sie sich auf my.caseware.com anmelden und die Registerkarte Katalog auswählen.

Gehen Sie wie folgt vor, um auf MyCaseware einen Einkauf zu tätigen:

  1. Vergewissern Sie sich, dass Sie über die Rolle des Administrators oder gleichwertige Rechte verfügen.

  2. Wählen Sie im Menü MyCaseware die Option Katalog.

    Die Lizenzen, die Sie erwerben können, werden angezeigt.

  3. Legen Sie Ihre Einkäufe in Ihren Warenkorb, indem Sie auf die gewünschte Cloud Lizenz klicken. Wählen Sie die Anzahl aus und wählen Sie dann In den Warenkorb.

  4. Wenn Sie weitere Einkäufe in Ihren Warenkorb legen möchten, wählen Sie Einkauf fortsetzen oder Warenkorb anzeigen, um zur Kasse zu gehen und den Zahlungsvorgang abzuschließen.

Ihr Kauf ist abgeschlossen. Die aktualisierten Lizenzen, die Sie zur Verfügung haben, werden auf der Registerkarte Lizenzen angezeigt.

Eine Lizenz erneuern

Caseware speichert aus Sicherheitsgründen keine Zahlungsinformationen. Um Ihre Lizenz zu erneuern, kontaktieren Sie unser Vertriebsteam.

Verspätete Zahlungen

Wenn die Zahlung nicht bis zum angegebenen Datum eingeht, wird Ihr Konto gesperrt und alle mit diesem Konto verbundenen Daten oder Dateien sind nicht mehr zugänglich. Sobald die Zahlung eingegangen ist, wird das Konto ohne Datenverlust wiederhergestellt.
