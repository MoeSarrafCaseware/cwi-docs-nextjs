Modify signoff roles

OnPoint PCR offers the ability to customize signoff roles. Custom signoff roles allow you to define labels for preparer and reviewer roles and create custom signoff schemes.

OnPoint PCR has 2 default roles:

  • Prepared By

  • Reviewed By

If your firm requires additional role types, you can create new roles. To view or customize your signoff roles, select the Information icon () at the top menu, then select Settings to open the Engagement Settings dialog.

To modify signoff roles, see Define signoff roles.

To create custom signoff schemes to track reviewer and preparer workflows, see Set up signoff schemes.
