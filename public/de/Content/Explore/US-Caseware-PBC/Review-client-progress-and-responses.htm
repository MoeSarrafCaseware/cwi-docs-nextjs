Überprüfung der Fortschritte und Antworten der Kunden

Öffnen Sie die Abfrage, um mit der Überprüfung der Antworten zu beginnen. Notice how the time of the response and the name of the contact displays with the provided documents at each file upload request in the document.

Wenn die in einer Antwort enthaltenen Informationen gültig sind, können Sie die Antwort akzeptieren. Select the Responded icon () next to the query question, then choose Accept response.

Sie können auch eine Antwort an die Kontaktperson zurückschicken, wenn die bereitgestellten Informationen oder Dokumente unzureichend oder falsch sind oder einer Klärung bedürfen. Select the Responded icon () next to the query question, then choose Return to contact.

Sie können auch die Filter im linken Fensterbereich verwenden, um den Fortschritt der Kontakte zu überprüfen und durch das Dokument zu navigieren, während Sie die Antworten überprüfen.
