What's new - Caseware Sherlock 4.1

Find out what's new in Caseware Sherlock 4.1.

Entity level security

Access to entity data is now restricted to users with the Viewer permission for the entity. When a user opens a Sherlock application, the entities available to them are automatically filtered to display only those that they can access.

Persistent visualization controls and filters

Customizations to visualization controls and filters in Sherlock Builder and Reader now persist across sessions. You no longer need to reapply these settings with each visit.
