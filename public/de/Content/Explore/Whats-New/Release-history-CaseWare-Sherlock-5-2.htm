What's new - Caseware Sherlock 5.2

Find out what's new in Caseware Sherlock 5.2.

Flush request prioritization

The File Storage Service (FSS) now prioritizes flush requests from licensed Sherlock firms. This ensures that all data extraction requests are processed in a timely manner.

Retry dataset refresh

We've added an automatic "retry" function to daily dataset refreshes in Sherlock Builder and Reader. This ensures completion of the daily refresh, even in the case of an internal error.
