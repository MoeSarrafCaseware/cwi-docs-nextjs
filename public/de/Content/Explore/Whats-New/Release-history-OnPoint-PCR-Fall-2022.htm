What’s new - OnPoint PCR Fall 2022 Release (v1.3320.0)

Here is what’s new in OnPoint PCR Fall 2022.

Annual tax export updates

The tax export feature now includes the updated 2022 tax exports and tax codes for the following tax vendors:

  • Intuit's Lacerte© Tax
  • Thomson Reuters UltraTax CS®

For a listing of changes, see Tax export codes.

Improved guidance

To provide better in-product guidance during your workflow, the following enhancements have been made:

Interactive welcome guide

To provide you with a seamless onboarding experience, a new three-minute welcome tour now automatically displays to introduce you to the key areas of an OnPoint PCR engagement. The tour includes screenshots, short videos and links to additional online help topics to help you effectively navigate and perform your engagement.

The welcome guide can be accessed at any time from the Resource Center.

Interactive Resource Center

A new Resource Center has been added to share information about upcoming OnPoint PCR releases and new product features. You can access the Resource Center from the bottom right corner of the engagement.

External documents export for OnPoint CloudBridge

To provide a more efficient way to transfer data from Working Papers to Cloud engagements, OnPoint CloudBridge now allows you to export several external document formats such as PDFs, Excel and Word documents.

For more information about how to add CloudBridge to Working Papers engagement files, see Transfer individual engagements with CloudBridge.

To download and install the latest version of OnPoint CloudBridge, see Get CloudBridge.

Fixes

The following list includes a brief description of the issues that have been resolved in this release.

  • When you create a Preparation or Compilation engagement, the 2-500 Transactions Import and 2-550 Data completeness documents display on the Documents page. The documents now only display in Review engagements (v.1.3320.0).
  • The tax export file for UltraTax® CS incorrectly displays a positive balance for credit accounts (v.1.3320.0).
  • In the 1120-S form:
    • The 2021 tax code for PPP Loans, EIDL and Oth COVID grants for CCH Axcess™ Tax/CCH ProSystem fx® Tax incorrectly displays 84422. The tax code has been updated to 04422 (v.1.3320.0).
    • Duplicate Depreciation expense line items exist (v.1.3320.0).
