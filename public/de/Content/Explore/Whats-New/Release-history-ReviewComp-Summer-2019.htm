What's new - Caseware ReviewComp Summer 2019

Here is a listing of what's new for the Summer 2019 release of Caseware ReviewComp.

Adding a firm logo

Your firm logo is now linked to the engagement. Once you upload the logo in Cloud, it automatically displays in the designated logo placeholders in letters and in the Notice to the Reader and Independent practitioners report sections in the financial statements document.

To learn more, see Add a firm logo and letterhead.

Financial statements notes

We have introduced a second subcategory in the notes to the financial statements to improve browsing, scannability and readability. For example, revenue recognition is now broken down to 8 subcategories, whereas previously, it included one long text section.

Zusammenfassung der falschen Darstellungen

The 4-10-1 Worksheet - Accumulation of identified misstatements - identification document now includes a bar graph to visually represent the adjusting entries.

The graph shows the five main categories (Assets, Liabilities, Equity, Revenues and Expenses) and the available adjustments. You can select which adjustment types to display on the graph.

In the example below, we have toggled Proposed - Factual adjustments and All Corrected to only view these types of entries on the graph.

Fixes

  • Fixed the Preview option in the Notes library. You can now view a preview of a note before you add it in the financial statements.

  • Trial balance group numbers no longer display next to the group name.

Known issues

3-02 Assets is incorrectly showing the wrong analysis chart.
