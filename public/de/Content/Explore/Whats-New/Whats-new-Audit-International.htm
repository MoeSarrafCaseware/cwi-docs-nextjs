What's new in Audit International v2024.1.0 (2.698)

This version of Audit International (Cloud) includes the changes and new requirements from ISA 600 (Revised), Special Considerations - Audits of Group Financial Statements (Including the Work of Component Auditors), which are intended to strengthen the auditor’s responsibilities related to group audits. Additionally, the documents listed in this summary have been enhanced to provide more user-friendly content to streamline the audit workflow, as a result of feedback provided by users and practice reviewers/regulators.

Here’s what’s new in Audit International v2024.1.0 (2.698).

Content updates

Changes related to ISA 600

To ensure compliance with the new ISA 600 changes, we've implemented the following:

  • Changes to the workflow when assessing group audit considerations: the decision around group audit has been moved to the A200 - Preliminary engagement activities document to ensure that the auditor performs this evaluation at the beginning of the audit work. Additional considerations for group audit will still be documented in A300 - Overall audit strategy.

  • Certain documents have been removed from the Audit International product (see below for the full list of documents). These have been replaced with new documents/procedures in existing documents.

New documents

To assist auditors with the new ISA 600 standard (and also in other areas), we've added the following documents:

  • A360 - Worksheet - Selecting a component auditor

  • A510 - Group audit - Component materiality

  • B335 - Understanding the financial reporting process

  • B510 - Group audit program - Engagement scoping components

  • C160 - Financial reporting - Audit procedures

  • C165 - Worksheet - Evaluating the work of a component auditor

  • C205 - Letter to a component auditor

  • C210 - Instructions for a component auditor

Analytics

To assist auditors and better integrate the analytics tests into the audit work, we've made some changes to the Analytics functionality of the product.

The following documents have been removed from the product:

  • B010 - Influencers

  • B040 - Analytics Insight

These documents have been replaced with a new technology stack that is embedded in our documentation. Users will now have access to the analytics tests directly from the C145 - Use of journal entries document.

Note: This functionality is being further developed. For any additional information, contact us at europe.support@caseware.com.

Settings form for Substantive sampling documents

We've added a settings form for the Substantive sampling documents to the System Documents folder, which will be available in the template. This form allows you to customize the confidence levels you'll use or any additional reliance factors at the firm level.

Bulk actions

The Documents page now includes the option to modify multiple documents at once. When you enable the Bulk actions option from the More actions () menu on the Documents page, you can select multiple documents and then move, delete or sign off all the documents at once.

Cloud Connector

Caseware Cloud Connector is a Microsoft Office add-in that creates a real-time data linkage between your engagement and Excel. With that linkage in place, you can access and retrieve trial balance data and engagement and firm properties, then insert the information into an Excel file. If a linked value or information changes in the engagement, it's automatically updated in the Excel file.

Caseware Cloud Connector will be available in your region soon. For any additional information, contact us at europe.support@caseware.com.
