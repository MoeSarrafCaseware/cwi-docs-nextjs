Here's what's new for the Caseware Accomplish Fall/Winter 2021 release.

Improvements have been made to Accomplish to enhance product translation.

Accomplish now has a brand new look and feel. You can continue to use the existing features and functionality within the new layout.

Fixes

The following list features a brief description of the issues that have been resolved in this release.
