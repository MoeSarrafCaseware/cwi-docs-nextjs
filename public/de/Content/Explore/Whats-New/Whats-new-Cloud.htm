Prior engagements

We’ve heard from many of you that you rarely access older engagements, so with that knowledge, we’ve added a new setting that will accelerate page load times and file searches in Caseware Cloud. This new setting lets you specify when to label engagements as ‘Prior engagements’ after a defined period elapses past the year-end date.

Once enabled, staff will only see the list of active engagements, but they can easily toggle between active and prior using the option at the top of the file list.
