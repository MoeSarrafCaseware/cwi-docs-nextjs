tag

  1. Keyword that you can assign to people, entities, or files in Caseware Cloud. Tags können als Suchbegriffe verwendet werden und erleichtern den Benutzern das Auffinden von Dokumenten und Mitarbeitern bzw. Kontakten. Administrators can create both system-wide tags that can be assigned to all categories and specialized tags that users can only assign to staff, contacts, files, or entities. See also Create custom fields, tags and workflows.

  2. In XBRL, a standardized identifier that is used to describe a specific item of financial data. The purpose of an XBRL tag is to provide a clear, concise definition of data items in business reports, which can be easily understood by both humans and machines. See also XBRL fact.
