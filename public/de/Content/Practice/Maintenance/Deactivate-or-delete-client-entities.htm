Kundenentität deaktivieren oder löschen

Sollten Sie Kundenentitäten haben, die nicht mehr aktiv sind, können Sie diese deaktivieren oder löschen.

Durch das Deaktivieren einer Entität bleiben ihre Daten erhalten und es werden die Mitglieder Ihrer Organisation informiert, dass weitere Arbeiten für diese Entität nicht abgeschlossen werden sollen. Durch das Löschen einer Entität wird sie komplett mit allen Aktivitäten und Dateien aus Cloud entfernt.

In den meisten Fällen sollte eine Entität deaktiviert und nicht gelöscht werden.

Eine Entität deaktivieren

Das Deaktivieren einer Entität kennzeichnet die Entität mit dem Status Inaktiv. Dies informiert die Mitarbeiter, dass die Entität nicht mehr länger aktiv ist, während alle Dateien und Aktivitäten erhalten bleiben.

Sie können eine inaktive Entität jederzeit reaktivieren, aber wenn die Entität gelöscht wurde, muss sie neu erstellt werden.

Hinweis: Entitäten mit Rechnungsentwürfen können nicht inaktiv werden. Stellen Sie alle Rechnungsentwürfe fertig, bevor Sie eine Entität als aktiv kennzeichnen.

Eine Entität deaktivieren:

  1. Stellen Sie sicher, dass Sie über die Rolle Entitäten-Administrator bzw. gleichwertige Berechtigungen verfügen.

  2. Stellen Sie sicher, dass alle Zwischenabrechnungen, die mit der Entität verbunden sind, beendet sind. Weitere Informationen zu Abschlussrechnungen finden Sie in Abschlussrechnungen erstellen.

  3. Wählen Sie im Caseware Cloud Hauptmenü auf Entitäten.

  4. Wählen Sie die Entität aus, die Sie deaktivieren möchten. Klicken Sie dann auf das Symbol Bearbeiten ().

  5. In der Seitenleiste wählen Sie Profil.

  6. Wählen Sie in der Drop-down-Liste Status die Option Inaktiv.

  7. Wenn eine Entität über nicht abgerechnete oder nicht abrechenbare Einträge verfügt, haben Sie folgende Möglichkeiten:

    • Wählen Sie Nicht abgerechnete Einträge… und dann Fortfahren. Weitere Informationen zum Übertragen von Einträgen finden Sie in Zeit- oder Kosteneinträge übertragen.

    • Wählen Sie Nicht abgerechnete Einträge löschen… und dann Fortfahren.

  8. Klicken Sie auf Speichern.

Die Entität wurde deaktiviert. Wenn Sie die Entität später reaktivieren möchten, folgen Sie bitte den unten angeführten Schritten.

Eine Entität neu aktivieren:

  1. Stellen Sie sicher, dass Sie über die Rolle Entitäten-Administrator bzw. gleichwertige Berechtigungen verfügen.

  2. Klicken Sie im Hauptmenü auf Entitäten.

  3. Wählen Sie die Entität aus, die Sie deaktivieren möchten. Klicken Sie dann auf das Symbol Bearbeiten ().

  4. Wählen Sie Profil.

  5. Wählen Sie in der Drop-down-Liste Status die Option Aktiv.

Entität löschen

Warnung: In den meisten Fällen wird das Löschen einer Entität nicht empfohlen. Deaktivieren Sie stattdessen die Entität. Weitere Informationen erhalten Sie in Eine Entität deaktivieren.

In seltenen Fällen möchten Sie möglicherweise eine Entität vollständig entfernen. Wenn Sie z. B. eine Entität löschen möchten, die unabsichtlich erstellt wurde.

Hinweis: Entitäten mit zugehörigen Rechnungen können nicht gelöscht werden und sollten stattdessen deaktiviert werden.

Eine Entität löschen

  1. Stellen Sie sicher, dass Sie über die Rolle Entitäten-Administrator bzw. gleichwertige Berechtigungen verfügen.

  2. Klicken Sie im Hauptmenü auf Entitäten.

  3. Wählen Sie die Entität aus, die Sie löschen möchten, und klicken Sie auf Weitere Aktionen () Löschen.

  4. Wenn eine Entität über nicht abgerechnete oder nicht abrechenbare Einträge verfügt, haben Sie folgende Möglichkeiten:

    • Wählen Sie Nicht abgerechnete Einträge… und dann Fortfahren. Weitere Informationen zum Übertragen von Einträgen finden Sie in Zeit- oder Kosteneinträge übertragen.

    • Wählen Sie Nicht abgerechnete Einträge löschen… und dann Fortfahren.

  5. Wählen Sie Ja.

Sie haben die ausgewählte Entität gelöscht. Weitere Informationen zur Erstellung und Verwaltung von Entitäten erhalten Sie in Arbeitsbereiche für Kunden erstellen.
