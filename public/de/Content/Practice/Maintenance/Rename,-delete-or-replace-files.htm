Dateien umbenennen, löschen oder ersetzen

Nachdem Sie eine Datei erstellt haben, können Sie sie bei Bedarf umbenennen, löschen oder ersetzen.

Bevor Sie starten, navigieren Sie zu Cloud Menü () | Dateien und wählen Sie die betreffende Entität aus der Drop-down-Liste.

Dateien umbenennen

Sie können Dateinamen und -beschreibungen bearbeiten, um den Inhalt der Datei besser wiederzugeben.

Hinweis: Wenn Sie eine Working Papers Datei, die Sie in Cloud publiziert haben, umbenennen möchten, empfehlen wir Ihnen, die Datei in einen nicht-synchronisierten Status zu konvertieren und sie mit Hilfe von Working Papers umzubenennen, anstatt die Datei in Cloud umzubenennen. Weitere Informationen erhalten Sie in Eine Datei umbenennen.

Dateiname und -beschreibung bearbeiten:

  1. Wählen Sie die Datei, die Sie bearbeiten möchten, mit Hilfe des Kontrollkästchens.

  2. Klicken Sie im Bereich Details auf das Symbol Bearbeiten ().

  3. Geben Sie einen neuen Namen und eine neue Beschreibung für die Datei ein und wählen Sie dann Speichern.

Dateien löschen

Wenn Sie eine Datei nicht mehr benötigen, können Sie sie löschen.

Dateien löschen:

  1. Wählen Sie die Dateien, die Sie löschen möchten.

  2. Wählen Sie im Bereich Details Weitere Aktionen ()| Löschen.

  3. Bestätigen Sie das Löschen der Dateien.

Wenn Sie eine Datei versehentlich gelöscht haben, können Sie sie aus dem Papierkorb wiederherstellen. Siehe Vorherige Version oder gelöschte Datei wiederherstellen, um weitere Informationen zu erhalten.

Dateien ersetzen

Wenn Sie über eine aktuelle Version einer Datei verfügen, die Sie hochladen möchten, können Sie die Originaldatei ersetzen.

Note: If you want to replace a Working Papers file that you have published to Cloud, you must first delete the Cloud copy, then upload the new copy from your local computer.

Eine Datei ersetzen:

  1. Wählen Sie die Datei, die Sie ersetzen möchten.

  2. Wählen Sie im Bereich Details Weitere Aktionen ()| Ersetzen.

  3. Klicken Sie auf die Schaltfläche Wählen Sie Dateien aus oder ziehen Sie die Datei per Drag-and-Drop in den dafür vorgesehenen Bereich.

Vorherige Version einer Datei löschen

Wenn Sie eine Datei ersetzt haben und Sie die alte Version der Datei nicht mehr benötigen, können Sie sie löschen.

Warnung: Nach dem Löschen eine vorherigen Version können Sie diese später nicht mehr herstellen.

Vorherige Version einer Datei löschen:

  1. Wählen Sie die Datei mit vorherigen Versionen, die Sie löschen möchten.

  2. Erweitern Sie bei Details den Bereich Vorherige Versionen.

  3. Wählen Sie Weitere Aktionen () für die Version, die Sie löschen möchten, und klicken Sie dann auf Löschen.

  4. Klicken Sie im Bestätigungsdialog auf Ja.

Sie können alle vorherigen Versionen einer Datei auch auf einmal löschen.

Alle vorherigen Versionen einer Datei löschen:

  1. Wählen Sie die Datei mit vorherigen Versionen, die Sie löschen möchten.

  2. Wählen Sie im Bereich Details Weitere Aktionen ()| Alle alten Versionen löschen.

  3. Klicken Sie im Bestätigungsdialog auf Ja.
