Beibehaltung definieren

Sie können die Beibehaltung von Dateien so festlegen, dass unerwünschte Elemente nach einer bestimmten Anzahl von Tagen, Monaten oder Jahren aus dem System entfernt werden. Dies gewährleistet, dass die Dateien des Folgejahres nur abgeschlossene Elemente und Berichte beinhalten.

Bevor Sie beginnen, stellen Sie bitte sicher, dass Sie über die Rolle Eigentümer bzw. gleichwertige Berechtigungen verfügen.

Periode Dateibeibehaltung festlegen:

  1. Klicken Sie im Caseware Cloud Hauptmenü () auf den Eintrag Dateien.

  2. Wählen Sie die betreffende Entität aus der Drop-down-Liste der Entitäten aus.

  3. Wählen Sie die Datei, für die Sie die Beibehaltung festlegen möchten, mit Hilfe des Kontrollkästchens aus.

  4. Wählen Sie Weitere Aktionen () | Beibehaltung definieren.

  5. Wählen Sie das Startdatum für die Aufbewahrung, wie lange die Datei aufbewahrt werden soll und alle Kommentare.

  6. Wählen Sie OK zur Bestätigung. Der Bereich Datei-Details wird mit den Informationen zum Beibehaltungsstatus aktualisiert.
