Feedback für andere in Caseware Cloud hinterlassen

Während Ihr Team im Engagement arbeitet, können Sie sich gegenseitig Feedback geben und wichtige Erkenntnisse teilen und besprechen. Beim Einsatz von Caseware Cloud können Sie in einem Engagement ohne die Notwendigkeit von ständigen Meetings zusammenarbeiten.

Eine Aktivität kommentieren

Sie können auf gepostete Diskussionen Ihrer Teammitglieder zur Arbeit reagieren, um Input oder Feedback zu geben.

Eine Diskussion posten

Sie können eine neue Diskussion zu allgemeinen Fragen oder Kommentaren zum Engagement erstellen.

Eine Diskussion erstellen:

  1. Stellen Sie sicher, dass Sie über die Rolle Editor bzw. über gleichwertige Berechtigungen verfügen.

  2. Wählen Sie im Cloud Hauptmenü den Eintrag Aktivitäten.

  3. Wählen Sie Neu und anschließend Diskussion.

  4. Geben Sie im Dialog Diskussion erstellen ein Betreff und den Inhalt der Nachricht ein. Zum Anhängen einer Datei klicken Sie auf das Büroklammersymbol.

  5. Wählen Sie den Drop-down-Pfeil rechts unten und anschließend Posten.

Sie haben eine neue Diskussion für Ihre ausgewählte Entität gepostet. Mitglieder des Engagementteams können diese Diskussionen sehen und Kommentare oder Feedback hinzufügen.

Eine Diskussion zu einer bestimmten Datei posten

Um einen besseren Audit Trail (Prüfpfad) zu gewährleisten, könnte es sinnvoll sein, Diskussionen zu einer Datei direkt an eine Datei zu posten. Benutzer, die keinen Zugriff auf die Entität haben, aber auf die Datei, können die Diskussionen sehen und darauf antworten.

Eine Diskussion über eine Datei hinzufügen:

  1. Stellen Sie sicher, dass Sie über die Rolle Editor bzw. gleichwertige Berechtigungen verfügen.

  2. Wählen Sie die Datei, für die Sie eine Diskussion erstellen möchten.

  3. Klappen Sie im Dialog Details den Bereich Diskussionen auf und klicken Sie auf Hinzufügen( ).

  4. Geben Sie Betreff und Inhalt für Ihre Diskussion ein.

  5. Klicken Sie auf Posten.

Sie haben eine neue Diskussion für Ihre ausgewählte Datei erstellt. Jeder Cloud-Benutzer, der die Datei sehen kann, kann auch Ihre Diskussion sehen und diese auch kommentieren.

Private Benachrichtigungen

Wenn Sie ein Teammitglied schnell kontaktieren müssen, können Sie das integrierte Chat-System verwenden.

Sie können das Chat-Fenster anzeigen oder verbergen, indem Sie das Chat-Symbol ein- oder ausschalten.

Sie können ein Chat-Fenster öffnen, indem Sie eine Person im Chat-Bereich auswählen. Wenn eine Person Ihnen eine Nachricht sendet, wird ein kleines Chat-Fenster unten in Ihrem Browser angezeigt. Sie können auch eine Nachricht an eine Person senden, die offline ist, und die Nachricht wird nach deren Anmeldung zugestellt.

Ihr Unternehmen kann die Chat-Historie in Caseware Cloud beibehalten. Weitere Informationen finden Sie unter Funktionen für alle Benutzer aktivieren oder deaktivieren.
