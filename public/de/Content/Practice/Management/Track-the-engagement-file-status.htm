Status der Engagementdatei nachverfolgen

Jede Engagementdatei verfügt über einen integrierten Workflow. Sie können den Workflow verwenden, um den Status der gesamten Engagementarbeit nachzuverfolgen. Beim Workflow handelt es sich um eine der Eigenschaften, die Sie im Pop-up im obersten Menü festlegen können.

Wählen Sie einen Workflow-Status aus dem Menü, um ihn dem Engagement zuzuweisen.

Bitte beachten Sie: Der Workflow-Status wird automatisch geändert, wenn Sie bestimmte Teile des Engagements fertig stellen. Wenn Sie z. B. die Hauptabschlussübersicht in Ihre Engagementdatei hochladen, wird der Workflow-Status automatisch in In Bearbeitung geändert.

Der Status der Engagementdatei wird auch in Cloud angezeigt.

Sie können detaillierte Informationen zum Engagementstatus auf der Seite Engagements in Cloud sehen. Wählen Sie ein Engagement, um Schlüsselinformationen, wie Schlüsseldaten, ausstehende Aufgaben und den Abzeichnungsstatus der Dokumente im Engagement anzeigen zu lassen.
