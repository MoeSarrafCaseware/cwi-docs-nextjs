Create a new entity

An entity is a workspace that you can use to organize and collaborate on your engagement files from a single location for a client. When your firm first starts to use Cloud, you need to create a new entity for each client.

Note: The instructions below explain the process for adding an individual client entity. You can also import a list of your clients in order to create all of your client entities at once. For more information, refer to Caseware's online help.

Practice set

From the Cloud menu, select Entities. This takes you to the Entities page.

Select New, and then select Clients. The Create Entity dialog displays.
