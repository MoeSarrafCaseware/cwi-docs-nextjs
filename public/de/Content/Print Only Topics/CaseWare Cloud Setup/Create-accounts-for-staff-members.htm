Create accounts for staff members

Once you create entities for each of your clients, grant Cloud access to other staff members in your firm. You can create a new account for each staff member who needs to collaborate on your engagements. After you create accounts for your staff members, you can assign them security roles to determine the level of access they have to your firm. The following table briefly describes the roles you can assign your staff members. For more details, see Built-in roles.

Rolle Anwendungsbereich Beschreibung
Administrator Systemweit

Assign the Admin role to Staff responsible for managing the entire site. Admins can access all content and perform any action. Administratoren muss aber für den Einsatz von Apps bei Bedarf eine Lizenz zugeordnet werden.
Einstellungen-Administrator Systemweit

Assign the Settings Admin role to staff responsible for Firm settings and template authoring.
Mitarbeiteradministrator Systemweit

Assign the Staff Admin role to staff responsible for managing staff. Staff Admins can add, view, edit, and delete all staff in the system.
Entitätsadministrator Systemweit

Assign the Entities Admin role to staff responsible for adding entities and contacts. Entities Admins can add, view, edit, and delete all entities and contacts in the system, and all content within entities.
Eigentümer Systemweit, Entitäten und Inhalte innerhalb dieser Entitäten Owners can view, edit, delete, and grant others roles on content where they have this role.
Editor Systemweit, Entitäten und Inhalte innerhalb dieser Entitäten Editors can view and edit content where they have this role.
Beobachter Systemweit, Entitäten und Inhalte innerhalb dieser Entitäten Viewers can view content where they have this role.
Zugriff auf Entität Systemweit, Entitäten und Inhalte innerhalb dieser Entitäten

Granting the Entity Access role to a user enables the assignment of a security role on content and grants access to the entity summary and details.

Practice set

Note: To add accounts for staff members, you must have either the Admin or Staff Admin role.

  1. Create an Excel file containing your staff member information with email, first-name and last-name columns.

  2. Save the file as Staff List.csv.

  3. From the Cloud menu (), select Staff.

  4. Select New and, under Import, select Staff.

  5. Wählen Sie in den ersten beiden Kontrollkästchen des Dialogs Mitarbeiter importieren, wie der Import bei existierenden Benutzern erfolgen soll.

    Bitte beachten Sie: Wählen Sie das zweite Kontrollkästchen nur, wenn Sie eine komplette Mitarbeiter- oder Kontaktliste importieren.

  6. Geben Sie an, ob Ihr Text oder Ihre CSV-Datei eine Kopfzeile beinhaltet und welcher Typ von Trennzeichen verwendet wird.

  7. Select Import from CSV and browse to your Staff List.csv directory file then select Next

  8. Ordnen Sie jeder Spalte die jeweilige Eigenschaft zu.

  9. Klicken Sie auf Import.

Ein Bestätigungsdialog erscheint, in dem die Anzahl der erfolgreich hinzugefügten oder geänderten Benutzer angezeigt wird.

Nach dem Import einer Benutzerliste sendet Caseware Cloud automatisch eine E-Mail mit einem Link zur Anmeldeseite an diese Benutzer. Eine Aufzeichnung dieser E-Mail finden Sie unter Einstellungen | E-Mail | E-Mail-Protokoll.

Auf dem Anmeldebildschirm werden die Benutzer dazu aufgefordert, ihre E-Mail-Adresse und ihr Kennwort einzugeben und zu bestätigen.
