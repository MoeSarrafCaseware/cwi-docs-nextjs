Complete the engagement acceptance

You have completed all the steps for the initial setup of your Caseware ReviewComp engagement.

You have:

  • Created the engagement

  • Reviewed the engagement properties

You are ready to begin working in the engagement itself. The next step is to complete the pre-engagement acceptance procedures.

The Documents page

When you open your Caseware ReviewComp engagement, you are automatically brought to the Documents page. This page is the starting point for most of the work that you will perform in the engagement.

Your engagement documents are grouped by their phase in the engagement workflow. There are three categories:

  • Planung

  • Feldforschung

  • Schlussfolgerung

When you first open your engagement, only the Planning phase will be visible.

As you complete procedures and sign off documents within your engagement, additional phases and documents will become available to you. Documents in each category are organized into subfolders for better accessibility and presentation.

Review and complete the engagement letter

In the Client queries - Engagement letters, documents and records query, navigate to the Engagement letter - Compilation 4200 question set and open the 1.10 Compilation engagement letter.pdf. The 1.10 Compilation engagement letter document opens, which is a draft version of your engagement letter that automatically generates based on your responses from previous procedures.

Your letter includes guidance information, as seen above. Guidance is displayed by default, but is not included when you print the letter. You can switch off guidance from More actions (...).

Edit the engagement letter

The engagement letter draft uses blue drop-down menus to indicate areas where you have to enter information specific to your engagement. At the top of the letter, the number of placeholders indicates the remaining number of areas you need to complete. You can select the Placeholder text to go to the next unset area.

Practice set

Complete the drop-downs in the engagement letter as required.

When you click on drop-downs that require a contact's name, for example, Person to whom the engagement letter is sent, the option to Search for contact displays. Select the Search box and begin typing.

A list of matches displays. If the desired person is not in the list, you can select Add New Contact and enter the contact's information as desired.

If an expected contact is not listed, edit their profile from the Contacts page in Cloud and ensure they have the Contact - Entity Collaborator role assigned.

To edit the text of the letter, select the Edit icon() at the top of the page.

Text areas will be displayed with gray backgrounds. You can select the text area to change the content. Once you have finished editing the area, select the Edit icon () again to save your changes.

Review the letter and update any sections with placeholder text. These are shown in blue text, for example, [name of organization].

Once you have finished editing the document, you will be ready to configure the print settings.

Configure the print settings for the engagement letter

You have filled in all the placeholders in your letter with real content, entering dates and client names and adding in any mandated content for letters in your firm. Next, you should add a header to your letter that includes your firm name, and you should set the page size for your letter.

Practice set

Select More actions (...) | Print Settings to display the Print Settings dialog.

Select the Header area to expand it and display the text editing menu. From the menu, select the Insert Formula icon (fx). The Data Link dialog displays. Select the Firm Properties tab, and then select the Firm name to add it to your header.

You can set a specific page size by choosing it from the Page Size pull-down. Set the page size to Letter.

You can also:

  • Change the page orientation to Landscape (the default is portrait)
  • Adjust the Margins
  • Include or exclude a document's table of contents from the page numbering

Once you have changed your print settings, select Save.

You can select the Print icon () at the top of the letter to see a sample of your engagement letter with the header included. Use this option to save a copy of the letter as a PDF.

Bulk Print Settings

You can also configure Bulk Print Settings in the Documents page to group two or more documents and print them as a single PDF. You can configure more than one document group and include the same document in multiple groups.

Once you configure a bulk print group, it displays in the print options in the Documents page and in each document within that group. Note that with bulk prints configured, you still have the option to print a document individually.
