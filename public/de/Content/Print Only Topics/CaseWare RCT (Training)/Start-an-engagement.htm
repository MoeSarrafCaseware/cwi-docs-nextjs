Start an engagement

At the beginning of a client engagement, you will need to create a new engagement file. Engagement files are created and hosted on Caseware Cloud. For more information about Caseware Cloud, visit Caseware Cloud.

Before completing the exercises in this section, check with your firm author to confirm that your firm template has been set up. For information, refer to Manage the firm template.

In this section, we will:

  • Create a new engagement file.

  • Set the engagement properties.

Create a new engagement

When you are ready to begin work in an engagement, the first thing you will do is create the new engagement file. In Caseware Cloud, you can create your engagement file in the client entity, set key dates and budgets and assign staff members to the file. Refer to docs.caseware.com for more information.

Practice set

To create a new Caseware ReviewComp engagement:

  1. From the Cloud menu (), select ReviewComp.

  2. Select the New button.

  3. In the ReviewComp dialog, choose your client Entity and enter the Name for the engagement.

  4. In the Preparer field, enter your first name to search for your account. Select your account to assign yourself as the preparer.

  5. In the Reviewer field, enter part of another staff member's name. Select their account to assign them as the reviewer.

  6. At the bottom of the ReviewComp dialog, select Save.

    The Engagements Properties popup displays.

You have created your new engagement and assigned staff in the firm to prepare the engagement and review it.

Set the engagement properties

When you first open the engagement, a Engagement Properties popup displays with information about the engagement.

Review the engagement properties and make any necessary edits.

  • Workflow - you can select a workflow stage to indicate the status of your engagement work.

  • Lockdown - you can lock down the engagement file once all engagement work is finalized.

  • Consolidate - you can add additional engagement files to create a consolidated engagement.

You can also change the Operating name, select a different Year end date, and change the currency.

Verify that your client's year end date is correct, then Save your changes.
