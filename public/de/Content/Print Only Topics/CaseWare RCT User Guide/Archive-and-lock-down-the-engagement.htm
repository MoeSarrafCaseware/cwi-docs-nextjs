Archive and lock down the engagement

Once all work on the engagement is complete, you can download a reference copy of the engagement to send to the client or for your own records. Die Referenzkopie ist eine Zip-Datei, die PDF-Kopien von allen Dokumenten in Ihrem Engagement beinhaltet.

To download a reference copy, select Information () | Download reference copy.

You can also can lockdown the file. Select the engagement file's name to open the Engagement Properties popup, then select Lockdown.
