Validation

In this stage, you will check your data for reliability and completeness.

Data reliability checklist

Use this document to assess the completeness and reliability of the data file.

Practice set

To complete the Data Reliability Checklist:

  1. Select 1-1 Data Reliability Checklist from the Documents page.
  2. Answer No for procedures 1 - 9 in the checklist.
  3. Select the appropriate response in the Conclusion section of the document.
    In this case, select Sufficiently reliable.

Vollständigkeit der Daten

You can use a data completeness check to see whether the account balances calculated from the trial balance match the account balances calculated from the transactions in an engagement. Discrepancies may indicate that you have received or uploaded an incomplete data set and may need to re-import the data.

Practice set

To review the Data Completeness:

  1. Select 1-2 Data Completeness from the Documents page.
    Notice the amount in the Difference column for Shareholders equity.
  2. In the Comments column, add the comment Opening equity was not included in the import.
  3. Return to the Documents page.
