Edit and send the PBC Request General Information Form

When you complete the 100 Client Optimization checklist, based on your answers, the Documents page includes one of the following query documents:

  • 100-10 PBC Request - Interim Fieldwork

  • 101 PBC Request - General Information

  • 102 PBC Request - Tax

  • 103 PBC Request - EBP

Eine Abfrage ist ein interaktives intelligentes Dokument, das aus den folgenden Komponenten besteht:

  • Query questions - The requests and inquiries that clients need to respond to.

  • Fragensets - Zur Gruppierung zusammengehöriger Fragen.

  • Response types - To specify the expected user response

Based on our answers to the checklist in the previous chapter, the 101 PBC Request - General Information query document is available in the Documents page. In this chapter, we'll look at how to edit and send a query.

101 PBC Request Form - General Information

Before sending the query, you can make modifications like adding instructions, question sets and questions. Today, we're going to go over adding question sets and questions, but you can learn more about customizing queries at docs.caseware.com.

Question sets

Question sets are used to group related questions. All questions must belong to a question set.

To add a question set, select Add () from the left navigation panel.

A new question set is added in the document with a default title New Group.

Sie können den Standardtitel auswählen, um ihn zu ändern. You can remove a question set by selecting it the Delete () icon.

Note: If you delete a question set, all questions in the set are deleted as well.

Eine Abfrage hinzufügen

After you have created a question set, you can add questions to it.

To add a query question:

  1. Go to the question set where you want to add a question.

  2. Select Add () and add one or more of the following response types:

    • Text - An input text field where users can enter a text response.

    • Picklist - A drop-down list where users can select one response from a list of responses. You can customize the list of responses to meet your needs. The default responses are Yes, No, and Not applicable.

    • Multi-Picklist - A drop-down box where users can select multiple responses from a list of responses. The default responses include Yes, No, and Not applicable.

    • File Upload - A file upload box where users can browse and add a file or drag and drop the file.

    • Date - A date picker field with an inline calendar icon where users can select a date.

    A new query question is added in the document with a default body text New Query Question.

  3. Select the default body text to change it. Once you select the text field, the formatting toolbar and the Cancel/Ok buttons () display.

    As you type, you can select Cancel () to clear the text field and start over. The text you enter is automatically saved but you can also use the Ok button () to save.

You can also select the Summary field above the text to enter a short description for the query question. Once you enter a summary for the question, it shows in the left navigation panel by default as the question locator. If the Summary field is empty, the question body text shows in the left panel instead.

Note: We recommend that you always enter a summary for questions to easily locate questions in the left navigation panel.

  1. If you've added a File Upload response type, you can set up a few more options:

    1. Select the () button at the bottom of the upload box.

    2. In the File response settings dialog, select a target folder or placeholder location, an identifier and a name for the uploaded file.

  2. Once you're done, select Save.

Send query

When you're finished modifying the query, you can send it to your client.

To send the 101 PBC Request Form - General Information to a client:

  1. Öffnen Sie das Abfragedokument und wählen Sie dann SENDEN.

  2. Enter the following information in the Open Query dialog:

    1. Enter today's date in the Query Due Date field.

    2. Select the client contact you created from the Assignees drop down list.

    3. Enter 'Please upload the requested items" in the Email Body text box.

  3. Klicken Sie auf OK, um die Abfrage zu senden.

A notification email is sent to your client contact.
