Getting started

Before you can start work on any OnPoint Collaborate engagements, you need to set up the Caseware Cloud portal for your organization. For information, refer to the Get Started with Caseware Cloud.

Once you have set up your portal, you can continue with the exercises in this guide.

This chapter walks you through the process for completing these setup steps. Once the following steps are complete, you will be ready to create an engagement:

  • Set up workspaces in Caseware Cloud for each of your clients
  • Create accounts for your staff members
  • Create accounts for your clients

Create a new entity

An entity is a workspace that you can use to organize and collaborate on your engagement from a single location for a client. You will need to create a new entity for each client.

Note: The instructions below explain the process for adding an individual client entity. You can also import a list of your clients in order to create all of your client entities at once. For more information, refer to Create client workspace.

Practice set

Klicken Sie im Cloud Hauptmenü () auf Entitäten. This takes you to the Entities page.

Select New, and then select Clients. The Create Entity dialog is displayed.

For each of the sections below, enter the information shown.

Once you have finished entering all of the relevant information for your client, select Save. The client entity is created, and you can see it added to the Entities list.

Create accounts for your staff members

Once you have created entities for each of your clients, you will be ready to grant access to your Cloud firm to other staff members in your organization. You can create a new account for each staff member who will need to collaborate on your engagements.

Practice set

Note: The instructions below explain the process for adding individual staff members. You can also import a list of your staff members in order to create all of their user profiles at once. For more information, refer to Caseware's online help.

From the Cloud menu (), select Staff. This takes you to the Staff page.

Select New, and then under Add, select Staff. The Add Staff dialog is displayed. Just as you saw when creating an entity, there are different areas where you will enter information about the new account.

Allgemein

Enter an email address and a first and last name for the new staff member. The email address will be the username for their account, and they will log into Cloud with it.

The First Name, Last Name, and Email (or Username) fields are mandatory.

Password

You can choose how the password for the new account will be set:

  • Set by User - the staff member will set their own password when they first log in. OnPoint Collaborate sends an email with instructions on how to log in to the portal.
  • Auto Generated - a random password is chosen by OnPoint Collaborate. It does not send an email to the staff member, so you will need to provide them with the login page URL, their user ID and their password.
  • Set - you choose a password for the staff member. OnPoint Collaborate does not send an email, so you will need to provide them with their login information.

Leave the password option as Set by User.

Gruppen

You can assign the staff member's account to one or more staff groups. If you create groups, you can specify permissions and access levels for members of that group, and then any accounts that you add as group members automatically inherit those settings.

OnPoint Collaborate only includes the All Staff group by default, and any staff members are automatically included in the group, so you won't have to do anything in this section.

Leave this page as-is.

Apps

You can grant access to apps to your staff member. You can grant the staff member the ability to view engagements for an app and the ability to create new engagements or files for it.

Note that by default, the All Staff group includes the ability to view and create engagements for all of the apps. This simplifies access, but if you need to control access to some apps more tightly, you can remove some of those permissions from the global group. This would enable you to pick and choose which staff members should be able to work with certain apps.

Standard Hours

The settings for Staff Standard Hours are only applicable to your organization if you use the CasewareCloud Time app to handle time and expense entries. Leave this section set to Organization standard hours.

Once you have finished entering all of the information for your staff member's new account, select Save. They will receive a welcome email notifying them that their account has been created and instructing them on how to log in.

Create client contacts

One of the major benefits of OnPoint Collaborate is that you can collaborate directly with your clients on engagements. For your clients to access their engagement, you need to set up a contact account for them on Cloud.

Practice set

From the Cloud menu (), select Contacts. This takes you to the Contacts page.

Select New, and then under Add, select Contact. The Add Contact dialog is displayed.

Allgemein

Enter a first and last name and an email address for your client contact.

Note: To avoid potential issues with Google, be sure to use a valid email address.

In the Entity field, select the client entity that you created earlier. Once you have selected the client entity, select the Contact - Entity Collaborator role to set the access level for this account. The entity collaborator role lets your client create discussions, tasks, and file requests, and lets them collaborate on engagements.

Password

The same password options that you had when creating your staff accounts are also available here. Leave the password option as Set by User.

Gruppen

Contact groups work the same way that staff groups do. By default, the only contact group is All Contacts — your client will automatically be part of that group.

Leave this page as-is.

Apps

The All Contacts group automatically grants client contacts the ability to view engagements or files for all app types in Cloud. Review the permissions and make sure that your contact can view OnPoint Collaborate engagements.

Once you have finished entering all of the information for your client contact's account, select Save. They will receive a welcome email notifying them that their account has been created and instructing them on how to log in.
