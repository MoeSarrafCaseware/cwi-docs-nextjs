Abfragen

Auf der Seite Abfrage-Dashboard können Sie Ihre Fortschritte und die Fortschritte Ihrer Kontakte bei den Abfragen verfolgen. Das Dashboard enthält eine Liste der sichtbaren Abfragen in der Engagementdatei. Sie besteht aus den folgenden Spalten:

Um auf das Abfrage-Dashboard zuzugreifen, gehen Sie auf die Seite Abfragen.

Auf der Seite wird eine Liste der verfügbaren Abfragen in der Engagementdatei angezeigt. Wählen Sie eine Spalte, um die Liste nach Status, Fortschritt, Fälligkeitsdatum oder alphabetisch zu sortieren.

Hinweis: Nur sichtbare Abfragen werden in der Liste angezeigt. Wenn eine Abfrage aufgrund von Sichtbarkeitseinstellungen ausgeblendet ist, wird sie im Abfrage-Dashboard nicht angezeigt.
