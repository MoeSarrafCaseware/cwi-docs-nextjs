Engagement acceptance

You have completed all the steps for the initial setup of your OnPoint PCR engagement. You have:

  • created the engagement file

  • reviewed the engagement properties

Now you are ready to begin working in the engagement file itself. The next step is to complete the pre-engagement acceptance procedures.

The Documents page

When you open your OnPoint PCR engagement, you are automatically brought to the Documents page. This page is the starting point for almost all of the work that you will do on the engagement.

Your engagement documents are grouped by their place in the engagement workflow. There are four large categories:

  • Engagement Acceptance and Planning

  • Feldforschung

  • Financial Statements and Wrap-up

  • Permanent File

Note: Some of these categories appear as you progress through your engagement.

Three of the sections have a progress indicator that measures how close that part of the engagement is to being completed. The progress indicator displays a percentage that indicates how many of the documents in the section have been signed off on by a reviewer.

In the Permanent File section, you can add supporting documents for the engagement. These documents will be carried forward when you create the next period's engagement for this client.

When you start a new engagement, there are only a few documents visible on the page. As you complete procedures and sign off documents, other documents you will subsequently need are added to the documents page.

Complete the Engagement Acceptance and Continuance document

Open the 1-110 Engagement Acceptance and Continuance document. In this document, you will provide initial information about the client and the engagement type.

The Engagement Acceptance and Continuance document is a checklist. Each checklist document contains a set of questions that the engagement preparer can answer. Most of the engagement work in OnPoint PCR engagements is completed in checklists. As you complete questions in the Engagement Acceptance and Continuance document, other documents will be added to your engagement file.

Complete the Quality Acceptance and Conclusion document

In the Quality Acceptance and Conclusion document, you respond to a series of questions to assist in compliance with the provisions of SSARS. Select the Guidance heading at the top of the document to see more details.

As you complete questions in this document, new questions are displayed. For each question, depending on your response, a warning message may be displayed, or you may be required to document an issue and the resolution.

Select the document name to open the draft engagement letter.

Complete the engagement letter

The first two checklist documents that you have completed outline the basic engagement information. The draft version of your engagement letter has automatically been generated based on your responses in those checklists.

At the top of the letter, you will see a guidance section. This contains information about the contents of the letter below, and includes instructions for finalizing the draft. Guidance is displayed by default, but it is not included when you print the letter. You can hide the guidance section by selecting More actions () |Guidance Off.

The engagement letter draft uses red text to indicate an area where you have to enter information specific to your engagement. Review the letter and update any sections with placeholder text.

For example, the first text area contains firm information. The firm name and address are populated automatically based on values from your firm profile on Cloud, but there is red text here indicating that you should insert firm letterhead.

To edit a text area in the letter, select the Edit icon () at the top-right of the document.

The text area will be displayed with a gray background, and you can select the text area to change the content. Select [Insert Firm Letterhead] and then from the text editing menu, select the Insert Image icon ().

In the Image dialog, select Choose File. Find the image file for your firm letterhead, and then select OK.

The image is added to your text area. Once you have finished editing the area, select the Edit icon () again to save your changes.

You will also find pull-down menus embedded in parts of the letter. Like the red text, these menus are placeholders for information that you need to add to the letter.

For example, in the second text area, you will see a pull-down menu that says Today's Date.

When you select this menu, a calendar is displayed. Select a date from the calendar menu, and the date area in the letter displays the chosen date. Note that although the pull-down menu's background is no longer visible, you can still select the date to display the calendar menu and change your date.

Review the rest of the letter draft, select values for each of the pull-down menus, and replace the red placeholder text with content specific for your engagement. Once you have finished editing the document and have inserted another image with your firm's signature, you will be ready to configure the print settings.

Configure the print settings for the engagement letter

You have filled in all the placeholders in your letter with real content, entering dates and client names and adding in any mandated content for letters in your firm. Next, add a header to your letter that includes your firm name, and set the page size for your letter.

Select More actions () | Print Settings to display the Print Settings dialog.

Select the Header text area to expand it and display the text editing menu. From the menu, select the Insert Formula icon (). The Data Link dialog is displayed. Select the Firm Properties tab, and then select the Firm name to add it to your header.

You can set a specific page size by choosing it from the Page Size pull-down menu. Set the page size to Letter.

You can also:

  • change the page orientation to landscape (the default is Portrait)
  • adjust the margin width
  • have a document's table of contents count in the page numbering

Once you have changed your print settings, select Save. You can select the Print icon () at the top of the letter to see a sample of your engagement letter with the header included.

Send the engagement letter to the client

Now that the draft of your engagement letter is ready, you need the client's signature in order to proceed with the engagement. You can send your draft directly to your client in OnPoint PCR by using a procedure.

A procedure is a document that you can use to communicate directly with the client. You can use procedures for a number of things:

  • Request for information - send questions to the client.

  • Document request - ask the client to upload one or more files.

  • Send deliverables - send documents to the client to sign and return.

If your client has a contact account in Cloud, they can sign in and respond directly on procedures.

Open the Documents page. Notice that there are more documents displayed here based on the questions that we responded to in the acceptance documents. In the Engagement Acceptance and Planning section, you will see a query document named 1-200 Engagement Letter (procedure to Client).

Open this document. It asks the client to review and sign the attached draft of the engagement letter.

Select Send to submit the query to the client. The Open Query dialog is displayed.

Select a due date from the calendar menu. Next, assign the query to one of your client contacts. Select the contact search field that is displayed below it and then choose one of the contacts from the quick list. Finally, enter the following custom email message:`

Please review the attached engagement letter, sign it, and return it to us. If you notice any problems in the letter, please inform us at once.

Select OK to open the procedure. This also automatically updates the status of the engagement file to Awaiting response.

The client will receive an email notifying them that there is an item in Cloud that requires their attention. When they log in, they will see your query under My Work.

The client can download a copy of the PDF letter, review it, add their signature, and upload the signed document back to the same procedure. They can also add comments to the procedure, and these comments will be visible to your firm members in the engagement file.

Once they have responded to all of the tasks on the procedure, they can mark the procedure as Responded. This updates the document status in the engagement file, and it also automatically updates the status of the engagement file to procedures Ready. Your client contacts can no longer see a procedure that has been marked as Responded on their Overview page, but they can still see the procedure on the Activities page.

Accept the signed engagement letter

When you review the procedure, you can see that the client has responded to the procedure, and you can either accept or decline the response. To accept the procedure, change the procedure status at the top of the document to Complete.

When you accept the procedure, the signed letter is attached to the file, replacing the placeholder that was on the Documents page. The procedure will be locked to prevent any further editing.
