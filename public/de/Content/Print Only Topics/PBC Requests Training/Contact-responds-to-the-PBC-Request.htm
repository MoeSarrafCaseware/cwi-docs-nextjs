Contact responds to the PBC Request

Sobald Sie das Dokument freigegeben haben, können Ihre Kontakte die Abfrage im Bereich Meine Arbeit sehen, wenn sie sich bei Cloud anmelden. Sie können das Dokument auswählen, um es zu öffnen und Antworten zu geben. In this chapter, we're going to go through how a client responds to a query.

Open the link you've received in a separate browser, such as Firefox or Microsoft Edge, and sign in to Cloud as your contact persona. The query you sent is displayed in the My Work pane.

Select the query to open it.

You can now answer the query questions and drag-drop documents into the appropriate response areas to respond to the query.

Mark the questions you have responded to by selecting the Respond icon ().

Once you mark a question as responded, the icon changes to the blue circle ().
