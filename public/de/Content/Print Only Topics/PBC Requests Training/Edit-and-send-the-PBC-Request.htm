Edit and send the PBC Request

In this chapter we're going to take a look at the Review records and documents PBC Request, and we're also going to learn how to create and send a PBC Request, or Query Document, to contacts.

Eine Abfrage ist ein interaktives intelligentes Dokument, das aus den folgenden Komponenten besteht:

  • Query questions - The requests and inquiries that clients need to respond to.

  • Fragensets - Zur Gruppierung zusammengehöriger Fragen.

  • Response types - To specify the expected user response

Based on our answers to the checklist in the previous chapter, the Review records and documents query document is available in the Documents page.

Review records and documents

Take a look at the questions included in this query document. The questions that are scoped into the query are based on the answers you provided in the Optimiser.

Before sending the query, you can make modifications like adding instructions, question sets and questions. Today, instead of modifying a query, we're going to go over adding your own custom query. You can learn more about customizing queries at docs.caseware.com.

Create a new Query Document

Although you can use the default queries that are scoped in to the engagement based on your answers in the Optimiser checklist, you also have the option to create your own query.

First, select Documents in the top menu to return to the Documents page.

In the Review phase, next to the Plan the Engagement folder, select Add (), then choose Query.

The Create Query page displays the query templates that are available to you. We're going to use a blank query.

Under Blank Query, enter "My Query" in the Name field and select Create Query.

The query opens.

Question sets

Question sets are used to group related questions. All questions must belong to a question set.

To add a question set, select Add () from the left navigation panel.

A new question set is added in the document with a default title New Group.

Rename the question set to "Training PBC Request Examples".

Add query questions

After you have created a question set, you can add query questions to it. We're going to add two questions to our questions set.

To add query questions:

  1. Move your cursor down until you see the blue Add () icon and line.

  2. Select Add () and choose the Picklist response type. Picklists are drop-down lists where users can select one response from a list of responses.

    A new query question is added in the document with a default body text New Query Question.

  3. Enter "Property Plant and Equipment" in the Summary field.

    Once you enter a summary for the question, it shows in the left navigation panel by default as the question locator. If the Summary field is empty, the question body text shows in the left panel instead.

    Note: We recommend that you always enter a summary for questions to easily locate questions in the left navigation panel.

  4. Select the default body text to change it. Enter "Were there any purchases of Property Plant and Equipment during the fiscal year?"

    Once you select the text field, the formatting toolbar and the Cancel/Ok buttons () display.

    As you type, you can select Cancel () to clear the text field and start over. The text you enter is automatically saved but you can also use the Ok button () to save.

  5. Enter "Yes" as Option 1.

  6. Select Add () to add "No" as a second option.

  7. Select More actions (), then select Settings.

  8. Select +Response to add an additional response.

  9. In the Response Type drop-down menu, select Upload Files.

  10. Select Edit Visibility.

  11. In the Visibility Settings dialog, change Hide to Show and select Response condition from the Add condition drop-down menu.

  12. Modify the response condition as follows:

  13. Select Copy. We're going to use this in a moment.

  14. Select Save.

  15. Select Save again in the Settings dialog.

  16. Now we're going to add a second query question. Move your cursor down again until you see the blue Add () icon and line below the query question.

  17. Select Add () and choose the Multi Picklist response type.

  18. Enter the summary, query description and picklist options as follows:

  19. Select the Include "Other" in the options check box.

  20. Select More actions (), then select Settings.

  21. Select +Response to add an additional response.

  22. In the Response Type drop-down menu, select Date.

  23. Select Edit Visibility.

  24. Select Paste.

    The visibility settings from the previous question are pasted.

  25. Change the procedure from "Request information about Property Plant and Equipment additions?" to "Request information about related party transactions?"

  26. Select Yes from the answered drop-down menu.

  27. Select Save.

  28. Select Save again in the Settings dialog.

Send query

We have now finished customizing the query and you can send it to your client.

To send My Query to a client:

  1. Select SEND.

  2. Enter the following information in the Open Query dialog:

    1. Enter today's date in the Query Due Date field.

    2. Select the client contact you created from the Assignees drop down list.

    3. Enter 'Please upload the requested items." in the Email Body text box.

  3. Klicken Sie auf OK, um die Abfrage zu senden.

A notification email is sent to your client contact.
