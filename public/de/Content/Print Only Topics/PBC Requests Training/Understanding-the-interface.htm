Understanding the interface

Before we start working on the engagement file, let's go over the engagement interface.

Engagement Properties

After you've created a new engagement, an Engagement Property dialog automatically displays in the engagement file. Here you can enter some high-level information about the engagement.

In this dialog, you can set the following properties:

  • Workflow-Stufe - wählen Sie eine Stufe für den Status der gesamten Engagementarbeit.

  • Operating name - this field shows the Cloud entity name by default. Sie können die Standardeinstellung ändern und einen anderen Namen eingeben.

  • Year end date - choose a year end date for this engagement.

The Engagement Properties dialog also features the following buttons:

  • Second Reviewer - select to add a second reviewer for all signoffs. This adds Final Review to all signoff options.

  • Lockdown - select to lock the engagement file, once all the work is complete.

For this exercise, set the following properties for your engagement file:

  • Workflow stage - In Progress

  • Operating Name - ACME Inc.

  • Year end date - December 31, 2019

You do not need to add a second reviewer. Once you have finished updating your engagement properties, select Save.

Aufgaben

Issues can be added to an engagement file to signal to the engagement team that additional work is required on a document or that specific tasks must be completed for an engagement.

Engagement level

If you want an issue to apply globally to the entire engagement, you can add the issue on the engagement level.

To add an issue to an engagement:

  1. Wählen Sie aus dem Menü oben die Option Aufgabe hinzufügen icon ().

    Ein Pop-up wird angezeigt, das ein Textfeld mit einer Formatierungsleiste beinhaltet.


  2. Geben Sie in das Textfeld eine Beschreibung Ihres Problems ein.

  3. Wählen Sie einen Aufgabentyp aus der Drop-down-Liste am unteren Rand des Editors.

  4. Auswahl von Aufgabe erstellen.

Document level

You can also add issues to specific documents within the engagement.

To add an issue to a document:

  1. Select the Add issue icon () for the document.

  2. Geben Sie in das Textfeld eine Beschreibung Ihres Problems ein.

  3. Wählen Sie einen Aufgabentyp aus der Drop-down-Liste am unteren Rand des Editors.

  4. Auswahl von Aufgabe erstellen.

Item level

You can also add issues to items within documents, For example, you can add an issue to an individual procedure in a checklist.

To add an issue to a procedure:

  1. Select the Add issue icon () for the procedure.

  2. Geben Sie in das Textfeld eine Beschreibung Ihres Problems ein.

  3. Wählen Sie einen Aufgabentyp aus der Drop-down-Liste am unteren Rand des Editors.

  4. Auswahl von Aufgabe erstellen.

Issues tab

You can view and manage all issues in an engagement from the Issues page. Select Issues to navigate to the page.

Information

You can select the Information icon () from the top menu to display the information menu. From this menu you can use the Download reference copy option, access the product help, and view information about your current version of the product.

Download Reference Copy

When you select Download reference copy, a zip folder downloads to your local drive. The default name of the folder is <Operating name><Year>. The folder contains PDFs of the following:

  • Engagement-Informationen

  • A list of documents

  • All engagement documents

  • A list of issues

  • A list of adjustments

  • Hauptabschlussübersicht

Help

When you select Help, a new browser window opens with the links to the PBC Requests online user help topics.

About

When you select About, a pop up displays the current PBC Requests version number, other software information and a link to the Latest Release Notes.

Documents tab

When you select the Documents tab from the top menu, the Documents page displays. This is where you can manage documents, check the progress of the engagement work, and start to perform various engagement tasks such as preparing a letter to your client or completing a checklist.

Navigation Panel

To have quick access to the engagement documents, you can use the navigation panel on the left of the Documents page.

To expand the navigation panel, you can select the Documents icon () on the top left, or the expand icon () on the bottom left. To minimize the panel, select the contract icon ().

Suchen

The search bar is located at the top of the navigation panel. You can search for a document by name. You can also search for a keyword in any documents.

Document list

On the navigation panel, you can see a list of all currently visible folders and documents. Once you have opened a document, its name will display under the Recent category.

Pinned documents

If you want a recently opened document to always appear on the top of the list, hover over the document under the Recent category, and select the pin icon (). The document will appear under the Pinned category at the top of the document list.

Documents page

The Documents page is the starting point for most of the engagement work.

Phases

In PBC Requests, documents populate within seven phases in the Documents page:

  • Start Here

  • Compilation

  • Personal Tax

  • Audit

  • Custom

  • Corporate Tax

  • Review

When you start a new engagement, there are only a few phases and documents available in the page. As you complete procedures and sign off documents, other documents populate in the Documents page.

Add content

You can add documents or folders to your engagement. Select the Add document icon () at the side of a phase or a folder, then select one of the options from the drop-down:

  • Select Folder to add a new folder

  • Select a document type to create a new document

  • Select Upload to add a document from your local drive

  • Select Copy from Caseware Cloud to add a file from the Cloud

Progress indicator

Each engagement phase has a progress indicator () that shows the percentage of signed off documents in that section.

Ordner

Within each engagement phase, the documents are organized in folders with blue labels. You can select the arrow icons to open () or close () the folder.

When you hover over a folder, the Add document icon () and the More actions icon () display to allow you to add a document to the folder or to edit the properties of the folder.

Documents

Each folder within an engagement phase contains documents. You can add the following types of documents:

  • Letter

  • Notiz

  • Worksheet

  • Checkliste

  • Query

  • Jahresabschluss

  • Placeholder

  • Link

Edit view

In the Documents page, you can select the Toggle edit view icon () to switch the edit view on or off. When the edit view is on, the Issues icon (), the More actions icon () and the Visibility icon () become available for each document. From here you can create issues, move documents, or change the visibility of documents. To learn more, see Manage the engagement documents.

Other options

Documents are displayed contextually: as you enter information in the planning documents, other documents relevant to your engagement will be visible in the Documents page. You can select the More actions icon (), then select the Hidden Items check box to show all hidden documents and folders. Every hidden item has a Currently hidden icon ().

The Documents page is shown as the Detailed View by default; it displays an icon for each document to indicate the document type; it also indicates the sign off status of each document. To have a simplified view, select the More actions icon () and clear the Detailed View check box.
