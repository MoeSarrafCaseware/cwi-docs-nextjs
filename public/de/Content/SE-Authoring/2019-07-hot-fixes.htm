2019-07 hot fixes

This is a summary of the hot fixes that have been deployed for SE 2019-07.

Note:The following listing only includes general fixes that have been deployed to all regions. It does not include hot fixes deployed to specific regions.

Hotfix 1

Here is the list of issues that have been fixed with the deployment of Hotfix 1.

  • [SE-13928] - Print preview of financial statements document fails to load.

  • [SE-13907] - Query issues in firm author template mode.

  • [SE-13874] - Carry forward issues post migration to SE 2019-07.

  • [SE-13864] - Text formulas (fx) not working as expected in dynamic tables.

  • [SE-13835] - Issues with the new query design:

    • Firm authors can modify some of the default product content.

    • The suppress content option is not available for firm authors.

  • [SE-13808] - Some of the subgroups for a specific financial group are not visible in the Data page.

  • [SE-13516] - API fixes.

  • [SE-13511] - Error in client logs.

  • [SE-11968] - API fixes.

Hotfix 2

Here is the list of issues that have been fixed with the deployment of Hotfix 2.

  • [SE-14330] - External documents imported from BGL 360 are blank.

  • Carry forward issues:

    • [SE-14249] - Prior period worksheet and memo are blank in the next period file.

    • [SE-14215] - Dynamic tables are missing in financial statements documents that users add in the next period file.

    • [SE-14112] - Prior year data displays for subsidiaries in the next period file instead of the current year data.

    • [SE-14109] - Fail to carry forward files where the new year dates are not adjacent to the previous year dates.

  • [SE-14185] - Console error in checklist documents.

  • [SE-14134]- "Invalid Engagement" error when creating a new engagement in AnalyticsAI.

  • [SE-14118] - Account assignment in Trial Balance page is not working properly.

Hotfix 3

Here is the list of issues that have been fixed with the deployment of Hotfix 3.

  • [SE-14526] - Dynamic table rows that are sourced from trial balance are deleted from the template once product authors import trial balance groupings.

  • [SE-14435] - Annotations in are not displayed once users print documents. The created PDF for print does not show the existing annotations in the document.

  • [SE-14305] - Attempting to snapshot a data URL to generate PDF for BGL360 documents fails.

  • [SE-14420] - Errors when opening engagement files.

  • [SE-14179] [SE-13292] - Carry forward failure in some regions.

Hotfix 5

Here is the list of issues that have been fixed with the deployment of Hotfix 5.

  • [SE-14554] - Unable to develop on the platform since SPLAT does not deploy.

  • [SE-14606] - Spontaneous and very frequent page refresh.

  • [SE-14714] - Signoffs missing from annotations in PDFs.

  • [SE-14629] - Accounts are not mapped to Tax Codes.

  • [SE-14669] - Adjustments do not display in Analytics charts.

  • [SE-14552] - Analysis chart missing after importing prior year balances.

  • [SE-14606] - Data page is available in all templates across all sites although it is disabled in the product settings.

Hotfix 6

Here is the list of issues that have been fixed with the deployment of Hotfix 6.

  • [SE-14959] - Some trial balance groups erroneously show a total balance of zero in dynamic tables.

  • [SE-14947] - API fixes.

Hotfix 7

Here is the list of issues that have been fixed with the deployment of Hotfix 7.

  • [SE-15113] - Cannot print financial statements.

  • [SE-15126] - Zero balances in dynamic tables display as zeros in the PDF output of the document when they should display as dashes (per template settings).
