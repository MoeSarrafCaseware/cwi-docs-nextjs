Accounting Estimates - Context view

The Accounting Estimates - Context view is a new approach to documenting the audit work required for accounting estimates. By combining several documents, it enables you to view all content “in context”. The revised workflow of this “in context” approach makes it easier for auditors to see the relevant information for each significant accounting estimate.

The following information outlines the key improvements included in the Accounting Estimates - Context view, compared to the historical approach (documenting work in various checklists):

  • The retrospective review of all significant accounting estimates is automatically populated from each of the detailed accounting estimate documents, ensuring consistency in the engagement file.

  • The overview of each estimate includes a summary of key information and audit work completed.

  • The primary work for a significant accounting estimate is completed in one document, from understanding the accounting estimate and related risks to responding to those identified risks, making it easier to understand the issues related to each accounting estimate.

  • Each significant accounting estimate has its own document and additional documents can be added as required for the engagement within each financial statement area.

The Accounting Estimates - Context view includes two SE Builder forms:

  • Accounting Estimates - A repeater type document that can be duplicated as many times as required.

  • Overview of the prior period accounting estimates - A summary of the information for all accounting estimate forms.

Content considerations

We developed the content of the Accounting Estimates - Context view to address the requirements for ISA 540 (Revised) Auditing Accounting Estimates and Related Disclosures, effective for audits of financial statements for periods ending on or after December 15, 2019. As the default content used in this form is based on the ISA 540 (Revised) Auditing Standard, the content contains copyright materials available through royalty agreements.

The content can be shared with any solutions that are derived from the Audit International Cloud solution. For any other solutions, we can provide a version of the form where copyright content has been removed.

Implement the Accounting Estimates - Context view

To implement the Accounting Estimates - Context view into your solution, you must submit a request to Caseware:

  1. In JIRA, click Create.

  2. In the Project drop-down, select Distributors (DIST).

  3. In the Issue Type drop-down, select Request.

  4. Complete the Summary and Description fields. Mention whether or not your solution is derived from Audit International Cloud.

  5. In the Component-Area drop-down, select Global Components, then in the secondary drop-down, select Other - Distributor Specific.

  6. Complete any other mandatory fields, then click Create.

We will provide implementation instructions based on whether or not your solution is derived from Audit International Cloud. After implementing the Accounting Estimates - Context view into your solution, you can add it to the Documents page by selecting Add (), then choosing Accounting Estimates - Context view.
