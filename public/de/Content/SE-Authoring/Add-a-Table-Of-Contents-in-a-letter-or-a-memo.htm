Add a Table Of Contents in a letter or a memo

To add a Table of Contents, select +Add section | Table of Contents. A Table of Contents area is added in the document with a default title Table of Contents. Sie können den Standardtitel auswählen, um ihn zu ändern. Beachten Sie, dass dieser Titel nicht zum Inhalt des Dokuments gehört.

To delete the Table of Contents, select Settings () | Delete.
