Einen Seitenumbruch in der dynamischen Tabelle hinzufügen

You can add a page break inside a table in the document to better present the table's information. For example, you can insert a page break inside a table to have similar or related information show on the same page rather than being spread over two pages.

To add a page break in a table:

  1. Wählen Sie die gewünschte Tabelle aus.

    2. Symbole für Tabellenoptionen werden rechts angezeigt.

  2. Select the More actions ()

  3. Choose Edit Dynamic Table from the list of actions.

    4. Die Tabellenoptionen im Bearbeitungsmodus.

  4. Locate where you want to add a page break.

  5. Select the Add() | Page Break at the left border of the table.

  6. Wählen Sie SPEICHERN oder SPEICHERN& SCHLIEßEN.
