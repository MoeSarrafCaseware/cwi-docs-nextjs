Add a query template

You can use Query documents to collect information from clients. You can create a standard for client requests by creating Query templates. Users can add a Query document in the engagement file using the query template.

Best Practice: We recommend that you create query documents from the template for users rather than having them do it themselves. This way, you can include the query document in the workflow of the engagement file template and apply visibility logic for it.

A query template (or document) consists of the following components:

  • Fragen - Die Anforderungen und Anfragen, auf die Kunden antworten müssen.

  • Question sets - To group related query questions.

  • Antworttypen - Zur Angabe der erwarteten Benutzerantwort. The following table describes the expected user for each response type.

    Antworttyp

    Erwartete Benutzerreaktion

    Text

    Geben Sie die gewünschten Informationen ein.

    Auswahlliste

    Select one item from the drop-down.

    Multi-Picklist

    Select one or more items from the drop-down.

    Upload Files

    Durchsuchen Sie die gewünschte Datei und fügen Sie sie hinzu (oder ziehen Sie die Datei per Drag & Drop).

    Datum

    Wählen Sie ein Datum aus dem Kalender aus.
    Einfache elektronische Signatur

    Wählen Sie, ob Sie das Dokument manuell unterschreiben oder elektronisch akzeptieren möchten.

    Bitte beachten Sie:

  • Instructions - To provide guidance information for the clients. Fügen Sie zum Beispiel die unterstützten Dateitypen für eine Dokumentenanfrage hinzu.

    Hinweis: If you leave this section empty, it does not show in the Query document when the assigned contacts access it.

  • Left panel - To easily navigate and arrange the items in your document. It shows an outline of available questions sets and questions. As you scroll through the document, the left pane automatically scrolls through the outlined items as well, highlighting the item you're currently reviewing or editing.

To add a query template, select Add () at the documents page and choose Query Template. A blank document opens with the Instructions section at the top.

To add a query document:

  1. Select Add () at the documents page, then choose Query.

  2. Select an existing query template to create your query document from.

  3. Add a name for the query document.

    4. You can optionally add a Number for the document.

  4. Select Create Query.

    5. A blank document opens with the client instructions section at the top.
