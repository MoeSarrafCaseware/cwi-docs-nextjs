Add a response to a question

You can add more than one response types for a query question. For Picklist or Multi Picklist response types, you can also modify the response options in the lists.

To add a response to a query question:

  1. Go to the question, then select More actions () | Settings.

  2. Select +Response to add a different response type.

  3. Enter the placeholder text.

    4. This is the default text that appears in the response field before the contact enters or selects a response.

  4. Select a response type.

    5. The following types are available:

    • Text - An input text field where contacts can enter a text response.

    • Picklist - A drop-down list where contacts can select one response from a list of responses. You can customize the list of responses to meet your needs. The default responses are Yes, No, and Not applicable.

    • Multi-Picklist - A drop-down box where contacts can select multiple responses from a list of responses. The default responses include Yes, No, and Not applicable.

    • File Upload - A file upload box where contacts can browse and add a file or drag and drop the file.

    • Date - A date picker field with an inline calendar icon where contacts can select a date.

    • Simple Electronic Signature - A file upload box where users can upload a document for contacts to review. Contacts can choose whether to sign manually or select Accept electronically to accept the document. Bitte beachten Sie:

  1. If you add a Picklist response type or a Multi-Picklist response type, select Edit icon () to modify the response options.

    • Edit the existing options in the text fields or select Delete () to remove the response from the list.

    • Select + Option to add a new option to the list.

    • Select Other to add a text input field where contacts can enter a text response within the drop-down list.

    • Select Display inline to display the responses next to each other in the procedure rather than in a drop-down list.

    • Use the arrow up () and arrow down () to rearrange the options.

    2. Once you're done, select the Checkmark () to save your changes.

  1. If you add a File Upload response type, you can set up a few more options:

    1. Select the () button at the bottom of the upload box.

    2. In the File response settings dialog, select a target folder or placeholder location, an identifier and a name for the uploaded file.

  2. Once you're done, select Save.
