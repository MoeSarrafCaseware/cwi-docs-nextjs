Add a response type to the procedure

You can modify the options for the default response type or remove it and add a different one.

To add a response in a procedure:

  1. Select the procedure or subprocedure.

  2. Select More actions () | Settings, then select the Override the checklist settings check box.

  3. Select +Response.

  4. Enter the placeholder text.

    5. This is the default text that appears in the response field before the user enters or selects a response.

  5. Select a response type.

    If you've selected a Picklist, Multi-Picklist, Entity Information or Contact Information, you need to also set up the options for the response.

    1. Select the Edit icon ().

    2. Modify the existing options or select Delete () to remove it.

    3. Select + Option to add a new option to the list.

    4. Select Other to include Other as a list option.

      A text field is also included with this option to allows users to optionally add more information if they choose Other as an answer.

    5. Select Display inline to display the responses next to each other in the procedure rather than in a drop-down list.

    6. Use the arrow up () and arrow down () to rearrange the order of appearance of the response options.

    7. Once you're done, select the Checkmark () to save your changes.

    Note that you can also set visibility logic for picklist response options.

  1. Once you're done, select Save.
