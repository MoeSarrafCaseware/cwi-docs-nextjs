Add columns

Add columns to define the data values in your table. You can also remove columns if needed.

Note: Die Standardschriftgröße für Inhalte in dynamischen Tabellen ist 12 und kann nicht geändert werden.

To add a column:

  1. Wählen Sie die Tabelle aus.

  2. Select More actions (), then choose Edit Dynamic Table.

    3. Die Tabellenoptionen im Bearbeitungsmodus.

  3. Gehen Sie zu der Stelle, an der Sie eine Spalte hinzufügen möchten.

  4. Wählen Sie Hinzufügen () am oberen Tabellenrand.

    5. Der Tabelle wird eine Spalte hinzugefügt und die Spalteneinstellungen werden auf der Seite angezeigt.

Specify the column settings

You need to set up various options for each column in the table to define data values.

To set up the columns:

  1. Markieren Sie die Spaltenüberschrift.

  2. Modify the column settings as needed.

  1. Select DONE.

  2. Wählen Sie SPEICHERN oder SPEICHERN & SCHLIEßEN.

Edit the column descriptions

You can also edit column descriptions to have custom titles for the table columns.

To edit column descriptions:

  1. Wählen Sie die Tabelle aus.

  2. Wählen Sie Weitere Aktionen () und anschließend Dynamische Tabelle bearbeiten.

    3. Die Tabellenoptionen im Bearbeitungsmodus.

  3. Select the column that you want to edit.

  4. Select the Heading field.

    5. The Data Link dialog displays.

  5. In the Output field, edit the name as desired.

    6. You can also select the Data Link () icon to have the table description include dynamic text.

  6. Wählen Sie FERTIG.

  7. Select SAVE or SAVE & CLOSE.
