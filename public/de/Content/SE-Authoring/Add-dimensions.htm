Dimensionen hinzufügen

A dimension is a group property. An account can be assigned to one dimension within a specific group for further sorting and categorization.

You can create one or more dimension categories then populate the categories with dimensions. For example, you can create a dimension category called Location then create a dimension for each location that your client has. Once you create dimensions, you can then assign them to financial groups.

Note that firm authors and end users also have the ability to create dimensions and assign them to financial groups, with the following restrictions:

Primary author-created dimensions and categories:

  • Firm authors and end users can:

    • Assign to financial groups

    • Rename

    • Reorder

    • Add new dimensions to existing categories. Firm authors have full permissions for the dimensions they create.

Firm author-created dimensions and categories:

  • End users can:

    • Assign to financial groups

    • Rename

    • Reorder

    • Add new dimensions to existing categories. Firm authors have full permissions for the dimensions they create.

Note: Make sure dimensions are enabled in the product.json file, otherwise, the Dimensions tab will not be available. To learn more, see Enable or disable features in your product.

To add a set of dimensions:

  1. Go to Features | Dimensions.

  2. Select + Category, then add a name for the new dimension category.

  3. Select + Dimension to add a dimension for the new category.

  4. Enter a name for the new dimension.

    If you want to use dimension numbers to assign a unique identifier for each dimension, enter a value in the Number field. The number can consist of letters or numbers. These dimension numbers can be used to associate dimensions with transactions.

    You can drag and drop dimensions to rearrange them within a category and use the up and down arrows to expand and collapse categories. You can also select Delete Dimension () to remove a dimension.

    You can select Add () to add a second-level dimension to your primary dimension. You can also add third-level dimensions to second-level dimensions.

    You can set any dimension without children to be the default dimension for the category. Accounts in groups with this category applied will be automatically mapped to this dimension. For dimension categories without a default dimension set, accounts will not be automatically mapped to any dimension.

When you have created your dimensions, you can then assign the dimension categories to financial groups. Note that you can assign multiple dimension categories to a single group.

To assign a dimension category to a financial group:

  1. Go to the category in the product settings.

  2. Select More actions () | Assign Groups.

  3. If your product has multiple financial grouping structures, select the grouping structure you want from the drop-down.

  4. Select all the applicable financial groups.

    You can expand and collapse financial groups to view sub-groups and use the search bar to find the appropriate groups.

Note that you can also assign dimension categories to a group from the Group properties dialog.
