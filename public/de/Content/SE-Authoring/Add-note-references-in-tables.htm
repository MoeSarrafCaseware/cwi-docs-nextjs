Add note references in tables

A note reference is a link to one or more existing notes in the financial statement. You can add note references in table rows, including total rows, or columns in the financial statement. Zum Beispiel, um ein Konto oder eine Gruppe mit einer bestehenden Erläuterung zu verknüpfen, die entsprechende Details oder Informationen enthält.

Add a note reference in a row

You can reference notes in rows.

So fügen Sie eine Nummerierungsreferenz in einer Zeile hinzu:

  1. Wählen Sie die Tabelle aus.

    Symbole für Tabellenoptionen werden rechts angezeigt.

  2. Select More actions (), then choose Edit Dynamic Table.

    3. Die Tabellenoptionen im Bearbeitungsmodus.

  3. Gehen Sie zu der Zeile, in der Sie eine Nummerierungsreferenz hinzufügen möchten.

  4. Wählen Sie die Schaltfläche Hinzufügen () oberhalb der Zeile.

  5. Choose Commentary row.

    6. Eine Nummerierungsreferenz wird direkt über der Zeile hinzugefügt.

  6. Select (Note#) to display a list the existing notes in the financial statements document.

    Tipp: If you have set visibility logic for the notes, select the Show hidden notes check box to include hidden notes in the list.

  7. Select one or more notes from the list to reference it in the row.

  8. Wählen Sie SPEICHERN oder SPEICHERN & SCHLIEßEN.

    9. Die Nummer der referenzierten Erläuterung/Notiz wird neben dem Zeilennamen angezeigt.

Add a note reference in a column

You can reference notes columns in the financial statements document.

So fügen Sie eine Nummerierungsreferenz in einer Spalte hinzu:

  1. Fügen Sie der Tabelle eine Spalte hinzu.

    1. Wählen Sie die Tabelle aus.

      Symbole für Tabellenoptionen werden rechts angezeigt.

    2. Select More actions (), then choose Edit Dynamic Table.

      3. Die Tabellenoptionen im Bearbeitungsmodus.

    3. Gehen Sie zu der Stelle, an der Sie eine Spalte hinzufügen möchten.

    4. Wählen Sie Hinzufügen () am oberen Tabellenrand.

      5. Der Tabelle wird eine Spalte hinzugefügt und die Spalteneinstellungen werden auf der Seite angezeigt.

  2. Markieren Sie die Überschrift der neuen Spalte und ändern Sie den Spaltentyp in Nummerierungsreferenz.

  3. Geben Sie eine Überschrift ein und legen Sie die gewünschte Breite fest.

  4. Klicken Sie auf Fertig.

    5. Die neue Spalte wird in der Tabelle mit leeren Verweisen (#) hinzugefügt.

  5. Geben Sie die Nummerierungsreferenz für jede Zeile an.

    1. Wählen Sie einen leeren Verweis (#), um eine Liste der im Jahresabschlussdokument vorhandenen Anmerkungen anzuzeigen.

      Tipp: If you have set visibility logic for the notes, select the Show hidden notes check box to include hidden notes in the list.

    2. Wählen Sie eine oder mehrere Anmerkungen aus der Liste aus.

  6. Wählen Sie SPEICHERN oder SPEICHERN & SCHLIEßEN.
