Add or remove content areas

Areas help organize your financial statement content. Use areas to group similar or related information.

To add an area to the financial statement:

  1. Go to the Document Map, then select Add () | New Area.

    2. A new area is added in the document map and displays on the page with a default title: New Area. Select the default title to change it.

  2. Start adding content to the new area by selecting it to display the Add icon ().

To add content to the new area, select Add (), then choose one of the following content types:

You can add more than one content type in an area. You can also drag and drop the new area to change its location in the document, see Rearrange areas.

To remove an area from the financial statement:

  1. Gehen Sie in die Dokumentennavigation und wählen Sie den Bereich aus, den Sie entfernen möchten.

  2. Wählen Sie Weitere Aktionen () und anschließend in der Liste Bereich löschen.

    3. The area is removed and is no longer available on the Document Map.
