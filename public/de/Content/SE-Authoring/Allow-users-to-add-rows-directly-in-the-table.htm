Allow users to add rows directly in the table

You can allow users to add input rows (custom rows) in a dynamic table directly without opening the table in edit mode. This simplifies the process of adding input rows for end users, especially in cases where adding custom rows in a table happens frequently.

For example, if you know that in a particular place in the table, users add labor costs on multiple lines, so they add input rows. You don't predefine the input rows for them, because you don't know how many rows they'd need to add. If this case happens frequently, it can be overwhelming for users to go to the table set up screen (edit mode) each time they need to add extra rows in the table.

Note: This option can only be enabled in specific places in the table. You must know the specific place or places in the table where you want to allow them to add input rows outside of edit mode.

To enable adding input rows directly in the table area:

  1. Wählen Sie die Tabelle aus.

    Symbole für Tabellenoptionen werden rechts angezeigt.

  2. Select More actions (), then choose Edit Dynamic Table.

    3. Die Tabellenoptionen im Bearbeitungsmodus.

  3. Locate where you want to allow users to add input rows, then select the Add () | Custom group at the left border of the table.

    4. Note: Adding a custom group marks the place in the table where you want to allow users to add input rows outside edit mode.

    A new group row is added in the table with a default title Title. Sie können den Standardtitel auswählen, um ihn zu ändern.

    A Total row is also added in the table. It holds the total value for the balances in the input rows added by the user.

    If you don't want the group row or the total row to show on the table, select the row then select the eye icon () to hide it. In this case, it's recommended to also select Settings () | Show hidden items to keep track of all existing rows in the table as you're working on it.

  4. Select the new custom group.

    5. The Enable Input Rows icon () displays on the top left corner.

  5. Select Enable Input Rows ().

  6. Select SAVE & CLOSE.

The Add Row () button is now available in the exact place in the table where you want to allow users to add input rows.

To disable this option for users, select the group you added in Step 3, the select ().
