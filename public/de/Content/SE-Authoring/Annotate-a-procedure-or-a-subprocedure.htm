Annotate a procedure or a subprocedure

You can annotate a procedure or a subprocedure. For example, to predefine a link to supporting or related documents at specific procedures or subprocedures.

Sobald Sie eine Referenz hinzugefügt haben, erscheint eines der folgenden Symbole auf der Seite, um den Zweck der Referenz darzustellen:

  • Agrees to ()

  • Comes from ()

  • Goes to ()

  • Calculation Verified ()

  • Addition Checked ()

  • Link ()

  • Note ()

  • Variance ()

To annotate a procedure or a subprocedure:

  1. Gehen Sie zu der Prozedur oder Unterprozedur, die Sie mit einer Referenz versehen möchten, und wählen Sie Referenz hinzufügen Symbol ().

  2. Geben Sie die Informationen für die Referenz ein.

    1. Wählen Sie den Zweck der Referenz aus.

    2. Link the annotation to a document.

    3. Geben Sie Notizen oder Kommentare ein.

  3. Klicken Sie auf Speichern.

    The icon corresponding to each annotation purpose is displayed in the document. If you select the icon, a pop-up displays showing a summary of the annotation.

    The timestamp and username don't show in the pop-up because this is product defined annotation created in the product template not an engagement file.
