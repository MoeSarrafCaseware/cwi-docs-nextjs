Change an existing note reference in tables

You can change existing note references in the financial statements tables.

To change an existing note reference in tables:

  1. Wählen Sie die Nummerierungsreferenz aus, die Sie ändern möchten.

    A dialog opens show an option to navigate to the referenced note and an option to edit the note reference.

  2. In the dialog, select Edit to view a list of the available notes.

    Tipp: If you have set visibility logic for the notes, select the Show hidden notes check box to include hidden notes in the list.

  3. Select the check boxes to reference one or more notes.

  4. Select Save once you're done.

In tables, you also have the option to change note references using the table edit mode. For example, to change existing note references while making other edits in the table.

To change a note reference in a table using edit mode:

  1. Wählen Sie die Tabelle aus.

    Symbole für Tabellenoptionen werden rechts angezeigt.

  2. Select More actions ()| Edit Dynamic Table to open the table in edit mode.

  3. Go to the row where you want to change a note reference.

  4. Select the note reference.

    A dialog opens showing an option to navigate to the referenced note and an option to edit the note reference.

  5. In the dialog, select Edit.

    A list of the existing notes in the financial statement displays.

    Tipp: If you have set visibility logic for the notes, select the Show hidden notes check box to include hidden notes in the list.

  6. Select the check boxes to reference one or more notes.

  7. Select Save once you're done.
