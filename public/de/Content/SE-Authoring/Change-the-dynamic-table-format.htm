Change the dynamic table format

You can change the dynamic table format to support the International Financial Reporting Standards (IFRS) and the Statement of Changes In Equity (SOCIE). The new format flips the table so you can have dimensions as rows and trial balance groups as columns.

Note: When you change the format, existing setup and information in the table is lost and you need to redo it in the new format.

To switch to IFRS/SOCIE table:

  1. Wählen Sie die gewünschte Tabelle aus.

    2. Symbole für Tabellenoptionen werden rechts angezeigt.

  2. Select More actions () | Edit Dynamic Table.

    3. Die Tabellenoptionen im Bearbeitungsmodus.

  3. Select Settings (), then select IFRS/SOCIE under Format.

    4. A confirmation prompt displays to notify you that switching the table format creates a new table and all existing setup and data will be lost.

  4. Select Create a new table.

  5. Setup the table rows and columns as desired.

The following row and column options are now available in the table:

Rows

  • Dimension row - select a dimension to set the column to only display data for that dimension. You can set the dimension row to include current year or prior year data.

  • Input row - adds a text field row. You can enter custom text.

  • Commentary row - adds a commentary row. Standardmäßig wird ein Verweis auf vorhandene Anmerkungen in den Jahresabschlüssen eingefügt. Sie können den Standardwert ändern, indem Sie die Nummerierungsreferenz löschen und einen Kommentar für die Zeile eingeben.

  • Spacing row - adds white space into your dynamic table. Sie können Leerzeichen hinzufügen, um Ihren Inhalt leichter lesbar zu machen.

  • Page break - adds a page break between table rows.

  • Custom group - adds a custom group row. Custom group rows insert a text field (to add a title or a brief description for the group) and a total row.
Spalten

  • Trial balance - adds a trial balance date column. You can select an account or group to source the column data from. Wenn die Gruppe Untergruppen hat, werden diese ebenfalls in die Tabelle eingefügt.

  • Input - adds a text field column. You can enter custom text.

  • Custom calculation - adds a custom calculation column where you can enter a calculation using the calculation editor. Die Formel wird auf alle Spaltenzellen angewendet.

  • Note reference - adds a column of references to existing notes in the financial statements.

  • Annotation - adds an annotation column. Select Add () in each cell in the column to add annotations.
