Conditions for glossary terms and dynamic text display logic

The following listing describes the conditions required to configure the display logic for a glossary term or dynamic text. It also describes the parameters for each condition.

Response condition

Logic is based on a checklist procedure response.

Parameter:

  • Document - select the checklist document.
  • Procedure - select the procedure.
  • Condition - select the response.
  • Output - enter the text to be displayed when the condition is met.

Condition is met when the selected response in Condition is provided in the selected checklist procedure.

Trial balance condition

Logic is based on the trial balance group properties.

Parameter:

  • Financial Group -select the trial balance group.
  • Condition - select an option from the drop-down.
    • Consistent
    • Contrary
    • Inconsistent
  • Output - enter the text to be displayed when the condition is met.

For example, if the group balance is Consistent → Statement of Income displays.

Consolidation condition

Logic is based on whether the engagement is a consolidation engagement or not.

Parameter:

  • Condition -select the consolidation state (Consolidated or Non-consolidated).
  • Output - enter the text to be displayed when the condition is met.

Scaling/Rounding

Select a rounding scale from the drop-down.

Example

If the rounding is in Millions, then display this item.

Reporting period condition

Logic is based on the reporting period for the engagement.

Parameter:

  • Condition -select the value of the reporting period (Monthly, Quarterly, Semi-Annual, Annual)
  • Output - enter the text to be displayed when the condition is met.

Organization type condition

Logic is based on the country selected for the engagement entity.

Parameter:

  • Condition - select a country.
  • Output - enter the text to be displayed when the condition is met.

Form Value condition

Logic is based on the value of a form field.

Parameter:

  • Condition - select a form and a field in the selected form.
  • Output - enter the text to be displayed when the condition is met.

Note: This condition is only enabled if you have SE Builder forms set up in your product.

Group of conditions

Select this option if you want to add a group of conditions, where the text output is based on how the conditions in the group are collectively evaluated.

Parameter:

  • When all conditions below are met - select this option if you want all conditions to be true for the Group Condition to be met.
  • When any condition below is met - select this option if you want any condition to be true for the Group Condition to be met.
  • Output -enter the text to be displayed when the Group Condition is met.
