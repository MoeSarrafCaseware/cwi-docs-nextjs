Create a risk library

Risk libraries are predefined lists of risks that you can associate with products. Users can select the applicable and relevant risks from the library and add them to engagements. You can create multiple risk libraries to accommodate different risk documentation requirements for different products.

Once the risk library is enabled for your site, the Risk Library tab is available in your firm settings. To access the Risk Library tab, select Settings from the Cloud menu (), then select Risk Library | Manage Risk Library.

Note that to create a risk library, you must have the Settings Admin role or equivalent permissions. For more information, see Built-in roles and Assign system-wide security roles.

To create a library:

  1. Select the Add () icon in the left pane.

  2. In the Create risk library dialog, enter the Name and ID of the library.

  3. Select CREATE.

You can now add and publish a risk library version. Note that a risk library must have at least one published version before it can be associated with a product. To learn more, see Add a risk library version.
