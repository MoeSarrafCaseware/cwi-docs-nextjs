Current limitations

The current Smart Copy-Paste functionality has the following limitations that will be resolved in future releases.

  • You can copy the following documents:

    • Jahresabschluss

    • Letters

    • Memos

    • Worksheets

    You cannot copy:

    • Checklisten

    • SE Builder Forms

    • Word/Excel/PDF

    • Risk, Materiality, Misstatements

    • Analysis

    • Placeholder

    • Documents that include visibility settings or formulas that reference forms or embedded form sections

  • Once you copy documents, potential missing or inconsistent data across templates are automatically identified. To learn more, see Resolve conflicts in financial statements.

  • For all other documents, you need to manually identify and resolve data discrepancies.

  • Conflict resolution for copied financial statements:

    If a dynamic text with a response condition and a data link (fx) in the output text generates multiple conflicts, resolving one conflict causes the other conflicts to be removed from the Broken Links pane.

    You resolve those conflicts by:

    1. Manually locate broken formulas in the document.

      Broken formulas display as #ValueUnavailable#.

    2. Select the broken formula to launch the data link dialog and resolve the conflict.

  • Existing Issues in the source template (that is, the template where you copy documents from) are not copied.

    • Best Practice: We recommend keeping the grouping structure between the source and destination template identical.
