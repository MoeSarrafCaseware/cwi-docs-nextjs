Customize the financial statements settings

You can customize the financial statements settings.

At Features | Financial Statements, the following check boxes are available to allow you to customize the financial statements settings:

  • Enable translation

    To learn more, see SE 2024-09 release notes.

  • Enable the new financial statements layout

    Bitte beachten Sie:

    • This setting is now enabled by default.

  • Show account numbers on dynamic tables

    If you select this check box, account numbers display with the account name in the Description column in dynamic tables.

  • Show negatives with brackets

    Bitte beachten Sie:

    • For dynamic tables, this setting overrides the Negative setting you specified in the Format | Numbers tab

    • This settings is available for firm authors in the firm template

  • Show zero as dash

    Bitte beachten Sie:

    • If you select this check box, zeros will only display as dashes in the financial statements - not globally in the Cloud app

    • This setting is available for firm authors in the firm template

  • Use consistent paragraph spacing between content areas

    This setting enforces consistent paragraph spacing across text areas (between paragraphs and between a heading and a paragraph) for the financial statements, letters, memos and worksheets.

    Note that this setting only applies to the print copy (PDF). It does not affect how the documents are displayed when you view them in a template or engagement file.

  • Enable rounding in financial statements

    This setting enables automated rounding for dynamic tables in the financial statements. Once you enable this setting, you can select whether to round to Ones, Thousands, Millions or Billions and set the number of decimal places to use for rounded values.

    You can also determine whether rounding information is shown in description headings and column headings and customize the terminology used.

    Note that when rounding is enabled, firm authors and end users can also modify the rounding settings used in the engagement.
