Lockdown-Einstellungen anpassen

Sie können anpassen, wie der Engagement-Lockdown in Ihrem Produkt funktioniert. Bitte beachten Sie, dass Firmenautoren diese Einstellungen auch in der Firmenvorlage erstellen können.

Unter Funktionen | Lockdown stehen die folgenden Optionen zur Verfügung, die Ihnen ermöglichen, die Einstellungen für den Jahresabschluss anzupassen:

  • Aufhebung des Lockdowns nur für Administratoren erlauben - Wenn Sie diese Option wählen, können nur Benutzer mit der Rolle Administrator in Cloud die Engagements entsperren. Administrator-Benutzer können die Lockdown-Einschränkungen vorübergehend entfernen, um Nicht-Administrator-Benutzern zu ermöglichen, das Engagement zu entsperren.

  • Ziel-Lockdown-Tage aktivieren - Wenn Sie diese Option aktivieren, wird Ihnen eine Reihe von weiteren Optionen angeboten, die Ihnen ermöglichen, Ziel-Lockdown-Tage festzulegen.

    • Ziel-Lockdown-Tage - Geben Sie in diesem Feld die Anzahl der Tage ein, die verstreichen sollen, bis ein Lockdown für das Engagement ausgeführt werden soll und wählen Sie anschließend aus, ob die Anzahl der Tage ausgehend vom Unterschriftdatum des Berichts oder dem Veröffentlichungsdatum des Berichts gerechnet werden. Die Benutzer können das Unterschriftdatum oder das Veröffentlichungsdatum im Mandantenstammdaten-Popup festlegen.

    • Systemgesteuerten automatischen Lockdown aktivieren - Wenn Sie diese Option auswählen, wird automatisch ein Lockdown für das Engagement ausgeführt, wenn das Zieldatum erreicht ist.
