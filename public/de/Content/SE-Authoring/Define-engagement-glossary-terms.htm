Define engagement glossary terms

You can define the terms in the engagement glossary and configure the display logic for each term using conditions.

For example, if the user answers Yes to Does your accounting practice have only one practicing partner? procedure in the 010-Optimizer checklist, then the output term is My. If the answer is No, the output term is Our.

To add terms in the Engagement Glossary:

  1. Select Information () | Engagement Glossary.

    2. The Engagement Glossary displays. At this point, it's empty.

  2. Select Add Group ().

  3. Enter a group name, for example Terminology.

    4. The Add term button is now available on the page.

  4. Select Add term to open the Term Settings dialog.

  5. Enter a Description.

  6. Select the Add Condition drop-down list to configure the display logic for the glossary term.

    Different set of parameters display based on the condition you choose.

    For information on how to set up conditions, see Conditions for glossary terms and dynamic text display logic.

    You can add more than one condition type to trigger additional text outputs. They could be the same type or different types.

    To add another condition, select Add Condition. To add the same condition type select Copy Condition ().

    You can also optionally specify an alternate output for the text if no conditions are met.

    If you select the Allow user to override output checkbox, end users and firm authors can modify the default output values you set.

  7. Once you're done, select Save.

    8. If a condition (or conditions) is met, the corresponding output text you specified for that condition displays. If no condition is met, the alternate text output (the one you specified for when none of the conditions are met) displays. Otherwise, #ValueEmpty# displays.

Once you define terms, you can use them in documents. For information on how to use the glossary terms, see the following topics:

  • Add a glossary term in the procedure description.

  • Add a glossary term in the query question description.

  • Add a glossary term in text fields in financial statements and letters or memos.
