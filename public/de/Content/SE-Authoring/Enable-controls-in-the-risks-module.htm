Enable controls in the risks module

You can enable the controls in the risk module to allow users to document controls and meet audit documentation requirements.

To enable controls in your product, select the Information icon () at the top menu and choose Settings to open the Product Settings dialog. At the Controls | General tab, select Enable controls in the risk module.

Once you enable controls, the Controls tab is available to end users in the risk module and the module name is changed to Risks & controls if you have both risks and controls enabled. In this tab, users can:

  • Add, modify and delete controls

  • Expand control cards to view and edit control details

  • Link controls to risks

  • Add annotations and issues to controls

  • Use the filters to search for and view relevant controls

Also, more settings are available for controls in the product settings:

  • Signoff - Once you enable the signoff option, two check boxes are available to allow you to specify how users sign off the controls report.

    • Overall report - One signoff button is available in the report for users to sign off the entire report.

    • Individual control - A signoff button is available in each control card in the report. Users sign off each control individually.

    Once you select how users sign off the risk report, you can select a custom signoff scheme. To learn more about signoff schemes, see Set up signoff schemes.

  • Identify controls by - Select whether controls are identified by Business cycle or Financial statement area (FSA).

  • Help - if you want to set up a help page for your clients to describe the functionality, fields and attributes for controls, you can enable the Help option and add the URL. Once you enable this option, a Help icon () displays in control cards.

Customize the control fields

Under the Controls drop-down, select Fields.

The Controls fields are organized into four categories that can be expanded and collapsed using the up and down arrows:

  • Identification

  • Attributes

  • Design and implementation

  • Operating effectiveness evaluation

Note that the Identification category and the Title field in that category cannot be disabled.

To add and customize control fields:

  1. Select a field to include it in the controls report.

  2. For example, select to enable it in the controls report.

  3. In the Customize this label field, enter your custom term.

  4. For each field, select the On Dialog checkbox to make the field available to users in the Create New Control dialog.

  5. If the field has multiple options, enter your custom term for each option.

    For some fields, you can use the Add () button to add more custom options.
