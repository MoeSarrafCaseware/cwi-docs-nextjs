Enable grouping structures You can enable multiple financial grouping structures in your Cloud app, as well as Tax and Prescribed Accounts (Chart of Accounts) grouping structures.

Go to Product Settings | Trial Balance | Groups and enable each grouping structure you want to add to your product. Note that you can add a maximum of three financial grouping structures. You can also enter a custom name for each grouping structure.

If you want group numbers to appear with the group description in the trial balance, select the Show group numbers for financials check box. This is useful for markets that don't have a standard chart of accounts and want to simplify the trial balance by not displaying group numbers. This setting also shows group numbers in the Caseware Cloud Connector Add-in (see image below). To learn more, see How to get started with Cloud Connector.
