Customize the Data page

The Data page is where users can view and edit the data they have imported into their engagement. You can customize the Data page from the product settings.

In the Trial Balance | Data Page tab, you can select the following options:

  • Enable planning balances

  • Set parent groups - Select this option to display groups in the groups views of the Data page.

  • Set accounts - Select this option to display accounts in the groups view of the Data page.

  • Enable sampling

  • Enable data page signoff

  • Enable adjustments signoff

  • Enable new workflow for CSV/Excel trial balance imports

    Note that the new data import workflow is enabled by default for the vast majority of products.

  • Enable account mapping latest version - Select this option to enable the new design of the Assign Accounts page.

  • Enable split adjustment columns toggle - Select this option to enable the Split Adjustments option. When users turn this option on, normal and reclassifying adjustments are displayed in separate columns.

  • Enable user-defined financial groups - Select this option to allow end users to add financial groups in engagements.
