Enable the new Queries layout

Queries are switching to a new and improved user interface. To enable the new query design in your product, select the Information icon () at the top menu and choose Settings to open the Product Settings dialog.

Note: Make sure the query functionality is enabled in the product.json file, otherwise, the Queries tab will not be available. To learn more, see Enable or disable features in your product.

At Features | Queries, select Enable new query layout.

Note: We strongly advise that you test the new design on existing queries in your product before enabling it in the production environment.

Once you enable the new query layout, you can also enable the Simple Electronic Signature response type. This response type allows users to upload documents that require client signatures, such as engagement terms, and send them to clients to accept electronically. Contacts can choose whether to:

  • Elektronisch unterzeichnen - Wählen Sie die Schaltfläche Akzeptieren, um zu unterzeichnen. Die Kontakte können sich auch dafür entscheiden, ihre Standortdaten beim Unterzeichnen weiterzugeben, die dann im Signaturprotokoll aufgezeichnet werden.

  • Unterzeichnetes Dokument hochladen - Laden Sie das angehängte Dokument herunter, fügen Sie dann die Signatur ein oder drucken und unterzeichnen Sie das Dokument manuell und laden Sie es hoch.

  • Unterzeichnetes Dokument aus Cloud kopieren - Wählen Sie das zuvor heruntergeladene, unterzeichnete und hochgeladene Dokument von der Cloud-Website des Unternehmens.

If clients choose to accept electronically, a signature log is generated and available for download. Das Protokoll enthält die folgenden Informationen:

  • Abfragedetails

  • Name und E-Mail des Unterzeichners

  • Standort des Unterzeichners (wenn die Kunden den Zugriff auf den Standort aktiviert haben)

  • IP-Adresse des Unterzeichners

  • Datum der Unterzeichnung

  • Name des Dokuments

  • Dokument MD5-Hash

To enable this response type, select the Enable simple electronic signature in response types checkbox.
