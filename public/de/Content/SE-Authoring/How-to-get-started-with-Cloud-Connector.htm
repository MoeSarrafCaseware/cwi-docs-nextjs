How to get started with Cloud Connector?

Caseware Cloud Connector ist ein Microsoft Office Add-in, das eine Echtzeit-Datenverknüpfung zwischen Ihrem Engagement und MS Excel ermöglicht. Mit dieser Verknüpfung können Sie auf die Daten der Summenbilanz, die Eigenschaften des Engagements und die Firmeneigenschaften zugreifen und sie abrufen und dann in eine MS Excel-Datei einfügen. If a linked value or information changes in the engagement, it's automatically updated in the Excel file.

Sie können den Cloud Connector verwenden, um Jahresabschlüsse, Arbeitsblätter und Zeitpläne zu erstellen oder um Berechnungen zu überprüfen.

Beachten Sie, dass Cloud Connector die folgenden Informationen nicht abrufen kann:

  • Salden für Tochtergesellschaften in konsolidierten Engagements

  • Dimensionensalden

  • Vorläufige Salden

  • Buchungen

  • Wesentlichkeitswerte

  • Salden aus verschiedenen Kontengruppenstrukturen

  • Salden aus Vorperiode 2-4. Nur die Salden des laufenden Jahres und der Vorjahre werden unterstützt.

Note that two versions of the Connector add-in are available on the Office Add-in Store; one for partner environments and one for production environments. When authoring or testing with Cloud Connector on the partner sites, you must use the partner version of the add-in. Formulas you add using the partner add-in will continue to work when accessed from a production firm with the production version of the add-in.

Before you start, ensure your firm administrator has enabled a Caseware Cloud Connector license for your account. See Enable Cloud apps to learn more.

Erste Schritte:

  1. Wenn Sie Windows verwenden, stellen Sie sicher, dass Sie eine lokale Installation der neuesten Version von Microsoft Office 365 Desktop haben und die neueste Version von Windows 10 verwenden.

    Wenn Sie einen Mac verwenden, stellen Sie sicher, dass Sie eine lokale Installation von Office für Mac, Version 16.35 oder höher, haben und ein Mac-Betriebssystem verwenden, das Microsoft Office 365 unterstützt.

    Hinweis: MS Office Version 2019 oder früher und Office Online werden nicht unterstützt.

  2. Öffnen Sie eine Excel-Datei in MS Office 365.

  3. Hochladen der Excel-Datei in Ihr Engagement.

  4. Öffnen Sie die Excel-Datei auf der Seite Dokumente.

  5. Holen Sie sich Caseware Cloud Connector aus dem Microsoft Add-ins Store.

    If you are testing the add-in in a partner environment, search "WA200002372" in the store to locate the partner add-in.

    If you are testing the add-in in a production environment, search "WA200002239" in the store to locate the production add-in.

    Note: You can only have one version of the Connector add-in installed. If you will be testing the add-in in both partner and production environments you will need to remove the add-in before installing the other version. To remove the add-in, right-click the add-in and select Remove Add-in.

  6. Verwenden Sie Caseware Cloud Connector Add-in, um Daten aus Ihrem Engagement in die Excel-Datei einzufügen.
