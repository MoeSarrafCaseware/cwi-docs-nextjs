Link analytic tests to a checklist procedure

You can link to analytics tests in a checklist procedure. Analytics tests allow users to identify and analyze risks related to their engagement data and enhance their ability to meet audit objectives.

Once you link tests to a procedure, users can run the test and then view the test results and re-configure the tests before running them again if necessary.

See View analytic test results for checklists for more information on how users can use analytics tests in their engagements.

To link analytic tests to a checklist procedure:

  1. Navigate to the procedure or sub-procedure where you want to link analytics.

  2. Select More actions () | Attach analytics.

  3. In the Attach analytics dialog, select the analytic tests to attach to the procedure from the drop-down.

  4. Klicken Sie auf die Schaltfläche SPEICHERN.
