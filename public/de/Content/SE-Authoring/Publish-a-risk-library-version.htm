Publish a risk library version

After you upload an Excel file, you can publish your risk library version. Once the version is published, you can associate the risk library to products.

Bitte beachten Sie:

  • You can publish versions in any order. For instance, you can publish a patch version (that includes urgent fixes, such as some typos) even if you have an unpublished minor version.

  • Once you publish a new version of a risk library, all new and existing engagement files for products that are associated to the library are also updated to use the latest version.

To start, select Settings from the Cloud menu(), then select Risk Library | Manage Risk Library.

To publish a risk library version:

  1. Select the risk library from the left pane.

  2. Go to the library version that you want to publish.

  3. Select PUBLISH.

  4. In the confirmation dialog, enter the library name.
