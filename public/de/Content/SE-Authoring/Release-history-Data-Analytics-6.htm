Data Analytics DA-6

AnalyticsAI

This release includes the following bug fixes:

  • Redirects for AAI and CSV import links for AAI engagements created from subfolders on Collaborate.
  • Failure to export files due to tagged and blank posting date.
  • Configuration page does not work well with narrow viewports.
  • Configuration page risk score should have an input limit.
  • Not showing spinners on AnalyticsAI test LOADING status.
  • MS service (AAI, CSV Import) link goes through constant loop when engagement is created from folder.
  • Issues with AAI for tagged but blank posting date.

Data import

Major improvements have been made to how you import data into an engagement from online accounting platforms.

With this new functionality enabled, the Data page has been simplified to include a single button to access online accounting software.

When you select Online Accounting Software, the Import Transactions wizard displays:

These platforms will continue to be supported:

  • Exact (NL and BE)
  • SnelStart
  • Visma - eAccounting
  • Octopus (NL and BE)
  • Yuki (NL and BE)
  • Twinfield
  • Quickbooks Online
  • Xero

Additionally, we have added support for:

  • Freshbooks
  • Microsoft Dynamics 365: Business Central

Limitations for the initial release:

  • Transactions import is limited to all transactions within the last five years to the import date.
  • You will not be able to filter import providers by product.
