Response types

The following listing describes the response types that you can add to a checklist procedure.

Manual

An input text field where users can enter a custom response.

Expected user response: Geben Sie die gewünschten Informationen ein.

Auswahlliste

A drop-down list where users can select one response from a list of responses. You can customize the list of responses to meet your needs. The default responses are Yes, No, and Not applicable.

Expected user response: Wählen Sie ein Element aus der Drop-down-Liste aus.

Multi-Picklist

A drop-down box where users can select multiple responses from a list of responses. The default responses include Yes, No, and Not applicable.

Expected user response: Wählen Sie ein oder mehrere Elemente aus der Drop-down-Liste aus.

Datum

A date picker field with an inline calendar icon where users can select a date.

Expected user response: Wählen Sie ein Datum aus dem Kalender aus.

Entity Information

A field that displays information about the client entity. Entity information is linked directly to the client's profile on CasewareCloud. You can link one or more of the following fields to your procedure:

  • Legal name
  • Operating name
  • Jahresabschlussdatum
  • Startdatum des Jahres
  • Current year
  • Prior year
  • Adresse
  • Website
  • Entity number
  • Business number
  • Organization type
  • Business phone
  • Business fax
  • Mobiltelefon
  • Home phone
  • Industry codes
  • Client since
  • Date of incorporation

Expected user response: View and verify client data. The displayed information links to Caseware Cloud. The user can edit any incorrect information if needed.

Contact Information

A field that displays information about one of the client contacts. Contact information is linked directly to the client's user profile on CasewareCloud. You can link one or more of the following fields to your procedure:

  • First name
  • Last name
  • Initials
  • Email
  • Job title
  • Beschreibung
  • Start
  • phone
  • Work
  • phone
  • Mobiltelefon

Expected user response: View and verify client contact data. The displayed information links to Caseware Cloud. The user can edit any incorrect information if needed.
