SE 2021-06 patches

Here is a summary of the patches that have been deployed for SE 2021-06.

SE 2021-06-U1

Version: 2021-06 Build 648

Here is the list of issues that have been fixed with the deployment of SE 2021-06-U1.

  • [SE-27953] - In letters, when users copy list items with visibility conditions, the visibility logic does not work as expected for the new (pasted) list items.

  • [SE-27785] - Authors cannot add risk reference codes in the Product Settings.

  • [SE-27847] - Importing data from BGL Simple Fund 360 or Class Super does not work as expected. The import dialog is blank and prompts the user to sign in to the accounting software.

  • [SE-27916] - In checklists, when users select Dismiss remaining carryforward suggestions, the previously accepted carry forward suggestions display as dismissed. When users refresh the page, the suggestions display correctly.

  • [SE-27924] - Visibility settings in checklists do not work as expected. If the visibility conditions for a procedure are met, but the conditions for that procedure’s group or subgroup are not met, then the group or subgroup title does not display.

  • [SE-27927] - A technical issue identified after reviewing production logs.

  • [SE-27928] - A technical issue causing Carry Forward errors.

  • [SE-28018] - A technical issue causing errors when users attempt to edit calculation formulas in dynamic tables.

  • [SE-27835] - In the Summary of Misstatements document, the total adjustment amount is calculated incorrectly.

SE 2021-06-U2

Version: 2021-06 Build 649

Here is the list of issues that have been fixed with the deployment of SE 2021-06-U2.

  • [SE-28045] - An error occurs when authors attempt to associate a risk library to a product.

  • [SE-28036] - An error occurs when authors attempt to query the getTags API to retrieve tag details for risk groups.

  • [SE-28044] - When staff view queries using the Activities page, they cannot open PDF documents uploaded by clients in query responses.

  • [SE-28046] - Users receive “Extend session” messages when the engagement file is idle for less than 5 minutes.

  • [SE-28069] - When users add risks from a risk library to an engagement, the name for the associated risk group does not display.

SE 2021-06-U3

Version: 2021-06 Build 650

One issue has been fixed with the deployment of SE 2021-06-U3.

[SE-28095] - Performance improvement related to data links (fx) in checklists.

SE 2021-06-U4

Version: 2021-06 Build 654

Here is the list of issues that have been fixed with the deployment of SE 2021-06-U4.

  • [SE-28094] - The risk library page fails to open for some users due to an authorization and access error.

  • [SE-28145] - Risk library APIs do not provide access to new risk properties.

  • [SE-28068] - When assertions change between template versions, an error occurs when users attempt to carry forward engagements.

SE 2021-06-U5

Version: 2021-06 Build 656

One issue has been fixed with the deployment of SE 2021-06-U5.

[SE-28194] - A technical issue causing carry forward errors.

SE 2021-06-U6

Version: 2021-06 Build 657

The deployment of SE 2021-06-U6 addresses a performance issue impacting the availability of SE engagements.

SE 2021-06-U7

Version: 2021-06 Build 658

Here is the list of issues that have been fixed with the deployment of SE 2021-06-U7.

  • [SE-28172] - Caseware Cloud Connector users cannot insert data in Excel files because the Cloud Connector pane is empty.

  • [SE-28236] - A technical issue affecting custom form development.

  • [SE-28128] - In dynamic tables, cells in title rows that contain formulas are empty, instead of being hidden, when the account group is expanded.

  • [SE-28138] - Translation issue causing the MARK ALL RESPONDED button in queries to display in English when the content language of the product is not English.

  • [SE-28200] - A performance improvement related to Simple Electronic Signature response types in queries.

  • [SE-28246] - The Insert risk from library option is visible to primary authors in the product template.

  • [SE-28118] - An error occurs when users attempt to set visibility conditions based on specific organization types.

  • [SE-28212] - In financial statements, when the Use consistent paragraph spacing between content areas option is selected, the print preview option does not work as expected - text at the bottom of the pages is cut off.

  • [SE-28243] - When new documents are available in an engagement, the Documents pane does not automatically display the newly added documents and users must refresh the page.

SE 2021-06-U9

Version: 2021-06 Build 665

Here is the list of issues that have been fixed with the deployment of SE 2021-06-U9.

  • [SE-28367] - Users cannot edit text fields in a query if they open the More actions () menu for any comment while changes to a comment are being saved.

  • [SE-28375] - The Export visibility conditions option does not work as expected for documents with visibility conditions that refer to values originating from custom forms. An error occurs and the export is not completed.

  • The following issues apply to engagements created from carry forward:

    • [SE-28421] - An error occurs when users attempt to bulk carry forward a checklist that contains a conclusion section with procedures.

    • [SE-28422] - An error occurs when users open checklists. The checklist content does not display.

    • [SE-28376] - Users cannot add notes to the financial statements.

SE 2021-06-U10

Version: 2021-06 Build 670

Here is the list of issues that have been fixed with the deployment of SE 2021-06-U10.

  • Carry forward:

    • [SE-28456] - An error occurs when users attempt to bulk carry forward the engagement documents.

    • [SE-28466] - Financial statements notes that are set to be excluded in the prior year file are visible in the current year file until the user refreshes the page.

    • [SE-28540] - A “#ValueUnavailable#” error occurs for data links (fx) within text areas for the following use case:

      1. Users modified the text area in the prior year’s file.

      2. The primary author modified the same text area in the current year’s template

  • [SE-28501] - An error occurs when users attempt to add custom forms to a bulk print document group.

  • [SE-28506] - The Export visibility conditions option does not work as expected. The .csv file does not include all visibility conditions for query questions.

  • [SE-28523] - An error occurs when users attempt to sign off on a checklist if any procedures in the checklist with a Date response type do not have a valid date selected.

  • [SE-28586] - An error occurs when authors attempt to set visibility conditions based on organization type.

SE 2021-06-U11

Here is the list of issues that have been fixed with the deployment of SE 2021-06-U11.
