SE 2022-02 patches

Here is a summary of the patches that have been deployed for SE 2022-02. The patches are listed in order with the latest patch first.

Note:To check your product's current version, select Information () | About in your template. The About dialog displays both the server version and the client version of your product. You can check the server version to determine whether you have the latest server patch and the client version to determine whether you have the latest client patch. The server and client versions are displayed in the following format:

  • Server version : 2022-02 Build 123 Release
  • Client version: release-se-2022-02-u1-123456789

SE 2022-02-U8 Client

Version: 2022-02 Build release-se-2022-02-u8-2489262212

Here is the list of issues that have been fixed with the deployment of SE 2022-02-U8 Client.

  • [SE-32513] - When users print the financial statements, the generated PDF does not display as expected. Trial balance rows with zero balance accounts are included in the PDF even if they are not set to display.

  • [SE-32501] - In the notes to the financial statements, dynamic tables with custom calculations that use values from custom forms do not display if the form value is incorrect or not complete.

  • [SE-32507] | [SE-32429] - On the Adjustments page and in the financial statements, decimal values with comma separators do not display as expected.
