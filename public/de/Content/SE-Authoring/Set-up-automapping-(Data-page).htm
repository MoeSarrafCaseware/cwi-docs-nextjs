Set up automapping (Data page)

You can set up automapping so that accounts are automatically assigned to the best suited groups when users import the client's trial balance data.

To set up automapping:

  1. Enable the automap feature in your product (see, List of product features (API)).

  2. Use the Universal Classification List to look up and determine the appropriate group classification for the accounting package.

  3. For each group:

    1. Select Edit () to open the Group Properties dialog.

    2. Enter the group classification in the Classification of group field.

    3. Klicken Sie auf Speichern.

    4. Tipp: You can also add the group classifications in the group import excel or CSV file and import it.
