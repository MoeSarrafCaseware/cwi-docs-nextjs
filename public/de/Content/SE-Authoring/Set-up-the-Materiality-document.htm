Set up the Materiality document

You can enable the materiality document to allow users to define materiality for financial areas.

To enable the Materiality report in your product, specify the com.caseware.ca.general.enhanced-materiality in the form property in the product.json file. To learn more, see Enable or disable features in your product.

Once you enable the document in your product, you can add it in your draft and set it up.

To add the materiality document, select Add () at the documents page, then choose Materiality.

To set up the Materiality document:

  1. Add guidance information.
