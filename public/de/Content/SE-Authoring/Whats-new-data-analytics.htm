What’s new - Data Analytics DA-12

AnalyticsAI

Notebooks updates

Distributors can now deploy and enable custom Notebooks in their AnalyticsAI product. Additionally, distributors and primary authors can define default parameters and autorun capability for a Notebook. The configuration options must be defined in the relevant product.json file.

See Get started with AnalyticsAI Notebooks SDK for more information.

Data import

Imports improvements

This release includes a minor update to the new desktop file import capabilities that were made available in the DA-11 release. With this update, users that navigate back to the file upload configuration form either during an initial import, or on re-import will be able to view and edit the files that have been uploaded.

Other improvements

Improved integration with New Relic APM (app performance monitoring) to support:

  • Increased transparency on import from start to finish
  • Improved metrics on app performance and load times
