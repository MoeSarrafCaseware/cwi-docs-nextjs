Workflow (how to author letters and memos?)

To add a letter, select Add () at the documents page, then choose Letter.

To add a memo, select Add () at the documents page, then choose Memo.

A blank letter or memo opens. Select Edit () to start authoring.

The following steps summarize the recommended workflow for authoring letters and memos. You can use this workflow for guidance.

  1. Add guidance

  2. Add reference to a document

  3. Add a Table of Contents

  4. Add a grouping area

  5. Add a text area

  6. Seitenumbruch hinzufügen

  7. Add header and footer

  8. Set up the print options

  9. Add visibility settings
