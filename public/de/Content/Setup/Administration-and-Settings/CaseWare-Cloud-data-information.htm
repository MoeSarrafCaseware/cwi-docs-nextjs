Caseware Cloud Datenschutzrichtlinie

Caseware Cloud ermöglicht die einfache Speicherung Ihrer Dateien für die Verwendung mit der Caseware Produktreihe.

Wo werden meine Daten gespeichert?

Alle Abonnentendaten werden in der Caseware Cloud Produktionsumgebung gespeichert. Die Verwendung von Kundendaten in Nicht-Produktionsumgebungen wird durch sichere Datenverarbeitungsprozesse gesteuert, die eine ausdrücklich dokumentierte Genehmigung der Kunden erfordern, deren Daten betroffen sind, und muss die gesetzlichen und behördlichen Anforderungen an die Bereinigung sensibler Datenelemente erfüllen.

Es gibt ein ausgewiesenes Team, das für alle Aufgaben in Bezug auf die Infrastruktur und die Speicherung verantwortlich ist und dessen Zuständigkeiten festgelegt, dokumentiert und kommuniziert wurden.

Caseware Cloud wird auf Amazon-Webservern auf der ganzen Welt gehostet. Beim Abonnieren von Caseware Cloud Diensten werden Kunden über den Gerichtsstand des Servers informiert, der für das Hosting der Abonnentendaten und der personenbezogenen Daten des Kunden zugeteilt wurde. Sie können entweder dieser Zuteilung zustimmen oder einen auf diese Weise zugeteilten Server ablehnen.

Aus Performance-Gründen werden Ihre Daten an folgenden Orten gehostet:

  • USA/Nord Virginia, wenn Sie in den USA oder Südamerika ansässig sind
  • Kanada/Montreal, wenn Sie in Kanada ansässig sind
  • Australien/New South Wales, wenn Sie im asiatisch-pazifischen Raum ansässig sind
  • Irland/Leinster, wenn Sie in einer anderen Region ansässig sind

Wie kann ich Einzelheiten über meine aktuelle Datennutzung erfahren?

Details zu den in Ihrem Caseware Cloud Konto gespeicherten Daten werden zu Ihrer Information katalogisiert. Sie können die aktuelle und die maximale Datenmenge sowie eine Reihe anderer Statistikdaten einsehen.

Zum Anzeigen von Details zu Ihrer aktuellen Datennutzung:

  1. Stellen Sie sicher, dass Sie über die Rolle Administrator verfügen. Administrator-Rolle oder gleichwertige Berechtigungen.

  2. Wählen Sie im Caseware Cloud Hauptmenü den Eintrag Einstellungen.

  3. Wählen Sie System | Website-Statistiken.

Wählen Sie dazu im Caseware Cloud Hauptmenü den Eintrag Einstellungen und öffnen Sie die Registerkarte Website-Statistiken.

Wie lange werden die Daten nach der Deaktivierung aufbewahrt?

In deaktivierten Caseware Cloud Konten bleiben alle Daten erhalten, die zum Zeitpunkt der Deaktivierung gehostet wurden. Wenn ein deaktiviertes Konto wieder aktiviert wird, ist der Zugriff auf diese Daten wieder möglich.

Derzeit gibt es kein Ablaufdatum für gespeicherte Daten. Wenn Sie Ihr Caseware Cloud Konto und alle damit verbundenen Daten vollständig löschen möchten, können Sie dies unter sales@audicon.net beantragen. Bitte beachten Sie, dass das Konto und die Daten nach dem Löschen nicht wiederhergestellt werden können.
