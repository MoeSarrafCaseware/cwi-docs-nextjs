Dateien mithilfe von Ordnern und Tags verwalten

In Caseware Cloud können Sie Ihre Dateien auf unterschiedliche Art und Weise verwalten.

Dateien in Ordnern verwalten

Verwenden Sie Ordner, um die Dateien in einer Entität leichter zu verwalten.

Zum Erstellen eines Ordners:

  1. Klicken Sie im Caseware Cloud Hauptmenü () auf den Eintrag Dateien.

  2. Wählen Sie die betreffende Entität aus der Drop-down-Liste der Entitäten aus.

  3. Wählen Sie im Bereich Ordner den Ordner aus, unter dem Sie einen neuen Ordner anlegen möchten.

  4. Öffnen Sie über den kleinen Pfeil () neben dem betreffenden Ordner die Drop-down-Liste und wählen Sie Neuen Ordner erstellen.

  5. Geben Sie einen Namen und eine Beschreibung für den Ordner ein.

  6. Klicken Sie auf die Schaltfläche OK.

Zum Verschieben von Dateien in einen Ordner:

  1. Wählen Sie die Dateien aus, die Sie in einen Ordner verschieben möchten.

  2. Klicken Sie im Bereich zu den Dateidetails auf das Symbol Weitere Aktionen () und wählen Sie dann den Eintrag In Ordner verschieben.

  3. Wählen Sie den Ordner aus, in den Sie die Datei verschieben möchten, und klicken Sie auf Verschieben. Alternativ besteht die Möglichkeit, die Datei per Drag-and-drop in einen anderen Ordner zu ziehen.

Zum Löschen eines Ordners:

  1. Wählen Sie den Ordner aus, den Sie löschen möchten.

  2. Öffnen Sie die zugehörige Drop-down-Liste () und wählen Sie den Eintrag Löschen.

  3. Klicken Sie in der daraufhin angezeigten Meldung auf Ja, um das Löschen zu bestätigen.

Wenn Sie einen Ordner versehentlich gelöscht haben, können Sie ihn wiederherstellen. Weitere Informationen hierzu finden Sie in Ordner und Dateien aus dem Papierkorb wiederherstellen.

Dateien mit Tags versehen

Mithilfe von Tags können Sie Ihre Dateien kategorisieren, sodass sie leichter zu finden sind.

Bevor Sie Tags zuweisen können, müssen Sie diese zunächst erstellen. Weitere Informationen zum Erstellen von Tags finden Sie in Tags erstellen.

Sie müssen außerdem sicherstellen, dass Sie über die Rolle Editor bzw. entsprechende Berechtigungen verfügen. Weitere Informationen zu Sicherheitsrollen finden Sie in Integrierte Rollen.

Gehen Sie wie folgt vor, um einer Datei Tags zuzuweisen:

  1. Wählen Sie eine Datei aus und klicken Sie dann auf das Symbol Tag ().

    Einer Datei Tags zuweisen

    Mehreren Dateien Tags zuweisen

  2. Wählen Sie die Tags aus, die der Datei zugewiesen werden sollen.

  3. Klicken Sie auf die Schaltfläche Speichern.

Mehrere Dateien auf einmal verschieben

In Caseware Cloud haben Sie die Möglichkeit, mehrere Dateien gleichzeitig in eine Entität oder einen Ordner zu verschieben.

Zum Verschieben von Dateien in eine andere Entität:

  1. Wählen Sie Alle Entitäten oder die Entität mit den zu verschiebenden Dateien in der Entitätsauswahl aus.

  2. Klicken Sie im Caseware Cloud Hauptmenü () auf den Eintrag Dateien.

  3. Wählen Sie die Dateien aus, die Sie in eine andere Entität verschieben möchten.

  4. Wählen Sie Weitere Aktionen () | In Entität verschieben.

  5. Wählen Sie die Entität aus, in die Sie die Dateien verschieben möchten, und klicken Sie auf OK.

  6. Klicken Sie im Bestätigungsdialog auf Ja.
