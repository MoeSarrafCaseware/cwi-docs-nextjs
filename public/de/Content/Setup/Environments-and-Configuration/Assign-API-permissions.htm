API-Berechtigungen zuweisen (Azure AD)

Die Konfiguration der API-Berechtigungen erlaubt Caseware Cloud Informationen aus Microsoft Azure zu lesen, um Benutzer richtig zu authentifizieren.

API-Berechtigungen zuweisen:

  1. Wählen Sie Azure Active Directory | App Registrations und wählen Sie anschließend Ihre App.

  2. Im linken Navigationsbereich wählen Sie API permissions.

  3. Wählen Sie Add a permission.

  4. Wählen Sie Microsoft Graph.

  5. Auswahl von Delegated permissions.

  6. Verwenden Sie das Kontrollkästchen, um die folgenden Berechtigungen auszuwählen:

    • Email

    • Openid

    • Profile

    • User.Read

  7. Wählen Sie Add permissions.

  8. Wählen Sie Grant admin consent for Default Directory.

  9. Klicken Sie auf Yes, wenn die Abfrage "Do you want to grant consent for the requested permissions for all accounts in Default Directory?“ angezeigt wird.
