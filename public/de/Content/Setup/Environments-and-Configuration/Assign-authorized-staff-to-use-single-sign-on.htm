Einmaliges Anmelden autorisierten Mitarbeiter zuweisen (Azure AD)

Wenn Sie sicherstellen möchten, dass nur autorisierte Mitarbeiter in Ihrem Unternehmen via Einmaliges Anmelden auf Caseware Cloud zugreifen können, haben Sie die Möglichkeit, autorisierte Benutzer Ihrer App in Azure zuzuweisen. Sollen alle Mitarbeiter Ihres Unternehmens auf Caseware Cloud zugreifen können, überspringen Sie ggf. diesen Schritt und fahren Sie damit fort API-Berechtigungen zuzuweisen.

Autorisierte Mitarbeiter zuordnen:

  1. Wählen Sie Azure Active Directory | App Registrations und anschließend Ihre App.

  2. Im linken Navigationsbereich wählen Sie Properties.

  3. Setzen Sie die Option User assignment required auf Yes.

  4. Klicken Sie auf Speichern.

  5. Im linken Navigationsbereich wählen Sie Users and Groups.

  6. Wählen Sie Add user.

  7. Wählen Sie Users und dann jeden Benutzer, den Sie der Anwendung zuordnen möchten.

    Hinweis: Wenn Sie ein Premium Microsoft Azure Benutzerkonto haben, können Sie Benutzergruppen erstellen und anschließend Gruppen auswählen, um den Zugriff für alle Mitglieder der Gruppe zu erlauben. Weitere Informationen finden Sie in Eine Basisgruppe erstellen und via Azure Active Directory Mitglieder hinzufügen.

  8. Wählen Sie Select, wenn Sie alle Benutzer, die Zugriff haben sollten, hinzugefügt haben.

  9. Wählen Sie Assign, um diesen Benutzern zu erlauben, auf die Anwendung zuzugreifen.
