Metadatenendpunkt kopieren (Azure AD)

Ein Metadatenendpunkt erteilt die Autorisierung für URLs und andere Metadaten, die erforderlich sind, damit Einmaliges Anmelden (Single Sign On/SSO) funktioniert. Er wird verwendet, wenn Sie Einmaliges Anmelden (SSO) in Caseware Cloud konfigurieren.

Metadatenendpunkt kopieren:

  1. Melden Sie sich an Ihrem Azure Portal an.

  2. Wählen Sie Azure Active Directory.

  3. Im linken Navigationsbereich wählen Sie App-Registrierungen.

  4. Wählen Sie die richtige App aus der Liste.

  5. Wählen Sie Endpunkte.

  6. Eine Liste der Endpunkte wird angezeigt.

    Hinweis: Kopieren Sie den Endpunkt des OpenId Connect Metadaten-Dokuments und fügen Sie ihn zur späteren Verwendung in ein Textdokument ein. Kennzeichnen Sie Metadatenendpunkt.
