Neue Anwendungsgruppe erstellen (ADFS)

Um Caseware Cloud in ADFS zu integrieren, müssen Sie zuerst eine neue Anwendungsgruppe für Cloud erstellen.

Eine neue Anwendungsgruppe erstellen:

  1. Starten Sie ADFS Administration Console.

  2. Klicken Sie auf den Ordner Application Groups und anschließend im Menü Actions auf Add Application Groups.

  3. Geben Sie einen Namen für die Gruppe ein. Wählen Sie aus der Template-Liste Web API. Klicken Sie auf Next.

  4. Das Feld Client Identifier wird automatisch ausgefüllt.

    Hinweis: Kopieren Sie die Client-ID und fügen Sie sie zur späteren Verwendung in ein Textdokument ein. Nennen Sie das Textdokument Client-ID.

    Geben Sie im Feld Redirect URI einen URL-Platzhalter (zum Beispiel https://www.caseware.com) ein und klicken Sie auf Add. Klicken Sie auf Next.

  5. Wählen Sie Generate a shared secret.

    Hinweis: Klicken Sie auf Copy to clipboard und fügen Sie es zur späteren Verwendung in ein Textdokument ein. Nennen Sie das Textdokument Geheimer Clientschlüssel.

    Klicken Sie auf Next.

  6. Klicken Sie auf Next, um die Konfiguration abzuschließen.
