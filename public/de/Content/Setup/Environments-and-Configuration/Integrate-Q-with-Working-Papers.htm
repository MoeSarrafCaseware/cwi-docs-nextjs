Caseware Q in Caseware Working Papers integrieren

Diese Funktion ist für die deutsche Version von Caseware Cloud und AuditTemplate nicht verfügbar. Wenn Ihr Unternehmen über die Q App verfügt, können Sie diese in ein Working Papers Audit Template integrieren, um die Funktionen dieser App zu integrieren.

Q in Working Papers integrieren:

  1. Wählen Sie in Working Papers Extras | Vorlagen, um den Dialog Vorlagen zu öffnen.

  2. Wählen Sie die Vorlage aus, die Sie integrieren möchten und klicken Sie dann auf das Symbol Bearbeiten. Die Vorlage wird geöffnet.

  3. Navigieren Sie im Menüband Start zum Menü Einfügen und wählen Sie Aus Bibliothek. Wählen Sie aus dem Drop-down-Menü die zu integrierende Vorlage.

  4. Q muss zur Vorlage hinzugefügt werden. Klappen Sie in der Dokumentenbibliothek die folgenden Ordner auf, um Q zu erreichen:

    • Caseware Audit System Documents | Control Documents | Q - Caseware Q

  5. Wählen Sie Q und klicken Sie auf Hinzufügen.

  6. Klicken Sie auf Schließen.

Die Q App wurde in Ihr Audit Template integriert. Sie können nun den Fortschritt Ihrer Prüfung in Caseware Cloud anzeigen, indem Sie die Q () App auswählen.
