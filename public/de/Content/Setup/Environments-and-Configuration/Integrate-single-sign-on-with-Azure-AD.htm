Einmaliges Anmelden in Azure AD integrieren

Überblick

Bevor Sie die Funktion Einmaliges Anmelden (Single-Sign-On/SSO) in Azure AD integrieren, sollten Sie Folgendes sicherstellen:

  • Einmaliges Anmelden für Ihre Firma aktivieren.

  • Mitarbeiter in Azure AD sind nun durch Microsoft Office 365 integriert. Sobald Sie die Funktion Einmaliges Anmelden integriert haben, können sich die Mitarbeiter mit Ihrer E-Mail-Adresse von Office 365 in Caseware Cloud anmelden. Benutzer mit Gastkonten in Ihrem Verzeichnis können Einmaliges Anmelden (SSO) verwenden.

    Hinweis: Bei Azure-Benutzernamen handelt es sich nicht um E-Mail-Adressen und sie können nicht für die Anmeldung in Cloud verwendet werden.

  • Bei Mitarbeitern mit existierenden Caseware Cloud Benutzerkonten wurde ihr Cloud Benutzerkonto aktualisiert, so dass sie ihre E-Mail-Adresse von Office 365 verwenden können. Sollte ein Mitarbeiter eine andere E-Mail-Adresse für sein Caseware Cloud Benutzerkonto verwenden, wird ein Duplikat des Benutzerkontos für ihn erstellt, sobald Einmaliges Anmelden integriert wurde.

Wir empfehlen, dass die Integration von Einmaliges Anmelden durch einen Administrator ausgeführt wird, der mit dem Einrichten der Funktion Einmaliges Anmelden in Azure AD vertraut ist.
