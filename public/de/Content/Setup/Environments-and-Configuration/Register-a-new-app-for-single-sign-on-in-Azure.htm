Registrieren einer neuen App für Einmaliges Anmelden (Azure AD)

Zur Integration von Caseware Cloud in Azure AD müssen Sie Caseware Cloud als neue App registrieren. Nach der Registrierung können Sie Caseware Cloud den Zugriff auf Ihre Active Directory-Domäne erlauben.

Eine neue App für Einmaliges Anmelden in Azure AD registrieren:

  1. Melden Sie sich an Ihrem Azure Portal an. Wenn Sie Zugriff auf mehr als ein Portal haben, können Sie Ihr Benutzerkonto in der rechten oberen Ecke wählen und anschließend den gewünschten Tenant (-Mandanten).

  2. Wählen Sie Azure Active Directory.

  3. Im linken Navigationsbereich wählen Sie App registrations.

  4. Wählen Sie New registration.

  5. Geben Sie auf der Seite Register an application Ihre Registrierungsdaten ein:

    • Name: Geben Sie einen passenden Namen ein.

    • Supported account types: Wählen Sie Accounts in this organizational directory only (Default Directory only - Single tenant).

    • Redirect URI: Wählen Sie Web. Sie müssen keine URL eingeben.

  6. Nach Eingabe Ihrer Informationen wählen Sie Register. Ihre Anwendungsdetails werden angezeigt.

    Hinweis: Kopieren Sie die Anwendungs-ID und fügen Sie sie zur späteren Verwendung in ein Textdokument ein. Nennen Sie das Textdokument Anwendungs-ID.
