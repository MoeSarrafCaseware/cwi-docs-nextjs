Entitätsverbund erstellen

Entitätsverbünde gruppieren zwei oder mehr Entitäten in Caseware Cloud und bieten Ihnen damit eine konsolidierte Ansicht der Aktivitäten und Dateien für jede Entität, die ein Bestandteil des Verbunds ist. Entitäten in Entitätsverbünden verbleiben unabhängig voneinander.

Mögliche Entitätsverbünde

Sie können einen Verbund für jeden Entitätstyp erstellen - Mandanten, Interne Entitäten oder Andere Entitäten.

Wenn Sie z. B. Aufgaben für verschiedene Niederlassungen eines großen Unternehmens ausführen, kann jede Niederlassung durch eine separate Mandantenentität repräsentiert werden und Sie erstellen anschließend daraus einen Entitätsverbund.

Ein Verbund darf auch unterschiedliche Entitätstypen umfassen. Sollten Sie z. B. eine interne Entität für Ihr Steuer-Team haben, können Sie sie mit den Mandanten-Entitäten, für die Sie Steuerdienstleistungen anbieten, verbinden.

Jede Entität kann nur Bestandteil eines Entitätsverbunds sein. Beim Versuch eine Entität mit Entitäten zu verbinden, die sich bereits in einem Verbund befinden, führt dies zu einem kombinierten Entitätsverbund.

Nehmen Sie zum Beispiel an, dass es zwei Entitätsverbünde gibt; einer besteht aus {A, B, C} und der andere aus {D, E, F}. Wenn Sie nun eine Entität aus dem einen Entitätsverbund mit einer Entität aus dem anderen verbinden (zum Beispiel wird die Entität C mit der Entität E verbunden), führt dies zu einem kombinierten Entitätsverbund {A, B, C, D, E, F}.

Entitätsverbünde anzeigen

Bei der Anzeige einer Entität, die Bestandteil eines Entitätsverbundes ist, erscheint ein Symbol neben dem Entitätsnamen. Dieses Symbol verhält sich wie ein Schalter, der zwischen einer Anzeige Entitätsverbund (), die den Inhalt aller verbundenen Entitäten umfasst, und einer Anzeige einzelne Entität (), die nur den Inhalt der ausgewählten Einheit anzeigt, wechselt.

Einen Entitätsverbund erstellen:

  1. Stellen Sie sicher, dass Sie über die Rolle Mitarbeiteradministrator bzw. entsprechende Berechtigungen verfügen.

  2. Klicken Sie im Caseware Cloud Hauptmenü auf den Eintrag Entitäten.

  3. Wählen Sie eine Entität, die Sie mit einer anderen Entität in einem Verbund zusammenfassen möchten und wählen Sie Weitere Aktionen () | Verbund zuweisen.

  4. Suchen Sie nach den Entitäten, die Sie hinzufügen möchten. Sie werden im Dialog entsprechend ihres Entitätstyps angezeigt.

  5. Klicken Sie auf Schließen.

Berechtigungen Entitätsverbünde

Benutzer können Entitäten in einem Entitätsverbund anzeigen, hinzufügen und entfernen. Es werden keine separaten Berechtigungen für Entitätsverbünde gewährt; ein Benutzer darf basierend auf den Berechtigungen für die individuellen Entitäten die Entitäten innerhalb des Entitätsverbunds anzeigen oder bearbeiten.

  • Zur Anzeige von Entitäten innerhalb eines Entitätsverbunds muss ein Benutzer über die Berechtigung Lesen in der Zusammenfassung der Attribute für jede Entität verfügen. Benutzer, die nicht für alle Entitäten in einem Entitätsverbund über diese Berechtigung verfügen, können nur die Teilmenge der Entitäten sehen, für die sie diese Berechtigung haben.

    Benutzer, die diese Berechtigung nur für eine Entität im Entitätsverbund haben, können den Entitätsverbund nicht sehen.

  • Zur Anzeige und zum Entfernen von Entitäten innerhalb eines Entitätsverbunds muss ein Benutzer über die Berechtigung Bearbeiten in den Details der Attribute für jede Entität verfügen. Eine Entität in einem Entitätsverbund kann nur von einem Benutzer hinzugefügt oder entfernt werden, wenn die Berechtigung für diese Entität gewährt wurde.

    Obwohl Benutzer möglicherweise in der Lage sind, eine Entität anzuzeigen, kann es sein, dass sie sie in Entitätsverbünden ohne die Berechtigung Bearbeiten nicht hinzufügen oder entfernen dürfen.
