Interne Entitäten organisieren

Interne Entitäten können eingesetzt werden, um die Dateien und Aktivitäten Ihres Unternehmens in Caseware Cloud zu organisieren. Interne Entitäten können Abteilungen, Aktivitäten oder Spezialisierungen in Ihrem Unternehmen repräsentieren.

In diesem Beispiel hat der Administrator interne Entitäten nach der Abteilung erstellt. Das Management-Team hat entschieden, dass das Administrations-Team davon profitieren würde, eine eigene Entität zu haben, und der Administrator fügt eine neue Entität hinzu.

Eine Interne Entität erstellen:

  1. Stellen Sie sicher, dass Sie über die Rolle Entitätsadministrator bzw. entsprechende Rechte verfügen.

  2. Klicken Sie im Caseware Cloud Hauptmenü auf Entitäten.

  3. Wählen Sie Neu | Interne Entität.

  4. Geben Sie einen Name(n) für die Entität ein und vervollständigen Sie die verbleibenden Felder nach Bedarf.

  5. Wählen Sie Speichern, um die Entität hinzuzufügen.

Die Entität wurde erstellt. Die neue Entität wird auf der Seite Entitäten angezeigt.

Zugriff auf Ihre internen Entitäten verwalten

Wenn Sie Ihre interne Entität erstellt haben, müssen Sie die Freigabeeinstellungen verwalten, um sicherzustellen, dass die passenden Mitarbeiter und Kontakte auf die damit verbundenen Dateien und Aktivitäten zugreifen können.

Benutzerberechtigungen für eine Entität anzeigen oder bearbeiten:

  1. Stellen Sie sicher, dass Sie über die Rolle Mitarbeiter: Eigentümer bzw. entsprechende Berechtigungen verfügen.

  2. Klicken Sie im Caseware Cloud Hauptmenü auf Entitäten.

  3. Wählen Sie die Entität, die Sie bearbeiten möchten, und anschließend Freigabe.

  4. Wählen Sie die Benutzer oder Gruppen aus, denen Sie Zugriff auf die Entität erteilen wollen. Öffnen Sie dann die Drop-down-Liste Rollen an (#) zuweisen und wählen Sie die betreffenden Rollen aus.

  5. Klicken Sie auf Freigabe.
