Systemweite Zugriffsrollen zuweisen

Sie können Anwendern oder Gruppen systemweite Rollen zuweisen. Diese Rollen geben den Benutzern die Fähigkeit, spezifische Typen von Inhalten in Ihrer gesamten Cloud-Instanz anzuzeigen und zu bearbeiten. Die Rolle Beobachter gewährt z. B. Benutzern die Möglichkeit, Dateien und Aktivitäten anzuzeigen, die in Entitäten beinhaltet sind, und Kommentare zu hinterlassen, während die Rolle Zugriff auf Entität nur zulässt, dass Benutzer Informationen über die Entität selbst anzeigen lassen können, wie z. B. wann die Entität hinzugefügt wurde oder deren organisatorische Struktur.

Alternativ können Sie entitätsspezifische, dateispezifische oder aktivitätsspezifische Rollen zuweisen. Sie können auch eine Kombination dieser Rollen einem Benutzer oder einer Gruppe zuweisen. Nehmen Sie z. B. an, Sie möchten, dass bestimmte Mitarbeiter sich alle Mandantenentitäten (und alle damit verbundenen Aktivitäten oder Dateien, die mit den Entitäten verbunden sind) anzeigen lassen können, aber nur einen spezifischen Satz von Entitäten bearbeiten können, für die sie spezifische Aufgaben haben. Sie können dies durchführen, indem Sie die betreffenden systemweiten und Entitätsrollen diesen Mitarbeitern zuordnen.

Hinweise:

  • Caseware Cloud erhalten Sie mit einer Reihe von vorgegebenen Rollen. Sie können die Details dieser Rollen ansehen und diese dann Anwendern oder Gruppen zuweisen. Sie können auch benutzerdefinierte Sicherheitsrolllen erstellen, falls dies erforderlich ist.

  • Systemweite Rollen bestimmen die Zugriffsebene, die Sie Mitarbeitern oder Mitarbeitergruppen gewähren möchten. Lesen Sie Vorgegebene Rollen, um sich eine Beschreibung für jede Rolle anzeigen zu lassen.

  • Einige systemweite Rollen können auch als unternehmensweite Rollen in Unternehmen bezeichnet werden, die die Funktion Business Units verwenden. Unternehmensweite Rollen gewähren Benutzern die Rechte der Rolle für alle Business Units, während systemweite Rollen die Rechte der Rolle nur für einen bestimmten Business Unit gewähren. Weitere Informationen finden Sie in Vorgegebene Rollen.

Systemweite Rollen zuweisen:

  1. Stellen Sie sicher, dass Sie über die Rolle Mitarbeiteradministrator bzw. entsprechende Berechtigungen verfügen.

  2. Wählen Sie aus dem Cloud-Menü Gruppen oder Mitarbeiter.

  3. Wählen Sie den Benutzer oder die Gruppe aus, für den/die Sie die Rollen bearbeiten möchten und wählen Sie dann Bearbeiten ().

  4. Wählen Sie die Registerkarte Systemweite Rollen und die geeigneten Rollen.

  5. Klicken Sie auf Speichern.
