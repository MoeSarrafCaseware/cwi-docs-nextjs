Benachrichtigungseinstellungen ändern

Während die meisten Aspekte, wie Benutzer die Anwendung Caseware Cloud sehen und verwenden können, von einem Vorgesetzten oder einem Administrator festgelegt werden, können einige Einstellungen nur vom Benutzer selbst geändert werden. Zum Beispiel kann jeder Benutzer Caseware Cloud so einrichten, dass persönliche E-Mail-Benachrichtigungen versendet werden, wenn andere Benutzer Online-Inhalte ändern. Sie können E-Mail-Benachrichtigungen in Caseware Cloud einrichten, indem Sie Ihre Abonnementeinstellungen ändern.

Abonnementeinstellungen kontrollieren, wann Caseware Cloud Ihnen eine E-Mail-Benachrichtigung sendet. Sie können auf Ihre Abonnementeinstellungen zugreifen, indem Sie auf Ihre Initialen oder Ihr Profilbild rechts oben in Caseware Cloud klicken und anschließend Meine Einstellungen | Globale Benachrichtigungen wählen.

Systemweite Abonnementeinstellungen

Abonnementeinstellungen sind in zwei Typen unterteilt: Systemweite Einstellungen und individuelle Einstellungen. Systemweite Abonnementeinstellungen gelten für alle Entitäten, auf die Sie Zugriff haben.

Angenommen, Sie möchten jederzeit eine E-Mail-Benachrichtigung erhalten, wenn ein Mitarbeiter einen neuen Kontakt für eine Entität erstellt, an der Sie arbeiten. Um diese E-Mails einzurichten, setzen Sie ein Häkchen bei Erstellen | Kontakte.

Alle systemweiten Abonnementeinstellungen sind standardmäßig deaktiviert. Klicken Sie auf Speichern, um Änderungen zu übernehmen.

Mit diesen ausgewählten Einstellungen erhalten Sie von Caseware Cloud eine E-Mail bei jeder ausgeführten Aktion (Bearbeiten, Löschen oder Freigeben) von Aktivitäten, Dateien oder Entitäten, die Sie abonniert haben.

Wenn Sie nur Benachrichtigungen für bestimmte Typen von Aktionen oder Inhalte von Entitäten erhalten möchten, die Sie abonniert haben, können Sie dies im Bereich Inhalte von abonnierten Entitäten konfigurieren.

Angenommen, Sie möchten nur eine Benachrichtigung erhalten, wenn Benutzer Working Papers Dateien in einer abonnierten Entität synchronisieren. Zum Einrichten dieser Benachrichtigungen wählen Sie Inhalte von bevorzugten Entitäten | Dateien und aktivieren Sie Erstellen | Working Papers mit einem Häkchen (im Gegensatz zum Häkchen bei Dateien, das Benachrichtigungen für alle Dateitypen aktiviert).

Klicken Sie auf Speichern, um Änderungen zu übernehmen.

Bitte beachten Sie, dass zusätzlich zu den von Ihnen gewählten Elementen für das Abonnement in Ihren Benachrichtigungseinstellungen automatisch ein Abonnement für Datenbankabfragen, die Sie aus einer Cloud App senden, erfolgt.

Sie können Ihre aktuellen Abonnements prüfen, indem Sie Abonnierte Elemente aus dem Menü Meine Einstellungen wählen.
