Persönliche Einstellungen ändern

  • Ein Profilbild hinzufügen
  • Datum, Uhrzeit und Einstellungen für die Zahlenanzeige ändern

Einige Caseware Cloud Anzeigeeinstellungen können nur auf der Benutzerebene geändert werden. Sie können Ihr Profilbild hinzufügen oder ändern und auch die regionalen Anzeigeeinstellungen zu Datum und Uhrzeit mit Ihren persönlichen Einstellungen überschreiben.

Ein Profilbild hinzufügen

Sie können ein Profilbild für Ihr Cloud-Benutzerkonto hochladen. Dieses Bild ist für alle Benutzer immer dann sichtbar, wenn Ihr Konto mit einer Datei oder Aktivität verknüpft ist, sowie auch in Diskussionen und Chats.

Profilbild hinzufügen:

  1. Wählen Sie oben rechts in Caseware Cloud Ihren Avatar und anschließend Meine Einstellungen.

  2. Wählen Sie unter Andere die Option Mein Bild.

  3. Klicken Sie auf Durchsuchen… und wählen Sie eine Bilddatei, die Sie verwenden möchten.

Sie haben Ihr Profilbild hochgeladen. Es wird nun oben rechts auf der Caseware Cloud Oberfläche angezeigt.

Datum, Uhrzeit und Einstellungen für die Zahlenanzeige ändern

Die regionalen Einstellungen zu Datum und Uhrzeit werden auf der Unternehmensebene festgelegt, Sie können diese Einstellungen aber ggf. ändern.

Ländereinstellungen auf Benutzerebene einrichten:

  1. Öffnen Sie das Benutzermenü, indem Sie Ihren Avatar rechts oben in Caseware Cloud auswählen.

  2. Wählen Sie Einstellungen und unter Andere den Eintrag Ländereinstellungen.

  3. Standardmäßig werden Einstellungen zu Datum, Uhrzeit und Zahlenanzeige an die Standardeinstellungen Ihres Unternehmens angepasst. Um diese zu ändern, deaktivieren Sie die Kontrollkästchen Unternehmensstandards verwenden und wählen die geeigneten Einstellungen.

  4. Klicken Sie auf Speichern.

Sie haben Ihre Anzeigeeinstellungen aktualisiert.
