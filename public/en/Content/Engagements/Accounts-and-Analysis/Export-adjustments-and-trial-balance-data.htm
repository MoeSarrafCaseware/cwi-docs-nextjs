Export adjustments and trial balance data

Auditors can export adjustment entries and trial balance data to a file and provide it to their clients. Clients can then update their books and accounting systems with the changes made during the engagement.

Export adjustments

You can export adjustments to a CSV file or a PDF file. You can also export to a CSV ASCII format, which is compatible with Working Papers.

Note: You can filter your adjustments by type using the All types drop-down filter. Your filter selections will be reflected in your export.

To export adjustments:

  1. Go to the Adjustments page.

  2. Select Export (), then choose the export option you want. Your options are:

    • Export to CSV

    • Export to PDF

    • Export to Working Papers (CSV ASCII)

The data is exported to the selected CSV file type and the file automatically downloads. If you've chosen PDF export, the output PDF file opens.

Export trial balance data (Data page)

You can export the trial balance accounts to a CSV file, PDF or Working Papers compatible file (CSV ASCII).

To export trial balance accounts:

  1. Select Accounts from the left panel at the Data page.

  2. Select Export (), then choose the export option you want. Your options are:

    • Export to CSV

    • Export to Working Papers (CSV ASCII)

    • Export to PDF

      Note: The Export to PDF option generates a PDF that reflects the columns you have selected to display on the Data page. If the engagement is consolidated, the PDF also reflects whether you have chosen to view consolidated data or data for a selected entity in the consolidation hierarchy.

The trial balance accounts data is exported to the selected file type and download starts immediately.
