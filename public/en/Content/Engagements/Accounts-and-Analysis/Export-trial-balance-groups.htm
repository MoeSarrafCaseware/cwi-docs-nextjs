Export trial balance groups

In the Data page, you can export the financial information in the group tabs. For example, to send to clients or to include in the reports.

Available options:

  • Export to CSV - allows you to export data to a CSV file. Trial balance groups that have no accounts are not included in the CSV file.
  • Export to Working Papers (CSV ASCII) - allows you to export data to a CSV file that is compatible with Working Papers. Trial balance groups that have no accounts are not included in the CSV file.
  • Export to PDF - allows you to export data to a PDF. Trial balance groups that have no accounts are not included in the PDF.
  • Export structure to CSV - allows you to export the groupings structure and its properties to a CSV file. It doesn't include accounts and adjustments.

To export trial balance groups:

  1. From the Data page, select a group tab from the left panel.
  2. Select Export (), then choose the desired export option.

    3. File download starts immediately.

    Note: The Export to CSV, Export to Working Papers (CSV ASCII) and Export to PDF options preserve the current selections and filters on the page. For example, if you've selected to view the adjustments column using the Columns drop-down, then adjustments are included in the exported file. If you've enabled the Parent Groups or the Accounts option, the parent groups or accounts are exported.
