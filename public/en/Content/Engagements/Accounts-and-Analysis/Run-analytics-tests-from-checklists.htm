Run analytic tests from checklists and Analytics Hub

Note: This feature may not be available in your product.

Analytic tests allow you to identify and analyze insights related to your engagement data and enhance your ability to meet audit objectives.

You can run an analytic test from any procedure in a checklist from the Documents tab or from the Analytics Hub tab. From the Documents tab, you can select checklist procedures that are available for an analytic test. They are identified by the Analytics icon []. From the Analytics Hub tab, you can see all analytic tests associated with all available checklist procedures.

Run an analytic test from a checklist

To run an analytic test from a checklist:

  1. From the Documents tab, click the Analytics icon [] next to any checklist procedures where an analytic test is available. The Analytics tab will open providing a list of tests available for that checklist item.

  1. Select the box to the test that you wish to run. The results are previewed on the right-side of the screen if any are available. You can rerun the test by clicking the Run button at the top right of the screen.

Run an analytic test from Analytics Hub

To run an analytic test from the Analytics Hub:

  1. Select the Analytics Hub page to display all analytic tests associated with all available checklist procedures.

  2. Select one or multiple tests from the checklist on the left-side of the screen.

  3. Click the Run button on the top right of the screen to run the test(s). The test results will be displayed on the right-side of the screen, defaulting to the Results subtab.

Other test result options

After running one or more tests, you can access the data in multiple ways from the right-side of the screen:

  1. Click the Results button to view the results of the test(s).

  2. Click the Export icon [] to export the results into a CSV file.

  3. Click the Information icon [] to be directed to our documentation site.

  4. Click the Expand button to view the results in a tabular view.

  5. Click the Test parameters subtab to customize the test specification.
