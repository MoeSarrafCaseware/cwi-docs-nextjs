Respond to file requests

As a contact, here's how to respond to file requests assigned to you by firm staff. File requests are indicated by the File Request icon () in the My Work list.

To respond to a file request:

  1. Select Respond to open the file request.

  2. Select the Select Files button or drag and drop the file.

  3. Enter any comments in the text box.

  4. Select Respond to complete the file request.
