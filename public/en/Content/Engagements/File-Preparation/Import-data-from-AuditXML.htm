Import data from XML Auditfile (XAF)

You can import client data from XML Auditfile (version 3.1 or 3.2) accounting software, using the data connector feature in Caseware Cloud. A data connector is a tool to connect to and extract data from a specific data provider. A provider could be third-party accounting software, a particular file format, or even another engagement. This feature works for both cloud and desktop versions of Caseware.

Note: Accounts with zero balances are not automatically imported. If you want to include zero-balance accounts, you must manually add them and assign them to the designated groups.

To import data from XML Auditfile:

  1. Navigate to the Data page. The Data page opens to the Datasets tab.

  1. If you’ve previously imported data, select Add data.

  2. If your engagement is consolidated and has internal subsidiary entities, select which entity you want to import data for from the list.

  1. Under IMPORT FROM DATA CONNECTOR, select XML Auditfile from the list. If your data source is not visible, enter its name into the Search box to find it.

  1. Click the upload window to select a XAF file from your computer to be uploaded or drag and drop your XAF file into the upload window.

  1. If applicable, select the RGS version from the dropdown menu under RGS Mapping. Note that versions 3.1 to 3.5 are supported.

 

  1. Select the dataset type for the XAF file.

  2. Click Next. The next window will allow you to review the dataset type you are importing.

  3. Click Import.
