Translate the financial statements

Based on your product configuration, you can automatically translate text sections in the financial statements to any of the content languages available in your product.

To translate the financial statements:

  1. Open the financial statement and go to the Document Map.

  2. Select Settings () , then select the Translation check box.

  3. The Translation icon appears next to text sections in the document.

  4. Select a translation icon to launch the Translation dialog. From this dialog, select the language to translate the content to, review the translation, then select Save to apply it to the text section.

The translated version of the text section displays in the engagement when you switch to the target content language. You can then make manual edits to the text if required.
