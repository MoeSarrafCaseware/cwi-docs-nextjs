Add custom calculations in text sections

You can add custom calculations to text areas in the financial statements.

To add custom calculations:

  1. Select the text section where you want to add the custom calculation.

  2. The formatting toolbar displays at the top of the text section. Select the Insert Calculation icon ().

  3. In the Calculation dialog, select Insert data (fx).

  4. The Data Link dialog opens. Select the Cell Reference tab.

  5. In the Select area/note drop-down, select from the list of available areas or notes to reference.

  1. In the Select dynamic table field, search for a table in the financial statement area or note, or select a table from the list below.

    1. In the dynamic table preview, select the applicable cell to reference it.

    1. Back in the Calculation dialog, enter the desired calculation. Continue to insert any additional cell references to complete the calculation.

    Select the formatting options for the result of the calculation as necessary:

    • Show decimals—the result is displayed with the number of decimal places as indicated in the Numbers, in Engagement Settings.

    • Show as positive—the result is shown as a positive number.

    • Format as currency—the result is shown as an amount of currency.

    • Show opposite sign—the result is shown as in the opposite sign (negative or positive). For example, if a value is positive, it will be displayed as negative, and vice versa.

    When finished, select Done.

    The custom calculation is added to the text section.
