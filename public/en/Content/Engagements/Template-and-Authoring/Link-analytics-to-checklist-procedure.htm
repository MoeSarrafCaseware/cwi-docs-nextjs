Link analytic tests to a checklist procedure

You can link to analytic tests in a checklist procedure. Analytic tests allow users to identify and analyze risks related to their engagement data and enhance their ability to meet audit objectives.

Once you link tests to a procedure, users can run the test and then view the test results and re-configure the tests before running them again if necessary.

See Run analytic tests from checklists and Analytics Hub for more information on how users can use analytics tests in their engagements.

To link analytic tests to a checklist procedure:

  1. From the Cloud menu (), select Settings () | <your product name> | Manage Template.

  2. Open the firm template.
    See Manage the firm template for more information.

  3. Navigate to the procedure or subprocedure where you want to link analytics.

  4. Select More actions () | Attach analytics.

  5. In the Attach analytics dialog, select the analytic tests to attach to the procedure from the drop-down.

  6. Select Save.
