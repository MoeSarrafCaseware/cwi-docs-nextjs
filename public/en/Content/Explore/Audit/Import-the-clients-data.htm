Import the client's data

To populate the trial balance, you need to import the financial data to your engagement file. For example, you can import data directly from an online accounting software package such as QuickBooks Online and Xero.

Here are the import options that are available from the Data page:

  • Caseware Working Papers

  • Desktop accounting software

  • Engagement file

  • Excel or CSV

  • Accounting software or structured file
