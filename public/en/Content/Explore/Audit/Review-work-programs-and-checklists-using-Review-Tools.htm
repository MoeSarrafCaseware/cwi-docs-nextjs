Review work programs and checklists using Review Tools

Review tools apply visual markup to the document, making potential content issues stand out on the page. Review tools are available in checklists, letters, financial statements and the Risks & controls tab.

To enable review tools, select the Review tools icon () and toggle on Review tools. To learn more, see Work with Review Tools.

Note: Any checklist responses marked as Completed with exceptions are flagged as non-optimal responses in the review tools.
