What is Caseware PBC Requests?

Caseware Provided By Client (PBC) Requests is a solution that's exclusively designed to address staff-client collaboration. It offers a unified interface where you can:

  • Request information and documents from clients

  • Track clients activity on each requested item

  • Simply and effectively provide and receive feedback or comments

  • Perform all the above within Caseware Cloud secure hub

The default content that comes with PBC Requests can be used in client collaboration cases that are part of a Canadian assurance engagement. For example, gather information from clients as part of the annual assurance, review or compliance engagements.

Firms and staff members can customize the default content to meet specific firm or client requirements. To learn more, see Manage the firm template and Review and customize query question settings.

PBC Requests can also directly Integrate with Working Papers to be used as an add-on to a desktop engagement file.

To learn more, see Get started with PBC Requests.
