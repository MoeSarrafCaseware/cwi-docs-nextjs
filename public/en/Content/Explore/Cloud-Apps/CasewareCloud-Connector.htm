What is Caseware Cloud Connector?

Caseware Cloud Connector is a Microsoft Office Add-in that provides real-time data linkage between your engagement and MS Excel. With that linkage in place, you can access and retrieve trial balance data, engagement properties and firm properties, then insert the information in a MS Excel file. If a linked value or information changes in the engagement, it's automatically updated in the excel file.

You can use the Cloud Connector to author financial statements, worksheets and schedules or to verify calculations.

Note that Cloud Connector cannot retrieve the following information:

  • Dimension balances

  • Preliminary balances

  • Adjustments

  • Materiality values

  • Balances from different grouping structures

To learn more, see Get started with Caseware Cloud Connector.
