Get started with Global Statutory Reporting Staff

Before you get started with Global Statutory Reporting Apps, ensure that your firm administrator completes the necessary prerequisites.

To start, sign in to your firm's Cloud site. As a staff member, the actions you can perform include:

  1. Create client entities

  2. Create and set up an engagement file

  3. Complete optimizer checklists

  4. Import and manage data

  5. Assign accounts to groups in the Data page (new design)

  6. Choose content language

  7. Define the engagement settings

  8. Customize the financial statements

  9. Work with review tools

  10. Print the financial statements document

  11. Lock down the engagement

  12. Carry forward an engagement
