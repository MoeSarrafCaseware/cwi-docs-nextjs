Account settings

From My Settings | Other | Account Settings, you can change your existing password and enable two-factor authentication.

You can also change your password prior to signing in via the login page.

Name Description

Confirm new password

 Re-enter the new password.
Country Select your country of residence
Current password Enter the current password used to sign into your Caseware Cloud account.
Mobile phone Enter your phone number.

New password

 Enter the new password you want to use for your Cloud account. This password cannot be one of the last ten used.
Password Quality Requirements Displays the standards for password quality.
Update Phone Number Select to change the phone number associated with your account.
Verification code Enter the verification code sent to your phone number.
