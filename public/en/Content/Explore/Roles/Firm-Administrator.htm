Firm Administrator

Firm administrators are staff members with an IT background that have the Admin security role. To learn more about security roles, see Built-in roles and Assign system-wide security roles.

Firm administrators define, plan and implement technology needs for the accounting firm. In Caseware Cloud, firm administrators are responsible for:

  • Onboarding users

  • Creating client entities

  • Assigning security roles to users

  • Adding custom branding to the firm Cloud site
