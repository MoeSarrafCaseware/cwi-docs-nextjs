Firm Author

Firm authors are staff members with the Settings Admin security role or similar which allows them to customize the default content in the Cloud App their firm is using to meet specific firm needs. To learn more about security roles, see Built-in roles and Assign system-wide security roles.

Firm authors need to create and manage the firm template, where they can make the necessary customizations to the Cloud App they use to cater for specific firm requirements. See Manage the firm template for more information.

Firm customizations include adding content, set up a firm logo and modifying or suppressing the default content.
