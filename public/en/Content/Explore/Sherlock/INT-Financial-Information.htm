Financial Information

This Sherlock International visualization provides details related to Financial Information, including:

  • Trial Balance Insights
  • Financial Statement Insights

Each tab includes controls to display data based on the client name and year end.

Trial Balance Insights

Widget Description

Significant Change Card

Displays the number of accounts with a variance greater than the amount in the Input Significant Change Threshold card, compared to the prior year.

New Accounts Card

Displays the number of new accounts in the engagement for the selected year.

Missing Accounts

Displays the accounts that are not included in the current year trial balance, but were present in prior years.

Analysis of Trial Balance

Displays details of the imported trial balance for the selected year. Accounts that have a variance greater than the amount entered in the Input Significant Change Threshold card are highlighted.

Financial Statement Insights

Widget Description

Trend in Materiality

Displays the Overall and Performance materiality amounts for Preliminary and Final, year over year.

Financial Statement Areas

Displays the Planning, Final and Prior Year balances for each Financial Statement Area, as well as their materiality assessment, for the selected year. Additional filtering is available on the Controls bar.

Trend in Key Financial Statement Areas and Financial Indicators

Displays details of key financial areas, year over year.

Balance Sheet and Income Statement Analysis

Displays financial details based on the engagement’s mapping structure. Additional filtering is available on the Controls bar.
