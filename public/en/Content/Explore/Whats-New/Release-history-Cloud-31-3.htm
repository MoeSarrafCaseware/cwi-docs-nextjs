What's new - Cloud 31.3

Caseware Cloud 31.3 adds APIs to integrate your firm’s external applications with Cloud, ensuring that your client data is always up-to-date.

API integration with external applications

Caseware Cloud now supports integration with external applications, such as CRM systems like Salesforce, MS Dynamics and more. This integration enables firms to easily transfer client data to Cloud, ensuring that it is always accurate and up-to-date. To get started, create an API client for the application in your Cloud site’s firm settings. For more information, see Firm settings - Integration.

API tokens

Firm developers can generate API tokens to authenticate API integrations. This allows integrated applications to automatically retrieve new information without requiring a firm developer to manually trigger an update. To generate an API token, firm administrators must create an API client and provide the client ID and secret to the firm developer. For more information, see Firm settings - Integration.

General enhancements and fixes

In addition to the latest features, Caseware Cloud 31.3 includes various enhancements and bug fixes to improve your overall workflow.
