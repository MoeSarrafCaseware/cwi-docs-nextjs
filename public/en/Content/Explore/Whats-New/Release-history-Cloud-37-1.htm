What's new - Cloud 37.1

Find out what’s new in Caseware Cloud 37.1.

Enhancements

This release includes the following enhancements:

  • Added support for the creation of Cloud firms in various new regions.

Fixes

This release includes the following fixes:

  • After modifying a user's app access between full access and read-only access, the license count now properly updates.

  • Licenses are no longer consumed upon assigning read-only access to staff groups.
