What's new - Cloud 38.0

Find out what’s new in Caseware Cloud 38.0.

'Bundle Guid' view column for Working Papers engagements

We've added a new 'Bundle Guid' column to custom views for Working Papers engagement files. The Bundle Guid is a unique engagement identifier that may be useful for reporting and reference purposes.

In your Cloud site's Firm Settings, select Customization | Views Management and create a new Working Papers view. Add the Bundle Guid column and Save the view.

Performance enhancements

This release includes several enhancements that align with our ongoing commitment to improve Cloud performance.
