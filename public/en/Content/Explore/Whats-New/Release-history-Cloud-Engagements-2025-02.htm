What's new in Caseware Cloud Engagements 2025-02

Welcome to Caseware Cloud Engagements 2025-02! This release introduces new numbered heading options for financial statements, the ability to open and edit PDFs locally and sync back to the Document Manager and more.

To learn more about the Caseware Cloud Engagements platform, see What is Caseware Cloud Engagements?

Numbered heading options in financial statements

You can now choose whether note and area headings are numbered. In addition, the style of numbering can be customized per each individual Area or applied across the entire financial statement. The new option is available from the More actions () menu for each Area.

In addition to being able to hide headings, users can now hide numbers that are applied to headings, including automatic numbering, for an individual heading. This feature is available from the More actions () menu for each heading for the notes and text areas in the financial statements. The number will also be hidden in the Table of Contents should the heading be displayed.

Rounding setup recommendations

If you have rounding set up in Format | Numbers on Product Settings, there are recommendations for settings found in the Features | Financial Statements tab. To learn more, see Rounding setup recommendations.

Fixes in Caseware Cloud Engagements 2025-02

The following list features a brief description of bugs that have been resolved in Caseware Cloud Engagements 2025-02.

  • Changes to the rounding setting in the financial statement require a page refresh to apply changes.

  • A horizontal scrollbar occurs in Role Settings that is not required.
