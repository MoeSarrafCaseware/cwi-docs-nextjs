What's new - Caseware PBC Requests Fall 2020 (v2.218.0)

Here is a listing of what's new for the Fall 2020 release of Caseware PBC Requests.

View and copy author IDs

Note: This feature requires Working Papers 2019 U1 or later.

Firm authors can now view and copy response author IDs to integrate with Working Papers and allow WP users to receive documents from PBC Requests. To learn more, see Copy Author IDs.

Subfolders in the Documents page

For an enhanced document presentation and access, subfolders are now available in the Documents page to further group related items. To learn more, see Manage the engagement documents.
