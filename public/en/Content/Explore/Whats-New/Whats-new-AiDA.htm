What’s new in AiDA 2025-07

Welcome to the Caseware AiDA 2025-07 release! AiDA 2025-07 introduces improvements in response quality and query handling.

Smarter query handling

As part of its prompt refinement process, AiDA now uses a language model to preprocess document-related queries. This enhancement enables AiDA to

  • Clean and simplify user queries to improve handling of a wider variety of syntactic structures, vocabulary usage, and grammar while preserving semantic structure and intent.

  • Decide whether to use the full document or selected relevant sections using RAG (retrieval-augmented generation) based on the query and conversation history.

  • Incorporate conversation history to better understand user intent and retrieve more relevant information.

These improvements help AiDA deliver more accurate, context-aware responses, especially when summarizing documents, answering complex questions, or supporting lengthy conversations.

Fixes in AiDA 2025-07

Improved document summarization for long PDFs

Summarizing long documents, such as lease agreements, sometimes returned incomplete results, omitting key sections. In AiDA 2025-07, summarization now includes the full content of the document, providing more complete and reliable results.

Citations no longer appear clickable when they aren’t

Fixed an issue where some citations appeared clickable but didn’t navigate to the related content in the document.

Document queries now return citations more reliably

Fixed an issue where some document-related queries returned no citations.
