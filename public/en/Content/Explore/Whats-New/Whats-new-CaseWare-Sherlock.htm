What’s new - Caseware Sherlock 7.0

Find out what's new in Caseware Sherlock 7.0.

Historical extraction improvements

We’ve improved the historical extraction process to ensure the integrity of extracted engagement data, including its continuity, performance, and reliability.

Support for SQM

The Caseware SQM app can now ingest engagement data using the new lastModified date range parameter.

The Caseware SQM app can now ingest engagement data using the new lastModified date range parameter.

Date range parameter

You can now use the lastModified date range parameter to retrieve a list of engagements within the specified range. This parameter is supported by the Sherlock API.
