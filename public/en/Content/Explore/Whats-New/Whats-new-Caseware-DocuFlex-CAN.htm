What's new - Caseware DocuFlex Summer 2025 (v 1.3007.0)

Here's a list of what's new in the Caseware DocuFlex Summer 2025 release.

French-language support

We’re excited to announce that Caseware DocuFlex now has French-language support. These items have been translated into French:

  • 1-100 Engagement setup

  • 1-200 Materiality

  • FS-1 Financial statements

  • Financial group names

    • Tax (GIFI) group names

Note: S100 – Balance sheet, S125 – Income statement, S140 – Summary statement, and S141D – Notes detail will be translated in a future release.

Set your Caseware Cloud language before you log in, and then change your product language under Content language in the Engagement Properties popup. See details below.

To change the language settings to French:

  1. Go to your Caseware Cloud portal and set the Language option to Français, then log in.

  1. Open your engagement file.

  2. Select the entity name in the upper left to open the Engagement Properties popup.

  3. From the Langue du contenu dropdown, select français.

  4. Select Enregistrer.

Note: When you carry forward an engagement, the selected language in the current period file is carried over to the next period's file.

Analytics Hub

This release introduces the Analytics Hub. The Analytics Hub is a central location that you can use to view a list of all analytic tests linked to a checklist. When you select a test, you can view configuration details as well as a list of checklist procedures linked to the test. To learn more, refer to Manage analytic tests in the Analytics Hub as well as other topics under Engagement Management | Embedded Analytics.

In your engagement, select the Analytics Hub page to:

  • View all analytic tests available in your engagement

  • Filter tests

  • Make adjustments to test configurations

  • Navigate to linked checklist procedures

In addition to the ability to view results for an individual test, this release also features the introduction of the Aggregated Results page. This page lets you view all transactions identified as part of any analytic test in one location. You can also filter and export results to a CSV file.

This new feature gives you a holistic view of test results and makes it easier to derive insights from your data.
