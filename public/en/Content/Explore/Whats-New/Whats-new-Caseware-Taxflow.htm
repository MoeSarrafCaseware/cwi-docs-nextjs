Tax reconciliation documents

We’ve added a new series of Tax reconciliation – detail tax line and account documents. The document that appears depends on the engagement entity type:

  • 2-300 appears where the engagement entity type is C Corporation.

  • 2-350 appears where the engagement entity type is S Corporation.

  • 2-400 appears where the engagement entity type is Partnership.

This document shows the accounts mapped under each tax group, as well as group level tax (reclassifying and book-to-tax) adjustments.

Cloud Connector support

The 1120 U.S. C-Corporation income tax return, 1120S U.S. Income tax return for an S-Corporation, 1065 U.S. Return of partnership income and 1125-A Cost of goods sold documents now support Cloud Connector allowing you to link information from these documents into Microsoft Excel.

 

Create an engagement based on an existing file

We’ve added the ability to create new engagements based on an existing file. This allows you to take advantage of customizations you have made to a previous engagement so you don’t have to redo this work in a new engagement. See Create a new engagement based on an existing file.
