What's new - Caseware PBC Requests Fall/Winter 2022 (v2.352.0)

Here is a listing of what's new for the Caseware PBC Requests Fall/Winter 2022 release.

Update to simple electronic signatures

The option to select the Simple Electronic Signature response type is now only available for letters. The Simple Electronic Signature question in the Optimiser checklist and corresponding Guidance reflects these changes.

Note: You’ll need to ensure that the use of electronic signatures is acceptable by the applicable tax authority and/or jurisdiction.

For more information on queries and response types, see Create a query document, Send the query to contacts, Respond to a query and Review contact responses.
