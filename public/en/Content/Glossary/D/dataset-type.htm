dataset type

Categorizes the various data schemas making up the Caseware Cloud data model. Each dataset type is structured into a set of columns, each with names and properties for data storage. See the dataset types linked below for more information.

  • General ledger

  • Trial balance

  • Inventory transactions

  • Accounts payable—open balances

  • Accounts payable—payments made described

  • Accounts payable—invoices

  • Accounts payable—suppliers

  • Accounts receivable—cash received

  • Accounts receivable—invoices

  • Accounts receivable—customers

  • Accounts receivable—open balances

  • Inventory on hand

  • Other (synonym of schemaless)— Unstructured transaction data that does not follow a predefined schema. This dataset type can be used to import data that does not correspond to any of the other dataset types. Each data file imported using this dataset type is considered its own dataset.
