Review and Compilation – Canada Quick Start Guide

This document serves as a quick reference for the common steps that engagement team members will perform when working in Review and Compilation – Canada

Set up Caseware Cloud

When your firm first begins using Caseware Cloud, you will need to perform some initial setup steps:

  • create entities for your clients

  • create accounts for the staff members in your firm

  • create accounts for your client contacts

Help:

  • Create entities

  • Add staff and contacts

Start an engagement

To start your Compilation and Review engagement, you will need to create the engagement file and set up the file to prepare staff members to work on it:

  • assign staff members to the engagement

  • set the engagement properties

Help:

  • Set up an engagement file

Perform engagement acceptance

In a new Compilation and Review engagement, there are preliminary checklist documents that you must complete at the start of the engagement. Based on the responses for these preliminary checklists, other checklists are added to the engagement and an engagement letter template is also created.

Once the engagement letter is created, you will edit the text in the letter, then submit it to the client to sign.

  • complete checklist - 0-01 Optimizer

    • if performing tax work, complete 0-02 Tax Optimizer

  • if applicable, complete 1-80 Letter to a predecessor accounting firm

  • complete engagement letter - 1-55 Client requests - Engagement letter

    • if performing a review, complete 1-50 Review engagement letter

    • if performing a compilation, complete 300 Compilation engagement letter and 305 Compilation engagement - Independence checklist

Help:

  • Complete a checklist

  • Add a letter or memo (see the Modifying text and formatting section)

  • Gather data for an engagement (skip to To send a client request to a client)

  • Complete a client request

Plan the engagement

To begin working on the engagement, you will need to add documents or information from the client. Compilation and Review includes PBC queries that you can submit for clients to attach any relevant files or documentation for the engagement.

To prepare the trial balance for your engagement, you will:

  • submit a client inquiries query (to request the trial balance)

    • if this is your firm's initial engagement with client, use 2-01 Client inquiries - Request for records and documents

    • if this is not your firm's initial engagement with client, use 2-930 PBC Document Requests - Engagements Subsequent to Initial Year

  • (For Clients) respond to a Client inquiries query (client should attach trial balance file)

  • import the trial balance

  • assign groupings

  • post adjustments

  • add and resolve review notes

Help:

  • Gather data for an engagement

  • Add an account or group to the trial balance

  • Post adjustments to the trial balance

  • Add review notes (see the Add review notes to an engagement file (SE) section)

  • Complete review notes (see the Complete review notes in an engagement file (SE) section)

Perform engagement work

With the trial balance imported into the engagement, users performing a Review engagement will be able to start the Performing stage. This will include:

  • work on checklists,

  • submit queries for client to complete

  • sign off on documents

  • add annotations

  • add and resolve review notes

  • attach external documents

Help:

  • Work in an engagement file

  • Attach documents to an engagement file

Complete the Review or Compilation engagement

At the conclusion of an engagement, you will create a financial statements document for the client where you explain your findings and present a conclusion. Review and Compilation – Canada has a built-in template that makes it easier for you to prepare the financial statements. While preparing the financial statements, you will:

  • replace placeholder text

  • add and edit notes

  • add cross-references to notes

  • add headers and footers

  • print the financial statements

Set up and complete the Tax engagement

If you're performing a Tax engagement, you'll need to complete the region-appropriate forms using Review and Compilation – Canada's built-in workcharts, and submit your completed return electronically via EFILE. You will:

  • complete the Tax Optimizer

  • complete the Corporation, Engagement, and Preparer Profiles

  • import the GIFI data from a file, via a T2 Auto-fill, or from the engagement's Trial Balance

  • complete the appropriate provincial tax forms

  • complete the federal tax forms

  • complete the EFILE information and submit the return
