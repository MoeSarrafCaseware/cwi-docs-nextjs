Perform a tax engagement

Canadian CRT enables you to perform the required tasks of a tax engagement along with the Compilation and Review functionality.

  • complete the Tax Optimiser checklist

  • upload your client's tax data

  • complete the Tax Preparer profile

  • complete the provincial and federal forms using the Tax worksheets

  • run a diagnostic check to ensure there are no outstanding errors

  • file the return electronically via EFILE or create a PDF for your client if required

Set up the tax engagement

The first step is to define the type of the engagement as a Tax engagement.

Practice set

  1. In the 0-10 Planning Optimiser, select Corporation Tax only beside What type of engagement is being performed?

  2. Select the following link to open the Tax Optimiser.

    The Tax Optimiser Checklist displays.

Complete the document with the details of the entity you're performing the tax engagement for, as well as the characteristics of the engagement. You can access different sections by selecting the corresponding tab above the document.

The engagement will automatically populate with the appropriate provincial and federal tax forms, as well as Caseware ReviewComp's worksheets. These documents will appear in the Document Map on the left panel. You can choose to complete the new documents as they appear with each answered procedure, or finish the checklist and complete the added documents later.

Import tax data

Caseware ReviewComp gives you a few different ways to import the engagement's tax data.

Select Import to view the import options.

Practice set 1- GIFI import

Select this option to browse your computer for a GIFI output file from your desktop accounting package.

Note: This option should only be used if you are not going to use the Trial Balance as any change in the Trial Balance will first clear the GIFI forms and transfer the Trial Balance data into the GIFI forms.

Practice set 2 - TaxPrep CSV Import

Select this option to upload a .csv output file generated that was generated by TaxPrep.

Notes:

  • Do a roll forward in TaxPrep before doing the CSV file Export within TaxPrep.
  • TaxPrep CSV Import will clear out all the data transferred from the Trial Balance into the GIFI and replace it with the data from the CSV Import. Therefore, you must import this CSV file before you perform any imports into the Trial Balance.

To export a .csv file from TaxPrep:

  1. Select | TaxPrep CSV.

  2. Navigate to the desired file and select Open.

Practice set 3 - T2 Auto-fill

If you've registered for the CRA's T2 Auto-fill program, you can import the engagement's tax data automatically based on the client's previously filed returns.

Practice set 4 - Download CloudBridge

CloudBridge is an add-on for Caseware Working Papers that lets you link a Working Papers engagement to a Cloud entity to import your engagement's tax data into your Cloud apps. Use this option if you have a Working Papers engagement that contains GIFI data.

To download CloudBridge:

  1. From Data | Import from, select Caseware Working Papers.

  2. Follow the instructions on the help page that appears to import your engagement data from Working Papers.

Complete the Tax Preparer profile document

The T-02 TPP - Tax preparer profile document requires you to enter key information about the entity performing the tax engagement.

Once this document is completed and signed off , you're ready to begin your tax engagement work.

Complete the tax documents

With the tax data imported and the profile documents completed, you're ready to begin completing the engagement's federal and provincial tax forms. Because you completed the Tax Optimiser, only the forms that are relevant to your tax engagement are shown in the Documents tab.

To see the complete list of available tax forms, select the Edit icon () on the Documents page, or open the Tax Forms List document. Use your browser's search function (Ctrl-F) to help locate specific documents.

Your selections also determine which worksheets, if any, are listed. Worksheets are documents designed to streamline your engagement work.

Practice set - Navigate between related tax documents

You can easily navigate between workcharts and their linked schedule documents. There are two options for moving between these documents.

When you select a field that has a single source, you will see this dialog:

When you select Jump To Source, the source document opens and jumps to the appropriate line, with the exact source field highlighted in flashing blue.

Example

In this example, your client has purchased a new fixed asset, a building.

Use Worksheet - S8 CCA to manage fixed assets.

Select the appropriate class for the firm's new fixed asset. Select the Add icon () in the CCA Class field.

Select the appropriate CCA class, then select OK.

Scroll to the bottom of the worksheet, then select Add (). On the new line, select the Edit icon (). A dialog opens, where you can enter the new asset's information.

Once tax documents are filled out, you can use the Enclosure letter under the TAX - GENERAL AND FEDERAL | Client communication folder as guidance for communicating key findings with your client.

Search tax forms and worksheets

When working on a tax form or worksheet, you can search, edit visibility and open all of the tax forms and worksheets in the engagement.

Practice Set

  1. Select Search from any open tax form or worksheet.

  2. Enter the name of the document you want to find.

    3. Alternatively, you can select Display all tax forms to view all tax forms and worksheets in the engagement.

  3. Select the switch beside a form to enable or disable it.

    4. A green switch displays next to the visible forms in the list.

    A gray switch displays next to the hidden forms in the list.

  4. Select a document to open it.

  5. Select Close when you're finished.

Correct outstanding errors or issues

If you have any outstanding EFILE diagnostics flagged, you will not be able to complete the EFILE until they are cleared.

Practice set

Select the Diagnostics page link in the warning shown above to see the engagement's outstanding issues. Click on the text beside an issue to display a link to the appropriate document.

Select the link to display the line in the document that requires attention.

Select | Efile to display the issues preventing an electronic filing.

E-file the return to the CRA

With the tax data imported, the engagement work complete, and all outstanding issues fixed, you can begin to plan your electronic filing via the CRA's EFILE program.

Practice set

Select EFILE from the top menu.

From the EI - Efile Information document, you can file your return to the CRA via one of two sets of credentials - if your firm is a Certified Tax Preparer, you can enter your CRA Login Information.

Alternatively, you can enter a WACode (Web Access Code) provided by the client. Select WACode from the drop-down beside Choose authentication mode.

If you require a Web Access Code and have not yet been provided with one, select Web Access Code (WAC) - Online Request and complete the request form.

Print the return

If your client requests a copy of the return for their records, you can create a PDF file which can then be downloaded and shared with the client electronically.

Practice set

Select Print from any open tax document to select the document you want to print.
