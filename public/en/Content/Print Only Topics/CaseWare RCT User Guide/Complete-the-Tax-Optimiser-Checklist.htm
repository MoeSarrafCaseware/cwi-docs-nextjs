Return to the Documents page and select the T-01 TOC - Tax optimiser checklist document. As with the first Optimiser checklist, your responses will determine the documents that will make up your Tax engagement. Some responses will be automatically pulled from your firm's Cloud Profile. Customize the engagement letter

ReviewComp contains a standard engagement letter for outlining the engagement parameters with your client. To customize the letter's contents, select Edit () and either edit the pre-existing content, or just paste in your firm's preferred content. When you're done, select the appropriate Sign Off status. You can print the letter to PDF or hard copy by selecting Print (). To learn more, see Edit the engagement letter.
