Import engagement data from Caseware Working Papers to Caseware ReviewComp

You can import your client's financial data from Working Papers and have it transferred to your Caseware ReviewComp engagement. The steps to carry out this task differ based on whether or not you have disabled your Working Papers license.

Note: Accounts with zero balances are not automatically imported. If you want to include zero-balance accounts, you must manually add them and assign them to the designated groups.

If you have a valid Working Papers license,

  1. Install CloudBridge.

  2. Add the CloudBridge (CLBG) document to your Working Papers engagement file.

  3. Transfer the data from Working Papers to your Caseware ReviewComp engagement.

If you have disabled your Working Papers license,

  1. Install CloudBridge.

  2. Open Working Papers and select Continue Without Registering.

  3. Make sure Working Papers is integrated with your Cloud firm. To learn more, see Working Papers - Cloud Integration.

  4. Carry forward the Working Papers engagement file.

    5. Note: If the engagement file is already carried forward, ignore this step and proceed to Step 5.

  5. In the Browse window, copy the below path and press Enter to open the CloudBridge (CLBG) document.

    6. cw:program/scripts/cloudbridge/index.html

  6. Transfer the data from Working Papers to your Caseware ReviewComp engagement.
