Import the client's data

To populate the trial balance, you need to import the financial data to your engagement file. For example, you can import data directly from an online accounting software package such as: QuickBooks online.

Here are the import options that are available at the Data page.

  • Import from an online accounting software

  • Import from a desktop accounting software using the Cloud Import Utility

  • Import trial balance from CSV or Excel file

  • Import transactions from CSV or Excel file

    Note: You must import the client's trial balance data before you import transactions.

  • Import from Caseware Working Papers to Caseware ReviewComp

  • Import from QuickBooks desktop to Caseware ReviewComp
