Transfer data from Working Papers to Caseware ReviewComp

After you add CLBG document to a Working Papers engagement, you can transfer your trial balance and general ledger tax data to Caseware ReviewComp.

To export your engagement data to a Cloud engagement using CloudBridge:

  1. In the Document Manager, double-click CLBG. The CLBG document displays.

  2. In the Step 1 section, choose an appropriate entity and engagement.

  3. In the Step 2 section, choose the appropriate trial balance data to export.

  4. In the Step 2 section, select the Use remap file check box.

    The remap file is used to transfer the current account mapping structure in Working Papers to your Cloud engagement. If your engagement uses the default Working Papers mapping or was created with the PEG template, use the remap file located in the folder where you installed CloudBridge (C:\Program Files (x86)\Caseware\Scripts\cloudbridge).

    Note: If your engagement uses a custom mapping structure specific to your firm, create a custom remap file then add it.

  5. Select Choose file to browse and add the remap file.

  6. (Optional) If you have other journal entries related to donations, meals and entertainment expenses or foreign advertising, the Step 3 section is available in the page. Step 3 section allows you to transfer GL data to the appropriate Tax Worksheet in the Caseware ReviewComp engagement.

    Note: Step 3 section is only available on the page if your engagement file contains other entries.

    In this example, we'll add gifts to client to the Donations worksheet.

  7. Once you're done, select Transfer Data.

Create a custom remap file (.ini)

If your firm uses a custom mapping structure, you need to create your own remap file (.ini) file based on Caseware ReviewComp mapping structure.

You can now use this remap file in Cloudbridge to transfer your Working Papers data to Caseware ReviewComp.
