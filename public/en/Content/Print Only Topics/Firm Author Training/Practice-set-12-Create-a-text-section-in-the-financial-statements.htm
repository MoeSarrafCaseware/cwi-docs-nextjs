Create a text section in the financial statements

There are cases where the accountant may want to add supplementary information to an engagement for a specific industry or client where it is not required by the standards but adds value to the overall financial reporting. In OnPoint, this can be done through several features which when combined can help us create supplementary information. For the first part, we need to create a text section.

Practice set

  1. Ensure the Hidden Items option is enabled.

  2. Open document 3-150 Financial Statements and Report, then go to the Document Map.

  3. Select Add () | New Area.

    The new area is appended to the document map and displays on the page with a default title New Area. You can select the default title to change it.

  4. In the new area, select Add () | Text Section.

  5. Select the text field and center its alignment ().

  6. Select Insert Text Placeholder.

    The Text Placeholder dialog displays.

  7. Enter Insert Client Operating Name at Label and select Save.

  8. On the next line, type Supplementary Information.

  9. On the next line, type Consolidated Schedules of General and Admin Expense.

  10. On the next line, type For the Years Ended.

  11. On the same line as Step 8, select Insert Formula (fx) and choose Current year from the Data Link dialog. Type and, select Insert Formula (fx) and choose Prior year.
