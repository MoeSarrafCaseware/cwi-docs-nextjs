Response documents

Once you accept a response, the document is added to the designated predefined folders in the Documents page.

Other options

From the More actions () menu for each response, you can perform the following actions:

  • Select Move to move the response document to another folder.

  • Select Edit to edit the Identifier, Name, and visibility settings of the document.

  • Select Delete to delete the response.

  • Select Save to Cloud to save the document to the Files section in your Cloud site.

    Open response documents inside a Working Papers file

    If your Cloud site is integrated with Working Papers, you can open the response document inside Working Papers.

    To integrate Working Papers with Cloud:

    1. Open Working Papers. If you have any files open, ensure you close your current Working Papers file (File | Close).

    2. On the ribbon, click File | Options | Caseware Cloud.

    3. Select Integrate with Caseware Cloud. In the Server field, enter the URL of your organization's Caseware Cloud site. You do not need to enter the web protocol (http:// or https://). Click OK.

    4. A Caseware Cloud login prompt displays. Enter your email address and password, then click Sign In.

    After integrating your Working Papers file with a Cloud entity, you can link it to a Queries file to start collaborating with your clients.

    To link to a Queries file:

    1. In Working Papers, open a file that is synchronized with a client entity in Cloud.

    2. On the ribbon, click Cloud | Open. The Queries Setup dialog displays.

    3. If applicable, select the Queries template that you want to use.

    4. Select Select Existing and click the drop-down menu to select the Queries file. Click OK.

    The file is linked to the Queries file and the Queries pane displays. You can now drag and drop your response documents to the Working Papers Document Manager.

    To add a document to the Document Manager:

    1. Ensure you have both the Document Manager and the Queries pane open.

    2. Drag the document you want to add from the Queries pane into the Document Manager.
