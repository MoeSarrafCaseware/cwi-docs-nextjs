Carry forward an engagement

Carry forward is a time-saving utility. You can carry forward an engagement to re-use information, as you can accept all of last year's changes or only work on the individual items that have changed.

When it comes to interim reporting, each interim period must have a separate file. However, the engagement file is based on the previous period.

For annual engagements, the prior year adjusted trial balances carry forward to the Prior Year column in documents and financial statements.

You have the option to accept or reject changes for the previous period in the new file. You can also view procedures that have been added or modified by the firm author between versions. For example, if you begin working on a quarter one file and there is a content change by the firm author, when you create your quarter two file, you can view the new changes made by firm authors.

Now let's carry the file forward into the next period.

Practice set

  1. Open the new engagement (created from carry forward).

  2. Select the 1-110 Engagement Setup document from the Documents page.

    You will now see options to accept or reject carried forward responses.

  3. Accept all the responses. Once complete, return to the Documents page

  4. Select the 1-125 Review Engagement Quality Acceptance and Conclusion document.

    You will be able to accept or reject carried forward responses again.
