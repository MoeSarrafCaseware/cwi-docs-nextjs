Completing the engagement

You have completed all of the fieldwork for your engagement:

  • completed leadsheet documents in the Fieldwork section

  • used a PBC (Prepared by Client) query to gather additional information from the client

With all of your fieldwork complete, you are ready to start preparing your conclusions.

Financial statements

At the conclusion of the engagement, you will create a financial statements document for the client where you explain your findings and present a conclusion. OnPoint PCR has a built-in template that will make it easier for you to prepare the financial statements.

On the Documents page, under the Financial Statements and Wrap-up section, open document 3-150 Financial Statements and Report.

Near the top of the report is a detailed guidance section. On that page, there is a table that outlines each section of the financial statements and indicates:

  • when it is included in the report

  • what elements are required in the section

  • what the AR-C references for the section are

In the guidance section, below the link, you will find additional information about completing some of the required sections of the financial statements.

Each financial statements document includes the following sections:

  • Title page

  • Table of Contents

  • Report section

  • Financial statement tables

    • Statement of Income

    • Statement of Comprehensive Income

    • Statement of Retained Earnings

    • Statement of Cash Flows

  • Notes to the Financial Statements

You may wish to review the linked document before proceeding.

Customize the financial statements

Much of the content in the financial statements document is already populated for you. The content in the review report is based on your answers to the questions in the Client Acceptance and Continuance document. Groups and accounts on the financial statements tables are from the trial balance.

However, you will need to make other changes to customize the financial statements for your client.

Replace the placeholder text

As you saw in the engagement letter, the review report paragraphs include placeholder text that you are meant to replace. Placeholder text locations may be displayed in red font and enclosed in brackets. They could also be displayed with pull-down menus where you can select the correct name, date, or information for this engagement.

Select the Edit icon (), and then select the text area you wish to edit. Replace any placeholder text with real content appropriate for your client and the engagement. Also select values for any placeholder pull-down menus. Once you have made all changes that you need to, select the Edit icon () again to save your changes.

Edit a financial statement table

While you cannot change the specific accounts and groups that are included in a financial statement table, you can make some changes to the content that is displayed.

Enter information into a financial statement table

Many of the tables in the financial statements are populated automatically. The information in them is pulled straight from the trial balance, to make financial statement preparation easier for you. However, you will also likely need to enter some additional information yourself.

Some of the lines in the financial statements are input rows. In an input row, you can enter amounts into the fields, and these amounts will be included in final calculations for the table.

For example, scroll to the Statement of Cash Flows, then select the Edit icon () at the top of the page.

Note: For the Statement of Cash Flows to be visible, you must have selected Indirect method in response to Which method, if any, is being used for the Statement of Cash Flows? in document 1-110 Engagement Acceptance and Continuance.

Normally, rows with no values are hidden from the standard view of a table, but in edit mode you can see all of the input rows. Once you enter any financial data into an input row, that row will be displayed in the statement of cash flows.

Note: If you want to enter a different amount in a row that already has automatic values, select the Settings icon () at the end of the row and then select Override calculated balances.

Expand a line in the financial statements

Many of the financial statement tables collapse accounts into their groupings by default. You can expand lines in the financial statement to give more details about some areas of the engagement file.

For example, go to the Balance Sheet in the financial statements.

Assets, Noncurrent displays totals for two areas: Property, Plant and Equipment, net and Long-term Investments, net. Now, you decide to give the client more information about the Property, Plant and Equipment, net group.

Select the Expand/hide lines icon () beside the group name.

All of the lines under your selected group are now displayed, and you can see the amounts for each different category under Property, Plant and Equipment, along with the amount for Accumulated Depreciation, Depletion and Amortization.

All of the lines under your selected group are now displayed, and you can see the amounts for Automobiles - cost, Property, Plant and Equipment and Accumulated Depreciation, Depletion and Amortization. Select the Expand/hide lines icon () beside the Property, Plant and Equipment subgroup to display all of the detail lines that make up the group.

To collapse a subgroup, select Expand/hide lines icon () again.

Note: If you want to see more details about a balance on the financial statement without changing a table, you can select the line with that balance to drill down and display more details.

Modify the description of a financial statement line

You can change the descriptions for lines in the financial statements. Some of the lines in the financial statements are input lines. You can select an input line and enter a new description. You can tell input lines from other lines by the different background color.

To change the descriptions of any lines that are not input rows, you will need to change the document settings. Select the Edit icon () , then select the Settings icon () and enable Allow override of line names.

Add a note to the financial statements

The Notes to the Financial Statements section is at the bottom of the financial statements document. If you need to provide any supplementary information about the results of your engagement, you can add that information here.

Each note can include a guidance section with information on any changes that you might need to make to it as the preparer. They also typically have one or more text areas that contain explanations relevant for the note.

Some of the notes in your engagement may also include a dynamic table. Dynamic tables contain specific lines from the trial balance or custom rows where you can input your own data. These tables are intended to highlight specific parts of the trial balance or to support your conclusions.

Normally, the firm author at your firm will add notes for the financial statements to the document template. When the author publishes that template, all new OnPoint engagement files will include the same notes in the financial statements document.

In this demo file, there are no notes added to the template yet, so we will create a sample note. This note will be titled "PROPERTY AND EQUIPMENT", and it will display a dynamic table with the current and prior year balances for Machinery and Equipment, Furniture and Fixtures and Leasehold Improvements.

To add a note to the financial statements:

  1. Scroll down to the Notes to the Financial Statements section, then select the Edit icon ().

  2. Scroll to the bottom of the document, then select (BLANK).

  3. Enter PROPERTY AND EQUIPMENT as the new title for your note.

New notes are empty by default, so you will need to add sections to the note to display your information. Next, you will add a text area to your note to introduce the dynamic table:

  1. Under the note, select + New section, then choose + Text Area.

  2. Select the gray text area, then enter the following:

    Property and equipment at June 30, 2017 consisted of:

You have added a description for the contents of the new note.

To add the balance information for the property and equipment account groups, you will need to add a dynamic table.

  1. Under the text area, select + New section, then choose + Dynamic Table.

  2. In the new Dynamic Table section, select the Settings icon () in the section heading, and choose Edit Layout.

  3. On the left side of the table, select the Add icon () to add a new row, then select Trial Balance Data.

  4. Enter the word property to filter the list. Select the Property, Plant and Equipment group (with the 45,000 balance) to add it to the row.

  5. At the end of the line, select the Expand lines icon ().

  6. At the top of the page, select Save & Close.

You have added a dynamic table to your note. When you scroll to the bottom of the financial statements document, you will see the note with all its new content.

Add a cross reference to a line

If a line in the financial statement table requires more explanation, you can add a cross reference to the line that links to one of the notes to the financial statements.

For example, you decide that you want to provide a breakdown of the Property, Plant and Equipment group, but you do not want to display it on the balance sheet. You can link a line in the balance sheet to the PROPERTY AND EQUIPMENT note that you created earlier.

To show the Notes column:

  1. Select the Edit icon (), then scroll to the Balance Sheet section of the financial statements and select the Settings icon () beside Dynamic Table - Balance Sheet.

  2. Select Edit Layout and the Add icon ()above the empty column.

  3. Select Note reference from the Type drop-down, then select SAVE & CLOSE.

At the top of the balance sheet, select the Edit icon ().

A Note column now appears in the Balance Sheet.

Select the Edit icon () again. You will see that each group name now has a # link in the Note column.

A pop-up is displayed, and you can see the name of the note to the financial statements that you created. Select the checkbox beside 1. PROPERTY AND EQUIPMENT.

Select the Edit icon () again to save your changes. The Property, Plant and Equipment group now includes a reference to the note in the Note column.

Print the financial statements

Once you have finished preparing the client's financial statements and your reviewer has signed off on the document, you will want to print a copy of the financial statements to present to the client. You might also print other copies of the document while you are still in the process of working on it in order to review the impact your changes will have on the finished product.

You cannot print directly from OnPoint PCR. Instead, printing creates a PDF copy of the file for your review.

Select More actions () | Print Settings to display the Print Settings dialog. You can enter content for the header and footer of your document, and you can also set the page size and orientation, the margin width and the table of contents settings. If you make any changes, select Save to close the dialog.

Select the Print icon () to generate a PDF copy of the Financial Statements. This automatically opens a new tab in your browser window, and the PDF version of the trial balance displays. You can choose to save the file as a PDF to your local hard drive, or you can select the Change button and find a local printer to print a physical copy.
