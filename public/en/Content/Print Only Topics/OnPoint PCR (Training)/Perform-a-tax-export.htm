Perform a tax export

The AICPA have enabled a more comprehensive workflow to ensure that users are able to seamlessly transition their data from Caseware Cloud OnPoint to their choice of tax software.

Practice set

To create a tax export file, open the 4-100 Tax Export document.

Select the View drop-down list to choose the display option for the document:

  • Show IDs - View the line items from the tax software where the information from Caseware will be transferred to.

  • Dynamic Form Width - Have the form automatically adjusts width based on the browser page width.

The left panel in the form displays the Entity Type, Tax Vendor, and Tax Year. Note that more options will be available in future releases.

The right side panel displays multiple tax forms. It is recommended that you select the applicable Tax Form and open it before starting the export so that you:

  • confirm the numbers were correctly transferred from the product into the forms.

  • reduce the export time (since the information has been populated and is already available in the forms).

Now, let's launch the 1120 U.S. Income Tax Return.

You can assign one or more accounts to a single tax code. Select the plus sign and choose an account to assign to a single tax code. To assign more than one account to a tax code, add a place a comma then select another account.

You'll find that the application adds the account balances for each tax code. The corresponding cell in the totals column is updated with the total amount.

Once you are satisfied with the data presented in the applicable forms, go back to document 4-100 to start the export.

Select Export, then specify where the exported file will be saved. In some browsers, the file is automatically saved in the Downloads folder.

If you need help to import the tax file into your tax software, please contact the respective tax vendor support center.
