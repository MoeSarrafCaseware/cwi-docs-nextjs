Process mining home screen

The process flow diagram at the top of the home screen shows how the data flows through the various processes in the PPP (Purchases, Payables, Payments) cycle. Each block in the process has a specific document type associated with it. For example, PR type data entries are associated with the Purchase Requisition block.

Note: If your data import failed, an error message displays and the home screen will be blank. If this happens, reimport your data.

Click on different process variants in the table to highlight them in the process flow diagram.

You can also click on users' initial icons to show entry types they are associated with.

Process variants summary report

The Process Variants Summary report contains information about the process, date and time of report generation, user who generated report as well as complete list of all the process variants in a process and list of users involved in the process.

To download the Process Variants Summary report:

  1. Click the Download Process Variants button ().
  2. Select Download, to download a PDF file.
    3. Or
  3. Select Save to Engagement, to save the report to your engagement file.

Note: You can also generate this report from the Variants dialog.
