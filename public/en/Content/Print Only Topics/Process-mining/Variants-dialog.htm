Variants dialog

Use the Variants dialog to view all of the process variants in your engagement data. This dialog lists all the process variants sorted in descending order of number of transactions.

To open the Variants dialog:

  • Select the View All button from the Process Mining home screen. This button only appears if there are more than three process variants in the engagement.

Use the Filters at the top of the dialog to select the contents of the dialog by Process Component and User.
