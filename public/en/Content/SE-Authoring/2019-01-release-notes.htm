SE 2019-01 release notes

All features below are automatically enabled if the corresponding component or module has been enabled.

Analysis charts

Decimals are now supported in Analysis charts. The decimal settings in the ratio table are applied in the chart.

Authoring

Primary authors can now:

  • Export the visibility settings to a CSV file to better verify and keep track of the existing settings in a template. For more information see, Visibility Export - Dist. Version

  • Add references to other documents in the Guidance area. For example, include a reference to a specific query document in the letter's guidance.

  • Add a Guidance area in Analysis documents. Note that engagement users can only view the Guidance area.

Carry forward

We've added a badge counter to the Carry Forward icon. The badge counter shows the number of items that can be carried forward.

Dynamic tables

  • Column headers now repeat on subsequent pages when you manually insert page breaks in the dynamic table.

  • Dynamic content is available in column headers. You can now include dynamic text and formulas in column headers.

  • We've improved the way symbols are displayed and added in column headers. Symbols now appear on a separate line under the header text.

  • You can now undo and redo actions when editing a dynamic table.

Financials new design

Carry forward

  • Financial statements can now be carried forward. This includes new, modified and hidden areas and content.

  • You can now view the number of items that you can carry forward in each area. An indicator of the carry forward items per area displays in the Document Map.

Consolidation

You now have the option to select whether or not to use non-consolidated data in your financial statement.

Notes Library

  • Preview option is available in the Notes Library. You can now preview a note before adding it in your financial statement.

  • You can now filter notes by Trial Balance group.

Formulas

Additional address fields are available in formulas for more flexibility. You can now arrange the address information to comply with your regional format specifications.

Letters, memos and financial statements

  • Letters and financial statements now include an indicator of the number of unset placeholders in the document.

  • Document sign-off now validates placeholder fields and text. If there are unset placeholders in the document, you get a warning message before you can proceed with the sign-off.

Translation

Spanish language is now supported in the formatting bar of a text area.

Queries

  • The name of the responder and the timestamp of the response are now displayed for all response types.

  • Sorting is now available in the Query Dashboard. You can sort queries in ascending or descending order based on status, due date, progress and name.

  • You now get a confirmation message once you send a query.

Fixes

  • We've added tooltips for various icons in the application and included translation support.

  • Firm authors can no longer suppress content that impacts the availability of other content in the template.

  • Firm authors can no longer edit financial statement content that's been added by the primary author.
