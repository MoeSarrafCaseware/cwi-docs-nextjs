New in SE 2024-12

Welcome to SE 2024-12! This release features the ability to add a lockdown history to the downloaded reference copy of an engagement, and more.

To view the most recent fixes in this release, see the following topic:

  • Fixes in SE 2024-12

Lockdown history included as a PDF

#primaryauthor#firmauthor

Users can now download the lockdown history as a PDF within the reference copy. To enable this feature, add the following code to the archive array of the product.json file for your product:

{ "title": "Lockdown", "uri": "index.jsp#/lockdown" }

The reference copy is available from the Information icon () at the top menu.

Cloud Engagements integration with Sherlock

#primaryauthor#firmauthor

For the latest integration updates with Caseware Sherlock, see What's new - Caseware-Sherlock.

Java upgrades to the Cloud Engagements server and microservices

The Caseware Cloud Engagement server is now upgraded from Java 11 to Java 17. In addition, its microservices are now upgraded from Java 17 to Java 21.
