New in SE 2025-02

Welcome to SE 2025-02! This release introduces new numbered heading options for financial statements, the ability to open and edit PDFs locally and sync back to the Document Manager and the ability to enable/disable the Sync trial balance function from the data import workflow.

To view the most recent fixes in this release, see the following:

  • Fixes in SE 2025-02

Numbered heading options in financial statements

#primaryauthor#firmauthor#enduser

Users can now choose whether note and area headings are numbered. In addition, the style of numbering can be customized per each individual Area or applied across the entire financial statement. The new option is available from the More actions () menu for each Area.

In addition to being able to hide headings, users can now hide numbers that are applied to headings, including automatic numbering, for an individual heading. This feature is available from the More actions () menu for each heading for the notes and text areas in the financial statements. The number will also be hidden in the Table of Contents should the heading be displayed.

Open / edit PDFs locally from Document Manager

#enduser

Note: This feature requires a one-time protocol installation and may not be available in your product.

Users can now open PDF files that have been added to the engagement in using a standard PDF viewer or editor app installed on your machine. This includes the ability to comment on and highlight PDFs, enabling better collaboration on your engagements. Once users save the file, the edits they made will be synced to the file in the engagement. To learn more, refer to Open and edit PDFs locally.

Enable/disable Sync trial balance from the engagement file import flow

#primaryauthor

Note: To include this feature, add the following to the features array in the product.json:

"feature": {

"syncTrialBalance": false,

For more information, see List of product features / Trial balance.

Primary authors can now enable or disable the Sync trial balance toggle from the engagement file import workflow.

The Sync trial balance function is, by default, enabled on all Caseware products that support engagement file imports. Since there are some use cases where this function should not be used, this enhancement allows the primary author to hide the function from the end user in the Engagement File import dialog.

Rounding setup recommendations

If you have rounding set up in Format | Numbers on Product Settings, there are recommendations for settings found in the Features | Financial Statements tab. To learn more, see Rounding setup recommendations.
