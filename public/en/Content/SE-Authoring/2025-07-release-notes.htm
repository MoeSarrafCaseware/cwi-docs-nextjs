New in SE 2025-07

Welcome to 2025-07! This release introduces improved PDF editing capabilities, the ability to copy individual risks across engagements and more.

On this page, you can view all the new features included in this release that apply to both authors and end users.

  • Authoring features

  • End-user features

Authoring features

Multilingual capabilities for Procedure Settings

#primaryauthor #firmauthor #enterpriseauthor

Note: This feature is enabled on all Caseware apps. It is only available for new engagements.

As a firm, enterprise or primary author, you now have multilingual capabilities in the editable fields of Procedure Settings within a checklist. This enhancement allows for greater flexibility and localization across global engagements.

The following fields now support multilingual input:

  • Note placeholder title

  • Response placeholder text

  • ‘Other’ field in response options

This update ensures that key elements of the checklist interface can be tailored to the language needs of your users.

Subtotal across multiple groups in a trial balance

#primaryauthor

Note: This feature is available for all primary authors and is enabled by default. It is only available for new engagements.

Authors can now select multiple financial groups to subtotal in the Advanced section of the Group Properties dialog, under the trial balance. The single-selector has been replaced by a multi-selection picklist allowing for subtotal balances across multiple groups at once rather than just one at a time.

This feature enables users to view and compare group subtotals more easily. Group selections are now saved together in one step, reducing errors and improving change tracking. To learn more, refer to Subtotal across multiple groups in a trial balance.

Support for embedded forms in Group Audit

#primaryauthor

Note: This feature must be enabled through source code (formData.js) and is available only to primary authors.

As a primary author, you can allow group auditors to include embedded forms within a group audit. This includes choosing which forms and values to use and preserving them to be carried forward across the audit.

As a result, group auditors can have more options on what to share with component auditors.

To learn more about embedded forms, refer to: Embed SE Builder forms in SE documents.

End-user features

Open and edit PDFs locally - now supported across Caseware

#enduser

Note: This feature requires a one-time protocol installation and is only available for Windows users.

The feature to open and edit PDFs locally, previously limited to select users, is now available across all Caseware applications.

As part of this rollout, this feature includes a new in-house protocol, Caseware WebDAV protocol, which improves installation and feature functionality.

To learn more about this feature, refer to: Attach documents to an engagement file.

Copy individual risks and risk assessments across engagements

#enduser

Note: This feature must be enabled in your Product settings. It is only available for new engagements.

You can now copy individual risks, including risk assessments, over to other engagements using the Copy from Other Engagement feature. This builds on a previous enhancement to perform a risk assessment on an individual risk. With this update, when you copy a risk from one engagement to another, any associated individual risk assessments are automatically copied as well. These assessments retain their structure and context within the destination engagement and are synced to changes made to the source engagement.
