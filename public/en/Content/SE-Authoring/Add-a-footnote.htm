Add a footnote

When applying tags, some regions require end users to provide additional text to further describe certain tagged item values. This concept is referred to as footnote tagging.

To add a footnote to a tagged value:

  1. Select the notepad icon () on the right-hand side of the value field in the XBRL Review panel to launch the Footnote Editor.

  2. Enter the required text, then select Submit.

  3. The notepad icon will turn green to indicate that a footnote tag has been successfully attached to the tag's value. Select Apply tag in the XBRL Review panel to save the new tag content.

  4. The tag will now appear on the financial statements as an edited tag.
