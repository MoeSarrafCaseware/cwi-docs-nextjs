Add a query group

Groups help you organize queries in the document by grouping related ones. Groups are mandatory in the Query document. You can't add a query without a group.

To add a group, select +Group. A new group is added in the document with a default title New Group. You can select the default title to change it.

You can remove a group by selecting More actions ()| Delete.

Note: If you delete a group, all queries in this group are deleted as well.
