Add new Word and Excel documents

You can create new Microsoft Word and Excel documents directly in the product template. Once you create a document in the template, end users can then open the document in Word or Excel and edit it.

To add a new Word and Excel document:

  1. On the Documents page, go to the phase or folder you want to add the document to and select Add () | Microsoft Word or Add () | Microsoft Excel.

  2. In the Document Properties dialog, enter the identifier and name of the document in the corresponding fields.

  3. You can also select whether a signoff scheme applies to the document and edit the visibility settings.

  4. Select SAVE.
