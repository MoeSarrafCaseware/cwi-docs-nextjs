Add or remove content areas

Areas help organize your financial statement content. Use areas to group similar or related information.

To add an area to the financial statement:

  1. Go to the Document Map, then select Add () | New Area.

    2. A new area is added in the document map and displays on the page with a default title: New Area. Select the default title to change it.

  2. Start adding content to the new area by selecting it to display the Add icon ().

To add content to the new area, select Add (), then choose one of the following content types:

You can add more than one content type in an area. You can also drag and drop the new area to change its location in the document, see Rearrange areas.

To remove an area from the financial statement:

  1. Go to the Document Map and select the area you want to remove.

  2. Select More actions (), then choose Delete Area from the list.

    3. The area is removed and is no longer available on the Document Map.

Add Notes and Area titles and headings as reference links to financial statements

You can add reference links to Notes and Area titles or headings to different areas within your financial statement. Reference links can be added to both text areas and dynamic tables.

Add a reference link to a text area:

  1. Go to your financial statement in the Documents page.

  2. Click any text area where you wish to add a reference link.

  3. Click the icon [] to Insert Note Reference. The Insert Note Reference dialog box will open.

  4. Select the Note or Area title or heading that you wish to insert into your text area.

Add Note reference

 

Add reference links to a dynamic table:

  1. Go to your financial statement in the Documents page.

  2. Click any dynamic table where you wish to add a reference link.

  3. Click the More Actions icon [] and click Edit dynamic table.

  4. Under the Note column, click the hashtag symbol in the row that you wish to add a reference to a Note or Areatitle or heading. The Insert Note Reference dialog box will open.

  5. Select the Note or Area title or heading link to insert into your dynamic table.

  6. If the Note column is not visible in your Dynamic table settings, you can add it by clicking the Add icon [] at the top of the table in the location where you wish to add the column.
