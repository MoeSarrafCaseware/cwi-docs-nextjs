Add visibility logic in a checklist

Visibility settings provide built-in automation so that only the relevant items are available in your document. This is achieved by setting up visibility conditions for various items.

You can add visibility conditions for groups, subgroups, procedure or sub-procedures so that your checklist only displays content that is relevant for the engagement. For example, if a trial balance group balance is immaterial, you can hide the related group of procedures.

To learn more, see What are visibility settings?

Add visibility logic for the document

You can add visibility logic for the entire document.

To set visibility for the document:

  1. Select the document name.

    2. The Document Properties dialog displays.

  2. Select Edit Visibility to display the Visibility Settings dialog.

  3. Select Hide or Show from the blue drop-down list to specify the visibility action.

  4. Select all or any to determine whether all conditions must be met or one condition must be met for the visibility action (from Step 3) to apply.

  5. Select Add Condition.

  6. Choose a condition type from the list, then specify the condition parameters.

  7. Once you're done, select Save.

Add visibility logic for a group or subgroup

You can add visibility logic for a group or a subgroup.

To set visibility for a group or subgroup:

  1. Select the group or the subgroup that you want to set visibility for.

  2. Select More actions () | Edit Visibility to display the Visibility Settings dialog.

  3. Select Hide or Show from the blue drop-down list to specify the visibility action.

  4. Select all or any to determine whether all conditions must be met or one condition must be met for the visibility action (from Step 3) to apply.

  5. Select Add Condition.

  6. Choose a condition type from the list, then specify the condition parameters.

  7. Once you're done, select Save.

Add visibility logic for a procedure or a subprocedure

You can add visibility logic for a procedure or a subprocedure.

To set visibility for a procedure or a sub-procedure:

  1. Select the procedure or sub-procedure that you want to set visibility for.

  2. Select More actions () | Settings.

  3. Select Edit Visibility to display the Visibility Settings dialog.

    4. You can also select the Visibility () | Settings ().

  4. Select Hide or Show from the blue drop-down list to specify the visibility action.

  5. Select all or any to determine whether all conditions must be met or one condition must be met for the visibility action (from Step 4) to apply.

  6. Select Add Condition.

  7. Choose a condition type from the list, then specify the condition parameters.

  8. Once you're done, select Save.

You can select Clear to clear all conditions and start over.

You can also copy the visibility settings to another item. Select Copy to copy the visibility settings into clipboard, then open the Edit Visibility dialog for the other item and select Paste to paste the copied settings.

Add visibility logic for a picklist response option

You can add visibility logic for an individual picklist response.

To set visibility for a picklist response:

  1. Navigate to the procedure or sub-procedure whose response options you want to set visibility for.

  2. Select More actions () | Settings.

  3. Select the Edit () icon for the picklist responses.

  4. Select the Visibility () icon for the response you want to edit.

  5. Select Hide or Show from the blue drop-down list to specify the visibility action.

  6. Select all or any to determine whether all conditions must be met or one condition must be met for the visibility action (from Step 5) to apply.

  7. Select Add Condition.

  8. Choose a condition type from the list, then specify the condition parameters.

  9. Once you're done, select Save.

You can select Clear to clear all conditions and start over.

Note that you can also set visibility conditions for picklist responses in the Checklist Settings. The selections you make there are set as the default for all procedures in the checklist.

You can also copy the visibility settings to another item. Select Copy to copy the visibility settings into clipboard, then open the Edit Visibility dialog for the other item and select Paste to paste the copied settings.
