Bulk change group names

If you're setting up multi-language support in your product, you can use the Bulk change group names option to upload a CSV file with the translated group names.

To upload the translated group names:

  1. Switch the Content language of the template to the target language.

  2. Go to the Data page, then select Add () | Bulk change group names.

  3. Select the CSV file from the Documents page (if you've uploaded it to the engagement) or browse your local drive.

    The CSV file must include:

    • Group number

    • Translated consistent name

    • Translated contrary name

    • Translated inconsistent name

    Note: If you are importing a large data set, split them into multiple imports of less than 1000 rows to avoid errors and guarantee a successful import.

    If you change the content language, the translated group names display. For example, if you upload French group names, then select French as the content language, the French group names display in the Data page. To learn more, see the Use-case: A product with French and English language support.
