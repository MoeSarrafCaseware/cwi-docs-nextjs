Change the default group for rounding differences

In the financial statements tables, rounding differences are applied by default to the first non-zero account in the lowest child level that appears in the first group in the trial balance.

For example, if the first parent group in the trial balance is Assets and is structured as follows:

Then once users import the trial balance data, the rounding differences apply to the first non-zero account in Land.

You can change the default group to ensure the rounding difference is applied to a frequently used financial statements line item in your market.

In dynamic tables, the row of the group where rounding differences apply is marked with an asterisk (*).

To change the group where rounding difference apply:

  1. Go to the dynamic table where you have the trial balance data.

    2. For example, while setting up the Balance Sheet.

  2. Select the row marked with an asterisk (*).

  3. Select the group name at Current year.

    4. A list showing the trial balance groups displays.

  4. Select the desired group from the list.

  5. Select the group name at Prior year to display a list of the trial balance groups.

  6. Select the desired group from the list.

Once users import the trial balance data in the engagement, rounding differences are automatically added to the first non-zero account in the groups you've selected for the current year and prior year respectively.

If you collapse the parent groups for the group you've selected for rounding differences, the asterisk relocates to the parent groups. This helps you spot where the selected group is located.

For example, let's assume the group selected for rounding differences is Land.

Notice how the asterisk relocates to the parent groups when they're collapsed. When the parent group Property, Plant and Equipment group is collapsed, the asterisk relocates there.

When the parent group Non-current assets is collapsed, the asterisk relocates there.

When the parent group Assets is collapsed, the asterisk relocates there.
