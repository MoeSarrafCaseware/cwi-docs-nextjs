Customize analytic tests

You can customize automated tests by duplicating them in the product template and then making any changes. The analytic test must be attached to one or more procedures before it can be duplicated and then customized. See Link analytic tests to a checklist procedure for more information. Generic analytic tests that are shipped with the product remain unchanged.

  1. From the Cloud menu (), select Settings () | <your product name> | Manage Template.

  2. Open the product template.

  3. Go to the Analytics Hub. From the Analytic tests list, select the analytic test you want to customize.

    • Any labels and descriptions in double curly brackets can be customized.

    • Any default parameter values can also be customized.

  4. Select the Copy icon ().

  5. In the Copy Analytic Test dialog, in Name, enter a new name for the customized analytic test.

  6. Select Confirm.

  7. Select the Edit icon () next to any label or description you want to modify. Make any changes as necessary.

  8. Select + Add filter to add a default filter to the analytic test.

  9. Under Configuration, modify any default values for parameters as necessary.

  10. Under Analytic details, select + Add new purpose to add your own text to explain the purpose of the analytic test.
