Customize the trial balance groups display settings

To view secondary trial balance grouping structures in your product, select the Information icon () at the top menu, then choose Settings to open the Product Settings dialog. Go to the Trial Balance | Groups tab and list the secondary trial balance grouping structures by name. Note that secondary trial balance grouping structures will only be visible if enabled (see Trial Balance).

If you want group numbers to appear with the description in the trial balance, select the Show group numbers for financials check box. This is useful for markets that don't have a standard chart of accounts and want to simplify the trial balance by not displaying group numbers.

This setting also shows group numbers in the Caseware Cloud Connector Add-in (see image below). To learn more, see How to get started with Cloud Connector.
