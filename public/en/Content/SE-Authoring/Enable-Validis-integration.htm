Enable integration with Validis

Note: Users will require a license from Validis to enable integration with any Caseware Cloud app.

To enable the Validis data connector:

  1. Obtain your Validis API key from Validis. To learn more, refer to Getting Started with Validis.

  2. Specify the Validis ID in the imports array of the product.json file of your SE product. To learn more, refer to Enable specific online data import sources.

  3. Update the collaborate.js file to enable the Validis firm settings page by specifying the following parameters:

this.firmSettingsPages = [
						{
						heading: 'Validis',
						anchorId: '900',
						url: '../c/validis-settings/'
						}
		];

  1. Provide a Validis API key on the Validis page under your app in Settings:

    • Ensure you have the Settings Admin role or equivalent permissions.

    • From the Cloud menu (), select Settings | Your app | Validis.

    • In API Key, enter the API key for the Validis API.

    • Select Save.

    Note: Your Cloud app can only store one API key.
