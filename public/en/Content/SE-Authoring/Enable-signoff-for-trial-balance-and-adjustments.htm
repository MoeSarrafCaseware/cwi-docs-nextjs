Enable signoff for trial balance and adjustments

You can enable the Sign Off option in the Adjustments page and the Data page so users can sign off on the trial balance and adjusting entries. If you haven't enabled the Data page in your product, you can enable sign off in the Trial Balance page.

To enable signoff for trial balance and adjustments:

  1. Go to Product Settings | Trial Balance | Data Page.

  2. Select the Enable Data Page Signoff check box to enable reviewer/preparer sign-off in the Accounts tab and the groups tabs in the Data page.

    If you haven't enabled the Data page in your product, this option displays as Enable Trial Balance Signoff.

  3. Select the Enable Adjustments Signoff check box to enable reviewer/preparer sign-off in the Adjustments page.

After you have enabled sign-off, the Sign Off option will display in the Data and Adjustments pages.
