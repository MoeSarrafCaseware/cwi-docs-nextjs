Enable the new Documents page layout

The new Documents page layout in SE products is designed to provide greater flexibility beyond the simple one preparer and a maximum of two reviewer signoff options.

To enable the new Documents page layout:

  1. Select the Information () icon at the top menu.

  2. Select Settings to open the Product Settings dialog.

  3. Select Enable the new Documents layout from the Features | Documents tab.

The Signoffs tab becomes available in the product settings dialog, enabling access to the Roles and Schemes sections. To learn more, see Define signoff roles and Set up signoff schemes.

Note: Once enabled, the new Documents page layout cannot be disabled again unless any additions or modifications to signoff roles and schemes are reverted back to default.
