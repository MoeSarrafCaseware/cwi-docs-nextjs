Enable the new workflow for trial balance imports from CSV or Excel

The new workflow for data imports features a revamped design to streamline the user experience and introduce functionality changes.

The following import options use the new import workflow by default:

  • Importing trial balance and transaction data from online accounting software

  • Importing trial balance and transaction data from desktop accounting software

  • Importing transaction data from a CSV or Excel file

You can also enable the new import workflow for CSV and Excel trial balance imports in the product settings.

To enable the new workflow in your product, select the Information icon () at the top menu and choose Settings to open the Product Settings dialog. At the Trial Balance | Data Page tab, select Enable new workflow for CSV/Excel trial balance imports.

As part of the new workflow, a new design is also available to map accounts to groups. To enable the new design for mapping accounts to groups, at the Trial Balance | Data Page tab, select Beta Feature: Enable account mapping latest version.
