Export groupings from Working Papers to Excel

To use the grouping structure you've defined in Working Papers, you need to export it to an Excel file, then convert the file to a .csv.

Open Working Papers and follow the steps below.

To export a grouping from Working papers to an Excel file:

  1. Open the file that you want to export.

  2. On the ribbon, click Engagement | Groupings.

    3. The Groupings worksheet displays.

  3. Right-click a column heading and click Reorder columns.

    4. In the Reorder Table Columns dialog, arrange the Active Columns as: Number > Name > Title > Type > Sign

  4. Click OK.

    5. Ensure that each group contains a unique number, a defined type and sign.

  5. On the ribbon, click Tools | Options.

  6. On the General tab, in the Table List Field Style drop-down menu, select Entry Codes.

  7. Click OK.

  8. On the worksheet, select all the groups by clicking any group number, then pressing the CTRL+A on your keyboard.

  9. Copy the selection by pressing CTRL+C on your keyboard.

  10. Open a new Excel workbook.

  11. Paste the groups by clicking cell A1, then hitting the CTRL+V keys on your keyboard.

  12. Verify that the information is correct in Excel:

    • Check for duplicate group numbers by selecting the whole column and pressing Home | Conditional Formatting | Highlight Cell Rules | Duplicate Values.

    • Check for missing or empty values in the Name, Title, and Type columns.

    • Check the Type column to ensure the classification is correct for each line.

    • Check the Sign column to ensure the sign is correct for each line. If the sign is blank, it will default to Debit.

  13. Save the Excel workbook as a .csv file.

  14. Import the file to your template.
