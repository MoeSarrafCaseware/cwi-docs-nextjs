How to author a query?

The following steps summarize the recommended workflow for authoring queries. You can use this workflow for guidance.

  1. Add a query template

  2. Author the query template

    • Add contact instructions

    • Add question sets

    • Add query questions

    • Review and customize the query question settings

    • Add a response type to a query question

    • Add a reference to a document in a question

    • Link a query question to a checklist procedure

    • Rearrange question sets and questions

    • Add visibility settings

  3. Create a query document using an existing template
