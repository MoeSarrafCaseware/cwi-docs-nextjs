Import groupings from a CSV or an Excel file (Data page)

Once you have the CSV or Excel file ready, you can start the import process.

To import a grouping from a CSV or Excel:

  1. Go to the Data page, then select Add () | Import groups.

  2. Drag and drop the .csv file into the pop-up dialog.

  3. Select Yes when prompted to confirm removal of the existing financial groups.

Once the import is complete, review the structure to ensure that the import has been successful. If necessary, you can also edit a group to add additional properties or change existing ones.
