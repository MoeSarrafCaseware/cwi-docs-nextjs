What's new in XBRL - April 2025

The following lists describe the improvements and fixed issues in the April 2025 XBRL updates.

SE Client

Improvements

  • [SE-53523] - The taxonomy structure in the Add tag dialog is now populated from the XBRL Context Microservice, enabling it to load instantaneously.

  • [SE-53230] - Cleaned up numeric generation and improved function labeling.

  • [SE-49556] - Added hypercube role information to the Context Editor for related dimensions.

  • [SE-41250] - Added "copy" functionality to the Numeric Options Editor, allowing new numerics to be created from existing ones.

Fixed issues

  • [SE-53948] - Dimensions with no “default” value cannot be modified to any other value than the first in the list.

  • [SE-53828] - The EnergyItemType tag incorrectly states that it does not require a numeric value.

  • [SE-53760] - The Close icon is not correctly aligned in the XBRL Review panel.

  • [SE-53587] - The Export button does not consistently initiate the XBRL export process.

  • [SE-53505] [SE-53506] [SE-53508] [SE-53510] - Users cannot tab through items in the XBRL Review panel.

  • [SE-53191] - Products without a context name appear as "null" in the Duplicate XBRL Fact dialog.

  • [SE-52819] - iXBRL exports erroneously appear in the SE Document Manager until the page is refreshed.

  • [SE-52586] - The snapshot service does not generate the necessary reference IDs for iXBRL review and tagging.

  • [SE-44537] - Custom tag identifiers do not display correctly when added to manually inserted tables.

  • [SE-43738] - Help text incorrectly allows tagging.

XBRL Context (Ver. 2.1.3)

Improvements

  • [SE-53995] - Added Irish taxonomy linkbases to the MS Context microservice. These can now be selected from the Add tag dialog.

Fixed issues

  • [SE-54914] - Automation tests that utilize the "HelloWorld" taxonomies no longer function.

XBRL Context Server (Ver. 2.2.1)

Improvements

  • [SE-55108] - Upgraded the MS Context server base to version 5.24.0.
