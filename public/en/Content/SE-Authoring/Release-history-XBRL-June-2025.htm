What's new in XBRL - June 2025

The following lists describe the improvements and fixed issues in the June 2025 XBRL updates.

SE Client

Enhancements

  • [SE-54956] - Added export support for the new footnote variable in the Tagging Merge Tool and the Export Tool.

  • [SE-55110] - Added the ability to apply tags to SE Builder forms.

  • [SE-54203] - Added the ability to apply block tags to entire notes.

  • [SE-55949] - Increased the code coverage of unit tests to over 80%.

Fixed issues

  • [SE-56206] - Decimal separator styles are not carried over to the iXBRL file.

  • [SE-56274] - Incorrect scaling attribute settings for Numeric Tags result in iXBRL filing errors.

  • [SE-55388] - The Delete Numeric dialog appears behind the Numeric Options Editor, making it inaccessible.

  • [SE-55458] - The Content drop-down menu does not appear in the XBRL Review pane after changing the UI language.

  • [SE-53708] - Custom tags on table cells do not display the correct content value until the form is refreshed.

  • [SE-53754] - The Value field in the XBRL Review pane does not show the correct native value of the cell being tagged.

XBRL Server (Ver 2.5.8.0)

Fixed issues

  • [RM-3889] - Empty tables in the iXBRL file result in validation errors.

XBRL Server (Ver 2.5.7.3)

Fixed issues

  • [SE-55372] - iXBRL exports to xHTML do not include a table note number reference column or the relevant exported data.

  • [SE-55798] - "(Continued)" appears in the title of all note headers in iXBRL exports to xHTML.

  • [SE-55892] - The microservice cannot download financial statements that exceed 4MB.

XBRL Context Server (Ver 2.2.2)

Enhancements

  • [SE-55730] - Added the SE IFRS taxonomy.

  • [SE-55691] - Generated and deployed English presentation linkbases for the South African taxonomy.
