What's new in XBRL - May 2025

The following lists describe the improvements and fixed issues in the May 2025 XBRL updates.

SE Client

Improvements

  • [SE-54538] [SE-54649] [SE-54548] [SE-54435] - Improved coverage of XBRL unit tests by adding new tests for the following components:

    • Unit viewer

    • Numeric viewer

    • Context selector

    • Context delete dialog

  • [SE-53907] [SE-55112] - Improved the search capabilities of the Add tag dialog to include ‘type-ahead’ functionality. These type-ahead results are sorted by presentation role.

  • [SE-54758] - Added footnote tagging functionality to the XBRL Review panel.

  • [SE-51104] - Added a temporary list that displays the resolved duplicate facts for the current session. This list is cleared when the page is reloaded.

  • [SE-50483] - Added the option to include review functionality of all present tags in exported iXBRL documents.

Fixed issues

  • [SE-49401] - The Documents pane becomes inaccessible after closing the XBRL Review panel.

  • [SE-49161] - System numerics do not appear in the Numeric editor.

  • [SE-54543] - System tags do not appear on tagged headers in the engagement and exports.

XBRL Server (Ver. 2.5.6.3)

Improvements

  • [SE-54758] - Added a new field to the XBRL Review panel that allows users to apply metadata values to footnotes.

  • [SE-54348] - Implemented various updates to security reports to facilitate further XBRL tagging enhancements.

Fixed issues

  • [SE-55433] - (DK) Exports from Årsrapport XBRL 2024 get stuck in a continuous validation cycle.

XBRL Server (Ver. 2.5.5.5)

Improvements

  • [SE-55109] - Updated the Gradle version to 8.6 and the Serverbase to 5.24.0.

  • [SE-55027] - Removed an unnecessary elastic search function that was affecting microservice performance.

Fixed issues

  • [SE-55359] - (NL) The microservice returns varying responses to the same call when validating XBRL instance documents.

  • [SE-53966] - (DK) Exports from Nettoomsætning 2024 return an error during validation.
