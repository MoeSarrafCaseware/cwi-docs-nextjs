Resolve conflicts in financial statements

After you have copied documents from your source template, you may encounter broken links to:

  • Response conditions

  • Financials group conditions

  • Engagement glossary terms

  • Missing visibility / dynamic text conditions

    • Note: if multiple broken links are generated by dynamic text, each link will need to be resolved separately.

  • Missing checklist response formula conditions

If there are conflicts, the Broken Links pane displays on the right to allow you to assess and resolve conflicts. The pane shows conflicts grouped by document name

Tip: You can also select the () button to display the Broken Links pane. Note that the number of conflicts are displayed in the red badge.

To resolve a broken link:

  1. Scroll to the document where you want to resolve conflicts.

  2. Select the broken link you want to resolve.

    3. Once selected, the document with the broken link will open. and the broken link dialog will appear.

    The Broken Link dialog displays details of the conflict and resolution options.

    Note: Smart Copy-Paste functionality has certain limitations which will be addressed in future releases; you can view these limitations here. An error dialog will display when a resolution is unsupported.

  3. Using the broken link dialog, you can:

    • Use the available drop-downs to select valid options.

      • For example, if an engagement glossary term copied over from a source document does not exist in your cloud app, select a term from your app's glossary to insert it in the document instead.

    • Select the delete button () to remove the item where the conflict resides or ignore the conflict to resolve it later

      • For example, if an engagement glossary term copied over from a source document does not exist in your cloud app, remove it from the document.

  4. Select SAVE to make your changes permanent.

    Note: The SAVE button is only enabled when you make valid selections in the dialog. CANCEL can be pressed at any time to discard your changes.
