Review and customize question settings

If you want to view or customize the settings for a specific query question in the document, select More actions () | Settings. Once you're done with your changes, select Save.

The following options are available in the Query Settings dialog.

Allow input notes in query

Select this option to add a text box below the response where users can enter notes. You can add a note placeholder in the displayed field.

Allow multiple rows

Select this option to display each response type you add to the query question as a column in a table. The placeholder for each response type will display as the column heading. You can then select Add Row to add more rows to the table.

Note that you can have a maximum of three columns in a table.

Response sets

  • Select +Response to add a response.
  • Select Remove () to remove a response.
  • Select Edit () to edit a response.

Show responses beneath procedure

Select this option to display the responses below the query.
