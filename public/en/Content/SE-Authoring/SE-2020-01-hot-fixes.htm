SE 2020-01 hot fixes

Here is a summary of the hot fixes that have been deployed for SE 2020-01.

SE 2020-01-H2

Version: 2020-01 Build 475

Here is the list of issues that have been fixed with the deployment of SE 2020-01-H2.

To view the details of the root cause and the fix, select the issue identifier (example: SE-12345).

  • [SE-17457] - If users exclude an area in the financial statements document then carry forward the file and accept suggestions, the content in the excluded area displays in an incorrect order (different in than the original file).

  • [SE-17646] - Adding a materiality Data Link in text fields does not work as expected. The word "Formula" displays indicating an error in the linked data.

  • [SE-17628] - Engagement files based on templates that have the Data page enabled display the old Trial Balance page.

  • [SE-17449] - Accounts are not properly assigned to groups after importing data using an accounting platform (BGL).
