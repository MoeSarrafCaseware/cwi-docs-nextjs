SE 2020-03 hot fixes

Here is a summary of the hot fixes that have been deployed for SE 2020-03.

SE 2020-03-H1

Version: 2020-03 Build 496

Here is the list of issues that have been fixed with the deployment of SE 2020-03-H1.

  • [SE-19545] - Grouping and ungrouping of trial balance accounts in dynamic tables does not function as expected.

  • [SE-19550] - A technical issue that affects the performance of the Data page.

  • [SE-19468] - Consolidating more than one entity does not work as expected. Some entities are out of balance.

SE 2020-03-H2

Version: 2020-03 Build 498

Here is the list of issues that have been fixed with the deployment of SE 2020-03-H2.

  • [SE-19610] - If users rename a completed query document from the Documents page, the uploaded client files are no longer available in the query questions.

  • [SE-19692] - In dynamic tables, iXBRL facts in custom groups are not exported when the groups are collapsed.

  • [SE-19742] - The list of dimensions is not available when users attempt to add a dimension row to an IFRS dynamic table.

  • [SE-17876] - The Guidance section does not display as expected in the old risk and control reports.

  • [SE-19655] - In the old query layout, users cannot manually hide query questions.

SE 2020-03-H3

Version: 2020-03 Build 500

Here is the list of issues that have been fixed with the deployment of SE 2020-03-H3.

  • [SE-19801] - Users cannot add or edit dynamic tables in the financial statements.

  • [SE-19773] - Procedure numbering in checklists does not work as expected.

SE 2020-03-H4

Version: 2020-03 Build 504

SE 2020-03-H4 addresses enhancements to:

  • The stability of production environments

  • The stability and performance of product authoring environments

Here is the list of issues that have been fixed with the deployment of SE 2020-03-H4.

  • [SE-19845] - In dynamic tables, engagement glossary terms in total rows do not update correctly when a term is modified elsewhere.

  • [SE-19826] - The More actions menu for financial statement areas does not work as expected. Authors cannot select options in the menu.

  • [SE-19907] - In product templates, pre-set dimensions in IFRS dynamic tables do not function as expected.

  • [SE-19859] - The Parent group information and group properties are not included when authors export groups.

  • [SE-19897] - When users select a control risk in a risk assessment card, the card collapses.

  • [SE-19903] - Technical issues causing consolidation engagements not to open.

  • [SE-19982] - A technical issue that affects performance when custom form developers are working remotely.

SE 2020-03-H5

Version: 2020-03 Build 506

Here is the list of issues that have been fixed with the deployment of SE 2020-03-H5.

  • [SE-20016] - Elimination adjustments posted to zero balance accounts do not display in the financial statements.

  • [SE-20107] - If users delete a document reference from the procedure guidance close the checklist and reopen, then the reference is also removed from the procedure.

  • [SE-20081] - Materiality data links added by authors do not function as expected.

  • [SE-19334] - Users can create engagements for products managed outside of the Caseware Cloud platform.

  • [SE-20042] - Analysis sections for certain trial balance groups display empty.

  • [SE-20114] - Carry forward does not function as expected when end users make changes to financial statements notes. Notes are not displayed in the correct order.

SE 2020-03-H6

Version: 2020-03 Build 507

Here is the list of issues that have been fixed with the deployment of SE 2020-03-H6.

  • [SE-20229] - When users print the financial statements, analysis graphs are not included in the generated PDF.

  • [SE-20219] - The Download reference copy option does not work as expected. Users receive an error message and the download fails.

  • [SE-20178] - The Download reference copy option produces corrupted files.

  • [SE-20175] - Dynamic text in financial statements does not display as expected. Extra html tags display before and after the dynamic text.

SE 2020-03-H7

Version: 2020-03 Build 510

Here is the list of issues that have been fixed with the deployment of SE 2020-03-H7.

  • [SE-20174] - In dynamic tables that only display one year of data, balances from prior years affect the visibility of a group or account row.

  • [SE-20478] - The Print and Save to Cloud options do not work as expected for documents that include a Response (checklist procedure response) data link (fx) with a Manual response type. The PDF file fails to generate.

  • [SE-20343] - An error occurs upon carry forward of an engagement where,

    • Adjustments have been made to a prior year group balance

    • That group has also been changed to a non-assignable group in the current year.

SE 2020-03-H8

Version: 2020-03 Build 517

Here is the list of issues that have been fixed with the deployment of SE 2020-03-H8.

  • [SE-21018] - Staff cannot view query questions when they attempt to view query documents using the Timeline or the Activities pages.

  • [SE-21111] - In the Trial Balance page, when users log in to a SuperMate account to import client data, they can't log out of the account.

  • [SE-19652] - When MacOS users edit and save a Microsoft Office document in an engagement file, the file saves correctly but an error message is displayed.

  • [SE-20246] - Data links (fx) referencing procedure responses do not evaluate if users make changes to the responses after they've inserted the data links. For more details, see the SE 2020-03 known issue SE-20961.
