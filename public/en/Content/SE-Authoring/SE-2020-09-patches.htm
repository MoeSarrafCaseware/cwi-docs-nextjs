SE 2020-09 patches

Here is a summary of the patches that have been deployed for SE 2020-09.

SE 2020-09-U1

Version: 2020-09 Build 551

The deployment of SE 2020-09-U1 addresses technical improvements related to security, stability, and logging capabilities.

SE 2020-09-U2

Version: 2020-09 Build 552

Here is the list of issues that have been fixed with the deployment of SE 2020-09-U2.

  • [SE-23213] - Analysis tables do not display as expected. The Annotation() and Issue () icons overlap with the last column of the table.

  • [SE-23236] - An "Invalid Engagement" error occurs when users attempt to open an engagement.

SE 2020-09-U3

Version: 2020-09 Build 554

Here is the list of issues that have been fixed with the deployment of SE 2020-09-U3.

  • [SE-23511] - Data links (fx) which include quotations ("") in their output do not display correctly and may result in errors.

  • [SE-23509] - Users are unable to select "February 28, 2021" as the Year end date in their engagement properties.

  • [SE-23402] - When users print the financial statements, the generated PDF does not display as expected. Content in lists does not display in full.

  • [SE-23428] - In XBRL review mode, the Documents pane overlaps with the XBRL Review pane and users are unable to select the Export button.

  • [SE-23446] - If users select not to include notes to the financial statements, an empty notes section is included when they print the financial statements.

  • [SE-23458] - In dynamic tables, when users select to display a row with the opposite sign, the opposite sign is not applied to Custom calculation columns.

  • [SE-23489] - When the Decimals setting is set to display zero decimal places in the product settings, decimal symbols do not display in the engagement file.

SE 2020-09-U4

Version: 2020-09 Build 559

Here is the list of issues that have been fixed with the deployment of SE 2020-09-U4.

  • [SE-23544] - Users are unable to select "February 28, 2021" as the Year end date in their engagement properties.

  • [SE-23592] - When users print financial statements, the table of contents in the generated PDF does not include page numbers.

  • [SE-23614] - When authors use the Copy From Another Product option in the product template, the Broken Links pane displays conflicts that cannot be resolved. The message "This link type is currently unsupported" displays when authors attempt to resolve the conflict.

  • [SE-23534] - Users receive multiple error messages when proxying to SE develop.

  • [SE-23435] - An error occurs when users attempt to carry forward an engagement.

SE 2020-09-U5

Version: 2020-09 Build 561

Here is the list of issues that have been fixed with the deployment of SE 2020-09-U5.

  • [SE-23697] - In dynamic tables, if both financial statement overrides and rounding differences are applied to accounts in a group, the total row does not calculate correctly when collapsed.

  • [SE-23450] - Annotations in analysis tables are not included in the generated PDFs when using the Download reference copy option.

  • [SE-23706] - Engagements created from carry forward do not include data links (fx) that have been added or modified in the original file.

  • [SE-23653] - Authors are unable to view financial statements notes that have more than 100 content sections, such as text or dynamic tables.

  • [SE-23619] - The print preview option does not work as expected. There are discrepancies between the page breaks in the print preview and the printed documents.

  • [SE-23557] - In IFRS tables (movement tables), users cannot post adjustments to groups that are assigned to second level dimensions.

SE 2020-09-U6

Version: 2020-09 Build 566

Here is the list of issues that have been fixed with the deployment of SE 2020-09-U6.

  • [SE-24058] - In financial statements, data links (fx) in text sections display as "#ValueUnavailable" if the guidance information for the section is modified.

  • [SE-23422] - In queries, when users try to add data links (fx) to the question text, the data link is moved to the beginning of the text.

  • [SE-23999] - In engagements created from carry forward, users are unable to add page breaks to the financial statements.

  • [SE-23831] - Existing connector formulas in files are still available after a firm's Caseware Cloud Connector license is disabled.

SE 2020-09-U8

Version: 2020-09 Build 569

One issue has been fixed with the deployment of SE 2020-09-U8.

[SE-24233] - External documents which were corrupted when they were uploaded to an engagement and have since been repaired are not accessible when users use the Download reference copy option. The PDF files are corrupted.

SE 2020-09-U9

Version: 2020-09 Build 578

SE 2020-09-U9 addresses a memory leak issue. As a result, users cannot upload external documents such as Word, Excel, and PDF.
