SE 2020-12 patches

Here is a summary of the patches that have been deployed for SE 2020-12.

SE 2020-12-U1

Version: 2020-12 Build 581

Here is the list of issues that have been fixed with the deployment of SE 2020-12-U1.

  • [SE-24714] - In the new Documents page, the signoff roles are not displayed in the correct order and do not function as expected.

  • [FLAT-6164] - An error occurs when users attempt to export XBRL files.

SE 2020-12-U3

Version: 2020-12 Build 583

Here is the list of issues that have been fixed with the deployment of SE 2020-12-U3.

  • [SE-25012] - A technical issue that causes performance degradation and intermittent outages.

  • [SE-24938] - When the new Documents page is enabled, users cannot perform bulk carry forward if the prior year's engagement file includes Consideration for next year issue types.

  • [SE-24965] - In letters and memos, formatting applied to bullet list items with visibility conditions is not saved.

SE 2020-12-U4

Version: 2020-12 Build 584

SE 2020-02-U4 mitigates a technical issue that causes performance degradation and intermittent outages.

SE 2020-12-U5

Version: 2020-12 Build 588

Here is the list of issues that have been fixed with the deployment of SE 2020-12-U5.

  • [SE-24382] - Product templates with large history files do not export as expected.

  • [SE-24985] - When the new Documents page is enabled, the URLs for external links are not carried forward.

  • [SE-25066] - Values entered in table cells are not saved if the user uses the Tab or Enter key to navigate between cells.

  • [SE-25042] - If the print settings of a document are modified in the firm template and the document is then removed in later versions of the product template, an error occurs when firm authors attempt to update the firm template.

SE 2020-12-U6

Version: 2020-12 Build 589

SE 2020-02-U6 addresses a configuration change that applies to the Canadian region.

Other regions can refer to the SE 2020-12-U5 notes for details about the issues relevant to your region that have been fixed in this release.

SE 2020-12-U7

Version: 2020-12 Build 590

Here is the list of issues that have been fixed with the deployment of SE 2020-12-U7.

  • [SE-25258] - Users cannot revert overridden balances in dynamic tables.

  • [SE-25142] - The None option for signoff schemes does not work as expected. When authors select this option for a document in the product template, end users can still sign off on the document.

  • [SE-25281] - An issue that causes some engagements (within specific apps - not all Cloud apps) to break.

  • [SE-25208] - When users export an XBRL file, hidden text sections and dynamic tables are not marked as hidden in the file.

  • [SE-24955] - In engagements that use the old query layout, contacts receive authorization errors when opening queries.

  • [SE-25193] - A technical issue that affects performance in checklists.

  • [SE-24437] - A "Failed to archive" error occurs when using the Download reference copy option if another user is uploading a file to the engagement at the same time.

SE 2020-12-U9

Version: 2020-12 Build 594

Here is the list of issues that have been fixed with the deployment of SE 2020-12-U9.

  • [SE-25342] / [SE-25334] - An error occurs when users add a condition to an engagement glossary term using the Copy Condition option, then attempt to save the term.

  • [SE-25344] - Users receive an error when they attempt to open some engagements (within specific apps - not all Cloud apps).

  • [SE-25296] - Spacing issues when users export iXBRL files that cause file validation errors.

  • [SE-25382] - In letters, text placeholders do not display as expected.

  • [SE-25033] - When the new Documents page is enabled, an error occurs when authors attempt to use the Copy From Another Product option in the product template.

  • [SE-25319] - A technical issue in the new Documents page that affects automatic testing scripts.

  • [SE-25332] - When the Decimal symbol setting is set to use commas in the product settings, the materiality report does not calculate materiality levels correctly.

SE 2020-12-U10

Version: 2020-12 Build 596

One issue has been fixed with the deployment of SE 2020-12-U10.

[SE-25474] - In engagements created from carry forward, user sessions time out before the maximum inactivity time is reached.

SE 2020-12-U11

Version: 2020-12 Build 599

Here is the list of issues that have been fixed with the deployment of SE 2020-12-U11.

  • [SE-25562] - In product templates that support multiple languages, data links (fx) in financial statements do not work as expected. When an author switches to a different language, the data link displays as "#ValueUnavailable#" and the author cannot open the Data Link dialog to edit the link.

  • [SE-25476] - In dynamic tables, users cannot revert table cells to their original values after they override the calculated value.

  • [SE-25254] - A technical improvement related to logging capabilities.

  • [SE-25316] - Attempting to carry forward an engagement for the following use case results in an error:

    1. Users added assertions to a checklist procedure in the prior year's file

    2. The primary or firm author added assertions for the same procedure in the current year's template.

  • [SE-25707] - A technical issue that causes performance issues in the new Documents page.

SE 2020-12-U12

Version: 2020-12 Build 601

Here is the list of issues that have been fixed with the deployment of SE 2020-12-U12.

  • [SE-25737] - User sessions time out before the maximum inactivity time is reached.

  • [SE-25842] - Users receive multiple prompts to log in to Cloud when opening Word or Excel files in an engagement.

SE 2020-12-U13

Version: 2020-12 Build 605

Here is the list of issues that have been fixed with the deployment of SE 2020-12-U13.

  • [SE-25990] - A technical issue that causes performance issues in checklists.

  • [SE-25974] - A technical issue that causes performance issues in the new Documents page.

Version: 2020-12 Build 606
