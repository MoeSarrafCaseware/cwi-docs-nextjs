SE 2021-03 patches

Here is a summary of the patches that have been deployed for SE 2021-03.

SE 2021-03-U1

Version: 2021-03 Build 611

Here is the list of issues that have been fixed with the deployment of SE 2021-03-U1.

  • [SE-26547] - In dynamic tables, balances for accounts in custom groups display with zero values after carry forward.

  • [SE-26495] - In the Data page, assigning accounts to a different parent group in a tax grouping does not work as expected. Users need to open the Edit dialog twice before they can select a new parent group.

  • [SE-26270] - The formatting preferences selected for headings in the product settings are applied to note titles.

  • [SE-26228] - When users print the financial statements, the last lines of text sections are cut off in the generated PDF.

  • [SE-26410] - An error occurs when authors attempt to open the product template if they remove a content language after they’ve already added content in that language.

  • [SE-26425] - Performance improvements related to the new Documents page.

  • [SE-26490] - In the new Documents page, the Download signature log option is not available for queries.

  • [SE-26515] - The XBRL validation errors dialog displays when the showValidationErrorMessage property is set to false.

  • [SE-26629] - When users export XBRL files, the areas, Table of Contents and Note columns in dynamic tables do not display as expected in the generated file.

SE 2021-03-U2

Version: 2021-03 Build 615

The deployment of SE 2021-03-U2 addresses technical improvements related to stability.

SE 2021-03-U3

Version: 2021-03 Build 616

One issue has been fixed with the deployment of SE 2021-03-U3.

[SE-26707] - Users cannot import data from Xero.

SE 2021-03-U4

Version: 2021-03 Build 622

Here is the list of issues that have been fixed with the deployment of SE 2021-03-U4.

  • [SE-26685] - Account lines in adjustments do not display in the order they were created.

  • [SE-26699] - In consolidation engagements, balances do not display for total rows in dynamic tables if the table is set to use consolidated data.

  • [SE-26703] - In products that support multiple languages, end user changes to dynamic text are not saved for all languages in carried forward engagements.

  • [SE-26803] - When users print the financial statements, the Accounts table analysis chart does not display as expected.

  • [SE-26789] - Users receive an error when they attempt to carry forward an engagement.

  • [SE-26762] - In checklists, when users manually hide procedures after reviewer sign-off, the changes are not displayed when using the Changes since engagement creation option in review tools.

  • [SE-26795] - When users print the Adjustments page, adjustments are broken across pages.

SE 2021-03-U5

Version: 2021-03 Build 623

Here is the list of issues that have been fixed with the deployment of SE 2021-03-U5.

  • [SE-26835] - Users cannot edit the materiality document.

  • [SE-26903] - Some Cloud apps are not working on smart devices (phones and tablets).

  • [SE-26916] - The Document Map pane is no longer available in checklists once authors embed a custom form.

  • [SE-26772] [SE-26913] - In engagements created from Carry Forward, dynamic tables do not display as expected.

  • [SE-26906] - Values in IFRS (movement) tables are not included in the XBRL export.

  • [SE-26929] - Incorrect formatting of numeric cells in custom forms that are embedded in financial statements.

  • [SE-26934] - In dynamic tables, flipped (dynamically reassigned) values in Total rows do not align properly.

SE 2021-03-U6

Version: 2021-03 Build 624

Here is the list of issues that have been fixed with the deployment of SE 2021-03-U6.

  • [SE-27046] - An error occurs when users attempt to import PDF documents from Class Super within the following use-cases:

    • Authors create the destination placeholder and then move it to a different folder in the product template, then users attempt to import the PDFs.

    • After importing the PDF documents, users move a document to another folder, then attempt to re-import.

  • [SE-27069] - In engagements created from carry forward, formatting in documents does not display until users accept or ignore the carry forward changes.

  • [SE-27015] - Dynamic tables do not display as expected. Title rows display a value of 0.0 in custom calculation columns and negative values are not aligned correctly.

SE 2021-03-U7

Version: 2021-03 Build 636

Here is the list of issues that have been fixed with the deployment of SE 2021-03-U7.

  • [SE-27419] - When users print the financial statements, font colors do not display as expected in the generated PDF.

  • The following issues apply to engagements created from carry forward:

    • [SE-27183] | [SE-27127] - Technical issues that cause errors when opening engagements.

    • [SE-27330] - When users print financial statements, the dynamic table rows they’ve added in the prior year file are highlighted in green in the generated PDF.

    • [SE-27246] - The carried forward file is temporarily available only to the user that performed the carry forward and users with admin security roles.

    • [SE-26854] - Annotations added to checklist procedures in the prior year file are duplicated.

    • [SE-27240] - Existing text formatting in the checklist procedures Summary field does not apply until users accept or ignore the carry forward changes.

  • [SE-27456] - An error occurs when users attempt to use the Download reference copy option. Queries are not included in the generated PDFs.

  • [SE-27413] - In IFRS tables, an error occurs when users select a primary level dimension for a column.

  • [SE-27138] - When query questions that include data links (fx) are linked to checklist procedures, the data links do not display in the checklist.

  • [SE-27194] - A “Download Reference Copy Failed” error occurs when users attempt to use the Download reference copy option if the Summary of Misstatements document is included in the generated PDFs.

SE 2021-03-U8

Version: 2021-03 Build 645

Here is the list of issues that have been fixed with the deployment of SE 2021-03-U8.

  • Carry forward:

    • [SE-27647] - A technical issue that causes errors when opening engagements created from carry forward.

    • [SE-27624] - A technical issue that causes errors when users attempt to bulk carry forward all financial statement sections created in the prior year file.

    • [SE-27530] - In the financial statements, users cannot accept carry forward suggestions for notes that were reordered in the prior year file.

    • [SE-27654] - A technical issue that has been identified after reviewing production logs.

    • [SE-27375] - A technical issue that causes errors when carrying forward an engagement.

  • [SE-27572] - A “Failed to update” error occurs when firm authors attempt to update the firm template.

  • [SE-27713] - When users print documents, the font and heading settings selected in the Product Settings are not applied to the generated PDF.

  • [SE-27548] - The Summary of Misstatements does not work as expected. If the tax rate is a non-numeric value, users receive console errors when they attempt to edit and save the document.

  • [SE-27749] - When users export the trial balance to PDF, the generated PDF does not display as expected. The column headings are not aligned with the column data.
