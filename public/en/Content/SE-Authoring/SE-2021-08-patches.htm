SE 2021-08 patches

Here is a summary of the patches that have been deployed for SE 2021-08.

SE 2021-08-U1

Version: 2021-08 Build 681

Here is the list of issues that have been fixed with the deployment of SE 2021-08-U1.

  • Financial statements:

    • [SE-29088] - When rounding is applied to an account in a dynamic table, other dynamic tables that include the account are not updated with the rounded value.

    • [SE-29229] - A performance improvement.

    • [SE-29163] - In the financial statements, zeros do not display as dashes when the Show zeros as dashes option is selected in the product settings.

    • [SE-29113] - Guidance sections do not display in the notes to the financial statements.

    • [SE-29119] - When users define note associations for parent-level notes, the association does not apply to the corresponding subnotes.

  • [SE-29284] - When users open a PDF document, annotations do not display and the option to add annotations is unavailable.

  • [SE-28920] - In products that support multiple languages, when firm authors mark a language as inactive in the firm template, the language is no longer visible to the firm author in the list of content languages in the product settings.

  • [SE-29161] | [SE-29329] - Checklist procedures with an Entity information response type that link to the Organization type field display “Unassigned” as the response if the organization type is “Limited Liability Company (LLC)”.

  • [SE-29226] - External documents are deleted from engagements if a user attempts to update the document properties at the same time as another user replaces the document.

  • [SE-29231] - When users export XBRL files, values in IFRS movement tables are not included in the export unless the user has the financial statements open when they perform the export.

  • [SE-29244] - Users cannot use the scroll bar when viewing long comments in queries.

  • [SE-29297] - In products that support multiple languages, document names display as “Untitled” for the following use-case:

    1. Users switch the content language to a non-default language and upload a file.

    2. Users then switch to another content language.

  • [SE-29042] - Changes made to the conclusion section of checklists are not included when users carry forward an engagement.

  • [SE-29050] - In dynamic tables, an error occurs when users enter non-numeric values in columns with a numeric column type. The dynamic table cannot be saved.

  • [SE-29084] - Table borders are not applied to dynamic tables in worksheets.

  • [SE-29218] - When users print checklists, the generated PDF does not display as expected. Sign-offs for procedures in the conclusion section are not aligned correctly.

  • [SE-29313] - In products that support multiple languages, when users switch the content language to a non-default language, picklist response options in checklists do not display.

  • [SE-29127] | [SE-29131] | [SE-29137] | [SE-29108] - Technical issues that have been identified after reviewing production logs.

SE 2021-08-U2

Version: 2021-08 Build 686

Here is the list of issues that have been fixed with the deployment of SE 2021-08-U2.

  • [SE-29365] - In the notes to the financial statements, the Insert from library option is not available if users have already added all hidden parent-level notes, even if there are hidden sub-level notes available in the library.

  • [SE-29484] - In engagements created from carry forward, engagement glossary terms that were edited by end users in the prior year file do not carry forward as expected.

  • [SE-29376] - When users import data from BGL Simple Fund 360, the import screen does not display the BGL entity that the data will be imported from.

  • [SE-29334] - When users import trial balance data, the data displays as out of balance if any of the account balances include exponential notation.

  • [SE-29449] - If translated content is not available for the selected content language, the content does not display at all instead of displaying in the default language.

SE 2021-08-U5

Version: 2021-08 Build 690

One issue has been fixed with the deployment of SE 2021-08-U5.

[SE-29885] - A performance improvement related to custom forms development.

SE 2021-08-U9

Version: 2021-08 Build 700

Here is the list of issues that have been fixed with the deployment of SE 2021-08-U9.

  • Dynamic tables

    • [SE-29813] - Diagnostic labels display as partially hidden and cannot be edited.

    • [SE-29712] - Adding annotations to a dynamic table row does not work as expected. When users refresh or navigate away, the annotation is not saved.

    • [SE-29894] - Decimal places do not display in input columns.

    • [SE-29728] - The Change Font Size menu does not display as expected. Users cannot access the menu to select options.

    • [SE-29852] - Data links (fx) do not work as expected when authors copy dynamic tables.

  • [SE-29826] - Clients cannot download files uploaded to query documents.

  • [SE-29918] - Prior year tax data does not display in custom forms.

  • [SE-29972] - An error occurs when users attempt to export iXBRL files. The files do not generate.

  • [SE-23001] - Data links (fx) with trial balance conditions do not work as expected if the financial group parameter for the condition is a net income group.

  • [SE-29317] - In consolidation engagements, the Data page incorrectly displays as out of balance.

  • [SE-29474] - In consolidation engagements, the Data page does not display the correct names for subsidiary entities.

  • [SE-29637] - When users create issues, there is a delay between the user typing and the typed text displaying in the description text box.

  • [SE-29769] - In analysis charts, tooltips for account balances do not display as expected. An equal sign (=) displays at the end of the account balance value in the tooltip.

  • [SE-29858] - When users open a document from the Documents page, the incorrect document opens if they have selected to view hidden items on the page.

  • [SE-30051] - In products that support multiple languages, when users change the content language of the engagement to a non-default language, the content continues to display in the default language.

  • [SE-28565] - An error occurs when users attempt to carry forward an engagement for the following use case:

    • The primary author removed a data formula (fx) from the prior year template.

    • The primary author then used the undo function (CTRL + Z) to undo the removal.

  • [SE-29529] - When users export XBRL documents, XBRL facts tagged with false-y tags are not included in the generated file.

  • [SE-29861] - When users print the financial statements, text with justified alignment does not display as expected.

  • [SE-29926] - In the financial statements, when users select an area in the Document Map, they are not navigated to the beginning of the area.

SE 2021-08-U10

Version: 2021-08 Build 701

Here is the list of issues that have been fixed with the deployment of SE 2021-08-U10.

  • [SE-30237] - A technical issue that causes errors when carrying forward an engagement if identical sign-off roles were added in both the prior year engagement file and the current year firm template.

  • [SE-30090] - The Risk tab in checklists does not work as expected. The names of some assertions are not displayed.

SE 2021-08-U11

Version: 2021-08 Build 707

Here is the list of issues that have been fixed with the deployment of SE 2021-08-U11.

  • [SE-30443] - When users print documents, dynamic tables that span more than one page do not display as expected in the generated PDF. After page breaks, the table headers are not formatted correctly.

  • [SE-30323] - When users print the financial statements, notes do not display as expected in the generated PDF if parent-level notes were suppressed by the firm author in the firm template. The sub-level notes for the suppressed parent notes are included in the PDF.
