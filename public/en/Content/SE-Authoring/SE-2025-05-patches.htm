SE 2025-05 patches

Here is a summary of the patches that have been deployed for SE 2025-05. The patches are listed from newest to oldest.

Note: To check your product's current version, select Information () | About in your template. The About dialog displays both the server version and the client version of your product. You can check the server version to determine whether you have the latest server patch and the client version to determine whether you have the latest client patch.

SE 2025-05-C2

Version: 2025-05 Build: SE-2025-05-C2

Here is the list of issues that have been fixed with the deployment of SE 2025-05-C2.

  • [SE-55714] - Fixed an issue where the Comparison Tool showed only a red indicator on note numbers without clearly identifying what was deleted or modified.

  • [SE-55716] - Fixed an issue where importing financial groupings into a template could fail with an Invalid tag order error.

  • [SE-55770] - Fixed an issue where totals didn’t display for calculated columns in dynamic tables when source columns were excluded from totals.

  • [SE-55784] - Fixed an issue where some engagements were missing from the consolidation page search in group audit engagements if a group audit component hadn’t yet been created.

  • [SE-55859] - Fixed an issue where guidance text was missing from the Summary of Misstatement document in certain templates.

  • [SE-55888] - Fixed an issue where bulk actions fail to apply a second signoff for a role when a first signoff is already present on the document, despite multiple signoffs being enabled.

  • [SE-55907] - Fixed an issue where formatting and content were not retained when carrying forward financial statements.
