Set up the Materiality document

You can enable the materiality document to allow users to define materiality for financial areas.

To enable the Materiality report in your product, specify the com.caseware.ca.general.enhanced-materiality in the form property in the product.json file. To learn more, see Enable or disable features in your product.

Once you enable the document in your product, you can add it in your draft and set it up.

To add the materiality document, select Add () at the documents page, then choose Materiality.

To set up the Materiality document:

  1. Add guidance information.

  2. Use the Applicable toggle () to specify which sections are included or excluded from the document.

  3. At the Settings drop-down list on the top left, modify the default settings if needed.

Edit Component performance materiality card fields

Note: The group audit feature must be enabled to use this feature.

The Component performance materiality card allows the user to assess the performance materiality of a component within a group audit directly in the Materiality form. You can edit the labels and descriptions within the Component performance materiality card. This allows customization according to different audit methodologies.
