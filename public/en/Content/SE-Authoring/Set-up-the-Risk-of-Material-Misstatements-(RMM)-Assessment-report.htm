Set up the Risk of Material Misstatements (RMM) Assessment report

Note: This topic is not applicable for Cloud apps that have the new risk module enabled.

You can enable the RMM Assessment report to allow users to assess RMM per financial area. For example, they'd review Cash and decide whether or not there is a risk that cash is materially misstated.

To enable the Risk Assessment report in your product:

  • Set the riskAssessment property to true

  • Specify the com.caseware.ca.general.risk-assessment in the form property in the product.json file. To learn more, see Enable or disable features in your product

Once you enable the report in your product, you can add it in your draft and set it up.

To add the risk assessment report, select Add () at the documents page, then choose Risk Assessment Report.

To set up the Risk Assessment report:

  1. At the Filters drop-down list on the top left, select the filter you want to apply to the document.

  2. At Settings (), modify the default settings if needed.

    3. The following options are available:

Setting

Description

Use non-consolidated data

Whether or not to use consolidated data

Show Materiality

Whether or not to show materiality (information is sourced from the materiality document)

Conclusion

Whether or not to include the Conclusion section at the end of the report

Additional column

Whether or not to include an extra column next to Reason column for each financial area

Allow assessments under Cycles or Areas

Whether or not to allow assessments under cycles or areas.

  1. At Group by drop-down list, select if you want the information in the report to be grouped or not.

    2. If you want the information grouped, you select to group by Areas or Cycles.

  2. Add guidance information.

    3. Select the text field to start adding content. The formatting toolbar displays. Use the toolbar to make the following changes:

  1. Rename the column headers to customize the terms if needed.

    2. You can rename the columns labeled RMM, Reason and Additional Information.

  2. Add a conclusion.

    3. Select the text field to start adding content. The formatting toolbar displays. Use the toolbar to format and edit the text.
