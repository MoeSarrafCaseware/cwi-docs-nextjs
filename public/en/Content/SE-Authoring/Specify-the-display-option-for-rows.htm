Specify the display option for rows

Once you add a row, you can specify the desired display option for the row in the financial statements and in the document PDF print copy.

Based on the row type, you can have three or two display options to choose from:

To specify the display option for a row:

  1. Select the table.

  2. Select More actions (), then choose Edit Dynamic Table.

    3. The table opens in edit mode.

  3. Using the check boxes on the left side, select the desired row.

  4. Select the eye icon ().

    5. A drop-down opens showing the available display options for the selected row.

    Apply the default behavior is selected by default.

    Note: If the row is an input row or a group row, the Always hide this row option is available in the list - otherwise, only Apply default behavior and Always display this row are available.

  5. Select the desired display option.

    6. Tip: If you don't want to change the current display option, clear the row check box.

  6. Select SAVE or SAVE&CLOSE.

Note that rows that will not be included in the document printout display in gray, so you can easily spot them.
