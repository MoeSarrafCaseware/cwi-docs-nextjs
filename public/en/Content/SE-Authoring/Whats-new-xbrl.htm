What's new in XBRL - August 2025

The following lists describe the improvements and fixed issues in the August 2025 XBRL updates.

XBRL Server (Ver 2.6.0.7)

Enhancements

  • [SE-57237] - Added a new endpoint to retrieve the XBRL Microservice version.

  • [SE-57224] - Reverted the Java version to Java 17 to restore compatibility with local development and UK taxonomy tests.

  • [SE-57238] - Removed deprecated, unused, and redundant XBRL functionalities and endpoints.

  • [SE-57087] - Reduced the maximum file size for gRPC snapshots from 512MB to 32MB to ensure appropriate limits for individual microservices.

Fixed issues

  • [SE-57047] - (NL only) The validate endpoint returns inconsistent responses between development and production environments.

  • [SE-57284] - Attempting to generate an iXBRL file results in an error indicating that the .xhtml file is too small.

  • [SE-57068] - Namespace handling in the ROI 102 product causes validation errors.
