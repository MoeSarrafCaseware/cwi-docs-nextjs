New in SE 2025-08

Welcome to 2025-08! This release introduces updates from our Angular migration, the ability to remove the dot leader from your table of contents and more.

On this page, you can view all the new features included in this release that apply to both authors and end users.

  • Authoring features

  • End-user features

Authoring features

Angular Migration: Cloud Engagements is moving to modern Angular

#primaryauthor #firmauthor #enterpriseauthor #enduser

Our Cloud Engagements platform is moving from legacy AngularJS to modern Angular. This update will improve long-term performance, styling consistency and set the foundation for future feature enhancements. Moving away from AngularJS reduces security risks and ensures compatibility with future Angular releases.

These updates are small first steps in a broader migration effort that will continue in future releases.

End-user features

New toggle for dot leaders in the table of contents

#primaryauthor #firmauthor #enterpriseauthor #enduser

Note: This feature is available for new and existing engagements. It does not need to be enabled.

You can now disable or enable the dot leader from the table of contents of your Financial Statements. The default setting for the table of contents includes the dot leader. A toggle for showing or hiding the dot leader can be found in Document Settings | Table of Contents Settings. The dot leader will be removed from both the UI, the print preview and PDF output.

  • Authors can set the default toggle state at the template level.

  • Users can override author settings at the engagement level.

Show Page Ranges in tables of contents

#primaryauthor #firmauthor #enterpriseauthor #enduser

Note: This feature is available for new and existing engagements. It does not need to be enabled.

You can now choose to display full page ranges (for example: “5–6” instead of just “5”) in the table of contents of your Financial Statements. This feature is available when a section spans 2 or more pages. Use the new toggle in Document Settings | Table of Contents Settings to enable this feature. When turned on:

  • Sections spanning multiple pages will show the full range.

  • Single-page sections will continue to show just one page number.

  • This change applies globally across the entire document (no need to set it individually for each section).

To learn more about table of contents within financial statements, refer to: Add table of contents in financial statements.
