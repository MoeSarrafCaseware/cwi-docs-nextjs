Workflow (how to author a checklist?)

Select Edit () to start authoring. The following steps summarize the recommended workflow for authoring checklists. You can use this workflow for guidance.

  1. Review and customize the checklist default settings

  2. Add document guidance

  3. Add groups and subgroups

  4. Add procedures and sub-procedures

  5. Review and customize procedure settings

  6. Add one or more response type to a procedure

  7. Add authoritative references in a procedure

  8. Add guidance information for a group or a procedure

  9. Add a document reference in the Guidance tab or a procedure description

  10. Rearrange procedures

  11. Add visibility settings
