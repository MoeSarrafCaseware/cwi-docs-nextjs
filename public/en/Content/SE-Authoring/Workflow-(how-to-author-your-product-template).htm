Workflow (how to author your product template?)

Your first step is to request a product, then enable the desired features in your product. Once you're done with that, you're ready to start authoring your product template.

You can use the following workflow to author your product template.

  1. Create your template

  2. Verify and customize the product settings

    In the Product Settings dialog, you can change display options or enable and disable features to customize your SE product for your local market requirements.

    To start, see Formatting and localization.

    Note: Any changes you make in the Product Settings dialog only apply to new engagements. Changes do not apply to existing engagements (that is, engagements that have been created based on older product templates).

  3. Define groupings

  4. Define the engagement glossary terms

  5. Copy content from other product templates, if applicable.

  6. Set up the built-in document types

    1. Add checklists

    2. Add queries

      Note: If you have not enabled the new query design, refer to the old query design topics for guidance on how to add queries.

    3. Add letters or memos

    4. Set up the financial statements document

    5. Set up the risk, control and risk assessment reports

    6. Set up materiality and summary of misstatements documents

  7. Develop custom forms

  8. Finalize the draft and distribute the template
