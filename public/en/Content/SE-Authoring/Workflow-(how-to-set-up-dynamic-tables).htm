Workflow (how to set up dynamic tables?)

Tables refer to the different schedules that you can include in your financial statements document.

To add a table in a financial statements area, select Add (), then choose Dynamic Table. An empty table is added in the area with a default title Dynamic Table. You can change the default title by selecting it and add a new title in the field.

Note: The default font size for content in dynamic tables is 12 and cannot be modified.

The following steps summarize the recommended workflow for setting up dynamic tables. You can use this workflow for guidance.

  • Add rows

  • Add columns

  • Create custom groups in tables

  • Exclude rows from a custom group total

  • Add a note reference in rows and columns

  • Allow users to add input rows in the table area without accessing the table's edit mode

  • Set the default display option for accounts and groups

  • Add a page break in the table

  • Copy the table to quickly create another one with similar setup
