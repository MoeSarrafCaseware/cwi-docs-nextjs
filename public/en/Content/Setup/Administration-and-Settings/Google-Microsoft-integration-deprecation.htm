Deprecation of Google & SharePoint integration

Due to the low usage of Google and SharePoint integration capabilities, we have decided to deprecate this feature. This change enables us to focus our efforts on solutions that provide greater value and long-term benefits to our entire user base. Please see below for more details.

What is changing?

With the release of Caseware Cloud 40.0, Google and SharePoint integrations will no longer be supported. The following features will be impacted:

  • Google Drive/SharePoint integration - Allows users to create and manage their files on these platforms directly from Caseware Cloud.

  • Google Calendar integration - Allows users to sync their calendar events with Caseware Cloud.

  • Google Viewer - Allows users to preview Microsoft Office documents from a web browser.

For more information, see Work with Google Drive and SharePoint files.

What is the timeline?

This change will take effect with the release of Caseware Cloud 40.0. Specific dates for each region are as follows:

  • Canada - July 16th

  • Asia–Pacific - July 22nd

  • Europe–Africa - July 22nd

  • United States - July 24th

How does this impact users?

Once the feature deprecation is complete, users will no longer be able to:

  • Access existing integrated files from Caseware Cloud

    (Note: These files can still be accessed through Google Drive and SharePoint.)

  • Create new Google or SharePoint files directly from Caseware Cloud

  • Sync Google Calendar events with Caseware Cloud

  • Preview documents in a web browser using Google Viewer

What are the alternatives?

  • Customers are advised to use New | URL Link if they wish to continue linking to files external to Caseware Cloud. Refer to the screenshot below.

  • No action is required to access existing calendar entries. They will remain available in Caseware Cloud.

Please stay tuned for more specific release dates, and reach out to us if you have any questions.
