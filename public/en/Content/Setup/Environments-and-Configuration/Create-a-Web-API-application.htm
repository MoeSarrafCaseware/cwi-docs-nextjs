Create a Web API application (ADFS)

Create a Web API application for your firm's Cloud site and define the application permissions.

To create a Web API application:

  1. In the ADFS Administration Console, click the Application Groups folder, then double-click your new application group.

  2. In the Properties dialog, click Add Application.

  3. In the Template list, select Web API. Click Next.

  4. In the Identifier field, paste the Client Identifier that you copied to a text document previously. Click Next.

  5. In the list of access control policies, select the policy that best applies to your firm. Click Next.

  6. In the Permitted scopes list, select the following scopes:

    • allatclaims

    • email

    • openid

    • profile

    Click Next.

  7. Click Next to complete the configuration.
