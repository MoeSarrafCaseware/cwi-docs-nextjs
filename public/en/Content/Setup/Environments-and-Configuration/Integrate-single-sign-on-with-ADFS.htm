Integrate single sign-on with ADFS

Overview

Before you integrate single sign-on with ADFS, ensure that:

  • You enable single sign-on for your firm.

  • The file server is using Windows Server 2016 or higher.

We recommend that single sign-on integration is performed by an administrator who is familiar with single sign-on setup in ADFS.
