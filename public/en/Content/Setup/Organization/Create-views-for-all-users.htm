Create views for all users

Using the Views feature, you can set which columns appear in each Cloud page for contacts and staff by default. Cloud comes with predefined views, but you can also create new views and assign them as needed. You can also give users the option to switch between multiple available views.

Create a new view

You can create new views from Settings.

The View is added, and can now be assigned to the desired user types.

Assign views

After you have created a view, you will need to configure what types of users can access it or have it set as a default.
