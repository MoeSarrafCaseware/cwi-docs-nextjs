New Contacts app (early access)

The new Contacts app is designed with the end user in mind. We've revamped the current version of the app by adding updated technology and improved user experience elements.

The new app will act as a strong foundation to enable efficient and faster implementation of advanced features as part of upcoming initiatives.

New user interface

The new Contacts app has a modern look and feel that aligns with our current design approach. Firms currently using the business units feature will notice that the new app is consistent with the Global Staff and Groups apps.

Besides the look and feel upgrade, the technological upgrade acts as a strong foundation for future initiatives. Front-end changes can now be implemented and deployed much faster, more efficiently and independently from major platform releases, allowing us to respond to customer feedback faster.

See details on key highlights below:

Number Description
1

Scrolling and sorting

With the new interface, users can now use the scroll bar to view the entire list of contacts. They no longer need to click through different pages. This behavior is more intuitive and eliminates a lot of extra clicks.
2

More actions

Access the More actions menu for the selected entity, where users can edit, tag or delete contacts.
3

Search

The Search bar is available at the top of the app. There has been no change in functionality.
4

Filters

The filters have also been enhanced to better suit users' needs.

As a result, for fields that can have multiple values (e.g. Tags), users can filter by more than one value and the results will include entities matching any of the selected values.

Note that all currently available filters have been retained.
5

Views

The Views menu includes built-in and custom views, as well as the option to export the current view to a CSV file. Admins and Setting Admins can access the Views Management panel in the Firm Settings to modify the views that are available to users.
6

Single and multi-selection – 2 menus

Select one or multiple entities from the boxes to the left of the entity name. The available options vary depending on how many entities are selected. For example, with one entity selected, the options mirror the More actions menu. With multiple entities selected, users can no longer edit the entity but can assign the entities to contact groups.

How can users access the new page?

The early access phase of the new Contacts app is only available to Entity Admins and higher roles. They can find the new app in the Cloud menu, among other applications. Once the early access phase is complete, this will replace the current Contacts app.

Future improvements

Bulk actions ('Select all' contacts option) - We currently support single and multi-select options for contacts. In a future release, we will consider including the option to select ALL contacts within the organization and perform actions in bulk.
