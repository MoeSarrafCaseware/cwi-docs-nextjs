Enable access to Caseware AiDA

You can enable access to Caseware AiDA for an individual user or group.

To enable access to AiDA for a user or group:

  1. Ensure you have the Staff Admin role or equivalent privileges.

  2. From the Cloud menu (), select Groups or Staff.

  1. Select the user or group for whom you want to enable AiDA.

  2. If you do not see the user or group, specify part or all of the user or group name in Search.

  3. Under the Full Name column, select the user or group.

  1. Select Edit () in the top left.

  1. Select Apps, then choose the appropriate access type.

    • No access: Do not enable access to the app for the user or group. This is the default option and does not use any licenses.

    • Full access: Enable the app for the user or group. This option uses a license for each user that receives app access.

  1. Click Save.

Note: After enabling or disabling access to an app, affected users or group members must refresh Cloud in their browser to make the changes visible in the Cloud menu.
