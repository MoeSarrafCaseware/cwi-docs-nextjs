Agregar anotaciones

Puede añadir anotaciones en el balance de sumas y saldos, listas de comprobación, ajustes o documentos PDF. Puede dar un estado a una entrada concreta, añadir comentarios o enlazar con otros documentos relevantes.

Una vez que añada una anotación, aparecerá en la página uno de los siguientes iconos para representar el propósito de la anotación:

  • Acepta ()

  • Viene de ()

  • Va a ()

  • Cálculo verificado ()

  • Suma comprobada ()

  • Enlace ()

  • Nota ()

  • Variación()

Anotar una lista de control

Puede añadir una anotación en los procedimientos y subprocedimientos de un documento de lista de comprobación.

To add annotations in a checklist:

  1. Open the engagement file and select Documents

  2. Seleccione un archivo para abrir.

  3. Go to the procedure or subprocedure that you want to annotate, then select the Add annotation icon ().

  4. Enter the information for the annotation.

    1. En Finalidad, seleccione la finalidad de la anotación.

      Note: You can select a suggestion to automatically apply the suggested annotation.

    2. Link the annotation to a document or an adjustment, if applicable. If you link the annotation to a checklist, you can select a specific procedure to link.

    3. Enter notes or comments.

  5. Seleccione Guardar.

The icon corresponding to each annotation purpose displays in the document. De vuelta a la página, si selecciona el icono o pasa el ratón por encima, aparece una ventana emergente que muestra un resumen de la anotación, la fecha de creación y el nombre de usuario del creador.

Anotar el balance de sumas y saldos (Página de datos)

Puede anotar cuentas y líneas de grupos en el balance de sumas y saldos.

Para añadir anotaciones en el balance de sumas y saldos:

  1. En la página Datos, vaya a la línea que desea anotar.

    2. Nota: Asegúrese de que la casilla Anotaciones está seleccionada en el desplegable Indicadores de la parte superior de la página.

  2. Seleccione Añadir anotación ().

  3. Introduzca los detalles de su anotación.

    1. En Finalidad, seleccione la finalidad de la anotación.

    2. En Enlace, seleccione un documento de un ajuste para enlazarlo con la anotación (si procede). If you link the annotation to a checklist, you can select a specific procedure to link.

    3. En Nota, introduzca notas o comentarios.

  4. Seleccione Guardar.

    5. El icono correspondiente al propósito de la anotación aparece junto a la línea anotada. De vuelta a la página, si selecciona el icono o pasa el ratón por encima, aparece una ventana emergente que muestra un resumen de la anotación, la fecha de creación y el nombre de usuario del creador.

Anotar un ajuste

Puede añadir una anotación para todo el asiento de regularización o para cada línea del asiento.

To add annotations in an adjustment:

  1. Vaya a la página Ajustes .

  2. Select Edit () next to the adjustment entry that you want to annotate.

    The Adjustment Details dialog opens.

  3. To annotate the entire adjustment entry, select Add Annotation () at the top right corner of the dialog and enter the annotation details.

  4. To annotate a line in the adjustment entry, go to the Details section and select Add Annotation () next to the line.

  5. Introduzca los detalles de su anotación.

    1. En Finalidad, seleccione la finalidad de la anotación.

      Note: You can select a suggestion to automatically apply the suggested annotation purpose.

    2. Link the annotation to a document or an adjustment, if applicable. If you link the annotation to a checklist, you can select a specific procedure to link.

    3. Enter notes or comments.

  6. Seleccione Guardar.

    El icono correspondiente al propósito de la anotación aparece junto a la línea anotada. De vuelta a la página, si selecciona el icono o pasa el ratón por encima, aparece una ventana emergente que muestra un resumen de la anotación, la fecha de creación y el nombre de usuario del creador.
