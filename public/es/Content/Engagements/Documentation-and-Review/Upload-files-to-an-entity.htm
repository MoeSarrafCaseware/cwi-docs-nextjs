Cargar archivos en una entidad

Puede añadir archivos y documentos a Cloud para que el material relevante para su encargo esté almacenado en una ubicación accesible. El material relevante pueden ser archivos que incluyan copias de la correspondencia con el cliente o documentos relacionados con el trabajo de encargo.

Si gestiona su trabajo fiscal en TaxPrep, también puede cargar sus archivos de TaxPrep a Cloud para realizar un seguimiento y asignar trabajo en estos archivos como lo haría con cualquier otro tipo de archivo.

El procedimiento para añadir un archivo TaxPrep es el mismo que para añadir cualquier otro archivo. Una vez cargado el archivo, Cloud creará automáticamente Otro encargo y almacenará allí el archivo TaxPrep.

Antes de empezar, asegúrese de que tiene el rol de Editor o los privilegios equivalentes. Para obtener más información, consulte Funciones incorporadas.

Para cargar archivos:

  1. Archivos

  2. Select the appropriate entity from the entity drop-down.

  3. Select New, and then choose Upload. The File Upload dialog displays.

  4. Seleccione el botón Seleccionar archivos o arrastre y suelte el archivo.

  5. In the Share Settings dialog, choose sharing options for the file and select OK.
