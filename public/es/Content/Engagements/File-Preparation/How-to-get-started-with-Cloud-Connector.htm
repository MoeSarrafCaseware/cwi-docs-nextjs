Introducción a Caseware Cloud Connector

Before you start, ensure your firm administrator has purchased and enabled a Caseware Cloud Connector license for your account.

Procedimientos iniciales

  1. If you are using Windows, make sure you have a local installation of the latest version of Microsoft Office 365 Desktop and are using the latest version of Windows 10.

    If you are using Mac, make sure you have a local installation of version 16.35 or later of Office on Mac and are using a Mac OS that supports Microsoft Office 365.

    Note: MS Office version 2019 or prior and Office Online are not supported.

  2. Open an Excel file in MS Office 365.

  3. Upload the Excel file to your engagement.

  4. Abra un archivo PDF desde la página Documentos .

  5. Get the Caseware Cloud Connector Add-in from the Microsoft Add-ins Store.

    Your firm administrator can pre-install the add-in for you. See Deploy add-ins in the admin center for instructions.

    If the add-in is not pre-installed, you can also search "WA200002239" in the store to locate the add-in and install it manually.

  6. Use the Caseware Cloud Connector Add-in to insert data from your engagement in the excel file.

    To learn more about inserting data using Caseware Cloud Connector, see the following:

    • Data linkage and realtime updates in Cloud Connector

    • Insertar datos mediante el panel derecho con Caseware Cloud Connector

    • Insertar datos mediante el cuadro de diálogo de fórmulas de MS Excel con Caseware Cloud Connector

    • Insertar datos mediante la barra de fórmulas de MS Excel con Caseware Cloud Connector
