Importar datos de AuditXML

To import client data from AuditXML 3.x (Beta) accounting software, you need to use the Caseware Transformer. To view the a complete list of supported desktop accounting tools, see Software de contabilidad de escritorio de terceros compatible.

Note: Accounts with zero balances are not automatically imported. If you want to include zero-balance accounts, you must manually add them and assign them to the designated groups.
