Comprender el conjunto de datos del balance de comprobación

When importing a trial balance from a CSV or Excel file, you must map the fields, so the trial balance data is displayed properly in the engagement. This topic describes the fields within the Trial balance dataset. For more information on importing transaction data from CSV files, see Import the trial balance from a CSV or Excel.

Campo Uso
Account number (account_id)

El identificador para la cuenta.

Valor devuelto: Cadena

Required: Yes
Account name (account_main_description)

El nombre o la descripción para la cuenta.Ejemplos:

efectivo, propiedad, planta y equipamiento, ventas

Valor devuelto: Cadena

Required: Yes
Número de grupo financiero

El número para el grupo financiero al que está asignada la cuenta en la estructura de grupos. This information is not provided by the client and is used to automatically map accounts to financial groups.

Las cuentas actualmente asignadas a grupos podrían reasignarse cuando importe este campo de datos.

Valor devuelto: Cadena

Required: No
