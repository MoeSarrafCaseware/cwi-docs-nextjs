Customize controls

Under the Controls drop-down, select Fields.

The Controls fields are organized into four categories that can be expanded and collapsed using the up and down arrows:

  • Identificación

  • Atributos

  • Diseño e implementación

  • Evaluación de la eficacia operativa

Note that the Identification category and the Title field in that category cannot be disabled.

To add and customize control fields:

  1. Select a field to include it in the controls report.

  2. For example, select to enable it in the controls report.

  3. In the Customize this label field, enter your custom term.

  4. For each field, select the On Dialog checkbox to make the field available to users in the Create New Control dialog.

  5. If the field has multiple options, enter your custom term for each option.

    For some fields, you can use the Add () button to add more custom options.
