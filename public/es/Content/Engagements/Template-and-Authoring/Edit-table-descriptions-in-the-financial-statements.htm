Editar las descripciones de los cuadros de los estados financieros

You can customize the row and column descriptions in the financial statements tables.

To edit row descriptions:

  1. Select the table.

  2. Select More actions (), then choose Edit Dynamic Table.

    3. The table opens in edit mode.

  3. Seleccione las estadísticas que desea mostrar.

  4. Edit the row description as desired.

    5. To include dynamic text, select the Data Link icon ().

    Note: If the row description is sourced from the trial balance data, the Revert icon () displays next to a modified description. Select Revert () to revert back to the trial balance description.

  5. Seleccione DONE.

  6. Select SAVE or SAVE & CLOSE.

To edit column descriptions:

  1. Select the table.

  2. Select More actions (), then choose Edit Dynamic Table.

    3. The table opens in edit mode.

  3. Seleccione las estadísticas que desea mostrar.

  4. Seleccione el campo necesario.

    5. The Data Link dialog displays.

  5. In the Output field, edit the name as desired.

    6. You can also select the Data Link () icon to have the table description include dynamic text.

  6. Seleccione DONE.

  7. Select SAVE or SAVE & CLOSE.
