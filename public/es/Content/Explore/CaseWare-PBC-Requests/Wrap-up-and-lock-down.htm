Wrap up and lock down the engagement

Now that you have collected the information and documents you need, you can wrap up and lock down the engagement.

To wrap up, download a reference copy of the engagement to share with the clients or to keep for your own records. The reference copy is a .zip file containing PDF copies of all the documents in your engagement. To download a reference copy, select Information () | Download reference copy.

To lock down the engagement, select the title to open the Engagement Properties dialog, then select Lockdown.
