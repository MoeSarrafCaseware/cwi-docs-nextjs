Print tax returns

If your clients request a copy of the return for their records, you can create a PDF file directly from Caseware ReviewComp. You can then share the file with your clients electronically or as a print copy.

You can print any tax form and customize the print settings.

Imprimir

You can print a single tax form, the tax return, client copy, or reporting documents.

To print the current tax form:

  1. Open a tax form you want to print.

  2. Select Print | Current form.

    A new tab opens with the tax form in PDF format.

To print the tax return, client copy, or reporting documents:

  1. Open a tax form.

  2. Select Print | Print return options.

    The print settings dialog opens.

  3. From the Default Print tab, select the option you want to print.

    The print confirmation dialog opens.

  4. Select Print.

Personalizadas

You can customize your print settings for the disclaimer, header and bulk print.

To customize the disclaimer:

  1. Open a tax form.

  2. Select Print | Print return options.

    The print settings dialog opens.

  3. Seleccione la pestaña Administrativo.

    The default settings are set to Yes.

  4. Change the response for the Would you like to use the default disclaimer question to No.

  5. Enter your custom disclaimer in the Text of the Disclaimer field.

  6. Change the response to the disclaimer options, if required.

To customize the header:

  1. Open a tax form.

  2. Select Print | Print return options.

    The print settings dialog opens.

  3. Select the Header Settings tab.

    The default setting is set to Yes and all options are enabled.

  4. Change the response for the Would you like to use the default print header question to No.

  5. Enter your custom header in the Please enter your customize header field.

To customize the tax return bulk print:

  1. Open a tax form.

  2. Select Print | Print return options.

    The print settings dialog opens.

  3. Select the Customize Print tab.

    The default setting is set to Customize. If you want to create a bulk print, change the response for the Select base to customize question to Bulk Print Setting.

  4. Select the tax return you want to customize, then select Customize.

  5. In the Bulk Print dialog, select ADD DOCUMENT then select the documents you want to include to print.

    You can change the print order by dragging and dropping each document.

  6. Rename the title to save your custom bulk print settings.

    Note: If you do not rename the title, your new print settings will automatically overwrite the original print settings.

  7. Select Close.

  8. In the tax form, select Print.

    The custom bulk print displays.

To remove a custom tax return bulk print:

  1. Open a tax form.

  2. Select Print | Print return options.

    The print settings dialog opens.

  3. Select the Customize Print tab.

  4. Select the tax return bulk print you want to remove, then select Customize.

  5. In the Bulk Print dialog, select the trash icon (), then Close.
