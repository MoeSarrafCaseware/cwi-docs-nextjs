Requisitos del sistema

We support using Cloud apps on machines with a minimum 8GB of RAM, but recommend using machines with a minimum of 16GB of RAM for improved performance.

Supported browsers

You can sign in to a Caseware Cloud product from any web browser using TLS v1.2 or higher, however we support and recommend using a stable release (non-beta) of one of the following:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

We recommend a minimum screen resolution of 1280 x 800 pixels and a minimum browser width of 1024 pixels.

Supported MS Office versions

Caseware Cloud products support:

  • Microsoft Office 2016 or later

  • Microsoft Office 365 (Desktop version)

Supported Working Papers versions

Integrate Working Papers with Cloud to access your client file data using a web browser. We support the latest two versions of Working Papers; however, Cloud integration is available starting from Working Papers 2017. Note that some functionality may require the most current version.
