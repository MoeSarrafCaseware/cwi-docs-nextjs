Insert a link or URL

Users can integrate links or URLs that lead to externally stored files. URLs may direct to documents within the Caseware Cloud instance or elsewhere.

To insert a link or URL to an external document:

  1. Select the Add a new annotation feature.

  2. In the dialogue, add the link to the field and a descriptive note to describe the document.

  3. Haga clic en Guardar.

The newly added link will be indicated by Link icon ().

Hover over the icon to see the URL and the note for that link.

If required, more than one link can be added.

To add an additional link, click on the Add a new annotation icon () located below the existing link icon.
