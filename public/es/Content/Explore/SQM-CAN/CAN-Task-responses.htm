Task responses

In the OPERATE OVERVIEW module, the Task Responses tab provides a summary of completed tasks. Monitoring task responses can guide in either taking action on specific responses or identifying areas for improving the company’s quality management system.

Users that require access

Access to this module is required for the following individuals:

  • Ultimate responsibility (UR) - Individual(s) with ultimate accountability for the firm’s system of quality management.

  • Operational responsibility (OR) - Individual(s) with operational responsibility for the firm’s system of quality management.

  • Independence responsibility (IND) - Individual(s) with operational responsibility for independence in the firm’s system of quality management.

  • Monitoring and remediation responsibility (MON) - Individual(s) in charge of the firm’s monitoring and remediation process

Select a task

To select a task click on Select a Task and choose the relevant task from the dropdown menu.

The table reflecting all the created records for that task will be displayed.

Users can review the responses in the table and manage outstanding items or query content by navigating to the New icon () and selecting Cloud collaboration features for Discussion or Task.
