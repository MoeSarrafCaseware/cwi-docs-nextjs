What's new in Audit International v2023.2.0 (2.665)

This version of Audit International Cloud includes enhancements to the Partner and Manager Summary document and additional platform functionality.

Here’s what’s new in Audit International Cloud v2023.2.0 (2.665).

Content updates

Resumen de socios y gerentes

In Audit International 2023.1.0 (2.660) a new Partner and manager summary document was added, which gathers key engagement information from across multiple phases of the engagement such as Planning, Risk Assessment and Response, Fieldwork and Completion.

In Audit International 2023.2.0 (2.665) the following enhancements have been added to the Partner and manager summary:

  • A new widget was added to the Fieldwork tab displaying the number of nonoptimal answers in the documents available in the Fieldwork phase.

  • In the Conclusion tab:

    • A new widget displaying engagement documents for which the visibility conditions have been manually overridden; and

    • A selection to the Adjustments widget that enables the user to display or hide prior year adjustments.

Create engagements based on other engagements

Users can now create new engagement files based on existing files. The following information will be copied from the source engagement:

  • Customizations to grouping structures

    Customizations to queries

  • External documents added to the engagement, such as PDFs and Microsoft Office files

  • Customizations and responses to checklists

  • Personalizar el contenido de los estados financieros

  • configuración de visibilidad

  • Risks, controls and risk assessments

  • Areas added to the engagement settings

  • Signoff schemes and roles.
