Novedades - Working Papers 6,0

Find out what's new in Caseware Sherlock 6.0.

userv2 and issuev2 datasets

We've introduced two new datasets — userv2 and issuev2. These datasets function similarly to the existing user and issue datasets, but with users' identities decrypted. This means that visualizations that would previously display an encrypted GUID to hide users' names (with user or issue) can now display their full names (with userv2 or issuev2).

Synchronization with Cloud user groups

To streamline the process of sharing visualizations, Sherlock now synchronizes with Cloud user groups. This enables Sherlock Builder users to share their visualizations in bulk with entire Cloud groups, rather than manually selecting individual users.
