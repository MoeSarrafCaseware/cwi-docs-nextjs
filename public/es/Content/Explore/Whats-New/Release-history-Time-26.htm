Novedades - Working Papers 26,0

Here is a listing of what's new for version 26.0 of CasewareCloud Time.

Ability to export invoices to QuickBooks Online

Administrators can now link Cloud with an existing QuickBooks Online (QBO) account. Invoices can then be exported directly to QBO to track payments and manage receivables. To learn more about exporting invoices, see Integrate Cloud with QuickBooks Online.

Ability to disable GL features

Administrators can now disable general ledger (GL) features. This ensures that only necessary information is collected during the billing process. If an organization has never used GL features, GL will be disabled during migration to 26.0. New organizations will have GL features disabled by default. Administrators can enable or disable GL features by selecting Firm Settings | Time | General | Time and Expense Settings | Enable general ledger.

Billing page improvements

A number of improvements have been made to the Billing page, including:

  • an option to send invoices directly from the Invoice Wizard,

  • search and filter functionality on the WIP tab and

  • the ability to revert the status of invoices from Sent to Client to Draft.

General improvements

Several improvements have been made for the latest release.

Análisis

  • Fixed an issue where certain Analysis reports did not display values in the Type column correctly.

  • Fixed an issue where some sorting options were not available for certain Analysis reports.

  • Fixed an issue where some Time Summary reports would export with incorrect formatting.

Carlos

  • Fixed an issue where users in certain time zones could not set an invoice billing date to the first day of the active period.

  • Fixed an issue where invoices were deselected after being sorted on the Invoices tab.

Firm Settings

  • Fixed an issue where inactive work codes could be marked as active.

  • Fixed an issue where enabling SharePoint could affect Time app permissions.
