Cambiar nombre, eliminar o sustituir archivos

After you create a file, you can rename, delete or replace it, as needed.

Before you start, navigate to Cloud menu () | Files and select the appropriate entity using the entity drop-down.

Cambiar nombre de archivo

You can edit file names and descriptions to better reflect the contents of the file.

Note: If you want to rename a Working Papers file that you have published to Cloud, we recommend you convert the file to a non-sync state and rename it using Working Papers, rather than renaming the file in Cloud. For more information, see Rename a File.

To edit a file name and description:

  1. Select the file you want to edit using the check box.

  2. In the file details pane, select Edit ().

  3. Enter a new name and description for the file, then select Save.

Eliminar archivo

If you no longer require a file, you can delete it.

¿Eliminar archivo?

  1. Seleccione el proyecto que desea eliminar.

  2. In the file details pane, select More actions () | Delete.

  3. Confirm deletion of the files.

If you remove a file by mistake, recover it from the Recycle Bin. See Recover a previous version or deleted file for more information.

Reemplazar archivo

If you have a newer version of a file you want to upload, you can replace the original file.

Note: If you want to replace a Working Papers file that you have published to Cloud, you must first delete the Cloud copy, then upload the new copy from your local computer.

To replace a file:

  1. Seleccione las aplicaciones que desea instalar.

  2. In the file details pane, select More actions () | Replace.

  3. Seleccione el botón Seleccionar archivos o arrastre y suelte el archivo.

Delete previous versions of a file

If you have replaced a file and no longer need the old versions of the file, you can delete them.

Warning: If you delete a previous version, you will not be able to recover it later.

Para restaurar una versión anterior de un documento:

  1. Select the file that has previous versions you want to delete.

  2. In the file details pane, expand the Previous Versions section.

  3. Select More actions () for the version you want to delete, and then select Delete.

  4. In the confirmation dialog, select Yes.

You can also delete all previous versions of a file at once.

To delete all previous versions of a file:

  1. Select the file that has previous versions you want to delete.

  2. In the file details pane, select More actions () | Delete All Old Versions.

  3. In the confirmation dialog, select Yes.
