Establecer la retención de archivos

You can set retention on files to remove any unwanted items from the system within a certain number of days, months, or years. This helps ensure that next year's files contain only finalized items and reports.

Antes de empezar, asegúrese de que tiene el rol de Editor o los privilegios equivalentes.

To set file retention period:

  1. From the Cloud menu (), select Files.

  2. Select the appropriate entity from the entity drop-down.

  3. Select the file you want to set the retention for using the check box.

  4. Select More Actions () | Set Retention.

  5. Select the retention start date, how long to retain the file and any comments.

  6. Select OK to confirm. The file details pane is updated with the retention status information.
