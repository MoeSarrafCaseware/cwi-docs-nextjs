An engagement file is where you collect, organize and review your client's data.

See Create an engagement file for more information.

To create a AnalyticsAI CAN engagement:

  1. From the Cloud menu (), select AnalyticsAI CAN ().
  2. Select the entity that you will be auditing from the drop-down menu, for example:

  3. Select New ().
  4. Enter a Name for the engagement, enter any other optional information, then select Save.
    The engagement properties display.

  5. Edit as desired and select Save.
    La página Proyectos muestra lo siguiente:

    Note: The Year end date selected must match what is in the trial balance and transactions files.

The Documents page contains links to documents for these stages of the engagement:

  • Planning – gather required information and data for the engagement.
  • Risk Assessment – check data for reliability and completeness, adjust influencers, run risk analysis and perform analytical review.
  • Conclusion – upload supporting documents here.
    Note: The Conclusion section is not visible initially. It will become visible when you complete the Optimiser file in the Planning section.
