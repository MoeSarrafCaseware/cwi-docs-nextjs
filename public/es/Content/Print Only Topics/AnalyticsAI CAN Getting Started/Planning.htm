Planificación

In this stage, you will complete the following:

  • 0-0 Optimiser
  • 0-1 Client queries
  • 0-2 Import trial balance & online transactions
  • 0-3 Import transactions CSV - If you import online transactions using the above document, you won't need to import a CSV.

Optimiser

The 0-0 Optimiser file allows you to:

  • Decide whether to use the Client queries file to request information from your client.
  • Select sections you would like to include in the engagement.

Based on your responses in this file, Caseware AnalyticsAI CAN automatically populates the engagement workspace with the files you need to complete the engagement.

Consultas

Use 0-1 Client queries to gather required information for the engagement. You can also use this document to request your client's trial balance and transactions data.

See Request information from a client for more information.

Importaciones

The trial balance and transaction data can be imported from Excel or CSV files, other desktop accounting software and online accounting platforms. You can also import trial balance data from an existing Caseware Working Papers file. See Import a client's trial balance in the Data page for more information.

If you've imported the client's trial balance data from a CSV file, you'll need to import the transactions CSV to store and run analyses on your data.

To import transactions from a CSV file:

  1. Select the 0-3 Import transactions CSV file from the Documents page.
  2. Select the transactions CSV file from your local directory.
  3. Map the columns in the transactions file.
  4. Seleccione IMPORT.

See Import the client's transactions for detailed instructions.
