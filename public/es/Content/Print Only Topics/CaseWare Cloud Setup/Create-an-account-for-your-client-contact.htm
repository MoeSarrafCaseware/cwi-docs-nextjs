Crear una cuenta para el personal o los contactos

One of the major benefits of Caseware Cloud is that you can collaborate directly with your clients on engagements. You need to set up a contact account so your clients can access their engagement file.

Practice set

From the Cloud menu, select Contacts. This takes you to the Contacts page.

Select New, then ADD | Contacts. The Add Contact dialog displays.

General

Enter a first and last name and an email address for your client contact. In the Entity field, select the client entity that you created earlier. Once you have selected the client entity, select the Contact - Entity Collaborator role to set the access level for this account. This role allows your client to create discussions, tasks, and file requests on Cloud. It also allows them to collaborate on engagements for Cloud apps.

Contraseña

The same password options for staff accounts apply here. Because the email address is yours, choose Set and uncheck User must change password on next sign in.

Grupos

Contact groups work the same way as staff groups. The contact group available out of the box is All Contacts and by default, your client is automatically a member of that group.

Apps

Just like the All Staff group, the All Contacts group grants its members the ability to view and create engagements for all the available Cloud apps.

Once you finish setting up your client contact's new account, select Save. They will receive a welcome email from Cloud notifying them that their account has been created and instructing them on how to log in.
