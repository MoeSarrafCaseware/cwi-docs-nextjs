Perform interim engagements

In an interim engagement, users can perform monthly, quarterly and semi-annual interim engagements, including comparative periods. However, they can only perform one type of engagement per interim period throughout the year. For example, they can only perform a review engagement in a specific interim period.

If a different type of engagement is performed at year end, users need to set up a new engagement for that period. One engagement letter is created in the first period file and covers all interim engagements. However, multiple reports and financial statements are created for each period. The signed engagement letter will roll forward during each covered period.

When importing from a CSV or an Excel file, users can define the periods during the import.
