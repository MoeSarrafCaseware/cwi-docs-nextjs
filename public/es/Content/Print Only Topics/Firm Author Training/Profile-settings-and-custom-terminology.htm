Profile settings and custom terminology

Before we dive into firm authoring, we have to populate a few areas from the Cloud menu, such as your profile and the custom terminology pertaining to the US market.

Practice set

  1. Select Cloud menu () | Settings.

  2. Under Organization, select Profile.

  3. Fill in the Profile, for example with your firm information, then select Save.

  4. From Customization | Terminology, enter the state in the Province field, the zip code in the Postal Code field, and select Save.
