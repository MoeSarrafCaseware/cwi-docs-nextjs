Start a client engagement

At the beginning of a new client engagement, you will need to create a new OnPoint AnalyticsAI engagement file. Engagement files are created and hosted on Caseware Cloud. For more information about Caseware Cloud, visit https://www.casewarecloud.com.

This section covers the following tasks:

  • Creating a new engagement file
  • Setting the engagement properties

Create an engagement file

To start working on a client engagement, you need to create an engagement file. In Caseware Cloud, you can create your engagement file in the client entity. You can also set key dates and budgets and assign staff members to the file.

Practice set

  1. From the Cloud menu, select OnPoint AnalyticsAI.
  2. Select New.
  3. In the OnPoint AnalyticsAI dialog, choose your client entity and enter the name for the engagement file.
    You can also select the start date and end date for your engagement and enter budget information.
  4. In the Preparer field, enter your first name to search for your account. Select your account to assign yourself as the preparer.
  5. In the Reviewer field, enter part of another staff member's name. Select the account to assign the staff as the reviewer.
  6. At the bottom of the OnPoint AnalyticsAI dialog, select Save.

You have now created your new engagement, and you have assigned staff in the firm to prepare the engagement and review it. When you select Save, the engagement properties dialog automatically opens in your browser.

Note: If you have a popup blocker set, click the new engagement name to open the engagement. You may wish to use the browser settings to turn off the pop-up blocker.

Set the engagement properties

When you open an engagement file for the first time, a pop-up with information about the engagement file displays. Use this to set some high-level engagement properties and perform some actions.

In this view, you can:

  • Set the Workflow stage - you can select a color-coded workflow stage to indicate the status of your engagement work. Set the Workflow stage to In Progress.
  • Enter engagement details - you can enter a Job number, change the Operating name, and select a different Year end date. Leave the Reporting period set to Annually for this exercise.
  • Lockdown the file- you can lock down the engagement file once all engagement work is finalized.
