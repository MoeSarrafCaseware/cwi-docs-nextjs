Descripción general

The purpose of this guide is to help firm authors and end users learn how they can customize their financial statements at either the firm template or engagement level.

The roles and limitations between firm authors and end users vary when modifying primary author content.

Autor de la empresa

  • Firm authors cannot modify primary author content at the template level.
  • If firm authors need to modify primary author content, reports or notes, they can add a new financial statement area, then copy the primary author content to it.
  • If firm authors need to modify the primary author’s dynamic tables, they can add a new financial statement area, then copy the primary author’s dynamic table to it.
  • Firm authors can suppress primary author content that is irrelevant to the template, including copied and modified content.

End users

  • End users can modify primary author content at the engagement level.

Setup

You must first perform the initial setup of a new OnPoint PCR engagement file and its accompanying financial statement document.

To set up the engagement file:

  1. Create a new OnPoint PCR engagement file.
  2. Complete the 1-100 Engagement set up document and one of the following engagement acceptance and conclusion documents:
    • 1-115 Preparation
    • 1-120 Compilation
    • 1-125 Review
  3. Import the current and prior year trial balance data to the file.
  4. Map the accounts to the appropriate financial statement groupings and ensure that there is no “out of balance” indication on the Data tab.

To set up the financial statement:

  1. Complete the Financial Statements and Letters section of the 1-100 Engagement set up document:
  2. Complete the 4-200 Financial Statements set up document.
  3. The criteria from both completed documents flows directly through to the financial statements.

Review the template document

To simplify financial statement preparation, OnPoint PCR includes a built-in template document — 4-250 Financial Statements and Report — with most of the document’s content pre-populated based on your answers to the questions in the documents 1-100 Engagement set up and 4-200 Financial Statements set up.

  1. Abrir este documento
  2. Note that each financial statement document should include, at minimum, the following sections (exact names may vary):
    • Cover Page
    • Contenido
    • Independent Accountant’s Compilation Report or Review Report
    • Balance general
    • Estado de resultados
    • Statement of Stockholders’ Equity
    • Cash Flow Statement
  3. Navigate to the Independent Accountant’s Compilation Report or Review Report area.
  4. Select the Guidance icon () to view more details.

Copiar tabla dinámica

  1. In the Document Map, select Income Statement.
  2. Select the table, then select Select More actions () | Copy dynamic table.
  3. In the area where you want to place the copied table, Add () | Paste copied dynamic table.
  4. Select More actions () | Edit print settings and set the page size to Landscape layout. Seleccione Guardar.

Copy a text section

  1. In the Document Map, select the Independent Accountant’s Compilation Report or Review Report.
  2. Highlight the first paragraph and copy it by pressing CTRL + C on your keyboard.
  3. Above the first paragraph, select Add () | Text Section.
  4. In the new text section, paste the copied paragraph by pressing CTRL + P on your keyboard.
  5. Note that all text, placeholders and formulas from the copied paragraph are copied over to the new text section.

Suppress primary author content

  1. Next to the content that you want to suppress, toggle Suppress to the “On” position ().

Copy primary author visibility conditions

  1. Next to the primary author content where you want to copy visibility conditions, select More actions () | Edit visibility.
  2. Seleccione Copiar.
  3. In the destination text section or dynamic table, select More actions () | Edit visibility.
  4. Select Paste, then select Save.

Crear nota

Assume you want to create a note titled “PROPERTY AND EQUIPMENT” that displays a dynamic table with the current and prior year balances for Machinery and Equipment, Furniture and Fixtures, and Leasehold Improvements.

  1. In the Document Map, select Notes to the Financial statements.
  2. Under NOTES TO THE FINANCIAL STATEMENTS, select Add () | Create Note.
  3. Select the Note title field and enter “PROPERTY AND EQUIPMENT”.
  4. Under the note title, select Add () | Text Section.
  5. In the text section, enter “Property and equipment at June 30, 2017 consisted of:”
  6. Under the text section, select Add () | Dynamic Table.
  7. Next to the dynamic table, select More actions () | Edit dynamic table.
  8. On the table row, select Add () | Trial balance data.
  9. Enter the word “Property” to filter the list. Select the Property, Plant and Equipment group and add it to the row.
  10. At the end of the line, select the Expand lines icon ().
  11. At the top of the page, select Save & close.
  12. The note and its content are now visible at the bottom of the financial statements document.

Configurar la biblioteca de notas

  1. Create a note and add the note content.
  2. (Optional) Create a note hierarchy by adding subheadings to the note:
    1. In the Document Map, select the note where you want to create the note hierarchy.
    2. Under the note title, select Add (), then in the drop-down, select the applicable heading format.
    3. The subheading is added with a default title. Select the default title to change it.
  3. You can also further categorize notes by associating them with a trial balance group:
    1. At the top of the note next to the note title, select More actions () | Associations.
    2. Next to FINANCIAL GROUPS, select Add ().
    3. Select the trial balance group to associate the note with.
    4. Seleccione Guardar.

  4. Exclude a note to store it in the Notes Library:

    • To automatically exclude a note from the document, set up visibility conditions for each note. When the conditions are met, the note is excluded and stored in the Notes Library.
    • To manually exclude a note from the document, select it, then select More actions () | Exclude.

    In the Document Map, if you select Add (), there will now be the option Insert from Library. All newly created OnPoint PCR engagements will include this option as long as there is at least one excluded note.

Cross reference notes with line items

Assume you want to provide a breakdown of the Property, Plant and Equipment group, but you do not want to display it on the balance sheet.

  1. In the Document Map, select Balance Sheet.
  2. Next to the dynamic table, select More actions () | Edit dynamic table.
  3. Above the empty column, select Add ().
  4. In the Type drop-down, select Note reference.
  5. Select Save & close.
  6. On the Property, Plant and Equipment Note line, select # then select the PROPERTY AND EQUIPMENT note (from Create a note).

Replace placeholder text

  1. In the Document Map, select the Independent Accountant’s Compilation Report or Review Report.
  2. Placeholder text displays in a blue, highlighted font, or as a drop-down for a date, staff or contact name. Select a placeholder to replace its content.

Editar tabla

While most of the tables in the financial statements are populated automatically from trial balance data, you can add more information to them by inserting your own input rows. The values that you enter into your input rows are included in the final calculations for the table.

  1. In the Document Map, select Cash Flow Statement.
  2. Next to the dynamic table, select More actions () | Edit dynamic table.
  3. Between the rows where you want to add an input row, select Add () | Input row.
  4. Add financial data to the input row.
  5. Select Save & close. The row now displays in the Cash Flow Statement.

Note: Input rows with no values display with empty column values.

Añadir o editar un cálculo en celdas de tabla en los estados financieros

  1. In the Document Map, select Balance Sheet.
  2. Next to the dynamic table, select More actions () | Edit dynamic table.
  3. Place your cursor in the input row or input column cell where you want to add or edit the calculation. Seleccione el icono XBRL ().
  4. In the Calculation menu:

    • Select Insert data () to insert a trial balance data link in the formula.

      By default, the Financial statement is selected as the source for trial balance data. You can change this to the Trial balance if required, which sources data from the Final column in the Data page. As you insert data links, the or prefix indicates the data source.

      If forms are available in your product, you can insert values from the Form Value tab.

    • Select Insert a number () to insert numbers in the formula, or use your keyboard.
    • Select any of the applicable operators (+, -, x, /, (, )) to build the formula.
  5. Seleccionar Listo. If the formula is invalid, the fx icon displays in red ().
  6. Back in the document, you can review the formulas you’ve added by selecting the cell to view the calculation details. Select Revert () to undo an overridden calculation.

Expand a table line

While most of the financial statement tables have the accounts (lines) collapsed into their respective groups, you can display more details about areas of the engagement file by expanding them.

  1. In the Document Map, select Balance Sheet.
  2. Assets, Noncurrent displays totals for two areas: Property, Plant and Equipment (net) and Long-term Investments (net). Next to the Property, Plant and Equipment (net) group name, select the Expand/hide lines icon ().
  3. All of the lines under your selected group display, including the amounts for each different category under Property, Plant and Equipment, Accumulated Depreciation, Depletion and Amortization. You can expand these categories further by selecting Expand/hide lines icon (), and collapse them by selecting the icon again.

Notas:

  • Select a line to preview its adjustments, preliminary, final and prior year balance details.
  • To change the description for a table line, select the row and modify the description. Rows sourced from trial balance data will display the Revert () icon after you change the description. Select it to discard your changes and return to the original description.

Review changes

  1. At the top of the Document Map, select Toggle review tools () and toggle Review tools to the “On” position ().

    Note: If the Toggle review tools icon displays a red dot (), there are balance discrepancies in the document.

  2. Enable the tools that you want to view on the page. The number of changes in each financial statements area displays next to the tool’s icon.
    • New content () - Content added by staff members.
    • Excluded content () - Content excluded by staff members.
    • Text area changes () - Modifications by staff members to product-default text sections.
    • Dynamic Table changes () - Modifications by staff members to product-default dynamic tables.
    • Overwritten balances () - Rows with overwritten balances.
    • Balance check () - Rows that have been compared.
  3. Select an area to review the changes. Changes in the area are marked with the tool’s icon. Rows with disagreeing balances are marked with a stop sign () while rows with agreeing balances are marked with a green checkmark ().

Imprimir el documento de estados financieros

  1. At the top of the Document Map, select Settings () | Document print settings.
  2. Customize the print settings as required.
    1. Enter header and footer content for the document.
    2. Set the page size, orientation and margins.
    3. Configure the table of contents.
    4. Seleccione Guardar.
  3. (Optional) At the top of the Document Map, select Settings () | Notes print settings. Customize the print settings for notes as required.
  4. At the top of the Document Map, select Print () | Download as PDF.
  5. The PDF displays in a new tab in your browser. Select Download () to save the PDF to your computer.
