Imprimir el documento de estados financieros

To conclude your engagement, you create the financial statements document for the client where you explain your findings and present a conclusion. OnPoint PCR has a built-in template that makes it easier for you to prepare the financial statements.

Practice set

  1. On the Documents page, under the Financial Statements and Wrap-up section, open the 3-150 Financial Statements and Report document.

  2. Navigate to the Independent Accountant's Review Report area.

  3. Select the guidance () button to view a detailed guidance section.

  4. Select the Review Report Required Elements Matrix link in the guidance to open a new page in your browser. The report consists of a table that lists each section of the financial statements and indicates:

    • when is each section included in the report

    • what elements are required in each section

    • the AR-C references for each section

There are also various guidance sections throughout the document to help you complete some of the required sections.

Each financial statements document includes the following sections:

  • Cover page

  • Contenido

  • Independent Accountant's Review Report

  • Balance general

  • Ajustes a los estados financieros

    • Statement of Comprehensive Income

    • Statement of Changes in Owner's Equity

    • Estado de flujo de fondos

  • Notas a los estados financieros

The Notes to the Financial Statements section is the last section in the financial statements document. If you need to provide any supplementary information about the results of your engagement, you can add that information in this section.

A note can have a guidance section with information on any changes that you might need to make to it as the preparer. The content of a note can be:

  • text section - this is a text field where you can add supplementary information in the form of one or more paragraphs.

  • dynamic table - this is a table that consists of selected lines from the trial balance or custom rows where you can input your own data to support your conclusions.

  • page break - to insert a page break within the note content.

Personalizar el contenido de los estados financieros

Most of the content in the financial statements document is already populated for you. The content in the review report is based on your answers to the questions in the 1-110 Engagement Setup document. Groups and accounts on the financial statements tables are sourced from the trial balance.

However, you may need to customize the financial statements document for your client. In this training session, we will make the following changes to the financial statements document.

Practice set 1 - Create a note

Let's create a note in the Notes to the financial statements area.

The note should be titled "PROPERTY AND EQUIPMENT" and displays a dynamic table with the current and prior year balances for Machinery and Equipment, Furniture and Fixtures, and Leasehold Improvements.

Crear una nota en los estados financieros

  1. Go to the Notes to the Financial Statements area on the Document Map.

  2. Select (BLANK) and enter PROPERTY AND EQUIPMENT as the new title for your note.

New notes are empty by default, so you need to add content areas to the note to include information.

Add a text area to your note to introduce the dynamic table:

  1. Under the note, select Add () | Text Section.

  2. En el campo Ecuación, escriba lo siguiente:

    Property and equipment at June 30, 2017 consisted of:

You have added a description for the contents of the new note.

To add the balance information for the property and equipment account groups, you need to add a dynamic table.

  1. At the text section, select Add () | Dynamic Table.

  2. Select the dynamic table to display the More actions icon ().

  3. Select More actions () | Edit Dynamic Table.

    The table opens in edit mode.

  4. Enter the word property to filter the list. Select the Property, Plant and Equipment group (with the 45,000 balance) to add it to the row.

  5. At the end of the line, select the Expand lines icon ().

  6. At the top of the page, select SAVE & CLOSE.

Note: In a dynamic table edit mode, you can undo or redo actions, show excluded items, expanded or collapsed groups or turn on auto-save.

You have added a dynamic table to your note. When you scroll to the bottom of the financial statements document, you should see the note with its newly added content.

Practice set 2 - Cross reference a note with a line in a financial statement table

You can cross reference a line in a table to one of the notes to the financial statements that holds related details or information.

For example, you want to provide a breakdown of the Property, Plant and Equipment group, but you do not want to display it on the balance sheet, so you link a line in the balance sheet to the PROPERTY AND EQUIPMENT note that you created earlier.

To show the Notes column in the financial statement table:

  1. Select the Balance Sheet area at the Document Map.

  2. Select the table, then select More actions () | Edit Dynamic Table.

    3. The table opens in edit mode.

  3. Select Add () above the empty column.

  4. Select Note reference from the Type drop-down, then select SAVE & CLOSE.

A Note column now appears in the Balance Sheet.

At the Property, Plant and Equipment, net line, select the # to display a pop-up with the name of the note to the financial statements that you just created. Select the checkbox beside 1. PROPERTY AND EQUIPMENT to link the line with the note.

Practice set 3 - Replace the placeholder text

The review report includes placeholders for information that you need to add. Placeholders for input text are displayed in blue highlighted font. There are also placeholders in the form of drop-downs where you select a date, staff or contact name.

Select a placeholder to replace it with real content appropriate for your client and the engagement. Also select values for any placeholder drop-downs.

Practice set 4 - Edit a financial statement table

While you cannot change the specific accounts and groups that are included in a financial statement, you can make some changes to the displayed content.

Most of the tables in the financial statements are populated automatically from the trial balance data, to make the financial statement preparation easier for you. However, you can add more information to the table

You can add rows in a financial statement table by adding input rows. The amounts that you enter in input rows are included in the final calculations for the table.

For example, scroll to the Statement of Cash Flows area in the Document Map.

Note: The Statement of Cash Flows is only available if you have answered Indirect method to Which method will be used for reporting cash flows? in document 1-110 Engagement Setup.

Input rows with no values are displayed with empty column values. Once you enter any financial data into an input row, that row is displayed in the statement of cash flows.

Note: If you want to enter a different amount in a row that already has automatic values, select the Settings () at the end of the row, then select Override calculated balances.

Practice set 5 - Add or edit a calculation in table cells

In the 3-150 Financial Statements and Report file, you can add or edit calculations in table cells. The calculation can include numbers, data sourced from the trial balance, and the financial statements, as well as form values (if forms are available in your product) or a combination of all.

Note: This feature is only available for cells in columns or rows with type Input.

Practice set 6 - Expand a line in the financial statements

Most of the financial statement tables have the accounts (lines) collapsed into their respective groups by default. You can expand lines in the financial statement to display more details about some areas of the engagement file.

For example, go to the Balance Sheet in the financial statements.

Assets, Noncurrent displays totals for two areas: Property, Plant and Equipment, net and Long-term Investments, net. Now, you want to give the client more information about the Property, Plant and Equipment, net group. Select the Expand/hide lines icon () beside the group name.

All of the lines under your selected group are now displayed, and you can see the amounts for each different category under Property, Plant and Equipment, along with the amount for Accumulated Depreciation, Depletion and Amortization.

All of the lines under your selected group are now displayed, and you can see the amounts for Automobiles - cost, Property, Plant and Equipment and Accumulated Depreciation, Depletion and Amortization. Select the Expand/hide lines icon () next to the Property, Plant and Equipment subgroup to display all of the detail lines that make up the group.

To collapse a subgroup, select Expand/hide lines icon () again.

Note: You can also view the details of a balance in the financial statement by selecting the line. A popup displays showing details for preliminary, adjustments, final and prior year for that balance.

You can change the descriptions for the financial statement lines. To change the description for lines , select the row and modify the description.

Note that rows sourced from trial balance data display the Revert () icon once you change the description. You can select Revert () to discard your changes and display the original description.

Print document copies

Once you have finished preparing the client's financial statements and your reviewer has signed off on the document, you can print a copy of the financial statements to present to the client. You might also print other copies of the document while you are still in the process of working on it in order to review the impact your changes have on the finished product.

You cannot print directly from OnPoint PCR as the print action creates a PDF copy of the file which you can print or share as required. To adjust how the financial statement looks once converted, modify the Print Settings.

Practice set 1 - Print the financial statements

Select Settings () | Document Print Settings to display the Print Settings dialog. You can also select Notes Print Settings to customize the print options for the notes to the financial statements area.

You can enter content for the header and footer of your document, and you can also set the page size and orientation, the margin width, letterhead options and the table of contents settings. If you make any changes, select Save to close the dialog.

Select Print() to generate a PDF copy of the Financial Statements. This automatically opens a new tab in your browser window, and the PDF version of the document is displayed. You can choose to save the file as a PDF to your local hard drive, or you can select the Change button and find a local printer to print a physical copy.
