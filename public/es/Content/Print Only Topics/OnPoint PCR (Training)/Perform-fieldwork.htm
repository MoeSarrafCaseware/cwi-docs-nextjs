Complete the materiality document

The materiality document is a form where you can define thresholds for overall materiality, performance and trivial misstatements to keep track of benchmarks.

Practice set

  1. Open the 2-102 Materiality document.

    If you want to view prior year balances, select Settings then, select the Show prior year balances check box.

  2. At the Principal Users of the Financial Statements section, enter the principal users and any factors to consider. You can select + Add row to add more principal users to the document.

  3. At the Materiality at the Financial Statement Level section, enter one or more factors to determine materiality. You can add two basis types:

    • Financial materiality basis - select a group from the trial balance and enter a benchmark percentage.

    • Other basis - enter a custom basis description and amount, and a benchmark percentage.

  4. At the Trivial misstatement section, enter a maximum amount for trivial misstatements.

Respond to work program procedures

On the Documents page there is a series of checklist documents under the Fieldwork section. These checklists are lead sheet documents (or work programs) that you will use to guide your analysis and document the results of your inquiries for the relevant financial statement areas.

Work programs have several features that help you complete them easily and quickly:

  • linked procedure responses

  • an analysis section

  • document guidance

At the top of each document, there is an Analysis tab that pulls information from the trial balance and presents it visually. A work program also includes a Guidance tab.

Practice set

Open the Cash and Cash Equivalents document.

Note: You must answer Yes in question 5 (Cash and Cash Equivalents?) in the 2-100 Review - Overall Analysis document for the Cash and Cash Equivalents document to be available.

Switch to the Analysis tab. The analysis section presents data in tables or graphs. It only includes trial balance information for relevant accounts and groups which helps you identify items that you need to inquire about from your client.

You can view either preliminary or final balances using the drop-down menu at the top of the tab. The toggles allow you to choose which information (such as Accounts) shows on the tab. Each figure also has a text field where you can add comments about the data displayed.

Switch to the Guidance tab to view the available guidance information.

Now go back to Procedures tab.

The Procedures tab contains questions to be answered or fields to fill in. If you are using the 2-106 PBC Requests- Review Inquiries query document to gather information from your client, you may see linked query responses under some procedures.

Linked query responses are response areas that are linked to matching questions in the PBC Requests - Review Inquiries query document. When you accept a response from a client in this query, the linked query response is displayed under the matching procedure in the checklist document.

Request additional information from the client

You are working on the 2-115 Cash and Cash Equivalents document, and you reach a procedure that asks What procedures are in place to ensure that all cash and cash equivalents accounts are reconciled on a timely basis?

In the analysis section, you noticed that the cash total for this year is much higher than the totals from the prior years. You decide to contact the customer to ask them to verify the response to this question, since a late cutoff could have inflated the current year total.

Practice set

From the Documents page, open the 2-106 PBC Requests - Review Inquiries document.

Note: For the purpose of this training, the Review Inquiries document includes linked queries for many different documents. In real life, you should identify all the questions where you need client input and delete all other questions from the query before you send it to the client.

Select SEND to display the Open Query dialog, and complete the dialog as required.

The client can enter information for the specific question and mark it as completed. You will see this response in the PBC Requests - Review Inquiries document. If you are satisfied with the information they have provided, then you can select Accept.

Notice how the time of the response and the name of the contact that has responded to each query question displays in the query document.

Once you have accepted the client response in the query, you should see the response in the linked procedure in the Checklist document.

Once you're satisfied with the information your client has provided and if procedure sign offs are available, you can sign off the procedure by selecting the Sign Off icon () next to it.

Access and modify Microsoft Word or Excel files

If a member of the engagement team has attached files from Microsoft Word or Microsoft Excel to your engagement, they're added to the list of documents in the Documents page.

Practice set

Select a file to open it to review the contents or make changes. If you enable editing in the Word or Excel file copy on your desktop, the file is locked for other users. If other members of the engagement team go to the Documents page and open the file, it opens in read-only mode. Once you've closed the file, other users can edit.

When a file is locked by a user, a lock icon with the user's initials shows on the Documents page next to the file.

Add annotations to PDF documents

If other members of your team attach a PDF document to the engagement, it is added to the document list in the Documents page.

Practice set

Select the PDF file to open it for review purposes or to add annotations. For example, you can add annotations to note work that you have performed or identify problems or potential inaccuracies.

To add annotations in the PDF file, go to where you want to add your annotation and select to display the Add Annotation icon (). Observe lo siguiente:

  • If you select anywhere inside the text, the icon shows on the right side at the end of the line.

  • If you select anywhere in the white space, the icon shows at the selected spot.

Select the icon to add an annotation at the indicated spot. From the dialog that displays, choose a Purpose, Link the annotation to a document or adjustment (if applicable) and enter a Note.

Haga clic en Guardar para agregar el gráfico al panel de información. The Annotation icon () displays in the document to indicate the annotation type. Other engagement team members can view the annotation by hovering their cursor over it or by selecting it.
