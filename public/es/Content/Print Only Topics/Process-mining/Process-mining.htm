Minería de procesos

Process Mining analyzes the transactions in a Cloud engagement. The initial version analyzes data related to the Purchases, Payables, Payments (PPP) cycle.

Process mining takes transactions and maps out the process related to the transactions in the data set. By doing this it is easy to see:

  • The main process for transactions
  • Potential issues, including:
    • Incorrect journal/document sequence
    • Skipping of required journals/documents
    • Users working where they shouldn't be
