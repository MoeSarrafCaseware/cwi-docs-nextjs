Detalles de la transacción

You can view details of transactions in a specific process variant.

To view transaction details:

  • From the home screen or the Variants dialog, double-click on a specific process variant or click the weblink in a transaction's Variant #.
    The Transaction Details dialog displays. This dialog lists all the transactions sorted in descending order of posting date. Scroll through to view all the transactions.

Transaction details report

The Transaction Details report contains all the component transaction groups in a specific process variant. It includes the date and time of report generation, the user who generated the report and a complete list of all the transaction groups in a process variant.

To download the transaction details report:

  1. From the Transaction Details dialog, click the Download Transactions button ().
  2. Select Download, to download a CSV file.
