Understand the interface

Before we start working on the engagement file, let's go over the engagement interface.

Propiedades del encargo

After you've created a new engagement, an Engagement Property dialog automatically displays in the engagement file. Here you can enter some high-level information about the engagement.

In this dialog, you can set the following properties:

  • Workflow stage - select a stage for the status of the overall engagement work.

  • Job number - enter your firm's job number for this engagement.

  • Operating name - this field shows the Cloud entity name by default. You can change the default and enter a different name.

  • Reporting period - choose a reporting period for this engagement.

  • Year end date - choose a year end date for this engagement.

The Engagement Properties dialog also features the following buttons:

  • Second Reviewer - select to add a second reviewer for all signoffs. This adds Final Review to all signoff options.

  • Lockdown - select to lock the engagement file, once all the work is complete.

For this exercise, set the following properties for your engagement file:

  • Workflow stage - In Progress

  • Job number - 2019-10000-<your initials>

  • Operating Name - ACME Inc.

  • Reporting period - Annually

  • Fecha de cierre del ejercicio

You do not need to add a second reviewer. Once you have finished updating your engagement properties, select Save.

Engagement level

If you want an issue to apply globally to the entire engagement, you can add the issue on the engagement level.

To add an issue to an engagement:

  1. From the top menu, select the Add Issue icon ().

    A pop-up displays on the page showing a text box with a formatting toolbar.


  2. Enter a description for your issue in the text box.

  3. Select an issue type from the drop-down menu at the bottom of the editor.

  4. Select Create Issue.

Document level

You can also add issues to specific documents within the engagement.

To add an issue to a document:

  1. Select the Add issue icon () for the document.

  2. Enter a description for your issue in the text box.

  3. Select an issue type from the drop-down menu at the bottom of the editor.

  4. Select Create Issue.

Item level

You can also add issues to items within documents, For example, you can add an issue to an individual procedure in a checklist.

To add an issue to a procedure:

  1. Select the Add issue icon () for the procedure.

  2. Enter a description for your issue in the text box.

  3. Select an issue type from the drop-down menu at the bottom of the editor.

  4. Select Create Issue.

Issues tab

You can view and manage all issues in an engagement from the Issues page. Select Issues to navigate to the page.

Información

You can select the Information icon () from the top menu to display the information menu. From this menu you can use the Download reference copy option, access the product help, and view information about your current version of the product.

Descargar copia de referencia

When you select Download reference copy, a zip folder downloads to your local drive. The default name of the folder is <Operating name><Year>. The folder contains PDFs of the following:

  • Información del encargo

  • A list of documents

  • All engagement documents

  • A list of issues

  • A list of adjustments

  • Balance de comprobación

Ayuda

When you select Help, a new browser window opens with the links to the online user help topics.

Acerca de

When you select About, a pop up displays the current product version number, other software information and a link to the Latest Release Notes.

Documents tab

When you select the Documents tab from the top menu, the Documents page displays. This is where you can manage documents, check the progress of the engagement work, and start to perform various engagement tasks such as preparing a letter to your client or completing a checklist.

Navigation Panel

To have quick access to the engagement documents, you can use the navigation panel on the left of the Documents page.

To expand the navigation panel, you can select the Documents icon () on the top left, or the expand icon () on the bottom left. To minimize the panel, select the contract icon ().

Buscar

The search bar is located at the top of the navigation panel. You can search for a document by name. You can also search for a keyword in any documents.

Document list

On the navigation panel, you can see a list of all currently visible folders and documents. Once you have opened a document, its name will display under the Recent category.

Pinned documents

If you want a recently opened document to always appear on the top of the list, hover over the document under the Recent category, and select the pin icon (). The document will appear under the Pinned category at the top of the document list.
