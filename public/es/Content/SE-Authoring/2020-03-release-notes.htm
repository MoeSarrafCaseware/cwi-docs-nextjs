SE 2020-03 release notes

SE 2020-03 brings in the big little features. Like, the ability to export groups and their structures or copy documents from one template to another. Additions that we sized up as "small". Yet, very unexpectedly, they've driven most of the discussion and questions in the 2020-03 release webinar!

New for authors

Here is what's new for authors in SE 2020-03.

Controles

#primaryauthor

Note: This feature is only available for testing purposes. It will not be deployed to partner or production environments.

A new section Controls is now available in the Product Settings, as we continue to expand the new risk module to include:

  • Automation of control implementation and design documentation.

  • Testing the operating effectiveness of controls.

In the General tab, select the Enable controls in the risk module check box to set up general options and fields related to compliance and documentation of control identification and testing.

In the Fields tab, you can specify which fields display in the control report, as well as customize labels.

The figure below shows the Operating effectiveness evaluation section in the Fields tab. Notice how each field can be enabled or disabled in the report. Labels for enabled fields can be also customized.

Note: The Out-Of-The-Box controls report will be available in a future release. However, you can use SE Builder to build a custom Control Report based on the Product Settings. Contact the Cloud Engagements Product Team for details and support.

Smart Copy-Paste

#primaryauthor #firmauthor

Adopting this approach helps you develop and release products efficiently, as well as reduces the likelihood of potential errors.

To enable Smart Copy-Paste, set the Import Documents property to true in the product.json of the template you want to copy documents into.

Once you enable Smart Copy-Paste, the Copy From Another Product option becomes available in the template you're currently working on and you can copy documents authored in other product templates.

In SE 2020-03, the Smart Copy-Paste feature has limited functionality that we plan to extend in future releases.

  • You can only copy Letters, Memos, Worksheets and FinancialStatements.

  • Copying documents from one template to the other does not identify missing or inconsistent data within visibility settings, dynamic text, formulas, engagement glossary terms and analysis charts.

Note: You need to manually identify and resolve data discrepancies for visibility settings, dynamic text, formulas, engagement glossary terms and analysis charts between the source and destination templates after you copy documents.

To learn more, see What is Smart Copy-Paste, Copy content from another template and Current limitations.

XBRL

#primaryauthor

Note: This feature is only available for testing purposes. It will not be deployed to partner or production environments.

You can now review XBRL facts in financial statements, as well as export the instance documents to validate that they generate as expected.

If you have enabled XBRL in your product, the XBRL Review and Export icon is available in your product template.

You can select the taxonomy that you want to review or export. Puede detener el proceso de revisión en cualquier momento y continuarlo más tarde. For more details on how to test, see XBRL Review and Export.

Note: In SE 2020-03, XBRL review and export is only available for dynamic table areas in the financial statements. We plan to extend the feature to other documents in future releases.

New for end users

Here is what's new for end users in SE 2020-03.

Impresión masiva

A new Bulk Print Settings option is now available. This option allows users to group two or more documents to be merged and printed as one PDF document.

Users can configure more than one bulk print group. Also, the same document can be included in multiple bulk print groups.

Once a bulk print group is configured, it displays in the print option in the Documents page and every engagement document. With bulk prints configured, users still have the option to print a document individually, if needed.

Página de datos

The Data page features new additions related to group tabs and the Adjustments column.

  • Export options are now available in the group tabs. Users can select Export () to export data to a CSV file, PDF or Working Papers. They can also export the grouping structure.

    • Note that the export takes into consideration existing selections on the page. For example, if users have disabled Parent Groups, then they're not included in the exported file.

  • Adjustments now display in blue. Users can select an adjustment to navigate to the Adjustments page, where the corresponding adjusting entries are highlighted and they can view their details.

    • The below figure shows how the corresponding adjusting entry is highlighted in the Adjustments page.

Problemas

In SE 2020-03, we have enhanced the logic around deleting issues to prevent loss of information and improve audit trail.

  • Users can no longer delete an issue unless they've created it.

  • Sólo un usuario administrador puede eliminar cuestiones creadas por otros usuarios. An Admin user has one or more Cloud security roles:

    • Admin

    • Configuración Admin

    • Entidades Admin

    • Personal Admin
