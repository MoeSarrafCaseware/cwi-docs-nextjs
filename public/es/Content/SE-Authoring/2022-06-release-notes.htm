SE 2022-06 release notes

Summer is here and so is our latest round of updates with SE 2022-06. This release introduces rounding values for dynamic tables, planning balances, improvements to the formatting and presentation of the financial statements and more.

Redondeo en estados financieros

#enduser #primaryauthor #firmauthor

You can now enable rounding in the financial statements. Enabling rounding removes the need for the user to manually round values and improves the readability of the financial statements.

Once you enable rounding in the product settings, you can set whether to round to Ones, Thousands, Millions, or Billions.

También podrá:

  • Select how many decimal places to display

  • Determine whether to display the rounding information in description headings and column headings

  • Customize the terminology used for rounding

Once rounding is enabled, in addition to displaying rounded values in dynamic tables, you can also use Scaling/Rounding conditions when adding engagement glossary terms and dynamic text, as well as when applying visibility settings to content.

Note that when rounding is enabled, firm authors and end users can also modify the rounding settings used in the engagement.

To learn more, see

  • Personalizar la configuración de los estados financieros

  • Visibility conditions

  • Conditions for glossary terms and dynamic text display logic

  • Personalizar el redondeo en los estados financieros

Balance de planificación

#enduser #primaryauthor #firmauthor

The release of SE 2022-06 introduces the planning balances feature. Planning balances allow auditors to record and track interim balances before the year end. This allows them to plan the audit ahead of time and adapt to changes as they occur, as well as fulfilling the requirement to document their work.

Primary authors can enable planning balances in the product settings.

Once you do so, you can then customize the terminology for Planning balances in the Terminology tab of the product settings. Note that you can also now customize the terminology used for Previous balances.

To learn more, see Enable planning balances and Customize terminology.

Currency symbols in dynamic tables

#enduser #primaryauthor #firmauthor

In order to further enhance the presentation and formatting options available for the financial statements, additional options are now available for currency symbols.

In addition to the existing option to display currency symbols in column headings by default for all dynamic tables, you can now also choose to display them by default on the first and last rows of dynamic tables.

Note that the currency symbol will only display on rows that are:

  • Included in the generated PDF when users print or download the financial statements.

  • Either a Trial Balance or Input row type.

  • Not a title row

If the first or last row in a table does not meet these conditions, the currency symbol will display on the next available row that does. For example, if you set the currency symbol to display on the last row, but the row is set to not be included when printing, the symbol will instead display on the previous row. If that row also does not meet the criteria, the symbol will display on the row above that one.

You also now have the option to select whether to display the currency symbol or the currency code in the tables.

Additionally, you can now select to display or hide currency symbols in any Trial balance, Input, or Dimension row in individual tables.

To learn more, see Set default currency options for dynamic tables and Specify the currency symbol option for rows.

#enduser #primaryauthor #firmauthor

You can now set Text Section areas to keep together when printing from the More actions () menu. This ensures that there are no page breaks in the middle of the section when you print or download the financial statements.

To learn more, see Add text in the financial statements.

Agrupar columnas en tablas dinámicas

#enduser #primaryauthor #firmauthor

You can now group columns in dynamic tables so you can present related columns together and clearly label them as part of a group.

Once you select which columns to group, you can add a header for the group which will display in the dynamic table above the headers for the individual columns.

To learn more, see Group columns in dynamic tables.

Decimal values in percent variance columns

#enduser #primaryauthor #firmauthor

In dynamic tables, you can now select the number of decimal places to display in Percent variance columns. The options available for the number of decimal places displayed are 0 to 5. By default, Percent variance columns are now set to display 2 decimal places.

To learn more, see Column settings in dynamic tables.

Search for trial balance groups in the Data Link dialog

#enduser #primaryauthor #firmauthor

The Data Link dialog now includes a search bar in the Trial Balance tab. The parent groups of the relevant trial balance groups are included in search results so users can ensure they are selecting the correct group.

New for end users

Here is what's new for end users in SE 2022-06.

Job titles for contacts

When users assign contacts to a query, the job titles of the contacts entered in Cloud now display in the drop-down so users can ensure they’re assigning the query to the correct contact.

Access entities from engagements

Users can now select the name of the entity from the Engagement Properties popup to quickly navigate to that entity in their Cloud site.

Edit PDFs

Users can now edit PDFs that have been uploaded to their engagement.

When users open a PDF in their engagement, they can select the Copy URL () button to copy the file URL. This URL can then be used to open the file in Adobe Acrobat.

After the user edits the PDF in Acrobat, they can then Save as the file to the same file pathway to sync the changes to the PDF in their engagement.

To learn more, see Edit an added PDF document.

Working Papers file links

This release includes technical improvements to link functionality for external documents, such as files uploaded to queries, in integrated Working Papers files.

Carry forward for contact responses

In checklists, responses for procedures with Contact information response types are now included in the next year’s file when users carry forward the engagement.

PDF/A download option

Users can now select to download the financial statements as PDF/A documents.

To learn more, see Print the financial statements document.

Tooltips for carry forward and review tools icons

Tooltips have now been added to carry forward and review tools icons in the financial statements, so users can understand at a glance what each icon means.

#primaryauthor

Fix

The following list features a brief description of bugs that have been resolved in SE 2022-06.

  • In the Data page, the prior year column does not use the correct materiality value.

  • UI inconsistencies on the Data page.

  • Translation issues.

  • In audit engagements, the balance from transactions is not calculated in data completeness tables if the trial balance data and transactions data were imported from two different sources.

  • In beta firms, the Use Default button is not available on the firm template page if firm author mode has been disabled.

  • In checklists, when users select to show hidden items, the items are not visible until the user refreshes the page.

  • Users need to click into text boxes twice before they can begin typing any text.

  • In products that support multiple languages, when the language users have selected when logging in to Cloud is different from the content language of the engagement, documents display content in both languages.

  • UI improvements to the firm template page to clarify the function of the Update button. The button now reads Migrate Customizations from Previous Template.

  • In checklists, dynamic text and engagement glossary terms that are dependent upon procedure responses do not display until the user refreshes the page.

  • When users export adjustments to PDF, the adjusting entries do not display as expected.

  • When users print the financial statements, input column cells in trial balance rows display as blank if the trial balance row has child rows and is collapsed.

  • In document print settings, when users adjust the margins of documents to a non-recommended size, the warning message always displays the size of the margin in inches even if centimeters are the selected unit of measurement for the engagement.
