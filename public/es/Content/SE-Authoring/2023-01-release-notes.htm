SE 2023-01 release notes

Ring in the new year with the latest SE release! SE 2023-01 is bringing some exciting new changes including the brand new controls report and significant enhancements to consolidation.

New controls report

#enduser #primaryauthor #firmauthor

SE 2023-01 features the official launch of the new controls report in the risk module. With the new controls report, you no longer need to create a custom controls report using SE Builder. This feature provides an improved user experience for documenting controls and enhances in-platform compliance to and support for audit documentation requirements.

You can enable controls for the risk module in the product settings.

También podrá:

  • Enable and customize sign-offs for the controls report. Note that once you enable signoff, end users and firm authors can customize sign-offs in the firm template and in engagement files

  • Customize fields used in the Create New Control dialog

  • Determine how controls are identified

  • Add a help link relevant to controls for your product

In the Controls | Fields tab, you can customize the fields available in the controls report and determine whether they are available to users in the Create New Control dialog.

Once you enable controls, the Controls tab is available to end users in the risk module and the module name is changed to Risks & controls if both risks and controls are enabled.

To learn more, see Enable controls in the risk module.

Enhancements to consolidation

#enduser #primaryauthor #firmauthor

A number of enhancements have been made to engagement consolidation to support more complicated consolidation hierarchies and circumstances where subsidiary entities do not require separate engagements.

The existing graph view for consolidation now includes:

  • A search option to allow users to easily find subsidiary entities

  • Updated designs for each subsidiary card

  • The ability to expand and collapse branches in the hierarchy

  • When subsidiaries with separate engagement files are added, a link to the engagement file is available in the subsidiary card to make it easier to navigate between engagement files

A new grid view has also been added.

In addition to these changes, you can now use the Add imported TB option to create internal subsidiary entities for your parent entity and upload trial balance data without linking a separate engagement file for the subsidiary.

To further support these improvements, the following updates have also been made:

  • You can now add adjustments for internal subsidiaries

  • The Data page has been updated to support filtering trial balance data for specific entities in the consolidation hierarchy

  • When you add a calculation from the Datalink dialog, you can specify whether the trial balance data linked is for a subsidiary, consolidated, or for the parent entity

  • You can now create data link formulas that determine which data to use based on whether the document is set to use consolidated or non-consolidated data.

  • You can now add a Total Consolidation column to dynamic tables to display the total consolidated balance

To learn more, see Create a consolidation engagement and View data for consolidation engagements in the Data page.

Enhancements to dimensions

#enduser #primaryauthor #firmauthor

A number of enhancements related to dimensions have been added as part of this release.

Dimensions settings

#enduser #primaryauthor #firmauthor

Firm authors and end users can now both create and edit dimensions in the Dimensions tab in the product settings. The Dimensions tab also now features an updated design to improve the user experience.

You can also now:

  • Add third-level dimensions

  • Drag-drop dimensions to reorder them within a category

  • Assign dimension categories to financial groups from the More actions() menu.

  • Firm authors and end users can also assign dimension categories to financial groups from the Group properties dialog.

    Tenga en cuenta que:

    • Firm authors can only modify dimension categories assigned to groups by other firm authors and end users

    • End users can only modify categories assigned by other end users.

    • Firm authors and end users cannot modify dimension categories for groups that have inherited categories from higher level groups

    Para más información, consulte Añadir zonas.

Dimensions filter for financial groups

#enduser #primaryauthor #firmauthor

There is now a Dimensions drop-down available in the Financial Groups tab in the Data page to allow users to filter the financial groups by dimension categories.

Note that for consolidated engagements, instead of a drop-down, there is a separate Dimensions view for the Financial Groups tab.

Dimensions in dynamic tables

#enduser #primaryauthor #firmauthor

To enhance the ability to use dimensions in dynamic tables, you can now filter by dimensions:

  • When inserting a trial balance row

  • When inserting a trial balance data link as part of a calculation in table cells

Añadir nuevos documentos Word y Excel

#enduser #primaryauthor #firmauthor

You can now create new Word and Excel documents from within an engagement on the Documents page.

To learn more, see Add new Word and Excel documents.

Embed PDFs in financial statements

#enduser #primaryauthor #firmauthor

You can now embed PDFs in financial statements. Tenga en cuenta que:

  • You can add multiple Embedded PDF areas to the financial statements

  • Only one PDF can be uploaded to each Embedded PDF area.

  • Once you embed a PDF, it’s included in the table of contents for the financial statements

To learn more, see Embed a PDF in the financial statements.

Hide header rows for dynamic tables

#enduser #primaryauthor #firmauthor

You can now choose to hide the header rows for individual dynamic tables in financial statements. When a header row is hidden, it does not display when users view the financial statements or in the generated PDF when users print the financial statements.

To learn more, see Hide dynamic table header rows.

New for end users

Here is what's new for end users in SE 2023-01.

Configuración de impresión en masa

Bulk print documents are now clearly labeled when users are copying files from other engagements.

Improvement to print preview for financial statements

When users select the Print preview option, the preview now opens to the section of the financial statements the user was viewing.

Fixes in SE 2023-01

The following list features a brief description of bugs that have been resolved in SE 2023-01.

  • In old queries, the target folder and file name for file upload type query questions does not display as expected.

  • When users edit a query name after it has been sent to contacts, the updated name does not display to contacts.

  • In dynamic tables, column headers for custom calculation columns do not display for primary authors.

  • Users can add new accounts without assigning them a name when they add accounts from the Add Adjustment dialog.

  • In dynamic tables, note references added to commentary rows do not save.

  • In consolidation engagements, materiality icons do not display for the subsidiary entity columns in the Consolidated view for the Financial Groups tab.

  • In consolidation engagements, duplicate balances display in the Consolidated view for the Financial Groups tab.

  • Condition details do not display for dynamic text with Group Conditions.

  • When users add a new account and assign it a parent group with a dimension category, the parent group does not save on the first attempt.

  • In consolidation engagements, the materiality document only displays consolidated data.

  • A technical improvement related to old queries.
