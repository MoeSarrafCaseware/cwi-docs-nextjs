SE 2023-06 release notes

Multiple financial grouping structures, improvements to the engagement lockdown process, and updated multi-language support for financial statements. These are just some of the exciting new features waiting for you in the SE 2023-06 release.

Multiple grouping structures

#enduser#primaryauthor#firmauthor #enduser #primaryauthor #firmauthor

In order to enhance the ability to build compliant financial statements, you can now add multiple financial grouping structures to your product in the product settings.

To align with this new feature and enhance the user experience, the Group properties dialog has also been updated.

Once you import your grouping structures, you can link groups from one structure to groups in another. This makes it easier for users to map their user accounts. When they assign an account to a group in the default grouping structure, it will automatically be assigned to the linked groups in any secondary structures.

After you define your financial grouping structures, both authors and end users will be able to select the grouping structure that meets their requirements when adding dynamic tables to worksheets or the financial statements. Note that you can only select one grouping structure per dynamic table.

You can also select a grouping structure when you:

  • Create dimension categories

  • Post adjustments

  • Añadir enlaces externos

  • View analysis charts in checklists

Para más información, consulte Añadir zonas.

Enhancements to engagement lockdown

#enduser#primaryauthor#firmauthor #enduser #primaryauthor #firmauthor

To give users more control over engagement lockdown, a new Lockdown settings section has been added to the product settings.

From the Lockdown settings, you can restrict the ability to unlock the engagement to users with Admin roles in Cloud. When this ability is restricted, admin users will be able to temporarily grant unlock permissions to non-admin users if required.

You can also select to Enable target lockdown days. If you toggle this option on, you can then set the target lockdown days and select whether the days are counted from the report signing date or the report release date. End users can specify the report signing date or release date in the Engagement Properties popover. You can also select for the engagement to lock automatically once the target lockdown day is met.

Note that the Lockdown settings are also available for firm authors.

The look and feel of the engagement lockdown process has also been updated.

To learn more, see Customize the lockdown settingsCustomize the lockdown settings and Lock down the engagement.

 

Documents page navigation

#enduser#primaryauthor#firmauthor #enduser #primaryauthor #firmauthor

When you navigate back to the Documents page after viewing a document, you are now automatically returned to the same location. This improvement ensures you don’t have to spend time navigating back to your previous location.

Tables in queries

#enduser#primaryauthor#firmauthor #enduser #primaryauthor #firmauthor

You can now use the Allow multiple rows option in query settings to add tables to queries.

When you select this option, each response type you add to the query question displays as a column in the table. You can then select Add Row to add more rows to the table.

Note that you can have a maximum of three columns in a table.

To learn more, see Review and customize question settingsReview and customize question settings.

Enhanced multi-language support for financial statements

#enduser#primaryauthor#firmauthor #enduser #primaryauthor #firmauthor

You can now add multi-language content for text areas, headers and footers in financial statements without setting visibility conditions on the content.

After you author content in the default language for your product, you can switch the content language of the template and then replace the content in the default language with content in other languages.

Users will only see content authored for the language they have set their engagement to use.

To learn more, see Use-case - A product with French and English language support.

To learn more, see Authoring content and deliverables in multiple languages.

Toggle options for risk assessment fields

#primaryauthor #primaryauthor

You can now select whether to include the Inherent Risk and Control Risk fields for risk assessments in your product. This allows you to assess the Risk of Material Assessment in products where users do not have to evaluate the inherent or control risk levels.

If these fields are toggled off, the Likelihood and Magnitude fields will also automatically be toggled off.

To learn more, see Enable the new risk module.

New for end users

Here is what's new for end users in SE 2023-06.

Edit author-created dimensions

End users can now edit dimensions created by primary authors and firm authors. Note that end users cannot delete dimensions created by authors.

Para más información, consulte Añadir zonas.

File restoration

End users can now restore and download deleted third-party files from the new Recycle Bin. Examples of third-party files include Word, Excel and PDF documents.

When users delete a third-party file, the Recycle Bin option appears in the left pane of the Documents page.

When users navigate to the Recycle Bin page, they can view a list of deleted documents and select to download them or view the file history from the More actions() menu.

From the File History page, users can view all previous versions of the document and select a version to restore.

Note that the ability to restore files directly from the Recycle Bin will be available at a later date.

To learn more, see Download a deleted document and Restore a previous version of a document.

Duplicate adjustments

Users can now use the Duplicate option to duplicate adjustments. Cuando se duplica un ajuste, el nuevo ajuste tiene el mismo tipo, descripción y detalles de cuenta que el ajuste original.

Note that any annotations, issues, or signoffs associated with the adjustment are not copied, and the duplicate adjustment defaults to the current year even if the original adjustment is for the prior year.

To learn more, see Post adjustments to the trial balance.

Search functionality for entity selector in dynamic tables

In consolidation engagements, users can now search for the correct entity when they’re selecting an entity while editing column settings in dynamic tables. The search option is only available if there are at least 15 subsidiary entities.

Carry forward for controls

Controls are now included when engagements are carried forward.
