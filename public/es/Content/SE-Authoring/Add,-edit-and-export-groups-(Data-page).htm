Add, edit and export groups (Data page)

You can add groups or edit the properties of existing groups. You can also export groups to a CSV file to review their properties or make changes easily.

Add a group (Data page)

You can manually add groups to define a grouping.

Para agregar un grupo:

  1. Go to the Data page, then select Add () | Add group.

    2. The Group Properties dialog displays.

  2. Set the properties for the new group.

    Refer to Group properties for a description of each group property.

  1. Select Save to add the group.

Edit a group (Data page)

You can edit the properties of a group in the structure.

Para editar el nombre de un grupo:

  1. Go to the Data page, then select the grouping structure from the left pane:

    • Grupos de contabilidad
    • Grupos de impuestos
    • Grupos prescritos

  2. Select the group you want to edit, then select Edit () to open the Group Properties dialog.

    3. Tip: You can search groups using the Search field on the top right corner of the page.

  3. Edit the group properties as needed.

    4. Refer to Group properties for a description of each group property.

  4. Once you're done, select Save.

Reorder a group (Data page)

You can drag and drop groups in the Data page to reorder them. Navigate to the grouping structure in the Data page and drag the group you want to reorder to the desired location.

Note that you can only use drag and drop to reorder groups under the following conditions:

  • The groups you are reordering are all sibling groups in the grouping structure. You cannot drag and drop a child group to be the same level as or higher level than the parent group.

  • All the sibling groups are collapsed or do not have any child groups.

Export groups to a CSV file

You can export groups to a CSV file to review the properties. You can also export them to have all the groups lined up so you can easily make changes, then save the file and import it.

Reimporting groups retains any existing group properties. Only updates to existing properties and adding values to empty properties take effect.

For example, you can export groups to a CSV and easily add the value 'Y' for the Default Group property for 15 groups by copy-pasting 'Y' across 15 rows. Once you're done, save the file and reimport it.

Note: When you export groups, the order of the groups in the structure upon export is maintained when you import them again. In the CSV file, however, the groups will be displayed in alphabetical or numerical order based on the values in the first column. See Reorder a group to learn how to reorder your grouping structure in the Data page.
