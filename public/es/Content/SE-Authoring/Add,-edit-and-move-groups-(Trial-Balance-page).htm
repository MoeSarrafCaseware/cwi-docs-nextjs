Add, edit and move groups (Trial Balance page)

You can edit the properties of existing groups or rearrange them. You can also add groups.

Add groups (Trial Balance page)

You can manually add groups to define a grouping.

Para agregar un grupo:

  1. Go to the Trial Balance page, then select Add () | Group.

    2. The Group Properties dialog displays.

  2. Set the properties for the new group.

  1. Select Save to add the group.

Edit groups (Trial Balance page)

You can edit the properties for any group in the structure.

Para editar el nombre de un grupo:

  1. Go to the Trial Balance page, then choose the grouping structure from the drop-down list at the top.

    2. You can choose to edit for Financials , Tax Code or Chart of Accounts grouping.

  2. Select the group you want to edit, then select Edit ().

    3. The Group Properties dialog displays.

  3. Edit the group properties as needed.

    4. See Group Properties for a detailed description of each property.

  4. Once you're done, select Save.

Move groups (Trial Balance page)

You can change the structure of the grouping by moving groups.

To move a group:

  1. Go to the Trial Balance page.

  2. Select the group that you want to move, then drag and drop into the desired place in the structure.

    3. A rectangular grid displays in the destination place to identify the target spot for the group.

  3. To convert the group to a subgroup under a specific parent group, drag it and drop in the target parent group.

    The target parent is marked with a green background.
