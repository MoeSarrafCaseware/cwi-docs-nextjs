Add a calculation in table cells

You can add calculation formulas in table cells. Once users import data, the cells automatically populate with calculated values based on your predefined formulas. For example, you can add cell calculations in your template to build:

  • complex notes to the financial statements, such as movement rows.

  • worksheets (which can be used as a base setup to various items in your template).

  • movement rows in the Statement of Cash Flows.

  • ratio analysis rows in the Earnings per share section in the Income Statement.

The calculation can include numbers, data sourced from the trial balance, data sourced from the financial statements, form values (if SE Builder forms are developed in your product) or a combination of all.

Note: This feature is only available for cells in columns or rows with type Input.

To add a calculation in table cells:

  1. Select the table.

  2. Select More actions (), then choose Edit Dynamic Table.

    3. The table opens in edit mode.

  3. Go to the desired cell and select the fx icon.

    Note: The cell must be an Input row or Input column cell.

  4. In the Calculation tab, enter your formula.

    1. Select Insert data (fx) to insert a trial balance data link in the formula.

      2. The Data Link dialog opens on the Trial Balance tab. The current document is selected by default as the source for trial balance data.

      You can change the default using the Source drop-down:

      • Trial balance - trial balance data is sourced from the Final column in the Data page. That is, the final amounts after any existing adjustments are applied.

      • Financial statements - trial balance data is sourced from the financial statements. That is, the amounts after rounding is applied and any manual overrides are done.

      As you insert data links, the () or () prefix indicates the data source.

      Notas:

      • The Cell Reference tab enables you to reference values from one dynamic table into another. See Add cell references for more information.

      • If you have developed SE Builder forms in your product, the Forms value tab is available in the Data Link dialog. You can insert values from forms using this tab.

    2. Select Numbers () to insert numbers in the formula.

      Tip: You can use your keyboard to insert numbers.

    3. Add the desired operators to build the formula.

      Tip: You can use your keyboard to insert operators.

    4. Select DONE to close the Calculation tab.

      Note: If you've entered an invalid formula, the fx icon displays in red () to indicate an error.

As you're setting up the financial statements, you can review the calculation formulas you've added. Select a cell to view details of the calculation.

End users can override cell calculations. To learn more, see Add or edit a calculation in table cells.
