Add a grouping area

You can use grouping areas to group multiple text areas and guidance sections. Grouped items are slightly indented.

To add a grouping area, select +Add section | Grouping Area.

A grouping area is added in the document with a default title Grouping Area. You can select the default title to change it. Note that this title is not part of the document content.

To delete a grouping area, select Settings () | Delete.

If you delete a grouping area, all text areas and guidance sections in that grouping area are deleted as well.
