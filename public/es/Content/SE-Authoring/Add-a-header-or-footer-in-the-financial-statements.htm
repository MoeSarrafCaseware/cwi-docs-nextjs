Add header or footer in an area

You can customize the header or footer for an area if you want to have a different header or footer information in that area than the rest of the document.

To add a custom header or footer in an area:

  1. Go to the desired area.

  2. Select the More actions () icon at the top right corner of the page.

  3. Choose Edit Print Settings from the list of actions.

  4. Select Custom header and Custom footer to display the text fields for the header or footer.

  5. Enter the header and footer content at the text fields.

    6. You can use the formatting toolbar to customize the header or footer content.

You can customize the header or footer for a page break if you want one page in an area to display a different header or footer than the rest of the area or the document.

To add a custom header or footer to a page break:

  1. Select the desired page break.

  2. Select More actions () |Print Settings.

  3. Select Custom header and Custom footer to display the text fields for the header or footer.
