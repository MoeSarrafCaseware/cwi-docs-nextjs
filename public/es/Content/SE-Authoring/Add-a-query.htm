Añadir una consulta

A query is a client request. A query must be part of a group. You cannot add a query in the document without a group.

Añadir una consulta

  1. Go to the group where you want to add a query.

  2. Select + Query.

    3. A new query is added with a default description New Query and a File Upload response type.

  3. Select the default description to change it.

    4. Once you select the text field, the formatting toolbar displays. Use the toolbar to make the following changes:

Añadir un enlace de datos en la descripción del procedimiento

Data links () allow you to have already existing information in the engagement or in Cloud automatically retrieved and inserted in text fields.

You can add the following data links:

  • Propiedades del encargo

    • Engagement properties as defined in Cloud, such as:

    • Nombre operativo
    • Nombre legal
    • Fecha de cierre del ejercicio
    • Fecha de inicio del ejercicio
    • Ejercicio en curso
    • Sitio web
    • Año anterior
    • Dirección de la entidad

  • Balance de comprobación

    Items in the trial balance in the engagement, such as: account name, grouping name, or values from the current or prior year.

  • Propiedades de firma

    Firm properties as defined in Cloud:

    • Nombre de la firma
    • Dirección de la firma

  • Respuestas

    Responses to procedures from checklists. Once you select a checklist document from the list, select the response you want to retrieve and insert in the text section.

  • Materialidad

    Values defines in the Materiality document.

  • Glosario del encargo

    Add a term from the engagement glossary.

    See Add an Engagement Glossary term.

  • Texto dinámico

    Dynamic text is conditional text. It's a field where different text displays based on a condition or a set of conditions.

    See Add dynamic text.

  • Valor de formulario

    Insert the value of a specific form field. Once you select a form and a field, the value of the selected field is inserted in the text field.

    Note: This option is only available in the Data Link dialog if you have developed forms using SE Builder in your product. To learn more, see Develop custom forms.

Añadir una referencia de documento en la pregunta de consulta

Adding a document reference inserts a link to an existing document in the engagement file. You can add a reference to a document in the query description. For example, you can add a link to a letter or checklist.

Añadir una referencia de documento en la pregunta de consulta

  1. Go to the query where you want to insert a document reference.

  2. Select the text field to display the formatting toolbar.

  3. Select Insert Document Reference ().

  4. A list of the available documents in the engagement file displays. You can select Show hidden documents to select a hidden document if needed.

  5. Seleccione el campo que desea eliminar de la base de datos.

    6. A reference to the selected document is created and added in the query description.
