Risk Library Sample File

The template is also available for download in the Unpublished Version dialog.

If you’re adding a risk library version for the first time, the version is automatically added as a Major version. You can also add subsequent risk library versions: Minor or Patch. Use minor versions for planned updates to the library and use patch versions for urgent updates. This allows you to simultaneously work on a planned update and release critical updates.

Note: You can only have one unpublished minor version and one unpublished patch version at a time.

To start, select Settings from the Cloud menu(), then select Risk Library | Manage Risk Library.

To add a risk library version:

  1. Select the risk library from the left pane.

  2. If you’re adding the first version of this risk library, select CREATE VERSION then go to Step 4

  3. If this is not the first version for this risk library:

    1. Seleccione ADD.

      The Unpublished Version dialog opens.

    2. Seleccione el tipo de extracción.

      Opciones disponibles:

      • Minor - Select this option for planned updates.

      • Patch - Select this option for urgent updates.

  4. Select the Upload Excel File box to browse or drag and drop the file.

    You can select DOWNLOAD TEMPLATE to download an empty template that you can use to set up your risk library Excel file.

  5. Seleccione CREATE.

    Errors and warnings display in the dialog - if there are issues with the Excel file data.

    You can ignore warnings, but you must fix all errors and reupload the file to continue.

Once you’ve added the version, you can also:

  • Select Edit () to replace the Excel file.

  • Select Summary Report () to view a summary of the risk library content.

  • Seleccione Sí para eliminar la cuestión.
