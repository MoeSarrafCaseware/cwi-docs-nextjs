Add an analysis section

You can add an analysis section in the financial statements to include a visual representation of the data through tables, graphs and charts.

You can specify the types of graphs and charts that are available in the analysis section , as well as the data they present in the product API. Para más información, consulte Añadir zonas.

To add an Analysis section:

  1. Locate where you want to add the analysis section.

    2. You can add an analysis section in an area or a note or a note heading.

  2. Go to the left side of the page and select Add ().

  3. Choose Analysis from the list of content types.

    4. A list of the available tables, charts and graphs displays. To customize this list, see Analysis charts.

  4. Select the desired figure to add it in the analysis section.

    5. Note: The data displayed in the analysis section reflects the balances in the financial statements not the trial balance. Financial statements balances can include overridden values and adjustments.

You can make a few modifications to the tables, charts or graphs in the analysis section to customize the default look of the section and data presentation.

To customize the Analysis section:

  1. At the Select Graph drop-down, choose the figures you want to include in the section.

    2. Tip: If you accidentally select a figure and decided it should be removed, you can simply close it () to have it removed.

  2. Use the preliminary/final switch () to select whether preliminary or final data should be represented by default in the figures.

  3. Select the Full size icon () to display the figure in a full size across the section.

  4. Select the Edit icon () in the figure's title bar, then select the Add icon () to add an item to the chart. Select an item from the displayed list to add it to the figure.

  5. In tables, select the COLUMNS button to specify which columns display by default in a specific table.

  6. If you want to reset a figure to its original state, select the Reset to default icon ().
