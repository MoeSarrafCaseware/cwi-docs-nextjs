Agregar afirmaciones

You can add an assertion to your SE product at Features | Assertions.

Note: Make sure assertions are enabled in the product.json, otherwise, the Assertions tab will not be available. To learn more, see Enable or disable features in your product.

Under the Features drop-down, select Assertions.

To add an assertion:

  1. Select + Add Assertion.

  2. Enter the abbreviation and the full name of the assertion.

Note that all assertions must be associated with one of the 12 base assertions; unassociated assertions will be deleted.
