Add guidance information in a query

The Guidance is the first section in the query template. If there are standard information related to filling out queries by the client contacts, you can add it in this section.

To add guidance information:

  1. Go to the Guidance area at the top.

  2. Select the text field to start adding content. The formatting toolbar displays. Use the toolbar to make the following changes:

Toolbar icon

Descripción

Bold, italicize or underline selected text.

Add a bullet or number list.

Indent selected text or paragraph.

Align selected text or paragraph.

Add a table.

Agregar o quitar un punto de interrupción.
