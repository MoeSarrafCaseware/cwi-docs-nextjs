Add note references in tables

A note reference is a link to one or more existing notes in the financial statement. You can add note references in table rows, including total rows, or columns in the financial statement. For example, to link an account or a group to an existing note that holds related details or information.

Add a note reference in a row

You can reference notes in rows.

To add a note reference in a row:

  1. Select the table.

    Icons for table options display on the right.

  2. Select More actions (), then choose Edit Dynamic Table.

    3. The table opens in edit mode.

  3. Go to the row where you want to add a note reference.

  4. Select the Add () button above the row.

  5. Choose Commentary row.

    6. A note reference is added right above the row.

  6. Select (Note#) to display a list the existing notes in the financial statements document.

    Tip: If you have set visibility logic for the notes, select the Show hidden notes check box to include hidden notes in the list.

  7. Select one or more notes from the list to reference it in the row.

  8. Select SAVE or SAVE & CLOSE.

    9. The number of the referenced note displays next to the row name.

Add a note reference in a column

You can reference notes columns in the financial statements document.

To add a note reference in a column:

  1. Add a column in the table.

    1. Select the table.

      Icons for table options display on the right.

    2. Select More actions (), then choose Edit Dynamic Table.

      3. The table opens in edit mode.

    3. Go to where you want to add a column.

    4. Select Add () at the upper table border.

      5. A column is added to the table and the column settings display on the page.

  2. Select the header of the new column and change the column type to Note reference.

  3. Enter a header and specify the desired width.

  4. No seleccionar nada.

    5. The new column is added in the table with empty references (#).

  5. Specify the note references for each row.

    1. Select an empty reference (#) to display a list the existing notes in the financial statements document.

      Tip: If you have set visibility logic for the notes, select the Show hidden notes check box to include hidden notes in the list.

    2. Select one or more notes from the list.

  6. Select SAVE or SAVE & CLOSE.
