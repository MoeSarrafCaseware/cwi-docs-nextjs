Personalizar el texto de los estados financieros

You can set up text content in the financial statements document to create standard text wording and format. Add text sections in the document to hold the text content.

To add a text section in a financial statements area:

  1. Select Add (), then choose Text Section.

  2. Select the text section to start editing the text as desired.

    3. The formatting toolbar displays. Use the toolbar to make the following changes:

The default font size for each style is as follows:

  • Paragraph - 11pt

  • Heading 3 - 18pt

  • Heading 4 - 13.5pt

  • Heading 5 - 10.5pt

To ensure there are no page breaks in a text area when printing, select More actions () and enable Keep Together. Note that Keep Together does not prevent page breaks if the text area is longer than a full page.

If you want to cut the text from an existing section and paste it into a new section, select More actions () | Cut section. You can then select the Paste section option when adding a new section to paste the text section. The section will be removed from its original location and moved to the new section. Using this option, instead of manually copying the section, ensures that the pasted text section is carried forward correctly.

To delete a text area, select More actions () | Delete.
