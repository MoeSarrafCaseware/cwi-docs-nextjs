Change an existing note reference in a text section

You can change existing note references in text sections in financial statements.

To change an existing note reference in text sections:

  1. Seleccione el saldo que desea cambiar.

    A dialog opens showing an option to navigate to the referenced note and an option to edit the note reference.

  2. In the dialog, select Edit to view a list of the available notes.

    Tip: If you have set visibility logic for the notes, select the Show hidden notes check box to include hidden notes in the list.

  3. Select the check boxes to reference one or more notes.

  4. Select Save once you're done.
