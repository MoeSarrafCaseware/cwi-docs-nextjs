Copiar tabla dinámica

You can copy an existing table and paste it anywhere in the document. Once you copy a table, it stays in the clipboard and you can paste it as many times as you want until you copy a different table to clipboard.

Copiar tabla dinámica

  1. Seleccionar el proyecto archivado que desea restaurar.

    Icons for table options display on the right.

  2. Select More actions ()| Copy Dynamic Table.

  3. Select where you want to add the copied table, then select Add()| Paste Copied Dynamic Table.
