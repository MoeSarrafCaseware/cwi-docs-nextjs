Copiar el contenido de otra plantilla de producto

Before you start, note that:

  • Source template - refers to the template you want to copy from.
  • Destination template - refers to the template you're currently authoring (your product template).

Requisitos previos

  • The source template is published and you have access to it. Para obtener más información, consulte Funciones incorporadas.

  • The Import Documents property is enabled in the destination template. To learn more, see Enable or disable features in your product.

  • If you're copying financial statements, the grouping numbers and structure in the destination template are the same as the ones in the source template.

  • If there are custom forms in the source template, then the values.json file in that destination template match the values.json file in the source template.

Copiar el contenido de otra plantilla de producto

  1. In the Documents page, select Add () | Copy From Another Product.

  2. From the Select Template drop-down, select the template you want to copy from.

    3. Aparece una lista de las dimensiones disponibles.

  3. From the list, select the documents you want to copy.

    4. You can select:

    • individual documents.

    • a folder to copy all documents in the folder.

    • a phase to copy all documents in the phase.

    Note: When you select a folder or a phase, only documents in that phase or folder are copied. File structures are not maintained.

  4. Seleccione OK.

    5. Note: If you recopy a document (that is, copy the same document from the same template again), you will override the existing document and lose any changes you have made to it.
