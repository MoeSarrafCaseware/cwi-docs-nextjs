Create and manage the product template

To start authoring, create a draft product template or open an existing draft (if you're updating an existing draft).

Note: If you're making changes to a template for an existing SE product, note that your changes will only apply Nuevo encargo Template changes do not apply to existing engagements (that is, engagements that have been created from the older product template).

Improvements to primary author and network author template versions

An improvement was introduced to template versioning that enhances the user experience when saving multiple drafts. This change does not have an impact on already published template versions and is meant to improve the template versioning experience.

1.2.3.4-x

  • 1 – Caseware version

  • 2 – Primary author version

  • 3 – Network author version (only displayed if applicable)

  • 4 – Firm author version

  • -x – Saved template version (only displayed if applicable)

The improvement will append the template version with '-x' to identify the saved template version. Each time a draft is saved, the saved template version is incremented by 1.

For example, a new draft is created with template version 1.3.0.0-1.

  • After the template is edited and 'saved', the saved template version is incremented to 1.3.0.0-2, then 1.3.0.0-3, 1.3.0.0-4, and so on.

  • When published, the published template version would align with the latest version saved template version, in this case, 1.3.0.0-4.

  • When a new draft is created, the template version is incremented to 1.4.0.0-1

Crear un borrador

If you don't have a product template draft, you'll need to create one.

Crear un borrador

  1. Navigate to the Cloud menu, then select Settings.

  2. Select your product name, then select Manage Template.

  3. Select Create Draft to create a product template draft.

    An engagement file opens in Primary Author mode. This is your draft product template.

Abrir un un IDEAScript existente.

You can close your draft at any point in time and come back to it later. When you're ready to continue working on the draft, you need to open it.

To open a draft:

  1. Navigate to the Cloud menu, then select Settings.

  2. Select your product name, then select Manage Template.

  3. Select Open Draft to continue working on your draft.

Delete an existing draft

You can delete the draft if you're not satisfied with it and start a new one.

To delete an existing draft:

  1. Navigate to the Cloud menu, then select Settings.

  2. Select your product name, then select Manage Template.

  3. Select Delete Draft.

    4. The Create Draft button is now available on the page.

  4. Select Create Draft to start working on a new product template draft.
