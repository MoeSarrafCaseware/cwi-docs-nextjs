Habilitar saldo de planificación

Planning balances allow auditors to record and track interim balances before the year end.

When the planning balances feature is enabled, users can:

  • Upload planning balances for the current and prior years.

  • View the planning balances in the Data page including multiple options for variance columns to view any changes between the planning and final balances.

  • Use planning balances when viewing analysis charts.

  • Use planning balances in the materiality document.

  • Include information related to planning balances in the risk and risk assessment reports.

  • Link planning balance values in Excel docs using Caseware Cloud Connector.

Tenga en cuenta que:

  • When planning balances are enabled, visibility conditions and dynamic text with Trial Balance conditions are evaluated based on the planning balance until the final balance is imported.

  • The financial statements document currently does not currently support planning balances.

  • Users can only use the planning balances feature if they are importing from an Excel or CSV file and if their engagement has an annual reporting period.

To enable planning balances in your product, select the Information icon () at the top menu and choose Settings to open the Product Settings dialog. At the Trial Balance | Data Page tab, select Enable planning balance.

Once you enable planning balances, you can customize the terminology used for planning balances in the Features | Terminology tab.
