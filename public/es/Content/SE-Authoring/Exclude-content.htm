Contenido excluido

You can exclude areas, content and notes in the financial statement. For example, you can exclude areas that won't be included in the final financial statement output file that will be mailed to the client.

Note: Although this option is available, we strongly recommend that you use visibility conditions to include or exclude content.

To exclude a dynamic table, a page break, a text section or a note heading:

  1. Go to the area where the content you want to exclude is located.

  2. Seleccione las aplicaciones que desea instalar.

    3. For example, if you want to exclude a dynamic table, select the table.

  3. Select More actions ()| Exclude.

To exclude an area or a note in the financial statement:

  1. Go to the area or the note that you want to exclude.

  2. At the top right corner of the page, select More actions () | Exclude.
