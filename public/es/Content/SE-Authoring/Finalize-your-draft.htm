Finalize your draft

Now that you've finished authoring, you need to finalize your engagement file template draft.

Add a file placeholder, a link or upload a file

You can add file placeholders in the template to predefine a location for a document you know the user needs to upload during the engagement workflow. You can also upload files or add links in the template to include relevant or supplementary information for users.

Para cargar archivos:

  1. Vaya a la página Ajustes .

  2. Select the Add icon (), then choose Upload.

  3. Browse and add the file you want to upload.

    4. The file displays on the page. You can move it to a specific folder if you want. See To add an item to a folder.

To add a file placeholder:

  1. Vaya a la página Ajustes .

  2. Select the Add icon (), then choose Placeholder.

    3. El cuadro de diálogo Propiedades del documento.

  3. Enter an identifier for the placeholder in the Identifier field.

  4. Enter a title for the link in the Name field.

    5. If you do not add a title, a default title for the link displays in the engagement file.

  5. Seleccione Guardar.

    6. The file placeholder displays on the page. You can move it to a specific folder if you want. See To add an item to a folder

To add a link:

  1. Vaya a la página Ajustes .

  2. Select the Add icon (), then choose Link.

    3. El cuadro de diálogo Propiedades del documento.

  3. Enter an identifier for the link in the Identifier field.

  4. Enter a title for the link in the Name field.

    5. If you do not add a title, a default title for the link displays in the engagement file.

  5. Enter the web address for the link in the URL field.

  6. Seleccione Guardar.

    7. The link displays on the page. You can move it to a specific folder if you want. See To add an item to a folder.

To add an internal link:

  1. Vaya a la página Ajustes .

  2. Select the Add icon (), then choose Link.

    3. El cuadro de diálogo Propiedades del documento.

  3. Enter an identifier for the link in the Identifier field.

    If you do not add a title, a default title Internal Link displays.

  4. Enter a title for the link in the Name field.

  5. At the Destination drop-down, choose the destination of your internal link.

    Opciones disponibles:

    • Trial Balance - creates an internal link to the Trial Balance page. If you've enabled the Data page in your product, this option links to the Data page instead.
    • CSV Transaction Import - creates an internal link to the transactions import page. Para obtener más información, consulte Ver transacciones de la cuenta.
    • AnalyticsAI Results - creates an internal link to AnalyticsAI results page. To learn more, see Enable AnalyticsAI integration.

    Note: CSV Transcation Import and AnalyticsAI Results options are only available in the list if you have enabled the Data page in your product.

  6. Seleccione Guardar.

    The internal link displays on the page. You can move it to a specific folder if you want.

Organize the items in your template

On the Documents page, you can group related documents, files, links or file placeholders in folders and subfolders to organize and better present the items in your template. You can also move or delete items, both individually and in bulk.

As you organize items, you can use the search option in the right panel to locate documents by searching for them by name, if necessary.

Matching results display as you enter a keyword for search.

Organize items in folders

You can create folders and subfolders and then add items to them.

To create a folder:

  1. In the Documents page, go to the phase where you want to add a folder.

  2. Select Add () | Folder.

  3. Escriba un nombre para el campo nuevo.

  4. Seleccione OK.

To create a subfolder:

  1. In the Documents page, go to the folder where you want to add a subfolder.

  2. Select Add () | Subfolder.

  3. Escriba un nombre para el campo nuevo.

  4. Seleccione OK.

To move an item to a folder or subfolder:

  1. Go to the item that you want to move to a folder or subfolder and select More actions () | Move.

  2. An item can be a document, file, link or placeholder.

    3. Arrows () next to the available target folders (or subfolders) display on the page.

  3. Select the arrow next to the folder or subfolder where you want to move the item.

    The item now displays in the selected folder or subfolder.

You can also rearrange folders and subfolders for better presentation and organization of documents.

If you want to move multiple items at once, you can use the Bulk actions option. Note that only documents, external files, links and placeholders can be deleted using Bulk actions, not folders and subfolders.

To move items in bulk:

  1. On the Documents page, select More actions () | Bulk actions.

  2. Use the checkboxes to select the items you want to move then select Move documents.

  3. In the dialog, select where you want to move the documents to, then select Move.

Delete items in your template

You can also delete an individual item, a folder or a subfolder by selecting More actions () | Delete.

Note: If you delete a folder or a subfolder, all the items in it are moved outside any folder or phase.

If you want to delete multiple items at once, you can use the Bulk actions option. Note that only documents, external files, links and placeholders can be deleted using Bulk actions, not folders and subfolders.

To delete items in bulk:

  1. On the Documents page, select More actions () | Bulk actions.

  2. Use the check boxes to select the items you want to delete then select Delete.

  3. In the confirmation dialog, select Delete.

Rename items in your template

You can rename an item, a folder or a subfolder name by selecting More actions () | Edit, and then entering a different name.

Add visibility logic for documents and items

Visibility settings provide built-in automation so that only the relevant items are available in your template. This is achieved by setting up visibility conditions for various items in the template.

To learn more, see What are visibility settings?

To set visibility for a folder:

  1. Vaya a la página Ajustes .

  2. Select the folder that you want to set visibility for.

  3. Select More actions () | Edit Visibility

    4. The Visibility Settings dialog displays.

  4. Select Hide or Show from the blue drop-down list to specify the visibility action.

  5. Select all or any to determine whether all conditions must be met or one condition must be met for the visibility action (from Step 4), to apply.

  6. Select Add Condition.

  7. Choose a condition type from the list, then specify the condition parameters.

  8. Once you're done, select Save.

To set visibility for an item:

  1. Vaya a la página Ajustes .

  2. Select the item that you want to set visibility for.

    3. An item can be a document, file placeholder or link.

  3. Select More actions () | Edit Visibility to display the Visibility Settings dialog displays.

    4. You can also select Visibility () | Settings ().

  4. Select Hide or Show from the blue drop-down list to specify the visibility action.

  5. Select all or any to determine whether all conditions must be met or one condition must be met for the visibility action (from Step 4) to apply.

  6. Select Add Condition.

  7. Choose a condition type from the list, then specify the condition parameters.

  8. Once you're done, select Save.

You can make an item show or hide despite of the defined visibility settings or even if it doesn't have any visibility settings. For example, if you want an item that's hidden (due to visibility settings) to temporarily show until you finish authoring.

To show or hide an item:

  1. Select the Visibility icon () next to the item.

  2. Select Show or Hide.

You can also select More actions () on the top right corner of the Documents page, then select Hidden Items to display all hidden items.
