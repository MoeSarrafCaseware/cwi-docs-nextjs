How to get started with Cloud Connector?

Caseware Cloud Connector is a Microsoft Office Add-in that provides realtime data linkage between your engagement and MS Excel. With that linkage in place, you can access and retrieve trial balance data, engagement properties and firm properties, then insert the information in a MS Excel file. If a linked value or information changes in the engagement, it's automatically updated in the Excel file.

You can use the Cloud Connector to author financial statements, worksheets and schedules or to verify calculations.

Note that Cloud Connector cannot retrieve the following information:

  • Balances for subsidiaries in consolidation engagements

  • Dimension balances

  • Preliminary balances

  • Ajustes

  • Materiality values

  • Balances from different grouping structures

  • Balances from prior period 2-4. Only current year and prior year balances are supported

Note that two versions of the Connector add-in are available on the Office Add-in Store; one for partner environments and one for production environments. When authoring or testing with Cloud Connector on the partner sites, you must use the partner version of the add-in. Formulas you add using the partner add-in will continue to work when accessed from a production firm with the production version of the add-in.

Before you start, ensure your firm administrator has enabled a Caseware Cloud Connector license for your account. See Enable Cloud apps to learn more.

Procedimientos iniciales

  1. If you are using Windows, make sure you have a local installation of the latest version of Microsoft Office 365 Desktop and are using the latest version of Windows 10.

    If you are using Mac, make sure you have a local installation of version 16.35 or later of Office on Mac and are using a Mac OS that supports Microsoft Office 365.

    Note: MS Office version 2019 or prior and Office Online are not supported.

  2. Open an Excel file in MS Office 365.

  3. Upload the Excel file to your engagement.

  4. Abra un archivo PDF desde la página Documentos .

  5. Get the Caseware Cloud Connector Add-in from the Microsoft Add-ins Store.

    If you are testing the add-in in a partner environment, search "WA200002372" in the store to locate the partner add-in.

    If you are testing the add-in in a production environment, search "WA200002239" in the store to locate the production add-in.

    Note: You can only have one version of the Connector add-in installed. If you will be testing the add-in in both partner and production environments you will need to remove the add-in before installing the other version. To remove the add-in, right-click the add-in and select Remove Add-in.

  6. Use the Caseware Cloud Connector Add-in to insert data from your engagement in the excel file.
