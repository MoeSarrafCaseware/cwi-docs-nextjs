Identify the procedure ID

To identify the procedure ID:

  1. Identify the procedure whose ID you require.

  2. In your browser, open Developer tools and navigate to the Network tab.

  3. Respond to the procedure whose ID you want to obtain. The Network tab populates with the required information about that procedure, including its ID. Identify the ID by expanding the properties of the procedure in the Network tab.

  4. Copy the ID without quotation marks (“ “) and paste it into its specific field in the Primary Author Settings.
