Import groupings (Data page)

You can import groupings from CSV or Excel files. For CSV imports, you need to properly prepare the files first. If you have groupings set up in Working Papers, you can export them to an Excel file and import the file.

To import groupings from a CSV file:

  1. Preparar el archivo CSV de las transacciones
  2. Importar grupos

To import groupings from Working Papers:

  1. Export groupings from Working Papers to Excel
  2. Importar grupos
