Vincular una pregunta de consulta con un procedimiento de la lista de control

You can link a query question to a related procedure in a checklist to help users complete the checklist document.

For example, if there is a procedure in a checklist related to the client's business that you know users won't have an optimal answer to and it's better they request that information from the client, you create a Query template, add this request and link it to the related procedure in the checklist document.

Once users collect the required information from the client using a query document, they add it in the checklist. If the client uploads a file as part of the response, the file displays as a link so users can access it directly from the checklist. See View client responses related to checklist procedures to learn more about how end users can use this feature to complete checklists.

Note: If the linked query question is hidden in the query document, its responses will not display in the checklist document.

Vincular una pregunta de consulta con un procedimiento de la lista de control

  1. Go to the question, then select More actions () | Link to checklist.

  2. Select the search field to display a list of the available checklists in the template.

    You can enter a keyword in the search field to filter the list. You can also select Show hidden documents check box at the bottom to include hidden checklists in the list.

  3. Select a checklist document.

    Once you add a checklist, another search field displays. It allows you to find and select procedures in this checklist.

  4. Select the search field to display a list of the procedures in the checklist.

    5. You can enter a keyword in the search field to filter the list. You can also select Show hidden procedures check box to include hidden procedures in the list.

  5. Select one or more procedures.

  6. Seleccione Guardar.

    The selected procedure are now linked to the question.

To remove the linked procedures, select the question, then select More actions () | Link to Checklist. Once the dialog opens, select Delete.
