New for end users

Here is what's new for end users in SE 2024-09.

Issues for dynamic table rows

Users can now add issues to dynamic table rows. The Issues icon displays when users hover the cursor over the right side of a dynamic table row. They can then select the icon to add an issue.

Tenga en cuenta que:

  • Issues added to a child row are displayed at the parent level when the group of rows is collapsed. When the row is expanded, the issues display at the row they were added to.

  • If a row should be hidden due to visibility logic, it will continue to display if an issue is attached to it and it is either outstanding or resolved. The row will be hidden if the issue is cleared. Note that whether an issue is attached to a hidden row or not, that row will not be included when printing the document.

  • If an issue is added to a row and the row is later deleted, the issue is moved to the dynamic table at the table level.

Note that issues cannot be added to page break rows.

Improvement to syncing trial balance

A warning message now displays to users when they attempt to sync their trial balance from another engagement. After they select the Confirm button, this new message displays and informs them that syncing the trial balance is an irreversible action.

Filter adjustments

A new All types dropdown is now available on the Adjustments page to allow users to filter adjustments by type. When users select to export their adjustments, only the filtered adjustments are included in the generated file. This allows users to select only the types of adjustments they need and provide a copy of them to their clients.

Saved views for the Data page

The Save views icon () is now available on the Data page to allow users to save their current view and access previously saved views.

When users save a view, their selections for the following settings are saved:

  • Columnas

  • Indicadores

  • Filtros

  • Variación

  • Ajustes de división

  • Filas por página:

Saved views are available to all users in the engagement file.
