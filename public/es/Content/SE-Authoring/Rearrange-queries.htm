Rearrange queries

You can move queries across groups in the document. Note that this option is only available if there is more than more group in the document.

To move a query:

  1. Seleccione las bases de datos que desea mover.

  2. Select the Move ().

    3. Arrows indicating the possible target locations where you can move the query display in the document.

  3. Select the arrow next to the desired target location.

    4. The query now displays in the new location.
