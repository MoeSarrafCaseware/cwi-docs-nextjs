Recomendado

For a smooth authoring experience, we recommend that you:

  • Use Google Chrome

  • Create a standardized design documents for content authoring

  • Save your template draft frequently

  • Communicate between team members on saving and deleting drafts

Note that multiple authors can author simultaneously. However, we recommend that authors are always assigned different sections to avoid overwriting each others work. Por ejemplo:

  • In a checklist document, multiple writers can work on the same document as long as they are assigned different procedures

  • In the financial statements, multiple writers can work on the same document at the same time as long as they are assigned different dynamic tables, text sections, analysis sections or note sections

  • In queries, multiple authors can work on the same document at the same time as long as they are assigned different questions
