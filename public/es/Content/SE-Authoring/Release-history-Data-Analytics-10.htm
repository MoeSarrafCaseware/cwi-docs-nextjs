Envision provides a quick visualization of the transaction data that’s been imported into an AnalyticsAI engagement.

Envision is divided into several widgets:

  • Descripción general
  • Resumen
  • Distribuida
  • Campo de fecha de registro

Notebooks

Notebooks are interactive documents that contain live code together with markdown text. Using Notebooks, you can build and explore custom analysis, and display the results. Each notebook consists of a series of cells. Run each cell individually in sequence to view its output or run the entire notebook.

Notebook files have the .ipynb file name extension and can be explored, modified and run in JupyterLab, an open-source notebook editor. The Data Analytics team has re-engineered out-of-the box Analytics AI tests to work with a new Notebooks engine. You can access these from the AnalyticsAI platform.

Note: The initial launch of a CWI Notebook in an engagement by each user may take several minutes. The Data Analytics team is working on reducing the initial load times. Changes made in notebooks, including those in the code cells as well as the markdown cells, are not saved.

Refer to Get started with AnalyticsAI Notebooks SDK for more information .
