  • When the test filter is used for a test that has zero flagged transactions, the widget now shows a "No relevant data" message

The following improvements have been made for this release:

  • A link to the valid csv file requirements has been added to the Import Transactions page. See CSV transaction file requirements for more information.
  • A message no longer appears when a data layout is successfully applied during transaction file import.

Import wizard

This improvements have been made to the Import Transactions wizard:

  • The Data Source search field has been moved into the list of available providers
  • Distributors can now filter the Data Source list by adjusting their app's product.json file. See List of product features (API) - Imports for more information.
  • You can import data from desktop accounting packages using the Import Transactions wizard. This parallels the functionality of the existing Desktop Accounting Packages button on the Data import page. Primary authors can turn on this import in their app from its product.json file. See List of product features (API) - Imports for more information.
