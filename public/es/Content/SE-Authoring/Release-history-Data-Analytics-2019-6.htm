AnalyticsAI 2019.6

This highlights the major changes since the last release of this product.

Detección de valores atípicos

These outlier detection tests have been added for this release:

  • Detección de valores atípicos basada en el ángulo
  • Factor atípico local basado en el grupo
  • Detección de valores atípicos basada en un histograma
  • Valores atípicos del bosque de aislamiento
  • K vecinos más próximos
  • Factor de valor atípico local (LOF)
  • Determinante de covarianza mínima
  • Análisis de componentes principales

See Outlier detection for more information.

Fiscal year filter

If you have data outside of the fiscal year of the audit, you now have the option to only test the transactions within the fiscal year.

In the following example, the engagement properties show a fiscal period of January 1, 2017 to December 31, 2017.

Because this engagement has data from more than just 2017, the following option appears on the Configuration page. Select the checkbox if you wish to run analyses on data solely from the period of the engagement.

Currency symbols removed

For easier Internationalisation, currency symbols have been removed from the Results page.
