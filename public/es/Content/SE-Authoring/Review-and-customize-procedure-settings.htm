Revisar y personalizar la configuración del procedimiento

If you want to view or customize the settings for a specific procedure or sub-procedure in the checklist, select More actions () | Settings, then select the Override the checklist settings check box. Once you're done, select Save.

The following options are available in the Procedure Settings dialog.

Opción Descripción
Permitir aprobaciones Select this option to add a signoff () to the procedure.
Permitir las notas de entrada en los procedimientos Select this option to add a text box below the response where users can enter notes. You can add a note placeholder in the displayed field.
Permitir múltiples filas

Select this option to allow users to provide more than one response for the procedure. This option enables the +Row button in the procedure.

Note: If your procedure has more than one response, selecting this setting displays your responses in a table beneath the procedure. The response placeholder labels are displayed in the header row.
Response sets
  • Select +Response to add a response type.
  • Select Remove () to remove a response.
  • Select Edit () to edit a response.
Mostrar respuestas debajo del procedimiento Select this option to display the responses below the procedure.
