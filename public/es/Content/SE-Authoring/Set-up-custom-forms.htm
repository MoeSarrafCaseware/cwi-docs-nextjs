Set up custom forms

In addition to the built-in document types, you can develop custom forms in your SE product using SE Builder. To learn more about custom form development, see Get started with SE Builder.

You can redirect users to Custom Forms in your product rather than default pages for the following features:

  • Riesgos

  • Controles

  • Materialidad

  • Resumen de incorrecciones

To start, select the Information icon () at the top menu, then choose Settings. Select Features | CustomForms from the Product Settings dialog that opens.

To replace a default page with a custom form, add the form to the Documents page and then select it from the feature's drop down menu.

Note: The Risks section will not appear in the Custom Forms settings unless it is enabled in your product.

You can also add your custom forms to the Add dialog.

To add a custom form to the Add dialog:

  1. Seleccione Añadir nuevo sitio.

  2. Escriba un nombre único para el panel de información. This name will display to users in the Add dialog.

  3. Select the form you want to add from the drop down.

  4. Seleccione Guardar.
