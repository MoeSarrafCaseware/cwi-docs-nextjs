Set up diagnostics

You can set up diagnostics in the financial statements to help users identify and locate discrepancies between balances across the financial statements tables.

To set up diagnostics in financial statements, you need to:

  1. Create the diagnostics labels.
  2. Label the groups and accounts that you want the application to compare their balances to each other.

When you label groups or accounts, the application automatically compares the balances for rows having the same label.

Comparison happens in real time. If you have labeled rows that are sourced from trial balance data, then once users import the trial balance, the results of the comparison display.

To learn more about how users view diagnostics, see View balance discrepancies.

To label rows in financial statements tables:

  1. Select the dynamic table where you want to label accounts or groups.

    2. Icons for table options display on the right.

  2. Select More actions () | Edit Dynamic Table.

    3. Table opens in edit mode.

  3. Seleccione las estadísticas que desea mostrar.

    4. The row can be an account or a group.

  4. Go to the Diagnostic Label field at the top. You can either create a label or select an existing label to assign it to the selected row.

    • To create a label, enter the desired name in the field. The label is created and assigned to the selected row once you press Enter or the Spacebar.

      • Note: Diagnostics labels cannot have spaces.

    • To find an existing label, start typing the keyword in the field to populate a list of matching results. Choose a label from the list to assign it to the selected row.

    • To select an existing label, select the drop-down and choose a label to assign it to the selected row.

      • The assigned label displays at the top replacing the Diagnostic Label field.

      Tip: To remove the assigned label, select Delete () next to the label. Note that if this is a new label (that is, you have not selected it from the drop-down), this button also deletes the label.

  5. Deselect the row.

  6. Select SAVE or SAVE & CLOSE.

Important notes for diagnostics setup

Note the following when setting up diagnostics,

  • The application only compares rows having the same label. Make sure you assign the same label to the rows that you want the application to compare them to each other.
  • La aplicación compara los valores absolutos de las filas aplicables. Las diferencias de signo no se marcan como discrepancias de saldo.
  • If you label a group, the application compares the total balance of the group to other rows having the same label. It does not compare the balance of each account in the group to rows having the same label.
  • You can assign one label to a row.
