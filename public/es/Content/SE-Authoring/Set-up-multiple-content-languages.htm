Set up multiple content languages

Multi-language support aims at facilitating SE product offerings in regions and markets where there is a requirement to present deliverables and content in multiple languages. To learn more, see What is multi-language support.

To start, select the Information icon () at the top menu, then choose Settings. The Product Settings dialog opens.

To add a language:

  1. Go to the Content Language tab.

    2. If this is the first time you're making changes in the Content Language tab, you'll find English already added and marked as the default language.

  2. Select +Language.

  3. En la lista desplegable Servidor seleccione el servidor con el cual desea conectar.

    4. The Active check box is selected by default for new languages.

  4. If you want the new language to be the default language for your product, select Default.

    5. Note: The default language cannot be inactive. If you select Default next to an inactive language, the language automatically becomes active and the Active check box is disabled.

If a language is no longer available in your market, you can select Remove () to delete it.

Tenga en cuenta que:

  • You cannot modify a language you've added. To make changes or corrections, delete the language and add a new one.

  • You can clear the Active check box to mark a language as inactive. For example, if the language is yet to be introduced in your market.

  • Inactive languages do not display in the Engagement Properties popup and the language visibility condition parameter.
