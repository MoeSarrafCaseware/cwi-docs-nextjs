Set up the Summary of Misstatements document

You can enable the summary of misstatements document which summarizes the impact of misstatements (omissions) of information in the financial statements areas, and presents a conclusion (whether or not the misstatements are considered to be material or immaterial).

To enable the Summary of Misstatements report in your product, specify the com.caseware.ca.general.misstatements-summary in the form property in the product.json file. To learn more, see Enable or disable features in your product.

Once you enable the document in your product, you can add it in your draft and set it up.

To add the summary of misstatements document, select Add () at the documents page, then choose Summary of Misstatements.

To set up the Summary of Misstatements document:

  1. Add guidance information.

    2. Select the text field to start adding content. The formatting toolbar displays. Use the toolbar to make the following changes:

  1. Use the Applicable toggle () to specify which sections are included or excluded from the document.

  2. In the Actual and Proposed Journal Entries, select the Specify classification groups option to specify the classification trial balance groups for the actual and proposed journal entries.

    Note: The Use group classification option is currently not functional.

  3. Select +New group, then select the text field to display a list of the trial balance groups. Choose the desired group for Assets, Liabilities, Equity, Revenues and Expenses.

  4. Rename the items under Key Data to customize the terms if needed.

    5. Once you rename an item, a Revert icon () is available so you can revert back to the original name.

  5. Especifique los siguientes detalles:

    • Evaluación cualitativa

    • Summary of Omitted Disclosures (Description of Omission or Other Misstatement, Other Area Affected, Discussion with Client, Reason for Not Disclosing)

    • Evaluación

    • Conclusión

    • Aprobación

Select the text field to start adding content. The formatting toolbar displays. Use the toolbar described in Step 1 to format and edit the text.
