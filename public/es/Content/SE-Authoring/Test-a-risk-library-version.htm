Test a risk library version

If you want to test an unpublished risk library version, you can do so in an engagement file in your development firm.

Note that you can only test unpublished risk library versions if there is an existing published version of the library.

Before you start, ensure the risk library is associated to the product in your development firm.

To view an unpublished risk library version:

  1. Open an engagement file for the product in your development firm.

  2. Select Add () | Risk from Library.

    You can also select Risk Library from the Risks page to open the risk library.

  3. In the Risk Library Settings dialog, select the unpublished version you want to view.

  4. Seleccione OPEN.
