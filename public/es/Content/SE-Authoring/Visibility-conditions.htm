Visibility conditions

The following listing describes the visibility conditions, the available parameters for each condition and shows an example for each condition.

Riesgo de incorrección material

Select an assertion type, a group from the trial balance and one of the following conditions:

  • RMM - selected group has RMM for this assertion type.
  • No RMM - selected group has no RMM for this assertion type.

If you've enabled the new risk module in your product, a difference set of parameters are available for this condition:

  • Seleccione el tipo de extracción.
  • Select the area or business cycle.
  • Specify the logical operation to compare and evaluate (=,<=,>=).
  • Seleccione la escala de tiempo.

The values in the RMM scales list are defined in the RMM scales setup in Product Settings.

Ejemplo

If Completeness for Cash has No RMM, then show this item.

Or, if you've enabled the new risk module:

If Accuracy for Inventory has RMM less than Medium, then hide this item.

Grupos de contabilidad

Select a group from the trial balance, specify the year (current or prior) and one of the following balance values:

  • es cero
  • es distinto de cero
  • es material
  • no es material

Ejemplo

If the balance for the Short term investment account for the Current year is material, then show this item

Respuesta

Select a source document, a procedure in the document and a response for the procedure.

Ejemplo

In the 200 Optimizer document, if the response to What is the type of engagement as defined under SSARS21? procedure is Compilation, then hide this item.

Note that if a response has been changed after you've set up a Response visibility condition that references that response, the value Response Not Recorded displays.

Consolidación

Specify whether the engagement is Consolidated or Non-consolidated, where:

  • Non-consolidated balance = Parent entity + Parent entity adjustments (excluding eliminating adjustments).

  • Consolidated balance = Parent entity + Parent entity adjustments + Subsidiary entities + Subsidiary entities adjustments + Eliminating adjustments to the Parent and Subsidiary entities.

Ejemplo

If the engagement is Consolidated, then show this item.

Escala/redondeo

Select a rounding scale from the drop-down.

Ejemplo

If the rounding is in Millions, then display this item.

Período de informe

Select the reporting period:

  • Mensual
  • Trimestral
  • Semestral
  • Anual

Ejemplo

If the reporting period is Annual, the show this item.

Fecha

To set a date condition, select the applicable parameter option from the available drop-downs.

  1. Seleccione el ancho del borde:
    • Efectivo
    • Expires on
  2. Select the period type options for fiscal year and interim periods within a fiscal year:
    • Fecha de inicio del ejercicio
    • Fecha de cierre del ejercicio
    • Fecha de inicio de período
    • Fecha de fin de período
  3. Select the mathematical operations used to compare and evaluate (<,>, <=,>=).
  4. Seleccione la fecha inicial

Ejemplo

This item shows if effective date for the engagement year start date is after November 30-2019.

Tipo de organización

Select country and select organization type.

Ejemplo

If the entity type is Canadian Private Corporation, then show this item.

Idioma

Select a content language from the drop-down.

Ejemplo

If the content language is French, then display this item.

Tenga en cuenta que:

  • The content language means the language currently selected in the Engagement Properties popup up

  • Only active languages are available for selection in the drop-down

Grupo de condiciones

Add the conditions to set up the group

Note: All condition types are available for you to add.

Ejemplo

This item shows if the engagement type is Review and the entity type is Canadian Private Corporationor if the year star tdate is after October-21-2019 .

Valor de formulario

You can include one or more forms in the condition setup. Select the form name, a field in the selected form and a value for the selected field.

The value options for a selected field depend on the field type. For example, numeric fields have logical operators and numerical values, while date fields have a calendar icon for date selection.

Note: If you have not developed forms using SE Builder in your product, the parameter drop-downs for this condition are disabled.

Ejemplo

If the field T2S8 in S8 - Capital cost allowance (CCA) form is > 10 or the field T3S8 in S8 - Capital cost allowance (CCA) form is > 10, then show this item.
