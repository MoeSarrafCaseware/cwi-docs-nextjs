What's dynamic text?

Dynamic text is conditional text. It's a field where different text displays based on a condition or a set of conditions. You can include dynamic text in various text fields while authoring checklists, queries, letters or memos and financial statements.

To find out how to include dynamic text in various text fields in your template, see:
