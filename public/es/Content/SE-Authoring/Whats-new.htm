Autor de la red

#networkauthor

SE 2024-09 features the official launch of the network author role.

The network author role enables accountancy networks and associations, as well as large firms with multiple offices, to customize product templates provided by primary authors and share those templates with all their member firms. This allows accountancy networks to easily maintain a standard methodology across their member firms, while still giving member firms the flexibility to make further customizations as required for their region.

To manage the network firm template in a central location, accountancy networks will have a Cloud site specifically for network authoring in addition to Cloud sites for each of the member firms in the network. Users from the member firms that the network wants to designate as network authors can be added to the network authoring firm site.

In the network authoring firm, network authors can access the template provided by primary authors on the Manage template page in their settings. From this page, they can:

  • Create a draft and make customizations to the template

  • Save the template so they can test it

  • Publish the template and make it available to all the member firms in their network

  • Restore previous versions of the template

Note: Any customizations made by network authors cannot be overridden by new updates made to the product template by primary authors.

Once their work is complete, network authors can then share this template with their member firms, where firm authors can make additional customizations. Note that member firms will always receive their templates from the network authoring firm and not directly from primary authors.

Additional authoring capabilities

In addition to the features already available to firm authors, network authors also have access to customize the following features:

  • Riesgos
    Network authors have the ability to customize all items on the Assessment, Fields, and Scales tabs of the Risks section in the product settings. Note that while network authors also have access to the General tab, the option to enable the new Risks page and the Library management section is still only available to primary authors.

  • Controles
    If controls are enabled by primary authors, network authors can customize all controls settings. Note that they do not have the ability to disable controls.

  • Afirmaciones
    All items in the Assertions tab of the product settings are available to network authors.

  • Documents page
    Network authors can modify the names, identifiers and visibility settings of primary author-created documents, as well as reorder primary author-created folders and subfolders. They can also add, rename and reorder phases from the product settings dialog.

  • Listas de verificación
    Network authors can modify visibility settings for items within a checklist and can also add annotations for procedures.

Translation for the financial statements

#primaryauthor #enduser

You can now enable automatic machine translation for the financial statements. When you make this feature available in your product, users can translate text sections in the financial statements to any of the content languages available in your product.

To enable this feature, select the Enable translation checkbox in the Financial Statements tab of the product settings and ensure you have more than one content language enabled in your template.

Users can then select the Translation option in the financial statements settings to enable translation for the document. The Translation icon then appears next to text sections in the document.

When users select the translation icon, the Translation dialog opens. From this dialog, they can select which language to translate the content to, view the translation, and select Save to apply it to the text section.

The translated version of the text section displays in the engagement when the user switches the content language of the engagement to the target language. They can then make manual edits to the text if required.

Custom calculations in financial statement text sections

#primaryauthor #firmauthor #networkauthor #enduser

You can now add custom calculations to text sections in the financial statements. Similarly to custom calculations in dynamic tables, you can reference trial balance values and dynamic table cells in your calculations. These calculations will update automatically if the referenced values change.

Cell references within the financial statements

#primaryauthor #firmauthor #networkauthor #enduser

This release introduces cell referencing within the financial statements. You can now reference values from one dynamic table into another. This eliminates the need to manually enter the same value multiple times in different tables in the financial statements.

Custom formulas for the Overall Materiality Table

#primaryauthor #firmauthor #networkauthor #enduser

You can now use custom formulas in the Overall Materiality Table.

Materiality and performance materiality for specific circumstances for areas

#primaryauthor #firmauthor #networkauthor #enduser

In the materiality document, you can now document materiality and performance materiality for specific circumstances for areas.
