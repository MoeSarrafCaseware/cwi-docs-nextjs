Copiar el punto final de metadatos (Azure AD)

A metadata endpoint provides authorization URLs and other metadata necessary for SSO to function. You'll use it when you configure SSO in Cloud.

To copy the metadata endpoint:

  1. Sign in to your Azure portal.

  2. Select Azure Active Directory.

  3. From the left-hand navigation pane, select App registrations.

  4. Seleccione el campo que desea en la lista.

  5. Select Endpoints.

  6. A list of endpoints displays.

    Important: Copy the OpenId Connect metadata document endpoint and paste it to a text document for later use. Label it Metadata Endpoint.
