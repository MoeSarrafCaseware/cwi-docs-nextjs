Crear una aplicación Web API (ADFS)

Create a Web API application for your firm's Cloud site and define the application permissions.

Crear una aplicación Web API (ADFS)

  1. In the ADFS Administration Console, click the Application Groups folder, then double-click your new application group.

  2. In the Properties dialog, click Add Application.

  3. In the Template list, select Web API. Haga clic en Siguiente.

  4. In the Identifier field, paste the Client Identifier that you copied to a text document previously. Haga clic en Siguiente.

  5. In the list of access control policies, select the policy that best applies to your firm. Haga clic en Siguiente.

  6. In the Permitted scopes list, select the following scopes:

    • allatclaims

    • Correo electrónico:

    • openid

    • Perfil

    Haga clic en Siguiente.

  7. Haga clic en Sí para completar la configuración.
