Integrar el inicio de sesión único

Cloud supports single sign-on (SSO) with Azure Active Directory (Azure AD), Active Directory Federation Services (ADFS) and Okta. Using SSO with Cloud reduces the number of passwords staff members are required to memorize and helps administrators manage account issues (such as password resets) from a single location.

Tenga en cuenta que:

  • The authorization protocol we use for SSO integration is OpenID Connect.

  • Time Desktop is not compatible with SSO. Once SSO is enabled, SSO users will no longer be able to log in to Time Desktop.

To get started, contact Caseware Support to request that single sign-on is enabled for your firm.
