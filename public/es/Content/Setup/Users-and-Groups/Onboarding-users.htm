Incorporación de usuarios

Every accounting firm has its own security policy and mode of operation. How firm administrators onboard users depends on these two factors.

Large firms

The following steps summarize how to onboard users in large firms.

  1. Crear funciones de seguridad personalizadas

  2. Create groups based on functional roles and access rights.

  3. Crear una cuenta de usuario

  4. Asignar funciones de seguridad a todo el sistema

  5. Grant contacts the appropriate access rights to entities.

  6. Grant users or groups access to specific files in an entity.

Small-size firms

The following steps summarize how to onboard users in small-size firms:

  1. Crear funciones de seguridad personalizadas

  2. Create groups based on functional roles and access rights (Optional).

  3. Crear una cuenta de usuario

  4. Asignar funciones de seguridad a todo el sistema

    5. Note: If you have created user groups, then assign system-wide security roles to the staff groups instead of individual staff.

  5. Grant contacts the appropriate access rights to entities.

  6. Grant users or groups access to specific files in an entity.
