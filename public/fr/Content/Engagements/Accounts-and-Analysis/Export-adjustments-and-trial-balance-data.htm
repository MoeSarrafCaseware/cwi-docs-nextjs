Exporter les ajustements et les données de la balance

Les auditeurs peuvent exporter les écritures d'ajustement et les données de la balance de vérification dans un fichier et le fournir à leurs clients. Les clients peuvent ensuite mettre à jour leurs livres et leurs systèmes comptables en fonction des changements apportés au cours de la mission.

Ajustements à l'exportation

Vous pouvez exporter les ajustements vers un fichier CSV ou un fichier PDF. Vous pouvez également exporter au format CSV ASCII, qui est compatible avec les Working Papers.

Remarque : Vous pouvez filtrer vos ajustements par type en utilisant le filtre déroulant Tous les types. Vos sélections de filtres seront reflétées dans votre exportation.

Pour exporter des ajustements :

  1. Accédez à la page Ajustements .

  2. Sélectionnez Exporter (), puis choisissez l'option d'exportation que vous souhaitez. Voici vos options :

    • Exporter au format CSV

    • Exporter au format PDF

    • Exportation vers les documents de travail (CSV ASCII)

Les données sont exportées vers le type de fichier CSV sélectionné et le fichier est automatiquement téléchargé. Si vous avez choisi l'exportation PDF, le fichier PDF de sortie s'ouvre.

Exporter les données de la balance générale (page Données)

Vous pouvez exporter les comptes de la balance de vérification vers un fichier CSV, PDF ou un fichier compatible avec Working Papers (CSV ASCII).

Pour exporter des groupes de balances :

  1. Sélectionnez Comptes dans le panneau de gauche de la page Données .

  2. Sélectionnez Exporter (), puis choisissez l'option d'exportation que vous souhaitez. Voici vos options :

    • Exporter au format CSV

    • Exportation vers les documents de travail (CSV ASCII)

    • Exporter au format PDF

      Remarque : L'option Exporter au format PDF génère un PDF qui reflète les colonnes que vous avez sélectionnées pour l'affichage sur la page Données . Si la mission est consolidée, le PDF indique également si vous avez choisi de visualiser les données consolidées ou les données d'une entité sélectionnée dans la hiérarchie de consolidation.

Les données de la balance des comptes sont exportées vers le type de fichier sélectionné et le téléchargement commence immédiatement.
