Exporter des groupes de balances

Dans la page Données , vous pouvez exporter les informations financières dans les onglets du groupe. Par exemple, pour l'envoyer aux clients ou pour l'inclure dans les rapports.

Options disponibles :

  • Exporter vers CSV - permet d'exporter des données vers un fichier CSV. Les groupes de balances de vérification qui n'ont pas de comptes ne sont pas inclus dans le fichier CSV.
  • Export to Working Papers (CSV ASCII) - permet d'exporter des données vers un fichier CSV compatible avec Working Papers. Les groupes de balances de vérification qui n'ont pas de comptes ne sont pas inclus dans le fichier CSV.
  • Exporter au format PDF - permet d'exporter des données au format PDF. Les groupes de balances de vérification qui n'ont pas de comptes ne sont pas inclus dans le PDF.
  • Exporter la structure vers CSV - permet d'exporter la structure des groupes et ses propriétés vers un fichier CSV. Il n'inclut pas les comptes et les ajustements.

Pour exporter des groupes de balances :

  1. Dans la page Données , sélectionnez un onglet de groupe dans le panneau de gauche.
  2. Sélectionnez Exporter (), puis choisissez l'option d'exportation souhaitée.

    3. Le téléchargement du fichier commence immédiatement.

    Remarque : Les options Exporter vers CSV, Exporter vers les documents de travail (CSV ASCII) et Exporter vers PDF préservent les sélections et les filtres actuels sur la page. Par exemple, si vous avez choisi d'afficher la colonne des ajustements à l'aide du menu déroulant Colonnes , les ajustements sont inclus dans le fichier exporté. Si vous avez activé l'option Groupes parents ou Comptes , les groupes parents ou les comptes sont exportés.
