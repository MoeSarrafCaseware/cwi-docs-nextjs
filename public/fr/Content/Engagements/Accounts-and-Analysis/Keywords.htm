Mots-clés

Le test analytique Mots-clés analyse les mots-clés donnés d'un ou de plusieurs champs. Ces mots-clés peuvent être saisis manuellement, choisis dans une liste prédéfinie de mots-clés ou dérivés d'une liste de termes produite à partir des champs sélectionnés.

Ce test comprend également un paramètre qui vous permet de choisir entre les correspondances exactes et les correspondances partielles lors de la recherche de mots-clés.

Ce test analytique peut être utilisé pour repérer :

  • Les transactions qui contiennent des estimations significatives

  • Les transactions fictives

  • Les transactions qui sont de nature complexe ou inhabituelle

  • Les transactions entre sociétés

  • Les transactions utilisées pour réconcilier les livres auxiliaires. Par exemple, vous pouvez rechercher les mots-clés « rapprocher le grand livre auxiliaire » ou « établir la balance du grand livre auxiliaire »

Champs utilisés pour l'analyse

Les champs suivants sont obligatoires pour cette analyse :

  • Champs de référence – Champs uniques utilisés pour créer un identifiant de transaction unique, comme le champ ID de l’écriture pour l’ensemble de données du grand livre général. Ces colonnes ne font pas partie du résultat, mais elles sont utilisées pour trouver les transactions qui font partie du résultat. Ce champ est déjà défini dans le test et ne peut pas être modifié.

  • Champs de mots-clés – Les champs de caractères dans lesquels les mots-clés sélectionnés sont recherchés.

Paramètres

Les paramètres suivants doivent être définis pour exécuter ce test :

  • Liste de mots-clés – Une liste de mots-clés enregistrée dans un fichier texte.

  • Correspondance exacte – Indiquez si les valeurs doivent correspondre exactement au mot-clé. Si vous ne choisissez pas de déterminer uniquement les correspondances exactes, les correspondances partielles sont également relevées.

  • Ignorer la casse – Indiquez si les correspondances des mots-clés sont sensibles à la casse ou non

  • Créer une colonne de mots correspondants – Créer une colonne supplémentaire qui répertorie les mots pour lesquels des correspondances ont été trouvées pour cette configuration.

Configurations de test

La seule configuration possible pour ce test est la recherche par mots-clés.

Spécifications techniques

Lorsque vous exécutez le test analytique des mots-clés, il faut réaliser les étapes suivantes :

  1. Vérifiez que les champs de référence nécessaires ont été sélectionnés. Si les champs n'ont pas été sélectionnés, créez un champ de référence unique. Cette étape est réalisée seulement si des champs spécifiques ont été sélectionnés. Si tous les champs sont accessibles, cette étape n'est pas nécessaire.

  2. Vérifiez qu'un ou plusieurs champs de caractères ont été sélectionnés pour l'analyse.

  3. Obtenez la liste de mots, selon l'interface, elle peut provenir de plusieurs sources ou d'un mélange d'une ou plusieurs sources :

    • Une liste préprogrammée de mots-clés.

    • Une liste de mots-clés téléversée par l'utilisateur.

    • L'utilisateur saisit les mots-clés directement dans l'interface.

    • Une combinaison d'un ou de plusieurs des éléments ci-dessus.

  4. Permettez à l'utilisateur d'ajouter ou de supprimer des mots-clés de la liste, en particulier s'ils proviennent de l'étape 3a ou 3b.

  5. Demandez si la correspondance est sensible à la casse ou non.

  6. En fonction de la sélection des correspondances exactes ou partielles, effectuez la recherche des mots-clés à partir de l'étape 3.

  7. Si une colonne de mots-clés correspondants a été sélectionnée pour être ajoutée, seule une colonne est ajoutée, même si plusieurs colonnes ont été sélectionnées pour la recherche.
