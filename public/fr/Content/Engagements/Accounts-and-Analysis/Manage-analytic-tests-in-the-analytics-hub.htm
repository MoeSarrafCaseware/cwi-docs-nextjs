Gérer les tests analytiques dans l'Analytics Hub

Remarque : Cette fonction n'est disponible que si elle a été activée dans votre produit.

Le centre d'analyse est un emplacement central que vous pouvez utiliser pour visualiser et gérer tous les tests d'analyse disponibles dans votre engagement. From the Analytics Hub, you can

  • Voir tous les tests analytiques disponibles dans votre engagement

    Remarque : Les tests liés à des procédures de listes de contrôle cachées ne sont pas affichés dans le centre d'analyse

  • Visualiser l'état et les résultats des tests

  • Tests des filtres

  • Ajustement des configurations de test

  • Naviguer vers les procédures de la liste de contrôle liée

Pour ouvrir le Hub analytique, sélectionnez Hub analytique dans le menu supérieur. Lorsque vous ouvrez la page, la liste de tous les tests disponibles dans votre engagement s'affiche sur la gauche. Vous pouvez effectuer une recherche ou utiliser les filtres pour trouver le test que vous recherchez.

Une fois que vous avez ouvert un test, vous pouvez ajuster la configuration ou naviguer vers les procédures de la liste de contrôle dans l'onglet Paramètres du test. Pour exécuter le test, vous pouvez sélectionner le bouton Exécuter , puis afficher les résultats dans l'onglet Résultats du test. Notez que vous pouvez également sélectionner plusieurs tests à l'aide des cases à cocher et les exécuter en même temps.

Pour ouvrir la page des résultats agrégés , sélectionnez le bouton Résultats en haut de la page. Cette page affiche toutes les transactions signalées à la suite de tous les tests que vous avez effectués. Vous pouvez utiliser les options Colonnes et Filtres pour filtrer les résultats affichés et utiliser le bouton Exporter pour exporter les résultats vers un fichier CSV.

Filtrer dynamiquement les données pour les tests analytiques

Vous pouvez filtrer l'ensemble de données source pour un test analytique, puis exécuter le test sur le sous-ensemble de données identifié par le filtre.

Filtrer dynamiquement les données pour un test analytique :

  1. Dans le Hub Analytique, sélectionnez le test analytique dont vous souhaitez filtrer l'ensemble des données d'entrée.

  2. Dans Filter dataset, ils sélectionnent + Add filter pour créer une condition de filtrage qui sera appliquée à une seule colonne de l'ensemble de données d'entrée.

  3. Select the column where you want to apply a filter.

  • Une ou plusieurs conditions de filtrage sont prises en charge dans un filtre.

  • Les conditions de filtrage multiples peuvent être reliées par des connecteurs logiques tels que AND, OR ou NOT.

  • Chaque condition de filtrage ne s'applique qu'à une seule colonne de l'ensemble de données d'entrée.

  • Les arguments conditionnels varient selon le format des données. Par exemple, dans une colonne de date, vous pouvez spécifier un intervalle.

  1. Créez la condition de filtrage et sélectionnez Filtre pour l'exécuter.

Le nombre de lignes de données correspondant à la condition s'affiche.

  1. Vous pouvez ensuite exécuter le test analytique sur ce sous-ensemble de données en cliquant sur Exécuter.
