Afficher les résultats des tests analytiques pour les listes de contrôle

Remarque : Cette fonction n'est disponible que si elle a été activée dans votre produit.

Les tests analytiques vous permettent d'identifier et d'analyser les informations relatives aux données de votre mission et d'améliorer votre capacité à atteindre les objectifs de l'audit.

L'icône d' analyse () s'affiche à côté de toutes les procédures de la liste de contrôle pour lesquelles un test d'analyse est disponible.

Sélectionnez l'icône pour afficher les résultats du test.

Vous pouvez exporter les résultats du test vers un fichier CSV en sélectionnant Autres actions () | Résultats CSV. Vous pouvez également reconfigurer le test si nécessaire à partir de l'onglet Configuration et l'exécuter à nouveau. Notez que les options de configuration ne sont pas disponibles pour tous les tests.
