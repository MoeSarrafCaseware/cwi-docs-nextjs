Consulter les données du grand livre et du livre auxiliaire du client

Si vous avez importé les données du grand livre et de la comptabilité auxiliaire de votre client, dans l'onglet Ensembles de données , sur la page Données , cliquez sur Afficher les données.

Remarque : The View data button is only visible on the Datasets tab if you've successfully imported data.

Trier les données

Tri des colonnes

Les colonnes par lesquelles les données de l'ensemble de données peuvent être triées sont marquées d'une flèche vers le haut () pour indiquer que les données sont triées de manière ascendante, ou d'une flèche vers le bas () pour indiquer que les données sont triées de manière descendante. Cliquez sur l'en-tête de la colonne pour modifier le sens du tri.

Tri multi-colonnes

Un nombre et une flèche de tri () apparaissent sur plusieurs colonnes pour indiquer un tri sur plusieurs colonnes. Dans l'exemple ci-dessous, les données sont triées par ID d'entrée , puis par numéro de ligne par défaut.

Pour modifier la colonne triée en premier :

Survolez la colonne marquée 2 et maintenez la touche Shift enfoncée tout en cliquant sur l'en-tête de la colonne.

Options au niveau des colonnes

Sélectionnez la liste déroulante () sur l'en-tête d'une colonne pour faire apparaître une liste d'options.

Sélectionnez Pin column pour que la colonne devienne la première colonne à gauche.

  • Sélectionnez Masquer pour masquer le document.

  • Sélectionnez Filtrer par colonne pour filtrer les lignes de données et n'afficher que celles qui contiennent la valeur spécifiée. Dans l'exemple ci-dessous, seules les lignes de données dont le numéro de ligne contient la valeur `96` seront affichées.

Filtrer l'ensemble des données

Vous pouvez filtrer l'ensemble des données en cliquant sur Filtres puis, dans le panneau Filtres, en sélectionnant n'importe quelle colonne de l'ensemble de données. Tous les filtres de colonne utilisent l'opérateur CONTAIN, tandis que l'ensemble des filtres utilise l'opérateur AND.

Exclure des colonnes de la vue

Vous pouvez exclure des colonnes des données affichées en cliquant sur Colonnes, puis en sélectionnant celles que vous ne souhaitez pas voir apparaître.
