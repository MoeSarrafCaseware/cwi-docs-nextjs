Ajouter ou modifier un contexte

Remarque : Cette fonction n'est disponible que si elle a été activée dans votre produit.

Vous pouvez ajouter de nouveaux contextes et copier des contextes existants à partir du panneau de révision XBRL.

Ajouter un nouvel élément

Si les contextes disponibles ne répondent pas à vos besoins, vous pouvez ajouter des contextes personnalisés à partir du panneau de révision XBRL. Si vous souhaitez créer des contextes dimensionnels, reportez-vous à la section Ajouter un contexte dimensionnel ou un contexte dimensionnel typé.

Avant de commencer, sélectionnez l'icône XBRL () dans le menu supérieur, sélectionnez la taxonomie que vous souhaitez réviser et Démarrer la révision pour ouvrir le panneau de révision.

Ajouter un nouveau format

  1. Dans les états financiers, sélectionnez le contenu qui nécessite un contexte personnalisé.

  2. Dans la section Contexte du panneau de révision, sélectionnez Autres actions () | Nouveau contexte.

  3. Saisissez les détails du contexte si nécessaire.

    Notez que chaque contexte doit avoir un nom unique.

    Avertissement : Veillez à ne pas modifier le champ Identifiant de l'entité . La modification de ce champ entraînera des erreurs de diagnostic et de soumission lors de l'exportation du rapport XBRL.

  4. Select Add context(s).

Le contexte que vous avez créé est maintenant disponible pour que vous puissiez l'ajouter aux étiquettes.

Copy an existing context

Si vous souhaitez ajouter un contexte similaire à un contexte existant, vous pouvez copier le contexte existant, puis le modifier pour créer votre nouveau contexte.

Avant de commencer, sélectionnez l'icône XBRL (icône) dans le menu supérieur, sélectionnez la taxonomie que vous souhaitez réviser et Démarrer la révision pour ouvrir le panneau de révision.

Pour copier un contexte existant :

  1. Dans la section Contexte, sélectionnez Editeur de contexte.

  2. Naviguez jusqu'au contexte que vous souhaitez copier dans la liste et sélectionnez l'icône Copier ().

    Remarque : Vous pouvez sélectionner l'icône étoile () d'un contexte pour le mettre en favori. Les contextes favoris s'affichent en haut de la liste des contextes dans l' éditeur de contexte et dans le menu déroulant Contexte lorsque vous ajoutez ou modifiez une balise.

  3. Modifiez les champs de contexte et les dimensions requis, qui seront pré-remplis avec les informations du contexte sélectionné à l'étape 3.

    Avertissement : Veillez à ne pas modifier le champ Identifiant de l'entité . La modification de ce champ entraînera des erreurs de diagnostic et de soumission lors de l'exportation du rapport XBRL.

  4. Sélectionnez Ajouter contexte(s) pour créer le contexte

Vous pouvez maintenant attribuer des balises au contexte nouvellement créé.

Edit an existing context

You can also edit any contexts you add.

Remarque : Vous ne pouvez pas modifier les contextes système préexistants.

Avant de commencer, sélectionnez l'icône XBRL (icône) dans le menu supérieur, sélectionnez la taxonomie que vous souhaitez réviser et Démarrer la révision pour ouvrir le panneau de révision.

Pour modifier un commentaire :

  1. Dans la section Contexte , sélectionnez Editeur de contexte.

  2. Naviguez jusqu'au groupe que vous souhaitez modifier et sélectionnez Modifier (). Vous pouvez utiliser l'option de recherche si nécessaire.

    Remarque : Vous pouvez sélectionner l'icône étoile () d'un contexte pour le mettre en favori. Les contextes favoris s'affichent en haut de la liste des contextes dans l' éditeur de contexte et dans le menu déroulant Contexte lorsque vous ajoutez ou modifiez une balise.

  3. Une fois que vous avez sélectionné un contexte, les détails de ce contexte s'affichent sous la liste des contextes. Modifiez les champs Nom, Période et Applicable si nécessaire.

    Avertissement : Veillez à ne pas modifier le champ Identifiant de l'entité . La modification de ce champ entraînera des erreurs de diagnostic et de soumission lors de l'exportation du rapport XBRL.

  4. Le cas échéant, sélectionnez Modifier les dimensions pour modifier toutes les dimensions du contexte, puis sélectionnez Confirmer une fois que vous avez effectué vos modifications.

  5. Sélectionnez Enregistrer.
