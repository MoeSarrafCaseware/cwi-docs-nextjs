Répondre aux requêtes de fichiers

En tant que contact, voici comment répondre aux demandes de dossiers qui vous sont attribuées par le personnel de l'entreprise. Les demandes de dossier sont signalées par l'icône Demande de dossier () dans la liste Mon travail.

Pour répondre aux requêtes de fichiers :

  1. Sélectionnez Répondre pour ouvrir la demande de fichier.

  2. Sélectionnez le bouton Sélectionner les fichiers ou faites glisser le fichier.

  3. Saisissez tout commentaire dans la zone de texte.

  4. Sélectionnez Répondre pour terminer la demande de dossier.
