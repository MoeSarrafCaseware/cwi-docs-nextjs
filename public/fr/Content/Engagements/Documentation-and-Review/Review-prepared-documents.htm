Examiner les documents préparés

Voici comment examiner les documents préparés dans le cadre de votre mission.

Étape 1 : Localiser les documents préparés

Dans le menu Cloud, sélectionnez Engagements pour ouvrir la page Engagements. L'onglet par défaut sélectionné sur la page est Mon travail. Les missions pour lesquelles vous avez été désigné comme préparateur ou réviseur s'affichent sur la page.

Sélectionnez un engagement pour en savoir plus. Sous Statut du document, vous pouvez voir une ventilation du statut de tous les documents du dossier de mission. Si vous voyez que le dossier contient des documents préparés , cela signifie que le préparateur a signé les documents et qu'ils sont maintenant prêts à être examinés.

Sélectionnez OUVRIR pour commencer à examiner les documents préparés.

Étape 2 : Examiner les documents préparés

Sur la page Documents, chaque document est affiché avec une icône de signature. Les documents prêts à être examinés sont précédés de l' icône de signature "Préparé" ().

Ouvrez un document préparé et commencez à le réviser. Lors de la révision, vous pouvez :

  • Travailler avec les outils de révision pour suivre les modifications apportées aux documents et les contrôles d'équilibre

  • Créer des problèmes pour signaler des préoccupations ou assigner des tâches

  • Compléter les questions signalées dans les différents documents par le préparateur

Étape 3 : Signer en tant que réviseur

Une fois que vous avez examiné un document et que vous avez déterminé que le contenu est correct, vous pouvez signer le document pour indiquer qu'il est complet.

Signer un document :

  1. En haut du document, sélectionnez le menu déroulant Signer. Les rôles d'approbation sont affichés.

  2. Sélectionnez Déconnexion en regard du rôle que vous souhaitez déconnecter.

Lorsque vous signez la révision, le statut du document est mis à jour. Votre nom et la date sont enregistrés dans le document, et lorsque vous allez sur la page Documents, vous verrez que l'icône indique maintenant que le document a été révisé.
