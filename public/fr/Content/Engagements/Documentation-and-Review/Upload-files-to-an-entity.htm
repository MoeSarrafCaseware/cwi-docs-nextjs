Télécharger des fichiers vers une entité

Vous pouvez ajouter des fichiers et des documents au nuage afin que le matériel pertinent pour votre mission soit stocké en un seul endroit accessible. Les documents pertinents peuvent être des dossiers comprenant des copies de la correspondance avec le client ou des documents relatifs à la mission.

Si vous gérez votre travail fiscal dans TaxPrep, vous pouvez également télécharger vos fichiers TaxPrep dans le nuage afin de suivre et d'assigner le travail sur ces fichiers comme vous le feriez avec n'importe quel autre type de fichier.

La procédure d'ajout d'un fichier TaxPrep est la même que pour tout autre fichier. Une fois le fichier téléchargé, Cloud crée automatiquement un autre engagement et y stocke le fichier TaxPrep.

Avant de commencer, assurez-vous que vous avez le rôle d' éditeur ou des privilèges équivalents. Pour en savoir plus, voir Rôles intégrés.

Pour télécharger un fichier :

  1. Fichiers

  2. Sélectionnez l'entité appropriée dans la liste déroulante des entités.

  3. Sélectionnez Nouveau, puis choisissez Télécharger. La boîte de dialogue File Upload s'affiche.

  4. Sélectionnez le bouton Sélectionner les fichiers ou faites glisser le fichier.

  5. Dans la boîte de dialogue Paramètres de partage, choisissez les options de partage pour le fichier et sélectionnez OK.
