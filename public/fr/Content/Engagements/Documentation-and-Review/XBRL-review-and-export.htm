Révision et exportation XBRL

Si XBRL est disponible dans votre produit, vous pouvez examiner les faits XBRL dans les états financiers, ainsi qu'exporter des documents d'instance XBRL et iXBRL.

Examiner les faits XBRL

Vous pouvez activer le mode de révision XBRL pour réviser les faits XBRL.

Examiner les faits XBRL :

  1. Sélectionnez l'icône XBRL ().

  2. Dans la section XBRL REVIEW , sélectionnez la taxonomie que vous souhaitez réviser.

  3. Sélectionnez START REVIEW.

    Le document des états financiers s'ouvre en montrant la première zone qui contient des faits XBRL. Les faits sont soulignés en vert.

    Remarque : Le premier fait XBRL de la taxonomie sélectionnée est mis en évidence par défaut en mode révision et ses détails s'affichent dans le volet XBRL REVIEW en bas de page. Toutefois, si le premier fait n'est pas situé sur la page en cours, il se peut que vous ne voyiez pas de champ en surbrillance sur la page (mais ses détails s'afficheront toujours dans le volet XBRL REVIEW ).

  4. Sélectionnez un fait pour l'examiner.

    Le fait est mis en évidence et ses détails s'affichent dans le volet XBRL REVIEW .

Notez que, lorsque vous êtes en mode révision, la seule action que vous pouvez effectuer dans le document est de créer des questions.

Pour quitter le mode révision, sélectionnez XBRL et STOP REVIEW ou sélectionnez le X dans le panneau de révision

Vous pouvez interrompre le processus de révision à tout moment et le reprendre plus tard. Cependant, lorsque vous relancez l'examen, vous n'êtes pas automatiquement dirigé vers le dernier fait que vous avez examiné.

Meilleure pratique : Nous vous recommandons de créer une question " À faire" et d'enregistrer le dernier fait révisé avant d'arrêter une révision.

Exporter des documents d'instance XBRL

Après avoir examiné les faits XBRL, vous pouvez exporter les documents d'instance XBRL.

Pour exporter un document d'instance XBRL :

  1. Sélectionnez l'icône XBRL ().

  2. Dans la section XBRL EXPORT , sélectionnez la taxonomie que vous souhaitez exporter.

  3. Sélectionnez Exporter.

    Vous pouvez également sélectionner le bouton XBRL Export directement à partir du volet XBRL REVIEW . La taxonomie que vous êtes en train de réviser est exportée.

    Si vous exportez un document iXBRL ou XBRL, il est automatiquement sauvegardé dans la page Documents ou dans l'entité client dans Cloud, en fonction de votre produit.

    Pour les documents XBRL, le fichier .xbrl peut également être automatiquement téléchargé localement, si cette option est disponible dans votre produit.

    Si le document contient des erreurs de validation, le fichier est téléchargé avec succès mais un message d'erreur s'affiche avec les détails de l'erreur.
