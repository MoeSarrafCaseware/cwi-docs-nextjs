Dépannage de Caseware Cloud Connector

Cette section présente les solutions possibles aux problèmes courants que les utilisateurs peuvent rencontrer lors de l'utilisation de Caseware Cloud Connector.

Le volet Cloud Connector est vide

Il y a plusieurs raisons pour lesquelles le contenu du volet Cloud Connector ne s'affiche pas :

  • Temps de chargement des pages

    Il se peut que vous deviez parfois attendre plus longtemps que d'habitude pour que la page se charge.

  • Lorsque de nouvelles mises à jour sont publiées pour Cloud Connector, le complément n'est pas toujours mis à jour immédiatement. Complete the following steps to resolve the issue:

    1. Dans la barre d'outils Accueil de votre document Excel, sélectionnez Compléments | Autres compléments | Mes compléments.

    2. Dans la boîte de dialogue, sélectionnez Modifier.

    3. Fermez et rouvrez Excel.

  • Un bogue connu d'Excel qui empêche le chargement du volet

    Si le volet ne se charge toujours pas, sélectionnez le menu déroulant Connecteur cloud et choisissez Redimensionner.

  • Une erreur d'exécution qui empêche le chargement du volet

    Si les conseils de dépannage précédents ne permettent pas de résoudre le problème, suivez les étapes suivantes avant de contacter notre équipe d'assistance.

    1. Cliquez avec le bouton droit de la souris sur le volet et sélectionnez Inspecter. La fenêtre des outils de développement s'ouvre.
    2. Sélectionnez l'onglet Console en haut de la fenêtre.
    3. Faites une capture d'écran du contenu de l'onglet Console et envoyez-la à notre équipe d'assistance pour l'aider à identifier et à résoudre le problème.

Message d'entreprise non valide

Si vous recevez un message d'entreprise invalide, il se peut que vous ayez installé la mauvaise version de l'add-in Caseware Cloud Connector.

Pour confirmer quelle version du complément est installée :

  1. Sélectionnez le menu déroulant en haut du volet Cloud Connector et sélectionnez Security Info.

  2. Si l'URL contient "cwcloudpartner", la version installée est incorrecte.

Pour désinstaller le complément :

  1. Sur le ruban Excel, sélectionnez Mes compléments dans l'onglet Insertion .

  2. Sélectionnez Caseware Cloud Connector, puis Supprimer.

Pour réinstaller le complément, voir Démarrer avec Caseware Cloud Connector.

Plusieurs versions de Cloud Connector installées

Si vous avez plusieurs versions de Caseware Cloud Connector installées, Cloud Connector ne fonctionnera pas comme prévu.

Pour confirmer le nombre de versions de Cloud Connector que vous avez installées, sélectionnez Mes compléments dans l'onglet Insertion .

Si plusieurs compléments Cloud Connector sont installés, désinstallez-les.

Pour désinstaller le complément :

  1. Sélectionnez Mes compléments dans l'onglet Insertion .

  2. Sélectionnez Caseware Cloud Connector, puis Supprimer.

Après avoir désinstallé les compléments, copiez vos feuilles dans un nouveau classeur Excel et téléchargez-le à nouveau dans votre dossier d'engagement pour remplacer l'ancien fichier. Vous pouvez ensuite réinstaller la version correcte de Cloud Connector. Pour plus d'informations, voir Démarrer avec Caseware Cloud Connector.

Dépannage général

Pour les problèmes généraux qui ne sont pas traités dans les sections précédentes, essayez de vider votre cache et de réinstaller le complément.

Pour désinstaller le complément :

  1. Sur le ruban Excel, sélectionnez Mes compléments dans l'onglet Insertion .

  2. Sélectionnez Caseware Cloud Connector, puis Supprimer.

  3. Effacez la mémoire cache d'Excel. Pour plus d'informations, voir Vider le cache de l'Office.

Vous pouvez ensuite réinstaller la version correcte de Cloud Connector. Pour plus d'informations, voir Démarrer avec Caseware Cloud Connector.
