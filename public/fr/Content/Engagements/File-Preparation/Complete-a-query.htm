Terminer une requête

Une fois que vous avez accepté toutes les réponses, vous pouvez marquer la requête comme étant complète.

Les informations fournies dans le document d'interrogation sont désormais définitives et ne peuvent plus être modifiées. Les requêtes terminées ne sont plus visibles pour les contacts dans le nuage.

Remarque : Vous ne pouvez pas modifier le contenu ou l'état d'une requête complète. Vous ne pouvez modifier que le nom de la requête et ses paramètres de visibilité. Dans la page Documents, sélectionnez Autres actions () | Modifier pour changer le nom de la requête et modifier les paramètres de visibilité.
