Créer une mission Caseware Cloud en utilisant Caseware Working Papers

Les fichiers d'engagement de bureau nécessitent que le logiciel soit installé sur votre ordinateur. Vous pouvez créer le fichier d'engagement localement et l'intégrer ensuite dans le nuage pour améliorer la collaboration et le partage.

Créer le fichier de mission

Les instructions pour créer un nouveau fichier d'engagement Working Papers sont disponibles ici.

Publier le fichier dans le nuage

Lorsque vous êtes prêt à travailler sur le nouveau fichier, vous pouvez le publier dans le nuage.

Remarque : Assurez-vous que vous avez intégré votre logiciel Working Papers avec Cloud. Pour plus d'informations, voir Intégrer les Working Papers avec Caseware Cloud.

Vous avez publié avec succès votre fichier Working Papers dans le Cloud. Cette opération crée une copie parentale du fichier sur le cloud, et vous pouvez maintenant la partager avec d'autres membres du personnel. La copie parentale remplace l'espace réservé créé lors de la première clôture de l'exercice sur le fichier de l'année précédente.

Une fois publié, votre équipe peut créer des copies du fichier pour travailler sur le nouvel engagement. Pour plus d'informations sur le partage du fichier, voir Partager un fichier Working Papers SmartSync sur le Cloud.

Affecter du personnel à un dossier de documents de travail

Dans la plupart des cas, les agents sont affectés aux dossiers des documents de travail en fonction de leur équipe de mission (pour plus d'informations, voir Créer une équipe de mission), mais dans certains cas, vous voudrez peut-être affecter le personnel individuellement.

Pour affecter du personnel à un dossier de documents de travail :

  1. Dans le menu Cloud, sélectionnez Working Papers.

  2. Sélectionnez un fichier Working Papers, puis la barre de recherche Assigned .

  3. Saisissez le nom d'un membre du personnel, puis sélectionnez-le dans le menu déroulant. Vous pouvez répéter cette étape pour affecter plusieurs membres du personnel.

  4. Cochez la case () pour terminer le travail.

Voir les affectations des documents de travail

L'application Documents de travail vous permet de consulter les dossiers de documents de travail qui vous ont été attribués ou qui ont été attribués à d'autres membres de votre organisation.

Pour consulter les affectations de fichiers des documents de travail :

  1. Dans le menu Cloud, sélectionnez Working Papers.

  2. Sélectionnez Filtre | Affectations.

  3. Dans le menu déroulant, sélectionnez Assigné.

  4. Saisissez le nom approprié dans la zone de texte Rechercher des personnes....

Les dossiers des documents de travail attribués au membre du personnel sélectionné sont affichés.

Après avoir été affecté à un dossier de Working Papers, vous pouvez vous voir attribuer des tâches pour ce dossier. Vous pouvez consulter ces tâches sur la page Activités. Pour plus d'informations, voir Visualisation du travail qui vous a été attribué.

Le personnel peut consulter le travail qui lui a été assigné sur la page Activités. Pour plus d'informations, voir Visualisation du travail qui vous a été attribué.
