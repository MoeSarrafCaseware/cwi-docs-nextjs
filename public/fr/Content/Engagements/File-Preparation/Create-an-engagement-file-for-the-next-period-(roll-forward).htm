Créer un fichier de mission pour la période suivante (report)

Lorsque la période de la mission touche à sa fin, vous devez créer un dossier de mission pour la période suivante.

Dans les Working Papers, le processus d'achèvement d'une mission est connu sous le nom de clôture de l'exercice. Au cours de ce processus, vous pouvez également créer un nouveau dossier pour la période suivante ; c'est ce que l'on appelle un report. Dans un transfert, les soldes de clôture du fichier actuel sont utilisés pour définir les soldes d'ouverture du nouveau fichier.

Effectuer la clôture de l'exercice

Vous pouvez effectuer une clôture de fin d'année sur un dossier de Working Papers lorsque tous les travaux de la mission sont terminés. Ce faisant, vous créez un nouveau fichier Working Papers et vous reportez les soldes et les données choisies pour établir le fichier de l'année suivante.

Effectuer la clôture de l'exercice :

  1. Assurez-vous que vous avez le rôle de propriétaire ou des autorisations équivalentes pour le fichier Working Papers.

  2. Dans le menu Cloud, sélectionnez Working Papers.

  3. Sélectionnez le fichier Working Papers correspondant à votre engagement. Dans le volet de détails, sélectionnez Ouvrir la copie de synchronisation.

  4. Dans le fichier Working Papers, sur le ruban, cliquez sur l'onglet Engagement . Dans la section Gérer, cliquez sur Clôture de l'exercice.

  5. Dans la boîte de dialogue Clôture de l'exercice et report, saisissez un nom de fichier pour le fichier d'engagement de l'année suivante. Choisissez vos options pour le report et sélectionnez les informations à inclure dans le dossier de l'année suivante. Lorsque vous avez fait vos choix, cliquez sur OK.

Working Papers effectuera la clôture de l'exercice et, une fois le processus terminé, ouvrira automatiquement le nouveau fichier pour l'année suivante.

Lorsque vous effectuez une clôture de fin d'année sur un fichier publié, vous créez également un espace réservé pour le fichier sous l'entité du client. Cloud affiche l'espace réservé, mais le fichier ne peut pas être ouvert tant que vous n'avez pas publié votre copie locale.

Publier le fichier du nouvel an dans le nuage

Lorsque vous êtes prêt à travailler sur le nouveau fichier, vous pouvez le publier dans le nuage.

Vous avez publié avec succès votre fichier Working Papers dans le Cloud. Cette opération crée une copie parentale du fichier sur le cloud, et vous pouvez maintenant la partager avec d'autres membres du personnel. La copie parentale remplace l'espace réservé créé lors de la première clôture de l'exercice sur le fichier de l'année précédente.
