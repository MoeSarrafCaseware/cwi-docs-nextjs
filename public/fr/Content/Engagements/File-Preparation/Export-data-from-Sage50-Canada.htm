Exporter des données à partir de Sage 50 Canada vers Caseware Cloud

Les données que vous devez exporter de Sage 50 Canada varient selon le type de jeu de données Caseware dans lequel vous voulez importer des données. Consultez le tableau ci-dessous pour passer aux instructions permettant d'exporter les données dont vous avez besoin. Les données de Caseware Cloud sont organisées en plusieurs ensembles de données standard en fonction de l'objectif des données, telles que la balance de vérification, le grand livre et les données de comptabilité auxiliaire, entre autres.

Pour plus d'informations sur l'importation de données dans Caseware Cloud, ainsi que sur les descriptions des types de données, voir Importer et gérer des données.

Rapport d'exportation des transactions par compte

Pour exporter le rapport Transactions par compte de Sage 50 Canada :

  1. Dans le menu Rapports , sélectionnez Centre de rapports.
  2. Dans la boîte de dialogue Centre de rapports , sous Sélectionner un type de rapport, sélectionnez Finances | Transactions par compte.
  3. Cliquez sur Créer. Un aperçu du rapport qui sera exporté s'affiche.

Remarque : Le Centre de rapport exporte par défaut les données de l'exercice en cours. Si vos dates de mission se rapportent à l'exercice précédent, vous pouvez cliquer sur l'option Modifier ce rapport et, dans la liste déroulante Rapport sur l'exercice , sélectionner l' année précédente (AAAA).

  1. Dans le menu Fichier , sélectionnez Exporter...
  2. Dans Nom du fichier, spécifiez un nom pour le rapport que vous reconnaîtrez lorsque vous l'importerez dans Caseware Cloud et cliquez sur Enregistrer.

Exporter l'enregistrement du grand livre des clients

Pour exporter l'enregistrement du grand livre des clients de Sage 50 Canada :

  1. Dans le menu Fichier , sélectionnez Import/Export | Exporter les enregistrements..
  2. Dans la boîte de dialogue Assistant d'exportation d'enregistrements , sous Sélectionner un type de rapport, sélectionnez Exporter les enregistrements vers un format de fichier Sage 50 Import. Cliquez sur Suivant.
  3. Dans Sélectionner le type d'enregistrements à exporter, sélectionnez Clients.
  4. Spécifiez un nom et un emplacement pour le fichier d'exportation et cliquez sur Suivant.

  1. Dans la zone Sélectionner les clients à exporter , sélectionnez les clients dont vous souhaitez exporter les enregistrements de grand livre ou cliquez sur Sélectionner tout pour exporter tous les enregistrements de grand livre des clients.
  2. Cliquez sur Suivant.
  3. Affichez le résumé des enregistrements d'exportation et cliquez sur OK.
  4. Cliquez sur Terminer.

Exporter le rapport détaillé sur l'ancienneté des clients

Pour exporter le rapport Détail cumulé client de Sage 50 Canada :

  1. Dans le menu Rapports , sélectionnez Centre de rapports.
  2. Dans la boîte de dialogue Centre de rapports , sous Sélectionner un type de rapport, sélectionnez Clients et ventes | Détail cumulé client.
  3. Cliquez sur Créer. Un aperçu du rapport qui sera exporté s'affiche.

Remarque : Le Centre de rapport exporte par défaut toutes les factures payées au cours des 999 derniers jours à compter de la date de clôture de l'exercice fiscal en cours. Vous pouvez modifier la date de clôture en cliquant sur l'option Modifier ce rapport. Assurez-vous que la date sélectionnée est postérieure ou égale à la date de fin de votre engagement Caseware.

 

 

  1. Dans le menu Fichier , sélectionnez Exporter..
  2. Dans Nom du fichier, spécifiez un nom pour le rapport que vous reconnaîtrez lorsque vous l'importerez dans Caseware Cloud et cliquez sur Enregistrer.
