CloudBridge (.cwp)

Cette version de CloudBridge est :

  • Destiné aux applications canadiennes, telles que Caseware Audit.

  • Utilisé pour transférer les données d'un engagement individuel de Working Papers vers le Cloud.

  • Ajouté directement à l'engagement des documents de travail à partir de la bibliothèque de documents.

  • Convient pour une migration progressive vers l'informatique en nuage.
