Importer et gérer des données

Vous pouvez importer dans votre engagement des données de balance de vérification, de grand livre et de comptabilité auxiliaire provenant de diverses sources.

Les fichiers d'importation de données, tels que les fichiers Excel ou CSV, sont appelés sources de données, que vous pouvez ensuite configurer et affecter à un ensemble de données sur la page Ensembles de données . Plusieurs sources de données peuvent être affectées à un seul jeu de données. En outre, vous pouvez également importer des données non structurées, c'est-à-dire des données qui ne peuvent pas être facilement attribuées à l'un des types d'ensembles de données existants, en sélectionnant le type d'ensemble de données "Autre". La source d'importation et les options d'ensemble de données disponibles varient en fonction de votre produit.

Toute mission peut comporter plusieurs types d'ensembles de données, qui sont des collections de données normalement présentes dans une mission. Chaque ensemble de données stocke des données relatives à un aspect différent d'un engagement et comporte des champs prédéterminés. Voir les descriptions des ensembles de données et de leurs champs dans Types d'ensembles de données décrits.
