Importer des données à partir d'AuditXML

Pour importer des données clients à partir du logiciel comptable AuditXML 3.x (Beta), vous devez utiliser le transformateur Caseware. Pour consulter la liste complète des outils de comptabilité de bureau pris en charge, voir Logiciel de comptabilité de bureau tiers pris en charge.

Remarque : Les comptes dont le solde est nul ne sont pas automatiquement importés. Si vous souhaitez inclure des comptes à solde nul, vous devez les ajouter manuellement et les affecter aux groupes désignés.
