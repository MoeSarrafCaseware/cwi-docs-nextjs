Import engagement data from Working Papers

You can import your client's financial data from Working Papers and have it transferred to your engagement. The steps to carry out this task differ based on whether or not you have disabled your Working Papers license.

Note: Accounts with zero balances are not automatically imported. If you want to include zero-balance accounts, you must manually add them and assign them to the designated groups.

To import engagement data from a licensed copy of Working Papers:

  1. Install OnPoint CloudBridge.

  2. Ensure that Working Papers is integrated with your Cloud site.

  3. Carry forward your Working Papers engagement file.

  4. Add the CloudBridge (CLBG) document to your Working Papers engagement file.

  5. Transfer the data from Working Papers to your Cloud engagement.

To import engagement data from a disabled copy of Working Papers:

  1. Install OnPoint CloudBridge.

  2. Open Working Papers and select Continue Without Registering.

  3. Ensure that Working Papers is integrated with your Cloud site.

  4. Carry forward your Working Papers engagement file.

  5. Copy the below path and paste it into the URL field of the Browser toolbar.

    cw:program/scripts/cloudbridge/index.html

    Press Enter on your keyboard to open the CloudBridge (CLBG) document.

  6. Transfer the data from Working Papers to your Cloud engagement.
