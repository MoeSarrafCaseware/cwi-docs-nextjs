Importer des données à partir d'un logiciel de comptabilité de bureau

Pour importer des données clients à partir d'un logiciel de comptabilité tiers, utilisez l'utilitaire d'importation Cloud pour générer un fichier d'importation que vous pourrez ensuite importer à partir de l'onglet Datasets de la page Data de votre engagement Caseware Cloud. L'utilitaire d'importation Caseware Cloud est compatible avec la plupart des logiciels de comptabilité.

Pour une liste complète des logiciels de comptabilité de tiers, voir Logiciel de comptabilité de bureau tiers pris en charge. Notez que les options disponibles lors de l'importation varient en fonction du produit.

Conditions préalables

Avant d'importer les données de vos clients, assurez-vous d'avoir effectué les étapes préalables pour votre logiciel de comptabilité. Pour plus d'informations, voir Conditions préalables à l'importation à partir d'un logiciel de comptabilité de bureau.

Pour savoir comment importer des données à partir de QuickBooks desktop ou de Sage desktop, voir ce qui suit :

  • Importer à partir de QuickBooks Desktop

  • Importer à partir de Sage Desktop

Pour tout autre logiciel de comptabilité tiers, utilisez les instructions de cette page.

Remarque : Les comptes avec un solde nul ne sont pas automatiquement importés dans les applications Caseware Cloud, à l'exception d' IDEA. Pour inclure les comptes à solde nul, vous devez les ajouter et les affecter manuellement aux groupes désignés.

Générer le fichier d'importation

Au cours de cette étape, vous téléchargez et installez l'utilitaire d'importation Caseware Cloud. Utilisez-le pour importer les données de vos clients et générer un fichier d'importation.

Pour générer le fichier d'importation :

  1. Téléchargez l'utilitaire d'importation Caseware Cloud.

    2. Caseware Cloud Import Utility (.exe)

  2. Ouvrez l'utilitaire, sélectionnez Autre source de données et cliquez sur Suivant.

  3. Dans les champs Importer de et Version , indiquez le logiciel de comptabilité à partir duquel vous effectuez l'exportation.

    Le contenu de la page change en fonction de votre sélection.

  4. Naviguez jusqu'au chemin d'importation du fichier client que vous souhaitez importer.

    Si votre logiciel de comptabilité vous demande de spécifier un fichier client, vous verrez apparaître la liste déroulante Sélectionner le fichier d'identification du client . S'il y a plus d'un fichier, assurez-vous de sélectionner le bon fichier ici.

  5. Sélectionnez Next. La page de fin d'exécution de l'assistant de nouveau fichier s'affiche.

    Il indique l'emplacement d'un fichier temporaire qui sera utilisé pour générer le fichier .zip final contenant les données importées.

  6. Sélectionnez Terminer pour effectuer la conversion.

    Une fois les données générées, la boîte de dialogue Enregistrer sous s'affiche.

  7. Naviguez jusqu'à l'emplacement souhaité pour enregistrer le fichier .zip et sélectionnez Enregistrer.

    Remarque : Si vous importez des données dans IDEA à l'aide de l'utilitaire d'importation dans le nuage, ne procédez pas à l'importation des données. L'étape suivante consiste à importer le fichier zip dans IDEA. Voir l'aide IDEA pour des instructions détaillées.

Importer les données

Après avoir généré le fichier zip à l'aide de l'utilitaire d'importation Caseware Cloud, vous pouvez importer les données dans votre engagement.

Pour importer les données dans votre engagement :

  1. Ouvrez votre fichier d'engagement Caseware Cloud. Lorsque vous ouvrez l'engagement pour la première fois, la fenêtre contextuelle Propriétés de l'engagement s'affiche. Définissez la date de fin d'année comme étant la même que celle des données que vous êtes sur le point d'importer.

    La date de fin d'année est le dernier élément du nom du fichier zip que vous avez généré. Par exemple, pour le fichier nommé CONSTRUCTION UNIVERSELLE 2025-06-30.zip, la date de fin d'année est le 30 juin 2025.

  2. Accédez à la page Données . The Data page opens to the Datasets tab.

  • Si vous avez déjà importé des données, sélectionnez Ajouter des données.

  • Si votre engagement est consolidé et comporte des filiales internes, sélectionnez dans la liste l'entité dont vous souhaitez importer les données.

  1. Sélectionnez votre logiciel de comptabilité de bureau dans la liste des sources de données.

    Si l'option n'est pas visible, vous pouvez utiliser la boîte de recherche pour la trouver.

  2. A l'étape Upload de l'assistant d'importation, sélectionnez ou glissez-déposez le fichier zip que vous avez généré en utilisant l'utilitaire d'importation Caseware Cloud, puis sélectionnez Next.

  3. A l'étape Configuration des données de l'assistant d'importation :

    • La période dedéclaration est automatiquement remplie avec la période de déclaration des données.

    • La date dedébut est automatiquement remplie avec la date de début des données.

    • La date defin est automatiquement remplie avec la date de fin des données.

  4. Dans Sélectionner le type de jeude données, choisissez le type de jeu de données que vous importez.

    Pour en savoir plus sur les options des jeux de données, voir Types de jeux de données. Tous les types de jeux de données ne sont pas disponibles s'ils n'ont pas été activés dans votre produit.

  5. Si vous avez sélectionné Balance de vérification, réduisez la liste des types d'ensembles de données en sélectionnant l'icône de réduction (). Remarquez que l'option Importer les 5 dernières années de TB apparaît. Sélectionnez-le pour importer jusqu'à cinq années de données de balance de vérification si elles sont disponibles dans le fichier zip que vous avez généré.

  6. Sélectionnez Next.

  7. À l'étape de confirmation de l'assistant d'importation, vérifiez les informations et sélectionnez Importer.

Pour afficher les données importées, cliquez sur Afficher les données en regard de l'ensemble de données que vous avez importé.

Une nouvelle page s'affiche avec les données.

Par défaut, toutes les colonnes de votre jeu de données s'affichent. Cliquez sur Colonnes pour désélectionner celles que vous souhaitez masquer.

Cliquez sur Filtres pour filtrer les données affichées dans l'ensemble de données.

Pour plus d'informations, reportez-vous à la section Consulter les données générales et auxiliaires du client.
