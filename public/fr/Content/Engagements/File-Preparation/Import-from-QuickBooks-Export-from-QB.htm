Importer à partir de QuickBooks Desktop (exporter à partir de QB)

Vous pouvez importer des données de QuickBooks desktop (Export from EB) en utilisant le module d'exportation, puis en important le fichier dans votre engagement.

Remarque : Les comptes avec un solde nul ne sont pas automatiquement importés dans les applications Caseware Cloud. Si vous souhaitez inclure des comptes à solde nul, vous devez les ajouter et les affecter manuellement aux groupes désignés.
Les comptes à solde nul sont importés dans les fichiers IDEA .

Avant de commencer, veillez à créer un fichier d'exportation à partir de QuickBooks à l'aide du module d'exportation.

Pour importer les données exportées dans votre engagement :

  1. Téléchargez l'utilitaire d'importation Caseware Cloud.

    2. Caseware Cloud Import Utility (.exe)

  2. Ouvrez l'utilitaire, sélectionnez Autre source de données et cliquez sur Suivant.

  3. Dans les menus déroulants Importer de et Version , sélectionnez QuickBooks (Exporter de QB).

    Remarque : Pour connaître la liste des versions prises en charge, reportez-vous à la rubrique Logiciels comptables tiers pris en charge . Si vous trouvez des versions obsolètes dans la liste, signalez le problème à Caseware pour enquête.

  4. Naviguez jusqu'au chemin d'importation du fichier client que vous souhaitez importer.

  5. Dans Fichiers texte exportés de QuickBooks, parcourez et sélectionnez le fichier du plan comptable.

  6. Sélectionnez Suivant, puis Terminer.

  7. Enregistrez le fichier zip généré à l'aide de l'invite affichée.

  8. In your engagement file, go to the Data page. The Data page opens to the Datasets tab.

  • Si vous avez déjà importé des données, sélectionnez Ajouter des données.

  • Si votre engagement est consolidé et comporte des filiales internes, sélectionnez dans la liste l'entité dont vous souhaitez importer les données.

  1. Sous IMPORTATION DEPUIS UN CONNECTEUR DE DONNÉES, sélectionnez QuickBooks Desktop dans la liste des sources de données, puis cliquez sur Suivant.

    Si l'option n'est pas visible, vous pouvez utiliser la boîte de recherche pour la trouver.

  2. Sélectionnez ou déposez par glisser-déposer le fichier zip que vous avez enregistré à l'étape 7.

Pour afficher les données importées, cliquez sur Afficher les données en regard de l'ensemble de données que vous avez importé.

Une nouvelle page s'affiche avec les données.

Par défaut, toutes les colonnes de votre jeu de données s'affichent. Cliquez sur Colonnes pour désélectionner celles que vous souhaitez masquer.

Cliquez sur Filtres pour filtrer les données affichées dans l'ensemble de données.

Pour plus d'informations, reportez-vous à la section Consulter les données générales et auxiliaires du client.
