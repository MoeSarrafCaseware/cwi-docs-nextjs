Importer à partir de QuickBooks Desktop

Vous pouvez importer des données de QuickBooks desktop en les exportant sous forme de fichier, en générant un fichier zip à l'aide de l'utilitaire d'importation Caseware Cloud, puis en les important dans votre engagement.

Remarque : Les comptes avec un solde nul ne sont pas automatiquement importés dans les applications Caseware Cloud, à l'exception d' IDEA. Pour inclure les comptes à solde nul, vous devez les ajouter et les affecter manuellement aux groupes désignés.

Exporter des données à partir de Quickbooks

Avant de commencer, assurez-vous d'avoir effectué les étapes préalables nécessaires en fonction de votre version de QuickBooks :

QuickBooks Canada / UK / Australie :

  • Installez une version applicable de QuickBooks sur votre poste de travail.

  • Installez l'utilitaire d'exportation QuickBooks sur votre poste de travail. Vous pouvez le télécharger à partir de MyCaseware sur la page Support Downloads.

  • Dans QuickBooks, vous devez autoriser l'utilitaire d'exportation QuickBooks à accéder au fichier. Sur le ruban, cliquez sur Édition | Préférences. .. | Pour ce faire, cliquez sur Applications intégrées | Préférences de l'entreprise et désélectionnez l'option Ne permettre à aucune application d'accéder à ce fichier d'entreprise.

QuickBooks USA :

  • Installez une version applicable de QuickBooks sur votre poste de travail.

Après avoir rempli les conditions préalables, vous pouvez exporter des données à partir de QuickBooks Canada / UK / Australie ou QuickBooks US.

Pour exporter des données de QuickBooks Canada / UK / Australie :

  1. Connectez-vous à votre compte QuickBooks, puis ouvrez le fichier de l'entreprise que vous souhaitez importer dans Caseware Cloud.

  2. Ouvrez l'utilitaire d'exportation QuickBooks.

    1. Dans la liste déroulante Version de QuickBooks , sélectionnez votre version de QuickBooks.

    2. dans le champ Chemin d'exportation , sélectionnez Browse....

      3. La boîte de dialogue Browse For Folder (Rechercher un dossier) s'ouvre.

    3. Naviguez jusqu'à l'endroit où vous souhaitez créer le fichier d'exportation, puis sélectionnez OK.

    • Sélectionnez Spécifier la date de la dernière transaction et saisissez la date applicable.

    • Dans la liste déroulante Méthode comptable , sélectionnez la méthode de comptabilité d'exercice applicable.

  3. Sélectionnez Exporter.

Les données comptables du fichier QuickBooks Desktop sont exportées vers un fichier à l'emplacement spécifié.

Pour exporter des données à partir de QuickBooks US :

  1. Connectez-vous à votre compte QuickBooks desktop, puis ouvrez le fichier de l'entreprise que vous souhaitez importer.

  2. Sur le ruban, sélectionnez Fichier | Utilitaires | Copier le fichier de l'entreprise pour QuickBooks Online....

    Si vous êtes invité à fermer les fenêtres ouvertes, sélectionnez OK.

  3. Dans la boîte de dialogue Enregistrer le fichier de la société exportée dans..., naviguez jusqu'à l'emplacement de votre poste de travail où vous souhaitez créer le fichier QuickBooks Online et sélectionnez Enregistrer.

  4. Sélectionnez OK.

Les données comptables du fichier QuickBooks Desktop sont exportées vers un fichier à l'emplacement spécifié.

Générer le fichier d'importation pour Quickbooks

Après avoir exporté vos données depuis QuickBooks, vous pouvez maintenant générer le fichier d'importation en utilisant l'utilitaire d'importation Caseware Cloud.

Pour générer le fichier d'importation pour Quickbooks :

  1. Téléchargez l'utilitaire d'importation Caseware Cloud.

    2. Caseware Cloud Import Utility (.exe)

  2. Ouvrez l'utilitaire, sélectionnez Autre source de données et cliquez sur Suivant.

  3. Dans les menus déroulants Importer de et Version , sélectionnez QuickBooks et la version à partir de laquelle vous effectuez l'importation.

    Remarque : Pour une liste des versions que nous prenons en charge, voir Logiciels comptables tiers pris en charge.

  4. Dans Chemin d'importation, sélectionnez Parcourir et naviguez jusqu'au dossier contenant la collection de fichiers que vous avez exportés de QuickBooks en utilisant l'utilitaire d'exportation QuickBooks de Caseware.

    • Sélectionner le fichier d'identification du client est automatiquement complété par le nom du fichier d'identification du client.

    • L'extension de fichier est automatiquement remplie avec l'extension correcte du fichier. (Les options d'extension de fichier varient en fonction de la version que vous avez sélectionnée précédemment)

    • Dans la liste déroulante Année comptable , sélectionnez l'année pour laquelle vous souhaitez importer des données.

  5. Déterminez les autres paramètres de l'importation.

    Les paramètres varient en fonction de la version que vous avez sélectionnée précédemment.

  6. Sélectionnez Suivant, puis Terminer.

  7. Enregistrez le fichier zip généré à l'aide de l'invite affichée.

Importer les données dans votre engagement

Après avoir généré le fichier zip à l'aide de l'utilitaire d'importation Caseware Cloud, vous pouvez importer les données dans votre engagement.

Pour importer les données dans votre engagement :

  1. Ouvrez votre fichier d'engagement Caseware Cloud. Lorsque vous ouvrez l'engagement pour la première fois, la fenêtre contextuelle Propriétés de l'engagement s'affiche. Définissez la date de fin d'année comme étant la même que celle des données que vous êtes sur le point d'importer.

    La date de fin d'année est le dernier élément du nom du fichier zip que vous avez généré dans Générer le fichier d'importation pour Quickbooks. Par exemple, pour le fichier nommé SUNBRIDGE FARMS LTD. Mastercard Mark 8386 2022-09-30.zip, la date de fin d'année est le 30 septembre 2022.

  2. Accédez à la page Données . Les paramètres du produit s'ouvrent sur l'onglet Zones .

  • Si vous avez déjà importé des données, sélectionnez Ajouter des données.

  • Si votre engagement est consolidé et comporte des filiales internes, sélectionnez dans la liste l'entité dont vous souhaitez importer les données.

  1. Sous IMPORTATION DEPUIS UN CONNECTEUR DE DONNÉES, sélectionnez QuickBooks Desktop dans la liste des sources de données, puis cliquez sur Suivant.

    Si l'option n'est pas visible, vous pouvez utiliser la boîte de recherche pour la trouver.

  1. A l'étape Upload de l'assistant d'importation, sélectionnez ou glissez-déposez le fichier zip que vous avez généré en utilisant l'utilitaire d'importation Caseware Cloud, puis sélectionnez Next.

  2. A l'étape Configuration des données de l'assistant d'importation :

    • La période dedéclaration est automatiquement remplie avec la période de déclaration des données.

    • La date dedébut est automatiquement remplie avec la date de début des données.

    • La date defin est automatiquement remplie avec la date de fin des données.

  3. Dans Sélectionner le type de jeude données, choisissez le type de jeu de données que vous importez. Sélectionnez Balance de vérification si vous importez une balance de vérification et Grand livre si vous importez des transactions.

  4. Si vous avez sélectionné Balance de vérification, vous pouvez sélectionner l'option Importer les 5 dernières années de TB pour importer jusqu'à cinq années de données de balance de vérification si elles sont disponibles dans le fichier zip que vous avez généré.

Pour afficher les données importées, cliquez sur Afficher les données en regard de l'ensemble de données que vous avez importé.

Une nouvelle page s'affiche avec les données.

Par défaut, toutes les colonnes de votre jeu de données s'affichent. Cliquez sur Colonnes pour désélectionner celles que vous souhaitez masquer.

Cliquez sur Filtres pour filtrer les données affichées dans l'ensemble de données.

Pour plus d'informations, reportez-vous à la section Consulter les données générales et auxiliaires du client.
