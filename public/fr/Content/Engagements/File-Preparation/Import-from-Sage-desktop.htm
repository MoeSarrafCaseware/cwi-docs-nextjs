Importer à partir de Sage Desktop

Vous pouvez importer des données de Sage en générant un fichier zip à l'aide de l'utilitaire d'importation Caseware Cloud, puis en l'important dans votre engagement. Nous prenons actuellement en charge les logiciels de comptabilité Sage suivants :

  • Importation à partir de Sage 50 Accounts (Sage-Line50)

  • Importation de Sage 50 Comptabilité Édition CDN (Simply)

Pour une liste des versions que nous prenons en charge, voir Logiciels comptables tiers pris en charge.

Remarque : Les comptes avec un solde nul ne sont pas automatiquement importés dans les applications Caseware Cloud, à l'exception d' IDEA. Pour inclure les comptes à solde nul, vous devez les ajouter et les affecter manuellement aux groupes désignés.

Générer le fichier d'importation pour Sage 50 Accounts (Sage-Line50)

Avant de commencer, assurez-vous que vous disposez des fichiers d'importation nécessaires :

  • Informations sur le profil : ACCDATA\COMPTE.DTA

  • Plan comptable : ACCDATA\NOMINAL.DTA

  • Indice CoA : ACCDATA\NOMINDEX.DTA

  • Détail du GL : ACCDATA\SPLITS.DTA

  • Informations sur l'entreprise : ACCDATA\SETUP.DTA

  • Types de comptes : ACCDATA\COA

  • Informations bancaires : ACCDATA\BANK.DTA

  • Index du grand livre : ACCDATA\NHEADER.DTA

  • Informations sur le département : DEPARTM.DTA

Pour générer le fichier d'importation pour Sage 50 Accounts (Sage-Line50) :

  1. Téléchargez l'utilitaire d'importation Caseware Cloud.

    2. Caseware Cloud Import Utility (.exe)

  2. Ouvrez l'utilitaire, sélectionnez Autre source de données et cliquez sur Suivant.

  3. Dans les menus déroulants Importer de et Version , sélectionnez Sage 50 Comptes et la version à partir de laquelle vous effectuez l'importation.

  4. Dans Chemin d'importation, sélectionnez Parcourir et naviguez jusqu'au dossier qui contient le dossier ACCDATA, où se trouvent les fichiers .DTA que vous souhaitez importer.

  5. Déterminez les autres paramètres de l'importation.

    Les paramètres varient en fonction de la version que vous avez sélectionnée précédemment.

  6. Sélectionnez Suivant, puis Terminer.

  7. Enregistrez le fichier zip généré à l'aide de l'invite affichée.

    Pour poursuivre le processus d'importation, voir Importer les données dans votre engagement.

Générer le fichier d'importation pour Sage 50 Comptabilité Edition CDN (Simply)

Avant de commencer, assurez-vous que vous avez installé Sage 50 Comptabilité (édition CDN) ou l'utilitaire Sage 50 Connection Manager de MyCaseware. Vous devez disposer de l'une de ces applications pour effectuer l'importation ; toutefois, n'installez pas les deux.

Pour générer le fichier d'importation pour Sage 50 Accounting CDN Edition (Simply) :

  1. Téléchargez l'utilitaire d'importation Caseware Cloud.

    2. Caseware Cloud Import Utility (.exe)

  2. Ouvrez l'utilitaire, sélectionnez Autre source de données et cliquez sur Suivant.

  3. Dans les menus déroulants Importer de et Version , sélectionnez Sage 50 Comptabilité Édition CDN (Simply) et la version à partir de laquelle vous effectuez l'importation.

  4. Dans Chemin d'importation, sélectionnez Parcourir et naviguez jusqu'au dossier contenant le fichier client à importer. Les fichiers clients pour Sage portent l'extension .SAI.

    Le champ ID client se remplit automatiquement avec le nom du fichier client.

  5. Déterminez les autres paramètres de l'importation.

    Les paramètres varient en fonction de la version que vous avez sélectionnée précédemment.

  6. Sélectionnez Suivant, puis Terminer.

  7. Enregistrez le fichier zip généré à l'aide de l'invite affichée.

Importer les données dans votre engagement

Après avoir généré le fichier zip à l'aide de l'utilitaire d'importation Caseware Cloud, vous pouvez importer les données dans votre engagement.

Pour importer les données dans votre engagement :

  1. Ouvrez votre fichier d'engagement Caseware Cloud. Lorsque vous ouvrez l'engagement pour la première fois, la fenêtre contextuelle Propriétés de l'engagement s'affiche. Définissez la date de fin d'année comme étant la même que celle des données que vous êtes sur le point d'importer.

    La date de fin d'année est le dernier élément du nom du fichier zip que vous avez généré dans Générer le fichier d'importation pour Sage 50 Comptes (Sage-Line50) ou Générer le fichier d'importation pour Sage 50 Comptabilité Edition CDN (Simply). Par exemple, pour le fichier nommé CONSTRUCTION UNIVERSELLE 2025-06-30.zip, la date de fin d'année est le 30 juin 2025.

  2. Accédez à la page Données . The Data page opens to the Datasets tab.

  • Si vous avez déjà importé des données, sélectionnez Ajouter des données.

  • Si votre engagement est consolidé et comporte des filiales internes, sélectionnez dans la liste l'entité dont vous souhaitez importer les données.

  1. Sélectionnez Desktop Accounting Software dans la liste des sources de données, puis sélectionnez Next.

    Si l'option n'est pas visible, vous pouvez utiliser la boîte de recherche pour la trouver.

  1. A l'étape Upload de l'assistant d'importation, sélectionnez ou glissez-déposez le fichier zip que vous avez généré en utilisant l'utilitaire d'importation Caseware Cloud, puis sélectionnez Next.

  2. A l'étape Configuration des données de l'assistant d'importation :

    • La période dedéclaration est automatiquement remplie avec la période de déclaration des données.

    • La date dedébut est automatiquement remplie avec la date de début des données.

    • La date defin est automatiquement remplie avec la date de fin des données.

  3. Dans Sélectionner le type de jeude données, choisissez le type de jeu de données que vous importez. Sélectionnez Balance de vérification si vous importez une balance de vérification et Grand livre si vous importez des transactions.

  4. Si vous avez sélectionné Balance de vérification, réduisez la liste des types d'ensembles de données en sélectionnant l'icône de réduction (). Remarquez que l'optionImporter les 5 dernières années de TB apparaît. Sélectionnez-le pour importer jusqu'à cinq années de données de balance de vérification si elles sont disponibles dans le fichier zip que vous avez généré.

  5. Sélectionnez Next.

  6. À l'étape de confirmation de l'assistant d'importation, vérifiez les informations et sélectionnez Importer.

Pour afficher les données importées, cliquez sur Afficher les données en regard de l'ensemble de données que vous avez importé.

Une nouvelle page s'affiche avec les données.

Par défaut, toutes les colonnes de votre jeu de données s'affichent. Cliquez sur Colonnes pour désélectionner celles que vous souhaitez masquer.

Cliquez sur Filtres pour filtrer les données affichées dans l'ensemble de données.

Pour plus d'informations, reportez-vous à la section Consulter les données générales et auxiliaires du client.
