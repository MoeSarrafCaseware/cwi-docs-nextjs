Insérer des données dans le panneau de droite avec Caseware Cloud Connector

Vous pouvez utiliser le volet de droite pour récupérer et insérer des données de l'engagement dans le fichier Excel.

Notez que votre point de départ est toujours la page Documents :

  1. Accédez à la page Documents .

  2. Ouvrez le fichier Excel.

  3. Sélectionnez l'icône Datalink dans le ruban.

    4. Un volet s'ouvre sur le côté droit, affichant deux liens vers des sources de données : Balance de vérification et propriétés.

Pour insérer les données de la balance de vérification :

  1. Sélectionnez la cellule dans laquelle vous souhaitez insérer les données du groupe de balances de vérification.

  2. Allez dans le volet de droite et localisez le groupe souhaité. Utilisez les flèches vers le bas () pour développer les groupes de parents si nécessaire.

    Conseil : Vous pouvez également utiliser le champ de recherche situé en haut de la page. Vous pouvez effectuer une recherche par nom de groupe ou par numéro de groupe.

  3. Utilisez les filtres () pour sélectionner les colonnes que vous souhaitez ajouter pour le groupe et ses sous-groupes. Par exemple, si vous souhaitez insérer le numéro de groupe, le nom, le solde de l'année en cours et le solde de l'année précédente, sélectionnez les filtres pour ces colonnes.

    Si vous souhaitez inclure les comptes à solde nul, sélectionnez le filtre Insérer les comptes à solde nul .

    Si votre mission est consolidée, vous pouvez sélectionner une entité dans les données de consolidation pour afficher les données de cette entité. Vous pouvez également utiliser le filtre Consolidé pour choisir d'insérer des données consolidées ou non consolidées.

  4. Select the Add groups and subgroups icon () next to the group.

    Le groupe sélectionné et tous ses sous-groupes sont insérés dans le fichier.

Pour insérer une propriété de fiançailles :

  1. Sélectionnez la cellule dans laquelle vous souhaitez insérer une propriété d'engagement.

  2. Allez dans le volet de droite et sélectionnez l'onglet Propriétés .

  3. Localisez la propriété d'engagement que vous souhaitez insérer.

    4. Conseil : Vous pouvez également utiliser le champ de recherche situé en haut de la page.

  4. Sélectionnez l'icône Insérer un lien de propriété ().

    5. Les informations sont extraites de l'engagement et insérées dans la cellule.

Pour insérer une propriété ferme :

  1. Sélectionnez la cellule dans laquelle vous souhaitez insérer une propriété d'engagement.

  2. Allez dans le volet de droite et sélectionnez l'onglet Propriétés .

  3. Faites défiler vers le bas jusqu'à la section Propriétés de l'entreprise .

  4. Sélectionnez l'icône Insérer un lien de propriété () pour insérer le nom ou l' adresse de l'entreprise.

    5. Les informations sont extraites de l'engagement et insérées dans la cellule.
