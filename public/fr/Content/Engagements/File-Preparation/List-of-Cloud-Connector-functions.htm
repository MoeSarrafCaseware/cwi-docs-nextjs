Liste des fonctions de Cloud Connector

La liste suivante décrit les fonctions de Cloud Connector que vous pouvez utiliser pour insérer des données de votre engagement dans des fichiers Excel.

Fonction Description

SOLDE DE COMPTE CWI.CWI.CWI.CWI.CWI.CWI.CWI.CWI.CWI

Renvoie le solde d'un compte dans la balance générale.

Paramètres :

  • Numéro - Le numéro de compte. Le numéro de compte doit être placé entre guillemets.

  • Year_offset - Numéro de l'année précédente. Pour obtenir le solde de l'année en cours, entrez 0 comme décalage de l'année. Pour l'année précédente, inscrire 1.

  • Balance_type - Le type de solde. Pour renvoyer le solde final, saisissez 0 comme type de solde. Pour le solde de planification, saisissez 1.

CWI.ACCOUNTNAME

Renvoie le nom d'un compte dans la balance générale.

Paramètres :

  • Numéro - Le numéro de compte. Le numéro de compte doit être placé entre guillemets.

CWI.MATÉRIALITÉ

Renvoie les informations relatives à l'importance relative.

Paramètres :

  • Materiality_type - Le type de matérialité. Pour restituer l'importance relative globale, entrez 0. Pour l'importance relative de la performance, saisissez 1.

  • Materiality_workflow - Le flux de travail relatif à la matérialité. Pour revenir au flux de travail de planification, entrez 0. Pour le flux de travail révisé, entrez 1.
