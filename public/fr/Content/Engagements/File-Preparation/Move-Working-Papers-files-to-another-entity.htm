Déplacer les fichiers de Working Papers vers une autre entité

Vous pouvez organiser vos documents de travail en les déplaçant vers les entités du nuage où ils sont pertinents.

Pour déplacer les fichiers Working Papers vers une autre entité :

  1. Sélectionnez Toutes les entités ou l'entité contenant les fichiers à déplacer dans le sélecteur d'entité.

  2. Dans le menu Cloud (), sélectionnez Working Papers.

  3. Sélectionnez les fichiers de Working Papers que vous souhaitez déplacer vers une autre entité.

  4. Sélectionnez More Actions () | Move to Entity.

  5. Choisissez l'entité vers laquelle vous souhaitez déplacer les fichiers des Working Papers et sélectionnez OK.

  6. Dans la boîte de dialogue de confirmation, sélectionnez Oui.

Move Working Papers files between business units

Si votre entreprise utilise la fonction "business units", vous pouvez déplacer des fichiers Working Papers vers des entités d'autres business units. Assurez-vous que vous avez le rôle d' accès à l' entité ou des droits équivalents sur l'entité.

Remarque : Les fichiers des Working Papers ne peuvent pas être transférés en bloc vers d'autres unités commerciales, ils doivent l'être individuellement.

To move a file between business units:

  1. Sélectionnez Toutes les entités ou l'entité contenant les fichiers à déplacer dans le sélecteur d'entité.

  2. Dans le menu Cloud (), sélectionnez Working Papers.

  3. Sélectionnez les fichiers de Working Papers que vous souhaitez déplacer vers une autre entité.

  4. Sélectionnez More Actions () | Move to Entity.

  5. Dans la boîte de dialogue Sélecteur d'entité, vous verrez une liste d'entités dans toutes les unités commerciales auxquelles vous avez accès. Choisissez l'entité vers laquelle vous souhaitez déplacer les fichiers des Working Papers et sélectionnez OK.

  6. Dans la boîte de dialogue de confirmation, sélectionnez Oui.

Répétez ce processus pour chaque fichier Working Papers supplémentaire que vous devez déplacer.
