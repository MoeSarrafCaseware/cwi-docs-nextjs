Mise en place d'une planification centrale

Remarque : Cette fonction n'est disponible que si elle a été activée dans votre produit.

La planification centralisée vous permet de synchroniser les risques, les contrôles et les listes de contrôle sélectionnées entre les missions, de sorte que le travail n'a pas besoin d'être dupliqué dans plusieurs missions. Lorsque vous définissez une mission comme une mission de planification centrale, tous les risques, contrôles et listes de contrôle sélectionnés sont synchronisés avec les missions de destination.

Créer un engagement central de planification

Pour mettre en place la planification centrale, vous devez d'abord désigner un engagement à utiliser comme engagement de planification centrale. C'est dans le cadre de cette mission que vous effectuerez tous les travaux relatifs aux risques, aux contrôles et aux listes de contrôle synchronisées.

Pour définir une mission comme une mission de planification centrale, dans la mission, sélectionnez l'icône Information (), puis sélectionnez Paramètres pour ouvrir la boîte de dialogue Paramètres de la mission. Dans l'onglet Planification centrale , sélectionnez Définir comme engagement de planification centrale.

Remarque : Une fois qu'un engagement est défini comme un engagement de planification centrale, cette action est irréversible.

L'icône de planification centrale () s'affiche désormais dans le menu supérieur et à côté des listes de contrôle synchronisées. Vous pouvez sélectionner l'icône dans le menu supérieur pour ouvrir le Central Planning Hub et afficher une liste de vos engagements de destination et vérifier s'ils sont à jour.

Créer une mission de consolidation

Après avoir défini une mission comme mission de planification centrale, vous pouvez créer des missions de destination.

Pour créer un engagement de destination, accédez à l'engagement de planification centrale dans votre site Cloud et utilisez l'option Créer un engagement à partir de la source . Les risques, les contrôles et les listes de contrôle sélectionnées dans ce nouvel engagement seront alors synchronisés à partir de l'engagement de planification central.

Lorsque vous ouvrez une liste de contrôle synchronisée dans une mission de destination, un message s'affiche pour vous informer que le document a été complété dans la mission de planification centrale, avec un lien vers la liste de contrôle dans cette mission. Les risques et les contrôles continueront à s'afficher dans les missions de destination, mais en lecture seule.

Vous pouvez sélectionner l'icône de planification centrale () dans la barre de menu supérieure pour ouvrir le Hub de planification centrale. Cette page vous informe si des modifications ont été apportées à l'engagement source et vous permet de synchroniser ces modifications.

Report pour la planification centrale

Vous pouvez poursuivre la planification centrale et les engagements de destination de la même manière que vous le feriez pour n'importe quel autre engagement. Cependant, les nouveaux engagements ne seront pas automatiquement liés pour la synchronisation. Vous devez relier manuellement chaque engagement de destination à l'engagement de planification centrale.

Si vous commencez à apporter des modifications à l'engagement de planification centrale avant de le lier aux engagements de destination, ces modifications seront synchronisées avec les engagements de destination une fois que vous les aurez liés.

Lier un engagement de destination créé à partir du report à l'engagement de planification centrale :

  1. S'assurer que les engagements relatifs à la destination et à la planification centrale ont été reportés.

  2. Dans un engagement de destination créé à partir d'un report, sélectionnez l'icône Planification centrale () dans le menu supérieur.

  3. Dans le Central Planning Hub, sélectionnez l'icône de synchronisation () pour lier l'engagement de destination à l'engagement de planification centrale de l'année en cours.

    Remarque : Il peut y avoir un petit délai avant que les engagements de destination nouvellement liés soient visibles dans le Central Planning Hub de l'engagement de planification centrale.
