Comprendre l'ensemble de données de la balance

Lorsque vous importez une balance de vérification à partir d'un fichier CSV ou Excel, vous devez mettre en correspondance les champs afin que les données de la balance de vérification s'affichent correctement dans la mission. Cette rubrique décrit les champs de l'ensemble de données Balance de vérification . Pour plus d'informations sur l'importation de données de transaction à partir de fichiers CSV, voir Importer la balance de vérification à partir d'un fichier CSV ou Excel.

Champ d'application Utilisation
Identifiant du compte(account_id)

Période du compte.

Type de données : Chaîne

Il s'agit d'une obligation : Oui
Account name (account_main_description)

Le nom ou la description du compte.

Exemples : " Trésorerie ", " Immobilisations corporelles ".

Type de données : Chaîne

Il s'agit d'une obligation : Oui
Numéro de groupe financier(financial_group_number)

Le numéro du groupe financier auquel le compte est affecté dans la structure de regroupement. Cette information n'est pas fournie par le client et est utilisée pour rattacher automatiquement les comptes aux groupes financiers.

Les comptes actuellement affectés à des groupes peuvent être réaffectés lors de l'importation de ce champ de données.
