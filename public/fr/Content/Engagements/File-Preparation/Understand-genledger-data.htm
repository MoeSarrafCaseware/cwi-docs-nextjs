Comprendre l'ensemble des données du grand livre général

Lorsque vous importez des transactions du grand livre à partir d'un fichier CSV ou Excel, vous devez affecter les champs du fichier aux champs Caseware dans le type de jeu de données du grand livre que vous sélectionnez lors de l'importation, afin que les données du grand livre s'affichent correctement dans la mission. Cette rubrique explique comment les identifiants uniques sont générés automatiquement lors de l'importation, ainsi que les descriptions des données du grand livre.

Pour plus d'informations sur l'importation de données de transaction à partir de fichiers CSV, voir Importer les transactions du client à partir d'un fichier CSV.

Comment les identifiants uniques des lignes de données du grand livre sont autogénérés ?

Il est important de savoir comment les identificateurs uniques sont autogénérés si des conflits de données surviennent lors de l'importation des données. Voir Résoudre les conflits d'importation de données pour plus d'informations sur la manière dont ils peuvent se produire et sur ce que vous pouvez faire pour les résoudre.

Lorsque vous importez des données du grand livre dans des fichiers CSV ou Excel, chaque ligne de données doit être identifiée de manière unique. Pour ce faire, Cloud crée un identifiant unique composé des valeurs des champs que vous attribuez aux champs Caseware Entry ID (entry_id) et Line Number (line_number)pour former un identifiant unique pour chaque ligne de données :

{entry_id}-{line_number}

  • Lorsque des champs du fichier d'importation entrant sont affectés à l' ID d'entrée mais pas au numéro de ligne ou vice versa, une valeur unique est autogénérée pour l' ID d'entrée ou le numéro de ligne, selon ce qui manque.
  • Lorsque les champs du fichier d'importation entrant ne sont pas affectés à l'ID d'entrée et au numéro de ligne, des combinaisons uniques sont générées automatiquement pour ces deux champs Caseware.
  • Lorsque les utilisateurs attribuent un champ au numéro d'entrée mais pas à l' ID d'entrée, l' ID d'entrée sera rempli automatiquement avec les valeurs du numéro d'entrée.
