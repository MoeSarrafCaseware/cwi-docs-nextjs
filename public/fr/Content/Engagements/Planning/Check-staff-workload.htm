Vérifier la charge de travail du personnel

Si vous utilisez les tâches dans le nuage pour gérer les activités de votre personnel, vous pouvez consulter les tâches en cours pour identifier les membres du personnel que vous pouvez affecter à la nouvelle équipe de mission.

Pour visualiser les tâches assignées au personnel :

  1. Assurez-vous que vous avez le rôle d' administrateur des entités ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Activités.

  3. Sélectionnez le menu Activités , puis Tâches.

  4. Sélectionnez l'icône de filtre (), puis choisissez Statut | Ouvert pour afficher uniquement les tâches ouvertes.
