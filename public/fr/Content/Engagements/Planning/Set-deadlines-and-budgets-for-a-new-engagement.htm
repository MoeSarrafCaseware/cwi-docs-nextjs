Définir les délais et les budgets pour une nouvelle mission

Vous vous préparez à travailler sur une nouvelle mission et vous devez déterminer la durée de la mission et son coût. Vous avez procédé à une première évaluation sur la base des dossiers de l'année précédente et vous avez réfléchi à l'étendue du travail de cette année.

Lorsque vous aurez déterminé ces délais et ces coûts, vous voudrez partager ces informations avec l'équipe chargée de la mission. Vous pouvez ajouter ces informations dans le nuage en mettant à jour les propriétés de votre fichier d'engagement et en créant des tâches d'étape pour l'entité.

Établir un budget

Vous pouvez saisir des informations budgétaires pour les fichiers d'engagement sur le cloud. En saisissant ces données sur les missions elles-mêmes, les entrées de temps de l'application Time peuvent être comptabilisées dans le total budgétisé.

Vous pouvez saisir des informations budgétaires pour les dossiers Working Papers, Other Engagement et US Review.

Pour saisir un budget sur un engagement :

  1. Assurez-vous que vous avez le rôle d' éditeur ou des droits équivalents pour l'entité cliente.

  2. Sélectionnez le fichier de votre engagement, puis sélectionnez Editer ().

  3. Saisissez le temps et le coût budgétés pour cette mission.

  4. Sélectionnez Enregistrer.

Vous avez maintenant ajouté des informations budgétaires à votre dossier d'engagement.

Fixer des délais

Dans le cadre du processus de planification, vous déterminerez les principales étapes de l'engagement et fixerez des délais pour chacune d'entre elles. Vous pouvez ajouter ces étapes à Cloud en créant une tâche pour chacune d'entre elles.

Pour créer une tâche avec une date limite :

  1. Assurez-vous que vous avez le rôle d' éditeur ou des droits équivalents pour l'entité cliente.

  2. Sélectionnez le fichier correspondant à votre engagement. Dans le volet de détail, sélectionnez Ajouter () à côté de Tâches.

  3. Saisissez un objet et une description pour votre tâche dans la boîte de dialogue Créer une tâche.

  4. Choisissez une date d'échéance pour cette tâche.

  5. Sélectionner le poste.

Vous avez créé une nouvelle tâche pour votre étape d'engagement avec une date limite spécifiée. Vous pouvez répéter ce processus pour toutes les étapes de votre engagement, en fixant des dates d'échéance pour chaque étape.
