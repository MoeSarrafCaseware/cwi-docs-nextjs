Partager un fichier Caseware SmartSync sur Caseware Cloud

Une fois que vous avez examiné le dossier de l'année précédente, constitué votre équipe de mission et fixé vos étapes et vos échéances, vous devriez être prêt à partager le nouveau dossier de mission avec votre équipe. Vous pouvez utiliser Cloud comme environnement SmartSync, en configurant une copie parentale dans l'espace de travail du client. Les membres de votre équipe peuvent alors créer des copies synchronisées de ce fichier parent.

Création d'une copie primaire (parent)

Vous avez créé un nouveau fichier Working Papers pour la mission de ce client et vous souhaitez maintenant rendre ce fichier accessible aux autres membres de votre équipe de mission. Pour ce faire, vous devrez publier le fichier local, en créant une copie primaire sur le cloud.

Vous avez publié avec succès votre fichier Working Papers dans le Cloud. Cette opération crée une copie principale du fichier sur le cloud, et vous pouvez maintenant la partager avec d'autres membres du personnel.

Création d'une copie de synchronisation (enfant)

Si le fichier de votre prochain engagement a été publié par quelqu'un d'autre, vous aurez besoin d'une copie du fichier pour travailler dessus. Vous pouvez le faire en ouvrant une copie de synchronisation à partir de Cloud.

Pour créer une copie de synchronisation :

  1. Assurez-vous que vous avez le rôle d' éditeur ou des droits équivalents pour le fichier Working Papers.

  2. Dans le menu Cloud, sélectionnez Working Papers.

  3. Sélectionnez le fichier Working Papers sur lequel vous allez travailler.

  4. Dans le panneau de détails, sélectionnez l'option Plus d'actions () | Ouvrir la copie de synchronisation.

Vous avez maintenant une copie de synchronisation du fichier parent ouvert dans les documents de travail, et vous serez prêt à commencer à répartir le travail ou à attribuer vous-même le travail sur le fichier lorsque la mission commencera.
