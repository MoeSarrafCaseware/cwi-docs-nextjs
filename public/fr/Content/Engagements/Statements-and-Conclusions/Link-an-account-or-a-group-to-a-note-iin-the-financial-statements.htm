Relier un compte ou un groupe à une note dans les états financiers

Une référence de note est un lien vers une note existante dans l'état financier. Vous pouvez ajouter des références de notes dans les lignes de tableau de l'état financier. Par exemple, pour relier un compte ou un groupe à une note existante contenant des détails ou des informations connexes.

Pour ajouter une référence de note dans une ligne :

  1. Accédez à la ligne où vous souhaitez ajouter une référence de note.

  2. Sélectionnez le bouton Ajouter () au-dessus de la ligne.

  3. Sélectionnez la ligne Commentaire.

    Une note de référence est ajoutée juste au-dessus de la ligne.

  4. Sélectionnez (Note#) pour afficher une liste des notes existantes dans l'état financier.

  5. Sélectionnez une note dans la liste pour la référencer dans la ligne.

    Vous pouvez également sélectionner Afficher les notes cachées pour référencer une note cachée.

  6. Sélectionnez SAVE ou SAVE & CLOSE.

Le numéro de la note référencée s'affiche à côté du nom de la ligne.

Vous pouvez également faire référence à des notes dans des colonnes.

Pour ajouter une référence de note dans une colonne :

  1. Ajouter une colonne dans le tableau.

    1. Sélectionnez le tableau.

      Les icônes des options du tableau s'affichent à droite.

    2. Sélectionnez Autres actions (), puis choisissez Modifier le tableau dynamique.

      3. Le tableau s'ouvre en mode édition.

    3. Allez à l'endroit où vous souhaitez ajouter une colonne.

    4. Sélectionnez Ajouter () dans la partie supérieure du tableau.

      5. Une colonne est ajoutée au tableau et les paramètres de la colonne s'affichent sur la page.

  2. Sélectionnez l'en-tête de la nouvelle colonne et modifiez le type de colonne en Référence de note.

  3. Saisissez un en-tête et spécifiez la largeur souhaitée.

  4. Sélectionnez Terminé.

    5. La nouvelle colonne est ajoutée dans le tableau avec des références vides(#).

  5. Spécifiez les références des notes pour chaque ligne.

    1. Sélectionnez une référence vide(#) pour afficher une liste des notes existantes dans le document des états financiers.

    2. Sélectionnez une ou plusieurs notes dans la liste.

      3. Vous pouvez également sélectionner Afficher les notes exclues pour référencer les notes exclues.

  6. Sélectionnez SAVE ou SAVE & CLOSE.

Remarque : Vous pouvez accéder directement à la note référencée dans le domaine des notes aux états financiers en utilisant la référence de la note que vous venez de créer.

Vous pouvez également utiliser les méthodes ci-dessus pour modifier une référence de note ou vous pouvez la modifier directement à partir du tableau.

Pour modifier une référence de note directement à partir du tableau :

  1. Sélectionnez la référence de note que vous souhaitez modifier.

    Une liste des notes existantes dans l'affichage des états financiers

  2. Sélectionnez dans la liste une ou plusieurs notes à référencer dans la ligne ou la colonne.

    Vous pouvez également sélectionner Afficher les notes cachées pour référencer les notes cachées.
