Utiliser des soldes non consolidés dans les états financiers

Une fois que vous avez créé une mission de consolidation, les soldes consolidés sont utilisés dans l'ensemble du document et s'affichent dans chaque tableau. Cependant, vous disposez de trois options pour visualiser les soldes non consolidés :

  • Paramétrer l'ensemble du document pour qu'il utilise des soldes non consolidés.

  • Définir un ou plusieurs tableaux pour afficher les soldes non consolidés.

  • Définir une ou plusieurs colonnes du tableau pour afficher les soldes non consolidés.

Par exemple, vous pouvez paramétrer le document pour qu'il utilise des données non consolidées si vous devez préparer des jeux d'états financiers distincts pour l'entité mère et ses filiales.

Paramétrer le document pour qu'il utilise des soldes non consolidés

Utiliser des soldes non consolidés dans les états financiers

  1. Ouvrez l'état financier et accédez au plan du document.

  2. Sélectionnez Settings () , puis cochez la case Use non-consolidated data (utiliser des données non consolidées ).

    Les données consolidées ne sont plus disponibles dans les états financiers.

Pour inclure à nouveau les données consolidées dans les états financiers, décochez la case Utiliser les données non consolidées .

Définir un tableau pour afficher les soldes non consolidés

Pour afficher des données non consolidées dans un tableau :

  1. Sélectionnez le tableau souhaité.

    2. Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez Autres actions () | Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Sélectionnez Paramètres () | Consolidation, puis cochez la case Utiliser les données non consolidées .

Pour inclure à nouveau les données consolidées dans le tableau, décochez la case Utiliser les données non consolidées .

Définir une colonne de tableau pour afficher les soldes non consolidés

Pour afficher des données non consolidées dans une colonne de tableau :

  1. Sélectionnez le tableau souhaité.

    2. Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez Autres actions () | Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Sélectionnez la colonne souhaitée.

  4. Cochez la case Utiliser des données non consolidées .

    Pour inclure à nouveau les données consolidées dans la colonne, décochez la case Utiliser les données non consolidées .
