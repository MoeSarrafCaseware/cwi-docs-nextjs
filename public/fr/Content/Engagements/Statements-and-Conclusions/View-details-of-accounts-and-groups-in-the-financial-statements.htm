Afficher les détails des comptes et des groupes dans les états financiers

Vous pouvez sélectionner un compte spécifique dans n'importe quel tableau des états financiers pour afficher les données de sa balance de vérification.

Pour visualiser les détails du compte, sélectionnez le compte pour afficher la fenêtre contextuelle Détails du compte qui présente les données de la balance de vérification pour ce compte. Notez que :

  • Si vous sélectionnez une cellule avec un solde écrasé, la fenêtre contextuelle Détails du compte affiche une ligne en surbrillance indiquant le solde original de cette cellule (le montant de la balance de vérification). Pour revenir au montant initial, sélectionnez Revert (). Pour en savoir plus, voir Soldes d'annulation dans les états financiers.

  • Si une différence d'arrondi est appliquée au compte, la fenêtre contextuelle Détails du compte affiche également une ligne en surbrillance indiquant les détails de l'arrondi. Pour en savoir plus, voir Voir les détails des arrondis dans les états financiers.

  • Dans les missions de consolidation, la fenêtre contextuelle Détails du compte affiche une colonne distincte supplémentaire pour l'élimination des ajustements.

    Remarque : Cette colonne n'est pas disponible si vous choisissez d'afficher les soldes non consolidés dans les états financiers. Pour en savoir plus, voir Utiliser des soldes non consolidés dans les états financiers.

Vous pouvez également développer ou réduire les groupes pour afficher les données de chaque compte du groupe ou le montant total des comptes d'un groupe.

Remarque: Vous ne pouvez pas réduire ou développer les groupes à moins que le paramètre pour les comptes dans le tableau soit Inclus (Réduit) ou Inclus (Développé). Pour en savoir plus, voir Définir une option d'affichage par défaut pour les comptes et les groupes dans les états financiers.
