Ajouter un saut de page dans les états financiers

Vous pouvez insérer des sauts de page dans vos états financiers. Vous pouvez utiliser un saut de page pour indiquer que vous prévoyez d'ajouter du contenu ou pour créer un espace vide entre deux zones.

Vous pouvez également ajouter un saut de page à l'intérieur d'un tableau dans le document afin de mieux présenter les informations du tableau. Par exemple, vous pouvez insérer un saut de page dans un tableau pour que des informations similaires ou connexes soient affichées sur la même page au lieu d'être réparties sur deux pages.

Remarque: Vous ne pouvez pas supprimer les sauts de page dans le contenu par défaut de votre produit.

Pour ajouter un saut de page dans les tableaux :

  1. Localisez l'endroit où vous souhaitez ajouter votre saut de page.

  2. Allez sur le côté gauche et sélectionnez l'icône Ajouter () pour ajouter un nouveau contenu.

  3. Choisissez Saut de page dans la liste des types de contenu.

Pour ajouter un saut de page dans les tableaux :

  1. Sélectionnez le tableau souhaité.

    Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez l'icône Autres actions ().

  3. Sélectionnez Modifier le tableau dynamique dans la liste des actions.

    Le tableau s'ouvre en mode édition.

  4. Localisez l'endroit où vous souhaitez ajouter un saut de page.

  5. Sélectionnez l'option Ajouter () | Saut de page sur le bord gauche du tableau.

  6. Sélectionnez SAVE ou SAVE & CLOSE en haut de la page.
