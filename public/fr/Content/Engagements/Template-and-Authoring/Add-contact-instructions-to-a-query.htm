Ajouter des instructions de contact

Chaque document d'interrogation comporte une section Instructions en haut de page. Vous pouvez utiliser cette section pour inclure des informations d'orientation pour vos contacts, en fournissant des instructions sur la manière de remplir la requête ou des clarifications supplémentaires sur les demandes.

Pour ajouter des instructions dans la requête, sélectionnez la zone de texte jaune et saisissez les informations d'orientation.

Remarque : Si vous laissez la section Instructions vide, elle ne s'affiche pas dans le document lorsque les contacts assignés y accèdent.
