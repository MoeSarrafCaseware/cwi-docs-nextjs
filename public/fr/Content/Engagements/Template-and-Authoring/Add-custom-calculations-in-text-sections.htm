Add custom calculations in text sections

Vous pouvez ajouter des zones PDF intégrées dans les états financiers.

Pour ajouter des calculs personnalisés :

  1. Select the text section where you want to add the custom calculation.

  2. La barre d'outils de formatage s'affiche en haut de la section de texte. Select the Insert Calculation icon ().

  3. Dans la boîte de dialogue Calcul, sélectionnez Insérer des données (fx).

  4. La boîte de dialogue Data Link s'ouvre. Sélectionnez l'onglet Référence de cellule .

  5. Dans la liste déroulante Sélectionner un domaine/une note , sélectionnez dans la liste des domaines ou des notes disponibles à référencer.

  6. Dans le champ Sélectionner un tableau dynamique, recherchez un tableau dans le domaine des états financiers ou dans la note, ou sélectionnez un tableau dans la liste ci-dessous.

  7. Dans l'aperçu du tableau dynamique, sélectionnez la cellule concernée pour y faire référence.

  8. De retour dans la boîte de dialogue Calcul, saisissez le calcul souhaité.

    Lorsque vous avez terminé, sélectionnez Terminé.

Le calcul personnalisé est ajouté à la section texte.
