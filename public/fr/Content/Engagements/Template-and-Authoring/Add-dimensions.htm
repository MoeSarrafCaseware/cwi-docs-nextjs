Ajouter des dimensions

Remarque : Cette fonction n'est disponible que si elle est activée dans votre produit.

Une dimension est une propriété de groupe. Un compte peut être affecté à une dimension au sein d'un groupe spécifique pour un tri et une catégorisation plus poussés.

Vous pouvez créer une ou plusieurs catégories de dimensions, puis les remplir de dimensions. Par exemple, vous pouvez créer une catégorie de dimension appelée Emplacement, puis créer une dimension pour chaque emplacement de votre client. Une fois les dimensions créées, vous pouvez les affecter à des groupes financiers.

Les restrictions suivantes s'appliquent à l'affectation des catégories de dimensions aux groupes :

  • Les utilisateurs finaux ne peuvent pas supprimer les dimensions créées par les auteurs.

  • Les auteurs du cabinet et les utilisateurs finaux ne peuvent pas modifier les catégories de dimension pour les groupes qui ont hérité des catégories de groupes de niveau supérieur

Pour ajouter un ensemble de dimensions :

  1. Aller à Caractéristiques | Dimensions.

  2. Sélectionnez + Catégorie, puis ajoutez un nom pour la nouvelle catégorie de dimension.

  3. Sélectionnez + Dimension pour ajouter une dimension à la nouvelle catégorie.

  4. Saisissez un nom pour la nouvelle dimension.

    Si vous souhaitez utiliser des numéros de dimension pour attribuer un identifiant unique à chaque dimension, saisissez une valeur dans le champ Numéro. Le numéro peut être composé de lettres ou de chiffres. Ces numéros de dimension peuvent être utilisés pour associer des dimensions à des transactions.

    Vous pouvez faire glisser et déposer les dimensions pour les réorganiser au sein d'une catégorie et utiliser les flèches vers le haut et vers le bas pour développer et réduire les catégories. Vous pouvez également sélectionner Supprimer la dimension () pour supprimer une dimension.

    Vous pouvez sélectionner Ajouter () pour ajouter une dimension de second niveau à votre dimension primaire. Vous pouvez également ajouter des dimensions de troisième niveau aux dimensions de deuxième niveau.

    Vous pouvez définir n'importe quelle dimension sans enfant comme dimension par défaut pour la catégorie. Les comptes des groupes auxquels s'applique cette catégorie seront automatiquement affectés à cette dimension. Pour les catégories de dimensions sans jeu de dimensions par défaut, les comptes ne seront pas automatiquement mis en correspondance avec une dimension.

Lorsque vous avez créé vos dimensions, vous pouvez ensuite affecter les catégories de dimensions à des groupes financiers. Notez que vous pouvez attribuer plusieurs catégories de dimensions à un seul groupe. Voir Propriétés du groupe pour plus d'informations.

Pour affecter une catégorie de dimension à un groupe financier :

  1. Accédez à la catégorie dans les paramètres du produit.

  2. Sélectionnez Autres actions () | Attribuer des groupes.

  3. Si votre produit possède plusieurs structures de regroupement financier, sélectionnez la structure de regroupement souhaitée dans la liste déroulante.

  4. Sélectionnez tous les groupes financiers applicables.

    Vous pouvez développer et réduire les groupes financiers pour afficher les sous-groupes et utiliser la barre de recherche pour trouver les groupes appropriés.
