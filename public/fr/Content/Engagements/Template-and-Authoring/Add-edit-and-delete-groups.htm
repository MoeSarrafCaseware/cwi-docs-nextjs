Visualisation des groupes financiers

Dans la page Données , vous pouvez ajouter manuellement des groupes de balances de vérification à une structure de regroupement dans le modèle de société, si nécessaire. Vous pouvez également modifier ou supprimer les groupes que vous créez. Notez que vous ne pouvez pas modifier ou supprimer les groupes préexistants dans le modèle.

Remarque : Pour permettre au personnel d'ajouter des groupes financiers, cochez la case Activer les groupes financiers définis par l'utilisateur dans l'onglet Données des paramètres du produit. Vous pouvez également décocher la case pour restreindre ce paramètre.

Ajouter un groupe

Vous pouvez ajouter manuellement des groupes à une structure de regroupement.

Pour ajouter un groupe :

  1. Accédez à la page Données , puis sélectionnez la structure de regroupement à laquelle vous souhaitez ajouter un groupe dans le volet de gauche :

    • Groupementsfinanciers

    • Groupementsfiscaux

    • Groupementsprescrits .

  2. Sélectionnez Ajouter () | Ajouter un groupe.

    La boîte de dialogue Propriétés du groupe s'affiche.

  3. Définir les propriétés du nouveau groupe.

    Reportez-vous à la section Propriétés du groupe pour une description de chaque propriété du groupe.

  4. Sélectionnez Enregistrer pour ajouter le groupe.

Modifier un groupe

Vous pouvez modifier les propriétés d'un groupe que vous avez ajouté au modèle d'entreprise.

Pour modifier un groupe :

  1. Accédez à la page Données , puis sélectionnez la structure de regroupement dans le volet de gauche :

    • Groupementsfinanciers

    • Groupementsfiscaux

    • Groupementsprescrits .

  2. Sélectionnez le groupe que vous souhaitez modifier, puis sélectionnez Editer () pour ouvrir la boîte de dialogue Propriétés du groupe.

    Conseil : Vous pouvez rechercher des groupes en utilisant le champ de recherche situé dans le coin supérieur droit de la page.

  3. Modifiez les propriétés du groupe si nécessaire.

    Reportez-vous à la section Propriétés du groupe pour une description de chaque propriété du groupe.

  4. Une fois que vous avez terminé, sélectionnez Enregistrer.

Réorganiser un groupe

Vous pouvez faire glisser et déposer des groupes dans la page Données pour les réorganiser. Naviguez jusqu'à la structure de regroupement de la page Données et faites glisser le groupe que vous souhaitez réorganiser à l'endroit désiré.

Notez que vous ne pouvez utiliser le glisser-déposer pour réorganiser les groupes que dans les conditions suivantes :

  • Les groupes que vous réordonnez sont tous des groupes frères dans la structure de regroupement. Vous ne pouvez pas glisser-déposer un groupe enfant pour qu'il soit au même niveau ou à un niveau supérieur que le groupe parent.

  • Tous les groupes de frères et sœurs sont réduits ou n'ont pas de groupes d'enfants.

Supprimer un groupe

Vous pouvez supprimer un groupe que vous avez ajouté au modèle d'entreprise.

Pour supprimer un groupe :

  1. Accédez à la page Données , puis sélectionnez la structure de regroupement dans le volet de gauche :

    • Groupementsfinanciers

    • Groupementsfiscaux

    • Groupementsprescrits .

  2. Sélectionnez le groupe que vous souhaitez modifier, puis sélectionnez Editer () pour ouvrir la boîte de dialogue Propriétés du groupe.

    Conseil : Vous pouvez rechercher des groupes en utilisant le champ de recherche situé dans le coin supérieur droit de la page.

  3. Sélectionnez Supprimer.
