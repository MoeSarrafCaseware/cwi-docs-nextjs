Attribuer des composants aux éléments de votre modèle

Components help you set up a market-driven, region-driven or product-driven visibility logic for content. Par exemple, si l'entité opère dans le secteur de l'agriculture, affichez les groupes de balances de vérification qui sont pertinents pour l'agriculture. Pour en savoir plus, voir Définir la matérialité.

If you associate multiple components to content and define visibility settings for each one of the components, then the visibility logic varies based on whether the assigned components belong to the same category or different categories:

  • Si les composants affectés appartiennent à la même catégorie, le groupe de balance générale est visible si au moins un composant est visible - sinon le groupe est masqué

  • Si les composants affectés appartiennent à des catégories différentes, le groupe de balance générale est visible si au moins un composant de chaque catégorie est visible - sinon, le groupe est masqué

Notez que :

  • Un élément auquel ont été attribués à la fois des paramètres de visibilité et des composants ne sera visible que si les paramètres de visibilité de l'élément et la visibilité du composant sont tous deux évalués comme étant visibles.

  • Si un élément est masqué en raison des paramètres de visibilité de l'élément, les utilisateurs finaux peuvent toujours le voir en cochant la case Afficher le masqué .

  • Si un élément est masqué en raison de la visibilité du composant, les utilisateurs finaux ne peuvent le voir qu'en cochant la case Afficher le masqué si l'option Occlude du composant n'est pas sélectionnée dans les paramètres du produit.

Affecter des composants à un document

Vous pouvez attribuer des composants à un document entier.

To assign components to a document:

  1. Sur la page Documents , sélectionnez Autres actions () | Affecter des composants pour le document afin d'afficher la boîte de dialogue Affecter des composants.

  2. Sélectionnez Ajouter ().

  3. Sélectionnez le formulaire que vous souhaitez ajouter dans la liste déroulante.

  4. Une fois que vous avez terminé, sélectionnez APPLY.

Attribuer des composants aux éléments de votre modèle

Vous pouvez affecter des composants aux éléments suivants dans les documents :

  • Listes de contrôle

    • Groupes et sous-groupes

    • Procédures et sous-procédures

  • Sections dans les lettres et mémos

  • Questions d'interrogation

  • Domaines et sections des états financiers

Pour affecter des composants à un élément d'un document :

  1. Naviguez jusqu'à l'élément auquel vous souhaitez affecter des composants.

  2. Select More actions () | Assign Components to display the Assign Components dialog.

  3. Sélectionnez Ajouter ().

  4. Sélectionnez le formulaire que vous souhaitez ajouter dans la liste déroulante.

  5. Une fois que vous avez terminé, sélectionnez APPLY.
