Copier un tableau dans les états financiers

Vous pouvez copier un tableau dynamique existant dans les états financiers et le coller n'importe où dans les états financiers. Une fois que vous avez copié un tableau, il reste dans le presse-papiers et vous pouvez le coller autant de fois que vous le souhaitez jusqu'à ce que vous copiez un autre tableau dans le presse-papiers.

Pour copier un tableau dynamique :

  1. Sélectionnez le tableau dynamique que vous souhaitez copier.

  2. Sélectionnez Autres actions () | Copier le tableau dynamique.

  3. Sélectionnez l'endroit où vous souhaitez ajouter le tableau copié, puis sélectionnez Ajouter | Coller le tableau dynamique copié.
