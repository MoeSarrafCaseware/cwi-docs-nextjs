Créer des groupes personnalisés dans les tableaux

Deux méthodes permettent de créer des groupes personnalisés dans les tableaux dynamiques :

  • Regrouper plusieurs lignes.

  • Ajouter une ligne de groupe personnalisée.

Regrouper plusieurs lignes de saisie

Vous pouvez regrouper plusieurs lignes pour créer un groupe personnalisé dans les tableaux dynamiques.

Pour créer un groupe personnalisé en regroupant plusieurs lignes :

  1. Sélectionnez le tableau.

    2. Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez Autres actions () | Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Sélectionnez les lignes que vous souhaitez regrouper.

    4. Notez que vous pouvez sélectionner :

    • Lignes de balance de vérification, lignes d'entrée ou une combinaison des deux
    • Lignes de compte unique ou lignes de groupe

  4. Sélectionnez l'icône Grouper les lignes sélectionnées ()

    5. Une ligne de groupe personnalisé est ajoutée dans le tableau avec un titre par défaut, Title. Vous pouvez modifier le titre par défaut.

    Une ligne de total est également ajoutée dans le tableau avec un titre par défaut : Total. Vous pouvez modifier le titre par défaut.

  5. Sélectionnez SAVE & CLOSE.

Ajouter une ligne de groupe personnalisée

L'ajout d'une ligne de groupe personnalisée ajoute une ligne de champ de texte. Vous pouvez l'utiliser pour saisir un titre ou une brève description dans le tableau.

Vous pouvez également l'utiliser pour inclure le bouton () dans le tableau. Le bouton +Row permet au personnel d'ajouter directement des lignes au groupe personnalisé sans devoir passer au mode d'édition du tableau.

Ceci est utile dans les cas où vous ne pouvez pas prédéfinir le nombre de lignes nécessaires. Par exemple, si chaque membre du personnel doit ajouter ses coûts de main-d'œuvre dans un tableau ou inclure des lignes supplémentaires dans l'état des flux de trésorerie.

Pour permettre l'ajout de lignes de saisie directement à un groupe personnalisé :

  1. Sélectionnez le tableau.

    2. Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez Autres actions () | Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Repérez l'endroit où vous souhaitez permettre au personnel d'ajouter des lignes de saisie, puis sélectionnez le groupe Ajouter () | Personnaliser dans le bord gauche du tableau.

    4. Remarque : L'ajout d'un groupe personnalisé marque l'endroit du tableau où vous souhaitez permettre aux utilisateurs d'ajouter des lignes de saisie en dehors du mode d'édition.

    Une nouvelle ligne de groupe est ajoutée dans le tableau avec un titre par défaut Titre. Vous pouvez sélectionner le titre par défaut pour le modifier.

    Si vous ne voulez pas que la ligne de groupe ou la ligne de total apparaisse dans le tableau, sélectionnez la ligne puis l'icône de l'œil () pour la masquer. Dans ce cas, il est recommandé de sélectionner également l'option Paramètres () | Afficher les éléments cachés pour garder une trace de toutes les lignes existantes dans le tableau pendant que vous travaillez dessus.

  4. Sélectionnez le nouveau groupe personnalisé.

    5. L'icône Activer les rangées d'entrée () s'affiche dans le coin supérieur gauche.

  5. Sélectionnez Activer les rangées d'entrée ().

  6. Sélectionnez SAVE & CLOSE.

Pour supprimer le bouton (), sélectionnez le groupe que vous avez ajouté à l'étape 3, puis sélectionnez ().
