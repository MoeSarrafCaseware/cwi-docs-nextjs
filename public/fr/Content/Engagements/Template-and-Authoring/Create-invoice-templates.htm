Créer des modèles de factures

Cloud vous permet de personnaliser les factures pour répondre aux besoins de votre organisation et de votre client. Vous pouvez modifier les options de facturation par défaut, créer des modèles de factures et gérer les codes de facturation directement à partir de Cloud.

Création de modèles de factures

La création d'un modèle de facture vous permet de générer de nouvelles factures qui répondent aux besoins spécifiques de votre organisation.

Vous pouvez créer une facture personnalisée dans les paramètres afin de répondre aux exigences de votre client ou de votre type d'engagement.

Vous avez créé un modèle de facture. Vous pouvez l'appliquer lorsque vous générez des factures à l'aide de l'application Facturation , ou consulter la liste des modèles disponibles dans Paramètres | Temps | Modèles.

Modification et suppression des modèles de factures

Si un modèle ne répond plus à vos besoins, vous pouvez le modifier ou le supprimer.

Pour modifier ou supprimer un modèle de facture :

  1. Assurez-vous d'avoir le rôle d' administrateur des paramètres ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Sélectionnez Temps puis, dans la section Facture, sélectionnez Modèles.

  4. Sélectionnez un modèle de facture.

  5. Sélectionnez Modifier ou Supprimer.

Vous avez modifié ou supprimé une facture.

Ajout d'un numéro d'enregistrement fiscal à un nouveau modèle de facture

Vous pouvez ajouter automatiquement un numéro d'enregistrement fiscal aux factures qui le nécessitent en créant un modèle de facture personnalisé.

Pour ajouter un numéro d'enregistrement fiscal à un nouveau modèle de facture :

  1. Assurez-vous d'avoir le rôle d' administrateur des paramètres ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Sélectionnez Temps puis, dans la section Facture, sélectionnez Modèles.

  4. Sélectionnez Nouveau.

  5. Ajoutez le widget Informations sur le client , le widget Informations sur l'organisation , ou les deux. Ces widgets peuvent être ajoutés aux sections Main, Header ou Footer de votre modèle.

  6. Sélectionnez l'icône Paramètres ( ).

  7. Cochez la case Numéro d'enregistrement fiscal principal et sélectionnez OK.

  8. Complétez votre modèle de facture comme vous le souhaitez et sélectionnez Enregistrer.

Les factures créées avec votre modèle afficheront un numéro d'enregistrement fiscal.
