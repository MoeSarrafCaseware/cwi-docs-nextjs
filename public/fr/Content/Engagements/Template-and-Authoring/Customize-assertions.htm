Personnaliser les assertions

Under the Features drop-down, select Assertions.

Pour ajouter une assertion :

  1. Sélectionnez Ajouter une condition.

  2. Saisir l'abréviation et le nom complet de l'assertion.

Notez que toutes les assertions doivent être associées à l'une des 12 assertions de base ; les assertions non associées seront supprimées.
