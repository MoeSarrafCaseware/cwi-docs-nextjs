Personnaliser la taille des polices et l'espacement dans les états financiers

Vous pouvez personnaliser la taille des polices, ainsi que les marges, les retraits et le remplissage du texte utilisé dans les états financiers. Notez que les sélections que vous faites pour ces paramètres dans la boîte de dialogue Options de formatage ont la priorité sur les options de police que vous sélectionnez dans les paramètres du produit.
