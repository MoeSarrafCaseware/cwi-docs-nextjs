Personnaliser les arrondis dans les états financiers

Si l'arrondi est activé dans votre produit, vous pouvez personnaliser les paramètres d'arrondi utilisés dans les états financiers du modèle de cabinet ou dans un fichier de mission individuel.

Pour personnaliser les arrondis dans les états financiers :

  1. Dans les états financiers, sélectionnez Paramètres du document () | Arrondissement.

  2. Dans le champ Arrondir aux nombres entiers , choisissez d'arrondir aux Unes, aux Milles, aux Millions ou aux Milliards.

  3. Dans le champ Afficher les décimales , sélectionnez le nombre de décimales à afficher dans la liste déroulante.

  4. Cochez les cases Afficher les informations d'arrondi sur les en-têtes de description et Afficher les informations d'arrondi sur les en-têtes de colonnepour afficher les informations d'arrondi dans ces en-têtes dans les tableaux dynamiques.

    Vous pouvez également personnaliser la terminologie utilisée dans chaque type de rubrique.

  5. Lorsque vous avez terminé de personnaliser les paramètres, sélectionnez Fermer.
