Personnaliser la langue du contenu

Dans l'onglet Format | Langue du contenu , vous pouvez modifier les paramètres par défaut de la langue utilisée pour le contenu et les produits livrables du cabinet.

Pour modifier les paramètres de langue du contenu par défaut :

  1. Allez dans l'onglet Langue du contenu.

    2. Remarque : Les langues disponibles dans cet onglet sont prédéfinies. Si votre produit ne prend pas en charge plusieurs langues, une seule langue sera disponible dans cet onglet.

  2. Modifiez la langue par défaut des dossiers de mission de votre cabinet en sélectionnant le bouton Défaut .

  3. Décochez la case Actif pour les langues qui ne s'appliquent pas à votre entreprise.

    4. Remarque : Les langues inactives ne s'affichent pas dans la fenêtre contextuelle des propriétés de l'engagement .

Notez que si vous mettez à jour votre modèle d'entreprise avec les dernières modifications du modèle de produit :

  • Votre choix de la langue par défaut est conservé et n'est pas remplacé.

    • Par exemple, si vous avez spécifié l'anglais comme langue par défaut et que vous mettez à jour le modèle d'entreprise, l'anglais reste la langue par défaut même si le modèle de produit a le français comme langue par défaut.

  • Votre sélection de langues actives peut être remplacée.

    • Si vous avez transformé une langue inactive en langue active, puis mis à jour le modèle de l'entreprise, vos modifications sont remplacées pour correspondre au modèle du produit.

    Par exemple, si vous avez rendu le français actif dans le modèle d'entreprise et que vous mettez à jour le modèle, le français reste actif s'il est défini comme actif dans le modèle de produit et redevient inactif s'il est défini comme inactif dans le modèle de produit.
