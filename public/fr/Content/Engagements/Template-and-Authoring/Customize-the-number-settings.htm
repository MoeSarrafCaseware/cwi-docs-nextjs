Personnaliser les paramètres des numéros

Dans Format | Numéros, vous pouvez personnaliser les paramètres suivants :

Paramètre

Description

Format des nombres négatifs

Comment les nombres négatifs s'affichent dans votre produit :

  • entre parenthèses ( (1 234 567,46) )
  • signe moins précédant le nombre ( -1,234,567.46 )
  • le signe moins après le nombre ( 1,234,567.46- )

Symbole décimal

Utilisation d'un point (.) ou d'une virgule (,) comme symbole décimal.

Décimales

Nombre de décimales affichées dans les soldes.

Symbole de regroupement des chiffres

Utiliser un point (.) ou une virgule (,) comme symbole de regroupement des chiffres.

Groupement de chiffres

Combien de chiffres sont regroupés entre les symboles de regroupement des chiffres.
