Modifier des notes dans les états financiers

Vous pouvez modifier le tableau dynamique ou la section de texte d'une note. Pour modifier une section de texte dans une note, voir Personnaliser le texte.

Pour modifier ou copier un tableau dynamique dans une note, reportez-vous aux rubriques suivantes :

  • Modifier les descriptions de tableaux dans les états financiers

  • Modifier les lignes des tableaux dans les états financiers

  • Modifier les colonnes des tableaux dans les états financiers

  • Copier un tableau dans les états financiers

Vous pouvez supprimer une note entière ou le contenu d'une note qui n'est plus nécessaire dans le document.

Sélectionnez Autres actions () | Supprimer pour supprimer le contenu ou les titres d'une note. Par exemple, pour supprimer un titre de note dans une note, sélectionnez le titre de note, puis sélectionnez Autres actions () | Supprimer.

Pour supprimer une note :

  1. Allez dans le plan du document, puis sélectionnez Notes to the Financial Statements (Notes sur les états financiers).

  2. Sélectionnez la note que vous souhaitez supprimer, puis sélectionnez Autres actions ()| Supprimer la note dans le coin supérieur droit de la page.

    3. Tous les titres et le contenu de cette note sont également supprimés.

Remarque : Vous ne pouvez pas supprimer les notes qui font partie du contenu par défaut de votre produit. Vous pouvez toutefois les exclure du document.
