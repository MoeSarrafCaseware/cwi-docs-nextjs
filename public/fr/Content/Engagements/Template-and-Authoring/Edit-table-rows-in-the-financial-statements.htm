Modifier les lignes des tableaux dans les états financiers

Vous pouvez supprimer toutes les lignes existantes dans un tableau dynamique ou les réorganiser. Vous pouvez également ajouter des lignes pour inclure plus d'informations ou pour ajouter des espaces blancs entre les lignes afin d'améliorer la lisibilité.

Remarque : La taille de police par défaut pour le contenu des tableaux dynamiques est de 12 et ne peut être modifiée.

Ajouter des lignes de tableau

Vous pouvez également ajouter des lignes de saisie directement sur le tableau dynamique sans devoir passer en mode édition, si cette option est activée dans votre produit.

Pour ajouter une ligne d'entrée directement sur le tableau :

  1. Allez à l'endroit où vous souhaitez ajouter une ligne de saisie.

  2. Sélectionner +Row.

    3. Remarque : L'option +Row n'est disponible que si elle est activée dans votre produit.

  3. Ajoutez les informations souhaitées dans la nouvelle ligne.

Vous pouvez supprimer directement une ligne d'entrée en sélectionnant le bouton () à la fin de la ligne.

Déplacer les lignes du tableau

Supprimer des lignes de tableau

Pour supprimer une ligne :

  1. Cochez la case située sur le bord droit de la ligne que vous souhaitez supprimer.

  2. Sélectionnez Supprimer () .

Masquer les lignes du tableau

Pour masquer une ligne de la balance générale :

Remarque : Vous ne pouvez masquer que les lignes dont la source de données est un groupe de balances de vérification et non un compte. Les icônes Masquer / Afficher (/) ne sont visibles que pour les lignes du groupe de balances de vérification.

  1. Cochez la case située à droite de la ligne de la balance générale que vous souhaitez supprimer.

  2. Sélectionnez Masquer () .

    Pour rendre la ligne à nouveau visible, sélectionnez Afficher ().

Afficher les comptes inversés

Vous pouvez également modifier le signe par défaut d'une ligne pour qu'elle s'affiche dans un signe opposé. Par exemple, dans le compte de résultat, vous souhaitez afficher les recettes et la marge brute comme des nombres positifs (bien que le signe par défaut de ces comptes soit négatif).
