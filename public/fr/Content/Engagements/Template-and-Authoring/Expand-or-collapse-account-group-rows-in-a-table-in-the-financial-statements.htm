Développer ou réduire les lignes d'un groupe de comptes dans un tableau des états financiers

Si un tableau dynamique contient des données de balance générale, vous pouvez développer ou réduire les lignes du groupe de comptes. Lorsque vous développez un groupe de comptes, vous visualisez les informations et les chiffres de chaque compte du groupe. Lorsque vous réduisez un groupe de comptes, vous visualisez le montant total des comptes du groupe.

Vous pouvez également développer ou réduire les groupes lors de l'édition du tableau.

Pour modifier les paramètres du groupe :

  1. Sélectionnez Autres actions (), puis choisissez Réduire pour masquer tous les comptes d'un groupe.

    Pour afficher tous les comptes du groupe de comptes, sélectionnez Développer ().

  2. Sélectionnez Autres actions (), puis choisissez Ne pas mettre en retrait les lignes de détail pour afficher les comptes du groupe sans retrait.
