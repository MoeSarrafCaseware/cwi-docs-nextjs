Achetez et activez Caseware Cloud Connector pour votre cabinet

Les licences pour Cloud Connector peuvent être achetées directement à l'aide d'un système de paiement automatique. Pour chaque licence achetée, un utilisateur supplémentaire pourra accéder à Cloud Connector. Pour plus d'informations sur les prix, voir MyCaseware.

Vous pouvez acheter des licences Cloud Connector pour votre entreprise auprès de MyCaseware.

To enable Cloud Connector so staff can use it, see Manage access for Caseware Cloud apps.
