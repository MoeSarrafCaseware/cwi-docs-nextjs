Supprimer le contenu du modèle de cabinet

Vous ne pouvez pas masquer ou modifier les paramètres de visibilité des documents par défaut ou du contenu fourni avec votre produit. Cependant, vous pouvez le supprimer dans le modèle du cabinet afin qu'il n'apparaisse pas dans les dossiers de mission de votre cabinet.

Remarque : Pour supprimer le contenu du modèle d'entreprise, vous devez avoir le rôle d' administrateur des paramètres ou un rôle similaire pour accéder au modèle d'entreprise. Pour plus d'informations, voir Rôles intégrés et Attribuer des rôles de sécurité.

Supprimer un document dans le modèle de cabinet

Si un document faisant partie du contenu par défaut de votre produit n'est pas pertinent pour votre engagement, vous pouvez le supprimer.
