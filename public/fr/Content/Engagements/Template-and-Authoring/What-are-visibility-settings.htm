Que sont les paramètres de visibilité ?

Lesmembres du personnel peuvent ajouter et modifier les paramètres de visibilité des éléments qu'ils ont ajoutés dans un engagement.

Les auteurs du cabinet peuvent ajouter et modifier les paramètres de visibilité pour les éléments qu'ils ont ajoutés dans le modèle de cabinet.

Notez que :

  • Le personnel ne peut pas ajouter ou modifier les paramètres de visibilité pour les documents ou le contenu par défaut fournis avec leur application cloud.

  • Les auteurs de l'entreprise ne peuvent pas ajouter ou modifier les paramètres de visibilité pour les documents ou le contenu par défaut fournis avec leur application Cloud.
