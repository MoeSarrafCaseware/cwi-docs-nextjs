Ajouter des feuilles de travail sur le terrain

Audit International comprend des fiches de travail personnalisées pour :

  • Procédures d'analyse de fond

  • Échantillonnage de fond - Tests de détail

  • Échantillonnage - Tests de contrôle

Les feuilles de travail prédéfinies sont mises à la disposition de l'utilisateur de la manière suivante :

  • Procédures analytiques de fond : ces feuilles de travail sont disponibles pour tous les domaines financiers prédéfinis.

  • Échantillonnage de substance - Tests de détail : ces feuilles de travail sont incluses pour une sélection de domaines financiers.

  • Échantillonnage - Tests des contrôles : ces feuilles de travail sont incluses pour les cycles économiques prédéfinis.

Si vous créez un nouveau domaine financier et que vous souhaitez ajouter l'une de ces trois feuilles de calcul, procédez comme suit :

  • Dans la page Documents, naviguez jusqu'au dossier dans lequel vous souhaitez ajouter la feuille de calcul du formulaire personnalisé.

  • Cliquez sur l'icône d'ajout (+) située à droite du nom du dossier.

  • Sélectionnez la feuille de calcul correspondante dans la section Formulaires personnalisés.

  • Fournir un numéro/identifiant de document et modifier le nom en conséquence dans le dialogue.

  • Sélectionnez Enregistrer pour ajouter la feuille de calcul du formulaire personnalisé à l'engagement.
