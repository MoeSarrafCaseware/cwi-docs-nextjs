Ajouter des contrôles déterminés

Si vous avez identifié des contrôles potentiels, ajoutez-les dans le dossier de mission.

Pour ajouter un contrôle :

  1. Dans la barre d'outils de l'engagement, sélectionnez Nouveau poste d'engagement () | Contrôle. La boîte de dialogue Nouveau contrôle s'affiche.

  2. Saisissez un nom et une description pour le dossier.

  3. Compléter toute autre option applicable.

  4. Sélectionnez Enregistrer.

Le contrôle s'affiche désormais dans le document de rapport 551 Contrôles .
