Définir la signification

Utilisez le document 420 Caractère significatif pour définir le caractère significatif de l'audit. Vous pouvez définir l' importance relative globale, l' importance relative de la performance et le seuil d'anomalies négligeables.

Compléter le document de matérialité :

  1. Dans la page Documents , ouvrez 420 Materiality.

  2. Dans la section Principaux utilisateurs des états financiers , indiquez les principaux utilisateurs et les facteurs à prendre en considération. Vous pouvez sélectionner + Ajouter une ligne pour ajouter d'autres utilisateurs principaux au document.

  3. Dans la section Importance relative globale , saisissez un ou plusieurs facteurs pour déterminer l'importance relative. Vous pouvez ajouter deux types de base supplémentaires :

    • Base de matérialité financière : Base de matérialité financière - sélectionnez un groupe dans la balance de vérification et saisissez un pourcentage de référence.

    • Autre base : Autre base - saisissez une description et un montant de base personnalisés, ainsi qu'un pourcentage de référence.

  4. Dans la section " Informations qualitatives" , identifiez toutes les inexactitudes possibles qui pourraient être significatives pour les utilisateurs prévus des états financiers.

  5. Dans la section Matérialité des performances , saisissez un pourcentage de référence à appliquer à la matérialité globale de la planification.

  6. Dans la section des inexactitudes négligeables , indiquez un montant maximal pour les inexactitudes négligeables.

  7. Dans la section Importance relative pour des circonstances spécifiques , saisissez un ou plusieurs facteurs d'importance relative pour des circonstances spécifiques.

  8. Dans la section Matérialité des performances pour des circonstances spécifiques, saisissez un ou plusieurs facteurs de matérialité pour des circonstances spécifiques.
