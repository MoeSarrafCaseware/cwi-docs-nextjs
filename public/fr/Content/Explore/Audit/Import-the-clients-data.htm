Importer les données du client

Pour alimenter la balance de vérification, vous devez importer les données financières dans votre fichier d'engagement. Par exemple, vous pouvez importer des données directement à partir d'un logiciel de comptabilité en ligne tel que QuickBooks Online et Xero.

Voici les options d'importation disponibles dans la page Données :

  • Caseware Working Papers

  • Logiciels de comptabilité de bureau

  • Engagement file

  • Excel/CSV

  • Accounting software or structured file
