Modifier et examiner les risques

L'onglet Risques affiche une liste des risques identifiés dans votre mission, où vous pouvez visualiser, ajouter, modifier et supprimer des risques. Vous pouvez également accéder à cet onglet à partir de 520 rapports sur les risques.

To modify a risk:

  1. Sélectionnez un risque pour l'ouvrir.

  2. Utilisez le menu déroulant pour modifier le type de risque selon vos besoins.

  3. Sélectionnez les champs de texte Titre et Description pour les modifier.

  4. Sélectionnez Ajouter () pour ajouter des domaines, des références, des contrôles et des réponses d'audit au risque.

  5. Sélectionnez une conclusion dans le menu déroulant.

Si les signatures sont activées dans vos documents, vous pouvez utiliser des outils de révision pour examiner les modifications apportées aux risques. Une fois qu'un évaluateur a approuvé vos risques, les outils d'évaluation suivants sont disponibles :

  • Ajouté Ajouté - Marque les risques qui ont été ajoutés.

  • Modifié Modifié - Marque les risques qui ont été modifiés.

  • Supprimé Supprimé - Marque les risques qui ont été supprimés.
