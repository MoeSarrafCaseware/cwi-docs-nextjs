Rédiger du contenu en plusieurs langues

PBC Requests vous permet d'ajouter du contenu en français. Le contenu par défaut fourni avec les demandes PBC n'est disponible qu'en anglais. Toutefois, si votre entreprise a besoin d'un support en français, les auteurs de l'entreprise peuvent créer et maintenir le contenu en français dans le modèle de l'entreprise. Les membres dupersonnel peuvent rédiger directement dans un dossier d'engagement.

Avant d'ajouter du contenu, assurez-vous que le français est la langue active du contenu.

Pour modifier les paramètres linguistiques :

  1. Allez sur votre portail Caseware Cloud et réglez l'option Langue sur Français, puis connectez-vous.

    Si vous sélectionnez l' anglais, vous pouvez toujours rédiger du contenu en français, mais la langue de l'application s'affichera en anglais.

  2. Si vous êtes un auteur de cabinet, ouvrez votre modèle de cabinet. Si vous êtes membre du personnel, ouvrez votre dossier d'engagement.

  3. Dans la fenêtre contextuelle Propriétés de l'engagement , sélectionnez Français dans le menu déroulant Langue du contenu .

    Sélectionnez Enregistrer.

Remarque : Lorsque vous reportez un engagement, la langue sélectionnée dans le fichier de la période en cours est reportée dans le fichier de la période suivante.

Après avoir modifié vos paramètres linguistiques, vous pouvez ajouter des documents aux dossiers français par défaut, tels que les requêtes, les listes de contrôle, les lettres et les mémos , ainsi que les zones d'états financiers. Pour en savoir plus, voir Rédaction.

Remarque : Si vous créez un dossier dans une phase ou si vous ajoutez un document à une phase au lieu de le placer dans un dossier français par défaut, vous devez modifier les paramètres de visibilité.

Pour modifier les paramètres de visibilité :

  1. Survolez le nouveau dossier ou document, puis sélectionnez Autres actions () | Modifier.

  2. Sélectionnez Modifier la visibilité, puis sélectionnez Langue dans la liste déroulante Ajouter une condition .

  3. Modifiez la logique de la condition pour qu'elle soit affichée.

  4. Réglez le paramètre Langue du contenu sur Français.
