Localiser les documents du client

Une fois la requête terminée, les documents fournis par le client sont automatiquement stockés dans les dossiers prédéfinis désignés dans la page Documents , en fonction de vos réponses dans le document Optimiser .

Par exemple, si vous répondez Utiliser les modèles PBC et Compilation comme type d'engagement dans l' Optimiseur, les documents fournis par les clients sont automatiquement stockés dans le dossier prédéfini des documents PBC .

Remarque : Si vous répondez Construire ma propre requête dans l' Optimiseur, les documents fournis par les clients sont automatiquement stockés dans le dossier Requête - Documents PBC .
