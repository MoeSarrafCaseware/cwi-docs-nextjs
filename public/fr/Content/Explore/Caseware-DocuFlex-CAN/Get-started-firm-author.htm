Démarrer avec Caseware PBC Requests - Auteur de l'entreprise

Avant de commencer à utiliser Caseware PBC Requests, assurez-vous que l 'administrateur de votre société a effectué les tâches suivantes :

  • Vous confère le rôle d' administrateur des paramètres (ou un rôle similaire)

  • Permet au personnel d'accéder à Caseware Cloud. Pour en savoir plus, voir Créer du personnel et Intégrer des rôles.

Pour commencer, connectez-vous au site Cloud de votre entreprise. En tant qu' auteur du cabinet, vous pouvez personnaliser le modèle de votre cabinet pour l'adapter aux types de missions que vous devez réaliser. Vous pouvez consulter les rubriques d'aide relatives à toutes les actions que vous pouvez effectuer pour personnaliser le modèle dans l' index de création.

Pour en savoir plus, voir Gérer le modèle d'entreprise.
