Effectuer une exportation fiscale

Caseware Taxflow vous permet de créer un fichier d'exportation basé sur un mappage ou un regroupement de comptes par défaut pour l'importer dans votre logiciel fiscal. Les logiciels fiscaux suivants sont actuellement pris en charge :

  • CCH ProSystem fx® Tax

  • CCH Axcess™ Tax

  • Taxe Lacerte

Sur la page Documents , le dossier Exportation fiscale est disponible dans la phase Préparation et exportation fiscale.

Pour effectuer une exportation fiscale :

  1. Ouvrez le formulaire d' exportation 2-500 Taxe .

  2. Sélectionnez votre type d'entité.

  3. Examinez les tableaux fiscaux qui deviennent visibles.

    Remarque : Les formulaires fiscaux affichés dépendent du type d'entité sélectionné.

  4. Sélectionnez le logiciel d'imposition.

  5. Sélectionnez l' année fiscale.

    Remarque : La fonction d'exportation des taxes n'exporte que les soldes fiscaux définitifs de l'année en cours calculés dans les calendriers fiscaux.

  6. Sélectionnez le bouton Exporter .

Le fichier d'exportation fiscale est téléchargé sur votre appareil. Vous pouvez vérifier que les soldes des comptes ont été affectés aux bons endroits sur les formulaires fiscaux et ajuster les affectations dans le logiciel fiscal, si nécessaire.
