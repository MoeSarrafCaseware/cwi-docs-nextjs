Planifier la mission

Pour commencer à planifier votre engagement, ouvrez la liste de contrôle de l 'engagement 1-100 à partir de la page Documents.

Pour modifier le type d'entité :

  1. Sélectionnez le type d'entité actuel.

  2. Sur la page Détails de l'entité , sélectionnez l'icône Modifier ().

  3. Mettre à jour le pays d'enregistrement avec les États-Unis d'Amérique.

  4. Sélectionnez un type d'organisation pris en charge et cliquez sur Enregistrer.

Il se peut que vous deviez actualiser votre engagement pour voir vos modifications.

Précisez l'année fiscale applicable à votre engagement.

Enfin, sélectionnez le logiciel fiscal pour lequel vous souhaitez préparer le fichier d'exportation.

Après avoir sélectionné les informations requises, assurez-vous que le bouton Confirmer votre sélection est sélectionné pour que les informations fiscales applicables soient chargées dans l'engagement en fonction de l'année fiscale et du vendeur de taxes applicables.

Après avoir examiné les données de votre mission, vous pouvez passer à la requête 1-100Q " Données financières et informations générales sur l'entité" et "Réponses" pour demander des informations supplémentaires à votre client. Pour en savoir plus, voir Correspondance avec le client (Requêtes).

Si vous disposez de toutes les informations nécessaires, vous pouvez procéder à l'importation des données de votre client en utilisant diverses sources. Pour en savoir plus, voir Importer les données du client.
