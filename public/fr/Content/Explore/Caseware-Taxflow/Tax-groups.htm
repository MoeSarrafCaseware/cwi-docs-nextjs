Groupes fiscaux

Caseware Taxflow vous permet d' assigner des groupes de taxes à vos comptes de groupes financiers. En fonction de votre type d'organisation, des groupes de taxes mères sont disponibles par défaut. Vous pouvez ajouter, modifier ou supprimer des groupes d'imposition à l'aide du formulaire 1-570 C-Corp - Groupes d'imposition.

Remarque : Les groupes d'imposition parents ne peuvent pas être modifiés ou supprimés.

Ajouter un groupe d'imposition

Pour ajouter un groupe de taxes :

  1. Dans l'onglet Tableau du formulaire 1-570 C-Corp - Tax Groups , sélectionnez le tableau auquel vous souhaitez ajouter un groupe de taxe.

    Une liste de tous les groupes de taxes disponibles s'affiche.

  2. Sélectionnez l'icône de développement () du groupe d'imposition parent.

  3. Sélectionnez l'icône Ajouter () pour ajouter une nouvelle ligne d'entrée.

  4. Complétez les champs Numéro et Nom.

    Remarque : Le numéro de sous-groupe fiscal doit être compris entre 1 et 79.

Modifier un groupe d'imposition

Après avoir ajouté un groupe de taxe, vous pouvez modifier le numéro et le nom du sous-groupe.

Pour modifier un groupe de taxes :

  1. Dans l'onglet Tableau du formulaire 1-570 C-Corp - Tax Groups , sélectionnez le tableau du groupe d'imposition que vous souhaitez modifier.

  2. Sélectionnez l'icône de développement () du code fiscal parent.

  3. Localisez le sous-groupe que vous souhaitez modifier et mettez à jour les champs Numéro et Nom.

Supprimer un groupe d'imposition

Si vous créez un groupe de taxe qui n'est plus nécessaire, vous avez la possibilité de supprimer le sous-groupe.

Pour supprimer un groupe de taxes :

  1. Dans l'onglet Tableau du formulaire 1-570 C-Corp - Tax Groups , sélectionnez le tableau du groupe d'imposition que vous souhaitez supprimer.

  2. Sélectionnez l'icône de développement () du code fiscal parent.

  3. Sélectionnez l'icône Supprimer () pour supprimer le sous-groupe.
