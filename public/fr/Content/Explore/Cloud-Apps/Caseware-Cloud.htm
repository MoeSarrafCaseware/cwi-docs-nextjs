Qu'est-ce que Caseware Hybrid Cloud ?

Caseware Cloud est la plateforme sous-jacente qui vous permet de gérer l'environnement cloud de votre entreprise dans divers domaines, y compris les paramètres, le personnel, les contacts, les entités, les missions et les fichiers - le tout dans un emplacement en ligne pratique.

Visitez la page Quoi de neuf dans Caseware Cloud pour découvrir les dernières fonctionnalités de la plateforme Caseware Cloud.
