Pour commencer avec Caseware PBC

Avant de commencer à utiliser Caseware PBC Requests, assurez-vous que l 'administrateur de votre entreprise a rempli les conditions préalablesnécessaires.

  1. Create a Caseware PBC engagement file

  2. Remplir le document sur l'optimisation des 100 clients

  3. Examiner le document de demande de PBC généré

  4. Envoyer le document de demande de PBC aux clients

  5. Les clients répondent au document de demande du PBC

  6. Examiner les progrès et les réponses

  7. Collaborer avec les clients

  8. Rouvrir le document de demande de PBC

  9. Remplir le document de demande de PBC

  10. Localiser les documents du client

  11. Clôture et verrouillage

  12. Reporter la mission (facultatif)
