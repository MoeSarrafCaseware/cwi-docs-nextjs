Configuration requise

Nous prenons en charge l'utilisation des applications Cloud sur des machines dotées d'un minimum de 8 Go de RAM, mais nous recommandons d'utiliser des machines dotées d'un minimum de 16 Go de RAM pour améliorer les performances.

Navigateurs pris en charge

Vous pouvez vous connecter à un produit Caseware Cloud à partir de n'importe quel navigateur web utilisant TLS v1.2 ou plus, cependant nous supportons et recommandons l'utilisation d'une version stable (non-bêta) de l'un des produits suivants :

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

Nous recommandons une résolution d'écran minimale de 1280 x 800 pixels et une largeur de navigateur minimale de 1024 pixels.

Versions de MS Office prises en charge

Support des produits Caseware Cloud :

  • Microsoft Office 2016 ou version ultérieure

  • Microsoft Office 365 (version bureau)

Versions des documents de travail soutenues

Intégrez Working Papers avec Cloud pour accéder aux données de vos dossiers clients à l'aide d'un navigateur web. Nous prenons en charge les deux dernières versions des Working Papers ; cependant, l'intégration dans le nuage est disponible à partir des Working Papers 2017. Notez que certaines fonctionnalités peuvent nécessiter la version la plus récente.

Pour en savoir plus, voir la politique de support de Caseware.
