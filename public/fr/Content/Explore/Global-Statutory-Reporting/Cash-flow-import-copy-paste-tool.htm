Outil d'importation et de copier-coller des flux de trésorerie

L'outil d'importation et de copier-coller des flux de trésorerie vous permet de transférer facilement des soldes à partir de fichiers Excel ou CSV dans des tableaux dynamiques au sein des états financiers.

Remarque : L'outil d'importation / copier-coller des flux de trésorerie n'est pas disponible dans toutes les applications de Global Statutory Reporting.

Vous devez fournir les informations suivantes :

  1. Indiquez si les montants que vous souhaitez importer ou copier sont déjà arrondis ou non.

    Remarque : Les états financiers peuvent être arrondis en milliers, millions et milliards.

  2. Indiquez si les montants importés/copiés sont déjà arrondis conformément aux paramètres d'arrondi des états financiers.

    • Si les montants ne sont pas arrondis, le formulaire appliquera automatiquement les mêmes paramètres d'arrondi que ceux spécifiés dans les paramètres d'arrondi des états financiers.

    • Pour permettre l'arrondi dans les états financiers, voir Personnaliser l'arrondi dans les états financiers.

  3. Sélectionnez la méthode pour insérer les montants des flux de trésorerie dans ce formulaire : Importation CSV | Copier-coller.

    Remarque : L'outil d'importation des flux de trésorerie permet d'insérer les montants dans ce formulaire de deux manières :

    • Importation CVS : Importez les montants en utilisant le fichier modèle CSV prédéfini fourni.

      Remarque : L'importation CSV permet de charger les informations à partir d'un fichier modèle CSV dont le format et la structure sont prédéfinis. An example file can be found here: Fichier de démonstration du modèle d'importation CSV.

    • Copier-coller : Permet de copier et de coller directement les montants pour chacune des sections de flux de trésorerie applicables.

  4. Sélectionnez si vous souhaitez importer les montants de l'année précédente ou non.

  5. Examinez la section " Résumé" pour vous assurer que le tableau des flux de trésorerie est équilibré.
