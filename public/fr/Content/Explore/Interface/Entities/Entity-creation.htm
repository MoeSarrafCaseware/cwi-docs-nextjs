Création d'entités

Lorsque vous créez une entité, certains champs sont mis à jour dynamiquement en fonction de vos choix ou des options activées par votre distributeur. Ces champs sont décrits ci-dessous.

Type d'organisation

Les choix disponibles dans le menu déroulant Type d'organisation dépendent du pays d'enregistrement sélectionné.

Les pays sélectionnés disposent d'une liste personnalisée de types d'organisations disponibles, énumérés dans le tableau ci-dessous. Pour tous les autres pays, les types d'organisation suivants sont disponibles :

  • Société

  • Gouvernement

  • Sans but lucratif

  • Partenariat

  • Confiance

  • Coopérative

  • Fonds de pension

  • Individuel

  • Entreprise publique
