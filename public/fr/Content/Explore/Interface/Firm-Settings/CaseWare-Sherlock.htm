Traitement des messages

Le rapport Messages de traitement affiche des informations sur les nombreux processus d'extraction de données qui ont lieu chaque jour.

Pour afficher le rapport Traitement des messages :

  1. Assurez-vous que vous avez le rôle d' administrateur des paramètres ou des privilèges équivalents.

  2. Ouvrez le menu Cloud () et sélectionnez Paramètres.

  3. Sélectionnez Sherlock Starter | Traitement des messages.

  4. Le rapport Messages de traitement charge une liste des processus d'extraction de données historiques lors de sa première ouverture.

    • (Facultatif) Utilisez l'icône d'actualisation () pour recharger le rapport sur le traitement des messages.

    • (Facultatif) Utilisez la barre de recherche ou le filtre () pour limiter les résultats du rapport à ceux qui correspondent aux propriétés spécifiées.

      Propriété Description
      Icônes d'état (,) Affiche en un coup d'œil si l'extraction a été un succès () ou un échec ().
      Temps d'extraction L'heure à laquelle la tentative d'extraction a été effectuée.
      Entité Le nom de l'entité où se trouve le fichier.
      Fichier Le nom du fichier d'engagement.
      Dernière modification Date de la dernière modification du dossier d'engagement.
      Statut Indique si l'extraction a réussi ou échoué.

  5. Sélectionnez un processus d'extraction de données dans la liste pour obtenir plus de détails, y compris le message d'erreur complet pour les extractions qui ont échoué.

Cartographie de référence

Si votre entreprise dispose de modèles d'audit personnalisés, vous pouvez utiliser la cartographie Gold Standard pour faire correspondre les risques, les contrôles et les éléments à signaler dans vos modèles avec le "gold standard" de Caseware. Grâce à ce processus de mise en correspondance, Sherlock peut extraire les risques, les contrôles et les éléments à déclarer des dossiers de mission qui sont basés sur le modèle du cabinet.

Pour configurer le mappage Gold Standard :

  1. Assurez-vous que vous avez le rôle d' administrateur des paramètres ou des privilèges équivalents.

  2. Ouvrez le menu Cloud () et sélectionnez Paramètres.

  3. Sélectionnez Sherlock Starter | Gold Standard Mapping.

  4. Sélectionnez l'icône Ajouter une règle de mappage () sous le type de mappage applicable pour créer une nouvelle règle de mappage, ou l'icône Modifier () à côté d'une règle existante.

    Remarque : Vous ne pouvez pas modifier les règles de mappage avec le statut ACTIF .

  5. Sur la page Select Source, sélectionnez un fichier Working Papers parmi les 20 fichiers les plus récents. Lesinformations de la base de données CaseView sont chargées à partir du fichier sélectionné. Sélectionnez jusqu'à 3 groupes applicables et un formulaire.

    Le numéro figurant à côté de chaque identifiant indique le nombre de fois qu'il a été associé à une cible. Sélectionnez Ajouter pour mapper les ID.

  7. La page Résumé affiche une vue d'ensemble de la règle de mappage. Si la règle de mappage est complète, sélectionnez Terminer.

  8. La règle de mappage est enregistrée en tant que projet. Sélectionnez l'icône Modifier () pour continuer à modifier la règle, ou sélectionnez ACTIVER au-dessus du type de mappage pour activer la règle.

    Notes :

    • S'il existe deux règles à l'état de PROJET pour un type de mappage particulier, le fait de sélectionner ACTIVER activera les deux règles.

    • Une fois activée, vous ne pouvez plus modifier ou supprimer une règle de mappage.

    • Chaque point de contrôle, de risque ou de gestion peut avoir jusqu'à deux règles de correspondance.
