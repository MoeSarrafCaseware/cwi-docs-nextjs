Paramètres du cabinet – Intégration

Paramètres de l'API

Intégrer le nuage avec des applications externes à l'aide de clients API. Vous pouvez créer un maximum de 50 clients API pour votre entreprise. Nous recommandons d'intégrer une application par client API.

Les informations sur les clients provenant d'applications intégrées sont automatiquement mises à jour dans le nuage grâce à l'utilisation de jetons API. Les jetons API sont utilisés par les développeurs de l'entreprise pour authentifier le processus d'intégration.

Pour créer un nouveau client API :

  1. Assurez-vous d'avoir le rôle d' administrateur ou des autorisations équivalentes.

  2. Dans le menu Cloud (), sélectionnez Paramètres | Intégration | Paramètres API.

  3. Dans le champ Alias de client , saisissez un nom d'identification pour le client API, tel que le nom de l'application externe que vous intégrez.

  4. Sélectionnez Générer un client.

    Détails de l'affichage du nouveau client API.

  5. Veillez à copier l'identifiant et le secret du client dans un document texte pour une utilisation ultérieure. Le secret du client ne sera plus affiché. Sélectionnez Fermer.

  6. Le client API est créé.

    • Editer : Modifier l'alias du client.
    • Supprimer : Supprimer le client API.
    • Réinitialiser le secret du client : Générer un nouveau secret client. Nous recommandons de réinitialiser le secret à intervalles réguliers pour des raisons de sécurité. Notez que vous devrez mettre à jour toutes les applications externes qui utilisaient le secret précédent.

  7. Fournissez l'identifiant et le secret du client API aux développeurs de votre entreprise afin qu'ils puissent générer le jeton API.
