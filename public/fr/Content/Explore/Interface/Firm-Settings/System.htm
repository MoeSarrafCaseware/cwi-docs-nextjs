Gestion des erreurs

La page Gestion des erreurs vous permet de consulter un journal de toutes les erreurs du système et de les gérer comme il convient. Ces messages d'erreur nous fournissent des informations qui nous permettent d'améliorer l'expérience de votre organisation avec Cloud. Utilisez la barre de recherche pour filtrer les erreurs affichées.

Pour en savoir plus sur une erreur et accéder aux options de gestion des erreurs, sélectionnez l'erreur. Le volet des détails de l'erreur s'affiche à droite. Vous pouvez développer le message d'erreur complet pour afficher le texte intégral de l'erreur, ou sélectionner Autres actions () pour supprimer l'erreur ou la marquer comme lue ou non lue.

Statistiques du site

La page des statistiques du site indique la quantité d'espace que l'organisation utilise actuellement dans le nuage. Naviguez vers Paramètres | Système | Statistiques du site pour afficher cette page.

Nombre Statistiques Description
Utilisation actuelle

La quantité d'espace utilisée en termes de gigaoctets (Go) de stockage sur disque et de nombre de personnes actives.

  • Lestockage sur disque actuel est la somme de toutes les données de l'organisation stockées sur Caseware Cloud. Il s'agit notamment de

    • tous les fichiers téléchargés sur le serveur

    • tous les fichiers du Working Papers synchronisés avec le serveur

    • tous les objets du nuage créés par l'utilisateur (par exemple, les utilisateurs, les entités, les rôles)

    • toutes les données de la base de données Cloud

  • Lepersonnel actif actuel est le nombre total de personnes dans votre organisation qui utilisent Caseware Cloud. Ce nombre exclut les contacts et les membres du personnel inactifs.
Utilisation en période de pointe

La quantité d'espace la plus élevée (pic) utilisée pour le mois en cours en termes de gigaoctets (Go) de stockage sur disque et de nombre de personnes actives.

Ce sont les montants utilisés pour la facturation. Des informations sur les prix sont disponibles sur le site de Caseware Cloud.

Pour plus d'informations sur les limites d'utilisation, contactez votre représentant Caseware ou envoyez-nous un e-mail à solutions@caseware.com.
Liste des éléments du nuage

Le nombre d'éléments de contenu dans chacune des catégories suivantes :

  • Fichiers

  • Activités

  • Les personnes

  • Entités

  • Autres éléments (y compris les rôles personnalisés, les vues personnalisées et les journaux de base).

Facturation en nuage

Affiche le nombre total de licences achetées et restantes de votre organisation, y compris la date de renouvellement. Vous pouvez gérer vos licences via MyCaseware en cliquant sur le lien fourni. For more information, see Manage your firm account in MyCaseware.
