À propos des paramètres de notification

Dans Mes paramètres | Notifications | Notifications globales, vous pouvez configurer les événements qui entraîneront la réception d'un courriel de notification.

Nom Description
Options d'abonnement rapide Cochez la case pour recevoir des notifications sur toutes les actions effectuées par les contacts sur les entités, les activités ou les fichiers auxquels vous êtes abonné.
Paramètres d'abonnement à l'échelle du système

Cochez une case pour vous abonner à toutes les actions de ce type effectuées dans le Cloud.

Par exemple, si vous cochez la case Entités dans la colonne Modifier , vous recevrez un courriel de notification chaque fois qu'un utilisateur modifiera une entité.
Paramètres de l'abonnement individuel

Cochez une case pour vous abonner à toutes les actions de ce type effectuées sur les activités, les fichiers, les entités ou le contenu des entités.

Par exemple, si vous sélectionnez Activités sous Contenu des entités abonnées dans la colonne Créer , vous recevrez un courriel de notification chaque fois qu'une nouvelle activité sera créée dans une entité à laquelle vous êtes abonné.

Si vous cochez la case Entités de premierniveau , vous recevrez des courriels de notification pour tout le contenu du nuage associé aux entités auxquelles vous êtes abonné.
