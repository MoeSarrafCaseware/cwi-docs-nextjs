Paramètres VoIP

Dans Mes paramètres | Autres | Paramètres VoIP, vous pouvez activer l'intégration VoIP dans le cloud. Lorsque cette option est activée, tous les numéros de téléphone sont convertis en liens qui peuvent être composés par votre ordinateur. En cliquant sur ces liens, vous lancerez votre application VoIP par défaut. Une application VoIP doit être installée sur votre ordinateur pour que cette fonction fonctionne.

Les clients téléphoniques pris en charge sont les suivants :

  • Skype
  • Google Voice
  • Microsoft NetMeeting

Si aucun indicatif de pays n'est spécifié dans un numéro de téléphone, l'indicatif de pays par défaut est préfixé au numéro de téléphone. Demandez à votre administrateur Cloud si ce paramètre est spécifié dans les Paramètres.
