Propriétés du taux du personnel

Des taux peuvent être attribués aux membres du personnel sur la base des codes de travail définis pour l'entreprise. Cela permet à l'entreprise d'attribuer des taux individuels aux membres du personnel tout en maintenant un ensemble standard de codes de travail dans l'ensemble de l'entreprise.

Vous pouvez également définir des codes de travail à un taux zéro (0) pour enregistrer le temps sans frais.

Remarque : Les membres du personnel ne bénéficiant pas d'un taux standard peuvent toujours enregistrer des entrées à temps. Leur taux sera de zéro.

Les taux de personnel ont les propriétés suivantes :

Taux Propriétés Description
Taux standard et taux du code du travail Statut

L'état actuel du taux standard ou du taux de code de travail :

  • Actif ()

    Le taux actuel pour ce membre du personnel. Voir la date d'entrée en vigueur ci-dessous.

  • Historique ()

    Le taux pour ce membre du personnel au cours d'une période antérieure.

    Note:;Un taux de période antérieure ne peut être ni modifié ni supprimé.

  • Futur ()

    Un taux futur pour ce membre du personnel. Un taux est un taux futur si la date d'entrée en vigueur est postérieure à la période en cours.
Tarif normal Date d'entrée en vigueur

Date d'entrée en vigueur du taux standard.

La date d'entrée en vigueur ne peut être antérieure à la date de début de la période.
Taux

Le taux standard (par défaut) pour ce membre du personnel. Le taux standard est appliqué lorsqu'il n'y a pas de taux correspondant défini pour le code de travail spécifique.

Par exemple, un membre du personnel saisit du temps en utilisant le code de travail 100. Si aucun taux correspondant n'est défini pour ce code de travail, le taux standard est appliqué pour cette saisie de temps.
Code de travail Taux Code du travail Code de travail prédéfini pour les saisies de temps. Pour plus d'informations, voir Création de nouveaux codes de travail .
Date d'entrée en vigueur

Date d'entrée en vigueur du taux pour le code de travail spécifié.

La date d'entrée en vigueur ne peut être antérieure à la date de début de la période.
Taux

Le taux pour le code de travail spécifié pour ce membre du personnel. S'il est défini, ce taux est appliqué lors de la saisie du temps à l'aide du code de travail spécifié.

Par exemple, un membre du personnel saisit du temps en utilisant le code de travail 100. Un taux de 100,00 $ est défini pour ce code de travail. Ce taux est ensuite appliqué pour les entrées de temps utilisant le code de travail 100.
