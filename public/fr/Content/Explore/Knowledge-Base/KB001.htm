Désactivation de l'accès à Caseware Cloud pour les navigateurs anciens et non supportés

Publié : jeudi 18 juin 2020

Peu après la sortie de Caseware Cloud 31, Caseware désactivera l'accès à Caseware Cloud pour les navigateurs web plus anciens et non supportés. Ces navigateurs plus anciens et non pris en charge utilisent des protocoles de cryptage obsolètes - TLS version 1.0 et v1.1.

Le Conseil des normes de sécurité PCI a suggéré à toutes les organisations qui suivent ses meilleures pratiques de passer de SSL/TLS à TLS v1.2 ou plus. Ce changement est appliqué dans l'ensemble du secteur afin de maintenir des connexions sécurisées qui cryptent et protègent vos données sensibles contre les violations malveillantes. Caseware suivra cette recommandation pour toutes ses connexions entrantes.

Qu'est-ce que TLS ?

TLS signifie "Transport Layer Security" (sécurité de la couche transport) Il s'agit d'un protocole qui assure la confidentialité et l'intégrité des données entre deux applications communicantes. Il s'agit du protocole de sécurité le plus largement déployé à l'heure actuelle. Il est utilisé pour les navigateurs web et d'autres applications qui nécessitent un échange sécurisé de données sur un réseau. TLS garantit qu'une connexion à un point d'accès distant est le point d'accès prévu grâce au cryptage et à la vérification de l'identité du point d'accès. Les versions de TLS, à ce jour, sont TLS 1.0, 1.1, 1.2 et 1.3.

Qui est concerné ?

Les clients de Caseware Cloud (personnel et contacts) et les clients de Hybrid Cloud utilisant des navigateurs web plus anciens et non supportés pour accéder aux produits Caseware seront impactés. La liste complète des navigateurs web concernés est détaillée dans la section Compatibilité des navigateurs Internet ci-dessous.

Pour vérifier si vous utilisez un navigateur non pris en charge :

  1. Utilisez cet outil pour vérifier les capacités SSL/TLS devotre navigateur.
  2. Si le navigateur doit être mis à jour (c'est-à-dire s'il n'affiche pas 1.2 ou une version plus récente), veuillez consulter ce site web pour savoir comment activer TLS 1.2 ou une version plus récente : Activation de TLS 1.1 et TLS 1.2 sur les navigateurs web.

Expérience de l'utilisateur en l'absence d'action

Les utilisateurs ne pourront pas se connecter à Caseware Cloud si leur navigateur ou ses paramètres ne sont pas mis à jour conformément aux directives de compatibilité mentionnées ci-dessous.

  1. Utilisateurs de Caseware Cloud

    Les utilisateurs qui n'utilisent pas un navigateur compatible recevront un message d'erreur sur la page de connexion de Caseware Cloud et ne pourront pas se connecter au serveur. Une fois leur navigateur ou ses paramètres mis à jour, les utilisateurs peuvent actualiser la page pour accéder à l'écran de connexion.

  2. Working Papers Utilisateurs avec l'intégration Cloud pour SmartSync

    Les utilisateurs qui ne disposent pas d'un navigateur compatible peuvent être confrontés à tout ou partie des éléments suivants :

    Working Papers 2018 ou supérieur

    • Vous recevrez un message d'erreur lorsque vous vous connecterez à l'écran de connexion de Caseware Cloud à partir de Working Papers.
    • Ne sera pas en mesure de voir ou d'accéder à des fichiers à partir de l'onglet SmartSync Server sous Fichier | Ouvrir.
    • Ne pourront pas se connecter au serveur (c'est-à-dire aller en ligne) pour synchroniser les modifications de leur copie de synchronisation locale. Une bulle de notification indique que le fichier est hors ligne dans le coin inférieur droit.

    Documents de travail 2017 ou moins

    • Ne sera pas en mesure de voir ou d'accéder à des fichiers à partir de l'onglet SmartSync Server sous Fichier | Ouvrir.
    • Ne pourront pas se connecter au serveur (c'est-à-dire aller en ligne) pour synchroniser les modifications de leur copie de synchronisation locale. Une bulle de notification indique que le fichier est hors ligne dans le coin inférieur droit.

Une fois le navigateur ou les paramètres du navigateur mis à jour, les fichiers Working Papers reviendront à leur état en ligne lorsqu'ils seront connectés à l'internet. Il se peut que vous deviez vous connecter à Caseware Cloud si les paramètres de votre entreprise ont été configurés pour exiger une connexion quotidienne à Caseware Cloud.

Compatibilité avec les navigateurs Internet

Selon les exigences denotre système, Caseware Cloud supporte les navigateurs suivants :

  • Google Chrome (recommandé)
  • Mozilla Firefox
  • Microsoft Edge

Vérifiez si votre navigateur prend en charge TLS 1.2 ou une version supérieure à l'aide de cet outil. Si le navigateur doit être mis à jour (c'est-à-dire s'il n'affiche pas 1.2 ou une version plus récente), veuillez consulter ce site web pour savoir comment activer TLS 1.2 ou une version plus récente.
