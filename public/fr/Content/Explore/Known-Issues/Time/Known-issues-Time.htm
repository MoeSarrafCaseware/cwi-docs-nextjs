Problèmes connus dans CasewareCloud Time

Les problèmes connus suivants affectent la version actuelle de CasewareCloud Time.

Erreurs avec les utilisateurs de Time supprimés

Si vous êtes connecté et que votre accès à Time a été révoqué par un administrateur, vous recevrez plusieurs messages d'erreur lorsque vous tenterez d'accéder à Time.

CORE-46707

Les factures exportées vers QuickBooks Online peuvent être affectées à des taxes inactives

Au cours du processus d'exportation, les factures peuvent être incorrectement affectées à des taxes inactives. Pour contourner ce problème, vous pouvez sélectionner manuellement une nouvelle taxe dans QuickBooks Online pour chaque facture concernée.

Erreur de récupération des données après un changement de compte Xero

Une erreur de récupération des données se produit lorsque vous dissociez un compte Xero de Time et que vous le reliez ensuite à un autre compte. L'actualisation de la page ne résout pas l'erreur. Pour contourner ce problème, vous pouvez gérer les organisations directement dans votre compte Xero au lieu de créer un nouveau compte. La possibilité de changer le compte Xero que vous avez lié à Time n'est actuellement pas disponible.

Impossible d'exporter de nouvelles factures vers le compte Xero lié

La tentative d'exportation d'une nouvelle facture vers Xero échouera si le compte a été lié avant août 2021.

Solution : Reliez votre compte Xero. Dans Xero, sélectionnez Paramètres | Applications connectées. À côté de l'application Caseware ImportHub, sélectionnez Déconnecter. Dans Caseware Cloud, sélectionnez le menu Cloud () | Paramètres | Temps | Intégration des logiciels de comptabilité, puis sélectionnez Unlink Xero. Ensuite, sélectionnez Link to Xero et connectez-vous avec votre compte.

erreur "User Name must be unique" lors de la création de nouveaux contacts

Si l'adresse électronique d'un contact existant dans un fichier Time Desktop intégré au cloud est manquante, l'erreur "Le nom d'utilisateur doit être unique" s'affiche lors de la création d'un nouveau contact. Pour contourner temporairement ce problème, vous pouvez saisir des adresses électroniques "fictives".
