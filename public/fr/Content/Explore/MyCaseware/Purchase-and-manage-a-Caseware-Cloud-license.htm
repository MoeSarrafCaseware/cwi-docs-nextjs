Acheter et gérer une licence Caseware Cloud

Pour acheter une licence Caseware Cloud, visitez https://www.caseware.com/contact/ et remplissez le formulaire d'information. Vous serez contacté par un représentant de Caseware pour finaliser votre achat.

Achat de licences supplémentaires

Additional Caseware Cloud app licenses can be purchased directly using a self-checkout system in MyCaseware. Pour chaque nouvelle licence achetée, vous pourrez créer un compte personnel supplémentaire. Pour chaque licence achetée, un utilisateur supplémentaire pourra accéder à Cloud Connector. Pour plus d'informations sur les prix, connectez-vous à my.caseware.com et sélectionnez l'onglet Catalogue.

Pour effectuer un achat sur MyCaseware :

  1. Assurez-vous d'avoir le rôle d'administrateur ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Groupes.

    Les licences que vous pouvez acheter s'affichent.

  3. Ajoutez des achats à votre panier en cliquant sur la licence cloud que vous souhaitez acheter. Choisissez la quantité, puis sélectionnez Ajouter au panier.

  4. Pour ajouter d'autres achats à votre panier, sélectionnez Continuer les achatsou Passer à la caisse pour terminer le processus de paiement.

Votre achat est terminé. Les licences mises à jour dont vous disposez s'affichent sous l'onglet Licences .

renouveler une licence,

Caseware ne stocke pas les informations de paiement pour des raisons de sécurité. Pour renouveler votre licence, contactez notre équipe de vente.

Retards de paiement

Si le paiement n'est pas reçu à la date indiquée, votre compte sera verrouillé et toutes les données ou tous les fichiers associés à ce compte seront inaccessibles. Une fois le paiement reçu, le compte sera rétabli sans perte de données.
