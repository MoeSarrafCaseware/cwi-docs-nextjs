renouveler une licence,

Caseware ne stocke pas les informations de paiement pour des raisons de sécurité. Pour renouveler votre licence, contactez notre équipe d'assistance.

Retards de paiement

Si le paiement n'est pas reçu à la date indiquée, votre compte sera verrouillé et toutes les données ou tous les fichiers associés à ce compte seront inaccessibles. Une fois le paiement reçu, le compte sera rétabli sans perte de données.

View Authorization code

Les licences Working Papers sont activées à l'aide de votre code d'autorisationpersonnalisé. Pour afficher votre code d'autorisation dans MyCaseware, sélectionnez Licences | Desktop Licenses.

Votre code s'affiche dans la colonne Code d'autorisation .

Voir les licences et les téléchargements de produits bureautiques Caseware

Dans MyCaseware, les utilisateurs peuvent accéder et gérer les licences et les téléchargements pour les produits Caseware Desktop.

Pour afficher les licences, sélectionnez Licences | Licences de bureau. Sur cette page, vous pouvez voir toutes les licences Desktop associées à votre compte. Cliquez sur le produit figurant dans la colonne Nom de la licence de bureau pour afficher les téléchargements disponibles pour votre licence.

Télécharger le logiciel

Sélectionnez Télécharger le logiciel pour trouver les dernières versions du logiciel Caseware Desktop disponibles pour le téléchargement.

Télécharger DataStore

Sélectionnez Télécharger le DataStore pour télécharger l'outil d'administration du Data Store pour les Working Papers. Cette option n'est disponible que pour les utilisateurs disposant d'une licence Working Papers.

Consultation du contrat de licence

Sélectionnez Rapport de licence pour consulter vos enregistrements de licence Working Papers à des fins de conformité et de suivi.
