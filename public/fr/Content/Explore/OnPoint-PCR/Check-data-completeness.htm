Vérifier l'exhaustivité des données

Si des transactions ont été importées dans votre fichier de mission, utilisez le document 2-550 sur l'exhaustivité des données pour vérifier si les soldes des comptes calculés à partir de votre balance de vérification correspondent aux soldes des comptes calculés à partir des transactions d'une mission. Les divergences peuvent indiquer que vous avez reçu ou téléchargé un ensemble de données incomplet et que vous devrez peut-être réimporter les données.

Pour plus d'informations, voir Vérifier l'exhaustivité des données.
