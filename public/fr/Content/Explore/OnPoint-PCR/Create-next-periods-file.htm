Créer le dossier de la période suivante

Vous pouvez reporter une mission afin de réutiliser les informations et les réponses de l'année précédente qui n'ont pas changé. Vous pouvez également choisir d'accepter toutes les modifications de l'année dernière ou de ne travailler que sur les éléments individuels qui ont été modifiés.

Pour en savoir plus sur ce qui est reporté et ce qui ne l'est pas, voir Report d'un engagement.

Remarque : Les réponses fournies par les clients dans les documents de demande PBC ne sont pas reportées.

Pour reporter la mission :

  1. Dans le menu Cloud (), sélectionnez OnPoint PCR pour ouvrir la vue de l'engagement.

  2. Sélectionnez votre engagement pour ouvrir le panneau latéral de droite, puis sélectionnez Créer le fichier de la période suivante.

  3. Spécifiez vos préférences pour le nouvel engagement et sélectionnez Enregistrer.

  4. Ouvrez le nouvel engagement (créé à partir du report).

    Des options de report vous sont proposées :

    • Transférer en une seule fois tous les documents et fichiers de l'engagement initial (report en bloc).

    • Annulez le report en bloc et sélectionnez individuellement les documents et fichiers à transférer de l'engagement initial vers le nouvel engagement.

Si vous ouvrez le document de configuration de l 'engagement 1-100 , les options de report sont également disponibles afin que vous puissiez sélectionner les réponses à transférer dans le nouvel engagement.

Si vous acceptez les suggestions de report, les réponses du dossier de l'année dernière sont reportées dans le document de configuration de l'engagement 1-100 . Lorsque vous passez au point 1-125 Acceptation de la qualité de la mission de revue et conclusion, les suggestions de report du dossier de l'année précédente vous sont à nouveau présentées - et ainsi de suite pour chaque document de mission.
