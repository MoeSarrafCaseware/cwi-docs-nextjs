Exporter toutes les données

Pour exporter les données d'un dialogue ou d'une page, cliquez sur l'icône Exporter toutes les données ().

Cette action déclenche le téléchargement d'un nouveau fichier Excel dans votre dossier Téléchargements, à partir duquel vous pouvez utiliser le fichier exporté selon vos besoins.

Notez que ce fichier exporté n'est pas formaté pour être réimporté dans SQM. Pour travailler sur un modèle Excel dont le contenu peut être importé dans SQM, naviguez vers Paramètres | Importation.
