Importer des éléments de réseau

Le paramètre d' importation du réseau permet aux utilisateurs d'importer des éléments du réseau, tels que les objectifs prescrits, les risques, les réponses aux politiques et procédures et les activités de surveillance, dans le système de gestion de la qualité d'une entreprise sur SQM.

Importer

Dans l'option de menu de la page Importation , l'entreprise peut importer les éléments du réseau dans le système de chaque entreprise membre du réseau sur SQM.

Ces éléments sont importés à l'aide d'un fichier au format .sqmex.

Un fichier .sqmex est créé lorsque l'entreprise du réseau exporte les bases de données d'articles du réseau à partir de son propre système de gestion de la qualité sur SQM.

Il s'agit notamment de bases de données sur les objectifs, les risques, les réponses politiques, les réponses procédurales et les activités de contrôle prescrites par le réseau.

To import:

  1. Sélectionnez Paramètres dans la barre de menu, puis SQM | Importer.

  2. Sélectionnez le(s) fichier(s) .sqmex à importer et glissez-déposez le(s) fichier(s) dans le bloc disponible.

  3. Une fois que tous les fichiers ont été déposés dans le bloc, cliquez sur Importer.

  4. Une boîte de dialogue apparaît. Pour exécuter l'importation, cliquez sur Oui.

Une fois le(s) fichier(s) importé(s), un message de notification s'affiche pour confirmer que les informations ont été importées avec succès.

Les fichiers sélectionnés ne sont plus visibles sur cet écran.

Le contenu a été importé dans les différents tableaux des articles.

Télécharger le modèle d'enregistrement des éléments du réseau

L'entreprise peut également enregistrer les éléments prescrits pour son réseau en les inscrivant sur le modèle Excel téléchargeable, en enregistrant les informations et en générant un fichier .sqmex pour l'importation.

Pour télécharger le fichier .sqmex, cliquez sur le lien hypertexte dans Paramètres sous SQM | Importer.

Enregistrer manuellement des enregistrements individuels

L'entreprise peut enregistrer manuellement chaque élément individuel sur la page correspondante.

Par exemple, pour enregistrer une réponse à une politique prescrite par le réseau, cliquez sur Paramètres et sélectionnez SQM | Réponses aux politiques.

To add a new record to the database:

  1. Sélectionnez Ajouter.

  2. Complétez le dialogue de réponse à la politique et sélectionnez SAUVEGARDER.

La réponse politique ajoutée apparaît dans le tableau.
