Ajouter un nouvel élément

  1. Cliquez sur l'icône de création rapide ().

  2. Choisissez l'élément à ajouter dans le menu déroulant.

    Notez que les éléments affichés dans le menu déroulant sont adaptés à votre rôle et au module dans lequel vous êtes actuellement engagé. Une boîte de dialogue s'ouvrira pour l'élément nouvellement créé.

  3. Complétez le dialogue.

  4. Pour enregistrer l'élément, cliquez sur Enregistrer.
