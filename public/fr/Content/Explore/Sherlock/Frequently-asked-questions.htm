Foire aux questions (FAQ)

Vous trouverez ci-dessous quelques-unes des questions les plus fréquemment posées (FAQ) sur Caseware Sherlock.

Configurer

  • Comment puis-je voir mes données d'engagement dans Caseware Sherlock ?

    Vous pouvez visualiser les données d'engagement dans Caseware Sherlock si vous remplissez les conditions suivantes :

    • Vous êtes un utilisateur actif de Hybrid Cloud avec une licence Sherlock Builder ou Sherlock Reader

    • Le dossier d'engagement a été publié dans Caseware Cloud

    • Vous disposez du droit de lecture dans l'entité où le fichier d'engagement est stocké

Données

  • Comment Caseware Sherlock sait-il quand extraire les données de mes fichiers d'engagement ?

    Une demande d'extraction de données est déclenchée automatiquement par les trois scénarios suivants :

    • Un dossier d'engagement est publié depuis votre application locale Working Papers vers Caseware Cloud avec des modifications supérieures au seuil

    • Un fichier d'engagement est téléchargé dans Caseware Cloud à l'aide de l'interface Cloud

    • Les fichiers d'engagement sont téléchargés vers Caseware Cloud en vrac en utilisant l'outil Migration Manager

    Une fois que la demande d'extraction de données a été déclenchée, les données sont extraites au cours du cycle programmé suivant.

  • Pourquoi ne vois-je pas mes modifications de fiançailles récentes dans Caseware Sherlock ?

    Il se peut que les données d'engagement existantes n'affichent pas les modifications récentes dans l'une ou l'autre des circonstances suivantes :

    • Le processus d'extraction des données n'a pas pu être achevé dans le cycle prévu en raison du grand nombre de demandes. L'extraction est automatiquement reportée au cycle du jour suivant.

    • Vous ne disposez pas du droit de lecture dans l'entité où le fichier d'engagement est stocké

Visualisations

  • Pourquoi ne puis-je pas voir certaines visualisations dans Sherlock Reader ?

    Si les visualisations ont été créées par un utilisateur de Sherlock Builder dans votre organisation, il doit les partager avec vous.

    Si les visualisations proviennent de Caseware ou d'un distributeur, l'administrateur de votre entreprise doit vous attribuer des licences pour les visualisations.

Sécurité

Remarque : Cette information concerne toutes les données de Caseware Cloud, y compris, mais sans s'y limiter, Caseware Sherlock.

  • Où sont stockées mes données ?

  • Mes données sont-elles cryptées ?

    Le stockage des données (data-at-rest) est crypté en utilisant le même niveau de cryptage que les services bancaires en ligne.

  • Sécurité et politique

    Caseware Cloud utilise une plateforme avancée de protection des points d'accès pour détecter et se défendre contre les attaques connues et inconnues.

    Caseware Cloud s'appuie également sur les politiques de sécurité d'Amazon Web Services et leurs accréditations qui sont un élément clé de la protection de vos informations sensibles.

    Pour les clients canadiens : Le Commissariat à la protection de la vie privée du Canada a déclaré que les accords d'externalisation n'enfreignent pas la loi sur la protection des renseignements personnels et les documents électroniques lorsque les dispositions relatives à la confidentialité et à la sécurité contenues dans l'accord d'externalisation satisfont à l'exigence de fournir un niveau comparable de protection des renseignements personnels.
