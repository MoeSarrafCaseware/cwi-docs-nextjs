Afficher une visualisation

Utilisez l'application Sherlock Reader pour visualiser et personnaliser les visualisations disponibles sur votre compte. L'icône précédant le nom de la visualisation indique si la visualisation est achetée () ou partagée ().

Notes :

  • Vous ne pouvez visualiser que les visualisations pour lesquelles votre entreprise a acheté des licences via MyCaseware, ou les visualisations qui ont été partagées avec vous par d'autres membres du personnel via Sherlock Builder.

  • Vous ne pouvez visualiser que les données des entités pour lesquelles vous disposez du droit Permis de visualisationPermis devisualisation .

Afficher une visualisation

  1. Assurez-vous que l'administrateur de votre entreprise vous a attribué des licencesvous a attribué des licences pour l'application Sherlock Reader et toutes les visualisations applicables.

  2. Dans le menu Cloud (), sélectionnez Sherlock Reader.

    Une liste des visualisations disponibles s'affiche.

  3. Dans la liste, sélectionnez la visualisation que vous souhaitez visualiser. Pour plus d'informations, voir Affichage de données visuelles dans Amazon QuickSight.

    Remarque : La fonction de recherche n'est pas disponible dans Sherlock Reader.

    La visualisation s'ouvre.

  4. Si nécessaire, appliquez des filtres ou développez un widget pour personnaliser l'apparence de la visualisation pour l'impressionl'impression.
