Remplir la demande de PBC

Une fois que vous avez accepté toutes les réponses, vous pouvez marquer le document comme complet pour clore l'interaction.

Pour marquer une requête comme complète, sélectionnez l'option Complèteen haut de la page après avoir accepté toutes les réponses. En outre, lorsque toutes les questions sont acceptées, un message de notification s'affiche avec le bouton MARK COMPLETE afin que le contact puisse facilement passer à l'étape suivante du flux de travail.
