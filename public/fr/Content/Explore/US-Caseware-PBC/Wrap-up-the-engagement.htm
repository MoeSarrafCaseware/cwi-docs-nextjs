Conclure la mission

Maintenant que vous avez rassemblé les informations et les documents dont vous avez besoin, vous pouvez conclure l'engagement.

Pour conclure :

  • Utilisez le dossier Documents du dossier permanent dans la page Documents pour télécharger ces documents permanents pertinents dans Collaborate.

    • Les documents permanents sont des métadonnées relatives à votre client qui ne changent pas de manière significative d'une année à l'autre, comme le plan d'affaires, l'organigramme et les statuts de l'entreprise.

  • Téléchargez une copie de référence de l'engagement à partager avec les clients ou à conserver pour vos propres dossiers. La copie de référence est un fichier .zip contenant des copies PDF de tous les documents de votre engagement. Pour télécharger une copie de référence, sélectionnez Information () | Télécharger la copie de référence.
