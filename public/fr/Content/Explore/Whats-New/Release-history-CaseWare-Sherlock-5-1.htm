Nouveautés - Caseware Sherlock 5.1

Découvrez les nouveautés de Caseware Sherlock 5.1.

Hybride sur site

Libérez la puissance de Sherlock practice insights avec Hybrid on Prem, un environnement idéal si vous utilisez des intégrations personnalisées avec Working Papers et souhaitez tirer parti des capacités de Caseware Cloud tout en limitant la gestion du changement.

Pour plus d'informations, contactez votre représentant local.

Ensemble de données sur les propriétés d'engagement

Nous avons ajouté le jeu de données engagementproperties aux applications Sherlock Builder et Sherlock Reader. Les utilisateurs peuvent extraire des données de n'importe quel champ de la boîte de dialogue Propriétés de la mission d'un fichier Working Papers en utilisant le code CLPassocié à chaque champ.

Processus d'extraction de l'engagement historique

Nous avons rationalisé le processus d'extraction des données historiques sur l'engagement. Le processus d'extraction est désormais automatiquement déclenché pour les nouvelles entreprises Sherlock. Le personnel de l'entreprise aura accès à une interface contenant les détails de l'extraction, y compris l'état d'avancement et la durée estimée de l'opération.
