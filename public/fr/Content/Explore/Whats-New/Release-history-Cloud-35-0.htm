Nouveautés - Cloud 35.0

Découvrez les nouveautés de Caseware Sherlock 35.0.

Période de renouvellement de la licence

Les administrateurs peuvent désormais consulter la période de renouvellement de toutes les licences actives de leur entreprise sur la page Paramètres de l'entreprise | Système | Facturation cloud.

Licences en lecture seule

Le personnel peut désormais se voir attribuer des licences en lecture seule pour les applications de Cloud Engagement. Les licences en lecture seule permettent aux utilisateurs de visualiser le contenu de l'engagement dans une application, sans consommer de licence d'application payante. Pour attribuer à un utilisateur une licence en lecture seule, accédez à son profil d'utilisateur et, dans l'onglet Applications , sélectionnez Lecture seule pour l'application concernée.

For more information, see Manage access for Caseware Cloud apps.

Dépréciation de la fonction Replace

Sur la base des commentaires des clients et d'une utilisation globalement faible au cours de l'année écoulée, nous avons décidé de supprimer la fonction Remplacer pour les fichiers des documents de travail. Cette dépréciation n'aura pas d'impact sur la fonction Remplacer la copie du serveur dans les documents de travail, qui est toujours disponible. À l'avenir, vous pourrez remplacer un fichier Working Papers en supprimant simplement la copie du nuage, puis en téléchargeant la nouvelle copie à partir de votre ordinateur local.
