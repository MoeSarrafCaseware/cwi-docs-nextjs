Quoi de neuf - Caseware ReviewComp Fall 2020 (Version 1.1680.0)

Voici une liste des nouveautés pour la version automne 2020 de Caseware ReviewComp.

Lettre d'accompagnement dans les missions fiscales

Une lettre d'accompagnement est désormais disponible dans le dossier de communication du client pour les missions d'impôt sur les sociétés. La lettre détaille les principaux résultats, tels que les impôts à payer, les montants des versements et les dates clés.

Amélioration de la navigation dans la liste de contrôle de l'Optimiseur fiscal

Pour éviter de faire défiler un long document, des onglets sont désormais disponibles dans la liste de contrôle de l'optimiseur fiscal (TOC) afin de vous permettre de naviguer facilement dans les différentes sections.

Filigrane dans les paramètres d'impression

Vous pouvez désormais ajouter un filigrane dans les états financiers. L'option Appliquer un filigrane est désormais disponible dans les paramètres d'impression du document. Une fois cette option activée, vous pouvez ajouter le contenu du filigrane dans le champ de texte associé. Pour en savoir plus, voir Imprimer les états financiers.

Signer les ajustements et la balance des comptes

Vous pouvez maintenant approuver la balance de vérification et les écritures de régularisation. Pour en savoir plus, voir Approuver la balance et les régularisations.

Sous-dossiers dans la page Documents

Pour améliorer la présentation et l'accès aux documents, des sous-dossiers sont désormais disponibles dans la page Documents afin de regrouper davantage d'éléments connexes. Pour en savoir plus, voir Gérer les documents d'engagement.
