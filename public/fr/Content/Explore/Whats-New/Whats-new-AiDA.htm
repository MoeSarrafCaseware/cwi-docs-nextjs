Nouveautés dans AiDA (v2.0)

Bienvenue à la version 2.0 de Caseware AiDA ! Cette version introduit les bases de connaissances comme source d'information pour AiDA, le résumé des PDF numérisés, les références au matériel source ainsi qu'un choix de modèles d'IA.

Basculer entre les modèles d'IA

Vous avez la possibilité de choisir le modèle d'IA qui correspond le mieux à vos besoins :

  • AiDA Fast : Basé sur le modèle Claude 3 Haiku, le modèle AiDA Fast est idéal pour les tâches rapides et efficaces. Il fournit des réponses rapides avec moins de puissance de traitement.

  • AiDA Enhanced : Basé sur le modèle Claude 3 Sonnet, le modèle AiDA Enhanced excelle dans les tâches complexes nécessitant profondeur et précision, comme l'analyse détaillée et le résumé de contenus hautement techniques.

Pour basculer entre les modèles d'IA, descendez au bas de la fenêtre de chat et cliquez sur l'icône du modèle d'IA (). Sélectionnez le modèle d'IA de votre choix. Saisissez vos questions et cliquez sur Envoyer ().

Bases de connaissances

Lorsque vous interagissez avec AiDA, vous pouvez sélectionner une base de connaissances et poser à AiDA des questions relatives à ses renseignements. Par défaut, une base de connaissances appelée Aide générale Caseware est accessible, quelle que soit l'application Caseware Cloud que vous utilisez. En outre, une base de connaissances propre à l'application, portant le nom de l'application en cours d'utilisation, est également accessible pour certaines applications disposant d'une base de connaissances Caseware. La base de connaissances Aide générale Caseware tire ses renseignements du centre d'aide Caseware Cloud à docs.caseware.com.

Pour utiliser cette fonction, descendez au bas de la fenêtre de chat et cliquez sur l'icône de la base de connaissances (). Sélectionnez la base de connaissances qui vous intéresse. Dans la zone de texte, tapez votre question relative à l'application ou à Caseware Cloud et cliquez sur Envoyer (). AiDA recherche les renseignements pertinents dans la base de connaissances sélectionnée, les traite et génère une réponse adaptée à votre requête.

Résumer les PDF numérisés

Vous pouvez également demander à AiDA de résumer les PDF numérisés, également appelés PDF à base d'images. Ces PDF sont des PDF dont le texte n'est pas modifiable et fait partie de la représentation visuelle du document. AiDA peut résumer les PDF numérisés en convertissant les images en texte.

Résumer les PDF numérisés de la même manière que n'importe quel autre document de votre mission.

Voir les références des documents sources

Dans les réponses d'AiDA à vos interactions, AiDA fournit maintenant des références que vous pouvez sélectionner pour voir les sources utilisées par AiDA pour générer la réponse. Les références ne sont pas accessibles si vous avez résumé un PDF numérisé.
