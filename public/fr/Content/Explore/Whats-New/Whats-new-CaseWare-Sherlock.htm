Nouveautés - Caseware Sherlock 6.1

Découvrez les nouveautés de Caseware Sherlock 6.1.

ensembles de données documentv2 et historyv2

Nous avons introduit deux nouveaux ensembles de données - documentv2 et historyv2. Ces ensembles de données fonctionnent de la même manière que les ensembles de données documentaires et historiques existants, mais l'identité des utilisateurs est décryptée. Cela signifie que les visualisations qui affichaient auparavant un GUID crypté pour masquer les noms des utilisateurs (avec document ou historique) peuvent désormais afficher leurs noms complets (avec documentv2 ou historiquev2).

Données de sortie des engagements dans le nuage

Sherlock capture désormais les données des engagements Cloud et les intègre dans les mêmes ensembles de données que les documents de travail. Cela vous permet d'interroger les données relatives aux problèmes dans toutes les missions, quelle que soit leur plateforme d'origine.

Métadonnées supplémentaires provenant d'engagements dans le nuage

Sherlock capture désormais des métadonnées supplémentaires à partir des engagements Cloud, telles que les informations de verrouillage, les dates d'approbation, les valeurs de matérialité et plus encore.
