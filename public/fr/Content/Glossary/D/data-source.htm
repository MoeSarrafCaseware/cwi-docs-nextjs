source de données

L'origine d'un fichier de données ou de données provenant d'un service en ligne, qui est importé dans Caseware Cloud pour faire partie d'un engagement. Le fichier de données ou l'importation de données en ligne est associé à une entité Caseware. Par exemple, un ensemble de données du grand livre comprend plusieurs fichiers de données de transaction. Chacun de ces fichiers est une source de données pour l'ensemble de données du grand livre.

Pour plus d'informations, voir également Type de données et Ensemble de données .
