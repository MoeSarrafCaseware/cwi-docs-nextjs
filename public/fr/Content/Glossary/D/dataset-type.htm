type d'ensemble de données

Catégorise les différents schémas de données qui composent le modèle de données de Caseware Cloud. Chaque type d'ensemble de données est structuré en un ensemble de colonnes, chacune ayant des noms et des propriétés pour le stockage des données. Pour plus d'informations, voir les types d'ensembles de données ci-dessous.

  • Grand livre

  • Balance

  • Mouvements de stock

  • Comptes fournisseurs – Soldes non utilisés

  • Comptes fournisseurs – Paiements effectués décrits

  • Comptes fournisseurs – Factures

  • Comptes fournisseurs – Fournisseurs

  • Comptes clients – Encaissements

  • Comptes clients – Factures

  • Comptes clients – Clients

  • Comptes clients – Soldes non utilisés

  • Stock disponible

  • Données transactionnelles non structurées qui ne suivent pas un schéma prédéfini. Ce type de jeu de données peut être utilisé pour importer des données qui ne correspondent à aucun des autres types de jeu de données. Chaque fichier importé à l'aide de ce type de dataset est considéré comme un dataset à part entière.
