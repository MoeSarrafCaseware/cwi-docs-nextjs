ensemble de données

Ensemble de données conformes à un schéma ou à une catégorie de haut niveau et associées à une entité Caseware. Par exemple, un jeu de données du grand livre peut comprendre plusieurs fichiers de données de transaction, que vous pouvez affecter à des colonnes de données du grand livre lors de l'importation dans un jeu de données du grand livre.

Pour plus d'informations, voir également Type de jeu de données et Source de données .
