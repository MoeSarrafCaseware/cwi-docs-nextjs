Vérifier l'historique du système et des fichiers

Le contrôle des modifications apportées à vos dossiers peut s'avérer impératif pour éviter toute modification injustifiée des dossiers achevés. Avec Caseware Cloud, vous pouvez travailler en toute tranquillité car les changements du système tels que les téléchargements de fichiers, les modifications des paramètres de sécurité et les autres activités des utilisateurs sont enregistrés et consignés.

Visualisation des activités du système

Les activités du système sont générées pour tous les types d'événements qui se produisent dans le nuage, y compris les téléchargements de fichiers, la modification des paramètres de sécurité, la publication d'activités et d'autres événements.

Pour visualiser les activités du système :

  1. Dans le menu Cloud , ouvrez l'application Activités.

  2. Sélectionnez une entité pour afficher uniquement les activités associées à cette entité.

  3. Dans le menu Activités, sélectionnez Activités système.

Visualisation de l'historique des fichiers

Vous pouvez consulter toutes les modifications apportées à un fichier afin de savoir si des changements indésirables ont été apportés ou de savoir qui a travaillé sur le fichier.

Cloud conserve également les versions précédentes des fichiers téléchargés afin que vous puissiez revenir en arrière si la nouvelle version téléchargée est incorrecte. Voir Récupération des versions précédentes pour plus d'informations.

Pour afficher l'historique des fichiers :

  1. Pour consulter l'historique d'un fichier, sélectionnez la page Fichiers . Pour consulter l'historique d'un dossier de mission, sélectionnez la page appropriée ( Documents de travail ou Autres missions).

  2. Sélectionnez un fichier.

  3. Dans le volet de détails du fichier, développez Historique du fichier.
