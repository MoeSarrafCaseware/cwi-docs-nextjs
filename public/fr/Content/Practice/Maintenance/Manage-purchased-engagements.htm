Gérer les missions achetées

Si vous utilisez une application Cloud facturée à l'engagement, vous pouvez gérer les engagements achetés dans Cloud.

Suivez vos engagements achetés

Vous pouvez suivre le nombre d'engagements que vous avez achetés et les engagements restants à partir de la page des paramètres de votre entreprise.

Pour suivre vos engagements achetés et restants :

  1. Assurez-vous d'avoir le rôle d' administrateur des paramètres ou des autorisations équivalentes.

  2. Dans le menu Cloud (), sélectionnez Paramètres.

  3. Sélectionnez Système | Facturation cloud.

  4. Sous Achats d'engagements, vous pouvez voir le nombre d'engagements achetés, créés et restants pour chaque application Cloud que vous utilisez.

Restaurer les jetons d'engagement

Si vous créez un engagement par erreur, vous pouvez le supprimer définitivement dans les 24 heures suivant la création de l'engagement et restaurer le jeton.

Pour supprimer définitivement l'engagement, vous devez

  1. Supprimez le fichier d'engagement de votre application Cloud

  2. Supprimer l'engagement de la corbeille

Le jeton sera ensuite ajouté au nombre d'engagements restants.

Pour supprimer définitivement un engagement :

  1. Dans le menu Cloud (), sélectionnez l'application Cloud appropriée.

  2. Sélectionnez l'engagement que vous souhaitez supprimer, puis sélectionnez Autres actions | Supprimer.

  3. Dans le menu Dossiers , sélectionnez la Corbeille.

  4. Sélectionnez le fichier que vous avez supprimé, puis sélectionnez Autres actions | Supprimer.

Vous pouvez également sélectionner Vider la corbeille dans le menu déroulant de la corbeille pour supprimer définitivement tous les fichiers d'engagement dans la corbeille.
