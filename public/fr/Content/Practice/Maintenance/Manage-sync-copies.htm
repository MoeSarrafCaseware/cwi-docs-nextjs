Gérer les copies de synchronisation

Cloud vous permet de gérer des copies synchronisées des documents de travail directement à partir de votre navigateur.

Pour ouvrir les copies de synchronisation :

Grâce à Cloud, vous pouvez ouvrir une copie synchronisée de votre fichier d'engagement Working Papers sur votre machine locale.

Pour ouvrir les copies de synchronisation :

  1. Assurez-vous d'avoir le rôle d' éditeur ou des privilèges équivalents.

  2. Documents de travail.

  3. Sélectionnez le fichier que vous souhaitez ouvrir, puis sélectionnez Autres actions () | Ouvrir la copie de synchronisation

  4. Working Papers se lance, télécharge le fichier et crée une copie synchronisée. Si vous avez déjà une copie synchronisée, vous avez la possibilité d'ouvrir la copie synchronisée existante.

Supprimer les copies de synchronisation inutiles

Lorsque vous avez terminé de travailler sur un engagement, supprimez votre copie de synchronisation de votre machine locale. La suppression d'une copie de synchronisation permet de s'assurer que toutes les modifications finales sont correctement synchronisées avec le cloud.

Les copies de synchronisation ne peuvent pas être supprimées à l'aide de Cloud. Pour supprimer les copies de synchronisation, utilisez plutôt Working Papers. Pour en savoir plus, voir Gestion des copies de synchronisation.

Gérer les copies de synchronisation

Vous pouvez abandonner des copies synchronisées en suspens à l'aide de Cloud. L'abandon des copies synchronisées non désirées ou inaccessibles permet d'éviter des modifications indésirables de votre fichier Working Papers. Pour en savoir plus, voir Abandon des copies de synchronisation.

Pour abandonner les copies de synchronisation dans le Cloud :

  1. Assurez-vous d'avoir le rôle d' éditeur ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Working Papers.

  3. Sélectionnez un fichier, puis sélectionnez Autres actions () | Abandonner la copie de synchronisation.

  4. Sélectionnez le fichier parent pour abandonner toutes les copies de synchronisation ou sélectionnez les copies de synchronisation spécifiques que vous souhaitez abandonner, puis sélectionnez OK.

Informations claires sur la synchronisation à partir du nuage

L'option Effacer les informations SmartSync permet d'effacer le dossier _Sync d'un fichier client Working Papers dans le Cloud. Pour plus d'informations, voir la commande Clear Synchronization Information (Effacer les informations de synchronisation).

Pour effacer les informations de synchronisation du Cloud :

  1. Assurez-vous d'avoir le rôle d' éditeur ou des privilèges équivalents.

  2. Documents de travail.

  3. Sélectionnez le fichier Working Papers contenant le dossier _Sync que vous souhaitez effacer, puis sélectionnez More Actions () | Details.

  4. Dans le volet Détails , sélectionnez Résumé SmartSync et assurez-vous que toutes les copies de synchronisation existantes du fichier ont été supprimées. Pour en savoir plus, voir Gérer les copies de synchronisation.

  5. Sélectionnez Plus d'actions ( ) | Effacer les informations SmartSync.

Réparer un fichier parent

Si vous rencontrez des erreurs de synchronisation, notamment des événements de synchronisation manquants ou corrompus, vous pouvez tenter de réparer le fichier parent pour les résoudre.

Conseil : Avant de tenter une réparation, sauvegardez votre fichier parent à l'aide de Working Papers. Pour en savoir plus, voir Sauvegarder un fichier.

Pour réparer un fichier parent :

  1. Assurez-vous d'avoir le rôle d' éditeur ou des privilèges équivalents.

  2. Dans le menu Cloud (), sélectionnez Working Papers.

  3. Sélectionnez le fichier que vous souhaitez réparer, puis sélectionnez Plus d'actions () | Exécuter Sync Repair.

  4. Dans la boîte de dialogue de confirmation, sélectionnez Oui pour lancer la réparation.

Si vous continuez à rencontrer des erreurs de synchronisation, il se peut que vous deviez effectuer un rapprochement manuel. Pour plus d'informations, contactez votre distributeur local.

Supprimer un fichier parent

Dans de rares cas, par exemple lorsqu'un fichier a été ajouté au nuage par accident, vous pouvez vouloir supprimer complètement un fichier parent.

Remarque : Vous ne pouvez pas supprimer un fichier parent avec des copies de synchronisation en cours. Si vous ne pouvez pas supprimer les copies de synchronisation en suspens, vous pouvez les abandonner. Pour plus d'informations, voir Gérer les copies de synchronisation.

Pour supprimer un fichier parent :

  1. Assurez-vous d'avoir le rôle d' éditeur ou des privilèges équivalents.

  2. Documents de travail.

  3. Sélectionnez un fichier, puis sélectionnez Autres actions () | Supprimer.

Si nécessaire, vous pouvez récupérer les fichiers supprimés dans la corbeille. Pour plus d'informations, voir Récupérer des dossiers et des fichiers de la corbeille.
