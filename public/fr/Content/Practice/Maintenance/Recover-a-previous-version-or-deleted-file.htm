Récupérer une version précédente ou un fichier supprimé

Caseware Cloud stocke les informations sur l'infrastructure du serveur Amazon Web Services qui peut être situé en dehors du pays où vous travaillez. Caseware Cloud utilise les serveurs Web d'Amazon aux Etats-Unis pour les Amériques, en Australie pour les clients d'Asie-Pacifique, et en Irlande pour les autres juridictions. Toutes vos données, y compris les sauvegardes, se trouvent dans la même région.

Caseware Cloud utilise Amazon Glacier pour le stockage des sauvegardes. Les données relatives aux glaciers sont spécifiques à une seule région [1] [2] et ne sont pas automatiquement répliquées ailleurs. Ainsi, si vous avez des exigences légales selon lesquelles vos données doivent être stockées géographiquement en Australie (par exemple), vos données ne seront stockées que dans ce pays.

Récupérer une version précédente ou un fichier supprimé

Lorsque vous remplacez un fichier téléchargé sur le cloud, vous pouvez facilement récupérer les versions précédentes de ce fichier. Accédez aux détails du fichier pour trouver les copies précédentes de votre document.

To recover a previous version of an uploaded file:

  1. Ouvrez la page Fichiers et sélectionnez une entité dans le sélecteur d'entité.

  2. Sélectionnez le fichier.

  3. Dans le volet Détails du fichier , sous le menu Versions précédentes , sélectionnez la version précédente du fichier.

  4. Sélectionnez l'option Plus d'actions () :

    • Télécharger - Télécharger le fichier sur votre ordinateur

    • View - Voir le fichier dans Caseware Cloud

    • Supprimer - Pour supprimer définitivement la version précédente. Ce fichier n'apparaîtra pas dans la corbeille.

Récupération de dossiers et de fichiers dans la corbeille

Lorsque vous supprimez un fichier ou un dossier, il est envoyé dans la corbeille. Vous pouvez récupérer n'importe quel dossier ou fichier supprimé de la corbeille.

Pour récupérer des dossiers et des fichiers de la corbeille :

  1. Ensure you have Admin, Entity Admin, or Owner role, or equivalent permissions.

  2. Dans le menu Cloud (), sélectionnez Fichiers.

    Si vous souhaitez récupérer un fichier Working Papers, sélectionnez Working Papers.

  3. Sélectionnez la corbeille dans le volet de gauche pour accéder à vos fichiers supprimés.

  4. Sélectionnez les fichiers à restaurer

  5. Dans le panneau de détails du fichier , sélectionnez Plus d'actions() | Restaurer pour déplacer ce fichier vers son entité d'origine.

Si vous rencontrez toujours des problèmes pour récupérer vos fichiers, vous pouvez envoyer une demande de récupération de données à notre équipe d'assistance. Pour plus d'informations, voir Limites de la récupération des données.
