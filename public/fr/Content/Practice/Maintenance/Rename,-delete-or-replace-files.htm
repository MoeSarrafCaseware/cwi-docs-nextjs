Renommer, supprimer ou remplacer des fichiers

Après avoir créé un fichier, vous pouvez le renommer, le supprimer ou le remplacer, selon vos besoins.

Avant de commencer, accédez au menu Cloud () | Fichiers et sélectionnez l'entité appropriée à l'aide de la liste déroulante des entités.

Renommer des fichiers

Vous pouvez modifier les noms de fichiers et les descriptions pour mieux refléter le contenu du fichier.

Remarque : Si vous souhaitez renommer un fichier Working Papers que vous avez publié dans Cloud, nous vous recommandons de convertir le fichier dans un état non synchronisé et de le renommer à l'aide de Working Papers, plutôt que de renommer le fichier dans Cloud. Pour plus d'informations, voir Renommer un fichier.

Pour modifier le nom et la description d'un fichier :

  1. Sélectionnez le fichier que vous souhaitez modifier à l'aide de la case à cocher.

  2. Dans le volet des détails du fichier, sélectionnez Modifier ().

  3. Saisissez un nouveau nom et une nouvelle description pour le fichier, puis sélectionnez Enregistrer.

Supprimer des fichiers

Si vous n'avez plus besoin d'un fichier, vous pouvez le supprimer.

Pour supprimer des fichiers :

  1. Sélectionnez les fichiers à supprimer.

  2. Dans le volet des détails du fichier, sélectionnez Autres actions () | Supprimer.

  3. Confirmer la suppression des fichiers.

Si vous supprimez un fichier par erreur, récupérez-le dans la corbeille. Voir Récupérer une version précédente ou un fichier supprimé pour plus d'informations.

Remplacer les fichiers

Si vous disposez d'une version plus récente d'un fichier que vous souhaitez télécharger, vous pouvez remplacer le fichier original.

Remarque : Si vous souhaitez remplacer un fichier Working Papers que vous avez publié dans le nuage, vous devez d'abord supprimer la copie du nuage, puis télécharger la nouvelle copie à partir de votre ordinateur local.

Pour remplacer un fichier :

  1. Sélectionnez le fichier à remplacer.

  2. Dans le volet des détails du fichier, sélectionnez Autres actions () | Remplacer.

  3. Sélectionnez le bouton Sélectionner les fichiers ou faites glisser le fichier.

Supprimer les versions précédentes d'un fichier

Si vous avez remplacé un fichier et que vous n'avez plus besoin des anciennes versions du fichier, vous pouvez les supprimer.

Avertissement : Si vous supprimez une version antérieure, vous ne pourrez pas la récupérer ultérieurement.

Pour supprimer toutes les versions précédentes d'un fichier :

  1. Sélectionnez le fichier contenant les versions antérieures que vous souhaitez supprimer.

  2. Dans le volet de détails du fichier, développez la section Versions précédentes.

  3. Sélectionnez Autres actions () pour la version que vous souhaitez supprimer, puis sélectionnez Supprimer.

  4. Dans la boîte de dialogue de confirmation, sélectionnez Oui.

Vous pouvez également supprimer toutes les versions précédentes d'un fichier en une seule fois.

Pour supprimer toutes les versions précédentes d'un fichier :

  1. Sélectionnez le fichier contenant les versions antérieures que vous souhaitez supprimer.

  2. Dans le panneau de détails du fichier, sélectionnez Autres actions () | Supprimer toutes les anciennes versions.

  3. Dans la boîte de dialogue de confirmation, sélectionnez Oui.
