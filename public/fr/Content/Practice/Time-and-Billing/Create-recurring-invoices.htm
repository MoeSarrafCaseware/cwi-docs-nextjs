Créer des factures récurrentes

Pour les clients réguliers, vous pouvez facilement créer des factures récurrentes en copiant une facture existante. Lorsque vous copiez une facture, vous créez une nouvelle facture pour le même client, avec le même type et le même montant. Si vous avez activé le grand livre, la facture est également appliquée au même compte du grand livre.

Vous pouvez modifier la facture copiée si vous lui attribuez le statut de brouillon.

Copier une facture existante

Si vous facturez régulièrement des clients, pensez à créer de nouvelles factures en copiant une facture existante. Le type de facture, le montant et le nom du client seront copiés sur la nouvelle facture. Si vous avez activé le grand livre, la nouvelle facture sera même appliquée au même compte du grand livre.

Pour copier une facture :

  1. Assurez-vous d'avoir le rôle de propriétaire ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Facturation.

  3. Sélectionnez l'onglet Factures .

  4. Sélectionnez la facture à copier.

  5. Sélectionnez l'option Plus d'actions () | Copier.

  6. Saisissez le pourcentage d'augmentation de la facture et sélectionnez Copier. L'assistant de facturation s'affiche.

  7. Effectuez les modifications nécessaires à l'aide de l'assistant de facturation.

  8. Sélectionnez Enregistrer pour créer un brouillon ou sélectionnez Finaliser.

Vous avez copié une facture. Il est affiché dans l'application Facturation .

Modification ou suppression d'une facture

Modifiez les factures pour ajouter de nouvelles informations ou supprimez les factures qui ne sont plus nécessaires. Vous ne pouvez modifier ou supprimer que les factures à l'état de brouillon.

Les factures provisoires peuvent être facilement modifiées ou supprimées.

Pour modifier ou supprimer un projet de facture :

  1. Assurez-vous que vous avez le rôle de propriétaire dans l'entité cliente.

  2. Dans le menu Cloud, sélectionnez Facturation.

  3. Sélectionnez l'onglet Factures .

  4. Sélectionnez le projet de facture que vous souhaitez modifier ou supprimer, puis sélectionnez Modifier() pour apporter des modifications, ou sélectionnez Supprimer() pour supprimer la facture.

Vous avez modifié ou supprimé un projet de facture.

Modification d'une facture finalisée

Pour modifier ou supprimer une facture après qu'elle a été finalisée, elle doit d'abord passer au statut de brouillon. Seuls les projets de facture peuvent être modifiés ou supprimés.

Pour modifier le statut d'une facture de finalisée à brouillon :

  1. Assurez-vous que vous avez le rôle de propriétaire dans l'entité cliente.

  2. Dans le menu Cloud, sélectionnez Facturation.

  3. Sélectionnez l'onglet Factures .

  4. Sélectionnez une facture finalisée, puis sélectionnez Autres actions () | Brouillon.

  5. Sélectionnez Oui. Le statut de la facture passe à " brouillon".
