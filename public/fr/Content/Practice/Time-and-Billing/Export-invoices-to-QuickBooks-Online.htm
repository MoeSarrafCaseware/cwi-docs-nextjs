Exporter des factures vers QuickBooks Online

Après l' intégration avec QuickBooks Online, vous pouvez exporter les factures finalisées.

Les factures ne sont pas synchronisées entre les services

Si vous devez modifier une facture dans Cloud après l'avoir exportée vers votre logiciel de comptabilité, vous devez effectuer à nouveau l'exportation pour vous assurer que les deux services sont à jour. Pour plus d'informations, voir Modifier une facture exportée.

L'exportation et la modification des informations sur les clients sont limitées

Lorsque vous exportez une facture, Time tente d'associer la facture à un client préexistant dans votre logiciel de comptabilité. Si aucune correspondance n'est trouvée, un nouveau client est créé en utilisant le nom de l'entité du nuage. Pour modifier les informations relatives à un client après une exportation, vous devez les mettre à jour manuellement dans Cloud et dans votre logiciel de comptabilité.

Les factures créées dans votre logiciel de comptabilité ne peuvent pas être exportées vers le cloud

Si vous créez une facture dans votre logiciel de comptabilité, vous ne pouvez pas exporter cette facture vers Cloud.

Les factures payées et approuvées sont automatiquement clôturées dans le nuage

Lorsque vous marquez une facture comme payée ou approuvée dans votre logiciel de comptabilité, le statut de la facture passe à Fermé dans le nuage. Ce processus peut prendre jusqu'à douze heures en fonction de votre logiciel comptable.

Exporter une facture

Avant de commencer, assurez-vous que vous avez l'autorisation d' exporter vers le logiciel de comptabilité Time.

Remarque : Si vous exportez à partir de QuickBooks Online US, tenez compte des éléments suivants :

  • Si les options de taxe primaire ou secondaire sont désactivées ou ont un taux de 0 dans Paramètres | Temps | Taxe, la ligne associée dans la facture exportée ne sera pas imposable.
  • Une erreur s'affiche si la taxe sur les ventes est définie dans la facture, mais pas dans QuickBooks Online US.
  • Si la taxe de vente automatisée est activée dans QuickBooks Online US, elle remplacera le taux de taxe existant dans la facture. Il peut en résulter un taux d'imposition différent de celui prévu. Assurez-vous que les options de l'impôt primaire ou secondaire sont réglées sur le taux correct.
  • Si la taxe de vente automatisée est désactivée dans QuickBooks Online US (valeur par défaut pour les entreprises créées avant le 10 novembre 2017), assurez-vous que le taux de taxe correspond, sinon une erreur s'affichera.

Exporter des factures vers QuickBooks Online

  1. Dans le menu Cloud (), sélectionnez Facturation.

  2. Sélectionnez les factures finalisées que vous souhaitez exporter, puis cliquez sur Plus d'actions () | Exporter les factures sélectionnées vers QuickBooks Online.

Si des erreurs ou des avertissements surviennent pendant l'exportation, ils sont affichés dans la colonne Exportation QuickBooks .

Modifier une facture exportée

Dans certains cas, il est possible de modifier une facture après son exportation. Par exemple, si un poste est incorrect dans la facture. La procédure de modification d'une facture après son exportation varie en fonction de votre logiciel comptable et du statut de la facture.

Modifier une facture impayée

Si vous avez exporté une facture vers QuickBooks Online mais que vous n'avez pas encore ajouté d'informations de paiement, nous vous recommandons d'effectuer les modifications nécessaires dans Cloud et non dans QuickBooks Online. Les modifications apportées à une facture dans QuickBooks Online ne sont pas synchronisées entre les deux services et peuvent entraîner des enregistrements inexacts dans Cloud.

La facture est supprimée de QuickBooks Online et peut maintenant être modifiée. Après avoir effectué les modifications nécessaires, exportez à nouveau la facture à l'aide du menu Autres actions.

Modifier une facture payée ou approuvée

Les factures marquées comme payées dans QBO sont automatiquement marquées comme fermées dans Cloud. Ce processus peut durer jusqu'à une heure.

  • Pour apporter des modifications à une facture finalisée qui a été marquée comme payée dans QBO, changez son statut en brouillon.

  • Pour apporter des modifications à une facture fermée qui a été marquée comme payée dans QBO, changez le statut de la facture de Fermée à Finalisée, puis changez son statut en Brouillon. Pour en savoir plus sur l'édition des factures clôturées, voir Afficher les factures clôturées.

Une fois que vous avez modifié le statut de la facture dans Cloud, son statut dans QBO passe automatiquement à En suspens. Notez que QBO conserve toutes les informations relatives au paiement.
