Exporter des factures vers Xero

Après l' intégration avec Xero, vous pouvez exporter les factures finalisées.

Limites

Avant d'exporter une facture, il convient de tenir compte des limitations suivantes.

Les factures ne sont pas synchronisées entre les services

Si vous devez modifier une facture dans Cloud après l'avoir exportée vers votre logiciel de comptabilité, vous devez effectuer à nouveau l'exportation pour vous assurer que les deux services sont à jour. Pour plus d'informations, voir Modifier une facture exportée.

L'exportation et la modification des informations sur les clients sont limitées

Lorsque vous exportez une facture, Time tente d'associer la facture à un client préexistant dans votre logiciel de comptabilité. Si aucune correspondance n'est trouvée, un nouveau client est créé en utilisant le nom de l'entité du nuage. Pour modifier les informations relatives à un client après une exportation, vous devez les mettre à jour manuellement dans Cloud et dans votre logiciel de comptabilité.

Les factures créées dans votre logiciel de comptabilité ne peuvent pas être exportées vers le cloud

Si vous créez une facture dans votre logiciel de comptabilité, vous ne pouvez pas exporter cette facture vers Cloud.

Les factures payées et approuvées sont automatiquement clôturées dans le nuage

Lorsque vous marquez une facture comme payée ou approuvée dans votre logiciel de comptabilité, le statut de la facture passe à Fermé dans le nuage. Ce processus peut prendre jusqu'à douze heures en fonction de votre logiciel comptable.

Les numéros de factures appliqués aux factures exportées ne peuvent pas être réutilisés après que la facture a été marquée comme approuvée dans Xero

Si une facture est marquée comme approuvée dans Xero, son numéro de facture ne peut pas être réutilisé. Si une facture est supprimée avant d'être marquée comme approuvée, son numéro peut être réutilisé.

Exporter une facture

Avant de commencer, assurez-vous que vous avez l'autorisation d' exporter vers le logiciel de comptabilité Time.

Exporter des factures vers Xero

  1. Dans le menu Cloud (), sélectionnez Facturation.

  2. Sélectionnez les factures finalisées que vous souhaitez exporter, puis cliquez sur Plus d'actions () | Exporter les factures sélectionnées vers Xero.

Si des erreurs ou des avertissements surviennent pendant l'exportation, ils sont affichés dans la colonne Xero Export.

Modifier une facture exportée

Dans certains cas, il est possible de modifier une facture après son exportation. Par exemple, si un poste est incorrect dans la facture. La procédure de modification d'une facture après son exportation varie en fonction de votre logiciel comptable et du statut de la facture.

Modifier une facture impayée

Si vous avez exporté une facture vers Xero mais que vous n'avez pas encore ajouté d'informations de paiement, nous vous recommandons d'effectuer les modifications nécessaires dans Cloud et non dans Xero. Les modifications apportées à une facture dans Xero ne sont pas synchronisées entre les deux services et peuvent entraîner des enregistrements inexacts dans Cloud.

La facture est supprimée de Xero et peut maintenant être modifiée. Après avoir effectué les modifications nécessaires, exportez à nouveau la facture à l'aide du menu Autres actions.

Modifier une facture payée ou approuvée

Si une facture a été marquée comme approuvée dans Xero, elle ne peut pas être modifiée directement. Au lieu de cela, annulez la facture dans Xero, puis supprimez-la du nuage et créez une nouvelle facture dans le nuage.
