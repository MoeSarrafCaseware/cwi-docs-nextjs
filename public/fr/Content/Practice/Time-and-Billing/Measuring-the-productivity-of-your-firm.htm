Mesurer la productivité

Lorsque votre organisation achève une mission, il peut être utile de déterminer la productivité des membres de l'équipe. Par exemple, si les membres de votre personnel enregistrent moins d'heures que la norme de votre organisation, vous devrez peut-être en rechercher la cause. Une façon de mesurer la productivité à l'aide du cloud est de créer un rapport sur la productivité du temps . Ce rapport affiche l'écart entre les heures standard d'une organisation et les heures réellement travaillées au cours d'une période sélectionnée. Il est généré à partir des entrées de temps créées par les membres de votre organisation.

Générer un rapport détaillé sur la productivité du temps

Un rapport de productivité temporelle au format détaillé montre l'écart total pour la période donnée, ventilé par jour. Si vous souhaitez savoir combien de jours, au cours du mois dernier, votre personnel a respecté les attentes en matière d'heures normales, par exemple, ce format est idéal.

Pour générer un rapport de productivité temporelle au format détaillé :

  1. Pour générer un rapport pour vous-même, assurez-vous que vous avez l'autorisation d' afficher vos propres entrées. Pour générer un rapport pour l'ensemble du personnel, assurez-vous que vous disposez de l'autorisation Voir toutes les entrées du personnel ou de privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Analyse.

  3. Dans le menu déroulant Type , choisissez Productivité du temps.

  4. Dans le menu déroulant Format , choisissez Détaillé.

  5. Dans le menu déroulant Grouper par, choisissez Personnel ou Semaine.

  6. Dans la section Sélectionner , sélectionnez Tous pour générer un rapport pour tous les membres du personnel, ou sélectionnez Personnel pour générer un rapport pour un individu.

  7. Sélectionnez Variance:

    • Tous - Affiche toutes les entrées disponibles.

    • En dessous - Affiche uniquement les écarts négatifs.

    • Sur - N'affiche que les écarts positifs.

    • Supérieur/inférieur - Affiche uniquement les entrées avec écart, tout en omettant les entrées sans écart.

  8. Sélectionnez les options de date et de tri appropriées.

  9. Sélectionnez Générer. Un rapport de productivité temporelle au format détaillé s'affiche.

Vous avez généré un rapport de productivité temporelle au format détaillé. Vous pouvez imprimer ou exporter ce rapport.

Générer un rapport de synthèse sur la productivité du temps

Le format Synthèse présente l'écart total pour une période donnée. Ce format est idéal lorsque vous souhaitez effectuer une simple comparaison entre les membres du personnel ou obtenir une vue d'ensemble de la manière dont votre organisation répond aux attentes.

    • En dessous - Affiche uniquement les écarts négatifs.

    • Sur - N'affiche que les écarts positifs.

    • Supérieur/inférieur - Affiche uniquement les entrées avec écart, tout en omettant les entrées sans écart.

  7. Sélectionnez les options de date et de tri appropriées.
