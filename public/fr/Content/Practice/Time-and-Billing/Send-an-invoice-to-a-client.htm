Envoyer une facture à un client

Lorsque le moment est venu d'envoyer une facture à votre client, vous pouvez la générer et l'envoyer directement à partir de Cloud.

Finaliser une facture

Avant d'envoyer une facture, assurez-vous que son statut est finalisé.

Pour modifier le statut d'une facture de Brouillon à Finalisé :

  1. Assurez-vous que vous avez le rôle de propriétaire sur l'entité client concernée et l'autorisation de traitement.

  2. Dans le menu Cloud, sélectionnez Facturation.

  3. Sélectionnez l'onglet Factures .

  4. Sélectionnez un projet de facture.

  5. Sélectionnez Editer ().

  6. Sélectionnez Finaliser.

  7. (Facultatif) Choisissez un modèle dans le menu déroulant Modèle de facture.

    Pour plus d'informations sur les modèles de facture, voir Créer des modèles de factures.

  8. Sélectionnez le bouton Finaliser . Si vous souhaitez envoyer immédiatement votre facture par courrier électronique, sélectionnez Envoyer.

Vous avez modifié le statut d'une facture de Brouillon à Finalisé.

Envoi des factures finalisées

Vous pouvez envoyer des factures finalisées directement à partir du nuage ou générer un PDF à envoyer par une autre méthode.

Envoi d'une facture finalisée par courriel

Envoyez les factures finalisées directement depuis le nuage par courrier électronique.

Pour envoyer par courrier électronique les factures finalisées sélectionnées :

  1. Assurez-vous que vous avez le rôle de propriétaire sur l'entité cliente et l'autorisation de traitement.

  2. Dans le menu Cloud, sélectionnez Facturation.

  3. Sélectionnez l'onglet Factures .

  4. Sélectionnez les factures finalisées que vous souhaitez envoyer.

  5. Sélectionnez Plus d'actions () | Envoyer les factures sélectionnées.

  6. Remplir tous les champs obligatoires dans la boîte de dialogue Email

  7. Sélectionnez Envoyer.

Vous avez envoyé les factures sélectionnées. Vous pouvez renvoyer les factures en suivant les mêmes étapes, mais chaque facture doit être renvoyée individuellement. Vous ne pouvez pas renvoyer des factures en masse ou dans le cadre d'un groupe de factures finalisées.

Envoyer une copie PDF d'une facture

Lorsque vous devez envoyer une facture par courrier ou à partir d'un autre compte de messagerie, vous pouvez générer une copie PDF.

Pour générer un PDF à partir d'une facture finalisée :

  1. Dans le menu Cloud, sélectionnez Facturation.

  2. Sélectionnez l'onglet Factures .

  3. Modification d'une facture finalisée

  4. Sélectionnez l'option Plus d'actions() | Afficher le PDF.

Vous avez généré une facture au format PDF qui peut être imprimée et envoyée par la poste ou en pièce jointe.

Fermer une facture

Lorsque vous avez terminé de traiter une facture, vous pouvez changer son statut en Closed pour indiquer qu'elle ne nécessite plus d'attention. Par exemple, vous pouvez changer le statut d'une facture en " clôturée " après qu'un client a payé la totalité de la facture. Après la clôture d'une facture, celle-ci est supprimée de l'onglet Factures . Vous pouvez à nouveau afficher les factures clôturées dans l'onglet Factures en utilisant le filtre Fermé . Pour en savoir plus, voir Afficher les factures clôturées.

Pour clôturer une facture :

  1. Assurez-vous que vous disposez de l'autorisation de traitement.

  2. Dans le menu Cloud, sélectionnez Facturation.

  3. Sélectionnez l'onglet Factures .

  4. Sélectionnez une facture finalisée, puis sélectionnez Autres actions () | Fermer.

Vous avez clôturé une facture. Les factures clôturées sont masquées dans l'onglet Factures , mais vous pouvez les afficher à l'aide du filtre Clôturé .

Afficher les factures clôturées

Par défaut, les factures clôturées sont masquées dans l'onglet Factures . Vous pouvez les afficher à l'aide du filtre Closed .

Pour afficher les factures clôturées :

  1. Assurez-vous que vous disposez de l'autorisation de traitement.

  2. Dans le menu Cloud, sélectionnez Facturation.

  3. Sélectionnez l'onglet Factures .

  4. Sélectionnez le bouton de filtrage , puis sélectionnez Statut | Fermé.

Les factures clôturées sont affichées dans l'onglet Factures . Si vous devez modifier une facture clôturée, changez d'abord son statut en Finalisé en sélectionnant l'option Plus d'actions () | Revenir à Finalisé.
