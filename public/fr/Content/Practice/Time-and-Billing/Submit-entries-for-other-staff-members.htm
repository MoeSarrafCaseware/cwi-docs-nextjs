Soumettre des entrées pour d'autres membres du personnel

Pour enregistrer avec précision les heures et les dépenses d'un autre membre du personnel, utilisez les applications de gestion du temps et de la facturation . Ces applications vous permettent de soumettre des candidatures à partir de n'importe quel navigateur, quel que soit l'endroit où vous vous trouvez.

Soumettre des entrées pour d'autres membres du personnel

Soumettre des entrées de temps pour enregistrer le travail d'autres membres du personnel à l'aide de l'application Time.

Soumettre des entrées pour d'autres membres du personnel

  1. Assurez-vous que vous disposez de l'autorisation Traiter toutes les entrées du personnel.

  2. Dans le menu Nuage, sélectionnez Temps.

  3. Sélectionnez Mon temps | Personnel, puis saisissez le nom d'un membre du personnel.

  4. Sélectionnez Nouveau.

  5. Saisir les détails applicables. Un code de travail et un nom ou un numéro de client sont nécessaires.

  6. Sélectionnez Enregistrer.

Une entrée de temps modifiable pour un autre membre du personnel est affichée dans l'application Temps.

Soumettre des entrées pour d'autres membres du personnel

Vous pouvez également effectuer des saisies groupées au nom d'autres membres du personnel.

To submit bulk time entries for other staff members:

  1. Assurez-vous que vous disposez de l'autorisation Traiter toutes les entrées du personnel.

  2. Dans le menu Nuage, sélectionnez Temps.

  3. Sélectionnez Mon temps | Personnel, puis saisissez le nom d'un membre du personnel.

  4. Sélectionnez Nouveau.

  5. Sélectionnez l'onglet Bulk .

  6. Sélectionnez les jours ouvrablesapplicables. Le nombre total d'heures sera réparti de manière égale entre les jours choisis.

  7. Saisissez les informations nécessaires. Un code de travail et un nom ou un numéro de client sont nécessaires.

  8. Sélectionnez Enregistrer.

Les saisies de temps en bloc pour un autre membre du personnel sont affichées dans l'application Temps.

Soumettre des entrées pour d'autres membres du personnel

Soumettre des notes de frais pour que d'autres membres du personnel enregistrent leurs dépenses à l'aide de l'application Dépenses.

Soumettre des entrées pour d'autres membres du personnel

  1. Assurez-vous que vous disposez de l'autorisation Traiter toutes les entrées du personnel.

  2. Dans le menu Cloud, sélectionnez Dépenses.

  3. Sélectionnez Mes dépenses | Personnel, puis saisissez le nom d'un membre du personnel.

  4. Sélectionnez Nouveau.

  5. Saisissez les informations nécessaires. Vous pouvez également joindre un reçu en sélectionnant Browse.... Les pièces jointes doivent être d'une taille inférieure ou égale à 20 Mo.

  6. Sélectionnez Enregistrer.

Une note de frais modifiable pour un autre membre du personnel s'affiche dans l'application Dépenses.

Importer les entrées de temps et de dépenses à l'aide d'un fichier CSV

Vous pouvez importer en masse les entrées de temps et de dépenses à l'aide d'un fichier CSV. Cette procédure est utile si vous importez des entrées à partir d'autres services ou si vous devez ajouter simultanément un grand nombre d'entrées pour plusieurs membres de votre organisation.

Pour importer plusieurs entrées de temps ou de dépenses à l'aide d'un fichier .csv :

  1. Assurez-vous d'avoir le rôle d' administrateur des paramètres et l'autorisation de traiter toutes les entrées.

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Sélectionnez Temps | Importation d'entrées de temps et de dépenses | Importation d'entrées.

  4. Sélectionnez le type d'entrée que vous importez :

    • L'heure

    • Dépenses

    • Temps et dépenses

  5. Sélectionnez Importer à partir de CSVou collez le contenu du fichier CSV dans la zone de texte prévue à cet effet. Sélectionnez Next.

  6. Sélectionnez un type pour chaque colonne ( code de travail ou courriel du personnel, par exemple) ou sélectionnez Ignorer pour ne pas en tenir compte.

  7. Sélectionnez Importer, puis choisissez l'une des options suivantes :

    • Enregistrer les entrées dont l'importation a échoué dans un fichier .csv - En choisissant cette option, vous créez un fichier CSV contenant toutes les entrées dont l'importation a échoué, ainsi que les messages d'erreur correspondants.

    • Enregistrer toutes les entrées dans un fichier .csv - Cette option permet d'enregistrer toutes les entrées importées, qu'elles soient réussies ou non, dans un nouveau fichier .csv.

    • Ne pas enregistrer d'entrées - Aucune entrée n'est enregistrée.

  8. Sélectionnez OK.

Toutes les entrées dont l'importation a échoué sont affichées dans la boîte de dialogue Importer les entrées de temps et de dépenses . Si vous avez enregistré un fichier CSV, ouvrez-le pour voir les erreurs qui ont causé l'échec de l'importation. Les erreurs simples (telles que les fautes d'orthographe) peuvent être corrigées à partir de la boîte de dialogue en sélectionnant la cellule appropriée et en y apportant des modifications. Dans le cas d'une erreur plus complexe (telle qu'une colonne manquante), vous pouvez modifier le fichier CSV et réimporter les entrées.
