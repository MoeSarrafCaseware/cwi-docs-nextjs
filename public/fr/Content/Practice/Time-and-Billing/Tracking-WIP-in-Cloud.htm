Suivre les travaux en cours dans Caseware Cloud

Cloud propose trois méthodes pour suivre les travaux en cours :

  • Utilisation de l'onglet WIP dans l'application Facturation .

  • Générer des rapports d'encours dans l'application Analyse.

  • Utilisation du gadget Top 10 WIP Clients ou du gadget WIP Analysis sur un tableau de bord personnalisé.

L'onglet WIP fournit une vue d'ensemble basée sur les soldes de WIP, tandis que l'application Analysis fournit des rapports plus détaillés qui peuvent être exportés et partagés. Les gadgets WIP peuvent être intégrés dans un tableau de bord personnalisé et permettent de comparer les données avec celles disponibles dans d'autres gadgets. Ces outils permettent à votre organisation d'analyser avec précision et en temps réel les données relatives aux encours de production.

Affichage de l'onglet WIP

L'onglet En-cours affiche le total des en-cours pour chaque client. Cet onglet permet de trier et de comparer rapidement les clients.

Pour afficher l'onglet WIP :

  1. Assurez-vous que vous disposez du rôle Accès entité ou des privilèges équivalents pour les clients dont vous souhaitez afficher les totaux d'encours

  2. Choisissez Toutes les entités dans le sélecteur d' entités .

  3. Ouvrez l'application Facturation . L'onglet WIP s'affiche. Vous pouvez personnaliser les données affichées en sélectionnant le filtre ( ).

Vous avez affiché l'onglet WIP.

Générer un rapport d'encours à l'aide de l'application Analyse

L'application Analyse fournit des rapports sur les encours en fonction des paramètres sélectionnés. Ces rapports fournissent des informations plus détaillées que l'onglet WIP et peuvent être exportés et partagés.

Pour créer un rapport d'encours :

  1. Assurez-vous que les rôles suivants, ou des rôles avec des privilèges équivalents, vous ont été attribués :

    • Traiter ses propres entrées est nécessaire pour accéder à ses propres transactions. Cette autorisation est attribuée à l'aide de l'application Time.

    • L'administrateur du personnel est nécessaire pour voir les transactions des autres membres du personnel.

    • L'accès à l'entité est requis pour chaque entité client que vous souhaitez inclure.

  2. Dans le menu Nuage, sélectionnez Analyse

  3. Dans le menu déroulant Type , choisissez un type de rapport d' encours :

    • WIP - As of - Afficher uniquement les WIP jusqu'au jour spécifié

    • WIP - As of - Engagement - Affiche uniquement les WIP jusqu'au jour spécifié qui sont associés à un engagement dans Cloud.

    • En-cours - Plage de dates - Affiche l'en-cours pour la plage de dates spécifiée.

    • WIP - Date Range - Engagement - Affiche uniquement les WIP pour la plage de dates spécifiée qui sont associés à un engagement dans Cloud.

  4. Choisissez Détaillé ou Résumé dans le menu déroulant Format.

  5. Choisissez les options appropriées dans les menus déroulants Sélectionner, Date et Trier .

  6. Sélectionnez Générer

Vous avez généré un rapport d'encours. Vous pouvez maintenant l' imprimer ou l' exporter .

Création d'un tableau de bord de l'état d'avancement

Un tableau de bord permet d'obtenir un aperçu rapide des données relatives aux encours de production sans avoir à générer un rapport à chaque fois. L'ajout d'autres widgets à ce tableau de bord vous permet de mettre en contraste les données de l'encours avec d'autres informations.

Pour créer un tableau de bord de l'encours :

  1. Assurez-vous d'avoir le rôle d' éditeur ou des privilèges équivalents et l'autorisation de voir l'encours et la facturation.

  2. Dans le menu Cloud, sélectionnez Tableaux de bord.

  3. Sélectionnez Nouveau.

  4. Saisissez un nom et une description.

  5. Choisissez les personnes avec lesquelles vous souhaitez partager ce tableau de bord en sélectionnant Tout le personnel, Tous les contactsou les deux. Sélectionnez ensuite OK.

  6. Sélectionnez Ajouter | Temps, puis ajoutez les gadgets WIP :

    • Top 10 WIP Clients - Affiche une liste des 10 premiers clients WIP par solde à une date sélectionnée.

    • Analyse de l'encours - Compare le solde de l'encours pour une période donnée avec le solde de l'encours pour la même période dans la période précédente.

  7. Sélectionnez Enregistrer.

Vous avez créé un tableau de bord qui permet d'accéder rapidement aux données relatives à l'encours de production. Affichez-le en sélectionnant Tableaux de bord dans le menu Nuage.
