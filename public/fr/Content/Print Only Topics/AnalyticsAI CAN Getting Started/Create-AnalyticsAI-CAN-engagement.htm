Créer l'engagement

Le dossier de mission est l'endroit où vous recueillez, organisez et examinez les données de votre client.

Pour plus d'informations, voir Valider les fichiers d'engagement .

Pour créer un engagement AnalyticsAI CAN :

  1. Dans le menu Cloud (), sélectionnez AnalyticsAI CAN.
  2. Sélectionnez l'entité que vous allez auditer dans le menu déroulant, par exemple :

  3. Sélectionnez Nouveau ().
  4. Saisissez un nom pour l'engagement, entrez toute autre information facultative, puis sélectionnez Enregistrer.
    Les propriétés de l'engagement s'affichent.

  5. Modifiez comme vous le souhaitez et sélectionnez Enregistrer.
    La page Documents s'affiche.

    Remarque: La date de fin d'année sélectionnée doit correspondre à celle qui figure dans les fichiers de la balance de vérification et des transactions.

La page Documents contient des liens vers les documents relatifs à ces étapes de l'engagement :

  • Planification - rassembler les informations et les données nécessaires à la mission.
  • Évaluation des risques - vérification de la fiabilité et de l'exhaustivité des données, ajustement des facteurs d'influence, analyse des risques et examen analytique.
  • Conclusion - téléchargez les documents justificatifs ici.
    Remarque: La section Conclusion n'est pas visible initialement. Il deviendra visible lorsque vous compléterez le fichier Optimiser dans la section Planification .
