Planification

Au cours de cette étape, vous effectuerez les opérations suivantes :

  • 0-0 Optimiser
  • 0-1 Questions des clients
  • Importation de la balance des comptes et des transactions en ligne
  • 0-3 Importer des transactions CSV - Si vous importez des transactions en ligne en utilisant le document ci-dessus, vous n'aurez pas besoin d'importer un CSV.

Optimiseur

The 0-0 Optimiser file allows you to:

  • Décidez s'il convient d'utiliser le document "Questions du client" pour demander des informations à votre client.
  • Sélectionnez les sections que vous souhaitez inclure dans l'engagement.

En fonction de vos réponses dans ce document, Caseware AnalyticsAI remplit automatiquement l'espace de travail de la mission avec les documents dont vous avez besoin pour mener à bien la mission.

Requêtes

Utiliser les questions 0-1 du client pour recueillir les informations nécessaires à la mission. Vous pouvez également utiliser ce document pour demander la balance de vérification et les données relatives aux transactions de votre client.

Pour plus d'informations, voir Demander des informations à un client .

Importations

La balance de vérification et les données de transaction peuvent être importées à partir de fichiers Excel ou CSV, d'autres logiciels de comptabilité de bureau et de plateformes de comptabilité en ligne. Vous pouvez également importer des données de balance de vérification à partir d'un fichier Caseware Working Papers existant. Importer la balance de vérification d'un client dans la page Données

Si vous avez importé les données de la balance de vérification du client à partir d'un fichier CSV, vous devrez importer le fichier CSV des transactions afin de stocker et d'exécuter des analyses sur vos données.

To import transactions from a CSV file:

  1. Sélectionnez le fichier CSV d' importation de transactions 0-3 dans la page Documents.
  2. Sélectionnez le fichier CSV des transactions dans votre répertoire local.
  3. Mettez en correspondance les colonnes du fichier de transactions.
  4. Sélectionner IMPORT.

Pour plus d'informations, voir Importer les transactions du client .
