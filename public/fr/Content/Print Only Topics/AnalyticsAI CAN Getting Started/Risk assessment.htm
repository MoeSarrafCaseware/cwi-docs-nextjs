Influenceurs

Les influenceurs sont des comptes sur lesquels vous souhaitez mettre l'accent dans le cadre de votre analyse des risques. Ils identifient les risques macroéconomiques et organisationnels et mettent en évidence les transactions susceptibles d'être intéressantes à la lumière de ces risques.

Par exemple, si vous soupçonnez des faiblesses de contrôle sur les comptes de trésorerie, vous pouvez définir l'influence de tout compte de trésorerie afin d'augmenter le score d'un pourcentage spécifique. Vous pouvez fixer l'influence des comptes en espèces à 50 %, par exemple, et les scores de risque de toutes les transactions contenant des comptes en espèces seront automatiquement augmentés de 50 %. Les transactions à risque zéro ne seront pas ajustées et resteront donc à risque zéro.

Pour plus d'informations, voir Détection des valeurs aberrantes .

Ajuster les influenceurs :

  1. Sélectionnez 2-2 Influenceurs (Risque organisationnel) sur la page Documents .
  2. Sélectionnez l'onglet Résumé .
  3. Cliquez sur le champ % d'influence du compte souhaité et saisissez une nouvelle valeur.
  4. Documenter la raison de l'augmentation du % d'influence dans la case Note.

Les couleurs qui apparaissent à côté des différents comptes sous Influence représentent le niveau de risque :

  • Vert - risque faible
  • Jaune - risque moyen
  • Rouge - risque élevé

Widgets

Cliquez sur une information ou survolez les graphiques dans les widgets pour afficher plus d'informations. Les informations affichées dans les widgets changent pour refléter votre sélection.

Vous pouvez cliquer sur les widgets Résumé du risque, Exceptions à haut risque, Utilisateurs à haut risque et Journaux à haut risque pour obtenir plus de détails. See Navigate a widget for more information.

Modifier la configuration de certains tests

Pour affiner les tests, sélectionnez l'onglet CONFIGURATION .

A partir de là, vous pouvez voir tous les tests qui sont exécutés dans AnalyticsAI. Vous pouvez sélectionner et désélectionner les tests à exécuter, configurer les paramètres de test et modifier l'importance(score de risque) de certains tests. Vous pouvez également affiner les paramètres du test.

Pour afficher la page CONFIGURATION :

  1. Cliquez sur 2-2 Analyse de risque (Risque de transaction) dans la page Documents .
    La page RÉSULTATS s'affiche.
  2. Sélectionnez CONFIGURATION.

Désélectionnez la case d'un test si vous ne souhaitez pas qu'il soit exécuté.

Vous pouvez également utiliser cette page pour mettre l'accent(score de risque) sur des tests spécifiques et leurs paramètres.

Tableaux et graphiques

Les tests comprennent un diagramme circulaire qui indique le pourcentage et la quantité des transactions signalées pour ce test. Des diagrammes à secteurs apparaissent lorsqu'un test a mis en évidence des transactions.

Certains tests, tels que le test " High Amounts" , comprennent un diagramme circulaire et un graphique supplémentaire qui vous aide à configurer les paramètres.

Les diagrammes à barres spécifiques au test montrent la distribution des données par rapport au test, ce qui permet à l'auditeur d'avoir une idée claire de l'endroit où fixer le paramètre. Par exemple, le diagramme à barres des montants élevés montre la distribution des montants de la ligne de transaction. Vous pouvez ajuster le percentile pour marquer les montants à droite de la barre orange.

Dans cet exemple, la valeur par défaut du percentile a été modifiée en 99,7. Remarquez que pour la valeur de percentile par défaut de 99,1, 21 transactions sont signalées. Lorsque nous modifions le percentile à 99,7 et que nous réexécutons les tests, le nombre de transactions signalées tombe à cinq.

Détection des valeurs aberrantes

AnalyticsAI dispose d'un certain nombre de tests de détection de valeurs aberrantes basés sur l'apprentissage machine (ML).

Ces tests sont désactivés par défaut. Utilisez la page CONFIGURATION pour les activer comme vous le souhaitez.

Pour plus d'informations, voir Détection des valeurs aberrantes .
