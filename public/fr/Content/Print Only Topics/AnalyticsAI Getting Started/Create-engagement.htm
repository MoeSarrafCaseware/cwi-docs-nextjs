Créer l'engagement

Le dossier de mission est l'endroit où vous recueillez, organisez et examinez les données de votre client.

Pour plus d'informations, voir Valider les fichiers d'engagement .

Pour créer un engagement AnalyticsAI :

  1. Dans le menu Cloud (), sélectionnez AnalyticsAI ().
  2. Sélectionnez l'entité que vous allez auditer dans le menu déroulant, par exemple :

  3. Sélectionnez Nouveau ().
  4. Saisissez un nom pour l'engagement, entrez toute autre information facultative, puis sélectionnez Enregistrer.
    Les propriétés de l'engagement s'affichent.

  5. Modifiez les propriétés de l'engagement comme vous le souhaitez et sélectionnez Enregistrer.

    Remarque : Pour éviter les erreurs, la date de fin d'année doit correspondre aux données de la balance de vérification.
    La page Documents s'affiche.

La page Documents contient des liens vers les documents relatifs à ces étapes de l'engagement :

  • Préparation - rassembler les données nécessaires à la mission
  • Validation - vérification de la fiabilité et de l'exhaustivité des données
  • Profil de risque - ajuster les influenceurs, effectuer une analyse de risque et procéder à un examen analytique
