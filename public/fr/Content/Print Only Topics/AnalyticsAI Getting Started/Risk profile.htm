Influenceurs

Les influenceurs sont des comptes sur lesquels vous souhaitez mettre l'accent dans le cadre de votre analyse des risques. Ils identifient les risques macroéconomiques et organisationnels et mettent en évidence les transactions susceptibles d'être intéressantes à la lumière de ces risques.

Par exemple, si vous soupçonnez des faiblesses de contrôle sur les comptes de trésorerie, vous pouvez définir l'influence de tout compte de trésorerie afin d'augmenter le score d'un pourcentage spécifique. Vous pouvez fixer l'influence des comptes en espèces à 50 %, par exemple, et les scores de risque de toutes les transactions contenant des comptes en espèces seront automatiquement augmentés de 50 %. Les transactions à risque zéro ne seront pas ajustées et resteront donc à risque zéro.

Pour plus d'informations, voir Détection des valeurs aberrantes .

Analyse des risques

La page Résultats vous donne un aperçu des transactions de l'engagement à l'aide de différents widgets.

To view the test results:

  • Cliquez sur 2-2 Analyse de risque (Risque de transaction) dans la page Documents .

Widgets

Cliquez sur l'information dans l'un des widgets pour afficher plus d'informations. Les informations affichées dans les widgets changent pour refléter votre sélection.

Vous pouvez cliquer sur les widgets Résumé du risque, Exceptions à haut risque, Utilisateurs à haut risque et Journaux à haut risque pour obtenir plus de détails. See Navigate a widget for more information.

Résumé des risques

Ce widget classe les transactions en fonction de leur score de risque global.

Exceptions à haut risque

Ce widget montre les transactions avec un score de risque qui sont regroupées par test. Les quatre tests les plus importants en termes de montant s'affichent.

Utilisateurs à haut risque

Ce widget affiche les cinq premiers utilisateurs ayant effectué des transactions avec le score de risque le plus élevé.

Revues à haut risque

Ce widget affiche les cinq revues ayant le score de risque le plus élevé.

Graphique de séries temporelles

Utilisez le graphique des séries chronologiques pour observer le score de risque et la valeur en dollars des transactions évaluées pour les 13 mois précédents à partir de l'ensemble de données téléchargé. Survolez un élément pour afficher plus d'informations.

Modifier la configuration de certains tests

A partir de là, vous pouvez voir tous les tests qui sont exécutés dans AnalyticsAI. Vous pouvez également configurer les tests en procédant comme suit :

  • Sélectionnez les tests à effectuer.
  • Modifier l'importance accordée à certains tests(score de risque).
  • Refine test parameters.

Désélectionnez la case d'un test si vous ne souhaitez pas qu'il soit exécuté.

Vous pouvez également utiliser cette page pour mettre l'accent(score de risque) sur des tests spécifiques et leurs paramètres.

Tableaux et graphiques

Les tests comprennent un diagramme circulaire qui indique le pourcentage et la quantité des transactions signalées pour ce test. Des diagrammes à secteurs apparaissent lorsqu'un test a mis en évidence des transactions.

Certains tests, tels que le test " High Amounts" , comprennent un diagramme circulaire et un graphique supplémentaire qui vous aide à configurer les paramètres.

Les diagrammes à barres spécifiques au test montrent la distribution des données par rapport au test, ce qui permet à l'auditeur d'avoir une idée claire de l'endroit où fixer le paramètre. Par exemple, le diagramme à barres des montants élevés montre la distribution des montants de la ligne de transaction. Vous pouvez ajuster le percentile pour marquer les montants à droite de la barre orange.

Dans cet exemple, la valeur par défaut du percentile a été modifiée en 99,7. Remarquez que pour la valeur de percentile par défaut de 99,1, 21 transactions sont signalées. Lorsque nous modifions le percentile à 99,7 et que nous réexécutons les tests, le nombre de transactions signalées tombe à cinq.

Détection des valeurs aberrantes

AnalyticsAI dispose d'un certain nombre de tests de détection de valeurs aberrantes basés sur l'apprentissage machine (ML). Ces tests sont désactivés par défaut.

Pour activer les tests de détection des valeurs aberrantes :

  • Cochez les cases des tests de détection des valeurs aberrantes sur la page CONFIGURATION

Pour plus d'informations, voir Détection des valeurs aberrantes .

Examen analytique

Le document d' examen analytique 2-3 identifie les relations (pour une utilisation éventuelle dans les procédures analytiques de fond), les risques et les incohérences. Les tableaux d'analyse figurant dans le présent document sont liés aux données tout au long de la mission afin d'aider les auditeurs à comparer visuellement les données et à découvrir des tendances et des schémas à l'aide d'analyses de données prédéfinies.

Utilisez le menu déroulant pour sélectionner les résultats préliminaires ou finaux . Utilisez les boutons de filtrage pour afficher les widgets d'analyse souhaités. Les widgets sont interactifs. Survolez un élément spécifique d'un graphique pour obtenir plus d'informations.
