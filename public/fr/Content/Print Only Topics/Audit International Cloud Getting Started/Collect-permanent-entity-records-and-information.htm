Collecter les dossiers et les informations sur les entités permanentes
Informations

Utilisez le A630Q Demande de registres et de documents - dossier permanent pour demander des documents qui peuvent être utilisés lors de l'audit de l'année suivante - les documents demandés ici sont généralement applicables à plusieurs années. Par exemple, le plan d'affaires, l'organigramme et les statuts de la société.

Pour plus d'informations, voir Collecte d'informations auprès du client .

Cette étape n'est pas facultative. Les auditeurs sont tenus de demander ces informations. Le document de requête D700Q est disponible si l'utilisateur répond "Collaboration élevée " ou "Collaboration limitée " dans la liste de contrôle de l'Optimizer A100 - Décisions initiales d'engagement.
