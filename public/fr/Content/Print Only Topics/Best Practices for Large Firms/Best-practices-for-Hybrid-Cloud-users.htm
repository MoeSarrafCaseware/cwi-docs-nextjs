Meilleures pratiques pour les utilisateurs de nuages hybrides

Afin d'optimiser les performances du site Cloud, nous recommandons aux utilisateurs de Cloud hybride de suivre les directives détaillées ci-dessous.

Utiliser le nuage uniquement pour les fichiers d'engagement

Nous recommandons aux utilisateurs de Cloud hybride d'utiliser leur site Cloud uniquement pour les missions et de minimiser leur utilisation d'autres fonctionnalités telles que les activités, le temps Cloud et les fichiers. L'application Fichiers, en particulier, peut entraîner des problèmes de performance pour les utilisateurs qui travaillent sur des missions si une grande quantité de fichiers est stockée sur votre site en nuage.

Désactiver le suivi du temps pour les fichiers d'engagement

Le suivi du temps permet de suivre automatiquement le temps passé par les membres du personnel dans les dossiers d'engagement. Nous recommandons aux utilisateurs de Cloud hybride de désactiver cette fonctionnalité dans les Working Papers afin d'optimiser les performances du Cloud.

Remarque : La désactivation du suivi du temps n'est possible qu'avec les Working Papers 2020 : Update 1 or later. Les utilisateurs de versions antérieures de Working Papers ne pourront pas désactiver le suivi du temps.

Pour désactiver le suivi du temps :

  1. Naviguez jusqu'au répertoire Caseware .

    L'emplacement par défaut de ce répertoire est le répertoire Program Files (x86) .

  2. Ouvrez le fichier CV.ver à l'aide du Bloc-notes ou d'un programme similaire.

  3. Ajoutez la ligne suivante au fichier : DisableTimeEntries=1

  4. Sélectionnez Fichier > Enregistrer ou appuyez sur CTRL + S pour enregistrer le fichier.

Définir les paramètres du délai d'attente

Nous recommandons également d'ajouter les paramètres suivants au fichier CV.ver pour améliorer encore les performances.

Pour définir les paramètres régionaux :

  1. Naviguez jusqu'au répertoire Caseware .

    L'emplacement par défaut de ce répertoire est le répertoire Program Files (x86) .

  2. Ouvrez le fichier CV.ver à l'aide du Bloc-notes ou d'un programme similaire.

  3. Ajoutez les lignes suivantes au fichier :

    MasterAvailabilityTimeout=30000

    LongerMasterAvailabilityTimeout=40000

    SSSTimeout=50000

  4. Sélectionnez Fichier > Enregistrer ou appuyez sur CTRL + S pour enregistrer le fichier.
