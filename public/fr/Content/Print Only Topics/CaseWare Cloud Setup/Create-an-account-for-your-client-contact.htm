Créer un compte pour le personnel ou les contacts

L'un des principaux avantages de Caseware Cloud est que vous pouvez collaborer directement avec vos clients sur les missions. Vous devez créer un compte de contact pour que vos clients puissent accéder à leur dossier d'engagement.

Kit d'entraînement

Dans le menu Cloud, sélectionnez Groupes. Vous accédez ainsi à la fonctionnalité AnalyticsAI.

Sélectionnez Nouveau, puis AJOUTER | Contacts. La boîte de dialogue Ajouter un contact s'affiche.

Général

Saisissez un prénom, un nom et une adresse électronique pour votre contact client. Dans le champ Entité, sélectionnez l'entité client que vous avez créée précédemment. Après avoir sélectionné l'entité cliente, sélectionnez le rôle Contact - Collaborateur de l'entité pour définir le niveau d'accès à ce compte. Ce rôle permet à votre client de créer des discussions, des tâches et des demandes de fichiers sur le cloud. Il leur permet également de collaborer sur des engagements pour des applications en nuage.

Mot de passe

Les mêmes options de mot de passe que pour les comptes du personnel s'appliquent ici. Comme l'adresse électronique est la vôtre, choisissez Définir et décochez l'option L'utilisateur doit modifier son mot de passe lors de la prochaine connexion.

Groupes

Les groupes de contact fonctionnent de la même manière que les groupes de personnel. Le groupe de contacts disponible d'emblée est Tous les contacts et, par défaut, votre client est automatiquement membre de ce groupe.

Applications

Tout comme le groupe " Tout le personnel" , le groupe " Tous les contacts" permet à ses membres d'afficher et de créer des engagements pour toutes les applications Cloud disponibles.

Une fois que vous avez terminé de configurer le nouveau compte de votre contact client, sélectionnez Enregistrer. Ils recevront un courriel de bienvenue de Cloud les informant que leur compte a été créé et leur indiquant comment se connecter.
