Pour commencer

Avant de commencer à travailler sur une mission, vous devez configurer le nouveau portail Caseware Cloud pour votre entreprise. Caseware propose un guide de démarrage complet (https://docs.caseware.com/2019/webapps/30/en/Explore/Getting-Started/Get-started-with-Caseware-Cloud.htm) qui vous guide à travers les tâches les plus courantes pour les nouvelles entreprises du Cloud. Pour les besoins de cette formation, vous devez suivre trois étapes :

  • créer des espaces de travail dans Caseware Cloud pour chacun de vos clients
  • créer des comptes pour les membres de votre personnel
  • créer des comptes pour vos clients

Cette formation vous guide tout au long des étapes ci-dessus. Une fois que vous les avez complétés, vous êtes prêt à créer un engagement. Après ces étapes, nous verrons également comment gérer les paramètres de l'entreprise.
