Effectuer une exportation fiscale

La CRT canadienne vous permet d'effectuer les tâches requises dans le cadre d'une mission fiscale, ainsi que les fonctions de compilation et d'examen.

  • Remplir la liste de contrôle de l'optimiseur fiscal

  • Importer les données fiscales du client

  • compléter le profil du préparateur fiscal

  • remplir les formulaires provinciaux et fédéraux à l'aide des feuilles de calcul de l'impôt

  • effectuer un contrôle de diagnostic pour s'assurer qu'il n'y a pas d'erreurs en suspens

  • déposer la déclaration par voie électronique via la TED ou créer un PDF pour votre client si nécessaire

Mise en place de l'engagement fiscal

La première étape consiste à définir le type de mission comme étant une mission fiscale.

Kit d'entraînement

  1. Dans l' Optimiseur de planification 0-10, sélectionnez Impôt sur les sociétés uniquement à côté de Quel type de mission est effectué ?

  2. Sélectionnez le lien suivant pour ouvrir l' Optimiseur fiscal.

    The Tax Optimiser Checklist displays.

Complétez le document en indiquant les coordonnées de l'entité pour laquelle vous effectuez la mission fiscale, ainsi que les caractéristiques de la mission. Vous pouvez accéder aux différentes sections en sélectionnant l'onglet correspondant au-dessus du document.

La mission sera automatiquement alimentée avec les formulaires fiscaux provinciaux et fédéraux appropriés, ainsi qu'avec les feuilles de travail de Caseware ReviewComp. Ces documents apparaîtront dans la carte des documents sur le panneau de gauche. Vous pouvez choisir de compléter les nouveaux documents au fur et à mesure qu'ils apparaissent pour chaque procédure à laquelle vous répondez, ou de terminer la liste de contrôle et de compléter les documents ajoutés plus tard.

Importer les données

Caseware ReviewComp vous propose plusieurs façons d'importer les données fiscales de la mission.

Sélectionnez Importer pour afficher les options d'importation.

Exercice 1 - Importation de l'IGRF

Sélectionnez cette option pour rechercher sur votre ordinateur un fichier de sortie GIFI provenant de votre logiciel de comptabilité.

Remarque : Cette option ne doit être utilisée que si vous n'avez pas l'intention d'utiliser la balance de vérification, car toute modification de la balance de vérification effacera d'abord les formulaires de l'IGRF et transférera les données de la balance de vérification dans les formulaires de l'IGRF.

Exercice 2 - TaxPrep CSV Import

Sélectionnez cette option pour télécharger un fichier de sortie .csv généré par TaxPrep.

Notes :

  • Effectuez un report dans TaxPrep avant d'exporter le fichier CSV dans TaxPrep.
  • TaxPrep CSV Import effacera toutes les données transférées de la balance de vérification dans l'IGRF et les remplacera par les données de l'importation CSV. Par conséquent, vous devez importer ce fichier CSV avant d'effectuer des importations dans la balance de vérification.

Pour exporter un fichier .CSV à partir de TaxPrep :

  1. Sélectionnez | TaxPrep CSV.

  2. Naviguez jusqu'au fichier souhaité et sélectionnez Ouvrir.

Practice set 3 - T2 Auto-fill

Si votre client est inscrit au programme T2 Auto-Fill de l'Agence du revenu du Canada (ARC), vous pouvez importer automatiquement les données fiscales de la mission en vous basant sur les déclarations précédemment déposées par le client.

Practice set 4 - Télécharger CloudBridge

CloudBridge est un add-on pour Caseware Working Papers qui vous permet de lier un engagement Working Papers à une entité Cloud pour importer les données fiscales de votre engagement dans vos applications Cloud. Utilisez cette option si vous avez un engagement de Working Papers qui contient des données de l'IGRF.

Pour télécharger CloudBridge :

  1. From Data | Import from, select Caseware Working Papers.

  2. Suivez les instructions de la page d'aide qui s'affiche pour importer vos données d'engagement à partir des Working Papers.

Remplir le document relatif au profil du préparateur fiscal

Le document T-02 TPP - Profil du préparateur fiscal vous demande de saisir des informations clés sur l'entité qui effectue la mission fiscale.

Une fois ce document complété et signé, vous êtes prêt à commencer votre travail d'engagement fiscal.

Remplir les formulaires fiscaux

Une fois les données fiscales importées et les documents de profil complétés, vous êtes prêt à commencer à remplir les formulaires fiscaux fédéraux et provinciaux de la mission. Comme vous avez rempli l'Optimiseur fiscal, seuls les formulaires pertinents pour votre mission fiscale sont affichés dans l'onglet Documents.

Pour consulter la liste complète des formulaires fiscaux disponibles, sélectionnez l'icône Modifier () sur la page Documents , ou ouvrez le document Liste des formulaires fiscaux . Utilisez la fonction de recherche de votre navigateur(Ctrl-F)pour localiser des documents spécifiques.

Vos sélections déterminent également les feuilles de travail qui sont listées, le cas échéant. Les fiches de travail sont des documents conçus pour rationaliser votre travail d'engagement.

Ensemble de pratiques - Naviguer entre les documents fiscaux connexes

Vous pouvez facilement naviguer entre les diagrammes de travail et les documents de planification qui y sont liés. Il existe deux options pour passer d'un document à l'autre.

Lorsque vous sélectionnez un champ ayant une source unique, la boîte de dialogue Remplacement du champ s'ouvre.

Lorsque vous sélectionnez Sauter à la source, le document source s'ouvre et passe directement à la ligne appropriée, avec le champ source en surbrillance.

Recherche de formulaires fiscaux et de feuilles de calcul

Lorsque vous travaillez sur un formulaire fiscal ou une feuille de calcul, vous pouvez rechercher, modifier la visibilité et ouvrir tous les formulaires fiscaux et feuilles de calcul de la mission.

Kit d'entraînement

  1. Sélectionnez Rechercher à partir de n'importe quel formulaire fiscal ou feuille de calcul ouvert.

  2. Enter the name of the document you want to find.

    3. Vous pouvez également sélectionner Afficher tous les formulaires fiscaux pour afficher tous les formulaires fiscaux et toutes les feuilles de calcul de la mission.

  3. Sélectionnez le commutateur à côté d'un formulaire pour l'activer ou le désactiver.

    4. Un interrupteur vert s'affiche à côté des formulaires visibles dans la liste.

    Un interrupteur gris s'affiche à côté des formulaires masqués dans la liste.

  4. Sélectionnez un document pour l'ouvrir.

  5. Sélectionnez Fermer lorsque vous avez terminé.

Corriger les erreurs ou les problèmes en suspens

Si des diagnostics TED en suspens sont signalés, vous ne pourrez pas effectuer la TED tant qu'ils n'auront pas été réglés.

Kit d'entraînement

Sélectionnez le lien de la page Diagnostics dans l'avertissement ci-dessus pour voir les problèmes en suspens de l'engagement. Cliquez sur le texte à côté d'une question pour afficher un lien vers le document approprié.

Sélectionnez le lien pour afficher la ligne du document qui nécessite une attention particulière.

Sélectionnez | Efile pour afficher les problèmes empêchant un dépôt électronique.

Transmettre la déclaration à l'ARC par voie électronique

Une fois les données fiscales importées, le travail de mission terminé et toutes les questions en suspens réglées, vous pouvez commencer à produire votre déclaration électronique par l'intermédiaire du programme TED de l'ARC.

Kit d'entraînement

Sélectionnez LICENCES dans le menu supérieur.

À partir du document AE - Renseignements sur la TED , vous pouvez transmettre votre déclaration à l'ARC en utilisant l'une des deux séries d'informations d'identification - si votre entreprise est un préparateur de déclarations de revenus agréé, vous pouvez saisir vos informations de connexion àl'ARC.

Vous pouvez également saisir un WACode (Web Access Code) fourni par le client. Sélectionnez WACode dans la liste déroulante située à côté de Choose authentication mode.

Si vous avez besoin d'un code d'accès au web et que vous n'en avez pas encore reçu, sélectionnez WAC - Web Access Code Online Request et remplissez le formulaire de demande.

Imprimer le retour

If your client requests a copy of the return for their records, you can create a PDF file which can then be downloaded and shared with the client electronically.

Kit d'entraînement

Sélectionnez Imprimer à partir de n'importe quel document fiscal ouvert pour sélectionner le document à imprimer.
