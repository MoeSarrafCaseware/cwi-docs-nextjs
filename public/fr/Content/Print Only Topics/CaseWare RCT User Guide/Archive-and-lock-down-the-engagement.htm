Archiver et verrouiller la mission

Une fois le travail sur la mission terminé, vous pouvez télécharger une copie de référence de la mission pour l'envoyer au client ou pour vos propres dossiers. La copie de référence est un fichier .zip contenant des copies PDF de tous les documents de votre engagement.

Pour télécharger une copie de référence, sélectionnez Information () | Télécharger la copie de référence.

Vous pouvez également verrouiller le fichier. Sélectionnez le nom de l'entité pour ouvrir la fenêtre contextuelle Propriétés de l'engagement, puis sélectionnez Verrouiller.
