Déclarations fiscales

Caseware ReviewComp vous permet de déposer la déclaration complétée auprès de l'ARC. Pour ce faire, vous devez disposer des informations d'identification appropriées. Vous pouvez également exporter la déclaration au format PDF, puis la partager électroniquement ou en imprimer une copie.

Produire la déclaration de revenus à l'aide de la TED

Une fois les données fiscales importées, le travail de mission terminé et toutes les questions en suspens réglées, vous pouvez commencer à produire votre déclaration électronique par l'intermédiaire du programme TED de l'ARC. Sélectionnez TED dans le menu supérieur pour ouvrir la page d'information sur la TED.

À partir de cette page, vous pouvez produire votre déclaration à l'ARC avec l'une des deux séries d'informations d'identification : Si votre entreprise est un préparateur fiscal agréé, vous pouvez entrer vos informations de connexion à l'ARC. Vous pouvez également saisir un WACode (Web Access Code) fourni par le client.

Si vous avez besoin d'un code d'accès au web et que vous n'en avez pas encore reçu, sélectionnez WAC - Web Access Code Online Request et remplissez le formulaire de demande.

Imprimer le retour

Si votre client demande une copie de la déclaration pour ses dossiers, vous pouvez créer un fichier PDF directement à partir de Caseware ReviewComp. Il peut être sauvegardé sur disque et partagé avec votre client par voie électronique ou imprimé sur papier.

Sélectionnez Imprimer à partir de n'importe quel document fiscal ouvert pour sélectionner le document ou la déclaration que vous souhaitez imprimer.

Déposer une déclaration modifiée

Si vous devez déposer une déclaration modifiée, voici comment procéder.

Pour déposer une déclaration modifiée :

  1. Assurez-vous d'avoir une copie de la déclaration originale que vous avez déjà produite.

    2. Conseil : Pour obtenir une copie PDF de la déclaration que vous avez déposée, sélectionnez Information () | Télécharger la copie de référence dans la barre de navigation principale.

  2. Sur la page Documents , ouvrez le document Tax Optimiser Checklist (TOC).

  3. Passez à la ligne 997 et répondez Oui à la question La société produit-elle une déclaration de revenus modifiée ?

    4. Un lien s'affiche sous votre réponse, vous invitant à vous rendre à la ligne 996.

  4. Sélectionnez le lien Aller au formulaire ligne 996 pour ouvrir le formulaire L966 .

  5. À la ligne 996 , indiquez les raisons de la modification dans le champ de texte prévu à cet effet.

    Vous pouvez également inclure un résumé des modifications à apporter et la date de dépôt.

  6. Apporter les modifications nécessaires aux formulaires ou feuilles de travail respectifs

    7. Conseil : Pour localiser les formulaires et les feuilles de calcul, voir Recherche de formulaires fiscaux et de feuilles de calcul.

  7. Accédez à la page Diagnostics et éliminez tout problème de blocage existant .

  8. Déposer la déclaration modifiée.
