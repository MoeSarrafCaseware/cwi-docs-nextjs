Importer les données fiscales du client

Si votre entreprise a déjà importé les données de la balance de vérification du client dans le cadre d'une mission de compilation ou de révision pour ce client, les données seront déjà disponibles pour être utilisées dans vos formulaires fiscaux. Dans le cas contraire, vous devrez effectuer l'importation vous-même.

Caseware ReviewComp offre une variété d'options pour importer les données des clients. Vous pouvez accéder à toutes les options d'importation à partir de n'importe quel formulaire fiscal en sélectionnant Importation.

Importation de l'IGRF

Sélectionnez cette option si vous disposez d'un fichier au format .gfi contenant les données GIFI du client. Dans votre document fiscal ReviewComp ouvert, sélectionnez Importer | GIFI, puis recherchez le fichier .gfi sur votre ordinateur et sélectionnez Ouvrir.

Importation depuis TaxPrep

Sélectionnez cette option si vous avez un fichier de sortie en format .CSV exporté par TaxPrep. Si vous ne savez pas comment exporter un fichier .CSV à partir de TaxPrep, suivez les instructions ci-dessous.

Dans votre document fiscal ReviewComp ouvert, sélectionnez Importer | TaxPrep CSV et recherchez le fichier exporté. Sélectionnez Ouvrir pour effectuer l'importation.

Importation de l'ARC

Si votre client est inscrit au programme T2 Auto-Fill de l'Agence du revenu du Canada (ARC), vous pouvez importer automatiquement les données fiscales de la mission en vous basant sur les déclarations précédemment déposées par le client.

Les données fournies par l'ARC sont importées dans votre engagement.
