Importer les données de la mission de Caseware Working Papers vers Caseware ReviewComp

Vous pouvez importer les données financières de votre client à partir des Working Papers et les transférer dans votre engagement Caseware ReviewComp. Les étapes à suivre pour effectuer cette tâche diffèrent selon que vous avez désactivé ou non votre licence Working Papers.

Remarque : Les comptes dont le solde est nul ne sont pas automatiquement importés. Si vous souhaitez inclure des comptes à solde nul, vous devez les ajouter manuellement et les affecter aux groupes désignés.

Si vous disposez d'une licence valide pour les documents de travail,

  1. Installer CloudBridge.

  2. Ajoutez le document CloudBridge (CLBG) à votre dossier d'engagement Working Papers.

  3. Transférez les données des Working Papers vers votre engagement Caseware ReviewComp.

Si vous avez désactivé votre licence Working Papers,

  1. Installer CloudBridge.

  2. Ouvrez Working Papers et sélectionnez Continue Without Registering.

  3. Assurez-vous que Working Papers est intégré à votre société Cloud. Pour en savoir plus, voir Working Papers - Cloud Integration.

  4. Reprendre le dossier d'engagement des documents de travail.

    5. Remarque : Si le dossier d'engagement est déjà reporté, ignorez cette étape et passez à l'étape 5.

  5. Dans la fenêtre Parcourir, copiez le chemin d'accès ci-dessous et appuyez sur Entrée pour ouvrir le document CloudBridge (CLBG).

    6. cw:program/scripts/cloudbridge/index.html

  6. Transférez les données des Working Papers vers votre engagement Caseware ReviewComp.
