Importer à partir de QuickBooks Desktop

Vous pouvez importer des données de QuickBooks desktop en les exportant sous la forme d'un fichier .cwq , puis en important ce fichier dans votre engagement.

Remarque : Les comptes dont le solde est nul ne sont pas automatiquement importés. Si vous souhaitez inclure des comptes à solde nul, vous devez les ajouter manuellement et les affecter aux groupes désignés.

Avant de commencer, assurez-vous que

  • Microsoft .NET Framework 4.0 est installé sur votre poste de travail.

  • L'utilitaire d'exportation QuickBooks est installé sur votre poste de travail.

    • Vous pouvez le télécharger à partir de MyCaseware sur la page Support Downloads.

  • Si la protection est activée dans le fichier Working Papers, vous devez avoir l'autorisation d' importer le fichier.

Étape 1 : Exporter des données à partir de QuickBooks

Pour exporter des données à partir de QuickBooks US :

  1. Ouvrez votre compte QuickBooks desktop, puis ouvrez le fichier de l'entreprise que vous souhaitez importer.

  2. Ouvrez l'utilitaire d'exportation QuickBooks.

    1. Dans la liste déroulante Version QuickBooks , sélectionnez QuickBooks 2008-2020 Canada.

    2. dans le champ Chemin d'exportation , sélectionnez Browse....

      3. La boîte de dialogue Browse For Folder (Rechercher un dossier) s'ouvre.

    3. Naviguez jusqu'à l'endroit où vous souhaitez créer le fichier d'exportation, puis sélectionnez OK.

    • Sélectionnez Spécifier la date de la dernière transaction et saisissez la date applicable.

    • Dans la liste déroulante Méthode comptable , sélectionnez la méthode de comptabilité d'exercice applicable.

  3. Sélectionnez Exporter.

    4. Les données comptables du fichier QuickBooks Desktop sont exportées vers un fichier au format .cwq à l'emplacement spécifié.

Étape 2 : Importer les données exportées dans votre engagement

Pour importer les données exportées dans votre engagement :

  1. Téléchargez l'utilitaire d'importation Caseware Cloud.

  2. Ouvrez l'utilitaire, sélectionnez Autre source de données et cliquez sur Suivant.

    3. Remarque : Ne sélectionnez pas l' option Fichier existant car elle n'est pas pertinente pour les utilisateurs du cloud. Pour plus d'informations, consultez la liste des problèmes connus dela plateforme.

  3. Dans le menu déroulantImportFrom , sélectionnez QuickBooks.

  4. Dans la liste déroulante Version , sélectionnez QuickBooks 2019 Canada/UK/AUS.

  5. Dans Chemin d'importation, parcourez et sélectionnez le fichier .cwq .

  6. Spécifiez les paramètres souhaités pour l'importation (fin d'exercice, séquence de dates de la période).

  7. Sélectionnez Suivant, puis Terminer.

  8. Enregistrez le fichier zip généré à l'aide de l'invite affichée.

  9. Dans votre fichier d'engagement, allez à la page Données et sélectionnez Logiciel de comptabilité de bureau.

  10. Sélectionnez ou déposez par glisser-déposer le fichier zip que vous avez enregistré à l'étape 8.

    11. Les données sont importées dans le fichier d'engagement et deviennent disponibles dans la page Données.
