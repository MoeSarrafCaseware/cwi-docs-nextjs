Pré-validation

La pré-validation est un service intégré à l'ARC qui est disponible sur la page TED. Il identifie les incohérences entre les dossiers du préparateur de déclarations et ceux de l'ARC dès le début de la mission afin de réduire le nombre de déclarations soumises contenant des erreurs

Prévalider :

  1. Allez à la page TED .

  2. Saisissez vos données d'identification de l'ARC.

  3. Sélectionnez le bouton Pre-Validate .

  4. Sélectionnez Oui dans la boîte de dialogue de confirmation.

  5. Vous pouvez sélectionner le lien Feuille de travail de pré-validation pour naviguer vers le tableau de travail de pré-validation, où vous pouvez visualiser les données qui seront envoyées à l'ARC avant de poursuivre la pré-validation.

Si la prévalidation est réussie, un message l'indique.

Si la prévalidation échoue, un message d'erreur s'affiche.

Sélectionnez Fermer pour fermer le message d'erreur et accéder à la page Diagnostics , où vous pouvez consulter un résumé des erreurs qui ont causé l'échec de la prévalidation et naviguer jusqu'à l'emplacement de l'erreur pour la corriger. Pour en savoir plus, lisez Corriger les erreurs dans votre déclaration.

Notez que la section Résultats de la dernière transmission dans le diagramme de travail de la prévalidation affiche un enregistrement des tentatives de prévalidation. En cas d'échec, vous pouvez sélectionner la description dans la colonne Erreurs pour accéder à la page où se trouve l'erreur et la corriger.

Une fois que vous avez corrigé toutes les erreurs, vous sélectionnez le bouton Prévalider dans le diagramme de travail pour tenter à nouveau la prévalidation.
