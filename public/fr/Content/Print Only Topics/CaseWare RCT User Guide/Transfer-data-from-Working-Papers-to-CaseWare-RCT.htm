Transférer les données de Caseware Working Papers vers Caseware ReviewComp

Après avoir ajouté le document CLBG à un engagement Working Papers, vous pouvez transférer votre balance de vérification et les données fiscales du grand livre vers Caseware ReviewComp.

Pour exporter vos données d'engagement vers un engagement Cloud à l'aide de CloudBridge :

  1. Dans le gestionnaire de documents, double-cliquez sur CLBG. Le document CLBG s'affiche.

  2. Dans la section Étape 1, choisissez une entité et un engagement appropriés.

  3. Dans la section Étape 2, sélectionnez les données de balance de vérification appropriées à exporter.

  4. Dans la section Étape 2, cochez la case Utiliser un fichier de remappage .

    Le fichier de remappage est utilisé pour transférer la structure de mappage des comptes actuels dans les Working Papers vers votre engagement Cloud. Si votre engagement utilise le mappage par défaut des documents de travail ou a été créé avec le modèle PEG, utilisez le fichier de remappage situé dans le dossier où vous avez installé CloudBridge(C:\NProgram Files (x86)\NCaseware\NScripts\Ncloudbridge).

    Remarque : Si votre mission utilise une structure de mappage personnalisée propre à votre entreprise, créez un fichier de remappage personnalisé , puis ajoutez-le.

  5. Sélectionnez Choisir un fichier pour rechercher et ajouter le fichier de remappage.

  6. (Facultatif) Si vous avez d'autres écritures de journal relatives aux dons, aux frais de repas et de représentation ou à la publicité étrangère, la section Étape 3 est disponible sur la page. La sectionÉtape 3 vous permet de transférer les données GL à la feuille de calcul fiscale appropriée dans l'engagement Caseware ReviewComp.

    Remarque : La sectionÉtape 3 n'est disponible sur la page que si votre dossier d'engagement contient d'autres entrées.

    Dans cet exemple, nous ajouterons les dons aux clients à la feuille de calcul Donations.

  7. Une fois que vous avez terminé, sélectionnez Transférer les données.

Créer un fichier de remappage personnalisé (.ini)

Si votre entreprise utilise une structure de mappage personnalisée, vous devez créer votre propre fichier de remappage (.ini) basé sur la structure de mappage de Caseware ReviewComp.

Vous pouvez maintenant utiliser ce fichier de remappage dans Cloudbridge pour transférer vos données de Working Papers vers Caseware ReviewComp.
