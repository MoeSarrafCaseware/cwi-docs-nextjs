Créer un brouillon

En tant qu'auteur de modèle, il est important que vous connaissiez la version exacte de votre produit OnPoint, car cela peut affecter le type de contenu que vos utilisateurs ont dans leur produit. Si, en tant qu'entreprise, vous rencontrez des problèmes, il est possible qu'ils aient été résolus dans la dernière version. En fait, lorsque vous contactez notre équipe d'assistance, la première chose qu'elle vous suggère est de mettre à jour votre version via le bouton de mise à jour, comme nous l'avons vu précédemment. En utilisant la dernière version, vous réduisez également votre travail en tant qu'auteur du modèle de l'entreprise, car le nouveau contenu peut également avoir résolu les lacunes qui existaient dans le produit.

Dans le projet de modèle, sélectionnez Information () , puis À propos pour afficher des informations sur votre version de OnPoint PCR.

Note à l'attention de l'instructeur : Les tâches suivantes ont pour but de clarifier le flux de travail de l'entreprise. Vous pouvez ou non choisir de montrer ces tâches aux élèves.
