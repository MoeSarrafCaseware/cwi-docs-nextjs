Créer une procédure pour les paramètres de visibilité

Plus tôt dans la formation, nous avons appris que le produit OnPoint n'affiche que les informations relatives à l'engagement sur la base des choix sélectionnés par les utilisateurs. Cela est possible grâce à nos paramètres de visibilité qui ont permis aux principaux auteurs de ce produit, l'AICPA, de créer une logique de visibilité pour des centaines d'éléments. Les auteurs du cabinet peuvent faire de même, mais uniquement pour le contenu qu'ils ont créé, et non pour celui de l'AICPA.

Que sont les paramètres de visibilité ?

Kit d'entraînement

Dans l'exercice précédent, nous avons créé une question sur la différence de cadre. Nous allons maintenant ajouter des paramètres de visibilité à une procédure.

  1. À côté de la procédure Différence de cadre financier , sélectionnez Autres actions () |Ajouter une procédure.

  2. Nommez la nouvelle procédure comme suit :

  3. Sélectionnez Autres actions () | Paramètres à côté de la procédure Détails du cadre financier .

  4. Dans la boîte de dialogue Paramètres de la procédure, cochez la case Remplacer les paramètres de la liste de contrôle .

  5. Sélectionnez +Réponse pour ajouter un autre type de réponse.

  6. Dans le premier champ de texte sous Response Placeholder, entrez "Name, Relationship" et choisissez Manual pour le Response Type.

  7. Dans le deuxième champ de texte sous Response Placeholder, entrez "Financial Framework" et choisissez Manual pour le Response Type.

  8. Sélectionnez Enregistrer.
