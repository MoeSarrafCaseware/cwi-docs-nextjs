Personnaliser les paramètres d'impression dans les domaines des états financiers

Les zones permettent d'organiser le contenu de vos états financiers en regroupant des informations similaires ou connexes. Vous devrez peut-être modifier les paramètres d'impression par défaut pour les zones du bilan et du compte de résultat. Étant donné que les auteurs d'entreprises ne peuvent pas modifier le contenu par défaut fourni avec le produit, nous allons étudier la possibilité de copier des tableaux dynamiques.

Kit d'entraînement

  1. Assurez-vous que l'option Éléments cachés est activée.

  2. Ouvrir le document 3-150 États financiers et rapport.

  3. Sélectionnez Bilan dans la carte du document.

  4. Sélectionnez le tableau, puis sélectionnez Autres actions () | Copier le tableau dynamique.

  5. Sélectionnez l'endroit où vous souhaitez ajouter le tableau copié, puis sélectionnez Ajouter | Coller le tableau dynamique copié.

  6. Select More actions () | Edit Print Settings.

    Les auteurs du cabinet ont la possibilité d'ajouter des en-têtes/pieds de page personnalisés ainsi que la mise en forme des pages de leurs états financiers. Les auteurs des entreprises peuvent choisir d'afficher leurs données financières en format paysage.

  7. Sélectionnez Compte de résultat dans la carte du document, puis sélectionnez tableau. Sélectionnez Autres actions () | Copier le tableau dynamique.

    Il est important de noter ici que vous pouvez continuer à copier et coller autant de tableaux dynamiques par défaut que nécessaire dans les états financiers.

    Sélectionnez Ajouter () | Coller un tableau dynamique copié à l'endroit où vous souhaitez insérer le tableau collé. Sélectionnez Autres actions () | Modifier les paramètres d'impression et réglez la zone du compte de résultat pour qu'elle soit imprimée en format paysage.

Le reste du rapport imprimé le sera en format portrait.
