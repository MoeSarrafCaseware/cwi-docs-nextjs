Modifier une liste de contrôle

Modifions maintenant divers contenus dans le modèle d'entreprise. Dans les prochains chapitres, nous aborderons l'édition d'une liste de contrôle, d'une requête et des états financiers.

Remarque : Nous vous recommandons d'activer la vue détaillée par défaut, car elle vous permettra d'afficher les signatures des documents directement dans le gestionnaire de documents.

Kit d'entraînement

  1. Activer la vue détaillée.

  2. Sélectionnez Autres actions () | Éléments cachés.

    Une fois que vous aurez activé les éléments cachés, vous remarquerez qu'un grand nombre de documents sont apparus. C'est ici que nous devons nous plonger dans la magie qui s'opère dans la partie arrière du produit. Comme nous l'avons expliqué lors de la formation des utilisateurs, OnPoint est construit sur l'idée de la simplicité. Les utilisateurs ne voient apparaître les informations qu'au fur et à mesure qu'ils utilisent le produit et répondent aux questions qui leur sont posées. Cependant, les auteurs de l'entreprise sont en mesure de voir tous les documents, ce qui est le cas par défaut. Il est important de noter que vous ne pouvez pas supprimer les documents par défaut fournis avec le produit. Lorsqu'une nouvelle version est publiée, les documents par défaut contenus dans le produit sont mis à jour.

    Toutefois, si vous remarquez bien, vous verrez que les documents sont accompagnés de curseurs qui vous permettent de désactiver les documents par défaut fournis avec le produit. Il peut s'agir d'une entreprise qui ne souhaite pas utiliser notre système d'interrogation intégré. Il leur suffirait de désactiver les documents d'interrogation dans le produit.

  3. Sélectionnez le document 1-110 Acceptation de l'engagement, puis sélectionnez Modifier ().

    Vous remarquerez que vous pouvez désactiver l'affichage de certaines procédures, mais que vous ne pouvez ni les modifier ni les supprimer.

Dans le document 1-110, vous remarquerez que vous pouvez ajouter des procédures de deux manières. L'une des possibilités consiste à créer une procédure directement au sein d'un groupe ; une autre consiste à créer son propre groupe, puis à créer une procédure. Dans l'exemple ci-dessous, l'optimisation de l'engagement est le groupe. Sélectionnez l'ellipse pour voir l'option d'ajout de procédure.

Une référence faisant autorité renvoie simplement la procédure à tous les liens externes disponibles sur le web. Il s'agit d'éléments tels que les normes comptables et les sites web relatifs à l'industrie.

Kit d'entraînement

  1. Allez dans Parties liées et relations similaires, puis sélectionnez Autres actions () |Ajouter uneprocédure.

    Conseil : Les éléments cachés doivent être activés pour voir cette procédure.

    Une nouvelle procédure apparaît.

    L'icône de l'ampoule se trouve à droite. En la sélectionnant, vous pouvez entrer toute orientation spécifique à la procédure. La flèche Haut/Bas permet de déplacer la procédure au-dessus ou au-dessous de son emplacement actuel.

  2. Sélectionnez Nouvelle inscription.

    Le titre se divise en deux cases.

  3. Dans la case Résumé, tapez

    Différence de cadre financier.

  4. Dans la case Nouvelle procédure, tapez

    L'une ou l'autre des parties liées ou filiales énumérées ci-dessus nécessite-t-elle un cadre financier différent de celui défini ci-dessus dans le cadre d'information financière applicable ?

Dans l'exercice précédent, nous avons créé une nouvelle question. Cependant, nous n'avons pas créé d'espace permettant à un utilisateur de répondre à cette question. C'est précisément ce que nous allons faire dans cet exercice.

Kit d'entraînement

  1. À côté de la procédure Différence de cadre financier , sélectionnez Autres actions ()| Paramètres.

  2. Dans la boîte de dialogue Paramètres de la procédure , cochez la case Remplacer les paramètres de la liste de contrôle .

    Un certain nombre d'options s'affichent.

Autoriser les signatures

Sélectionnez cette option pour ajouter une signature à cette procédure.
Autoriser les notes de saisie dans les procédures

Sélectionnez cette option pour ajouter une zone de texte sous la réponse où les utilisateurs peuvent saisir des notes. La sélection de cette option permet aux auteurs d'ajouter un espace réservé aux notes.
Autoriser les lignes multiples

Sélectionnez cette option pour créer une option Ajouter une ligne au bas de la réponse. Cela permet aux utilisateurs de saisir davantage d'informations pour la procédure.

Remarque : Si votre procédure comporte plus d'une réponse, la sélection de ce paramètre affichera automatiquement vos réponses dans un tableau sous la procédure. Les étiquettes des espaces réservés de la réponse sont affichées dans la ligne d'en-tête.
Afficher les réponses sous la procédure Affiche les réponses en dessous de la procédure.

  1. Sélectionnez l'une des options suivantes :

    À ce stade, nous allons nous concentrer sur la création d'un ensemble de réponses. Examinons quelques-uns des types de réponses disponibles pour les utilisateurs.

Type de réponse Description
Manuel Un champ de texte dans lequel les utilisateurs peuvent saisir une réponse personnalisée.
Liste de sélection Une liste déroulante dans laquelle les utilisateurs peuvent sélectionner une réponse parmi une liste de réponses. Vous pouvez personnaliser la liste des réponses en fonction de vos besoins. Les réponses par défaut sont Oui, Nonet Sans objet.
Multi-Picklist Une boîte déroulante dans laquelle les utilisateurs peuvent sélectionner plusieurs réponses à partir d'une liste de réponses. Les réponses par défaut sont Oui, Nonet Sans objet.
Date Un champ de sélection de date avec une icône de calendrier en ligne où les utilisateurs peuvent sélectionner une date.
Informations sur l'entité Un champ de sélection de date avec une icône de calendrier en ligne où les utilisateurs peuvent sélectionner une date.
Informations sur le contact Un champ qui affiche des informations sur l'un des contacts du client. Les informations de contact sont directement liées au profil utilisateur du client sur CasewareCloud.

  1. Sous Response Placeholder, laissez le texte du Placeholder tel quel.

  2. Pour Type de réponse, choisissez Liste de sélection.

  3. Sélectionnez le bouton Modifier à côté des options pour supprimer l'option "Sans objet".

  4. Sélectionnez Enregistrer. L'écran suivant s'affiche.
