Copier un tableau dans les états financiers

Comme nous l'avons vu précédemment, les utilisateurs ont la possibilité de copier des tableaux dynamiques. Dans cet exercice, nous allons créer notre propre tableau dynamique. Les tableaux dynamiques sont utilisés pour la majeure partie du contenu des états financiers, notamment le bilan, le compte de résultat et l'état des bénéfices non distribués.

  1. Sélectionnez Ajouter () | Nouvelle zone.

    La nouvelle zone est ajoutée à la carte du document et s'affiche sur la page avec un titre par défaut, Nouvelle zone. Vous pouvez sélectionner le titre par défaut pour le modifier.

  2. Dans la nouvelle zone, sélectionnez Ajouter () | Tableau dynamique.

    Un tableau vide s'affiche dans la page.

  3. Sélectionnez le tableau, puis sélectionnez Autres actions () | Modifier le tableau dynamique.

    Le tableau s'ouvre en mode édition.

  4. Sélectionnez Ajouter une ligne (), puis sélectionnez Données de la balance de vérification.

  5. Commencer à taper les ventes ..Lesfrais de vente, généraux et administratifs apparaissent au fur et à mesure de la saisie. Sélectionnez-le pour remplir la ligne.

    Remarquez que les colonnes de l'année précédente et de l'année en cours sont ajoutées par défaut dans le tableau.

  6. Sélectionnez Ajouter une colonne (), puis choisissez Écart en pourcentage pour le Type.

  7. Select the text field at Heading to open the Data Link dialog.

  8. Saisissez Pourcentage d'écart en sortie pour modifier l'en-tête de colonne par défaut.

  9. Sélectionnez FAIT pour fermer l'onglet Calcul.

  10. Sélectionnez Ajouter une colonne (), puis choisissez Entrée pour le Type.

  11. Select the text field at Heading to open the Data Link dialog. Saisissez des commentaires en sortie pour modifier l'en-tête de colonne par défaut, puis sélectionnez Enregistrer.

  12. Choisissez Texte pour le format, et choisissez Moyen pour la largeur.

  13. Sélectionnez FAIT pour fermer l'onglet Calcul.
