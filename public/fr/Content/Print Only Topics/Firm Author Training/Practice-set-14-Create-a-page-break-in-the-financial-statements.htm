Ajouter un saut de page dans les états financiers

Comme pour les produits de bureau Caseware, les sauts de page dans OnPoint sont utilisés pour créer un espace vide entre le contenu des états financiers. Un exemple serait d'insérer un saut de page quelque part dans le document si le compte de résultat présente un solde de l'année en cours.

  1. Sélectionnez Ajouter () | Nouvelle zone.

    La nouvelle zone est ajoutée à la carte du document et s'affiche sur la page avec un titre par défaut, Nouvelle zone. Vous pouvez sélectionner le titre par défaut pour le modifier.

  2. Dans la nouvelle zone, sélectionnez Ajouter () | Saut de page.

  3. Sélectionnez le saut de page, puis sélectionnez Autres actions () | Modifier la visibilité.

  4. Dans les paramètres de visibilité, choisissez d'afficher cette page en fonction d'une condition de groupe financier dans laquelle le compte de résultat présente un solde pour l'année en cours .

  5. Sélectionnez Enregistrer.
