Pratique 15 - Ajouter le papier à en-tête de l'entreprise aux états financiers

Il est courant pour les entreprises d'imprimer leurs états financiers sur du papier à en-tête, car il s'agit du produit final qu'elles fournissent à leurs clients dans le cadre de nombreuses missions.

Firm authors can add a logo or a letterhead in the firm template. Une fois le logo ajouté, il remplace automatiquement les espaces réservés au logo dans les produits livrables. Une fois l'en-tête ajouté, il s'affiche automatiquement dans la copie PDF (copie d'impression) des produits livrables.

Ajoutons un en-tête.

Si vous ouvrez le document 3-150 États financiers et rapport et que vous sélectionnez Imprimer () dans le plan du document, vous trouverez l'en-tête dans la copie d'impression PDF.

Remarque : Assurez-vous que l'option Papier à en-tête est activée dans les paramètres d'impression du document.

Dans cet exemple, nous avons défini dans les options d'impression du document que l'en-tête ne doit apparaître que sur la première page.

Note à l'attention de l'instructeur : Vous pouvez ou non choisir de faire la démonstration de la procédure suivante.
