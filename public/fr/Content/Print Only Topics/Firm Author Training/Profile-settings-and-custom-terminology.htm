Paramètres du profil et terminologie personnalisée

Avant de nous lancer dans la création de cabinets, nous devons remplir quelques rubriques du menu Cloud, telles que votre profil et la terminologie personnalisée relative au marché américain.

Kit d'entraînement

  1. Sélectionnez le menu Cloud () | Paramètres.

  2. Sous Organisation, sélectionnez Profil.

  3. Remplissez le profil, par exemple avec les informations relatives à votre entreprise, puis sélectionnez Enregistrer.

  4. Dans Personnalisation | Terminologie, saisissez l'État dans le champ Province , le code postal dans le champ Code postal et sélectionnez Enregistrer.
