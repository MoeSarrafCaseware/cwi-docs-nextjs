Recueillir des renseignements auprès du client

Dans cette section, nous rassemblerons les informations nécessaires à la mission.

Document d'information

Une mission OnPoint AnalyticsAI comprend un document d'enquête que vous pouvez envoyer à votre client pour qu'il le remplisse.

Kit d'entraînement

Pour accéder au document d'information :

  • Sélectionnez 0-1 Demandes de renseignements des clients (Requête) dans la page Documents .

Vous pouvez déterminer les procédures que vous souhaitez envoyer en utilisant la fonctionnalité "afficher/masquer". Par exemple, sous Téléchargement de fichiers, vous pouvez masquer Télécharger la balance de vérification si vous avez l'intention d'utiliser une méthode différente pour ajouter une balance de vérification à l'engagement.

Vous pouvez également ajouter des commentaires à des procédures individuelles dans le document de requête.

Pour cet exercice, nous laisserons toutes les procédures du document 0-1 Demandes de renseignements des clients avec leur contenu par défaut.

Pour envoyer la requête :

  1. Sélectionnez 0-1 Demandes de renseignements des clients (Requête) dans la page Documents .
  2. En haut à droite de la page, sélectionnez Envoyer.

    La boîte de dialogue Ouvrir une requête s'affiche.

  3. Saisir une date d'échéance de la requête - à utiliser aujourd'hui.
  4. Sélectionnez un destinataire - utilisez votre personne de contact.
  5. Saisissez un message dans le corps de l'e-mail - "Veuillez fournir les informations demandées"
  6. Sélectionnez OK.

    Un courriel est automatiquement envoyé à l'assigné et le statut du document passe à " En attente de réponse".

    L'e-mail de l'assigné contient un lien permettant de consulter les détails de la requête.

    Lorsque le cessionnaire sélectionne le lien, une version du document apparaît. L'attributaire peut répondre aux questions et fournir les fichiers requis, puis sélectionner Répondu pour indiquer que l'élément a été complété.

    Remarque : Le destinataire peut voir tous les commentaires qui ont été saisis.

Annoter la balance générale (page de données)

Vous devez importer une balance de vérification pour votre engagement afin de réconcilier vos données sur la page Complétude des données et de pouvoir utiliser les influenceurs.

Vous pouvez importer une balance de vérification à partir de :

  • Caseware Working Papers
  • Logiciels de comptabilité de bureau
  • Un fichier Excel ou CSV
  • Plateformes en ligne telles que Quickbooks Online et Xero

Si vous importez des données à partir d'un logiciel de comptabilité en ligne, vous pouvez choisir d'importer les données de transaction en même temps que les données de la balance de vérification.

Pour alimenter la balance de vérification, vous devez importer les données financières dans votre fichier d'engagement. Dans certains cas, vous pouvez importer ces données directement à partir d'un autre logiciel de comptabilité en utilisant l'utilitaire d'importation natif (pour les logiciels de comptabilité en ligne) ou un utilitaire d'importation de bureau que nous fournissons.

Kit d'entraînement

Dans cet exemple, nous allons importer une balance de vérification à partir d'un fichier CSV.

Pour importer la balance à partir d'un autre fichier de mission :

  1. Sélectionnez 0-2 Importation de la balance de vérification et des transactions en ligne dans la page Documents .
  2. Sélectionnez Excel ou CSV.
    La boîte de dialogue d'importation s'affiche.
  3. Faites glisser et déposez un fichier CSV dans la zone d'importation située dans la boîte grise en pointillés.
  4. Pour mettre en correspondance les colonnes du fichier d'importation, choisissez dans la liste déroulante le champ qui correspond le mieux aux données. Si vous ne souhaitez pas importer une colonne, sélectionnez Ne pas importer pour l'exclure.

  5. Sélectionnez Importer.
    La page BALANCE DE TRIBUNAL - COMPTES s'affiche avec la balance de vérification importée.

Attribuer des comptes

Lorsque vous importez une balance de vérification, OnPoint mappe autant de comptes que possible, mais dans certains cas, vous devrez peut-être évaluer les comptes et les affecter manuellement.

Kit d'entraînement

En haut de la page, dans le menu Données (à gauche de l'écran), vous remarquerez que l'icône Attribuer des comptes comporte désormais un numéro.

Cela indique qu'un compte n'est actuellement pas affecté à un groupe. Sélectionnez l'icône pour ouvrir la page Affecter des comptes .

Sur cette page, un compte est affiché dans la colonne Non affecté.

Sélectionnez-le pour le déplacer dans la section Comptes sélectionnés. Un certain nombre de suggestions s'affichent. Sélectionnez 2.2.1.1 Comptes créditeurs.

Sélectionnez la catégorie Groupes financiers pour afficher la balance de vérificationgroupée. Select the Documents tabs.

Importer des transactions

Si vous avez importé les données du client à partir d'un fichier CSV, vous devrez importer le fichier CSV des transactions pour stocker et effectuer des analyses sur vos données.

Kit d'entraînement

Pour importer des transactions à partir de Transactions.csv :

  1. Sélectionnez 0-3 Revue analytique dans la page Documents.
  2. Glissez-déposez le fichier Transactions.csv dans la zone de téléchargement de fichiers ou cliquez et naviguez jusqu'à son emplacement.

  3. Pour cet exemple, sélectionnez Virgule ( , ) comme délimiteuret CONTINUEZ.
    Des informations sur le fichier de transactions importé apparaissent dans la section Transactions au bas de la page.

  4. Sélectionner MAP. La page Projets affiche : C'est ici que vous ferez correspondre les champs sources du fichier CSV aux champs cibles correspondant aux données. La page comporte deux sections. La partie supérieure montre les champs sources du CSV et la partie inférieure montre les données de chaque champ.

Vous pouvez enregistrer la mise en page du mappage de données que vous venez de réaliser. Cela vous permet de réutiliser la structure de mappage que vous avez définie initialement dans les réimportations ultérieures de transactions.

Pour enregistrer une disposition de données :

  1. Sélectionnez 0-3 Revue analytique dans la page Documents.
  2. Sélectionnez SAUVEGARDER dans la page Importation de transactions | ACME Inc.
    Cette option n'est disponible que pour les fichiers de données de transaction mappés.
  3. Saisissez un nom de modèle de données dans la boîte de dialogue Enregistrer le modèle de données, par exemple, Modèle d'importation standard.
    Le nom du DATA LAYOUT est mis à jour sur la page Import Transactions .
    Désormais, lorsque vous sélectionnez MAP ou REMAP, une liste déroulante contenant les modèles disponibles s'affiche.
  4. Fermez l'onglet MAP/REMAP.
