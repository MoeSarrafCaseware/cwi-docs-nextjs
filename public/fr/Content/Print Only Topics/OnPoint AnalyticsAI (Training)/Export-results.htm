Exporter les résultats

Lorsque vous avez terminé votre examen analytique, vous pouvez exporter les données d'analyse à titre de preuve d'audit.

Kit d'entraînement

Pour exporter les résultats de l'analyse de risque :

  1. Dans la page Document , sélectionnez le document 2-2 Analyse de risque (risque transactionnel) .
  2. Sélectionnez l'icône de téléchargement () en haut de la page RESULTS et sélectionnez Download.
  3. Choisissez un format de fichier dans la boîte de dialogue Exporter les données localement et sélectionnez Exporter.

Les formats disponibles sont les suivants :

  • Résumé de l'analyse (PDF ) - présente les tests effectués, y compris leur description, les scores de base et les paramètres. Il contient

    • Graphique de la série chronologique - Risque/valeur des transactions

    • Liste des tests effectués

    • Liste des comptes influencés

    • Annexe A : Description des tests effectués

  • Résultats (CSV ) - contient une liste de toutes les transactions triées par score de risque, ainsi que les scores totaux et les scores de test pour chaque ligne de transaction.
  • Toutes les données (CSV ) - contient toutes les informations relatives à chaque transaction et à son score de risque.

Noms des fichiers d'exportation par défaut

Les formats de noms de fichiers par défaut pour les fichiers de données exportés sont les suivants :

Type de fichier exporté Format du nom de fichier
Résumé de l'analyse (PDF) <year>_<month>_<day>_<entity>_AAi Summary.pdf
Résultats (CSV) <year>_<month>_<day>_<entity>_AAi Audit Evidence.csv
Toutes les données (CSV) <year>_<month>_<day>_<entity>_AAi All Data.csv
