Créer les années suivantes OnPoint Collaborate engagement

Lorsque le moment est venu de créer l'engagement OnPoint Collaborate de l'année suivante, vous pouvez reporter l'engagement. Le report est un utilitaire qui permet de gagner du temps et de réutiliser les informations, car vous pouvez accepter toutes les modifications de l'année précédente ou ne travailler que sur les éléments individuels qui ont changé.

Pour reporter la mission :

  1. Dans le menu Cloud (), sélectionnez OnPoint Collaborate pour ouvrir la vue d'engagement.

  2. Sélectionnez votre engagement pour ouvrir le panneau latéral de droite, puis sélectionnez Créer le fichier de la période suivante.

  3. Dans la boîte de dialogue OnPoint Collaborate, choisissez ACME Inc. comme entité et entrez 2020 - ACME Inc - <your initials> comme nom pour le dossier de mission.

  4. Dans le champ Préparateur , saisissez votre prénom pour rechercher votre compte. Sélectionnez votre compte pour vous désigner comme préparateur.

  5. Dans le champ Réviseur , saisissez le nom d'un autre membre du personnel. Sélectionnez le compte pour désigner le membre du personnel comme réviseur.

  6. En bas de la boîte de dialogue OnPoint Collaborate, sélectionnez Enregistrer.

Le nouveau dossier d'engagement s'ouvre.
