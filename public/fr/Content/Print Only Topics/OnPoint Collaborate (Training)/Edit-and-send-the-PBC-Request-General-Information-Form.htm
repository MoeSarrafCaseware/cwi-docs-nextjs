Editer et envoyer le formulaire de demande d'informations générales de la CBP

Une fois la liste de contrôle Optimisation du client complétée, la page Documents comprend l'un des documents de requête suivants :

  • 100-10 Demande PBC - Travail intérimaire sur le terrain

  • 101 Demande de PBC - Informations générales

  • 102 Demande PBC - Taxe

  • 103 Demande PBC - EBP

Une requête est un document intelligent interactif qui se compose des éléments suivants :

  • Questions - Demandes et requêtes auxquelles les clients doivent répondre.

  • Ensembles de questions - Pour regrouper des questions liées entre elles.

  • Types de réponse - Pour spécifier la réponse attendue de l'utilisateur.

Sur la base de nos réponses à la liste de contrôle du chapitre précédent, le document 101 PBC Request - General Information query est disponible sur la page Documents. Dans ce chapitre, nous verrons comment modifier et envoyer une requête.

101 Demande de PBC - Informations générales

Avant d'envoyer la requête, vous pouvez y apporter des modifications telles que l'ajout d'instructions, de séries de questions et de questions. Aujourd'hui, nous allons ajouter des ensembles de questions et des questions, mais vous pouvez en savoir plus sur la personnalisation des requêtes sur docs.caseware.com.

Ensembles de questions :

Les ensembles de questions sont utilisés pour regrouper des questions connexes. Toutes les questions doivent appartenir à un ensemble de questions.

Pour ajouter un ensemble de questions, sélectionnez Ajouter () dans le panneau de navigation de gauche.

Un nouvel ensemble de questions est ajouté au document avec un titre par défaut Nouveau groupe.

Vous pouvez sélectionner le titre par défaut pour le modifier. Vous pouvez supprimer un ensemble de questions en sélectionnant l'icône Supprimer ().

Remarque : Si vous supprimez un ensemble de questions, toutes les questions de cet ensemble sont également supprimées.

Ajouter une question de requête

Après avoir créé un ensemble de questions, vous pouvez y ajouter des questions.

Pour ajouter une question de requête :

  1. Accédez à l'ensemble de questions dans lequel vous souhaitez ajouter une question.

  2. Select Add () and add one or more of the following response types:

    • Un champ de texte dans lequel les utilisateurs peuvent saisir une réponse textuelle.

    • Une liste déroulante dans laquelle les utilisateurs peuvent sélectionner une réponse parmi une liste de réponses. Vous pouvez personnaliser la liste des réponses en fonction de vos besoins. Les réponses par défaut sont Oui, Nonet Sans objet.

    • Une boîte déroulante dans laquelle les utilisateurs peuvent sélectionner plusieurs réponses à partir d'une liste de réponses. Les réponses par défaut sont Oui, Non et Sans objet.

    • Une boîte de téléchargement de fichiers où les utilisateurs peuvent parcourir et ajouter un fichier ou glisser-déposer le fichier.

    • Un champ de sélection de date avec une icône de calendrier en ligne où les utilisateurs peuvent sélectionner une date.

    Une nouvelle question est ajoutée au document avec un corps de texte par défaut Nouvelle question.

  3. Sélectionnez le texte du corps par défaut pour le modifier. Une fois le champ de texte sélectionné, la barre d'outils de formatage et les boutons Annuler/Ok () s'affichent.

    Pendant que vous tapez, vous pouvez sélectionner Annuler () pour effacer le champ de texte et recommencer. Le texte que vous saisissez est automatiquement sauvegardé, mais vous pouvez également utiliser le bouton Ok () pour sauvegarder.

Vous pouvez également sélectionner le champ Résumé au-dessus du texte pour saisir une brève description de la question. Une fois que vous avez saisi un résumé pour la question, il s'affiche par défaut dans le panneau de navigation de gauche en tant que localisateur de la question. Si le champ Résumé est vide, le corps du texte de la question s'affiche dans le panneau de gauche.

Remarque : Nous vous recommandons de toujours saisir un résumé pour les questions afin de les localiser facilement dans le panneau de navigation de gauche.

  1. Si vous avez ajouté un type de réponse File Upload, vous pouvez définir quelques options supplémentaires :

    1. Sélectionnez le bouton () en bas de la boîte de téléchargement.

    2. Dans la boîte de dialogue Paramètres de réponse au fichier, sélectionnez un dossier cible ou un emplacement fictif, un identifiant et un nom pour le fichier téléchargé.

  2. Une fois que vous avez terminé, sélectionnez Enregistrer.

Envoyer une requête

Lorsque vous avez terminé de modifier la requête, vous pouvez l'envoyer à votre client.

Pour envoyer le formulaire de demande 101 PBC - Informations générales à un client :

  1. Ouvrez le document d'interrogation, puis sélectionnez ENVOYER.

  2. Saisissez les informations suivantes dans la boîte de dialogue Ouvrir une requête :

    1. Saisissez la date du jour dans le champ Date d'échéance de la requête.

    2. Sélectionnez un contact client dans la liste déroulante Attributaires .

    3. Saisissez "Veuillez télécharger les éléments demandés" dans la zone de texte du corps de l'e-mail.

  3. Sélectionnez OK pour envoyer la requête.

Un courriel de notification est envoyé aux contacts désignés.
