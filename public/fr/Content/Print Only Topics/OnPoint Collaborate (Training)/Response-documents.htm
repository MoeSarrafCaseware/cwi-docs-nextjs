Documents de réponse

Lorsque vous acceptez une réponse, le document est ajouté aux dossiers prédéfinis désignés dans la page Documents.

Autres options

Dans le menu Autres actions () de chaque réponse, vous pouvez effectuer les actions suivantes :

  • Sélectionnez Déplacer pour déplacer le document de réponse dans un autre dossier.

  • Sélectionnez Modifier pour modifier l' identifiant, le nomet les paramètres de visibilité du document.

  • Sélectionnez Oui pour supprimer la question.

  • Sélectionnez Enregistrer dans le nuage pour enregistrer le document dans la section Fichiers de votre site Cloud.

    Ouvrir les documents de réponse dans un fichier Working Papers

    Si votre site Cloud est intégré à Working Papers, vous pouvez ouvrir le document de réponse dans Working Papers.

    Pour intégrer Working Papers à un site Cloud :

    1. Caseware Working Papers Si vous avez des fichiers ouverts, veillez à fermer votre fichier Working Papers actuel(Fichier | Fermer).

    2. Sur le ruban, cliquez sur Fichier | Options | Caseware Cloud.

    3. Sélectionnez Intégrer avec Caseware Cloud. In the Server field, enter the URL of your organization's Caseware Cloud site. Il n'est pas nécessaire d'inclure le protocole web(http:// ou https://). Cliquez sur OK.

    4. Une invite de connexion à Caseware Cloud s'affiche. Saisissez votre adresse électronique et votre mot de passe, puis cliquez sur Se connecter.

    Après avoir intégré votre fichier Working Papers à une entité Cloud, vous pouvez le lier à un fichier Queries pour commencer à collaborer avec vos clients.

    Pour établir un lien avec un fichier Queries :

    1. Dans Working Papers, ouvrez un fichier synchronisé avec une entité cliente dans Cloud.

    2. Sur le ruban, cliquez sur Cloud | Upload. La boîte de dialogue Queries Setup s'affiche.

    3. Le cas échéant, sélectionnez le produit Queries que vous souhaitez utiliser.

    4. Sélectionnez Sélectionner existant et cliquez sur le menu déroulant pour sélectionner le fichier Queries. Cliquez sur OK.

    L'engagement est lié au fichier Queries et le volet Cloud s'affiche. Vous pouvez désormais glisser-déposer vos documents de réponse dans le gestionnaire de documents de travail.

    Pour lier un document de requête au gestionnaire de documents :

    1. Assurez-vous que le Gestionnaire de documents et le volet Requêtes sont ouverts.

    2. Faites glisser le document que vous souhaitez ajouter depuis le volet Requêtes vers le gestionnaire de documents.
