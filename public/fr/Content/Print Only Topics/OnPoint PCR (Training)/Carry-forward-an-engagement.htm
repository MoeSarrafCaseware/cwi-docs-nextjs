Reporter une mission

Le report est un outil qui permet de gagner du temps. Vous pouvez également choisir d'accepter toutes les modifications de l'année dernière ou de ne travailler que sur les éléments individuels qui ont été modifiés.

En ce qui concerne les rapports intermédiaires, chaque période intermédiaire doit faire l'objet d'un fichier distinct. Cependant, le dossier d'engagement est basé sur la période précédente.

Pour les missions annuelles, les balances de vérification ajustées de l'exercice précédent seront reportées dans la colonne AP des documents, y compris les états financiers.

Vous avez la possibilité d'accepter ou de refuser les modifications de la période précédente dans le nouveau fichier. Vous pouvez également consulter les procédures qui ont été ajoutées ou modifiées par l'auteur du cabinet entre les versions. Par exemple, si vous commencez à travailler sur un fichier du premier trimestre et que le contenu est modifié par l'auteur de l'entreprise, lorsque vous créez votre fichier du deuxième trimestre, vous pouvez voir les nouvelles modifications apportées par les auteurs de l'entreprise.

Reprenons maintenant le dossier pour la période suivante.

Kit d'entraînement

  1. Ouvrez le nouvel engagement (créé à partir du report).

  2. Select the 1-110 Engagement Setup document from the Documents page.

    Vous avez maintenant la possibilité d'accepter ou de rejeter les réponses reportées.

  3. Accepter toutes les réponses. Une fois l'opération terminée, revenez à la page Documents

  4. Sélectionnez le document 1-125 Review Engagement Quality Acceptance and Conclusion.

    Vous pourrez à nouveau accepter ou rejeter les réponses reportées.
