Imprimer le document des états financiers

Pour conclure votre mission, vous créez le document relatif aux états financiers pour le client, dans lequel vous expliquez vos constatations et présentez une conclusion. OnPoint PCR dispose d'un modèle intégré qui facilite la préparation des états financiers.

Kit d'entraînement

  1. Sur la page Documents, dans la section États financiers et synthèse , ouvrez le document 3-150 États financiers et rapport.

  2. Naviguez jusqu'à la zone Rapport d'examen du comptable indépendant .

  3. Sélectionnez le bouton d'orientation () pour afficher une section d'orientation détaillée.

  4. Sélectionnez le lien Matrice des éléments requis du rapport d'examen dans le guide pour ouvrir une nouvelle page dans votre navigateur. Le rapport se compose d'un tableau qui énumère chaque section des états financiers et indique :

    • quand chaque section est-elle incluse dans le rapport

    • les éléments requis dans chaque section

    • les références AR-C pour chaque section

Le document contient également diverses sections d'orientation qui vous aideront à remplir certaines des sections requises.

Chaque document relatif aux états financiers comprend les sections suivantes :

  • Page de couverture

  • Table des matières

  • Rapport révisé de l'expert-comptable indépendant

  • Bilan

  • Aperçus des états financiers

    • État du résultat global

    • État des variations des capitaux propres

    • État des flux de trésorerie

  • Notes sur les états financiers

La section " Notes aux états financiers " est la dernière section du document des états financiers. Si vous devez fournir des informations supplémentaires sur les résultats de votre mission, vous pouvez les ajouter dans cette section.

Une note peut comporter une section d'orientation contenant des informations sur les modifications que vous pourriez être amené à y apporter en tant que préparateur. Le contenu d'une note peut être

  • section texte - il s'agit d'un champ de texte dans lequel vous pouvez ajouter des informations supplémentaires sous la forme d'un ou de plusieurs paragraphes.

  • tableau dynamique - il s'agit d'un tableau composé de lignes sélectionnées dans la balance générale des comptes ou de lignes personnalisées dans lesquelles vous pouvez saisir vos propres données pour étayer vos conclusions.

  • saut de page - pour insérer un saut de page dans le contenu de la note.

Personnaliser la mise en page des états financiers

La plupart des informations contenues dans le document des états financiers sont déjà remplies pour vous. Le contenu du rapport d'examen est basé sur vos réponses aux questions du document 1-110 Préparation de la mission. Les groupes et les comptes figurant dans les tableaux des états financiers proviennent de la balance de vérification.

Toutefois, il se peut que vous deviez personnaliser le document relatif aux états financiers pour votre client. Dans cette session de formation, nous allons apporter les modifications suivantes au document des états financiers.

Exercice 1 - Créer une note

Créons une note dans la zone " Notes sur les états financiers" .

La note doit s'intituler "BIENS ET ÉQUIPEMENTS" et présenter un tableau dynamique avec les soldes de l'exercice en cours et de l'exercice précédent pour les machines et équipements, le mobilier et les agencements, et les améliorations locatives.

Créer une note dans les états financiers

  1. Sélectionnez la zone Notes aux états financiers dans la carte du document.

  2. Sélectionnez (BLANC) et entrez PROPRIÉTÉ ET ÉQUIPEMENT comme nouveau titre pour votre note.

Les nouvelles notes sont vides par défaut, vous devez donc ajouter des zones de contenu à la note pour y inclure des informations.

Ajoutez une zone de texte à votre note pour présenter le tableau dynamique :

  1. Sous la note, sélectionnez Ajouter () | Section de texte.

  2. Dans le champ Équation, entrez les éléments suivants :

    Au 30 juin 2017, les immobilisations corporelles se composaient de :

Vous avez ajouté une description du contenu de la nouvelle note.

Pour ajouter les informations relatives au solde des groupes de comptes de biens et d'équipements, vous devez ajouter un tableau dynamique.

  1. Dans la section texte, sélectionnez Ajouter () | Tableau dynamique.

  2. Sélectionnez le tableau dynamique pour afficher l'icône Autres actions ().

  3. Sélectionnez Autres actions () | Modifier le tableau dynamique.

    Le tableau s'ouvre en mode édition.

  4. Saisissez la propriété du mot pour filtrer la liste. Sélectionnez le groupe Immobilisations corporelles (avec le solde de 45 000) pour l'ajouter à la ligne.

  5. À la fin de la ligne, sélectionnez l'icône Développer les lignes ().

  6. En haut de la page, sélectionnez SAVE & CLOSE.

Remarque : Dans le mode d'édition d'un tableau dynamique, vous pouvez annuler ou rétablir des actions, afficher les éléments exclus, les groupes développés ou réduits ou activer la sauvegarde automatique.

Vous avez ajouté un tableau dynamique à votre note. Lorsque vous faites défiler la page jusqu'au bas du document des états financiers, vous devez voir la note avec son nouveau contenu.

Travaux pratiques 2 - Renvoi d'une note à une ligne dans un tableau d'états financiers

Vous pouvez renvoyer une ligne d'un tableau à l'une des notes des états financiers qui contient des détails ou des informations connexes.

Par exemple, vous souhaitez fournir une ventilation du groupe Immobilisations corpore lles, mais vous ne voulez pas l'afficher dans le bilan. Vous reliez donc une ligne du bilan à la note IMMOBILIER ET ÉQUIPEMENT que vous avez créée précédemment.

Pour afficher la colonne Notes dans le tableau des états financiers :

  1. Sélectionnez la zone Bilan dans la carte du document.

  2. Sélectionnez le tableau, puis sélectionnez Autres actions () | Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Sélectionnez Ajouter () au-dessus de la colonne vide.

  4. Sélectionnez Référence de la note dans la liste déroulante Type , puis sélectionnez ENREGISTRER ET FERMER.

Une colonne Note apparaît désormais dans le bilan.

À la ligne Immobilisations corporelles, nettes, sélectionnez le # pour afficher une fenêtre contextuelle avec le nom de la note aux états financiers que vous venez de créer. Cochez la case située à côté de 1. PROPRIÉTÉ ET ÉQUIPEMENT pour relier la ligne à la note.

Sélectionnez le groupe Biens, installations et équipements , qui contient désormais une référence à la note dans la colonne Note.

Travaux pratiques 3 - Remplacer le texte de remplacement

Le rapport d'examen comprend des espaces réservés pour les informations que vous devez ajouter. Les espaces réservés au texte à saisir sont affichés en caractères bleus surlignés. Il existe également des espaces réservés sous la forme de menus déroulants permettant de sélectionner une date, un membre du personnel ou un nom de contact.

Sélectionnez un espace réservé pour le remplacer par un contenu réel adapté à votre client et à l'engagement. Sélectionnez également des valeurs pour toutes les listes déroulantes.

Travaux pratiques 4 - Modifier un tableau d'états financiers

Bien que vous ne puissiez pas modifier les comptes et les groupes spécifiques qui sont inclus dans un état financier, vous pouvez apporter certaines modifications au contenu affiché.

La plupart des tableaux des états financiers sont remplis automatiquement à partir des données de la balance de vérification, afin de faciliter la préparation des états financiers. Toutefois, vous pouvez ajouter des informations supplémentaires au tableau

Vous pouvez ajouter des lignes dans un tableau d'états financiers en ajoutant des lignes d'entrée. Les montants que vous saisissez dans les lignes de saisie sont inclus dans les calculs finaux de la table.

Par exemple, faites défiler jusqu'à la zone du tableau des flux de trésorerie dans le plan du document.

Remarque: Le tableau des flux de trésorerie n'est disponible que si vous avez répondu Méthode indirecte à la question Quelle méthode sera utilisée pour déclarer les flux de trésorerie ? dans le document 1-110 Configuration de la mission.

Les lignes d'entrée sans valeur sont affichées avec des colonnes vides. Dès que vous saisissez des données financières dans une ligne de saisie, cette ligne est affichée dans l'état des flux de trésorerie.

Remarque: Si vous souhaitez saisir un montant différent dans une ligne qui comporte déjà des valeurs automatiques, sélectionnez les options () au bout de la ligne, puis sélectionnez Remplacer les soldes calculés.

Pour ajouter ou modifier un calcul dans les cellules d'un tableau :

Dans le fichier 3-150 États financiers et rapports , vous pouvez ajouter ou modifier des calculs dans les cellules des tableaux. Le calcul peut inclure des chiffres, des données provenant de la balance de vérification, des données provenant des états financiers, des valeurs de formulaire (si des formulaires sont disponibles dans votre produit) ou une combinaison de tous ces éléments.

Remarque : Cette fonction n'est disponible que pour les cellules des colonnes ou des lignes de type Entrée.

Travaux pratiques 6 - Développer une ligne dans les états financiers

Dans la plupart des tableaux des états financiers, les comptes (lignes) sont regroupés par défaut dans leurs groupes respectifs. Vous pouvez développer les lignes de l'état financier pour afficher plus de détails sur certains domaines du dossier d'engagement.

Par exemple, allez au bilan dans les états financiers.

Lesactifs non courants affichent des totaux pour deux domaines : Immobilisations corporelles, nettes , et investissements à long terme, nets. Vous souhaitez maintenant donner au client plus d'informations sur le groupe Immobilisations corporelles nettes . Sélectionnez l'icône Développer/masquer les lignes () à côté du nom du groupe.

Toutes les lignes du groupe sélectionné sont maintenant affichées et vous pouvez voir les montants pour chaque catégorie différente sous Immobilisations corporelles, ainsi que le montant de l' amortissement cumulé, de l'amortissement pour dépréciation et de l'amortissement.

Toutes les lignes du groupe sélectionné sont maintenant affichées et vous pouvez voir les montants des rubriques Automobiles - coût, Immobilisations corporelles et Amortissements cumulés. Sélectionnez l'icône Développer/masquer les lignes () à côté du sous-groupe Immobilisations corporelles pour afficher toutes les lignes de détail qui composent le groupe.

Pour réduire un sous-groupe, sélectionnez à nouveau l'icône Développer/masquer les lignes ().

Remarque: Vous pouvez également consulter le détail d'un solde dans l'état financier en sélectionnant la ligne. Une fenêtre contextuelle s'affiche et présente les détails de l'exercice préliminaire, des ajustements, de l'exercice final et de l'exercice précédent pour ce solde.

Vous pouvez modifier les descriptions des lignes des états financiers. Pour modifier la description des lignes, sélectionnez la ligne et modifiez la description.

Notez que les lignes provenant des données de la balance de vérification affichent l'icône Revert ( ) lorsque vous modifiez la description. Vous pouvez sélectionner Revert ( ) pour annuler vos modifications et afficher la description originale.

Imprimer des copies de documents

Une fois que vous avez terminé la préparation des états financiers du client et que votre réviseur a signé le document, vous pouvez imprimer une copie des états financiers pour les présenter au client. Vous pouvez également imprimer d'autres copies du document pendant que vous travaillez dessus afin de vérifier l'impact de vos modifications sur le produit fini.

Vous ne pouvez pas imprimer directement à partir de OnPoint PCR car l'action d'impression crée une copie PDF du fichier que vous pouvez imprimer ou partager selon vos besoins. Pour ajuster l'apparence de l'état financier une fois converti, modifiez les paramètres d'impression.

Travaux pratiques 1 - Imprimer les états financiers

Select Settings () | Document Print Settings to display the Print Settings dialog. Vous pouvez également sélectionner Paramètres d'impression des notes pour personnaliser les options d'impression des notes de la zone des états financiers.

Vous pouvez saisir du contenu pour l'en-tête et le pied de page de votre document, et vous pouvez également définir la taille et l'orientation de la page, la largeur des marges, les options d'en-tête et les paramètres de la table des matières. Si vous apportez des modifications, sélectionnez Enregistrer pour fermer la boîte de dialogue.

Sélectionnez Imprimer() pour générer une copie PDF des états financiers. Cette opération ouvre automatiquement un nouvel onglet dans la fenêtre de votre navigateur, et la version PDF du document s'affiche. Vous pouvez choisir d'enregistrer le fichier au format PDF sur votre disque dur local, ou vous pouvez sélectionner le bouton Modifier et trouver une imprimante locale pour imprimer une copie physique.

Pratique 2 - Imprimer des documents groupés

Tous les documents composant la liste de contrôle pour la présentation et la divulgation peuvent désormais être imprimés en tant que groupe, plutôt qu'en tant que documents individuels.

Lancement 3-305 Optimiseur de liste de contrôle pour la présentation et la divulgation.

Select the Print () | Print all documents in group: Liste de contrôle pour la présentation et la divulgation pour produire un document PDF avec tous les documents regroupés et l'imprimer.

Sélectionnez Imprimer () | Imprimer ce document uniquement pour imprimer uniquement le document en cours.

Documenter les problèmes

Vous pouvez ajouter des questions à un document spécifique, à un élément spécifique d'un document ou globalement (pour l'ensemble de la mission).

Kit d'entraînement

Vous pouvez également étiqueter des utilisateurs spécifiques pour leur attribuer le problème. Dans la barre d'outils de formatage, sélectionnez @ et choisissez un utilisateur dans la liste.

Note à l'attention de l'instructeur : Vous pouvez ou non démontrer comment gérer les problèmes.

Si l'on vous attribue un problème, vous devez l'aborder en le résolvant.

Après avoir résolu un problème, ou si le problème n'est plus nécessaire, vous pouvez l'effacer à partir du menu des problèmes. L'effacement de la question supprime la notification sur l'icône des questions, tout en conservant l'historique de la question dans le dossier d'engagement.

Il est recommandé d'effacer une question si vous souhaitez conserver l'historique de la question. Si vous supprimez une question, elle est supprimée du dossier de mission et du rapport sur les questions.

Add supporting documents

Vous pouvez télécharger des fichiers pour inclure des documents justificatifs dans l'engagement.

Kit d'entraînement

Allez à la phase de travail sur le terrain dans la page Documents et sélectionnez Ajouter () | Télécharger. Recherchez le fichier à inclure dans l'engagement (l'instructeur fournira le fichier pour cet exercice).
