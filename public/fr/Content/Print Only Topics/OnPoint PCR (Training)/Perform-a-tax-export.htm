Effectuer une exportation fiscale

L'AICPA a mis en place un flux de travail plus complet pour s'assurer que les utilisateurs puissent transférer leurs données de Caseware Cloud OnPoint vers le logiciel fiscal de leur choix.

Kit d'entraînement

Pour créer un fichier d'exportation fiscale, ouvrez le document 4-100 Exportation fiscale.

Sélectionnez la liste déroulante Affichage pour choisir l'option d'affichage du document :

  • Show IDs - Permet de visualiser les lignes du logiciel fiscal vers lesquelles les informations de Caseware seront transférées.

  • Largeur dynamique du formulaire - Le formulaire s'adapte automatiquement à la largeur de la page du navigateur.

Le panneau gauche du formulaire affiche le type d'entité, le fournisseuret l' année d'imposition. D'autres options seront disponibles dans les prochaines versions.

Le panneau latéral droit affiche plusieurs formulaires fiscaux. Il est recommandé de sélectionner le formulaire fiscal applicable et de l'ouvrir avant de lancer l'exportation, de manière à ce que vous puissiez le consulter :

  • confirmer que les numéros ont été correctement transférés du produit vers les formulaires.

  • réduire le temps d'exportation (puisque l'information a été remplie et est déjà disponible dans les formulaires).

Lançons maintenant la déclaration d'impôt sur le revenu 1120.

Vous pouvez affecter un ou plusieurs comptes à un seul code fiscal. Sélectionnez le signe plus et choisissez un compte à affecter à un code fiscal unique. Pour affecter plus d'un compte à un code fiscal, ajoutez une virgule puis sélectionnez un autre compte.

Vous constaterez que l'application ajoute les soldes des comptes pour chaque code fiscal. La cellule correspondante de la colonne des totaux est mise à jour avec le montant total.

Une fois que vous êtes satisfait des données présentées dans les formulaires applicables, revenez au document 4-100 pour lancer l'exportation.

Sélectionnez Exporter, puis indiquez où le fichier exporté sera enregistré. Dans certains navigateurs, le fichier est automatiquement enregistré dans le dossier Téléchargements.

Si vous avez besoin d'aide pour importer le fichier fiscal dans votre logiciel fiscal, veuillez contacter le centre d'assistance du fournisseur de services fiscaux concerné.
