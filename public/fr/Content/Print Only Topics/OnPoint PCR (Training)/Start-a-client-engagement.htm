Start a client engagement

Au début d'une nouvelle mission pour un client, vous devez créer un nouveau fichier de mission OnPoint PCR. Les fichiers d'engagement sont créés et hébergés sur Caseware Cloud (pour plus d'informations sur Caseware Cloud, visitez https://www.casewarecloud.com).

Cette section couvre les tâches suivantes :

  • Créer le fichier de mission

  • Mise en place des propriétés d'engagement

Créer un fichier de mission

Pour commencer à travailler sur une mission, vous devez créer un dossier de mission. Dans Caseware Cloud, vous pouvez créer votre dossier de mission dans l'entité client. You can also set key dates and budgets and assign staff members to the file.

Kit d'entraînement

  1. Dans le menu Cloud (), sélectionnez OnPoint PCR.

  2. Sélectionnez Nouveau.

  3. Dans la boîte de dialogue suivante, choisissez votre entité cliente et saisissez le nom du dossier de mission. Vous pouvez également sélectionner la date de début et la date de fin de votre engagement et saisir les informations relatives au budget.

  4. Dans le champ Préparateur , saisissez votre prénom pour rechercher votre compte. Sélectionnez votre compte pour vous désigner comme préparateur.

  5. Dans le champ Réviseur , saisissez le nom d'un autre membre du personnel. Sélectionnez le compte auquel le personnel doit être affecté en tant que réviseur.

  6. En bas de la boîte de dialogue PCR OnPoint, sélectionnez Enregistrer.

Vous avez créé votre nouvelle mission et vous avez chargé des membres du cabinet de préparer la mission et de la revoir. Lorsque vous sélectionnez Enregistrer, le fichier d'engagement est automatiquement ouvert dans votre navigateur.

Mise en place des propriétés d'engagement

Lorsque vous ouvrez un fichier de mission pour la première fois, une fenêtre contextuelle contenant des informations sur le fichier de mission s'affiche. Vous pouvez ici définir certaines propriétés de haut niveau de l'engagement et effectuer certaines actions.

Voici ce que vous pouvez faire dans cette vue :

  • Définir l'étape du flux de travail - vous pouvez sélectionner une étape du flux de travail avec un code couleur pour indiquer l'état de votre travail d'engagement.

  • Saisir les détails de l'engagement - vous pouvez saisir un numéro de travail, modifier le nom del' opérationet sélectionner une autre date de fin d'année. Pour cet exercice, laissez la période de rapport sur Annuel .

  • Marquer un deuxième réviseur - vous pouvez activer une deuxième série de signatures de révision pour les documents de la mission. Normalement, vous ne le ferez qu'une fois la première série d'examens terminée.

  • Verrouiller le dossier - vous pouvez verrouiller le dossier de mission une fois que tout le travail de mission est terminé.

  • Consolidation - vous pouvez consolider plusieurs entités détenues à 100 %.
