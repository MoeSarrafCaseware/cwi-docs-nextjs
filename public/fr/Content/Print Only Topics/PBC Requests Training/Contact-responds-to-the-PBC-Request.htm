Les clients répondent à la demande de la PBC

Une fois que vous avez partagé le document, vos contacts peuvent voir la requête dans le volet Mon travail lorsqu'ils se connectent à Cloud. Ils peuvent sélectionner le document pour l'ouvrir et fournir des réponses. Dans ce chapitre, nous allons voir comment un client répond à une requête.

Ouvrez le lien que vous avez reçu dans un autre navigateur, tel que Firefox ou Microsoft Edge, et connectez-vous à Cloud en tant que personne de contact. La requête que vous avez envoyée est affichée dans le volet Mon travail.

Sélectionnez un risque pour l'ouvrir.

Vous pouvez maintenant répondre aux questions de la requête et glisser-déposer les documents dans les zones de réponse appropriées pour répondre à la requête.

Marquez les questions auxquelles vous avez répondu en sélectionnant l'icône Répondre ().

Once you mark a question as responded, the icon changes to the blue circle ().
