Modifier et envoyer la demande de PBC

Dans ce chapitre, nous allons examiner la demande PBC de révision des enregistrements et des documents, et nous allons également apprendre à créer et à envoyer une demande PBC, ou un document de requête, à des contacts.

Une requête est un document intelligent interactif qui se compose des éléments suivants :

  • Questions - Demandes et requêtes auxquelles les clients doivent répondre.

  • Ensembles de questions - Pour regrouper des questions liées entre elles.

  • Types de réponse - Pour spécifier la réponse attendue de l'utilisateur.

Sur la base de nos réponses à la liste de contrôle du chapitre précédent, le document de requête Examiner les enregistrements et les documents est disponible dans la page Documents.

Demander des dossiers et des documents

Jetez un coup d'œil aux questions incluses dans ce document d'interrogation. Les questions qui sont incluses dans la requête sont basées sur les réponses que vous avez fournies dans l'Optimiseur.

Avant d'envoyer la requête, vous pouvez y apporter des modifications telles que l'ajout d'instructions, de séries de questions et de questions. Aujourd'hui, au lieu de modifier une requête, nous allons ajouter une requête personnalisée. Pour en savoir plus sur la personnalisation des requêtes, consultez docs.caseware.com.

Create a new Query Document

Bien que vous puissiez utiliser les requêtes par défaut qui sont définies pour l'engagement en fonction de vos réponses dans la liste de contrôle Optimiser, vous avez également la possibilité de créer votre propre requête.

Tout d'abord, sélectionnez Documents dans le menu supérieur pour revenir à la page Documents .

Dans la phase de révision, à côté du dossier Planifier la mission, sélectionnez Ajouter (), puis choisissez Interroger.

La page Créer une requête affiche les modèles de requête disponibles. Nous allons utiliser une requête vide.

Sous Requête vierge, saisissez "Ma requête" dans le champ Nom et sélectionnez Créer une requête.

La requête s'ouvre.

Ensembles de questions :

Les ensembles de questions sont utilisés pour regrouper des questions connexes. Toutes les questions doivent appartenir à un ensemble de questions.

Pour ajouter un ensemble de questions, sélectionnez Ajouter () dans le panneau de navigation de gauche.

Un nouvel ensemble de questions est ajouté au document avec un titre par défaut Nouveau groupe.

Renommez l'ensemble de questions en "Exemples de demandes de formation PBC".

Ajouter une question de requête

Après avoir créé un ensemble de questions, vous pouvez y ajouter des questions d'interrogation. Nous allons ajouter deux questions à notre jeu de questions.

Ajouter une question de requête

  1. Déplacez votre curseur vers le bas jusqu'à ce que vous voyiez l'icône bleue Ajouter () et la ligne.

  2. Sélectionnez Ajouter () et choisissez le type de réponse Picklist . Une liste déroulante dans laquelle les utilisateurs peuvent sélectionner une réponse parmi une liste de réponses.

    Une nouvelle question est ajoutée au document avec un corps de texte par défaut Nouvelle question.

  3. Saisissez "Immobilisations corporelles" dans le champ " Résumé" .

    Une fois que vous avez saisi un résumé pour la question, il s'affiche par défaut dans le panneau de navigation de gauche en tant que localisateur de la question. Si le champ Résumé est vide, le corps du texte de la question s'affiche dans le panneau de gauche.

    Remarque : Nous vous recommandons de toujours saisir un résumé pour les questions afin de les localiser facilement dans le panneau de navigation de gauche.

  4. Sélectionnez le texte du corps par défaut pour le modifier. Saisir "Y a-t-il eu des achats d'immobilisations corporelles au cours de l'exercice ?"

    Une fois le champ de texte sélectionné, la barre d'outils de formatage et les boutons Annuler/Ok () s'affichent.

    Pendant que vous tapez, vous pouvez sélectionner Annuler () pour effacer le champ de texte et recommencer. Le texte que vous saisissez est automatiquement sauvegardé, mais vous pouvez également utiliser le bouton Ok () pour sauvegarder.

  5. Inscrivez "Oui" pour l'option 1.

  6. Sélectionnez Ajouter () pour ajouter "Non" comme deuxième option.

  7. Select More actions (), then select Settings.

  8. Sélectionnez +Réponse pour ajouter une réponse.

  9. In the Response Type drop-down menu, select Upload Files.

  10. Select Edit Visibility.

  11. Dans la boîte de dialogue Paramètres de visibilité, remplacez Masquer par Afficher et sélectionnez Condition de réponse dans le menu déroulant Ajouter une condition.

  12. Modifiez les en-têtes du rapport comme suit :

  13. Sélectionnez Copier. Nous allons l'utiliser dans un instant.

  14. Sélectionnez Enregistrer.

  15. Sélectionnez à nouveau Enregistrer dans la boîte de dialogue Paramètres.

  16. Nous allons maintenant ajouter une deuxième question. Déplacez à nouveau votre curseur vers le bas jusqu'à ce que vous voyiez l'icône bleue Ajouter () et la ligne sous la question de l'interrogation.

  17. Sélectionnez Ajouter () et choisissez le type de réponse Multi Picklist .

  18. Saisissez le résumé, la description de la requête et les options de la liste de sélection comme suit :

  19. Cochez la case Inclure "Autres" dans les options .

  20. Select More actions (), then select Settings.

  21. Sélectionnez +Réponse pour ajouter une réponse.

  22. Dans le menu déroulant Statut , sélectionnez Terminé.

  23. Sélectionnez Modifier la visibilité.

  24. Sélectionnez Coller.

    Les paramètres de visibilité de la question précédente sont collés.

  25. Modifier la procédure de "Demande d'informations sur les acquisitions d'immobilisations corporelles" à "Demande d'informations sur les transactions avec les parties liées"

  26. Sélectionnez Clients dans le menu déroulant Contacts .

  27. Sélectionnez Enregistrer.

  28. Sélectionnez à nouveau Enregistrer dans la boîte de dialogue Paramètres.

Envoyer une requête

Nous avons maintenant terminé la personnalisation de la requête et vous pouvez l'envoyer à votre client.

To send My Query to a client:

  1. Sélectionner SEND.

  2. Saisissez les informations suivantes dans la boîte de dialogue Ouvrir une requête :

    1. Saisissez la date du jour dans le champ Date d'échéance de la requête.

    2. Sélectionnez un contact client dans la liste déroulante Attributaires .

    3. Saisissez "Veuillez télécharger les éléments demandés" dans la zone de texte du corps de l'e-mail .

  3. Sélectionnez OK pour envoyer la requête.

Un courriel de notification est envoyé aux contacts désignés.
