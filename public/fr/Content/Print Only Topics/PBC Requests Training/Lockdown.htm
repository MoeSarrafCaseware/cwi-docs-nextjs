Verrouillage

Une fois que vous avez terminé et archivé le dossier, vous pouvez verrouiller l'engagement. Sélectionnez le titre pour ouvrir la boîte de dialogue Propriétés de l'engagement, puis sélectionnez Verrouillage.

Après avoir verrouillé l'engagement, vous pouvez fermer l'onglet du navigateur pour fermer le fichier d'engagement.
