Comprendre l'interface

Avant de commencer à travailler sur le dossier d'engagement, passons en revue l'interface d'engagement.

Propriétés d'engagement

Après avoir créé un nouvel engagement, une boîte de dialogue Propriété de l'engagement s'affiche automatiquement dans le fichier d'engagement. Vous pouvez ici saisir des informations de haut niveau sur l'engagement.

Dans cette fenêtre, vous pouvez définir les propriétés suivantes :

  • Étape du flux de travail - sélectionnez une étape pour l'état d'avancement de l'ensemble du travail de la mission.

  • Nom d'exploitation - ce champ affiche par défaut le nom de l'entité Cloud. Vous pouvez modifier la valeur par défaut et saisir un nom différent.

  • Date de fin d' année - choisissez une date de fin d'année pour cet engagement.

La boîte de dialogue Propriétés de l'engagement comporte également les boutons suivants :

  • Sélectionnez le bouton Deuxième réviseur pour ajouter un deuxième réviseur pour toutes les signatures. Cette option ajoute l'examen final à toutes les options de signature.

  • Verrouillage - sélectionnez cette option pour verrouiller le dossier d'engagement une fois que tout le travail est terminé.

Pour cet exercice, définissez les propriétés suivantes pour votre fichier d'engagement :

  • Étape du flux de travail - En cours

  • Nom de l'entreprise - ACME Inc.

  • Date de fin d'année

You do not need to add a second reviewer. Une fois que vous avez terminé la mise à jour de vos propriétés d'engagement, sélectionnez Enregistrer.

Problèmes

Les notes de révision peuvent signaler à l'équipe de mission qu'un travail supplémentaire est nécessaire sur un document ou que des tâches spécifiques doivent être accomplies dans le cadre d'une mission.

Niveau d'engagement

Si vous souhaitez qu'une question s'applique globalement à l'ensemble de la mission, vous pouvez l'ajouter au niveau de la mission.

Pour ajouter une question à un document:

  1. Dans le menu supérieur, sélectionnez l'icône Ajouter une question ().

    Une fenêtre contextuelle s'affiche sur la page, présentant une zone de texte avec une barre d'outils de formatage.


  2. Saisissez une description de votre problème dans la zone de texte.

  3. Sélectionnez un type de problème dans le menu déroulant situé en bas de l'éditeur.

  4. Sélectionnez Créer un problème.

Niveau du document

Vous pouvez également ajouter des questions à des documents spécifiques au sein de l'engagement.

Pour ajouter une question à un document:

  1. Sélectionnez l'icône Ajouter une question () pour le document.

  2. Saisissez une description de votre problème dans la zone de texte.

  3. Sélectionnez un type de problème dans le menu déroulant situé en bas de l'éditeur.

  4. Sélectionnez Créer un problème.

Niveau du poste

Par exemple, vous pouvez ajouter un problème à une procédure individuelle dans une liste de contrôle.

Pour ajouter une question à un document:

  1. Sélectionnez l'icône Ajouter une question () pour la procédure.

  2. Saisissez une description de votre problème dans la zone de texte.

  3. Sélectionnez un type de problème dans le menu déroulant situé en bas de l'éditeur.

  4. Sélectionnez Créer un problème.

Onglet Questions

Vous pouvez consulter et gérer toutes les questions relatives à un engagement à partir de la page Questions. Sélectionnez Questions pour accéder à la page.

Informations

Vous pouvez sélectionner l'icône d' information () dans le menu supérieur pour afficher le menu d'information. Dans ce menu, vous pouvez utiliser l'option Télécharger une copie de référence , accéder à l'aide du produit et consulter des informations sur la version actuelle du produit.

Télécharger l'exemplaire de référence

Lorsque vous sélectionnez Télécharger la copie de référence, un dossier zip est téléchargé sur votre disque local. Le nom par défaut du dossier est <Nom d'exploitation><Année>. Le dossier contient les PDF suivants :

  • Informations sur l'engagement

  • Une liste de documents

  • Tous les documents d'engagement

  • Une liste de questions

  • Une liste d'ajustements

  • Balance

Aide

Lorsque vous sélectionnez Aide, une nouvelle fenêtre de navigateur s'ouvre avec les liens vers les rubriques d'aide en ligne de PBC Requests.

A propos de

Lorsque vous sélectionnez À propos, une fenêtre contextuelle affiche le numéro de version actuel des requêtes PBC, d'autres informations sur le logiciel et un lien vers les dernières notes de mise à jour.

Onglet Documents

Lorsque vous sélectionnez l'onglet Documents dans le menu supérieur, la page Documents s'affiche. C'est là que vous pouvez gérer les documents, vérifier l'état d'avancement de la mission et commencer à effectuer diverses tâches telles que la préparation d'une lettre à votre client ou l'établissement d'une liste de contrôle.

Panneau de navigation

Pour accéder rapidement aux documents d'engagement, vous pouvez utiliser le panneau de navigation situé à gauche de la page Documents.

Pour développer le panneau de navigation, vous pouvez sélectionner l'icône Documents () en haut à gauche, ou l'icône Développer () en bas à gauche. Pour réduire le panneau, sélectionnez l'icône de contrat ().

Rechercher

The search bar is located at the top of the navigation panel. Vous pouvez rechercher un document par son nom. Vous pouvez également rechercher un mot clé dans n'importe quel document.

Liste des documents

Le panneau de navigation affiche une liste de tous les dossiers et documents actuellement visibles. Une fois que vous avez ouvert un document, son nom s'affiche dans la catégorie Récent .

Documents épinglés

Si vous souhaitez qu'un document récemment ouvert apparaisse toujours en haut de la liste, survolez le document dans la catégorie Récent et sélectionnez l'icône d'épingle (). Le document apparaît dans la catégorie Épinglé en haut de la liste des documents.

Page des documents

La page Documents est le point de départ de la plupart des travaux d'engagement.

Phases

Dans OnPoint Collaborate, les documents s'affichent en quatre phases dans la page Documents :

  • Commencer ici

  • Compilations

  • Impôt sur le revenu des personnes physiques

  • Audit

  • Personnaliser

  • Impôt sur les sociétés

  • Révision du travail effectué

Lorsque vous commencez un nouvel engagement, la page ne contient que quelques phases et documents. Au fur et à mesure que vous complétez les procédures et signez les documents, d'autres documents apparaissent dans la page Documents.

Add content

Vous pouvez ajouter des documents ou des dossiers à votre engagement. Sélectionnez l'icône Ajouter un document () sur le côté d'une phase ou d'un dossier, puis sélectionnez l'une des options dans le menu déroulant :

  • Sélectionnez +Rôles pour ajouter un nouveau rôle.

  • Sélectionnez un type de document pour créer un nouveau document

  • Sélectionnez Télécharger pour ajouter un document à partir de votre disque local

  • Sélectionnez Copier à partir de Caseware Cloud pour ajouter un fichier à partir du Cloud

Indicateur de progrès

Chaque phase d'engagement est assortie d'un indicateur de progrès () qui indique le pourcentage de documents signés dans cette section.

Dossiers

Pour chaque phase de l'engagement, les documents sont classés dans des dossiers avec des étiquettes bleues. Vous pouvez sélectionner les icônes fléchées pour ouvrir () ou fermer () le dossier.

Lorsque vous survolez un dossier, l'icône Ajouter un document () et l'icône Autres actions () s'affichent pour vous permettre d'ajouter un document au dossier ou de modifier les propriétés du dossier.

Documents

Chaque dossier d'une phase d'engagement contient des documents. Vous pouvez ajouter les balises suivantes :

  • Lettre

  • Mémo

  • Feuille de travail

  • Liste de contrôle

  • requête

  • États financiers

  • Marqueur.

  • Liens

Modifier la vue

Dans la page Documents , vous pouvez sélectionner l'icône Basculer la vue d'édition () pour activer ou désactiver la vue d'édition. Lorsque la vue d'édition est activée, l'icône Questions (), l'icône Autres actions () et l'icône Visibilité () deviennent disponibles pour chaque document. À partir de là, vous pouvez créer des questions, déplacer des documents ou modifier la visibilité des documents. Pour en savoir plus, voir Gérer les documents d'engagement.

Autres options

Les documents sont affichés de manière contextuelle : au fur et à mesure que vous saisissez des informations dans les documents de planification, d'autres documents pertinents pour votre dossier sont ajoutés à la page Documents. Vous pouvez sélectionner l'icône Autres actions (), puis cocher la case Éléments cach és pour afficher tous les documents et dossiers cachés. Chaque élément caché possède une icône actuellement cachée ().

La page Documents est affichée par défaut en vue détaillée ; elle affiche une icône pour chaque document afin d'indiquer le type de document ; elle indique également le statut de signature de chaque document. Pour obtenir une vue simplifiée, sélectionnez l'icône Autres actions () et décochez la case Vue détaillée.
