Accès au processus d'exploitation minière

Pour utiliser le Process Mining dans votre engagement Cloud, vous devez ajouter le document Process Mining.

Pour ajouter le processus d'exploitation minière :

  1. Sélectionnez Création de documents () dans la section souhaitée de la page Documents .
  2. Sélectionnez le document Process Mining dans le menu qui s'affiche.
  3. Sélectionnez Enregistrer dans la boîte de dialogue Propriétés du document qui s'affiche.

Pour lancer Process Mining :

  • Sélectionner le processus d'exploitation minière - p2p.
