Écran d'accueil pour l'exploitation minière

Le diagramme de processus en haut de l'écran d'accueil montre comment les données circulent à travers les différents processus du cycle PPP (Achats, Dettes, Paiements). Chaque bloc du processus est associé à un type de document spécifique. Par exemple, les entrées de données de type PR sont associées au bloc Demande d'achat.

Remarque : Si l'importation des données a échoué, un message d'erreur s'affiche et l'écran d'accueil est vide. Dans ce cas, réimportez vos données.

Cliquez sur les différentes variantes du tableau pour les mettre en évidence dans le diagramme de processus.

Vous pouvez également cliquer sur les icônes initiales des utilisateurs pour afficher les types d'entrées auxquels ils sont associés.

Rapport de synthèse sur les variantes de processus

Le rapport de synthèse des variantes de processus contient des informations sur le processus, la date et l'heure de la génération du rapport, l'utilisateur qui a généré le rapport ainsi que la liste complète de toutes les variantes de processus dans un processus et la liste des utilisateurs impliqués dans le processus.

Pour télécharger le rapport de synthèse des variantes de processus :

  1. Cliquez sur le bouton Télécharger les variantes de processus ().
  2. Sélectionnez Télécharger, pour télécharger un fichier PDF.
    3. Ou
  3. Sélectionnez Enregistrer dans la missionpour enregistrer le rapport dans votre dossier de mission.

Remarque : Vous pouvez également générer ce rapport à partir de la boîte de dialogue Variantes.
