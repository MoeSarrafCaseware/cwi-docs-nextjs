Détails de la transaction

Vous pouvez consulter les détails des transactions dans une variante de processus spécifique.

Pour afficher les détails de la transaction :

  • Dans l'écran d'accueil ou dans la boîte de dialogue Variantes, double-cliquez sur une variante de processus spécifique ou cliquez sur le lien web dans le numéro de variante d'une transaction.
    La boîte de dialogue Détails de la transaction s'affiche. Cette boîte de dialogue répertorie toutes les transactions triées par ordre décroissant de la date de comptabilisation. Faites défiler l'écran pour voir toutes les transactions.

Détails de la transaction décrits

L'état Détails de la transaction contient tous les groupes de transactions de composants dans une variante de processus spécifique. Il comprend la date et l'heure de la génération du rapport, l'utilisateur qui a généré le rapport et une liste complète de tous les groupes de transactions dans une variante de processus.

Pour télécharger le rapport détaillé de la transaction :
