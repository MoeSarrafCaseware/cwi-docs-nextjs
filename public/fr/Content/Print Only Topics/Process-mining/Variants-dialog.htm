Dialogue sur les variantes

La boîte de dialogue Variantes permet de visualiser toutes les variantes de processus dans vos données d'engagement. This dialog lists all the process variants sorted in descending order of number of transactions.

Pour ouvrir la boîte de dialogue Variantes :

  • Sélectionnez le bouton View All dans l'écran d'accueil de Process Mining. Ce bouton n'apparaît que s'il y a plus de trois variantes de processus dans l'engagement.

Utilisez les filtres situés en haut de la boîte de dialogue pour sélectionner le contenu de la boîte de dialogue par composant de processus et par utilisateur.
