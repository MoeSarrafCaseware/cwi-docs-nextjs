Interface cloud

Avant d'examiner un dossier d'engagement, passons en revue quelques caractéristiques clés de l'interface Cloud.

Filtre pour les entités

Vous pouvez utiliser la liste déroulante de l'entité pour afficher les éléments d'une entité spécifique.

Par exemple, sur la page Fichiers , vous pouvez sélectionner une entité spécifique pour afficher les fichiers téléchargés vers cette entité.

Menu du nuage

Dans le menu Cloud , vous pouvez afficher et accéder à vos produits d'engagement Cloud actifs.
