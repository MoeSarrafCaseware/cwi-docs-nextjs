Aperçu

Caseware Sherlock est un pionnier de l'intelligence comptable pour les entreprises du monde entier. Il s'agit d'une plateforme de veille stratégique puissante, sécurisée et basée sur le cloud, dotée d'une interface utilisateur élégante. Sherlock recueille et stocke automatiquement les données des missions Caseware actuelles et historiques de votre entreprise, et offre un chemin flexible pour transformer ces données en informations exploitables à l'échelle de la pratique en temps quasi réel grâce à des analyses et des visualisations de données avancées. Désormais, vous pouvez contrôler la santé de votre cabinet, mieux gérer les risques, identifier les pistes de croissance et devenir le conseiller stratégique de confiance de vos clients grâce à ce nouveau portail aux possibilités illimitées.

Comment cela fonctionne-t-il ?

Sherlock analyse automatiquement et continuellement les données de tous les engagements qui résident dans l'écosystème Caseware. Les données d'engagement sont extraites, transformées, organisées et stockées dans le lac de données Sherlock, ce qui permet une analyse de l'engagement à l'échelle de la pratique. Les cabinets peuvent accéder à leurs données d'engagement et les interroger pour prendre des décisions commerciales, tant pour leur cabinet que pour leurs clients.

La plateforme Sherlock de Caseware se compose de trois solutions principales : Sherlock Reader, Sherlock Builder et Sherlock API.Sherlock Reader, Sherlock Builder et Sherlock API. Ensemble, ces produits constituent une solution applicable à l'ensemble de l'entreprise et utilisable par des membres du personnel occupant des fonctions diverses.

Sherlock Reader

Sherlock Reader permet au personnel de présenter les données clés de l'engagement des entités clientes dans des visualisations faciles à utiliser. Vous pouvez exploiter les données de ces visualisations pour obtenir des informations sur les finances de vos clients, vous aider à gérer votre propre pratique, créer de nouvelles opportunités de revenus avec des services de conseil, et bien plus encore. Les visualisations sont disponibles auprès de votre distributeur Caseware, ou conçues sur mesure par l'auteur de votre entreprise à l'aide de Sherlock Builder.

Découvrez comment démarrer avec Sherlock Reader.

Sherlock Builder

Sherlock Builder est un outil de visualisation de données qui consomme les données des Working Papers et permet aux auteurs des entreprises de développer des modèles personnalisés.

Sherlock Builder se compose de trois éléments :

  • Les analyses sont des modèles de visualisation personnalisables. Vous pouvez partager une analyse avec d'autres utilisateurs de Sherlock Builder pour collaborer à son développement.
  • Les visualisations sont les versions en lecture seule des analyses que les utilisateurs de Sherlock Reader partagent avec leurs clients.
  • Les ensembles de données contiennent les données collectives qui alimentent une analyse. Les jeux de données dans Sherlock Builder sont préchargés et ne peuvent pas être modifiés. Les données sont actualisées quotidiennement.

Découvrez comment démarrer avec Sherlock Builder. (Identifiants MyCaseware et licence Sherlock Builder requis)

Pour commencer

Avant que le personnel puisse accéder à Sherlock, un administrateur doit lui attribuer une licence pour l'application concernée (Sherlock Reader ou Sherlock Builder).

Avertissement : Ne pas modifier l'adresse email ou révoquer la licence d'un utilisateur de Sherlock Builder. Toutes les analyses créées par l'utilisateur deviendront inaccessibles à tous les utilisateurs, y compris les administrateurs.

Pour attribuer une licence Sherlock à un membre du personnel :

  1. Assurez-vous d'avoir le rôle d' administrateur du personnel ou des privilèges équivalents.

  2. Dans le menu Nuage, sélectionnez Personnel.

  3. Sélectionnez le membre du personnel auquel vous souhaitez attribuer la licence, puis sélectionnez Modifier ().

  4. Sélectionnez Apps, puis choisissez les autorisations appropriées ( Sherlock Reader ou Sherlock Builder).

  5. Sélectionnez Enregistrer.

Sherlock Builder Créer une analyse

Vous pouvez créer une analyse pour chaque ensemble de données et personnaliser la façon dont vous souhaitez présenter les données d'engagement dans la visualisation.

To create an analysis:

  1. Dans le menu Cloud (), sélectionnez Sherlock Reader.

    Un cadre interactif alimenté par les écrans QuickSight d'Amazon.

  2. Sélectionnez l'onglet Factures .

    Une liste des visualisations disponibles s'affiche.

  3. Sélectionnez Nouvelle inscription.

  4. Sélectionnez l'ensemble de données applicable à l'analyse. Les options disponibles sont les suivantes :

    • Résumé des comptes
    • cv (CaseView)
    • cvcontrol
    • cvmlp (management letter points)
    • cvrisk
    • Groupe
    • hi (histoire)
    • Mises en correspondance
    • métadonnées (propriétés d'engagement)
    • sh (document)
    • solde du procès
    • ts (questions)
    • Utilisateur
    • groupe d'utilisateurs

    Notes :

    • Les noms des ensembles de données sont complétés par le nom abrégé de l'entreprise (par exemple, trialbalance-testfirm-001).
    • Le jeu de données entityrls contient des informations de sécurité au niveau de l'entité et ne doit pas être utilisé comme jeu de données pour une analyse ou partagé avec d'autres.

  5. Sélectionnez Créer un brouillon.

  6. Personnaliser l'analyse.

    Des exemples de personnalisation pour différents scénarios peuvent être trouvés en suivant cette procédure.

  7. Si vous souhaitez collaborer à l'analyse avec d'autres utilisateurs de Sherlock Builder dans votre entreprise, sélectionnez Partager () | Partager l'analyse.

    La boîte de dialogue Partager l'analyse s'affiche.

  8. Saisissez l'adresse électronique de chaque utilisateur et sélectionnez Partager.
