Notes de mise à jour SE 2019-01

Toutes les fonctionnalités ci-dessous sont automatiquement activées si le composant ou le module correspondant a été activé.

Tableaux d'analyse

Les décimales sont désormais prises en charge dans les graphiques d'analyse. Les paramètres décimaux du tableau des ratios sont appliqués dans le graphique.

Création

Les auteurs principaux peuvent désormais :

  • Exporter les paramètres de visibilité dans un fichier CSV afin de mieux vérifier et suivre les paramètres existants dans un modèle. Pour plus d'informations, voir Visibility Export - Dist. Version

  • Ajouter des références à d'autres documents dans le domaine des orientations. Par exemple, inclure une référence à un document d'interrogation spécifique dans les conseils de la lettre.

  • Ajouter une zone d'orientation dans les documents d'analyse. Notez que les utilisateurs engagés ne peuvent voir que la zone d'orientation.

Reporter

Il y a maintenant un compteur de badges sur l'icône de report. Le compteur de badges indique le nombre d'articles pouvant être reportés.

Tableaux dynamiques

  • Les en-têtes de colonnes se répètent désormais sur les pages suivantes lorsque vous insérez manuellement des sauts de page dans le tableau dynamique.

  • Le contenu dynamique est disponible dans les en-têtes de colonne. Vous pouvez désormais inclure du texte dynamique et des formules dans les en-têtes de colonne.

  • Nous avons amélioré la façon dont les symboles sont affichés et ajoutés dans les en-têtes de colonnes. Les symboles apparaissent désormais sur une ligne distincte sous le texte de l'en-tête.

  • Il est désormais possible d'annuler et de rétablir des actions lors de la modification d'un tableau dynamique.

Nouveau design pour les finances

Reporter

  • Les évaluations des risques peuvent désormais être reportées. Il s'agit de zones et de contenus nouveaux, modifiés ou cachés.

  • Vous pouvez désormais consulter le nombre de postes que vous pouvez reporter dans chaque domaine. Un indicateur des postes de report par domaine s'affiche dans la carte du document.

Consolidation

Vous avez maintenant la possibilité de choisir d'utiliser ou non des données non consolidées dans vos états financiers.

Bibliothèque de notes

  • Preview option is available in the Notes Library. Vous pouvez désormais afficher un aperçu d'une note avant de l'ajouter dans les états financiers.

  • Vous pouvez désormais filtrer les notes par groupe de balance de vérification.

Formule :

Des champs d'adresse supplémentaires sont disponibles dans les formules pour plus de flexibilité. Vous pouvez maintenant organiser les informations relatives à l'adresse de manière à respecter les spécifications de votre format régional.

Lettres, mémos et états financiers

  • Les lettres et les états financiers comportent désormais un indicateur du nombre de caractères génériques non paramétrés dans le document.

  • La signature des documents valide désormais les champs et le texte des espaces réservés. Si le document contient des espaces réservés non définis, un message d'avertissement s'affiche avant que vous ne puissiez procéder à la signature.

Traduction

La langue espagnole est désormais prise en charge dans la barre de formatage d'une zone de texte.

Requêtes

  • Le nom du répondant et l'horodatage de la réponse sont désormais affichés pour tous les types de réponse.

  • Le tri est désormais disponible dans le tableau de bord des requêtes. Vous pouvez trier les requêtes par ordre croissant ou décroissant en fonction du statut, de la date d'échéance, de l'état d'avancement et du nom.

  • Vous recevez désormais un message de confirmation lorsque vous envoyez une requête.

Corrections

  • Nous avons ajouté des infobulles pour diverses icônes dans l'application et inclus la prise en charge de la traduction.

  • Les auteurs de cabinets ne peuvent plus supprimer le contenu qui a un impact sur la disponibilité d'autres contenus dans le modèle.

  • Les auteurs du cabinet ne peuvent plus modifier le contenu des états financiers qui a été ajouté par l'auteur principal.
