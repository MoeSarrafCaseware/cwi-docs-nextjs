SE 2019-04 hot fixes

Voici un résumé des correctifs qui ont été déployés pour la SE 2019-04.

Note :La liste suivante ne comprend que les correctifs généraux qui ont été déployés dans toutes les régions. Il n'inclut pas les correctifs déployés dans des régions spécifiques.

Correctif 1

Voici la liste des problèmes qui ont été corrigés avec le déploiement du Hotfix 1.

  • [SE-12161] Les tableaux dynamiques ne conservent pas les valeurs dans le formulaire de complétude des données
  • [SE-12162] Les tableaux dynamiques affichent des ajustements en double
  • [SE-12190] Les conversions ne sauvegardent pas correctement les fichiers

Correctif 2

Voici la liste des problèmes qui ont été corrigés avec le déploiement du Hotfix 2.

  • [SE-12215] La date n'apparaît plus sous les questions d'interrogation
  • [SE-12377] Tableaux dynamiques (Financiers) - Les lignes de TB ne s'affichent pas en mode auteur s'il s'agit d'un enfant d'un groupe personnalisé
