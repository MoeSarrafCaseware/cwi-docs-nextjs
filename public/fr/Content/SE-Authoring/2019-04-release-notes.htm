Notes de mise à jour SE 2019-04

Toutes les fonctionnalités ci-dessous sont automatiquement activées si le composant ou le module correspondant a été activé.

La page Données

La page Données est une refonte de l'ancienne page Balance de vérification. L'objectif de cette refonte est de fournir des méthodes de reporting flexibles, de prendre en charge davantage de sources de données et de paramètres, de fournir un flux de travail guidé pour l'importation et la mise en correspondance, ainsi que d'améliorer les performances et la facilité d'utilisation.

Les nouvelles fonctionnalités de la page Données sont les suivantes

  • Une nouvelle vue des groupes et des comptes

  • Définition des groupes de balances de vérification par défaut pour le niveau de détail requis par le produit

  • Drill-down to transactional details

  • Un volet gauche pour changer facilement de vue

  • Tri et filtrage des données dans chaque vue.

  • Capacités de recherche.

  • Exportation de données (balance générale vers CSV)

Les fonctionnalités suivantes sont disponibles dans l'interface précédente et seront améliorées dans les versions ultérieures :

  • Importation de données et mise en correspondance des comptes (pour la balance de vérification et les transactions)

  • Régularisations

  • Rapports prédéfinis accessibles en enregistrant des filtres de données

Les auteurs principaux peuvent activer la page Données dans leurs produits en cochant la case Activer la page Données dans les paramètres du produit.

Recommandations : Nous suggérons aux distributeurs de consacrer beaucoup de temps à tester leur produit avant de déployer la nouvelle interface utilisateur auprès des clients.

Lettres, mémos et états financiers

  • Pour localiser facilement et efficacement les espaces réservés non spécifiés dans le document, vous pouvez désormais sélectionner l'indicateur d'espace réservé pour naviguer dans les espaces réservés qui n'ont pas été spécifiés.

  • Les auteurs peuvent désormais ajouter un logo et un en-tête dans le modèle de produit ou d'entreprise. L'espace réservé se remplit automatiquement à partir des nouveaux champs Logo et En-tête dans Caseware Cloud. L'en-tête s'affiche automatiquement dans le fichier PDF lorsque les utilisateurs impriment le document.

Nouveau design pour les finances

Création

Les auteurs peuvent désormais ajouter des informations d'orientation dans le document des états financiers.

  • The Add Guidance menu option is now available for areas, text sections, dynamic tables and notes. Les utilisateurs peuvent activer ou désactiver les informations d'orientation à l'aide de l'icône en forme d'ampoule située à côté de la zone, de la section de texte ou du tableau.

  • Une nouvelle zone d' orientation est également disponible pour ajouter des informations d'orientation générale dans le document.

Notes

  • la possibilité d'avoir une liste de notes à trois niveaux dans le document des états financiers en ajoutant des notes de deuxième niveau (Heading1) et de troisième niveau (Heading2). Si les utilisateurs suppriment une note, tous les titres et le contenu de cette note sont également supprimés. Notez que les notes de troisième niveau ne s'affichent pas dans le plan du document à moins que les utilisateurs ne développent leurs notes respectives de deuxième niveau (notes parentes) dans le document.

  • La bibliothèque de notes affiche désormais les notes en fonction de leur hiérarchie (notes de sous-niveau ou notes principales). Vous pouvez parcourir la hiérarchie des notes affichées en sélectionnant Voir note.

  • Lorsque vous ajoutez une note à partir de la bibliothèque, un indicateur bleu s'affiche désormais à côté de la note nouvellement ajoutée dans la carte du document. L'indicateur bleu disparaît une fois la note sélectionnée.

Tableaux dynamiques

  • Vous avez désormais la possibilité de modifier les références des notes dans une ligne ou une colonne. Ils peuvent également naviguer jusqu'à la note référencée (dans la zone Notes aux états financiers ) pour en visualiser ou en modifier le contenu.

  • La colonne Description peut désormais afficher des numéros de compte en plus du nom du compte et du total. Notez que les auteurs doivent activer cette fonctionnalité dans le produit.

  • Un nouveau format de tableau est désormais disponible pour prendre en charge les normes internationales d'information financière (IFRS) et l'état des variations des capitaux propres (SOCIE). Le nouveau format renverse le tableau et présente les dimensions en lignes et les groupes de balances de vérification en colonnes.

    • Notez que toute configuration et information existante est perdue lorsque le format du tableau est modifié. Les auteurs doivent refaire toute configuration de tableau existante s'ils passent au format IFRS/SOCIE.

Paramètres de visibilité

Dans les états financiers, les utilisateurs reçoivent désormais une demande de confirmation s'ils tentent d'exclure du contenu disponible en raison de paramètres de visibilité. L'invite indique les détails de la logique des conditions de réponse et fournit un lien vers le document de la liste de contrôle afin que vous puissiez revoir et modifier les réponses si nécessaire.

Notez qu'actuellement, cette fonctionnalité n'est disponible que pour les paramètres de visibilité basés sur les conditions de réponse.

Corrections

  • Correction d'un problème où les nombres décimaux ne s'affichaient pas correctement lorsque la virgule était le séparateur décimal et le point le séparateur de groupe de chiffres.

  • Le contenu exclu est supprimé du menu Ajouter du contenu (+) pour les auteurs principaux dans la nouvelle conception des données financières.

  • Correction de certains problèmes liés à l'indentation et à la troncature des titres de notes dans le plan du document dans le nouveau design financier.

  • Correction d'un problème lié aux outils de révision après un report, où certaines réponses aux procédures étaient incorrectement marquées comme modifiées.

  • Correction d'un problème d'incohérence dans la taille des icônes pour les annotations dans la copie de référence de l'engagement (fichier PDF).

  • Correction d'un problème de style dans la zone de la table des matières dans le nouveau design financier.

  • L'icône d'impression a été supprimée de la page d'interrogation.

  • Correction d'un problème où l'icône Outils de révision était disponible pour les auteurs dans le nouveau design financier. Il a maintenant été supprimé.

  • Correction de plusieurs problèmes de traduction où certains textes s'affichaient incorrectement en anglais.

  • Correction des problèmes de performance dans le domaine financier avec des tables de grande taille.
