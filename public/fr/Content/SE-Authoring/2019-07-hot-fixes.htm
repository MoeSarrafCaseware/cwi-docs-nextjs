2019-07 correctifs

Voici un résumé des correctifs qui ont été déployés pour la SE 2019-07.

Note :La liste suivante ne comprend que les correctifs généraux qui ont été déployés dans toutes les régions. Il n'inclut pas les correctifs déployés dans des régions spécifiques.

Correctif 1

Voici la liste des problèmes qui ont été corrigés avec le déploiement du Hotfix 1.

  • [SE-13928] - L'aperçu avant impression du document sur les états financiers ne se charge pas.

  • [SE-13907] - Problèmes de requête dans le mode modèle de l'auteur de l'entreprise.

  • [SE-13874] - Report des problèmes après la migration vers la SE 2019-07.

  • [SE-13864] - Les formules de texte (fx) ne fonctionnent pas comme prévu dans les tableaux dynamiques.

  • [SE-13835] - Problèmes liés à la nouvelle conception des requêtes :

    • Les auteurs d'entreprises peuvent modifier une partie du contenu des produits par défaut.

    • L'option de suppression du contenu n'est pas disponible pour les auteurs d'entreprise.

  • [SE-13808] - Certains sous-groupes d'un groupe financier spécifique ne sont pas visibles dans la page de données.

  • [SE-13516] - Corrections de l'API.

  • [SE-13511] - Erreur dans les journaux du client.

  • [SE-11968] - Corrections de l'API.

Correctif 2

Voici la liste des problèmes qui ont été corrigés avec le déploiement du Hotfix 2.

  • [SE-14330] - Les documents externes importés depuis BGL 360 sont vides.

  • Transmettre les suggestions.

    • [SE-14249] - La feuille de calcul et le mémo de la période précédente sont vides dans le fichier de la période suivante.

    • [SE-14215] - Les tableaux dynamiques sont absents des documents d'états financiers que les utilisateurs ajoutent dans le fichier de la période suivante.

    • [SE-14112] - Les données de l'année précédente s'affichent pour les filiales dans le fichier de la période suivante au lieu des données de l'année en cours.

    • [SE-14109] - Ne pas reporter les dossiers dont les dates de la nouvelle année ne sont pas adjacentes à celles de l'année précédente.

  • [SE-14185] - Erreur de console dans les documents de la liste de contrôle.

  • [SE-14134]- Erreur "Engagement invalide" lors de la création d'un nouvel engagement dans AnalyticsAI.

  • [SE-14118] - L'affectation des comptes dans la page Balance de vérification ne fonctionne pas correctement.

Correctif 3

Voici la liste des problèmes qui ont été corrigés avec le déploiement du correctif 3.

  • [SE-14526] - Les lignes du tableau dynamique qui proviennent de la balance de vérification sont supprimées du modèle lorsque les auteurs de produits importent les groupes de la balance de vérification.

  • [SE-14435] - Les annotations ne s'affichent pas lorsque les utilisateurs impriment les documents. Le PDF créé pour l'impression n'affiche pas les annotations existantes dans le document.

  • [SE-14305] - La tentative de snapshot d'une URL de données pour générer un PDF pour les documents BGL360 échoue.

  • [SE-14420] - Erreurs lors de l'ouverture des fichiers d'engagement.

  • [SE-14179][SE-13292] - Échec du report dans certaines régions.

Hotfix 5

Voici la liste des problèmes qui ont été corrigés avec le déploiement du correctif 5.

  • [SE-14554] - Impossible de développer sur la plateforme car SPLAT ne se déploie pas.

  • [SE-14606] - Rafraîchissement spontané et très fréquent de la page.

  • [SE-14714] - Signoffs manquants dans les annotations des PDF.

  • [SE-14629] - Les comptes ne sont pas associés aux codes fiscaux.

  • [SE-14669] - Les ajustements ne s'affichent pas dans les graphiques d'analyse.

  • [SE-14552] - Absence de tableau d'analyse après l'importation des soldes de l'année précédente.

  • [SE-14606] - La page de données est disponible dans tous les modèles sur tous les sites bien qu'elle soit désactivée dans les paramètres du produit.

Hotfix 6

Voici la liste des problèmes qui ont été corrigés avec le déploiement du Hotfix 6.

  • [SE-14959] - Certains groupes de balances de vérification affichent par erreur un solde total de zéro dans les tableaux dynamiques.

  • [SE-14947] - Corrections d'API.

Hotfix 7

Voici la liste des problèmes qui ont été corrigés avec le déploiement du Hotfix 7.

  • [SE-15113] - Impossible d'imprimer les états financiers.

  • [SE-15126] - Les soldes zéro dans les tableaux dynamiques s'affichent comme des zéros dans la sortie PDF du document alors qu'ils devraient s'afficher comme des tirets (selon les paramètres du modèle).
