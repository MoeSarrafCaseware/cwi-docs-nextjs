SE 2019-10 hot fixes

Voici un résumé des correctifs qui ont été déployés pour la SE 2019-10.

Correctif 1

Version : 2019 / 10 / 465

Voici la liste des problèmes qui ont été corrigés avec le déploiement du Hotfix 1.

Pour afficher les détails de la cause première et de la solution, sélectionnez l'identifiant du problème (exemple : SE-12345).

  • [SE-16229] - Les tableaux dynamiques n'affichent pas correctement les comptes ajoutés manuellement dans la page Données.

  • [SE-16218] - Dans le nouveau module de risque, les utilisateurs ne peuvent pas saisir correctement une raison d'exclusion pour les zones dans l'onglet Évaluations .

  • [SE-16205] - Les paramètres de visibilité basés sur les conditions de matérialité ne fonctionnent pas comme prévu.

  • [SE-16171] - Dans la section des notes des états financiers, les lignes de groupe réduites dans les tableaux dynamiques n'affichent pas les soldes des comptes.

  • La page Données

    • [SE-16117] - les paramètres du séparateur décimal du produit ne s'appliquent pas aux comptes ajoutés manuellement.

    • [SE-15998] - les utilisateurs ne peuvent pas attribuer de comptes après avoir consolidé des entités.

    • [SE-15968] - les en-têtes de colonnes pour les années précédentes s'affichent de manière incorrecte après l'importation des données de l'année précédente.

  • [SE-15948] - Les soldes surchargés dans les tableaux dynamiques ne s'affichent pas correctement.

  • [SE-15941] -Les ajustements d'élimination ne s'affichent pas dans les tableaux dynamiques des états financiers.

  • [SE-15902] - Les utilisateurs ne peuvent pas sélectionner un espace réservé de texte copié-collé.

  • [SE-15844] - Dans les lettres, les états financiers et les mémos, une nouvelle ligne est automatiquement insérée lorsque les utilisateurs ajoutent une formule de texte d'adresse d'entreprise (fx).

  • [SE-15658] - Les soldes des comptes changent lorsqu'un groupe est développé et réduit.

  • [SE-15354] - Problèmes de saisie dans les champs de texte dans la nouvelle présentation des requêtes.

Correctif 2

Version : 2019 / 10 / 466

Voici la liste des problèmes qui ont été corrigés avec le déploiement du Hotfix 2.

Pour afficher les détails de la cause première et de la solution, sélectionnez l'identifiant du problème (exemple : SE-12345).

  • [SE-16376] - Les auteurs principaux ne peuvent pas télécharger des fichiers excel avec macro dans le modèle de produit.

  • Tableaux dynamiques

    • [SE-16311][SE-16282] - Les étiquettes de diagnostic sont manquantes dans les tableaux dynamiques dont la formule de calcul personnalisée inclut une division.

    • [SE-16386] - Lorsque les utilisateurs remplacent des montants, les soldes dans d'autres tableaux dans plusieurs domaines ne sont pas mis à jour tant que les utilisateurs n'ont pas actualisé la page.

    • [SE-16229] - les nouveaux comptes ne s'affichent pas dans les tableaux tant que les utilisateurs n'ont pas réduit et développé le groupe parent de ce compte.

  • [SE-16413] - Les soldes de la colonne Amount Variance ne s'affichent pas dans les états financiers. Les soldes ne s'affichent que dans la page du mode d'édition du tableau dynamique.

  • [SE-16373] - Les soldes des groupes ne s'affichent pas dans les onglets d' analyse.

  • [SE-16425] - Le contour des soldes annulés s'affiche dans le fichier d'impression des états financiers (PDF).

Correctif 3

Version : 2019 / 10 / 467

Un problème a été identifié lors de l'examen des journaux de production et a été corrigé avec le déploiement du correctif 3.

Pour afficher les détails de la cause première et de la solution, sélectionnez l'identifiant du problème (exemple : SE-12345).

[SE-16510] - Les utilisateurs obtiennent une erreur "Engagement non valide" lorsqu'ils tentent d'ouvrir ou de reporter certains fichiers d'engagement. Plus précisément, les dossiers qui n'ont pas été ouverts depuis 2019-04 et qui comprennent des documents justificatifs téléchargés.

Hotfix 4

Version : 2019 / 10 / 469

Voici la liste des problèmes qui ont été corrigés avec le déploiement du Hotfix 4.

Pour afficher les détails de la cause première et de la solution, sélectionnez l'identifiant du problème (exemple : SE-12345).

  • [SE-16545] - Les soldes supérieurs au montant trivial défini dans le document de matérialité sont marqués comme matériels dans la page Balance de vérification .

  • [SE-16537] - Les séparateurs de groupes de chiffres et les symboles décimaux sont absents des valeurs affichées dans les graphiques d' analyse.

  • [SE-16526] - Lorsque le nouveau module de risque est activé dans le produit, l'option Risque d'anomalies significatives dans l'onglet Risque des listes de contrôle affiche des informations incorrectes.
