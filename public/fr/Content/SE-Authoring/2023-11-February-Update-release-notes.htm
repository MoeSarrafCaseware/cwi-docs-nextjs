Notes de mise à jour SE 2023-11

La mise à jour de février pour la version SE 2023-11 offre la possibilité de personnaliser les styles de numérotation des titres dans les états financiers, une nouvelle fonction de copie pour les sections de texte et bien plus encore.

Titres numérotés pour les notes et les zones de texte dans les états financiers

#enduser#premier auteur#firmauthor #enduser #primaryauthor #firmauthor

Cette version apporte de nombreuses nouvelles options pour personnaliser les titres numérotés dans les états financiers. Le nombre de niveaux de titres a été porté à cinq et vous pouvez désormais personnaliser le style de numérotation des titres. Outre ces options de personnalisation des en-têtes de notes, vous pouvez désormais ajouter des en-têtes avec ou sans numérotation à n'importe quelle zone de texte des états financiers.

Vous pouvez accéder aux paramètres des titres numérotés et les personnaliser en sélectionnant Options des titres numérotés dans les paramètres du document ().

Dans la boîte de dialogue, vous pouvez choisir d'activer ou non la numérotation de vos titres, afficher le format de numérotation actuellement sélectionné et sélectionner un nouveau format parmi les options disponibles.

Notez que pour les titres de zones, en plus des options disponibles pour les titres de notes, vous avez également la possibilité de sélectionner un format de numérotation pour les titres de zones qui s'affichent dans la carte du document.

Pour en savoir plus, voir le texte personnalisé dans les états financiers.

Cut and paste text sections in the financial statements

#primaryauthor #primaryauthor

Les auteurs primaires peuvent désormais facilement couper et coller des sections de texte dans les états financiers en sélectionnant Couper la section dans le menu Plus d'actions() pour les sections de texte.

Vous pouvez ensuite sélectionner l'option Coller la section lors de l'ajout d'une nouvelle section pour coller la section de texte. La section sera retirée de son emplacement d'origine et déplacée vers la nouvelle section. L'utilisation de cette option, au lieu de couper manuellement la section, garantit que la section de texte collée est reportée correctement.

Pour en savoir plus, voir le texte complémentaire dans les états financiers.

Modifier les noms des groupes financiers

Les utilisateurs finaux peuvent désormais modifier les noms des groupes financiers à partir de la page Données. Notez que lorsque les utilisateurs modifient le nom d'un groupe, le nom qu'ils saisissent sera utilisé pour le groupe indépendamment des changements de signe et remplacera les noms cohérents, contraires et incohérents du groupe. Les utilisateurs ont également la possibilité de revenir sur leurs modifications à partir de la boîte de dialogue Propriétés du groupe.

Pour en savoir plus, voir Editer la lettre de mission.
