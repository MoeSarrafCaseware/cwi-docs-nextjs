Notes de mise à jour SE 2023-11

La version SE 2023-11 apporte des améliorations considérables au formatage des états financiers, une nouvelle forme de consolidation offrant une plus grande flexibilité aux utilisateurs, la possibilité d'attribuer plusieurs catégories de dimensions aux groupes financiers, etc.

Catégories de dimensions multiples pour les groupes financiers

#enduser#premier auteur#firmauthor #enduser #primaryauthor #firmauthor

Nous pouvons maintenant affecter les catégories de dimensions aux groupes financiers. Cela permet d'organiser et de présenter les données de différentes manières dans les états financiers. Par exemple, vous pouvez disposer d'un tableau dynamique qui ventile vos données par lieu, ainsi que d'un tableau organisé à des fins de rapports sur les fonds.

Les auteurs de cabinets et les utilisateurs finaux ont désormais la possibilité d'attribuer plusieurs catégories de dimensions à un seul groupe financier, ainsi que la possibilité d'attribuer plusieurs dimensions à un compte à partir de la boîte de dialogue Propriétés du compte. Notez qu'une seule dimension par catégorie de dimension peut être affectée à un compte à la fois.

To learn more, see Add dimensions and Group properties.

Paramètres d'impression de l'en-tête.

L'option Afficher les titres des colonnes sur les pages suivantes a été ajoutée aux états financiers. Lorsque vous sélectionnez cette option, les tableaux dynamiques dont la taille est supérieure à une page affichent les en-têtes des colonnes du tableau sur chaque page où il se trouve.

Conserver les paragraphes ensemble

Les options Conserver les titres des notes avec les paragraphes et Conserver les paragraphes ensemble sont désormais disponibles dans les paramètres d'impression du document. La sélection de cette option permet de s'assurer que les en-têtes de notes ne sont pas imprimés sur des pages séparées du contenu de la note et que les paragraphes ne sont pas répartis sur plusieurs pages, sauf si le paragraphe est plus long qu'une page.

En-têtes pour les notes de plusieurs pages

Dans les paramètres d'impression des notes, vous pouvez maintenant sélectionner l'option Activer le texte d'en-tête continu . Lorsque cette option est sélectionnée, vous pouvez saisir un suffixe personnalisé pour les en-têtes des notes qui s'étendent sur plusieurs pages.

Par exemple, si vous saisissez le suffixe "suite", une note dont l'en-tête est "Principales méthodes comptables" s'affichera comme "Principales méthodes comptables suite" sur les pages suivantes.

Notez que le suffixe que vous saisissez est personnalisable pour chaque langue dans les modèles multilingues.

Nouveau pour les utilisateurs finaux

Voici les nouveautés pour les utilisateurs finaux dans la SE 2023-11.

consolidation externe

Une nouvelle forme de missions consolidées est désormais prise en charge dans SE afin de tenir compte des missions de consolidation pour lesquelles les utilisateurs ne reçoivent pas les soldes individuels des entités subsidiaires de la part de leurs clients ou effectuent la consolidation en dehors de SE.

Notez que pour que l'option de consolidation externe soit disponible dans votre produit, vous devez activer l'option Activer le nouveau flux de travail pour les importations de balances de vérification CSV/Excel dans les paramètres du produit.

Pour marquer un engagement comme étant consolidé en externe, les utilisateurs peuvent sélectionner l'option Consolider en externe dans le popover Propriétés de l'engagement.

Ils peuvent ensuite importer leurs données, en veillant à ce qu'elles incluent à la fois les soldes consolidés et les soldes de la société mère. Comme pour les missions consolidées à l'aide de SE, les utilisateurs peuvent filtrer leurs données financières en fonction des valeurs consolidées et des valeurs propres à la société mère sur la page Données.

Pour en savoir plus, voir Créer un dossier d'engagement.

Créer une nouvelle mission sur la base d'un fichier existant

Les utilisateurs peuvent désormais créer de nouveaux fichiers d'engagement basés sur des fichiers existants à partir du menu Plus d'actions () pour l'engagement existant dans Cloud ou à partir de la boîte de dialogue Nouvel engagement.

Pour activer cette fonctionnalité, vous devez définir la propriété this.canCreateEngagementFromSource = true dans le fichier collaborate.js de votre produit.

Les informations suivantes seront copiées à partir de l'engagement source :

  • Personnalisation des structures de regroupement

  • Personnalisation des requêtes

  • Documents externes ajoutés à la mission, tels que les PDF et les fichiers Microsoft Office

  • Personnalisations et réponses aux listes de contrôle

  • Personnaliser le contenu des états financiers

  • Paramètres de visibilité

  • Risques, contrôles et évaluations des risques

  • Zones ajoutées aux paramètres d'engagement

  • Schémas de signature et rôles

Notez que :

  • Les comptes et soldes de la balance de vérification ne sont pas inclus

  • Les liens vers les documents internes et externes dans les requêtes ne sont pas inclus

  • Les engagements verrouillés ne peuvent pas être utilisés comme engagement source

Un lien vers l'engagement source est disponible dans la boîte de dialogue "A propos" du nouvel engagement.

Mentionner les contacts dans les numéros

Vous pouvez désormais mentionner des contacts dans les commentaires des clients.

Lorsque vous saisissez le symbole "@" dans une question, une liste complète des contacts et du personnel ayant accès à l'engagement s'affiche. Vous pouvez alors sélectionner le contact approprié dans la liste.

Pour en savoir plus, consultez la section Questions relatives aux documents dans le dossier de mission.

Mise à jour de l'exportation des données de la balance de vérification au format PDF

Lorsque les utilisateurs finaux exportent leurs données au format PDF à partir des onglets Comptes ou Groupes de la page Données , le PDF exporté reflète désormais avec précision les paramètres de colonne que l'utilisateur a sélectionnés lorsqu'il a choisi le mode d'affichage de ses données dans la page Données.

Dans les missions de consolidation, le PDF indiquera également si l'utilisateur a choisi de consulter les données consolidées ou les données d'une entité sélectionnée dans la hiérarchie de consolidation.

Pour en savoir plus, voir Exporter des groupes de balances de vérification.
