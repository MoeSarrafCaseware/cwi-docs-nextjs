Ajouter, modifier et exporter des groupes (page Données)

Vous pouvez ajouter des groupes ou modifier les propriétés des groupes existants. Vous pouvez également exporter des groupes vers un fichier CSV afin de revoir leurs propriétés ou d'apporter des modifications facilement.

Ajouter un groupe (page de données)

Vous pouvez ajouter manuellement des groupes à une structure de regroupement.

Pour ajouter un groupe :

  1. Accédez à la page Données , puis sélectionnez Ajouter () | Ajouter un groupe.

    2. La boîte de dialogue Propriétés du groupe s'affiche.

  2. Définir les propriétés du nouveau groupe.

    Reportez-vous à la section Propriétés du groupe pour une description de chaque propriété du groupe.

  1. Sélectionnez Enregistrer pour ajouter le groupe.

Modifier un groupe (page de données)

Vous pouvez modifier les propriétés d'un groupe dans la structure.

Pour modifier un groupe :

  1. Accédez à la page Données , puis sélectionnez la structure de regroupement dans le volet de gauche :

    • Groupementsfinanciers
    • Groupementsfiscaux
    • Groupementsprescrits .

  2. Sélectionnez le groupe que vous souhaitez modifier, puis sélectionnez Editer () pour ouvrir la boîte de dialogue Propriétés du groupe.

    3. Conseil : Vous pouvez rechercher des groupes en utilisant le champ de recherche situé dans le coin supérieur droit de la page.

  3. Modifiez les propriétés du groupe si nécessaire.

    4. Reportez-vous à la section Propriétés du groupe pour une description de chaque propriété du groupe.

  4. Une fois que vous avez terminé, sélectionnez Enregistrer.

Réorganiser un groupe (page de données)

Vous pouvez faire glisser et déposer des groupes dans la page Données pour les réorganiser. Naviguez jusqu'à la structure de regroupement de la page Données et faites glisser le groupe que vous souhaitez réorganiser à l'endroit désiré.

Notez que vous ne pouvez utiliser le glisser-déposer pour réorganiser les groupes que dans les conditions suivantes :

  • Les groupes que vous réordonnez sont tous des groupes frères dans la structure de regroupement. Vous ne pouvez pas glisser-déposer un groupe enfant pour qu'il soit au même niveau ou à un niveau supérieur que le groupe parent.

  • Tous les groupes de frères et sœurs sont réduits ou n'ont pas de groupes d'enfants.

Exporter des groupes vers un fichier CSV

You can export groups to a CSV file to review the properties. Vous pouvez également les exporter pour que tous les groupes soient alignés et que vous puissiez facilement apporter des modifications, puis enregistrer le fichier et l'importer.

La réimportation des groupes conserve toutes les propriétés des groupes existants. Seules les mises à jour des propriétés existantes et l'ajout de valeurs aux propriétés vides prennent effet.

Par exemple, vous pouvez exporter des groupes vers un fichier CSV et ajouter facilement la valeur"Y" à la propriété Groupe par défaut pour 15 groupes en copiant-collant"Y" sur 15 lignes. Une fois que vous avez terminé, enregistrez le fichier et réimportez-le .

Remarque : Lorsque vous exportez des groupes, l'ordre des groupes dans la structure lors de l'exportation est conservé lorsque vous les importez à nouveau. Dans le fichier CSV, cependant, les groupes seront affichés par ordre alphabétique ou numérique en fonction des valeurs de la première colonne. Voir Réorganiser un groupe pour savoir comment réorganiser votre structure de regroupement dans la page Données.

Le fichier CSV exporté présente 19 colonnes pour les propriétés du groupe et leurs valeurs dans l'ordre suivant :

  1. Nombre
  2. Nom cohérent
  3. Allow account assignment
  4. Type
  5. Signe
  6. Classification du groupe
  7. Ajuster la préférence du groupe
  8. Restrictions sur le solde du compte
  9. Lien vers un groupe dans la structure de regroupement du code des impôts
  10. Lien vers un groupe dans la structure de regroupement prescrite
  11. Cycle
  12. Zone
  13. Nom contraire
  14. Nom incohérent
  15. Répartition par dimension
  16. Inclure les dimensions
  17. Réaffectation de l'objectif pour les comptes contraires
  18. Groupe par défaut
  19. Composants
