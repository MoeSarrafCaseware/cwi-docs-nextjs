Ajouter, modifier et déplacer des groupes (page Balance de vérification)

Vous pouvez modifier les propriétés des groupes existants ou les réorganiser. Vous pouvez également ajouter des groupes.

Add groups (Trial Balance page)

Vous pouvez ajouter manuellement des groupes à une structure de regroupement.

Pour ajouter un groupe :

  1. Allez sur la page Balance de vérification , puis sélectionnez Ajouter () | Groupe.

    2. La boîte de dialogue Propriétés du groupe s'affiche.

  2. Définir les propriétés du nouveau groupe.

  1. Sélectionnez Enregistrer pour ajouter le groupe.

Edit groups (Trial Balance page)

Vous pouvez modifier les propriétés de n'importe quel groupe de la structure.

Pour modifier un groupe :

  1. Accédez à la page Balance de vérification , puis choisissez la structure de regroupement dans la liste déroulante située en haut.

    2. Vous pouvez choisir d'éditer pour le groupe Finances , le code fiscal ou le plan comptable .

  2. Sélectionnez le groupe que vous souhaitez modifier, puis sélectionnez Modifier ( ).

    3. La boîte de dialogue Propriétés du groupe s'affiche.

  3. Modifiez les propriétés du groupe si nécessaire.

    4. See Group Properties for a detailed description of each property.

  4. Une fois que vous avez terminé, sélectionnez Enregistrer.

Move groups (Trial Balance page)

Vous pouvez modifier la structure du regroupement en déplaçant les groupes.

To move a group:

  1. Sinon, reportez-vous à la page Balance de vérification .

  2. Sélectionnez le groupe que vous souhaitez déplacer, puis faites-le glisser et déposez-le à l'endroit souhaité dans la structure.

    3. Une grille rectangulaire s'affiche sur le lieu de destination pour identifier l'endroit ciblé par le groupe.

  3. Pour convertir le groupe en sous-groupe d'un groupe parent spécifique, faites-le glisser et déposez-le dans le groupe parent cible.

    Le parent cible est marqué d'un fond vert.
