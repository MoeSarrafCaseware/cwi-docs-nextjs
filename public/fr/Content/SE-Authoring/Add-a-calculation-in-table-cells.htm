Pour ajouter ou modifier un calcul dans les cellules d'un tableau :

Vous pouvez ajouter ou modifier des calculs dans les cellules des tableaux. Lorsque les utilisateurs importent des données, les cellules se remplissent automatiquement avec des valeurs calculées basées sur vos formules prédéfinies. Par exemple, vous pouvez ajouter des calculs de cellules dans votre modèle pour construire :

  • des notes complexes aux états financiers, telles que les lignes de mouvement.

  • (qui peuvent être utilisées comme configuration de base pour divers éléments de votre modèle).

  • movement rows in the Statement of Cash Flows.

  • les lignes d'analyse des ratios dans la section Bénéfice par action du compte de résultat.

Le calcul peut inclure des chiffres, des données provenant de la balance de vérification, des données provenant des états financiers, des valeurs de formulaire (si des formulaires sont disponibles dans votre produit) ou une combinaison de tous ces éléments.

Remarque : Cette fonction n'est disponible que pour les cellules des colonnes ou des lignes de type Entrée.

Pour ajouter ou modifier un calcul dans les cellules d'un tableau :

  1. Sélectionnez le tableau.

  2. Sélectionnez Autres actions (), puis choisissez Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Allez dans la cellule souhaitée et sélectionnez l'icône fx .

    Remarque : La cellule doit être une cellule de la ligne d'entrée ou de la colonne d' entrée.

  4. In the Calculation tab, enter your formula.

    1. Sélectionnez Lien () pour insérer un lien de données de balance générale dans la formule.

      2. La boîte de dialogue Data Link s'ouvre sur l'onglet Trial Balance. Le document en cours est sélectionné par défaut comme source des données de la balance de vérification.

      Vous pouvez modifier la valeur par défaut à l'aide du menu déroulant Source :

      • Balance de vérification - les données de la balance de vérification proviennent de la colonne Finale de la page Données. Il s'agit des montants définitifs après application des ajustements existants.

      • États financiers - les données de la balance de vérification proviennent des états financiers. Il s'agit des montants après l'application de l'arrondi et de toute dérogation manuelle.

      Lorsque vous insérez des liens de données, le préfixe () ou () indique la source des données.

      Notes :

      • L'onglet Référence de cellule vous permet de référencer des valeurs d'un tableau dynamique dans un autre. See Add cell references for more information.

      • Si des formulaires sont disponibles dans votre produit, l'onglet Formulaires est disponible dans la boîte de dialogue Liaison de données . Cet onglet permet d'insérer des valeurs à partir de formulaires.

    2. Sélectionnez Numbers () pour insérer des nombres dans la formule.

      Conseil : Vous pouvez utiliser votre clavier pour insérer des chiffres.

    3. Ajoutez les opérateurs souhaités pour construire la formule.

      Conseil : Vous pouvez utiliser votre clavier pour insérer des opérateurs.

    4. Sélectionnez FAIT pour fermer l'onglet Calcul.

      Remarque : Si vous avez saisi une formule non valide, l'icône fx s'affiche en rouge () pour indiquer une erreur.

As you're setting up the financial statements, you can review the calculation formulas you've added. Sélectionnez une cellule pour afficher les détails du calcul.

Les utilisateurs finaux peuvent modifier les calculs des cellules. Pour en savoir plus, voir Ajouter ou modifier un calcul dans les cellules du tableau des états financiers.
