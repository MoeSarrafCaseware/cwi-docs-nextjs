Ajouter une référence au document dans la question de requête

L'ajout d'une référence à un document permet d'insérer un lien vers un document existant dans le dossier de mission. Par exemple, vous pouvez ajouter un lien vers une lettre ou une liste de contrôle.

Ajouter une référence au document dans la question de requête

Vous pouvez ajouter une référence à un document dans la question d'interrogation.

Ajouter une référence au document dans la question de requête

  1. Sélectionnez l'onglet Factures .

  2. Sélectionnez le champ de texte pour afficher la barre d'outils de formatage.

  3. Sélectionnez Insérer une référence de document ().

    4. Une liste des documents disponibles dans le dossier d'engagement s'affiche. Vous pouvez sélectionner Afficher les documents cachés pour sélectionner un document caché si nécessaire.

  4. Sélectionnez le document que vous souhaitez référencer dans la liste.

    5. Une référence au document sélectionné est créée et ajoutée dans l'onglet Orientation.

Ajouter une référence au document dans la question de requête

Vous pouvez ajouter une référence à un document dans la question d'interrogation.

Pour ajouter une référence au document dans la question de requête :

  1. Accédez à la question de la requête dans laquelle vous souhaitez insérer une référence de document.

  2. Sélectionnez le champ de texte pour afficher la barre d'outils de formatage.

  3. Sélectionnez Insérer une référence de document ().

    4. Une liste des documents disponibles dans le dossier d'engagement s'affiche. Vous pouvez sélectionner Afficher les documents cachés pour sélectionner un document caché si nécessaire.

  4. Sélectionnez le document que vous souhaitez référencer dans la liste.

    5. Une référence au document sélectionné est créée et ajoutée à la question de la requête.

Ajouter une référence de document dans l'onglet Guidance ou une description de procédure

Vous pouvez ajouter une référence à un document dans les informations d'orientation d'un groupe, d'un sous-groupe ou d'une procédure.

Ajouter une référence de document dans l'onglet Guidance ou une description de procédure

  1. Accédez à la ligne où vous souhaitez ajouter une référence de note.

    2. L'élément peut être un groupe, un sous-groupe ou une procédure.

  2. Sélectionnez l'icône de l'ampoule () à côté de l'élément.

    3. La section Orientation s'ouvre dans le panneau latéral gauche.

  3. Sélectionnez le champ de texte pour afficher la barre d'outils de formatage.

  4. Sélectionnez Insérer une référence de document ().

    5. Une liste des documents disponibles dans le dossier d'engagement s'affiche. Vous pouvez sélectionner Afficher les documents cachés pour sélectionner un document caché si nécessaire.

  5. Sélectionnez le document que vous souhaitez référencer dans la liste.

    6. Une référence au document sélectionné est créée et ajoutée dans l'onglet Orientation.
