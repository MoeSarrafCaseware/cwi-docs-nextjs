Ajouter un en-tête et un pied de page dans une lettre ou un mémo

Vous pouvez ajouter des informations sur l'engagement dans l'en-tête et le pied de page de votre lettre ou de votre note. Par exemple, vous pouvez ajouter le nom de l'engagement dans l'en-tête et le nom du produit dans le pied de page.

Ajouter un en-tête ou un pied de page

Vous pouvez également personnaliser l'en-tête ou le pied de page pour chaque zone de regroupement, saut de page ou table des matières.

Pour ajouter un en-tête et un pied de page dans une lettre ou un mémo :

  1. Sélectionnez l'option Plus d'actions () | Paramètres d'impression.

  2. Saisissez le contenu de l'en-tête et du pied de page dans les zones de texte En-tête et Pied de page .

Pour ajouter un en-tête ou un pied de page personnalisé à une zone de regroupement, à un saut de page ou à la table des matières :

  1. Accédez à la zone de regroupement, au saut de page ou à la table des matières souhaités.

  2. Sélectionnez l'option Paramètres () | Modifier les paramètres d'impression.

  3. Sélectionnez En-tête personnalisé et Pied de page personnalisé pour afficher les champs de texte de l'en-tête ou du pied de page.

  4. Saisissez le contenu de l'en-tête et du pied de page dans les champs de texte.

    5. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour personnaliser le contenu de l'en-tête et du pied de page.

Ajouter des liens de données dans l'en-tête ou le pied de page

Les liens de données () vous permettent de récupérer automatiquement des informations déjà existantes dans l'engagement ou dans le nuage et de les insérer dans des champs de texte.

Vous pouvez ajouter les liens de données suivants :

  • Propriétés d'engagement

    • Propriétés d'engagement telles que définies dans le nuage, telles que

    • Nom d'exploitation
    • Nom légal
    • Date de fin d'année
    • Date de début de l'année
    • Année en cours
    • Site web
    • Année précédente
    • Adresse de l'entité

  • Balance

    Éléments de la balance générale dans la mission, tels que : nom du compte, nom du groupe ou valeurs de l'année en cours ou de l'année précédente.

  • Propriétés fermes

    Propriétés fermes telles que définies dans le Cloud :

    • Nom du cabinet
    • Adresse du cabinet

  • Réponses

    Réponses aux procédures à partir de listes de contrôle. Une fois que vous avez sélectionné un document de la liste de contrôle dans la liste, sélectionnez la réponse que vous souhaitez récupérer et insérer dans la section texte.

  • Matérialité

    Valeurs définies dans le document de matérialité .

  • Glossaire de l'engagement

    Ajouter un terme du glossaire del' engagement.

    Voir Ajouter un terme dans le glossaire de l'engagement.

  • Texte dynamique

    Un texte dynamique est un texte conditionnel. Il s'agit d'un champ dans lequel un texte différent s'affiche en fonction d'une condition ou d'un ensemble de conditions.

    Voir Ajout d'un texte dynamique.

  • Valeur des formulaires

    Insérer la valeur d'un champ de formulaire spécifique. Lorsque vous sélectionnez un formulaire et un champ, la valeur du champ sélectionné est insérée dans le champ de texte.

    Remarque : Cette option n'est disponible dans la boîte de dialogue Data Link que si vous avez développé des formulaires à l'aide de SE Builder dans votre produit. Pour en savoir plus, voir Développer des formulaires personnalisés.
