Ajouter un saut de page dans les états financiers

Vous pouvez également ajouter un saut de page à l'intérieur d'un tableau dans le document afin de mieux présenter les informations du tableau. Par exemple, vous pouvez insérer un saut de page dans un tableau pour que des informations similaires ou connexes soient affichées sur la même page au lieu d'être réparties sur deux pages.

Pour ajouter un saut de page dans les tableaux :

  1. Sélectionnez le tableau souhaité.

    2. Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez l'option Plus d'actions () :

  3. Sélectionnez Modifier le tableau dynamique dans la liste des actions.

    4. Le tableau s'ouvre en mode édition.

  4. Localisez l'endroit où vous souhaitez ajouter un saut de page.

  5. Sélectionnez l'option Ajouter () | Saut de page sur le bord gauche du tableau.

  6. Sélectionnez SAVE ou SAVE & CLOSE.
