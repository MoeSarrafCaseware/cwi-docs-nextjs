Ajouter un saut de page dans les états financiers

Vous pouvez ajouter un saut de page n'importe où dans le document pour marquer manuellement l'endroit où une page spécifique se termine et où une nouvelle page commence. Cela vous permet de contrôler la mise en page pour l'impression. Vous pouvez également utiliser un saut de page pour marquer un endroit où vous prévoyez d'ajouter du contenu, pour créer un espace vide entre deux zones ou pour personnaliser l'en-tête ou le pied de page d'une page spécifique du document.

Ajouter un saut de page

  1. Localisez l'endroit où vous souhaitez ajouter votre saut de page.

  2. Allez sur le côté gauche et sélectionnez l'icône Ajouter () pour ajouter un nouveau contenu.

  3. Choisissez Saut de page dans la liste des types de contenu.
