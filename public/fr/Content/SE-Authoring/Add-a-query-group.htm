Ajouter un groupe d'imposition

Les groupes permettent d'organiser les requêtes dans le document en regroupant les requêtes apparentées. Les ensembles de questions sont obligatoires dans le document de requête. Vous ne pouvez pas ajouter une requête sans groupe.

Pour ajouter un groupe à votre document, sélectionnez + Groupe. Un nouvel ensemble de questions est ajouté au document avec un titre par défaut Nouveau groupe. Vous pouvez sélectionner le titre par défaut pour le modifier.

Vous pouvez supprimer un groupe en sélectionnant Autres actions ()| Supprimer.

Remarque : If you delete a group, all queries in this group are deleted as well.
