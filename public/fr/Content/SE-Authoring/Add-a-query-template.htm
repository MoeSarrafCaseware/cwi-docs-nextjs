Ajouter une question de requête

Vous pouvez utiliser les documents de requête pour collecter des informations auprès des clients. Vous pouvez créer une norme pour les demandes des clients en créant des modèles de requête. Les utilisateurs peuvent ajouter un document de requête dans le fichier d'engagement en utilisant le modèle de requête.

Meilleure pratique : Nous vous recommandons de créer des documents de requête à partir du modèle pour les utilisateurs plutôt que de leur demander de le faire eux-mêmes. Vous pouvez ainsi inclure le document de requête dans le flux de travail du modèle de dossier d'engagement et lui appliquer la logique de visibilité .

Une requête est un document intelligent interactif qui se compose des éléments suivants :

  • Questions - Les questions auxquelles les clients doivent répondre.

  • Groupes - Pour regrouper des requêtes connexes.

  • Types de réponses - Pour spécifier la réponse attendue de l'utilisateur. Le tableau suivant décrit l'utilisateur attendu pour chaque type de réponse.

    Type de réponse

    Réponse attendue de l'utilisateur

    Manuel

    Saisissez les informations demandées.

    Liste de sélection

    Sélectionnez un élément dans la liste déroulante.

    Multi-Picklist

    Sélectionnez un ou plusieurs éléments dans la liste déroulante.

    Pour télécharger des fichiers :

    Parcourir et ajouter un fichier demandé (ou glisser-déposer le fichier).

    Date

    Sélectionnez une date dans le calendrier.

  • Orientation - Fournir des informations d'orientation aux utilisateurs. Par exemple, ajouter les types de fichiers pris en charge pour une demande de document.

Ajouter une question de requête

  1. Sélectionnez Ajouter ( ) dans la page des documents, puis choisissez Liste de contrôle.

    2. Un document vierge s'ouvre avec une section d'orientation en haut.

  2. Sélectionnez Editer ( ) pour commencer la rédaction.

Pour ajouter une requête à un document:

  1. Sélectionnez Ajouter ( ) dans la page des documents, puis choisissez Liste de contrôle.

  2. Sélectionnez un modèle de requête existant à partir duquel vous créerez votre document de requête.

  3. Ajouter un nom pour le document de requête.

    4. Vous pouvez éventuellement ajouter un numéro au document.

  4. Sélectionnez Créer une requête.

    5. Un document vierge s'ouvre avec une section d'orientation en haut.

  5. Sélectionnez Editer ( ) pour commencer la rédaction.
