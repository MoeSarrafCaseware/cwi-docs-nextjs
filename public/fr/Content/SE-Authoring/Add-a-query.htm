Ajouter une requête

A query is a client request. A query must be part of a group. Vous ne pouvez pas ajouter une requête dans le document sans groupe.

Pour ajouter une question de requête :

  1. Accédez au groupe dans lequel vous souhaitez ajouter des données.

  2. Sélection + requête.

    3. Une nouvelle requête est ajoutée avec une description par défaut Nouvelle requête et un type de réponse Téléchargement de fichier.

  3. Sélectionnez la description par défaut pour la modifier.

    4. Une fois le champ de texte sélectionné, la barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour effectuer les modifications suivantes :

Ajouter un lien de données dans la description de la procédure

Les liens de données () vous permettent de récupérer automatiquement des informations déjà existantes dans l'engagement ou dans le nuage et de les insérer dans des champs de texte.

Vous pouvez ajouter les liens de données suivants :

  • Propriétés d'engagement

    • Propriétés d'engagement telles que définies dans le nuage, telles que

    • Nom d'exploitation
    • Nom légal
    • Date de fin d'année
    • Date de début de l'année
    • Année en cours
    • Site web
    • Année précédente
    • Adresse de l'entité

  • Balance

    Éléments de la balance générale dans la mission, tels que : nom du compte, nom du groupe ou valeurs de l'année en cours ou de l'année précédente.

  • Propriétés fermes

    Propriétés fermes telles que définies dans le Cloud :

    • Nom du cabinet
    • Adresse du cabinet

  • Réponses

    Réponses aux procédures à partir de listes de contrôle. Une fois que vous avez sélectionné un document de la liste de contrôle dans la liste, sélectionnez la réponse que vous souhaitez récupérer et insérer dans la section texte.

  • Matérialité

    Valeurs définies dans le document de matérialité .

  • Glossaire de l'engagement

    Ajouter un terme du glossaire del' engagement.

    Voir Ajouter un terme dans le glossaire de l'engagement.

  • Texte dynamique

    Un texte dynamique est un texte conditionnel. Il s'agit d'un champ dans lequel un texte différent s'affiche en fonction d'une condition ou d'un ensemble de conditions.

    Voir Ajout d'un texte dynamique.

  • Valeur des formulaires

    Insérer la valeur d'un champ de formulaire spécifique. Lorsque vous sélectionnez un formulaire et un champ, la valeur du champ sélectionné est insérée dans le champ de texte.

    Remarque : Cette option n'est disponible dans la boîte de dialogue Data Link que si vous avez développé des formulaires à l'aide de SE Builder dans votre produit. Pour en savoir plus, voir Développer des formulaires personnalisés.

Ajouter une référence au document dans la question de requête

L'ajout d'une référence à un document permet d'insérer un lien vers un document existant dans le dossier de mission. Vous pouvez ajouter une référence à un document dans la question d'interrogation. Par exemple, vous pouvez ajouter un lien vers une lettre ou une liste de contrôle.

Pour ajouter une référence au document dans la question de requête :

  1. Accédez à la question de la requête dans laquelle vous souhaitez insérer une référence de document.

  2. Sélectionnez le champ de texte pour afficher la barre d'outils de formatage.

  3. Sélectionnez Insérer une référence de document ().

  4. Une liste des documents disponibles dans le dossier d'engagement s'affiche. Vous pouvez sélectionner Afficher les documents cachés pour sélectionner un document caché si nécessaire.

  5. Sélectionnez le document que vous souhaitez référencer dans la liste.

    6. Une référence au document sélectionné est créée et ajoutée à la question de la requête.
