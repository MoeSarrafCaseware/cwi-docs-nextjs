Ajouter des jeux de questions

Les séries de questions peuvent vous aider à organiser vos demandes de renseignements et vos requêtes. Utilisez les ensembles de questions si vous souhaitez regrouper des questions liées dans le document. Les ensembles de questions sont obligatoires dans le document de requête. Vous ne pouvez pas ajouter une question isolée dans le document, elle doit faire partie d'un ensemble de questions.

Pour ajouter un ensemble de questions, sélectionnez Ajouter () dans le panneau de navigation de gauche. Un nouvel ensemble de questions est ajouté au document avec un titre par défaut Nouveau groupe. Vous pouvez sélectionner le titre par défaut pour le modifier ; le titre peut être un lien de données. Vous pouvez supprimer un ensemble de questions en le survolant et en sélectionnant l'icône () ou en le sélectionnant puis en sélectionnant Autres actions () | Supprimer.

Remarque : Si vous supprimez un ensemble de questions, toutes les questions de cet ensemble sont également supprimées.
