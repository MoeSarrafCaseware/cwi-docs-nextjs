Ajouter un type de réponse à une procédure

Vous pouvez modifier les options du type de réponse par défaut ou le supprimer et en ajouter un autre.

Pour ajouter une réponse dans une procédure :

  1. Sélectionnez la procédure ou la sous-procédure.

  2. Sélectionnez Autres actions () | Paramètres, puis cochez la case Remplacer les paramètres de la liste de contrôle .

  3. Sélectionnez +Réponse.

  4. Saisissez le texte de l'espace réservé.

    5. Il s'agit du texte par défaut qui apparaît dans le champ de réponse avant que l'utilisateur ne saisisse ou ne sélectionne une réponse.

  5. Sélectionnez un type de réponse.

    Si vous avez sélectionné une liste de sélection, une liste multiple, des informations sur l'entité ou des informations sur le contact, vous devez également définir les options de la réponse.

    1. Sélectionnez l'icône Editer ().

    2. Modifiez les options existantes ou sélectionnez Supprimer () pour les supprimer.

    3. Sélectionnez + Option pour ajouter une nouvelle option à la liste.

    4. Sélectionnez Autre pour inclure Autre dans la liste des options.

      Un champ de texte est également inclus dans cette option pour permettre aux utilisateurs d'ajouter éventuellement des informations supplémentaires s'ils choisissent Autre comme réponse.

    5. Sélectionnez Afficher en ligne pour afficher les réponses les unes à côté des autres dans la procédure plutôt que dans une liste déroulante.

    6. Utilisez la flèche vers le haut () et la flèche vers le bas () pour réorganiser l'ordre d'apparition des options de réponse.

    7. Une fois que vous avez terminé, sélectionnez la coche () pour enregistrer vos modifications.

    Notez que vous pouvez également définir la logique de visibilité pour les options de réponse de la liste de sélection.

  1. Une fois que vous avez terminé, sélectionnez Enregistrer.
