Le modèle est également disponible au téléchargement dans la boîte de dialogue Version inédite.

Si vous ajoutez une version de la bibliothèque des risques pour la première fois, la version est automatiquement ajoutée en tant que version majeure. Vous pouvez également ajouter des versions ultérieures de la bibliothèque des risques : Mineur ou correctif. Utilisez les versions mineures pour les mises à jour planifiées de la bibliothèque et les versions correctives pour les mises à jour urgentes. Cela vous permet de travailler simultanément sur une mise à jour planifiée et de publier des mises à jour critiques.

Remarque : Vous ne pouvez avoir qu'une seule version mineure non publiée et qu'une seule version de correctif non publiée à la fois.

Pour commencer, sélectionnez Settings dans le menu Cloud(), puis sélectionnez Risk Library | Manage Risk Library.

Pour ajouter une version de la bibliothèque des risques :

  1. Sélectionnez la bibliothèque de risques dans le volet de gauche.

  2. Si vous ajoutez la première version de cette bibliothèque de risques, sélectionnez CRÉER UNE VERSION puis passez à l'étape 4

  3. S'il ne s'agit pas de la première version de cette bibliothèque de risques :

    1. Sélectionner ADD.

      La boîte de dialogue Version inédite s'ouvre.

    2. Sélectionnez le type d'extraction.

      Options disponibles :

      • Mineur - Sélectionnez cette option pour les mises à jour planifiées.

      • Patch - Sélectionnez cette option pour les mises à jour urgentes.

  4. Select the Upload Excel File box to browse or drag and drop the file.

    Vous pouvez sélectionner TÉLÉCHARGER LE MODÈLE pour télécharger un modèle vide que vous pouvez utiliser pour configurer votre fichier Excel de bibliothèque de risques.

  5. Sélectionner CREATE.

    Des erreurs et des avertissements s'affichent dans la boîte de dialogue - en cas de problèmes avec les données du fichier Excel.

    Vous pouvez ignorer les avertissements, mais vous devez corriger toutes les erreurs et télécharger à nouveau le fichier pour continuer.

Une fois la version ajoutée, vous pouvez également :

  • Sélectionnez Editer () pour remplacer le fichier Excel.

  • Sélectionnez Rapport de synthèse () pour afficher un résumé du contenu de la bibliothèque des risques.

  • Sélectionnez Oui pour supprimer la question.
