Pour ajouter une section Analyse :

Vous pouvez ajouter une section d'analyse dans les états financiers afin d'inclure une représentation visuelle des données au moyen de tableaux, de graphiques et de diagrammes.

Vous pouvez spécifier les types de graphiques et de diagrammes disponibles dans la section d'analyse, ainsi que les données qu'ils présentent dans l'API du produit. Pour en savoir plus, consultez le rapport Controls.

Pour ajouter une section Analyse :

  1. Localisez l'endroit où vous souhaitez ajouter la section d'analyse.

    2. Vous pouvez ajouter une section d'analyse dans un domaine, une note ou un en-tête de note.

  2. Allez sur le côté gauche de la page et sélectionnez Ajouter ().

  3. Sélectionnez Analyse dans la liste des types de contenu.

    4. Une liste des tableaux, diagrammes et graphiques disponibles s'affiche. Pour personnaliser cette liste, voir Tableaux d'analyse.

  4. Sélectionnez la figure souhaitée pour l'ajouter à la section d'analyse.

    5. Remarque : Les données affichées dans la section d'analyse reflètent les soldes des états financiers et non la balance de vérification. Les soldes des états financiers peuvent inclure des valeurs et des ajustements ignorés.

Vous pouvez apporter quelques modifications aux figures pour personnaliser l'aspect de la section et la présentation des données.

Pour personnaliser la section Analyse :

  1. Dans le menu déroulant Sélectionner un graphique , choisissez les chiffres que vous souhaitez inclure dans la section.

    2. Conseil : Si vous sélectionnez accidentellement une figure et que vous décidez de la supprimer, il vous suffit de la fermer () pour qu'elle soit supprimée.

  2. Utilisez le commutateur préliminaire/final () pour choisir si les données préliminaires ou finales doivent être représentées par défaut dans les figures.

  3. Sélectionnez l'icône Taille réelle () pour afficher la figure en taille réelle dans toute la section.

  4. Sélectionnez l'icône Modifier () dans la barre de titre de la figure, puis sélectionnez l'icône Ajouter () pour ajouter un élément au graphique. Sélectionnez un élément dans la liste affichée pour l'ajouter à la figure.

  5. Dans les tableaux, sélectionnez le bouton COLONNES pour spécifier les colonnes que vous souhaitez afficher dans un tableau spécifique.

  6. Si vous souhaitez rétablir l'état initial d'une figure, sélectionnez l'icône Rétablir les paramètres par défaut ().
