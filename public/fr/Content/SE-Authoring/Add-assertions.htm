Pour ajouter des assertions :

Vous pouvez ajouter une assertion à votre produit SE dans Fonctionnalités | Assertions.

Remarque : Assurez-vous que les assertions sont activées dans le fichier product.json, sinon l'onglet Assertions ne sera pas disponible. Pour en savoir plus, voir Activer ou désactiver des fonctionnalités dans votre produit.

Under the Features drop-down, select Assertions.

Pour ajouter une assertion :

  1. Sélectionnez Ajouter une condition.

  2. Saisir l'abréviation et le nom complet de l'assertion.

Notez que toutes les assertions doivent être associées à l'une des 12 assertions de base ; les assertions non associées seront supprimées.
