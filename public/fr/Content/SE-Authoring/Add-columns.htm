Ajouter des colonnes

Ajoutez des colonnes pour définir les valeurs des données de votre tableau. Vous pouvez également supprimer des colonnes si nécessaire.

Remarque : La taille de police par défaut pour le contenu des tableaux dynamiques est de 12 et ne peut être modifiée.

To add a column:

  1. Sélectionnez le tableau.

  2. Sélectionnez Autres actions (), puis choisissez Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Allez à l'endroit où vous souhaitez ajouter une colonne.

  4. Sélectionnez Ajouter () dans la partie supérieure du tableau.

    5. Une colonne est ajoutée au tableau et les paramètres de la colonne s'affichent sur la page.

Spécifier les paramètres de la colonne

Vous devez définir diverses options pour chaque colonne du tableau afin de définir les valeurs des données.

Pour configurer les colonnes :

  1. Sélectionnez l'en-tête de la colonne.

  2. Modifiez les paramètres suivants si nécessaire :

  1. Sélectionnez DONE.

  2. Sélectionnez SAVE ou SAVE & CLOSE.

Modifier la description des colonnes

Vous pouvez également modifier les descriptions des colonnes afin d'obtenir des titres personnalisés pour les colonnes du tableau.

Pour modifier les descriptions des colonnes :

  1. Sélectionnez le tableau.

  2. Sélectionnez Autres actions (), puis choisissez Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Sélectionnez la colonne que vous souhaitez modifier.

  4. Sélectionnez le champ Intitulé .

    5. La boîte de dialogue Data Link s'affiche.

  5. Dans le champ Sortie , modifiez le nom comme vous le souhaitez.

    6. Vous pouvez également sélectionner l'icône Data Link () pour que la description du tableau comprenne un texte dynamique.

  6. Sélectionner DONE.

  7. Sélectionnez SAVE ou SAVE & CLOSE.
