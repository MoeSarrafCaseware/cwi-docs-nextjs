Ajouter du texte dans une lettre ou un mémo

Vous pouvez inclure des informations d'orientation dans une lettre ou un mémo. Les informations d'orientation peuvent être des hypothèses, des instructions ou une référence à un autre document.

Pour ajouter des informations d'orientation :

Vous pouvez créer une zone d'orientation en haut du document afin d'inclure des informations pour l'ensemble de la lettre ou du mémo. Vous pouvez également ajouter une zone d'orientation n'importe où dans le corps du document pour inclure des informations d'orientation relatives à des sections spécifiques du document.

Pour ajouter des informations d'orientation :

  1. Sélectionnez +Ajouter une section | Orientation.

    2. Un champ de texte s'affiche avec une ampoule () qui indique que la zone est une information de guidage.

    Le titre par défaut de la section "Conseils" est " Conseils". Vous pouvez sélectionner le titre par défaut pour le modifier. Notez que ce titre ne fait pas partie du contenu du document.

  2. Sélectionnez le champ de texte pour commencer à ajouter du contenu. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour effectuer les modifications suivantes :

Pour supprimer la section de guidage, sélectionnez Paramètres () | Supprimer.

Ajouter une référence au document dans la question de requête

L'ajout d'une référence à un document permet d'insérer un lien vers un document existant dans le dossier de mission. Vous pouvez ajouter une référence à un document dans la question d'interrogation. For example, you can add a reference to a Query document.

Insérer une référence à un document dans le guide :

  1. Sélectionnez le champ de texte pour afficher la barre d'outils de formatage.

  2. Sélectionnez Insérer une référence de document ().

    3. Une liste des documents disponibles dans le dossier d'engagement s'affiche. Vous pouvez sélectionner Afficher les documents cachés pour sélectionner un document caché si nécessaire.

  3. Sélectionnez le document que vous souhaitez référencer dans la liste.

    4. Une référence au document sélectionné est créée et ajoutée dans l'onglet Orientation.

Ajouter ou supprimer un lien dans la description de la procédure

Ajouter un lien de données dans la description de la procédure

Les liens de données () vous permettent de récupérer automatiquement des informations déjà existantes dans l'engagement ou dans le nuage et de les insérer dans des champs de texte.

Vous pouvez ajouter les liens de données suivants :

  • Propriétés d'engagement

    • Propriétés d'engagement telles que définies dans le nuage, telles que

    • Nom d'exploitation
    • Nom légal
    • Date de fin d'année
    • Date de début de l'année
    • Année en cours
    • Site web
    • Année précédente
    • Adresse de l'entité

  • Balance

    Éléments de la balance générale dans la mission, tels que : nom du compte, nom du groupe ou valeurs de l'année en cours ou de l'année précédente.

  • Propriétés fermes

    Propriétés fermes telles que définies dans le Cloud :

    • Nom du cabinet
    • Adresse du cabinet

  • Réponses

    Réponses aux procédures à partir de listes de contrôle. Une fois que vous avez sélectionné un document de la liste de contrôle dans la liste, sélectionnez la réponse que vous souhaitez récupérer et insérer dans la section texte.

  • Matérialité

    Valeurs définies dans le document de matérialité .

  • Glossaire de l'engagement

    Ajouter un terme du glossaire del' engagement.

    Voir Ajouter un terme dans le glossaire de l'engagement.

  • Texte dynamique

    Un texte dynamique est un texte conditionnel. Il s'agit d'un champ dans lequel un texte différent s'affiche en fonction d'une condition ou d'un ensemble de conditions.

    Voir Ajout d'un texte dynamique.

  • Valeur des formulaires

    Insérer la valeur d'un champ de formulaire spécifique. Lorsque vous sélectionnez un formulaire et un champ, la valeur du champ sélectionné est insérée dans le champ de texte.

    Remarque : Cette option n'est disponible dans la boîte de dialogue Data Link que si vous avez développé des formulaires à l'aide de SE Builder dans votre produit. Pour en savoir plus, voir Développer des formulaires personnalisés.
