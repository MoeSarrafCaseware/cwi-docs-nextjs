Ajouter des informations d'orientation dans une liste de contrôle

Vous pouvez inclure des informations d'orientation pour les utilisateurs dans la liste de contrôle. Les informations d'orientation peuvent être des instructions ou des références à d'autres documents.

Pour ajouter des informations d'orientation générale :

Vous pouvez ajouter des informations générales dans la liste de contrôle pour les utilisateurs dans l'onglet Conseils.

Pour ajouter des informations d'orientation générale :

  1. Allez dans l'onglet Orientation.

  2. Sélectionnez le champ de texte pour commencer à ajouter du contenu. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour effectuer les modifications suivantes :

    Icône de la barre d'outils

    Description

    Mettre en gras, en italique ou souligner le texte sélectionné.

    Ajouter une liste de puces ou de numéros.

    Mettre en retrait le texte ou le paragraphe sélectionné.

    Aligner le texte ou le paragraphe sélectionné.

    Ajouter une table.

    Ajouter ou supprimer un lien.

    Insérer une référence à un document visible ou caché dans le dossier de mission.

    Supprime toutes les mises en forme d'un texte ou d'un paragraphe sélectionné et rétablit la valeur par défaut.

Pour ajouter des informations d'orientation dans un groupe, un sous-groupe ou une procédure :

You can include guidance information for users in each group and procedure in the checklist. Les informations d'orientation peuvent être des instructions ou des références à d'autres documents.

Pour ajouter des informations d'orientation dans un groupe, un sous-groupe ou une procédure :

  1. Accédez à l'élément dans lequel vous souhaitez ajouter des informations d'orientation.

    2. L'élément peut être un groupe, un sous-groupe ou une procédure.

  2. Sélectionnez l'icône de l'ampoule () à côté de l'élément.

    3. La section Orientation s'ouvre dans le panneau latéral gauche.

  3. Sélectionnez le champ de texte pour commencer à ajouter du contenu. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour effectuer les modifications suivantes :
