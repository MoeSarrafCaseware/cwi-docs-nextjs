Add note references in tables

Une référence de note est un lien vers une ou plusieurs notes existantes dans l'état financier. Vous pouvez faire référence aux notes dans les lignes de tableau, y compris les lignes de total, dans les états financiers. Par exemple, pour relier un compte ou un groupe à une note existante contenant des détails ou des informations connexes.

Ajouter une référence de note dans une ligne

Vous pouvez également faire référence à des notes dans des colonnes.

Pour ajouter une référence de note dans une ligne :

  1. Sélectionnez le tableau.

    Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez Autres actions (), puis choisissez Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Accédez à la ligne où vous souhaitez ajouter une référence de note.

  4. Sélectionnez le bouton Ajouter () au-dessus de la ligne.

  5. Sélectionnez la ligne Commentaire.

    6. Une note de référence est ajoutée juste au-dessus de la ligne.

  6. Sélectionnez(#) pour afficher une liste des notes existantes dans le document des états financiers.

    Conseil : Si vous avez défini une logique de visibilité pour les notes, cochez la case Afficher les notes cachées pour inclure les notes cachées dans la liste.

  7. Sélectionnez une note dans la liste pour la référencer dans la ligne.

  8. Sélectionnez SAVE ou SAVE & CLOSE.

    9. Le numéro de la note référencée s'affiche à côté du nom de la ligne.

Pour ajouter une référence de note dans une colonne :

Vous pouvez faire référence aux colonnes de notes dans le document des états financiers.

Pour ajouter une référence de note dans une colonne :

  1. Ajouter une colonne dans le tableau.

    1. Sélectionnez le tableau.

      Les icônes des options du tableau s'affichent à droite.

    2. Sélectionnez Autres actions (), puis choisissez Modifier le tableau dynamique.

      3. Le tableau s'ouvre en mode édition.

    3. Allez à l'endroit où vous souhaitez ajouter une colonne.

    4. Sélectionnez Ajouter () dans la partie supérieure du tableau.

      5. Une colonne est ajoutée au tableau et les paramètres de la colonne s'affichent sur la page.

  2. Sélectionnez l'en-tête de la nouvelle colonne et modifiez le type de colonne en Référence de note.

  3. Saisissez un en-tête et spécifiez la largeur souhaitée.

  4. Sélectionnez Terminé.

    5. La nouvelle colonne est ajoutée dans le tableau avec des références vides(#).

  5. Spécifiez les références des notes pour chaque ligne.

    1. Sélectionnez une référence vide(#) pour afficher une liste des notes existantes dans le document des états financiers.

      Conseil : Si vous avez défini une logique de visibilité pour les notes, cochez la case Afficher les notes cachées pour inclure les notes cachées dans la liste.

    2. Sélectionnez une ou plusieurs notes dans la liste.

  6. Sélectionnez SAVE ou SAVE & CLOSE.
