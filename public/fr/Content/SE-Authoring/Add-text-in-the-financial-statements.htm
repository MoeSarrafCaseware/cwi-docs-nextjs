Personnaliser le texte dans les états financiers

Vous pouvez configurer le contenu du texte dans le document des états financiers pour créer un libellé et un format de texte standard. Ajouter des sections de texte dans le document pour contenir le contenu du texte.

Pour ajouter une section de texte dans une zone d'états financiers :

  1. Sélectionnez Ajouter (), puis choisissez Section de texte.

  2. Sélectionnez la section de texte pour commencer à modifier le texte comme vous le souhaitez.

    3. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour effectuer les modifications suivantes :

Icône de la barre d'outils

Description

Appliquer un style au texte ou au paragraphe sélectionné.

La taille de police par défaut pour chaque style est la suivante :

  • Paragraphe - 11pt

  • Rubrique 1 - 27pt

  • Rubrique 2 - 22.5pt

  • Titre 3 - 18pt

  • Titre 4 - 13.5pt

  • Rubrique 5 - 10.5pt

  • Titre 6 - 9pt

Pour savoir comment personnaliser la taille de la police des titres, voir Personnaliser les polices et les titres.

Pour éviter les sauts de page dans une zone de texte lors de l'impression, sélectionnez Autres actions () et activez l'option Conserver ensemble. Notez que la fonction Garder ensemble n'empêche pas les sauts de page si la zone de texte est plus longue qu'une page entière.

Si vous souhaitez couper le texte d'une section existante et le coller dans une nouvelle section, sélectionnez Autres actions () | Couper la section. Vous pouvez ensuite sélectionner l'option Coller la section lors de l'ajout d'une nouvelle section pour coller la section de texte. La section sera retirée de son emplacement d'origine et déplacée vers la nouvelle section. L'utilisation de cette option, au lieu de copier manuellement la section, garantit que la section de texte collée est reportée correctement.

To delete a text area, select More actions () | Delete.
