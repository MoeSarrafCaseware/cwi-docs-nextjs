Ajouter une logique de visibilité pour une requête

Les paramètres de visibilité permettent une automatisation intégrée afin que seuls les documents ou contenus pertinents soient disponibles dans votre dossier d'engagement. Pour ce faire, des conditions de visibilité sont définies pour les différents éléments de l'engagement.

Vous pouvez définir des conditions de visibilité pour les groupes, les sous-groupes, les procédures ou les sous-procédures afin que votre liste de contrôle n'affiche que le contenu pertinent pour l'engagement. Par exemple, si le solde d'un groupe de balances de vérification n'est pas significatif, vous pouvez masquer le groupe de procédures correspondant.

Pour en savoir plus, voir Qu'est-ce que Caseware ReviewComp ?

Ajouter une logique de visibilité pour le document

Vous pouvez ajouter une logique de visibilité pour une réponse de liste de sélection individuelle.

Pour définir la visibilité du document :

  1. Sélectionnez le nom du document.

    2. La boîte de dialogue Propriétés du document s'affiche.

  2. Sélectionnez Modifier la visibilité pour afficher la boîte de dialogue Paramètres de visibilité .

  3. Sélectionnez Masquer ou Afficher dans la liste déroulante bleue pour spécifier l'action de visibilité.

  4. Sélectionnez tout ou n'importe quelle condition pour déterminer si toutes les conditions doivent être remplies ou si une seule condition doit être remplie pour que l'action de visibilité (de l'étape 3) s'applique.

  5. Sélectionnez Ajouter une condition.

  6. Choisissez un type de condition dans la liste, puis spécifiez les paramètres de la condition.

  7. Une fois que vous avez terminé, sélectionnez Enregistrer.

Ajouter une logique de visibilité pour une requête

You can add visibility logic for a group or a subgroup.

Pour définir la visibilité d'un groupe ou d'un sous-groupe :

  1. Sélectionnez le groupe ou le sous-groupe pour lequel vous souhaitez définir la visibilité.

  2. Sélectionnez Autres actions () | Modifier la visibilité pour afficher la boîte de dialogue Paramètres de visibilité .

  3. Sélectionnez Masquer ou Afficher dans la liste déroulante bleue pour spécifier l'action de visibilité.

  4. Sélectionnez tout ou n'importe quelle condition pour déterminer si toutes les conditions doivent être remplies ou si une seule condition doit être remplie pour que l'action de visibilité (de l'étape 3) s'applique.

  5. Sélectionnez Ajouter une condition.

  6. Choisissez un type de condition dans la liste, puis spécifiez les paramètres de la condition.

  7. Une fois que vous avez terminé, sélectionnez Enregistrer.

Add visibility logic for a procedure or a subprocedure

You can add visibility logic for a procedure or a subprocedure.

Pour définir la visibilité d'une procédure ou d'une sous-procédure :

  1. Sélectionnez la procédure ou la sous-procédure pour laquelle vous souhaitez définir la visibilité.

  2. Sélectionnez Autres actions () | Paramètres.

  3. Sélectionnez Modifier la visibilité pour afficher la boîte de dialogue Paramètres de visibilité .

    4. Vous pouvez également sélectionner l'option Visibilité () | Paramètres ().

  4. Sélectionnez Masquer ou Afficher dans la liste déroulante bleue pour spécifier l'action de visibilité.

  5. Sélectionnez tout ou n'importe quelle condition pour déterminer si toutes les conditions doivent être remplies ou si une seule condition doit être remplie pour que l'action de visibilité (de l'étape 4) s'applique.

  6. Sélectionnez Ajouter une condition.

  7. Choisissez un type de condition dans la liste, puis spécifiez les paramètres de la condition.

  8. Une fois que vous avez terminé, sélectionnez Enregistrer.

Vous pouvez sélectionner Effacer pour effacer toutes les conditions et recommencer.

Vous pouvez également copier les paramètres de visibilité sur un autre élément. Sélectionnez Copier pour copier les paramètres de visibilité dans le presse-papiers, puis ouvrez la boîte de dialogue Modifier la visibilité pour l'autre élément et sélectionnez Coller pour coller les paramètres copiés.

Ajout d'une logique de visibilité pour une option de réponse de type liste de sélection

Vous pouvez ajouter une logique de visibilité pour une réponse de liste de sélection individuelle.

Pour définir la visibilité d'une réponse de type liste de sélection :

  1. Naviguez jusqu'à la procédure ou la sous-procédure dont vous souhaitez définir la visibilité des options de réponse.

  2. Sélectionnez Autres actions () | Paramètres.

  3. Sélectionnez l'icône Editer () pour les réponses de la liste de sélection.

  4. Sélectionnez l'icône Visibilité () de la réponse que vous souhaitez modifier.

  5. Sélectionnez Masquer ou Afficher dans la liste déroulante bleue pour spécifier l'action de visibilité.

  6. Sélectionnez tout ou n'importe quelle condition pour déterminer si toutes les conditions doivent être remplies ou si une seule condition doit être remplie pour que l'action de visibilité (de l'étape 5) s'applique.

  7. Sélectionnez Ajouter une condition.

  8. Choisissez un type de condition dans la liste, puis spécifiez les paramètres de la condition.

  9. Une fois que vous avez terminé, sélectionnez Enregistrer.

Vous pouvez sélectionner Effacer pour effacer toutes les conditions et recommencer.

Notez que vous pouvez également définir des conditions de visibilité pour les réponses de la liste de sélection dans les paramètres de la liste de contrôle. Les sélections que vous faites ici sont définies par défaut pour toutes les procédures de la liste de contrôle.

Vous pouvez également copier les paramètres de visibilité sur un autre élément. Sélectionnez Copier pour copier les paramètres de visibilité dans le presse-papiers, puis ouvrez la boîte de dialogue Modifier la visibilité pour l'autre élément et sélectionnez Coller pour coller les paramètres copiés.
