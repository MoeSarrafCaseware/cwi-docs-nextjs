Pour en savoir plus, voir Autoriser les utilisateurs à ajouter des lignes directement dans le tableau.

Vous pouvez permettre aux utilisateurs d'ajouter des lignes de saisie (lignes personnalisées) dans un tableau dynamique directement sans ouvrir le tableau en mode édition. Cela simplifie le processus d'ajout de lignes de saisie pour les utilisateurs finaux, en particulier dans les cas où l'ajout de lignes personnalisées dans un tableau est fréquent.

Par exemple, si vous savez qu'à un endroit particulier du tableau, les utilisateurs ajoutent des coûts de main-d'œuvre sur plusieurs lignes, ils ajoutent des lignes de saisie. Vous ne prédéfinissez pas les lignes d'entrée pour eux, car vous ne savez pas combien de lignes ils devront ajouter. Si ce cas se produit fréquemment, il peut être fastidieux pour les utilisateurs d'accéder à l'écran de configuration du tableau (mode édition) chaque fois qu'ils ont besoin d'ajouter des lignes supplémentaires dans le tableau.

Remarque : Cette option ne peut être activée qu'à certains endroits du tableau. L'ajout d'un groupe personnalisé marque l'endroit du tableau où vous souhaitez permettre aux utilisateurs d'ajouter des lignes de saisie en dehors du mode d'édition.

To enable adding input rows directly in the table area:

  1. Sélectionnez le tableau.

    Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez Autres actions (), puis choisissez Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Repérez l'endroit où vous souhaitez permettre au personnel d'ajouter des lignes de saisie, puis sélectionnez le groupe Ajouter () | Personnaliser dans le bord gauche du tableau.

    4. Remarque : L'ajout d'un groupe personnalisé marque l'endroit du tableau où vous souhaitez permettre aux utilisateurs d'ajouter des lignes de saisie en dehors du mode d'édition.

    Une nouvelle ligne de groupe est ajoutée dans le tableau avec un titre par défaut Titre. Vous pouvez sélectionner le titre par défaut pour le modifier.

    Une ligne Total est également ajoutée dans le tableau. Il contient la valeur totale des soldes dans les lignes d'entrée ajoutées par l'utilisateur.

    Si vous ne voulez pas que la ligne de groupe ou la ligne de total apparaisse dans le tableau, sélectionnez la ligne puis l'icône de l'œil () pour la masquer. Dans ce cas, il est recommandé de sélectionner également l'option Paramètres () | Afficher les éléments cachés pour garder une trace de toutes les lignes existantes dans le tableau pendant que vous travaillez dessus.

  4. Sélectionnez le nouveau groupe personnalisé.

    5. L'icône Activer les rangées d'entrée () s'affiche dans le coin supérieur gauche.

  5. Sélectionnez Activer les rangées d'entrée ().

  6. Sélectionnez SAVE & CLOSE.

Le bouton Ajouter une ligne () est désormais disponible à l'endroit exact du tableau où vous souhaitez permettre aux utilisateurs d'ajouter des lignes de saisie.

To disable this option for users, select the group you added in Step 3, the select ().
