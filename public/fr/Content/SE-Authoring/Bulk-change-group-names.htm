Modification en bloc des noms de groupes

Si vous mettez en place un support multilingue dans votre produit, vous pouvez utiliser l'option Bulk change group names pour télécharger un fichier CSV contenant les noms de groupes traduits.

Pour télécharger les noms de groupes traduits :

  1. La langue de contenu du modèle est remplacée par la langue cible.

  2. Accédez à la page Données , puis sélectionnez Ajouter () | Modifier en bloc les noms des groupes.

  3. Sélectionnez le fichier CSV à partir de la page Documents (si vous l'avez téléchargé dans le cadre de l'engagement) ou parcourez votre disque local.

    Le fichier CSV doit comprendre

    • Numéro du groupe

    • Nom cohérent traduit

    • Nom contraire traduit

    • Nom incohérent traduit

    Remarque : Si vous importez un grand ensemble de données, fractionnez-les en plusieurs importations de moins de 1 000 lignes pour éviter les erreurs et garantir une importation réussie.

    Si vous changez la langue du contenu, les noms de groupes traduits s'affichent. Par exemple, si vous téléchargez des noms de groupe en français, puis sélectionnez le français comme langue de contenu, les noms de groupe en français s'affichent dans la page Données. Pour en savoir plus, consultez les rubriques suivantes : Un produit qui prend en charge les langues française et anglaise.
