Outil d'importation et de copier-coller des flux de trésorerie

L'outil d'importation / copier-coller des flux de trésorerie a été développé pour aider nos utilisateurs à transférer efficacement les soldes des tableaux de flux de trésorerie à partir de sources externes (feuilles de calcul Excel ou CSV) dans des tableaux dynamiques au sein des états financiers.

Une fois que l' outil d'importation / copier-coller de flux de trésorerie a été intégré à votre solution, sélectionnez Ajouter () sur votre page Documents et ajoutez les documents suivants :

  • Importation de flux financiers - Le formulaire principal qui comprend toutes les fonctionnalités, tant pour la méthode des flux financiers indirects que pour la méthode des flux financiers directs.

  • PA - Tableau des structures du tableau des flux de trésorerie - Un formulaire SE Builder nécessaire pour soutenir la fonctionnalité du formulaire principal. Vous devez ajouter cette feuille de calcul à votre page Documents, mais vous pouvez toujours la masquer en utilisant les paramètres de visibilité.

  • GC - Formulaire de visibilité en mode auteur - Un formulaire SE Builder qui pilote les paramètres de visibilité du contenu du formulaire en fonction du type d'auteur qui y accède. Vous devez ajouter cette feuille de calcul à votre page Documents , mais vous pouvez toujours la masquer en utilisant les paramètres de visibilité.

    Remarque : GC - Le formulaire de visibilité en mode auteur est un formulaire utilisé par d'autres formulaires de composants globaux. S'il a déjà été ajouté à votre solution, il n'y a pas d'autres modifications à apporter.

Changements requis pour l'auteur principal

Série sur le cadre d'information financière

Si votre application prend en charge plusieurs cadres ou secteurs d'activité, insérez les numéros d'identification des procédures où ces informations sont documentées.

Remarque : Si votre application ne prend en charge qu'un seul cadre de reporting financier ou un seul secteur d'activité, laissez ces champs vides.

Méthode d'établissement du tableau des flux de trésorerie

Procédez de la même manière pour insérer l'ID de la procédure lorsque le type de méthode de flux de trésorerie est documenté dans votre solution (méthode directe ou indirecte).

Remarque : Si votre application ne prend en charge qu'une seule méthode de trésorerie, laissez ce champ vide. Par défaut, le formulaire applique la méthode indirecte.

Désactiver l'édition de la description de la ligne

Sélectionnez Oui pour désactiver la possibilité pour les utilisateurs finaux de modifier les descriptions des lignes.

Groupes financiers de la trésorerie et des équivalents de trésorerie

Sélectionnez les groupes financiers qui totalisent le solde de trésorerie et d'équivalents de trésorerie, utilisé pour le tableau des flux de trésorerie. Ce solde sera utilisé pour recouper les montants figurant dans la section récapitulative.

Informations complémentaires sur les flux de trésorerie

Sélectionnez Oui si une section d'informations supplémentaires sur les flux de trésorerie s'applique à votre solution.

Tableau de structure des flux de trésorerie

Utilisez le tableau pour configurer les structures de flux de trésorerie pour chaque cadre de reporting financier et chaque secteur d'activité pris en charge dans votre application, en fonction des informations fournies dans la section Paramètres ci-dessus :

  • Sélectionnez chaque configuration applicable en fonction du cadre de reporting financier et du secteur d'activité pris en charge par votre application.

  • Indiquez le nombre de lignes à inclure dans chaque section de flux de trésorerie (exploitation / investissement ou financement).

    Remarque : Le nombre maximum de lignes pour chaque section est défini par l'auteur du SE Builder dans le fichier values.json . Un maximum de 40 lignes pour chaque section a été défini dans la version maître de ce formulaire. Des lignes supplémentaires peuvent être ajoutées par l'auteur de SE Builder en ajoutant des clés supplémentaires dans le fichier values.json.

  • Cliquez sur Modifier pour définir les descriptions des postes individuels pour chacune des structures de flux de trésorerie requises.

La boîte de dialogue Modifier propose les options suivantes :

  • Effacer tout - Supprime toutes les descriptions de lignes de toutes les sections.

  • Rétablir l'original - Rétablit les descriptions de lignes par défaut, telles que définies dans le fichier values.json .

  • Définir une nouvelle valeur par défaut - Enregistre les valeurs actuelles saisies en tant que nouvelle valeur par défaut à utiliser pour les structures futures.

  • Sauvegarder - Sauvegarde les valeurs saisies.

    Remarque :

    • Si votre application ne prend en charge qu'un seul cadre de reporting financier ou qu'un seul secteur d'activité, une ligne "Engagement par défaut" sera affichée pour définir la structure du flux de trésorerie.

    • La structure par défaut du tableau des flux de trésorerie incluse dans l'outil est basée sur la taxonomie XBRL de l'International Full IFRS. Modifiez la structure en fonction des spécificités de votre marché et veillez à enregistrer les changements.

    • Pour modifier ou traduire les valeurs par défaut, voir Ajouter des traductions ou des mises à jour de libellés

Création d'états financiers Tableaux dynamiques

L' outil d'importation et de copier-coller des flux de trésorerie enregistre les informations à l'aide d' ID de valeurs de formulaire. Pour lier les valeurs entre le formulaire SE Builder et les états financiers, l'auteur principal doit autoriser ces ID de valeurs de formulaire dans les tables dynamiques.

  • Pour accéder à une liste d' identifiants de valeurs de formulaires liés aux tableaux dynamiques financiers, cliquez ici. Notez qu'il s'agit de valeurs par défaut partagées et qu'elles peuvent déjà avoir été modifiées par le développeur de votre solution SE.

  • Lier chaque ID de valeur de formulaire à la valeur de cellule correspondante dans le tableau dynamique des états financiers.

End user workflow

En tant qu'utilisateur final, vous devez documenter les informations suivantes :

  1. Indiquez si les montants que vous souhaitez importer ou copier sont déjà arrondis ou non.

    Remarque :

    • Les états financiers peuvent être arrondis en milliers, millions et milliards.

    • Indiquer si les montants importés/copiés sont déjà arrondis conformément aux paramètres d'arrondi des états financiers.

    • Si les montants ne sont pas arrondis, le formulaire appliquera automatiquement les mêmes paramètres d'arrondi que ceux spécifiés dans les paramètres d'arrondi des états financiers.

    • To enable rounding in the financial statements refer to

      Personnaliser les arrondis dans les états financiers

  2. Sélectionnez la méthode pour insérer les montants des flux de trésorerie dans ce formulaire : Importation CSV | Copier-coller.

    Remarque : L'outil d'importation des flux de trésorerie propose deux méthodes pour insérer les montants dans ce formulaire :

    1. Importation CVS : Importation CSV : Importez les montants en utilisant le fichier modèle CSV prédéfini fourni.

      Remarque : L'importation CSV permet de charger les informations à partir d'un fichier CSV dont le format et la structure sont prédéfinis. An example file can be found here: Fichier de démonstration du modèle d'importation CSV.

    2. Copier-coller : Permet de copier et de coller directement les montants pour chacune des sections de flux de trésorerie applicables.

  3. Sélectionnez si vous souhaitez importer les montants de l'année précédente ou non.

  4. Examinez la section " Résumé" pour vous assurer que le tableau des flux de trésorerie est équilibré.
