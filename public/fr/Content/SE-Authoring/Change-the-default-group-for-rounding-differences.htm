Modifier le groupe par défaut pour les différences d'arrondi

Dans les tableaux des états financiers, les différences d'arrondi sont appliquées par défaut au premier compte non nul du niveau enfant le plus bas qui apparaît dans le premier groupe de la balance générale.

Ensuite, lorsque les utilisateurs importent les données de la balance de vérification, les différences d'arrondi s'appliquent au premier compte non nul dans Land.

Vous pouvez modifier le groupe par défaut pour vous assurer que la différence d'arrondi est appliquée à un poste des états financiers fréquemment utilisé sur votre marché.

Dans les tableaux dynamiques, la ligne du groupe où les différences d'arrondi s'appliquent est marquée d'un astérisque(*).

Pour modifier le compte où la différence d'arrondi est appliquée :

  1. Accédez au tableau dynamique dans lequel se trouvent les données de la balance générale.

    2. Par exemple, lors de l'établissement du bilan.

  2. Sélectionnez la ligne avec un astérisque (*).

  3. Sélectionnez le nom du compte dans Année en cours.

    4. Une liste des groupes de balances de vérification et des comptes s'affiche.

  4. Sélectionnez le compte souhaité dans la liste.

  5. Sélectionnez le nom du groupe dans Exercice précédent pour afficher la liste des groupes de balances de vérification.

  6. Sélectionnez le compte souhaité dans la liste.

Une fois que les utilisateurs ont importé les données de la balance de vérification dans l'engagement, les différences d'arrondi sont automatiquement ajoutées au premier compte non nul dans les groupes que vous avez sélectionnés pour l'année en cours et l'année précédente respectivement.

Si vous réduisez les groupes parents pour le groupe que vous avez sélectionné pour les différences d'arrondi, l'astérisque est déplacé vers les groupes parents. Cela vous aide à repérer l'endroit où se trouve le groupe sélectionné.

Par exemple, supposons que le groupe sélectionné pour les différences d'arrondi soit Land.

Remarquez que l'astérisque se déplace vers les groupes parents lorsqu'ils sont réduits. Lorsque le groupe parent " Immobilisations corporelles " est réduit, l'astérisque y est déplacé.

Lorsque le groupe parent Actifs non courants est réduit, l'astérisque y est déplacé.

Lorsque le groupe parent Actifs est réduit, l'astérisque y est déplacé.
