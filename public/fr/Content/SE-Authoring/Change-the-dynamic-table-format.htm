Change the dynamic table format

Vous pouvez modifier le format du tableau dynamique pour prendre en charge les normes internationales d'information financière (IFRS) et l'état des variations des capitaux propres (SOCIE). Le nouveau format renverse le tableau de manière à ce que les dimensions soient présentées en lignes et les groupes de balances de vérification en colonnes.

Remarque : Lorsque vous changez de format, la configuration et les informations existantes dans le tableau sont perdues et vous devez les refaire dans le nouveau format.

Pour passer au tableau IFRS/SOCIE :

  1. Sélectionnez le tableau souhaité.

    2. Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez Autres actions () | Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Sélectionnez Paramètres (), puis sélectionnez IFRS/SOCIE sous Format.

    4. Une demande de confirmation s'affiche pour vous informer que le changement de format de table crée une nouvelle table et que toutes les configurations et données existantes seront perdues.

  4. Sélectionnez Créer un nouveau tableau.

  5. Configurez les lignes et les colonnes du tableau comme vous le souhaitez.

Les options de ligne et de colonne suivantes sont désormais disponibles dans le tableau :

Rangs

  • Vous pouvez sélectionner une dimension pour que la colonne n'affiche que les données relatives à cette dimension. Vous pouvez définir la ligne de dimension pour inclure les données de l'année en cours ou de l'année précédente.

  • Ligne de saisie - ajoute une ligne de champ de texte. Vous pouvez saisir un texte personnalisé.

  • Ligne de commentaire: ajoute une ligne de commentaire. Par défaut, il insère une référence aux notes existantes dans les états financiers. Vous pouvez modifier la valeur par défaut en supprimant la référence de la note et en saisissant un commentaire pour la ligne.

  • Ligne d'espacement : pour ajouter des espaces blancs dans votre tableau dynamique. Vous pouvez ajouter des espaces blancs pour faciliter la lecture de votre contenu.

  • Saut de page : pour ajouter un saut de page entre les lignes du tableau.

  • Custom group - adds a custom group row. Les lignes de groupes personnalisés insèrent un champ de texte (pour ajouter un titre ou une brève description du groupe) et une ligne de total.
Colonnes

  • Balance de vérification - ajoute une colonne de date pour la balance de vérification. Vous pouvez sélectionner un compte ou un groupe pour obtenir les données de la colonne. Si le groupe a des sous-groupes, ceux-ci sont également insérés dans le tableau.

  • Input - ajoute une colonne de champs de texte. Vous pouvez saisir un texte personnalisé.

  • Calcul personnalisé - ajoute une colonne de calcul personnalisé dans laquelle vous pouvez saisir un calcul à l'aide de l'éditeur de calcul. Le calcul est appliqué à toutes les cellules de la colonne.

  • Référence aux notes - ajouter des références aux notes existantes dans les états financiers.

  • Annotation - ajouter une colonne d'annotations. Sélectionnez Ajouter () dans chaque cellule de la colonne pour ajouter des annotations.
