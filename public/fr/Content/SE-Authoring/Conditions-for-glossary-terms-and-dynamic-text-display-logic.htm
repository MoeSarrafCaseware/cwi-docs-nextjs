Conditions relatives aux termes du glossaire et à la logique d'affichage dynamique du texte

La liste suivante décrit les conditions requises pour configurer la logique d'affichage d'un terme de glossaire ou d'un texte dynamique. Il décrit également les paramètres de chaque condition.

Condition de réponse

La logique est basée sur une procédure de réponse par liste de contrôle.

Paramètres :

  • Document - sélectionner le document de la liste de contrôle.
  • Procédure - sélectionner la procédure.
  • Condition - sélectionner la réponse.
  • Sortie - saisissez le texte à afficher lorsque la condition est remplie.

La condition est remplie lorsque la réponse sélectionnée dans la condition est fournie dans la procédure de la liste de contrôle sélectionnée.

Condition de la balance de vérification

La logique est basée sur les propriétés du groupe de la balance générale.

Paramètres :

  • Groupe financier - sélectionner le groupe de balance de vérification.
  • Condition - sélectionnez une option dans la liste déroulante.
    • Cohérent
    • Contrairement à ce que l'on pourrait penser
    • Incohérent
  • Sortie - saisissez le texte à afficher lorsque la condition est remplie.

Par exemple, si le solde du groupe est Cohérent → Le compte de résultat s'affiche.

Condition de consolidation

La logique est basée sur le fait que la mission est une mission de consolidation ou non.

Paramètres :

  • Condition - sélectionner l'état de consolidation (consolidé ou non consolidé).
  • Sortie - saisissez le texte à afficher lorsque la condition est remplie.

Mise à l'échelle/arrondi

Sélectionnez une échelle d'arrondi dans la liste déroulante.

Exemple

Si l'arrondi est en millions, afficher cet élément.

Condition relative à la période de référence

La logique est basée sur la période de référence de la mission.

Paramètres :

  • Condition - sélectionner la valeur de la période de déclaration (mensuelle, trimestrielle, semestrielle, annuelle)
  • Sortie - saisissez le texte à afficher lorsque la condition est remplie.

Condition relative au type d'organisation

La logique est basée sur le pays sélectionné pour l'entité d'engagement.

Paramètres :

  • Condition - sélectionnez un pays.
  • Sortie - saisissez le texte à afficher lorsque la condition est remplie.

Forme Valeur Condition

La logique est basée sur la valeur d'un champ de formulaire.

Paramètres :

  • Condition - sélectionner un formulaire et un champ dans le formulaire sélectionné.
  • Sortie - saisissez le texte à afficher lorsque la condition est remplie.

Remarque : Cette condition n'est activée que si des formulaires SE Builder sont configurés dans votre produit.

Groupe de conditions

Sélectionnez cette option si vous souhaitez ajouter un groupe de conditions, où l'édition de texte est basée sur la façon dont les conditions du groupe sont évaluées collectivement.

Paramètres :

  • Lorsque toutes les conditions ci-dessous sont remplies - sélectionnez cette option si vous souhaitez que toutes les conditions soient remplies pour que la condition de groupe le soit.
  • Lorsque l'une des conditions ci-dessous est remplie - sélectionnez cette option si vous souhaitez que l'une des conditions soit vraie pour que la condition de groupe soit remplie.
  • Sortie - saisirle texte à afficher lorsque la condition de groupe est remplie.
