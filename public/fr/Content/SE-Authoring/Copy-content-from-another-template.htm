Copier le contenu d'un autre modèle

Avant de commencer, notez que

  • Modèle source - désigne le modèle à partir duquel vous souhaitez effectuer une copie.
  • Modèle de destination - fait référence au modèle que vous êtes en train de créer (votre modèle de produit).

Prérequis :

  • Le modèle source est publié et vous y avez accès. Pour en savoir plus, voir Rôles intégrés.

  • La propriété Importation de documents est activée dans le modèle de destination. Pour en savoir plus, voir Activer ou désactiver des fonctionnalités dans votre produit.

  • Si vous copiez des états financiers, les numéros de groupe et la structure du modèle de destination sont les mêmes que ceux du modèle source.

  • Si le modèle source contient des formulaires personnalisés , le fichier values.json du modèle de destination correspond au fichier values.json du modèle source.

Pour copier le contenu d'un autre modèle :

  1. Dans la page Documents , sélectionnez Ajouter () | Copier à partir d'un autre produit.

  2. Dans la liste déroulante Sélectionner un modèle , sélectionnez le modèle à partir duquel vous souhaitez effectuer la copie.

    3. Une liste des documents disponibles s'affiche.

  3. Dans la liste, sélectionnez la visualisation que vous souhaitez visualiser.

    4. Vous pouvez sélectionner :

    • des documents individuels.

    • un dossier pour copier tous les documents qu'il contient.

    • une phase pour copier tous les documents de la phase.

    Remarque : Lorsque vous sélectionnez un dossier ou une phase, seuls les documents de cette phase ou de ce dossier sont copiés. Les structures de fichiers ne sont pas gérées.

  4. Sélectionnez OK.

    5. Remarque : Si vous recopiez un document (c'est-à-dire que vous copiez à nouveau le même document à partir du même modèle), vous remplacerez le document existant et perdrez toutes les modifications que vous y avez apportées.
