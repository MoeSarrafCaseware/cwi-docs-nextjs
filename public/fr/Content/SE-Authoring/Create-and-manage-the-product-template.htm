Créer et gérer le modèle de produit

Pour commencer à rédiger, créez un modèle de projet de produit ou ouvrez un projet existant (si vous mettez à jour un projet existant).

Remarque : Si vous apportez des modifications à un modèle pour un produit SE existant, notez que vos modifications ne s'applique que Nouvel engagement Les changements de modèle ne s'appliquent pas aux missions existantes (c'est-à-dire aux missions qui ont été créées à partir de l'ancien modèle de produit).

Amélioration des versions des modèles d'auteur principal et d'auteur de réseau

Une amélioration a été apportée à la gestion des versions des modèles afin d'améliorer l'expérience de l'utilisateur lors de l'enregistrement de plusieurs brouillons. Ce changement n'a pas d'impact sur les versions de modèles déjà publiées et vise à améliorer l'expérience de versionnement des modèles.

1.2.3.4-x

  • 1 - Version Caseware

  • 2 - Version de l'auteur principal

  • 3 - Version de l'auteur du réseau (affichée uniquement si applicable)

  • 4 - Version de l'auteur ferme

  • -x - Version enregistrée du modèle (affichée uniquement si applicable)

L'amélioration ajoutera la version du modèle avec '-x' pour identifier la version sauvegardée du modèle. Chaque fois qu'un projet est enregistré, la version du modèle enregistrée est incrémentée de 1.

Par exemple, un nouveau projet est créé avec la version de modèle 1.3.0.0-1.

  • Une fois le modèle édité et "sauvegardé", la version du modèle sauvegardé est incrémentée à 1.3.0.0-2, puis 1.3.0.0-3, 1.3.0.0-4, et ainsi de suite.

  • Lors de la publication, la version publiée du modèle s'aligne sur la dernière version enregistrée du modèle, dans ce cas, 1.3.0.0-4.

  • Lorsqu'un nouveau projet est créé, la version du modèle est incrémentée à 1.4.0.0-1

Créer un brouillon

Si vous n'avez pas de projet de modèle de produit, vous devez en créer un.

Créer un brouillon

  1. Naviguez jusqu'au menu Cloud, puis sélectionnez Paramètres.

  2. Sélectionnez le nom de votre produit, puis sélectionnez Gérer le modèle.

  3. Sélectionnez Créer une ébauche pour créer une ébauche de modèle de produit.

    Un dossier d'engagement s'ouvre en mode auteur principal. Il s'agit de votre projet de modèle de produit.

Ouverture d'un IDEAScript existant

Vous pouvez fermer votre brouillon à tout moment et continuer à travailler dessus plus tard. Lorsque vous êtes prêt à continuer à travailler sur le projet, vous devez l'ouvrir.

Pour ouvrir un brouillon :

  1. Naviguez jusqu'au menu Cloud, puis sélectionnez Paramètres.

  2. Sélectionnez le nom de votre produit, puis sélectionnez Gérer le modèle.

  3. Sélectionnez Ouvrir le brouillon pour continuer à travailler sur votre brouillon.

Pour supprimer un brouillon existant :

Vous pouvez supprimer le projet si vous n'en êtes pas satisfait et en commencer un nouveau.

Pour supprimer un brouillon existant :

  1. Naviguez jusqu'au menu Cloud, puis sélectionnez Paramètres.

  2. Sélectionnez le nom de votre produit, puis sélectionnez Gérer le modèle.

  3. Sélectionnez Supprimer le brouillon.

    4. Le bouton Créer un brouillon est désormais disponible sur la page.

  4. Sélectionnez Créer un projet pour commencer à travailler sur un nouveau projet de modèle de produit.
