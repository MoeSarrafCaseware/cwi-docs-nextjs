Limites actuelles

La fonctionnalité actuelle du copier-coller intelligent présente les limitations suivantes, qui seront résolues dans les prochaines versions.

  • Vous pouvez ajouter les balises suivantes :

    • États financiers

    • Lettre

    • Mémo

    • Feuille de travail

    Vous ne pouvez pas copier :

    • Listes de contrôle

    • Formulaires du constructeur SE

    • Word/Excel/PDF

    • Risque, importance relative, inexactitudes

    • Analyse

    • Marqueur.

    • Documents comprenant des paramètres de visibilité ou des formules qui font référence à des formulaires ou à des sections de formulaire intégrées

  • Lorsque vous copiez des documents, les éventuelles données manquantes ou incohérentes entre les modèles sont automatiquement identifiées. Pour en savoir plus, voir Soldes d'annulation dans les états financiers.

  • Pour tous les autres documents, vous devez identifier et résoudre manuellement les divergences de données.

  • Résolution des conflits pour les états financiers copiés :

    Si un texte dynamique avec une condition de réponse et un lien de données (fx) dans le texte de sortie génère plusieurs conflits, la résolution d'un conflit entraîne la suppression des autres conflits dans le volet Liens brisés .

    Vous résolvez ces conflits en :

    1. Localiser manuellement les formules erronées dans le document.

      Les formules cassées s'affichent sous la forme #ValueUnavailable#.

    2. Sélectionnez la formule interrompue pour lancer la boîte de dialogue de liaison de données et résoudre le conflit.

  • Les questions existantes dans le modèle source (c'est-à-dire le modèle à partir duquel vous copiez les documents) ne sont pas copiées.

    • Meilleure pratique : Il est recommandé de conserver une structure de regroupement identique entre le modèle source et le modèle de destination.
