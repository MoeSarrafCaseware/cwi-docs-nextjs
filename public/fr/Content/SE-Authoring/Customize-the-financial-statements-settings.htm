Personnaliser les paramètres des états financiers

Vous pouvez personnaliser les paramètres des états financiers.

Sous Fonctionnalités | États financiers, les cases à cocher suivantes sont disponibles pour vous permettre de personnaliser les paramètres des états financiers :

  • Activer la traduction

    Pour en savoir plus, consultez les notes de mise à jour de la SE 2024-09.

  • Enable the new financial statements layout

    Notez que :

    • Cette option est activée par défaut.

  • Afficher les numéros de compte dans les tableaux dynamiques

    Si vous cochez cette case, les numéros de compte s'affichent avec le nom du compte dans la colonne Description des tableaux dynamiques.

  • Afficher les négations avec des parenthèses

    Notez que :

    • Pour les tableaux dynamiques, ce paramètre remplace le paramètre Négatif que vous avez spécifié dans l'onglet Format | Numéros

    • Ce paramètre est disponible pour les auteurs d'entreprise dans le modèle d'entreprise

  • Afficher le zéro sous forme de tiret

    Notez que :

    • Si vous cochez cette case, les zéros ne s'afficheront que sous forme de tirets dans les états financiers - et non globalement dans l'application Cloud

    • Ce paramètre est disponible pour les auteurs d'entreprise dans le modèle d'entreprise

  • Utiliser un espacement cohérent entre les paragraphes des différentes parties du contenu

    Ce paramètre permet d'appliquer un espacement de paragraphe cohérent dans les zones de texte (entre les paragraphes et entre un titre et un paragraphe) pour les états financiers, les lettres, les mémos et les feuilles de travail.

    Notez que ce paramètre ne s'applique qu'à la copie d'impression (PDF). Elle n'affecte pas l'affichage des documents lorsque vous les visualisez dans un modèle ou un dossier d'engagement.

  • Arrondissement dans les états financiers

    Ce paramètre permet l'arrondi automatique des tableaux dynamiques dans les états financiers. Une fois ce paramètre activé, vous pouvez choisir d'arrondir à un, à des milliers, à des millions ou à des milliards et définir le nombre de décimales à utiliser pour les valeurs arrondies.

    Vous pouvez également déterminer si les informations d'arrondi sont affichées dans les titres de description et les titres de colonne et personnaliser la terminologie utilisée.

    Notez que lorsque l'arrondi est activé, les auteurs du cabinet et les utilisateurs finaux peuvent également modifier les paramètres d'arrondi utilisés dans la mission.
