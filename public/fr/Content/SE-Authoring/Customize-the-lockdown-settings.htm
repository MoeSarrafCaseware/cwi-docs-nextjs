Personnaliser les paramètres de verrouillage

Vous pouvez personnaliser le fonctionnement du verrouillage de l'engagement dans votre produit. Notez que les auteurs d'entreprises peuvent également personnaliser ces paramètres dans le modèle d'entreprise.

Dans la section Fonctionnalités | Verrouillage, les options suivantes sont disponibles pour vous permettre de personnaliser les paramètres des états financiers :

  • Limiter le déverrouillage aux seuls administrateurs - Si vous sélectionnez cette option, seuls les utilisateurs ayant un rôle d'administrateur dans Cloud peuvent déverrouiller les engagements. Les administrateurs peuvent également lever temporairement les restrictions de verrouillage pour permettre aux utilisateurs non administrateurs de déverrouiller l'engagement.

  • Activerles jours de fermeture cibles - Si vous activez cette option, un certain nombre d'autres options sont disponibles pour vous permettre de définir des jours de fermeture cibles.

    • Jours de blocage cibles - Dans ce champ, entrez le nombre de jours qui doivent s'écouler avant que la mission ne soit bloquée, puis choisissez si le nombre de jours doit être compté à partir de la date de signature du rapport ou de la date de publication du rapport. Les utilisateurs peuvent définir la date de signature du rapport ou la date de publication du rapport à partir de la fenêtre contextuelle des propriétés de l'engagement.

    • Activer le verrouillage automatique piloté par le système - Si vous sélectionnez cette option, l'engagement se verrouillera automatiquement lorsque la date cible sera atteinte.
