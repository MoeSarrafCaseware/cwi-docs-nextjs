Personnaliser les paramètres d'affichage des groupes de balances de vérification

To view secondary trial balance grouping structures in your product, select the Information icon () at the top menu, then choose Settings to open the Product Settings dialog. Allez dans l'onglet Balance de vérification | Groupes et listez les structures de regroupement de la balance de vérification secondaire par nom. Notez que les structures de regroupement des balances de vérification secondaires ne seront visibles que si elles sont activées (voir Balance de vérification).

Si vous souhaitez que les numéros de groupe apparaissent avec la description dans la balance générale, cochez la case Afficher les numéros de groupe pour la comptabilité générale . Cette option est utile pour les marchés qui n'ont pas de plan comptable standard et qui souhaitent simplifier la balance de vérification en n'affichant pas les numéros de groupe.

Ce paramètre affiche également les numéros de groupe dans le complément Caseware Cloud Connector (voir l'image ci-dessous). Pour en savoir plus, voir Comment démarrer avec Cloud Connector.
