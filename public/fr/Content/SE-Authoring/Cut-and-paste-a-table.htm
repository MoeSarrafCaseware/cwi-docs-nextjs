Couper et coller un tableau

Vous pouvez copier un tableau dynamique existant dans les états financiers et le coller n'importe où dans les états financiers.

Pour couper et coller un tableau dynamique :

  1. Naviguez jusqu'au tableau dynamique que vous souhaitez copier et coller.

  2. Sélectionnez Plus d'actions () | Section de coupe.

  3. Navigate to where you want to add the table, then select Add () | Paste Section.

    Remarque : Vous ne pouvez coller le tableau qu'une seule fois.
