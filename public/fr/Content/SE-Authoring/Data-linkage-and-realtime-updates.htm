Data linkage and realtime updates

Si une valeur de données change dans l'engagement, le changement est automatiquement reflété dans le fichier Excel.

Exemple:

  1. Vous insérez le nom du groupe de balance comptable Cash dans le fichier Excel en utilisant le complément Caseware Cloud Connector .

  2. Pendant que vous travaillez sur votre engagement, vous modifiez le nom du groupe en Trésorerie et équivalents de trésorerie.

  3. Vous remarquerez que les liquidités ont été automatiquement transformées en liquidités et que les équivalents de liquidités s'affichent automatiquement dans le fichier Excel.

Cependant, si vous modifiez un numéro de groupe dans votre engagement, le changement n'est pas automatiquement reflété dans les données que vous avez insérées dans le fichier Excel. Vous devez réinsérer les données dans le fichier Excel.

Exemple :

  1. Vous insérez le nom du groupe de balance de vérification Cash dans le fichier Excel en utilisant le complément Caseware Cloud Connector .

  2. Lorsque vous travaillez sur votre engagement, vous modifiez le numéro de groupe pour Cash.

  3. Vous remarquerez que la cellule affiche maintenant un nom incorrect ou aucune valeur.

    4. Pour y remédier, supprimez la valeur existante et réinsérez le nom du groupe de la balance de vérification "Liquidités" dans le fichier Excel.
