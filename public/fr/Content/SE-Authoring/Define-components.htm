Définir les composants

Les composants permettent d'appliquer une logique de visibilité universelle à l'ensemble des groupes financiers de la balance générale des comptes. Vous pouvez définir des composants et les organiser dans des catégories pertinentes pour votre produit, votre marché ou votre région, telles que les types d'industries, les types d'entités et les cadres de reporting. Par exemple, vous pouvez regrouper tous les composants liés au type d'industrie dans une catégorie, et tous les composants liés aux cadres de reporting dans une autre.

Notez que vous ne pouvez pas définir de composants autonomes. Les composants doivent appartenir à une catégorie.

Une fois que vous avez défini vos composants, définissez la logique de visibilité pour chacun d'entre eux afin de pouvoir l'appliquer au contenu. Par exemple, si l'entité opère dans le secteur de l'agriculture, affichez les groupes de balances de vérification qui sont pertinents pour l'agriculture.

To add a category:

  1. Go to Features | Components.

  2. Sélectionner + Catégorie.

  3. Saisissez le nouveau nom.

Vous pouvez supprimer un schéma en sélectionnant le bouton ().

To add a component:

  1. Développez la catégorie appropriée.

  2. Sélectionner + Composant.

    Remarque : Vous devez créer au moins une catégorie pour que l'option +Composant soit disponible dans la boîte de dialogue.

  3. Saisissez le nom et l'identifiant du composant.

  4. Sélectionnez l'icône de l'œil () pour définir la logique de visibilité du composant.

  5. Cochez la case Occlude pour vous assurer que lorsque le contenu étiqueté avec le composant n'est pas visible pour les utilisateurs en raison de la visibilité du composant, il reste caché même si les utilisateurs choisissent d'afficher le contenu caché.

Vous pouvez supprimer un schéma en sélectionnant le bouton ().

Pour ajouter des paramètres de visibilité à un composant :

  1. Sélectionnez l'icône Settings () pour ouvrir la boîte de dialogue Checklist Settings.

  2. Sélectionnez Masquer ou Afficher dans la liste déroulante bleue pour spécifier l'action de visibilité.

  3. Sélectionnez tout ou n'importe quelle condition pour déterminer si toutes les conditions doivent être remplies ou si une seule condition doit être remplie pour que l'action de visibilité (de l'étape 2) s'applique.

  4. Sélectionnez Ajouter une condition.

  5. Choisissez un type de condition dans la liste, puis spécifiez les paramètres de la condition.

    6. Voir Conditions de visibilité pour une description détaillée des types de conditions et des paramètres disponibles.

  6. Une fois que vous avez terminé, sélectionnez Enregistrer.
