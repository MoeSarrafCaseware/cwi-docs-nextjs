Définir les termes du glossaire de l'engagement

Vous pouvez définir les termes dans le glossaire de l'engagement et configurer la logique d'affichage pour chaque terme à l'aide de conditions.

Par exemple, si l'utilisateur répond Oui à la procédure Votre cabinet comptable n'a-t-il qu'un seul associé en exercice ? de la liste de contrôle 010-Optimizer , le terme de sortie est Mon. Si la réponse est Non, le terme de sortie est Notre.

Pour ajouter des termes dans le glossaire de l'engagement :

  1. Sélectionnez Information () | Engagement Glossary.

    2. Le glossaire d'engagement s'affiche. À ce stade, il est vide.

  2. Sélectionnez Ajouter une annotation ().

  3. Saisissez un nom de groupe, par exemple Terminologie.

    4. Le bouton Créer un brouillon est désormais disponible sur la page.

  4. Sélectionnez Ajouter pour ouvrir la boîte de dialogue Ajouter un lien.

  5. Saisissez une description.

  6. Sélectionnez la liste déroulante Ajouter une condition pour configurer la logique d'affichage du terme de glossaire.

    Différents paramètres s'affichent dans la boîte de dialogue Liaison de données en fonction de la condition choisie.

    Pour plus d'informations sur la définition des conditions, voir Conditions pour les termes du glossaire et la logique d'affichage dynamique de texte.

    Vous pouvez ajouter plusieurs types de conditions pour déclencher des sorties de texte supplémentaires. Ils peuvent être du même type ou de types différents.

    Pour ajouter une autre condition, sélectionnez Ajouter une condition. Pour ajouter le même type de condition, sélectionnez Copier la condition ().

    Vous pouvez également spécifier une sortie alternative pour le texte si aucune condition n'est remplie.

    Si vous cochez la case Autoriser l'utilisateur à modifier l'édition , les utilisateurs finaux et les auteurs de l'entreprise peuvent modifier les valeurs d'édition par défaut que vous avez définies.

  7. Une fois que vous avez terminé, sélectionnez Enregistrer.

    8. Si une (ou plusieurs) condition(s) est (sont) remplie(s), le texte d'édition correspondant que vous avez spécifié pour cette condition s'affiche. Si aucune condition n'est remplie, le texte alternatif (celui que vous avez spécifié lorsque aucune des conditions n'est remplie) s'affiche. Sinon, #ValueEmpty# s'affiche.

Une fois les termes définis, vous pouvez les utiliser dans les documents. Pour plus d'informations sur l'utilisation des termes du glossaire, voir les rubriques suivantes :

  • Ajouter un lien de données dans la description de la procédure

  • Add a glossary term in the query question description.

  • Ajouter un terme de glossaire dans les champs de texte des états financiers et des lettres ou mémos.
