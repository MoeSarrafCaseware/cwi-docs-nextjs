Dimensions Allocation Worksheets

Desfeuilles de calcul pour l'attribution des dimensions ont été élaborées pour aider les utilisateurs à mener à bien leurs missions auprès des organisations à but non lucratif. Dans d'autres juridictions, ces entités peuvent être désignées sous le nom d'organismes sans but lucratif (OSBL) ou d'organismes de bienfaisance.

Ces formulaires ont pour but d'aider les utilisateurs à répartir les soldes des groupes financiers entre les différentes dimensions lorsque la balance de vérification n'inclut pas ces détails au niveau des comptes.

Une fois que les feuilles de calcul d'attribution des dimensions ont été intégrées à votre solution, vous pouvez ajouter les formulaires en sélectionnant le menu Ajouter () sur votre page Documents. Il existe deux formulaires SE Builder :

  • Résumé des fiches de répartition des dimensions - Document de type répétiteur qui peut être dupliqué autant de fois que nécessaire. Un exemplaire de ce formulaire est destiné à être utilisé pour répartir les soldes dans une catégorie de dimension à la fois.

  • Feuille de calcul d'allocation des dimensions - Un formulaire SE Builder répétitif qui inclura toutes les instances de feuilles de calcul d'allocation générées à partir des documents de synthèse des feuilles de calcul d'allocation des dimensions. Vous devez ajouter cette feuille de calcul à votre page Documents , mais vous pouvez toujours la masquer en utilisant les paramètres de visibilité.

Pour en savoir plus sur les feuilles de calcul de l'allocation des dimensions, voir cet enregistrement.

Pour en savoir plus sur la manière d'accéder aux formulaires personnalisés des feuilles de travail d'allocation de Dimensions à partir de la bibliothèque des composants globaux, voir Démarrer avec les feuilles de travail d'allocation de Dimensions.

Ces feuilles de travail s'intègrent de manière transparente à la fonctionnalité de la plateforme SE autour de Dimensions, sans nécessiter de modifications supplémentaires du code ou de configurations de l'auteur principal pour les cas d'utilisation qu'elles sont censées résoudre.

Les concepts utilisés pour développer ces feuilles de calcul peuvent également être utilisés pour d'autres types d'allocations, moyennant quelques modifications. Si vous envisagez de faire changements importants au contenu ou à la fonctionnalité des formulaires SE Builder, comme l'ajout, la suppression ou la modification d'une fonctionnalité, vous devez être propriétaire du code. Voir Demande d'accès au code source des feuilles de calcul pour l'attribution des dimensions.

Remarque : Si vous devenez propriétaire du code, vous devez maintenir les formulaires SE Builder à l'avenir, ce qui inclut l'application manuelle de toutes les modifications, mises à jour et corrections de bogues futures qui sont déployées pour la source de formulaire d'origine.
