Pour afficher les soldes des lignes avec le signe opposé :

Dans les propriétés du groupe, vous définissez le signe par défaut pour les comptes d'un groupe spécifique (débit ou crédit). Toutefois, vous pouvez choisir d'afficher les soldes des lignes avec le signe opposé dans les tableaux dynamiques.

Par exemple, dans le compte de résultat, vous souhaitez afficher les recettes et la marge brute comme des nombres positifs (bien que le signe par défaut de ces comptes soit négatif).
