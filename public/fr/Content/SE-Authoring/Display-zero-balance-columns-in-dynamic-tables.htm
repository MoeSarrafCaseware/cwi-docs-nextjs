Afficher les colonnes à solde nul dans les tableaux dynamiques

Les colonnes du tableau dynamique créées après la version SE 2024-05 qui ne contiennent que des valeurs nulles sont automatiquement masquées par défaut. Lorsque les utilisateurs consultent les états financiers dans leur mission, ces colonnes sont grisées et ne sont pas incluses lorsque les états financiers sont imprimés.

Si vous souhaitez afficher les colonnes de solde nul pour un tableau dynamique, vous pouvez le faire à partir des paramètres du tableau.

Remarque : Ce paramètre n'affecte pas les colonnes de saisie ou de calcul personnalisé. Si vous souhaitez afficher une colonne de saisie ou de calcul personnalisé avec uniquement des valeurs nulles, vous pouvez le faire à partir des options d'affichage dans les paramètres de la colonne.
