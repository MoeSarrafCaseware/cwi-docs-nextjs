Intégrer un PDF dans les états financiers

Vous pouvez ajouter des zones PDF intégrées dans les états financiers. Cela vous permet d'ajouter et de prévisualiser tous les documents PDF qui doivent être inclus dans les états financiers avant d'imprimer les états et de générer le PDF.

Notez que,

  • Vous pouvez ajouter plusieurs zones PDF intégrées aux états financiers

  • Un seul PDF peut être téléchargé dans chaque zone de PDF incorporé

  • Une fois que vous avez intégré un PDF, vous pouvez l'inclure dans la table des matières des états financiers

Intégrer un PDF dans les états financiers

  1. Allez dans le plan du document, puis sélectionnez Ajouter () | PDF incorporé.

    Une nouvelle zone PDF intégrée est ajoutée au plan du document et s'affiche sur la page avec un titre par défaut : PDF intégré. Sélectionnez le titre par défaut pour le modifier.

  2. Cochez la case pour rechercher le fichier PDF ou faites glisser le fichier pour l'intégrer.

    Une fois le PDF intégré, vous pouvez sélectionner View pour visualiser le fichier ou Delete () pour le supprimer des états financiers.
