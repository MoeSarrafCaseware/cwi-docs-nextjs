Activer les contrôles dans le module des risques

Vous pouvez activer les contrôles dans le module de risque pour permettre aux utilisateurs de documenter les contrôles et de répondre aux exigences en matière de documentation d'audit.

Pour afficher ou personnaliser vos rôles d'approbation, sélectionnez l'icône Information () dans le menu supérieur, puis sélectionnez Paramètres pour ouvrir la boîte de dialogue Paramètres de l'engagement. Dans l'onglet Contrôles | Général , sélectionnez Activer les contrôles dans le module de risque.

Une fois les contrôles activés, l'onglet Contrôles est accessible aux utilisateurs finaux dans le module de risque et le nom du module devient Risques et contrôles si vous avez activé à la fois les risques et les contrôles. Dans cet onglet, les utilisateurs peuvent

  • Ajouter, modifier et supprimer des contrôles

  • Développez les cartes de contrôle pour afficher et modifier les détails du contrôle

  • Lier les contrôles aux risques

  • Ajouter des annotations et des questions aux contrôles

  • Utilisez les filtres pour rechercher et afficher les contrôles pertinents

En outre, davantage de paramètres sont disponibles pour les contrôles dans les paramètres du produit :

  • Signature - Une fois l'option de signature activée, deux cases à cocher sont disponibles pour vous permettre de spécifier comment les utilisateurs signent le rapport de contrôle.

    • Rapport global - Un bouton d'approbation est disponible dans le rapport pour permettre aux utilisateurs d'approuver l'ensemble du rapport.

    • Contrôle individuel - Un bouton de signature est disponible dans chaque carte de contrôle du rapport. Les utilisateurs signent chaque contrôle individuellement.

    Une fois que vous avez sélectionné la manière dont les utilisateurs approuvent le rapport de risque, vous pouvez choisir un schéma d'approbation personnalisé. Pour en savoir plus sur les schémas de signature, voir Configuration des schémas de signature.

  • Identifier les contrôles par - Choisir si les contrôles sont identifiés par cycle d'activité ou par domaine des états financiers (FSA).

  • Aide - si vous souhaitez créer une page d'aide pour vos clients afin de décrire les fonctionnalités, les champs et les attributs des contrôles, vous pouvez activer l'option Aide et ajouter l'URL. Une fois cette option activée, une icône d' aide () s'affiche dans les cartes de contrôle.

Personnaliser les champs de contrôle

Under the Controls drop-down, select Fields.

Les champs de contrôle sont organisés en quatre catégories qui peuvent être développées et réduites à l'aide des flèches vers le haut et vers le bas :

  • Identification.

  • Attributs

  • Conception et mise en œuvre.

  • Évaluation de l'efficacité opérationnelle.

Notez que la catégorie Identification et le champ Titre de cette catégorie ne peuvent pas être désactivés.

Pour ajouter et personnaliser des champs de contrôle :

  1. Sélectionnez un champ pour l'inclure dans le rapport de contrôle.

  2. Par exemple, sélectionnez pour l'activer dans le rapport sur les contrôles.

  3. Dans le champ Personnaliser cette étiquette , saisissez votre terme personnalisé.

  4. Pour chaque champ, cochez la case Dans la boîte de dialogue pour rendre le champ disponible aux utilisateurs dans la boîte de dialogue Créer un nouveau contrôle.

  5. Si le champ comporte plusieurs options, saisissez votre terme personnalisé pour chaque option.

    Pour certains champs, vous pouvez utiliser le bouton Ajouter () pour ajouter d'autres options personnalisées.
