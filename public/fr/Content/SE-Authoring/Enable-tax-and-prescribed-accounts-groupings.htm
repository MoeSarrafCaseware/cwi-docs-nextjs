Enable grouping structures

Activer les structures de regroupement Vous pouvez activer plusieurs structures de regroupement financier dans votre application Cloud, ainsi que des structures de regroupement pour les taxes et les comptes prescrits (plan comptable).

Allez dans Paramètres du produit | Balance de vérification | Groupes et activez chaque structure de regroupement que vous souhaitez ajouter à votre produit. Note that you can add a maximum of three financial grouping structures. Vous pouvez également saisir un nom personnalisé pour chaque structure de regroupement.

Si vous souhaitez que les numéros de groupe apparaissent avec la description du groupe dans la balance générale, cochez la case Afficher les numéros de groupe pour la comptabilité générale . Cette option est utile pour les marchés qui n'ont pas de plan comptable standard et qui souhaitent simplifier la balance de vérification en n'affichant pas les numéros de groupe. Ce paramètre affiche également les numéros de groupe dans le complément Caseware Cloud Connector (voir l'image ci-dessous). Pour en savoir plus, voir Comment démarrer avec Cloud Connector.
