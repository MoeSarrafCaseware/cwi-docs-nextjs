La page Données

La page Données permet aux utilisateurs de visualiser et de modifier les données qu'ils ont importées dans leur engagement. Vous pouvez personnaliser la page de données à partir des paramètres du produit.

Dans l'onglet Balance de vérification | Page de données , vous pouvez sélectionner les options suivantes :

  • Permettre des bilans de planification

  • Définir les groupes parents - Sélectionnez cette option pour afficher les groupes dans les vues de groupes de la page Données.

  • Définir les comptes - Sélectionnez cette option pour afficher les comptes dans la vue des groupes de la page Données.

  • Activer l'échantillonnage

  • Activer la signature de la page de données

  • Activer la signature des ajustements

  • Activation d'un nouveau flux de travail pour les importations de balances de vérification CSV/Excel

    Notez que le nouveau processus d'importation de données est activé par défaut pour la grande majorité des produits.

  • Activer la dernière version du mappage des comptes - Sélectionnez cette option pour activer la nouvelle conception de la page Attribuer des comptes.

  • Activer le fractionnement des colonnes d'ajustement - Sélectionnez cette option pour activer l'option de fractionnement des ajustements . Lorsque l'utilisateur active cette option, les ajustements normaux et les ajustements de reclassement sont affichés dans des colonnes séparées.

  • Activer les groupes financiers définis par l'utilisateur - Sélectionnez cette option pour permettre aux utilisateurs finaux d'ajouter des groupes financiers dans les missions.
