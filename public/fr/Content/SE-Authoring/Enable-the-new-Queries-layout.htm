Activer la nouvelle présentation des requêtes

Queries are switching to a new and improved user interface. To enable the new query design in your product, select the Information icon () at the top menu and choose Settings to open the Product Settings dialog.

Remarque : Assurez-vous que les dimensions sont activées dans le fichier product.json, sinon l'onglet Dimensions ne sera pas disponible. Pour en savoir plus, voir Activer ou désactiver des fonctionnalités dans votre produit.

Dans Fonctionnalités | Requêtes, sélectionnez Activer la nouvelle présentation de la requête.

Remarque : Nous vous conseillons vivement de tester la nouvelle conception sur les requêtes existantes dans votre produit avant de l'activer dans l'environnement de production.

Une fois que vous avez activé le nouveau modèle de requête, vous pouvez également activer le type de réponse Signature électronique simple. Ce type de réponse permet aux utilisateurs de télécharger des documents nécessitant la signature du client, tels que les termes de la mission, et de les envoyer aux clients pour qu'ils les acceptent par voie électronique. Les contacts peuvent choisir de :

  • Accepter électroniquement - Sélectionnez le bouton Accepter pour signer. Les contacts peuvent également choisir de partager leurs données de localisation lorsqu'ils signent, ce qui sera enregistré dans le journal des signatures.

  • Téléchargerun document signé - Téléchargez le document joint, puis insérez la signature ou imprimez et signez le document manuellement, puis téléchargez-le.

  • Copier le document signé à partir du nuage - Sélectionnez le document précédemment téléchargé, signé et téléchargé à partir du site du nuage de l'entreprise.

Si les clients choisissent d'accepter le document par voie électronique, un registre des signatures est généré et peut être téléchargé. Le journal contient les informations suivantes :

  • Détails de la question

  • Nom et courriel du signataire

  • Localisation du signataire (si les clients ont choisi d'activer l'accès à la localisation)

  • Adresse IP du signataire

  • Date de la signature

  • Nom du document

  • Diagramme MD5 du document

Pour activer ce type de réponse, cochez la case Activer la signature électronique simple dans les types de réponse .
