Activer le nouveau module de risque

Le module actuel de gestion des risques simples est remplacé par une nouvelle interface utilisateur permettant de prendre en charge des missions plus complexes. Le nouveau module introduit des changements massifs de fonctionnalité.

To enable the new risk functionality in your product, select the Information icon () at the top menu and choose Settings to open the Product Settings dialog. Dans l'onglet Risque | Général , sélectionnez Activer la fonctionnalité de nouveau risque.

Remarque : Le nouveau module de risque ne s'affiche pas si un formulaire personnalisé est sélectionné pour les risques dans l'onglet Fonctionnalités | Formulaires personnalisés. Vous pouvez sélectionner Aucun dans le menu déroulant Risques de l'onglet pour effacer toute sélection antérieure et afficher le module.

Une fois le nouveau module de risque activé, la nouvelle page Risques est disponible dans le modèle. De plus, d'autres paramètres s'affichent dans la boîte de dialogue Paramètres du produit :

  • L'évaluation

  • Domaines

  • Échelle

  • Option de signature :

    • Deux cases à cocher sont disponibles pour vous permettre de spécifier comment les utilisateurs signent le rapport de risques.

    • Signature de l'ensemble du rapport - Un bouton de signature est disponible dans le rapport pour permettre aux utilisateurs de signer l'ensemble du rapport.

    • Approuverchaque risque dans le rapport - Un bouton d'approbation est disponible dans chaque fiche de risque du rapport. Les utilisateurs signent chaque risque individuellement.

    Une fois que vous avez sélectionné la manière dont les utilisateurs approuvent le rapport de risque, vous pouvez choisir un schéma d'approbation personnalisé. Pour en savoir plus sur les schémas de signature, voir Configuration des schémas de signature.

Remarque : Les paramètres suivants ne sont disponibles dans la boîte de dialogue Paramètres du produit que si vous activez la nouvelle fonctionnalité de gestion des risques.

En outre, l'onglet Risque - Général offre une option permettant d' inclure le champ des commentaires dans les widgets d'analyse de risque. Les commentaires sont inclus par défaut dans les widgets et peuvent être désactivés en cochant la case correspondante.

Si vous avez activé la bibliothèque de risques pour votre site, vous pouvez également associer une bibliothèque de risques au produit dans la section Gestion des bibliothèques. Pour plus d'informations, voir Associer une bibliothèque de risques à un produit.

Activer l'onglet Évaluation

Pour activer l'onglet Évaluation dans la page Risques, sélectionnez Risque | Évaluation et activez l'option Évaluation des risques .

Une fois les évaluations activées, d'autres options sont disponibles dans la boîte de dialogue :

  • MÉTHODE D'ÉVALUATION - vous pouvez choisir d'activer soit le cycle de zone, soit le cycle d'entreprise. Une fois cette option activée, les fiches d'évaluation de chaque domaine ou cycle économique disponible s'affichent dans l'onglet Évaluation.

  • Allez-vous tester les contrôles ? - si vous activez cette option, les utilisateurs peuvent choisir s'ils testent les contrôles pour chaque domaine ou cycle dans le rapport d'évaluation des risques. Si les utilisateurs répondent qu'ils ne testent pas les contrôles, le risque de contrôle pour chaque assertion sera automatiquement fixé au niveau de risque le plus élevé. Notez que cette opération n'écrase pas les sélections manuelles effectuées précédemment.

  • Évaluation des risques inhérents et des risques de contrôle - si vous activez cette option, les utilisateurs finaux peuvent alors documenter les niveaux de risques inhérents et de risques de contrôle dans le rapport d'évaluation des risques. Notez que cette option doit être activée pour que les options Probabilité et Ampleur soient activées. Cette option est activée par défaut.

  • Probabilité et ampleur - si vous activez cette option, les utilisateurs finaux peuvent alors documenter la probabilité et l'ampleur au niveau du domaine ou du cycle dans le rapport d'évaluation des risques.

  • Aide - si vous souhaitez créer une page d'aide pour vos clients afin de décrire les fonctionnalités, les champs et les attributs des évaluations, vous pouvez activer l'option Aide et ajouter l'URL. Une fois cette option activée, une icône d' aide () s'affiche dans les fiches d'évaluation.

  • OPTIONS DE SIGNATURE - Deux cases à cocher sont disponibles pour vous permettre de spécifier comment les utilisateurs signent le rapport d'évaluation des risques.

    • Signature de l'ensemble du rapport - Un bouton de signature est disponible dans le rapport pour permettre aux utilisateurs de signer l'ensemble du rapport.

    • Approuverles évaluations individuelles - Un bouton d'approbation est disponible dans chaque fiche d'évaluation du rapport. Les utilisateurs signent chaque risque individuellement.

    Une fois que vous avez sélectionné la façon dont les utilisateurs approuvent le rapport d'évaluation des risques, vous pouvez choisir un schéma d'approbation personnalisé. Pour en savoir plus sur les schémas de signature, voir Configuration des schémas de signature.

  • Champs de texte supplémentaires - vous pouvez également activer des champs de texte personnalisés supplémentaires dans la fiche d'évaluation, si nécessaire. Sélectionnez Zone de texte personnalisée 1 ou Zone de texte personnalisée 2 pour activer des champs personnalisés supplémentaires. Enter a title for the field in the TITLE text area.

  • Personnalisation de l'étiquette - vous saisissez votre étiquette personnalisée pour la forme courte et longue des champs Risque inhérent, Risque de contrôle et Risque d'anomalies significatives dans l'onglet Évaluation. Vous pouvez également saisir des libellés personnalisés pour la forme courte et longue de la probabilité de survenance d'une anomalie et de l'ampleur de l'anomalie potentielle si vous avez activé l'option Probabilité et ampleur.

Personnaliser les champs de risque

Under the Risks drop-down, select Fields.

Pour ajouter et personnaliser les champs d'identification du risque, de description du risque, d'assertions, de méthode d'évaluation et de références :

  1. Sélectionnez un champ pour l'inclure dans le module des risques.

    2. Par exemple, sélectionnez pour l'inclure dans le module des risques.

    Une fois que vous avez sélectionné un champ, sa section de personnalisation est disponible dans la boîte de dialogue.

  2. Sous ÉTIQUETTE PERSONNALISÉE, saisissez votre terme personnalisé pour chaque champ.

  3. Pour chaque champ, sélectionnez :

    • la case à cocher Dialogue pour afficher le champ dans la boîte de dialogue Nouveau risque.
    • la case à cocher Carte pour afficher le champ dans la carte de risque.
    • les cases à cocher Dialogue et Carte pour afficher le champ dans le dialogue Nouveau risque et la carte de risque.

Dans la section Assertions, vous pouvez accéder directement à Fonctionnalités | Assertions pour ajouter ou modifier une assertion existante en sélectionnant le lien Assertions .

Pour ajouter et personnaliser les champs de type de risque et de niveau de risque :

  1. Sélectionnez Type et Niveau pour inclure ces champs dans le module des risques.

    2. Une fois que vous avez sélectionné un champ, sa section de personnalisation est disponible dans la boîte de dialogue.

  2. Dans la section de personnalisation pour le niveau,

    1. entrez votre terme personnalisé pour niveau de risque àÉTIQUETTE PERSONNALISÉE .

    2. entrez votre terme personnalisé pour Normal, significatif et frauduleux.

  3. Dans la section de personnalisation de Type,

    1. entrez votre terme personnalisé pour le type de risque à CUSTOM LABEL.

    2. entrez votre terme personnalisé pour État financier et affirmation.

Pour ajouter et personnaliser les champs de réponse d'audit, de contrôle et de conclusion :

  1. Sélectionnez un champ pour l'inclure dans le module des risques.

    2. Par exemple, sélectionnez pour l'inclure dans le module des risques.

    Une fois que vous avez sélectionné un champ, sa section de personnalisation est disponible dans la boîte de dialogue.

  2. Sous ÉTIQUETTE PERSONNALISÉE, saisissez votre terme personnalisé pour chaque champ.

  3. Dans la section Personnalisation des conclusions, saisissez un terme personnalisé pour chaque type de conclusion.

    1. Entrez un terme personnalisé pour Le risque a été traité.

    2. Saisissez un terme personnalisé pour " Il existe une déficience significative du contrôle".

    3. Saisissez un terme personnalisé pour Une réponse d'audit est requise.

    4. Saisissez un terme personnalisé pour Autre.

Si vous souhaitez créer une page d'aide pour vos clients afin de décrire les fonctionnalités, les champs et les attributs des risques, vous pouvez activer l'option Aide et ajouter l'URL. Une fois cette option activée, une icône d' aide () s'affiche dans les fiches de risque et dans la boîte de dialogue Nouveau risque.

Vous pouvez également choisir d'inclure les deux cases à cocher Inclure dans les éléments à déclarer et Les procédures de fond ne sont pas suffisantes dans la fiche de risque. Vous pouvez également ajouter un texte personnalisé pour chaque case à cocher.

Personnaliser les échelles et la cartographie des RMM

Vous pouvez choisir le nombre d'échelles dans votre produit pour les risques inhérents (RI), les risques de contrôle (RC) et les risques d'anomalies significatives (RMM) et personnaliser leurs noms d'affichage. Vous pouvez également prédéfinir le mappage RMM dans le produit.

Sélectionnez Risque | Échelles pour commencer.

Adapter les barèmes aux risques inhérents :

  1. Allez à la section Risque inhérent .

  2. Dans l'option Échelle d'évaluation, sélectionnez le nombre d'échelles souhaité.

  3. Dans la rubrique Personnalisation de l'étiquette, saisissez le terme personnalisé et le nom abrégé souhaités pour chaque échelle.

    Par exemple, vous pouvez renommer Very Low en Mild et définir le nom court en ML.

Personnaliser les échelles pour les risques de contrôle :

  1. Accédez à la section " Risque de contrôle" .

  2. Dans l'option Échelle d'évaluation, sélectionnez le nombre d'échelles souhaité.

  3. Dans la rubrique Personnalisation de l'étiquette, saisissez le terme personnalisé et le nom abrégé souhaités pour chaque échelle.

    Par exemple, vous pouvez renommer Very Low en Mild et définir le nom court en ML.

Adapter les échelles aux risques d'inexactitudes significatives :

  1. Go to the RMM section.

  2. Dans l'option Échelle d'évaluation, sélectionnez le nombre d'échelles souhaité.

  3. Dans la rubrique Personnalisation de l'étiquette, saisissez le terme personnalisé et le nom abrégé souhaités pour chaque échelle.

    Par exemple, vous pouvez renommer Very Low en Mild et définir le nom court en ML.

Pour prédéfinir le mappage RMM :

  1. Allez dans la section RMM Mapping.

  2. Sélectionner l'échelle RMM souhaitée pour chaque combinaison d'échelle IR et CR.

    3. Par exemple, vous pouvez prérégler l'échelle RMM sur Très faible (VL)si l'IR et la CR ont une échelle Très faible (VL).

Personnaliser les échelles de probabilité de survenance des risques d'anomalies :

  1. Passez à la section Probabilité de survenance d'une inexactitude .

  2. Dans l'option Échelle d'évaluation, sélectionnez le nombre d'échelles souhaité.

  3. Dans la rubrique Personnalisation de l'étiquette, saisissez le terme personnalisé et le nom abrégé souhaités pour chaque échelle.

    Par exemple, vous pouvez renommer Very Low en Mild et définir le nom court en ML.

Personnaliser les échelles en fonction de l'ampleur des risques potentiels d'inexactitude :

  1. Passez à la section " Ampleur de l'inexactitude potentielle ".

  2. Dans l'option Échelle d'évaluation, sélectionnez le nombre d'échelles souhaité.

  3. Dans la rubrique Personnalisation de l'étiquette, saisissez le terme personnalisé et le nom abrégé souhaités pour chaque échelle.

    Par exemple, vous pouvez renommer Very Low en Mild et définir le nom court en ML.
