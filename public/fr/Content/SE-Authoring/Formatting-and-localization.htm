Configurer les préférences de formatage

Vous devez vérifier et personnaliser les paramètres de police, spécifier l'affichage des dates, des devises et des nombres dans votre produit, ainsi que l'unité de mesure préférée.

Notez que ces paramètres sont également disponibles pour les auteurs d'entreprise.

Ajouter une monnaie

Vous pouvez ajouter une devise à votre produit SE pour prendre en charge des devises supplémentaires :

  1. Allez dans Format | Devise, puis sélectionnez Ajouter une devise.

  2. Saisissez le nom de la devise dans le champ de texte pour filtrer la liste des devises.

    3. Vous pouvez également filtrer la liste en utilisant le code de devise à trois lettres de la norme ISO 4217.

  3. Sélectionnez la devise souhaitée pour l'ajouter à votre produit SE.

    4. Vous pouvez également supprimer une devise de votre produit en sélectionnant ().

Les devises que vous ajoutez dans cet onglet sont globalement disponibles dans votre application Cloud et s'affichent à tous les endroits où les devises sont applicables. Par exemple, dans la page de données et les états financiers.

Personnaliser le format de la date

Allez dans Format | Dates et choisissez les formats pour les dates longues, courtes et moyennes en les sélectionnant dans les listes déroulantes. Les formats que vous spécifiez s'appliquent globalement à votre application Cloud chaque fois que vous sélectionnez ou insérez des dates. Par exemple, dans les listes de contrôle, les requêtes, la balance des comptes, les espaces réservés aux champs de date, les propriétés d'engagement et les formules dans les zones de texte.

Personnaliser les paramètres des numéros

Dans Format | Numéros, vous pouvez personnaliser les paramètres suivants :

Paramètre

Description

Format des nombres négatifs

La façon dont les nombres négatifs s'affichent dans votre produit SE :

  • entre parenthèses ( (1 234 567,46) )
  • signe moins précédant le nombre ( -1,234,567.46 )
  • le signe moins après le nombre ( 1,234,567.46- )

Symbole décimal

Utilisation d'un point (.) ou d'une virgule (,) comme symbole décimal.

Décimales

Le nombre de points décimaux affichés pour les soldes.

Symbole de regroupement des chiffres

Utiliser un point (.) ou une virgule (,) comme symbole de regroupement des chiffres.

Groupement de chiffres

La façon dont les chiffres sont regroupés.

Les paramètres de nombre que vous spécifiez ici s'appliquent globalement à votre application Cloud lorsque les soldes s'affichent. Par exemple, dans la page des données , le document sur l'importance relative et les états financiers.
