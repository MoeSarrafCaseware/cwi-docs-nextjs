Propriétés du groupe

La liste suivante décrit chaque propriété de la boîte de dialogue Propriétés du groupe.

Nombre

Saisissez un numéro de mappage unique. Si vous avez déjà importé des groupes, le numéro de mappage doit suivre la structure du masque de mappage existant.

Nom cohérent

Saisir une description du groupe à afficher si le solde est cohérent avec le signe normal du groupe. Cette description apparaît dans la colonne " Préliminaire " des états financiers.

Nom contraire

Le cas échéant, saisissez une description du groupe à afficher si le solde est opposé au signe normal du groupe. Cette description apparaît dans la colonne " Préliminaire " des états financiers.

Par exemple, "Découvert bancaire" apparaîtra à la place de "Banque" lorsque le solde est créditeur.

Nom incohérent

Le cas échéant, saisissez une description alternative pour le groupe à afficher lorsque l'année en cours et l'année précédente ont des signes opposés. Cette description apparaît dans la colonne " Préliminaire " des états financiers.

Remarque : Ce nom a la priorité sur le nom cohérent et le nom contraire, quel que soit le solde du groupe dans la balance de vérification.

Signe

Sélectionnez le signe applicable (débit ou crédit).

Type

Sélectionnez le type d'état approprié pour ce groupe (bilan, compte de résultat, résultat global).

Formule :

Le cas échéant, saisissez une formule AOCI pour calculer les valeurs de ce groupe. Une fois que l'utilisateur a importé la balance générale, ce groupe affiche un montant dérivé des soldes des autres groupes.

Exemple de formule pour les AOCI :

Tagbal("1.234",perspec({yearOffset}),{type},{code})

où 1.234 est le groupe OCI dans la section du compte de résultat

Répartition par dimension

Le cas échéant, sélectionnez une catégorie de dimension pour le groupe dans la liste déroulante Ventilation par dimension (les valeurs de la liste sont définies dans les paramètres du produit). Cela permet aux utilisateurs de filtrer la balance de vérification afin de visualiser les informations relatives aux soldes par dimension.

Inclure les dimensions

Ce bouton n'est disponible que si une catégorie de dimension est sélectionnée dans Ventilation par dimension. Il vous permet d'exclure des dimensions spécifiques de la catégorie sélectionnée.

Le bouton affiche les dimensions disponibles dans la catégorie sélectionnée. Vous pouvez exclure une dimension spécifique en décochant la case située à côté.

Cycle et zone

Sélectionnez un cycle d'activité et un domaine dans les menus déroulants du groupeDomaine/Cycle (les valeurs des menus déroulants sont définies dans les paramètres du produit). Si votre application Cloud comprend un document d'évaluation des risques, lorsque vous affectez un groupe à un cycle d'activité, les éléments de ce groupe seront automatiquement affichés sous ce cycle dans le document, ce qui facilitera l'audit par cycle d'activité.

Catégories de composants

Les catégories de composants organisent les composants de votre produit pour plusieurs secteurs d'activité, types d'entités ou cadres de reporting. Par exemple, les éléments liés au type d'industrie peuvent être regroupés dans une catégorie et les éléments liés aux cadres de reporting dans une autre catégorie. Pour en savoir plus, voir Définir la matérialité.

Sélectionnez une ou plusieurs catégories à attribuer au groupe dans la liste déroulante Catégories de composants (les catégories de la liste déroulante sont définies dans les paramètres du produit). Une fois que vous avez attribué une catégorie à un groupe, elle est incluse dans la boîte de dialogue Propriétés du groupe et vous pouvez attribuer les composants applicables.

Composants

Les composants vous aident à mettre en place une logique de visibilité axée sur le marché ou sur la région pour les groupes de balances de vérification. Par exemple, si l'entité opère dans le secteur de l'agriculture, affichez les groupes de balances de vérification qui sont pertinents pour l'agriculture. Pour en savoir plus, voir Définir la matérialité.

Avant de pouvoir affecter des composants, vous devez sélectionner une ou plusieurs catégories de composants. Une fois que vous avez sélectionné une catégorie pour le groupe, celle-ci est incluse dans la boîte de dialogue. Sélectionnez un ou plusieurs composants à affecter au groupe dans la liste déroulante des catégories (les composants de la liste déroulante des catégories sont définis dans les paramètres du produit).

Si vous associez plusieurs composants à un groupe de balances de vérification et que des paramètres de visibilité sont définis pour chaque composant dans votre application Cloud, la visibilité du groupe de balances de vérification varie selon que les composants affectés appartiennent à la même catégorie ou à des catégories différentes :

  • Si les composants affectés appartiennent à la même catégorie, le groupe de balance générale est visible si au moins un composant est visible - sinon le groupe est masqué

  • Si les composants affectés appartiennent à des catégories différentes, le groupe de balance générale est visible si au moins un composant de chaque catégorie est visible - sinon, le groupe est masqué

Il existe une exception aux règles ci-dessus si les membres du personnel ajoutent un compte à un engagement, cochent la case Afficher les comptes cachés (dans l'onglet Attribuer des comptes ) et attribuent le nouveau compte à un groupe caché. Dans ce cas, le groupe et ses groupes parents deviennent visibles.

Supposons, par exemple, que le groupe Frais d'exploitation soit masqué en raison des paramètres de visibilité des composants. L'utilisateur crée un compte Autres dépenses et l'affecte aux Dépenses d'exploitation. Dans ce cas, les dépenses d'exploitation et ses groupes parents deviennent automatiquement visibles.

Structures de groupes liés

Sélectionnez des groupes dans d'autres structures de regroupement pour y lier le groupe financier. Cela vous permet d'affecter automatiquement des comptes à des groupes dans d'autres structures de regroupement sur la base de leur affectation initiale à un groupe.

Allow account assignment

Cochez cette case pour autoriser l'affectation de comptes à ce groupe.

Réaffectation de l'objectif pour les comptes contraires

Le cas échéant, sélectionner un groupe ("réaffecter la cible") pour les comptes avec un solde contraire. Si un compte est automatiquement affecté à ce groupe mais qu'il ne correspond pas au signe sélectionné (débit ou crédit) pour ce groupe, il sera automatiquement affecté à la cible de réaffectation.

Par exemple, actif négatif --> passifpositif

Classification

Saisir ou sélectionner une classification à partir d'un plan comptable général. Cette liste de base des classifications permet une mise en correspondance automatique directe à partir d'un logiciel comptable en faisant correspondre la classification appliquée au groupe au type de compte dans le logiciel comptable.

Ajuster la préférence du groupe

Le poids est utilisé pour rattacher automatiquement des comptes à un groupe sur la base d'une échelle. Par exemple, les groupes ayant une pondération positive seront préférés aux groupes sans pondération.

Restrictions sur le solde du compte

Spécifiez les types de comptes qui peuvent être affectés en fonction de leur solde :

  • les comptes dont le solde est à affecter
  • seuls les comptes ayant un solde débiteur (positif) peuvent être affectés
  • seuls les comptes présentant un solde créditeur (négatif) peuvent être affectés

Groupe de parents

Sélectionnez le groupe de niveau supérieur dans la structure.

Groupe par défaut

Cette propriété vous permet de marquer le groupe comme l'un de vos groupes par défaut ou préférés. Si l'option Groupes par défaut uniquement est activée dans la vue Groupes (Financiers ou Fiscaux, ou Prescrits), seuls les groupes par défaut s'affichent dans la page.

Par exemple, si vous disposez d'une structure de regroupement importante, vous pouvez marquer les groupes les plus courants avec lesquels vous travaillez (vos groupes préférés) en cochant la case Groupe par défaut. Dans la vue Groupes, sélectionnez l'option Groupes par défaut uniquement pour n'afficher que les groupes marqués (ceux dont la case Groupe par défaut est cochée) dans la page.
