Comment rédiger une requête ?

Les étapes suivantes résument le processus recommandé pour finaliser votre projet. Vous pouvez utiliser ce flux de travail pour vous guider.

  1. Ajouter une question de requête

  2. Créer le modèle de requête

    • Ajouter des instructions de contact

    • Ajouter des jeux de questions

    • Ajouter une question de requête

    • Examiner et personnaliser les paramètres des questions de requête

    • Ajouter un type de réponse à une procédure

    • Add a reference to a document in a question

    • Lier une question de requête à une procédure de la liste de contrôle

    • Réorganiser les séries de questions et les questions

    • Copier les paramètres de visibilité

  3. Créer un document de requête à partir d'un modèle existant
