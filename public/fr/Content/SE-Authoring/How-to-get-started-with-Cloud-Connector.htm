How to get started with Cloud Connector?

Caseware Cloud Connector est un complément de Microsoft Office qui permet de relier les données en temps réel entre votre engagement et MS Excel. Grâce à ce lien, vous pouvez accéder aux données de la balance des comptes et les récupérer, ainsi qu'aux propriétés de la mission et du cabinet, puis insérer ces informations dans un fichier MS Excel. Si une valeur ou une information liée est modifiée dans l'engagement, elle est automatiquement mise à jour dans le fichier Excel.

Vous pouvez utiliser le connecteur cloud pour créer des états financiers, des feuilles de calcul et des tableaux ou pour vérifier des calculs.

Notez que Cloud Connector ne peut pas récupérer les informations suivantes :

  • Soldes pour les filiales dans les missions de consolidation

  • Bilans dimensionnels

  • Soldes préliminaires

  • Régularisations

  • Valeurs de matérialité

  • Soldes de différentes structures de regroupement

  • Soldes de la période précédente 2-4. Seuls les soldes de l'année en cours et de l'année précédente sont pris en charge

Notez que deux versions du complément Connector sont disponibles sur l'Office Add-in Store, l'une pour les environnements partenaires et l'autre pour les environnements de production. Lorsque vous créez ou testez avec Cloud Connector sur les sites partenaires, vous devez utiliser la version partenaire du complément. Les formules que vous ajoutez à l'aide du complément partenaire continueront à fonctionner si vous y accédez à partir d'une entreprise de production équipée de la version de production du complément.

Avant de commencer, assurez-vous que l'administrateur de votre entreprise a acheté et activé une licence Caseware Cloud Connector pour votre compte. Voir Enable Cloud apps pour en savoir plus.

Pour commencer :

  1. Si vous utilisez Windows, assurez-vous que vous disposez d'une installation locale de la dernière version de Microsoft Office 365 Desktop et que vous utilisez la dernière version de Windows 10.

    Si vous utilisez un Mac, assurez-vous que vous disposez d'une installation locale de la version 16.35 ou ultérieure d'Office sur Mac et que vous utilisez un système d'exploitation Mac compatible avec Microsoft Office 365.

    Remarque : La version 2019 ou antérieure de MS Office et Office Online ne sont pas pris en charge.

  2. Ouvrir un fichier Excel dans MS Office 365.

  3. Téléchargez le fichier Excel sur votre engagement.

  4. Ouvrez le fichier Excel à partir de la page Documents.

  5. Obtenez le complément Caseware Cloud Connector dans la boutique Microsoft Add-ins.

    Si vous testez le complément dans un environnement partenaire, recherchez "WA200002372" dans le magasin pour localiser le complément partenaire.

    Si vous testez le complément dans un environnement de production, recherchez "WA200002239" dans le magasin pour localiser le complément de production.

    Remarque : Vous ne pouvez installer qu'une seule version du complément Connector. Si vous testez le complément dans des environnements partenaires et de production, vous devrez supprimer le complément avant d'installer l'autre version. Pour supprimer le complément, cliquez avec le bouton droit de la souris sur le complément et sélectionnez Supprimer le complément.

  6. Utilisez le complément Caseware Cloud Connector pour insérer les données de votre engagement dans le fichier Excel.
