Importer des groupes (page Données)

Vous pouvez importer des regroupements à partir de fichiers CSV ou Excel. Pour les importations CSV, vous devez d'abord préparer correctement les fichiers. Si vous avez créé des groupes dans les documents de travail, vous pouvez les exporter dans un fichier Excel et importer ce fichier.

To import groupings from a CSV file:

  1. Préparer le fichier CSV des transactions
  2. Importer des groupes

Pour importer des groupes de documents de travail :

  1. Exporter des groupes de documents de travail vers Excel
  2. Importer des groupes
