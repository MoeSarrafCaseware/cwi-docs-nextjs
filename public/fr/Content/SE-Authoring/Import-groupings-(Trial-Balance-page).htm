Import groupings (Trial Balance page)

Vous pouvez importer des regroupements à partir de fichiers CSV ou Excel. Pour les importations CSV, vous devez d'abord préparer correctement les fichiers. Si vous disposez de groupes définis dans des documents de travail, vous pouvez les exporter dans un fichier Excel et importer ce fichier.

To import groupings from a CSV file:

  1. Préparer le fichier CSV des transactions
  2. Importer des groupes

Pour importer des groupes de documents de travail :

  1. Exporter des groupes de documents de travail vers Excel
  2. Importer des groupes

Préparer un fichier CSV pour l'importation (page Balance de vérification)

Le fichier CSV doit comprendre les cinq colonnes suivantes. Elles peuvent avoir des valeurs vides mais chaque colonne doit être disponible dans le fichier :

  • Numéro de carte (appelé numéro de groupe dans le SE)

  • Nom du compte (appelé nom du groupe dans SE)

  • Comptable (les comptes peuvent être affectés au regroupement)

  • Type (bilan ou compte de résultat)

  • Signe (débit ou crédit)

Vous pouvez éventuellement inclure les colonnes suivantes :

  • Poids
  • Autoriser
  • Nom contraire
  • Nom incohérent
  • Catégorie de dimension
  • Dimensions exclues
  • Réaffectation de l'objectif pour les comptes contraires
  • Classification des groupes
  • Lien vers un groupe fiscal
  • Lien vers un groupe de plans comptables
  • Cycle
  • Zone
  • Composants
  • Groupe par défaut

Vous pouvez utiliser ces colonnes pour faciliter la mise en place de l'automapping (affectation automatique des comptes aux regroupements) lorsque les utilisateurs importent une nouvelle balance générale.

Le tableau suivant décrit toutes les colonnes d'information :

Colonne

Description

Numéro de carte

(obligatoire)

Saisir un numéro de carte pour le groupement. Également connu sous le nom de numéro de groupe dans le SE.

Peut contenir des caractères alphanumériques, des points décimaux, des traits d'union et des traits de soulignement.

Nom du compte

Saisissez un nom pour le formulaire. Également connu sous le nom de nom de groupe dans le SE.

Peut contenir des caractères alphanumériques, des points décimaux, des traits d'union et des traits de soulignement.

Si ce champ n'est pas renseigné, le groupement sera nommé par défaut "[Sans titre]".

Postable

Indiquer si des comptes peuvent être affectés au regroupement.

Peut contenir une seule lettre. Valeur attendue

  • Y - les comptes ne peuvent pas être affectés au regroupement.
  • N - des comptes peuvent être affectés au groupement.

Si ce champ n'est pas renseigné, vous pourrez affecter des comptes au regroupement par défaut.

Type

(obligatoire)

Indiquer si le regroupement appartient au bilan ou au compte de résultat.

Peut contenir une seule lettre. Valeur attendue

  • B - bilan
  • Compte de résultat

Signe

Indiquer si le regroupement doit se faire pour les comptes de débit ou les comptes de crédit.

Peut contenir une seule lettre. Valeur attendue

  • D=débit
  • C=crédit;

Si ce champ n'est pas renseigné, le groupement aura par défaut le signe du débit.

Classification des groupes

Saisissez un nom pour le formulaire.

Peut contenir des caractères alphanumériques, des points décimaux, des traits d'union et des traits de soulignement.

Poids

Saisissez un nom pour le formulaire. Dans les cas où plusieurs groupes ont la même classification, tout compte ayant cette classification sera affecté au groupe ayant le numéro de pondération le plus élevé.

Peut contenir un nombre décimal de 1 à -1.

Si ce champ n'est pas renseigné, le groupement aura un poids de 0 par défaut.

Autoriser

Indiquer si le groupement est soumis à des restrictions de solde de compte.

Peut contenir un seul caractère. Valeur attendue

  • + - seuls les comptes ayant un solde débiteur (positif) peuvent être affectés.
  • - - seuls les comptes présentant un solde créditeur (négatif) peuvent être affectés.
  • * - des comptes avec des soldes débiteurs ou créditeurs peuvent être affectés.

Si ce champ n'est pas renseigné, vous pourrez attribuer des comptes avec des soldes débiteurs ou créditeurs par défaut.

Exporter des groupes de documents de travail vers Excel

Pour utiliser la structure de regroupement que vous avez définie dans les Working Papers, vous devez l'exporter vers un fichier Excel, puis convertir le fichier en .csv.

Ouvrez Working Papers et suivez les étapes ci-dessous.

Pour exporter un regroupement de documents de travail vers un fichier Excel :

  1. Open the file that you want to export.

  2. Sur le ruban, cliquez sur Engagement | Propriétés de l'engagement.

    3. La feuille de calcul Regroupements s'affiche.

  3. Faites un clic droit sur l'en-tête de la colonne, puis cliquez sur Éditeur de filtre....

    4. Dans la boîte de dialogue Réorganiser les colonnes du tableau, organisez les colonnes actives comme suit : Numéro > Nom > Titre > Type > Signe

  4. Cliquez sur OK.

    5. Veillez à ce que chaque groupe contienne un numéro unique, un type défini et un signe.

  5. Sur le ruban, cliquez sur Cloud | Upload.

  6. Sous l'onglet Général, dans le menu déroulant Style de champ de la liste de tableaux, sélectionnez Codes de saisie.

  7. Cliquez sur OK.

  8. Sur la feuille de calcul, sélectionnez tous les groupes en cliquant sur n'importe quel numéro de groupe, puis en appuyant sur la touche CTRL+A de votre clavier.

  9. Copiez la sélection en appuyant sur CTRL+C sur votre clavier.

  10. Ouvrez un nouveau classeur Excel.

  11. Collez les groupes en cliquant sur la cellule A1, puis en appuyant sur les touches CTRL+V de votre clavier.

  12. Assurez-vous que les renseignements sont exacts

    • Vérifiez la présence de numéros de groupe en double en sélectionnant la colonne entière et en appuyant sur Accueil | Mise en forme conditionnelle | Règles de mise en évidence des cellules | Valeurs en double.

    • Vérifiez que les colonnes Nom, Titre et Type ne contiennent pas de valeurs manquantes ou vides.

    • Vérifiez la colonne Type pour vous assurer que la classification est correcte pour chaque ligne.

    • Vérifiez dans la colonne Signe que le signe est correct pour chaque ligne. Si le signe n'est pas renseigné, la valeur par défaut est Débit.

  13. Enregistrez le classeur Excel en tant que fichier .csv.

Importer des regroupements à partir d'un fichier CSV ou Excel (page Balance de vérification)

Une fois le fichier CSV ou Excel prêt, vous pouvez lancer le processus d'importation.

Pour importer un groupement à partir d'un fichier CSV ou Excel :

  1. Allez dans l'onglet Balance de vérification , puis sélectionnez Autres actions () | Importer des groupes.

  2. Glissez-déposez le fichier .csv dans la boîte de dialogue.

  3. Sélectionnez Oui lorsque vous êtes invité à confirmer la suppression des groupes financiers existants.

Une fois l'importation terminée, vérifiez la structure pour vous assurer que l'importation s'est bien déroulée. Vous pouvez sélectionner l'option Développer tout/Réduire tout () pour revoir la structure importée. Si nécessaire, vous pouvez également modifier un groupe pour ajouter des propriétés supplémentaires ou modifier les propriétés existantes.
