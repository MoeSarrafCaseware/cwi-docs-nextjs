Importer des regroupements à partir d'un fichier CSV ou Excel (page Données)

Une fois le fichier CSV ou Excel prêt, vous pouvez lancer le processus d'importation.

Pour importer un groupement à partir d'un fichier CSV ou Excel :

  1. Go to the Data page, then select Add () | Import groups.

  2. Glissez-déposez le fichier .csv dans la boîte de dialogue.

  3. Sélectionnez Oui lorsque vous êtes invité à confirmer la suppression des groupes financiers existants.

Une fois l'importation terminée, vérifiez la structure pour vous assurer que l'importation s'est bien déroulée. Si nécessaire, vous pouvez également modifier un groupe pour ajouter des propriétés supplémentaires ou modifier les propriétés existantes.
