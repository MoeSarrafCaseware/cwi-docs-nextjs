Insérer des données en utilisant la barre de formule de MS Excel avec Cloud Connector

Vous pouvez utiliser la barre de formule de Microsoft Excel pour récupérer et insérer des données du fichier d'engagement dans le fichier Excel.

Pour insérer des données à l'aide de la barre de formule :

  1. Sélectionnez la barre de formule et entrez =cwi pour afficher une liste de toutes les fonctions disponibles de Cloud Connector. Vous pouvez également consulter la liste des fonctions de Cloud Connector pour voir toutes les fonctions de Cloud Connector disponibles et leurs paramètres.

  2. Sélectionnez la fonction souhaitée et appuyez sur Tab.

    Conseil : Si vous connaissez le nom de la fonction, tapez-le dans la barre de formule et appuyez sur la touche Tab.

  3. Si des paramètres sont associés à la fonction, entrez les paramètres comme indiqué à l'écran.

    Utilisez la boîte de dialogue de la formule si vous avez besoin d'aide sur les paramètres (leur signification ou le format attendu). Il fournit des instructions et une description à l'écran.
