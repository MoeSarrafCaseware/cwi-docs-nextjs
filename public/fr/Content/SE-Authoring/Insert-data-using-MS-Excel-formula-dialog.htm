Insérer des données à l'aide de la boîte de dialogue des formules MS Excel avec Cloud Connector

Vous pouvez utiliser la boîte de dialogue de formule de Microsoft Excel pour récupérer et insérer des données du fichier d'engagement dans le fichier Excel.

Pour insérer des données à l'aide de la boîte de dialogue de formule :

  1. Sélectionnez l'icône de la formule () pour lancer la boîte de dialogue Insérer une fonction.

  2. Dans le champ Ou sélectionnez une catégorie , sélectionnez SE Connector Develop dans la liste déroulante.

  3. Sélectionnez la fonction souhaitée.

    Si vous connaissez le nom de la fonction, vous pouvez utiliser le champ de recherche pour la trouver. Pour afficher toutes les fonctions disponibles, voir Liste des fonctions de Cloud Connector.

    Dans l'exemple ci-dessous, nous avons sélectionné la fonction CWI.GROUPBALANCE . Notez qu'une brève description de la fonction sélectionnée et de ses paramètres s'affiche.

  4. Sélectionnez OK.

  5. Si des paramètres sont associés à la fonction sélectionnée, entrez-les et sélectionnez OK.
