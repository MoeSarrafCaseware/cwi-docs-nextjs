Lier une question de requête à une procédure de la liste de contrôle

Vous pouvez lier une question à une procédure connexe dans une liste de contrôle afin de compléter le document de la liste de contrôle.

For example, if there is a question in a checklist related to the client's business that you know users won't have an optimal answer to and it's better they request that information from the client. Vous créez donc un modèle de requête et ajoutez cette demande, puis vous la reliez à la question correspondante dans le document de la liste de contrôle. Une fois que vous avez recueilli les informations requises auprès du client, vous pouvez ajouter la réponse dans la liste de contrôle.

Pour lier une requête à une liste de contrôle :

  1. Go to the query that you want to link to a checklist.

  2. Accédez à la question, puis sélectionnez Plus d'actions () | Lien vers la liste de contrôle.

  3. Sélectionnez le champ de recherche pour afficher une liste des listes de contrôle disponibles dans le modèle.

    4. Vous pouvez saisir un mot clé dans le champ de recherche pour filtrer la liste. Vous pouvez également cocher la case Afficher les documents cachés en bas de page pour inclure les listes de contrôle cachées dans la liste.

  4. Select a checklist document to link the query to.

    5. Lorsque vous ajoutez une liste de contrôle, un autre champ de recherche s'affiche. Il vous permet de sélectionner des procédures spécifiques dans cette liste de contrôle pour y associer la requête.

  5. Sélectionnez le champ de recherche pour afficher une liste des procédures de la liste de contrôle.

    6. Vous pouvez saisir un mot clé dans le champ de recherche pour filtrer la liste. Vous pouvez également cocher la case Afficher les procédures cachées pour inclure les procédures cachées dans la liste.

  6. Sélectionnez une procédure à laquelle lier la requête.

  7. Sélectionnez Enregistrer.

Pour supprimer les procédures liées, sélectionnez la question, puis sélectionnez Autres actions () | Lier à la liste de contrôle. Lorsque la boîte de dialogue s'ouvre, sélectionnez Supprimer.
