Lier une question à une procédure de la liste de contrôle

Vous pouvez établir un lien avec les tests d'analyse dans une procédure de liste de contrôle. Les tests analytiques vous permettent d'identifier et d'analyser les informations relatives aux données de votre mission et d'améliorer votre capacité à atteindre les objectifs de l'audit.

Une fois les tests liés à une procédure, les utilisateurs peuvent exécuter le test, puis consulter les résultats du test et reconfigurer les tests avant de les exécuter à nouveau si nécessaire.

Voir Voir les résultats des tests analytiques pour les listes de contrôle pour plus d'informations sur la façon dont les utilisateurs peuvent utiliser les tests analytiques dans leurs missions.

Pour lier une question de requête à une procédure de la liste de contrôle :

  1. Navigate to the procedure or sub-procedure where you want to link analytics.

  2. Sélectionnez Plus d'actions () | Attacher des analyses.

  3. Dans la boîte de dialogue Joindre des analyses, sélectionnez les tests analytiques à joindre à la procédure dans la liste déroulante.

  4. Sélectionner SAVE.
