Gérer les tests analytiques dans le Hub Analytique

Remarque : Pour accéder à cette fonctionnalité, vous devez ajouter manuellement "dataset-results/analytics-hub" après "index.jsp#/" dans votre URL.

Le hub analytique est un emplacement central que vous pouvez utiliser pour afficher une liste de tous les tests analytiques qui ont été liés à une liste de contrôle dans votre produit. Lorsque vous sélectionnez un test, vous pouvez afficher les détails de la configuration ainsi que la liste des procédures de la liste de contrôle liées au test. Notez que vous ne pouvez pas modifier les valeurs des paramètres d'un test, mais les utilisateurs pourront le faire avant de l'exécuter.

Vous pouvez également sélectionner Supprimer tous les liens de procédure pour supprimer les liens vers les procédures de la liste de contrôle pour un test. Notez que si vous sélectionnez cette option, le test ne s'affichera plus dans le Hub Analytique.
