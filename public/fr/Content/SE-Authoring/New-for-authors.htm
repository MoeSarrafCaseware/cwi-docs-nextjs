Nouveau pour les auteurs

Voici les nouveautés pour les utilisateurs finaux dans la SE 2024-09.

Définir la possibilité d'ajouter des groupes à partir des paramètres du produit

#firmauthor #networkauthor

Les auteurs de réseaux et les auteurs de cabinets peuvent désormais déterminer si les utilisateurs finaux peuvent ajouter des groupes financiers dans les missions. Si les auteurs principaux conservent la possibilité de définir si cette fonctionnalité est disponible pour les utilisateurs finaux dans le fichier product.json, les auteurs du réseau et les auteurs de l'entreprise peuvent désormais remplacer les paramètres de l'auteur principal pour personnaliser leur modèle.

Pour déterminer si les utilisateurs finaux peuvent ajouter des groupes financiers, les auteurs peuvent modifier la case à cocher Activer les groupes financiers définis par l'utilisateur dans l'onglet Données des paramètres du produit.

Notez que la case à cocher Activer les groupes financiers définis par l'utilisateur est également disponible pour les auteurs principaux, de sorte que vous pouvez contrôler si les utilisateurs finaux peuvent ajouter des groupes financiers sans avoir à modifier le fichier product.json.

Amélioration des échelles de risque

#primaryauthor #networkauthor

Vous pouvez désormais définir sept niveaux pour les échelles de risque au lieu des cinq qui étaient disponibles auparavant.
