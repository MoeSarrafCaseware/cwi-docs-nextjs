Nouveau pour les utilisateurs finaux

Voici les nouveautés pour les utilisateurs finaux dans la SE 2024-09.

Problèmes liés aux lignes de tableaux dynamiques

Les utilisateurs peuvent désormais ajouter des questions aux lignes des tableaux dynamiques. L'icône Issues s'affiche lorsque l'utilisateur passe le curseur sur le côté droit d'une ligne d'un tableau dynamique. Ils peuvent ensuite sélectionner l'icône pour ajouter un problème.

Notez que :

  • Les questions ajoutées à une ligne enfant sont affichées au niveau parent lorsque le groupe de lignes est réduit. Lorsque la ligne est développée, les questions s'affichent sur la ligne à laquelle elles ont été ajoutées.

  • Si une ligne doit être masquée en raison de la logique de visibilité, elle continuera à s'afficher si un problème y est associé et qu'il est soit en suspens, soit résolu. La ligne sera masquée si le problème est résolu. Notez que si une question est attachée à une ligne cachée ou non, cette ligne ne sera pas incluse lors de l'impression du document.

  • Si une question est ajoutée à une ligne et que cette ligne est ensuite supprimée, la question est déplacée vers la table dynamique au niveau de la table.

Il est à noter que les numéros ne peuvent pas être ajoutés aux lignes de saut de page.

Amélioration de la synchronisation de la balance des comptes

Un message d'avertissement s'affiche désormais à l'intention des utilisateurs lorsqu'ils tentent de synchroniser leur balance de vérification à partir d'un autre engagement. Après avoir sélectionné le bouton Confirmer , ce nouveau message s'affiche et les informe que la synchronisation de la balance de vérification est une action irréversible.

Réglages du filtre

Une nouvelle liste déroulante Tous les types est désormais disponible sur la page Ajustements pour permettre aux utilisateurs de filtrer les ajustements par type. Lorsque les utilisateurs choisissent d'exporter leurs ajustements, seuls les ajustements filtrés sont inclus dans le fichier généré. Cela permet aux utilisateurs de sélectionner uniquement les types d'ajustements dont ils ont besoin et d'en fournir une copie à leurs clients.

Vues sauvegardées pour la page Données

L'icône Enregistrer les vues () est désormais disponible sur la page Données pour permettre aux utilisateurs d'enregistrer leur vue actuelle et d'accéder aux vues précédemment enregistrées.

Lorsque les utilisateurs enregistrent une vue, leurs sélections pour les paramètres suivants sont sauvegardées :

  • Colonnes

  • Index

  • Filtres

  • Variance

  • Ajustements du fractionnement

  • Lignes par page

Les vues sauvegardées sont accessibles à tous les utilisateurs du dossier d'engagement.
