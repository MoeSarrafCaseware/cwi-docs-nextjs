Préparer un fichier CSV pour l'importation (page Données)

Le fichier CSV doit comprendre les colonnes suivantes Elles peuvent avoir des valeurs vides mais chaque colonne doit être disponible dans le fichier :

  • Map number
  • Nom du compte
  • Postable
  • Type
  • Signe

Vous pouvez éventuellement inclure les colonnes suivantes :

  • Poids
  • Autoriser
  • Nom contraire
  • Nom incohérent
  • Catégorie de dimension
  • Dimensions exclues
  • Réaffectation de l'objectif pour les comptes contraires
  • Classification des groupes
  • Lien vers un groupe fiscal
  • Lien vers un groupe de plans comptables
  • Cycle
  • Zone
  • Composants
  • Groupe par défaut

Vous pouvez utiliser ces colonnes pour faciliter la mise en place de l'automapping (affectation automatique des comptes aux regroupements) lorsque les utilisateurs importent une nouvelle balance générale.

La liste suivante décrit les colonnes et la valeur attendue pour chaque colonne.

Map Number (mandatory)

Saisir un numéro de carte pour le groupement. Également connu sous le nom de numéro de groupe dans le SE.

Valeur attendue

Nombre - peut contenir des caractères alphanumériques, des points décimaux, des traits d'union et des traits de soulignement.

Nom du compte

Saisissez un nom pour le formulaire. Également connu sous le nom de nom de groupe dans le SE.

Valeur attendue

Texte - peut contenir des caractères alphanumériques, des points décimaux, des traits d'union et des traits de soulignement.

Si ce champ n'est pas renseigné, le nom du groupe est "[Sans titre]".

Postable

Indiquer si des comptes peuvent être affectés au groupe.

Valeur attendue

  • Y ou 1 ou VRAI-les comptes ne peuvent pas être affectés au groupe
  • N ou 0 ou FALSE - les comptes peuvent être affectés au groupe

Si ce champ n'est pas renseigné, le système suppose que les comptes peuvent être affectés à un groupe et l'une de ces valeurs est ajoutée(N ou 0 ou FAUX).

Type (obligatoire)

Indique si le regroupement appartient au bilan, au compte de résultat, au résultat global ou au résultat net.

Valeur attendue

  • B - Bilan
  • I - pour Compte de résultat
  • C - pour Résultat global
  • N - pour revenu net

Signe

Indiquer si le regroupement doit se faire pour les comptes de débit ou les comptes de crédit.

Valeur attendue

  • D=débit
  • C=crédit;

Si ce champ n'est pas renseigné, le système considère que le débit est le signe de regroupement et la valeur D est appliquée.

Classification des groupes

Saisir la classification de l'automapping pour le groupement.

Valeur attendue

Texte

Peut contenir des caractères alphanumériques, des points décimaux, des traits d'union et des traits de soulignement.

Poids

Saisissez un nom pour le formulaire.

Dans les cas où plusieurs groupes ont la même classification, tout compte ayant cette classification sera affecté au groupe ayant le numéro de pondération le plus élevé. Par exemple, les groupes ayant une pondération positive seront préférés aux groupes sans pondération.

Valeur attendue

Valeur numérique comprise entre -1 et +1.

Si ce champ est laissé vide, la valeur 0est attribuée au poids.

Autoriser

Indiquer si le groupement est soumis à des restrictions de solde de compte.

Valeur attendue

  • * (Comptes avec solde à céder)
  • + (Seuls les comptes ayant un solde débiteur (positif) doivent être affectés)
  • - (Seuls les comptes avec un solde créditeur (négatif) doivent être affectés)

Si ce champ n'est pas renseigné, le système considère que les comptes ayant un solde quelconque peuvent être affectés au groupe et la valeur * est appliquée.

Lien vers un groupe fiscal

Saisissez un numéro de groupe dans la liste déroulante Lien vers un groupe dans la structure de regroupement du code fiscal de la boîte de dialogue Propriétés du groupe.

Cela vous permet d'affecter automatiquement des comptes à des groupes dans d'autres structures de regroupement sur la base de leur affectation initiale à un groupe.

Valeur attendue

Texte

Lien vers un groupe de plans comptables

Saisissez un numéro de groupe dans la liste déroulante Lien vers un groupe dans la structure de regroupement prescrite de la boîte de dialogue Propriétés du groupe.

Cela vous permet d'affecter automatiquement des comptes à des groupes dans d'autres structures de regroupement sur la base de leur affectation initiale à un groupe.

Valeur attendue

Texte

Cycle

Saisissez un cycle pour le groupe (tel que défini dans les paramètres du produit).

Si votre application Cloud comprend un document d'évaluation des risques, lorsque vous affectez un groupe à un cycle d'activité, les éléments de ce groupe seront automatiquement affichés sous ce cycle dans le document, ce qui facilitera l'audit par cycle d'activité.

Valeur attendue

Texte

Zone

Saisissez une zone pour le groupe (telle que définie dans les paramètres du produit).

When you assign a group to an area, items in that group will automatically be shown under this area in the Risk Assessment document, making it easier to audit by area.

Valeur attendue

Texte

Composants

Components provide a more market-driven, region-driven or product-driven visibility logic for trial balance groups. Par exemple, si l'entité opère dans le secteur de l'agriculture, affichez les groupes de balances de vérification qui sont pertinents pour l'agriculture. Pour en savoir plus, voir iXBRL.

Saisissez un ou plusieurs composants à affecter au groupe (tel que défini dans les paramètres du produit).

Valeur attendue

Texte

Utilisez deux points pour séparer plusieurs composants associés à un groupe (exemple : composant1;composant2;composant3).

Nom contraire

Affiche les montants du groupe qui sont opposés au signe normal du groupe.

Par exemple, "Découvert bancaire" apparaîtra à la place de "Banque" lorsque le solde est créditeur.

Valeur attendue

Texte

Nom incohérent

Le cas échéant, saisissez une description alternative pour le groupe à afficher lorsque l'année en cours et l'année précédente ont des signes opposés.

Valeur attendue

Texte

Catégorie de dimension

Saisissez une catégorie de dimension pour le groupe (sélectionnez-la dans la liste définie dans les paramètres du produit).

Valeur attendue

Texte

Dimensions exclues

Saisissez les dimensions qui sont exclues de la catégorie sélectionnée.

Remarque : Cette propriété n'est disponible que s'il existe une valeur pour la catégorie de dimension.

Valeur attendue

Texte

Utilisez deux points pour séparer plusieurs dimensions exclues (exemple : dimension1;dimension2)

Réaffectation de l'objectif pour les comptes contraires

If applicable, enter a group from the Financials grouping structure for accounts with a contrary balance.

Si un compte est automatiquement affecté à ce groupe mais qu'il ne correspond pas au signe sélectionné (débit ou crédit) pour ce groupe, il sera automatiquement affecté à la cible de réaffectation. Par exemple, actif négatif --> passifpositif

Valeur attendue

Texte

Groupe par défaut

Indiquer si le groupe est un groupe par défaut ou non. Pour en savoir plus, voir Types de réponses.

Valeur attendue

  • Y ou 1 ou VRAI
  • N ou 0 ou FAUX
