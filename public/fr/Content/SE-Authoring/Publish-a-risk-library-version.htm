Publier une version de la bibliothèque des risques

Après avoir téléchargé un fichier Excel, vous pouvez publier la version de votre bibliothèque de risques. Une fois la version publiée, vous pouvez associer la bibliothèque de risques aux produits.

Notez que :

  • Vous pouvez publier les versions dans n'importe quel ordre. Par exemple, vous pouvez publier une version corrective (qui comprend des corrections urgentes, telles que des fautes de frappe) même si vous disposez d'une version mineure non publiée.

  • Lorsque vous publiez une nouvelle version d'une bibliothèque de risques, tous les fichiers d'engagement nouveaux et existants pour les produits associés à la bibliothèque sont également mis à jour pour utiliser la dernière version.

Pour commencer, sélectionnez Settings dans le menu Cloud(), puis sélectionnez Risk Library | Manage Risk Library.

Pour publier une version de la bibliothèque des risques :

  1. Sélectionnez la bibliothèque de risques dans le volet de gauche.

  2. Accédez à la note que vous souhaitez masquer.

  3. Sélectionnez PUBLISH.

  4. Dans la boîte de dialogue de confirmation, saisissez le nom de la bibliothèque.
