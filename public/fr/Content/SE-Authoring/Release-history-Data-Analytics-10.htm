Envision

Envision permet de visualiser rapidement les données transactionnelles qui ont été importées dans un engagement AnalyticsAI.

Envision est divisé en plusieurs widgets :

  • Vue d'ensemble
  • Résumé
  • Distribution
  • Date de publication Distribution

Se référer à Envision pour plus d'informations.

Notebooks

Les carnets de notes sont des documents interactifs qui contiennent du code en direct ainsi que du texte en format markdown. Les Notebooks vous permettent d'élaborer et d'explorer des analyses personnalisées et d'en afficher les résultats. Chaque cahier est constitué d'une série de cellules. Exécutez chaque cellule individuellement dans l'ordre pour visualiser son résultat ou exécutez l'ensemble du carnet.

Les fichiers de carnet de notes portent l'extension .ipynb et peuvent être explorés, modifiés et exécutés dans JupyterLab, un éditeur de carnet de notes open-source. L'équipe chargée de l'analyse des données a remanié les tests d'IA analytique prêts à l'emploi pour qu'ils fonctionnent avec un nouveau moteur Notebooks. Vous pouvez y accéder à partir de la plateforme AnalyticsAI.

Remarque : Le lancement initial d'un carnet de notes CWI dans un engagement par chaque utilisateur peut prendre plusieurs minutes. L'équipe chargée de l'analyse des données s'efforce de réduire les temps de chargement initiaux. Les modifications effectuées dans les carnets, y compris dans les cellules de code et les cellules de démarquage, ne sont pas sauvegardées.

Pour plus d'informations, reportez-vous à la section Démarrer avec AnalyticsAI Notebooks SDK .
