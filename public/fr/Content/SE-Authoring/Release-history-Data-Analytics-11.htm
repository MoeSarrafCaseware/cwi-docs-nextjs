Quoi de neuf - Data Analytics DA-11

AnalyticsAI

Des améliorations ont été apportées au widget des exceptions à haut risque :

  • Les tests dont les transactions marquées sont nulles sont désormais filtrés
  • Lorsque le filtre de test est utilisé pour un test qui n'a aucune transaction signalée, le widget affiche désormais un message "Pas de données pertinentes"

Quelques améliorations ont été apportées à la présentation des widgets d'Envision pour cette version. Pour plus d'informations, voir la rubrique Carnets de notes .

Importer des données

Importation de fichiers de transactions

Les améliorations générales suivantes ont été apportées à cette version.

  • Un lien vers les exigences relatives aux fichiers csv valides a été ajouté à la page d'importation des transactions. See CSV transaction file requirements for more information.
  • Un message n'apparaît plus lorsqu'une mise en page de données est appliquée avec succès lors de l'importation d'un fichier de transaction.

Assistant d'importation

Ces améliorations ont été apportées à l'assistant d'importation de transactions :

  • Le champ de recherche de la source de données a été déplacé dans la liste des fournisseurs disponibles
  • Les distributeurs peuvent désormais filtrer la liste des sources de données en ajustant le fichier product.json de leur application. Voir Liste des caractéristiques du produit (API) - Importations pour plus d'informations.
  • Vous pouvez importer des données à partir de progiciels comptables de bureau à l'aide de l'assistant d'importation de transactions. Cette fonction est parallèle à celle du bouton Paquets comptables de bureau existant sur la page d' importation de données . Les auteurs principaux peuvent activer cette importation dans leur application à partir de son fichier product.json. Voir Liste des caractéristiques du produit (API) - Importations pour plus d'informations.
