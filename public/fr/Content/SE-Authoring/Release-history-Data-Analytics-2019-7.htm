AnalyticsAI 2019.7

Elle met en évidence les principaux changements intervenus depuis la dernière version de ce produit.

Mises à jour de l'importation des transactions

Le processus d'importation et de mise en correspondance des fichiers de transaction .csv a été amélioré dans cette version.

Vous pouvez désormais importer plusieurs fichiers de transaction. Dans ce cas, les fichiers sont traités en un seul fichier de données pour l'analyse.

Le processus de mise en correspondance des fichiers de transaction importés a également été amélioré, avec notamment de meilleures descriptions pour chaque type de données.

Pour plus d'informations, voir Importer des transactions de clients .

Amélioration de l'exportation des données

Les données figurant dans le PDF de la synthèse de l'analyse comprennent désormais :

  • Graphique de la série chronologique - Risque/valeur des transactions
  • Liste des tests effectués
  • Liste des comptes influencés
  • Annexe A : Description des tests effectués

Statut des tests

La page de configuration affiche désormais les états des tests comme suit :

  • Un indicateur de progression s'affiche pour tout test en cours.
  • Non exécuté - icône d'état grise.
  • La condition préalable a échoué - icône d'état jaune.
  • Exécution en cours - icône d'état grise.
  • Échec de l'exécution - icône d'état rouge.
  • Exécution réussie - icône d'état verte.
