AnalyticsAI 2019.8

Elle met en évidence les principaux changements intervenus depuis la dernière version de ce produit.

Amélioration de la page de configuration

Lorsque les dates d'un fichier de données ne correspondent pas aux dates de l'engagement, la liste déroulante Exécuter les tests le apparaît désormais sur la page Configuration. Vous pouvez effectuer des tests sur :

  • Toutes les données
  • Année en cours

En outre, le bouton Enregistrer et exécuter les tests s 'appelle désormais Exécuter les tests.

Amélioration du style de la liste déroulante d'étiquetage

Ces améliorations ont été apportées à la boîte de dialogue Importer comme qui apparaît lorsque vous mettez en correspondance un fichier de transactions .csv importé :

  • Le bouton Sélectionner a été supprimé. Il suffit maintenant de cliquer sur un élément disponible dans la liste pour le sélectionner.
  • Les flèches qui apparaissaient à côté de chaque sélection potentielle ont été supprimées.
  • Les propriétés inutilisées qui étaient visibles lors du processus de mappage des fichiers de transactions ont été masquées dans cette version.
