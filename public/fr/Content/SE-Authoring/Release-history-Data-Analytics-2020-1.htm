AnalyticsAI 2020.1

Elle met en évidence les principaux changements intervenus depuis la dernière version de ce produit.

Changements dans l'AnalyticsAI

Un certain nombre d'améliorations ont été apportées à AnalyticsAI pour cette version.

Les traductions sont désormais prises en charge et mises en œuvre

Vous pouvez maintenant sélectionner la langue d'AnalyticsAI dans le menu déroulant Langue en haut à droite de l'écran de connexion à Caseware Cloud. Vous devez être déconnecté pour effectuer cette opération.

Lorsque vous vous reconnectez, vous verrez les éléments suivants d'AnalyticsAI dans la langue sélectionnée :

  • L'écran de connexion
  • Tous les éléments de la page Résultats
  • Tous les éléments de la page Configuration
  • La boîte de dialogue Exporter les données localement
  • Le contenu du PDF de la synthèse de l'analyse lors de l'exportation

Amélioration des noms de fichiers d'exportation par défaut

Les formats de noms de fichiers par défaut pour les fichiers de données exportés ont été améliorés.

Type de fichier exporté

Format du nom de fichier

Résumé de l'analyse PDF

<year>_<month>_<day>_<entity>_AAi Summary.pdf

Résultats (CSV)

<year>_<month>_<day>_<entity>_AAi Audit Evidence.csv

Toutes les données (CSV)

<year>_<month>_<day>_<entity>_AAi All Data.csv

Pour plus d'informations sur l'exportation des résultats, voir Exporter les résultats d'AnalyticsAI.

Cohérence des en-têtes d'exportation CSV

Les informations d'en-tête des fichiers CSV "Résultats" et "Toutes les données" exportés sont désormais cohérentes.

Ajout de la propriété approved_by à l'exportation CSV des résultats

Le fichier CSV créé par une exportation de résultats à partir de la boîte de dialogue Exporter les données localement comprend désormais une colonne approved_by .

Changements dans l'importation des données

Des améliorations ont été apportées au processus d'importation des fichiers CSV de balance de vérification.

Mise en page des données

Cette nouvelle fonctionnalité vous permet d'enregistrer des présentations de données pour les propriétés des transactions. Cela permet de gagner du temps en évitant d'avoir à mapper des fichiers de transactions similaires pour plusieurs importations.

Vous pouvez maintenant mapper un fichier de transactions importé de la manière habituelle, puis enregistrer la mise en page pour une utilisation ultérieure.

Pour plus d'informations, voir Importer les transactions du client .
