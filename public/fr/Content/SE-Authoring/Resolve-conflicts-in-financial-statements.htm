Résoudre les conflits dans les états financiers

Après avoir copié des documents à partir de votre modèle source, il se peut que vous rencontriez des liens rompus vers.. :

  • Condition de réponse

  • Financials group conditions

  • Engagement glossary terms

  • Visibilité manquante / conditions de texte dynamique

    • Remarque : si plusieurs liens brisés sont générés par un texte dynamique, chaque lien devra être résolu séparément.

  • Conditions de la formule de réponse à la liste de contrôle manquante

En cas de conflit, le volet Liens brisés s'affiche à droite pour vous permettre d'évaluer et de résoudre les conflits. Le volet affiche les conflits regroupés par nom de document

Conseil : Vous pouvez également sélectionner le bouton () pour afficher le volet Liens brisés. Notez que le nombre de conflits est affiché dans le badge rouge.

Pour résoudre un lien brisé :

  1. Faites défiler l'écran jusqu'au document dans lequel vous souhaitez résoudre les conflits.

  2. Select the broken link you want to resolve.

    3. Une fois sélectionné, le document contenant le lien brisé s'ouvre et la boîte de dialogue du lien brisé apparaît.

    La boîte de dialogue Lien brisé affiche les détails du conflit et les options de résolution.

    Remarque : La fonctionnalité de copier-coller intelligent présente certaines limites qui seront corrigées dans les prochaines versions ; vous pouvez consulter ces limites ici. Une boîte de dialogue d'erreur s'affiche lorsqu'une résolution n'est pas prise en charge.

  3. En utilisant la boîte de dialogue des liens brisés, vous pouvez

    • Utilisez les menus déroulants disponibles pour sélectionner les options valables.

      • Par exemple, si un terme du glossaire d'engagement copié à partir d'un document source n'existe pas dans votre application cloud, sélectionnez un terme dans le glossaire de votre application pour l'insérer à la place dans le document.

    • Sélectionnez le bouton de suppression () pour supprimer l'élément où se trouve le conflit ou ignorez le conflit pour le résoudre plus tard

      • Par exemple, si un terme du glossaire d'engagement copié à partir d'un document source n'existe pas dans votre application en nuage, supprimez-le du document.

  4. Sélectionnez Enregistrer pour appliquer vos modifications.

    Remarque : Le bouton SAUVEGARDER n'est activé que lorsque vous effectuez des sélections valides dans la boîte de dialogue. Vous pouvez appuyer sur la toucheCANCEL à tout moment pour annuler vos modifications.
