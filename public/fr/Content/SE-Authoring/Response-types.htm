Types de réponses :

La liste suivante décrit les types de réponses que vous pouvez ajouter à une procédure de liste de contrôle.

Manuel

Un champ de texte dans lequel les utilisateurs peuvent saisir une réponse personnalisée.

Réponse attendue de l'utilisateur Saisissez les informations demandées.

Liste de sélection

Une liste déroulante dans laquelle les utilisateurs peuvent sélectionner une réponse parmi une liste de réponses. Vous pouvez personnaliser la liste des réponses en fonction de vos besoins. Les réponses par défaut sont Oui, Non et Sans objet.

Réponse attendue de l'utilisateur Sélectionnez un élément dans la liste déroulante.

Multi-Picklist

Une boîte déroulante dans laquelle les utilisateurs peuvent sélectionner plusieurs réponses à partir d'une liste de réponses. Les réponses par défaut sont Oui, Nonet Sans objet.

Réponse attendue de l'utilisateur Sélectionnez un ou plusieurs éléments dans la liste déroulante.

Date

Un champ de sélection de date avec une icône de calendrier en ligne où les utilisateurs peuvent sélectionner une date.

Réponse attendue de l'utilisateur Sélectionnez une date dans le calendrier.

Informations sur l'entité

Un champ qui affiche des informations sur l'entité cliente. Les informations sur l'entité sont directement liées au profil du client sur CasewareCloud. Associez un ou plusieurs des champs suivants à votre procédure :

  • Nom légal
  • Nom d'exploitation
  • Date de fin d'année
  • Date de début de l'année
  • Année en cours
  • Année précédente
  • Adresse
  • Site web
  • Numéro de l'entité
  • Numéro d'entreprise
  • Type d'organisation
  • Téléphone professionnel
  • Fax commercial
  • Téléphone portable
  • Téléphone fixe
  • Codes de l'industrie
  • Client depuis
  • Date de constitution

Réponse attendue de l'utilisateur Visualiser et vérifier les données des clients. Les informations affichées renvoient à Caseware Cloud. L'utilisateur peut modifier toute information incorrecte si nécessaire.

Informations sur le contact

Un champ qui affiche des informations sur l'un des contacts du client. Les informations de contact sont directement liées au profil utilisateur du client sur CasewareCloud. Associez un ou plusieurs des champs suivants à votre procédure :

  • Prénom
  • Nom de famille
  • Initiales
  • Courriel
  • Titre du poste
  • Description
  • Accueil
  • Téléphone
  • travail*
  • Téléphone
  • Téléphone portable

Réponse attendue de l'utilisateur Visualiser et vérifier les données de contact des clients. Les informations affichées renvoient à Caseware Cloud. L'utilisateur peut modifier toute information incorrecte si nécessaire.
