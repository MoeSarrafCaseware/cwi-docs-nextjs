Réviser et ajouter des balises XBRL

Remarque : Cette fonction n'est actuellement pas disponible dans toutes les régions.

Le panneau de révision XBRL vous permet d'identifier et de réviser les balises du système, d'ajouter des balises personnalisées et d'activer la mise en évidence de la révision pour les faits XBRL balisés dans vos états financiers

Examiner les faits XBRL

Vous pouvez lancer le panneau XBRL à partir de l'icône XBRL dans le menu supérieur et commencer à réviser vos étiquettes XBRL.

Examiner les faits XBRL :

  1. Sélectionnez la vue Personnes dans le menu supérieur.

  2. Si plusieurs taxonomies sont disponibles dans votre produit, sélectionnez la taxonomie que vous souhaitez examiner.

  3. Sélectionnez START REVIEW.

    Le panneau de révision XBRL est lancé et s'affiche sur le côté droit de la page.

  4. Vous pouvez maintenant examiner les faits XBRL marqués dans vos états financiers et effectuer les modifications nécessaires. Les faits marqués sont mis en évidence par les couleurs suivantes pour indiquer leur statut de marquage :

    • Vert - Balise du système qui n'a pas été éditée

    • Rouge - Étiquette du système qui a été éditée

    • Jaune - Etiquette personnalisée qui a été créée dans l'engagement

    • Bleu - La balise actuellement sélectionnée. La couleur de la bordure de la balise sélectionnée sera verte, rouge ou jaune pour indiquer son état de balisage

Ajouter une marque personnalisée

Si vous avez besoin d'étiqueter des faits XBRL supplémentaires, vous pouvez ajouter des étiquettes personnalisées à partir de votre taxonomie XBRL au cours du processus de révision. Assurez-vous d'avoir lancé le panneau de révision XBRL avant d'ajouter des balises personnalisées.

Pour ajouter une balise personnalisée :

  1. Sélectionnez le paragraphe ou le tableau que vous souhaitez baliser, puis sélectionnez Ajouter une balise.

    Si vous devez baliser une formule ou un espace réservé dans un paragraphe, sélectionnez le paragraphe. Vous pouvez spécifier la formule ou l'espace réservé que vous souhaitez baliser plus tard dans le processus de balisage.

  2. Localisez et sélectionnez la balise dans la taxonomie. Vous pouvez utiliser la boîte de recherche pour trouver la bonne étiquette.

    Si vous avez déjà ajouté des balises personnalisées, la taxonomie s'ouvrira pour afficher la dernière balise sélectionnée.

  3. Lorsque vous sélectionnez une balise, des informations clés la concernant s'affichent sous la barre de recherche, telles que le nom de l'élément, la période et le type. Examinez ces informations pour vous assurer que vous avez sélectionné la bonne étiquette, puis sélectionnez Confirmer.

  4. Si vous souhaitez baliser un lien de données ou un espace réservé spécifique dans un paragraphe, sélectionnez l'élément à baliser dans le menu déroulant Contenu.

    Vous devrez alors sélectionner à nouveau la balise dans la taxonomie en sélectionnant l'icône Editer () à côté du champ Élément .

  5. Sélectionnez un contexte pour votre balise dans la liste déroulante. Si un contexte approprié n'existe pas, vous pouvez en créer un.

    Si vous sélectionnez un contexte qui entraînera des erreurs de diagnostic lors de l'exportation de rapports XBRL, un message s'affichera pour indiquer pourquoi le contexte ne peut pas être choisi pour la balise.

  6. S'il y a lieu, sélectionnez une option de journalisation. S'il n'existe pas de code numérique approprié, vous pouvez en créer un.

    Si vous sélectionnez un numérique qui entraînera des erreurs de diagnostic lors de l'exportation de rapports XBRL, un message s'affichera pour indiquer pourquoi le contexte ne peut pas être choisi pour la balise.

  7. Select Apply tag.

Votre contenu est maintenant balisé et surligné en jaune pour indiquer qu'il s'agit d'un fait balisé personnalisé.

Supprimer une balise personnalisée

Vous pouvez supprimer les balises personnalisées que vous avez ajoutées.

Remarque : Les balises système ne peuvent pas être supprimées, mais elles peuvent être modifiées

Pour supprimer une balise personnalisée :

  1. Sélectionnez le fait XBRL qui ne nécessite plus de balise personnalisée.

  2. Sélectionnez l'icône Editer ().

  3. Dans la boîte de dialogue de confirmation, sélectionnez Oui.
