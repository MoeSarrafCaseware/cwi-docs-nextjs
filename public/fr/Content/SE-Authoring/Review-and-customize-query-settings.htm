Examiner et personnaliser les paramètres des questions de requête

Si vous souhaitez afficher ou personnaliser les paramètres d'une question spécifique dans le document, sélectionnez Autres actions () | Paramètres. Once you're done with your changes, select Save.

Les options suivantes sont proposées dans la boîte de dialogue Paramètres de la requête :

Autoriser la saisie de notes dans la requête

Sélectionnez cette option pour ajouter une zone de texte sous la réponse où les utilisateurs peuvent saisir des notes. Vous pouvez ajouter un espace réservé à la note dans le champ affiché.

Autoriser les lignes multiples

Sélectionnez cette option pour permettre aux utilisateurs de fournir plus d'une réponse pour la procédure. Cette option permet d'activer le bouton +Row dans la procédure.

Remarque : Si votre procédure comporte plus d'une réponse, la sélection de ce paramètre affichera automatiquement vos réponses dans un tableau sous la procédure. Les étiquettes des espaces réservés de la réponse sont affichées dans la ligne d'en-tête.

Ensembles de réponses

  • Sélectionnez +Réponse pour ajouter une réponse.
  • Sélectionnez Supprimer () pour supprimer une réponse.
  • Sélectionnez Editer () pour éditer une réponse.

Afficher les réponses sous la procédure

Sélectionnez cette option pour afficher les réponses sous la requête.
