SE 2020-03 hot fixes

Voici un résumé des correctifs qui ont été déployés pour SE 2020-03.

2020-03-H1

Version : 2020 / 03 / 496

Voici la liste des problèmes qui ont été corrigés avec le déploiement de la SE 2020-03-H1.

  • [SE-19545] - Le regroupement et le dégroupement des comptes de la balance de vérification dans les tableaux dynamiques ne fonctionnent pas comme prévu.

  • [SE-19550] - Un problème technique qui affecte les performances de la page de données.

  • [SE-19468] - La consolidation de plusieurs entités ne fonctionne pas comme prévu. Certaines entités sont déséquilibrées.

2020-03-H2

Version : 2020 / 03 / 498

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-03-H2.

  • [SE-19610] - Si les utilisateurs renomment un document de requête complété à partir de la page Documents , les fichiers clients téléchargés ne sont plus disponibles dans les questions de la requête.

  • [SE-19692] - Dans les tableaux dynamiques, les faits iXBRL dans les groupes personnalisés ne sont pas exportés lorsque les groupes sont réduits.

  • [SE-19742] - La liste des dimensions n'est pas disponible lorsque les utilisateurs tentent d'ajouter une ligne de dimension à un tableau dynamique IFRS.

  • [SE-17876] - La section " Guidance " ne s'affiche pas comme prévu dans les anciens rapports sur les risques et les contrôles.

  • [SE-19655] - Dans l'ancienne présentation de la requête, les utilisateurs ne peuvent pas masquer manuellement les questions de la requête.

2020-03-H3

Version : 2020 / 03 / 500

Voici la liste des problèmes qui ont été corrigés avec le déploiement de la SE 2020-03-H3.

  • [SE-19801] - Les utilisateurs ne peuvent pas ajouter ou modifier des tableaux dynamiques dans les états financiers.

  • [SE-19773] - La numérotation des procédures dans les listes de contrôle ne fonctionne pas comme prévu.

2020-03-H4

Version : 2020 / 03 / 504

La SE 2020-03-H4 apporte des améliorations aux :

  • La stabilité des environnements de production

  • La stabilité et la performance des environnements de création de produits

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-03-H4.

  • [SE-19845] - Dans les tables dynamiques, les termes du glossaire d'engagement dans les lignes totales ne sont pas mis à jour correctement lorsqu'un terme est modifié ailleurs.

  • [SE-19826] - Le menu Plus d'actions pour les domaines des états financiers ne fonctionne pas comme prévu. Les auteurs ne peuvent pas sélectionner d'options dans le menu.

  • [SE-19907] - Dans les modèles de produits, les dimensions prédéfinies dans les tableaux dynamiques IFRS ne fonctionnent pas comme prévu.

  • [SE-19859] - Les informations sur le groupe parent et les propriétés du groupe ne sont pas incluses lorsque les auteurs exportent des groupes.

  • [SE-19897] - Lorsque les utilisateurs sélectionnent un risque de contrôle dans une carte d'évaluation des risques, la carte s'effondre.

  • [SE-19903] - Problèmes techniques empêchant l'ouverture des engagements de consolidation.

  • [SE-19982] - Un problème technique qui affecte les performances lorsque les développeurs de formulaires personnalisés travaillent à distance.

2020-03-H5

Version : 2020 / 03 / 506

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-03-H5.

  • [SE-20016] - Les ajustements d'élimination comptabilisés sur les comptes à solde nul ne s'affichent pas dans les états financiers.

  • [SE-20107] - Si les utilisateurs suppriment une référence de document de la procédure d'orientation, ferment la liste de contrôle et la rouvrent, la référence est également supprimée de la procédure.

  • [SE-20081] - Les liens vers les données de matérialité ajoutés par les auteurs ne fonctionnent pas comme prévu.

  • [SE-19334] - Les utilisateurs peuvent créer des engagements pour des produits gérés en dehors de la plateforme Caseware Cloud.

  • [SE-20042] - Les sections d'analyse de certains groupes de balances de vérification s'affichent vides.

  • [SE-20114] - Le report ne fonctionne pas comme prévu lorsque les utilisateurs finaux modifient les notes des états financiers. Les notes ne sont pas affichées dans le bon ordre.

2020-03-H6

Version : 2020 / 03 / 507

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-03-H6.

  • [SE-20229] - Lorsque les utilisateurs impriment les états financiers, les graphiques d'analyse ne sont pas inclus dans le PDF généré.

  • [SE-20219] - L'option Télécharger la copie de référence ne fonctionne pas comme prévu. Les utilisateurs reçoivent un message d'erreur et le téléchargement échoue.

  • [SE-20178] - L'option Télécharger la copie de référence produit des fichiers corrompus.

  • [SE-20175] - Le texte dynamique dans les états financiers ne s'affiche pas comme prévu. Des balises html supplémentaires s'affichent avant et après le texte dynamique.

2020-03-H7

Version : 2020 / 03 / 510

Voici la liste des problèmes qui ont été corrigés avec le déploiement de la SE 2020-03-H7.

  • [SE-20174] - Dans les tableaux dynamiques qui n'affichent qu'une seule année de données, les soldes des années précédentes affectent la visibilité d'une ligne de groupe ou de compte.

  • [SE-20478] - Les options Imprimer et Enregistrer dans le nuage ne fonctionnent pas comme prévu pour les documents qui incluent un lien de données (fx) Réponse (réponse à la procédure de la liste de contrôle) avec un type de réponse Manuel. Le fichier PDF ne se génère pas.

  • [SE-20343] - Une erreur se produit lors du report d'un engagement où,

    • Des ajustements ont été apportés à un solde de groupe de l'année précédente

    • Ce groupe a également été transformé en groupe non assignable pour l'année en cours.

2020-03-H8

Version : 2020 / 03 / 517

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-03-H8.

  • [SE-21018] - Le personnel ne peut pas voir les questions de requête lorsqu'il essaie de voir les documents de requête en utilisant les pages Chronologie ou Activités.

  • [SE-21111] - Dans la page Balance de vérification, lorsque les utilisateurs se connectent à un compte SuperMate pour importer les données d'un client, ils ne peuvent pas se déconnecter du compte.

  • [SE-19652] - Lorsque les utilisateurs de MacOS modifient et enregistrent un document Microsoft Office dans un fichier d'engagement, le fichier s'enregistre correctement mais un message d'erreur s'affiche.

  • [SE-20246] - Les liens de données (fx) faisant référence à des réponses de procédures ne sont pas évalués si les utilisateurs modifient les réponses après avoir inséré les liens de données. Pour plus de détails, voir le problème connu SE 2020-03 SE-20961.
