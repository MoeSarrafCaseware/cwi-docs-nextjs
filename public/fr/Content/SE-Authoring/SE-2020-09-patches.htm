SE 2020-09 correctifs

Voici un résumé des correctifs qui ont été déployés pour SE 2020-09.

2020-09-U1

Version : 2020 / 09 / 551

Le déploiement de la SE 2020-09-U1 apporte des améliorations techniques liées à la sécurité, à la stabilité et aux capacités de journalisation.

2020-09-U2

Version : 2020 / 09 / 552

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-09-U2.

  • [SE-23213] - Les tableaux d'analyse ne s'affichent pas comme prévu. Les icônes Annotation() et Issue () se superposent à la dernière colonne du tableau.

  • [SE-23236] - Une erreur "Engagement invalide" se produit lorsque les utilisateurs tentent d'ouvrir un engagement.

2020-09-U3

Version : 2020 / 09 / 554

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-09-U3.

  • [SE-23511] - Les liens de données (fx) qui incluent des guillemets ("") dans leur sortie ne s'affichent pas correctement et peuvent entraîner des erreurs.

  • [SE-23509] - Les utilisateurs ne peuvent pas sélectionner "28 février 2021" comme date de fin d'année dans leurs propriétés d'engagement.

  • [SE-23402] - Lorsque les utilisateurs impriment les états financiers, le PDF généré ne s'affiche pas comme prévu. Le contenu des listes ne s'affiche pas dans son intégralité.

  • [SE-23428] - En mode révision XBRL, le volet Documents se superpose au volet Révision XBRL et les utilisateurs ne peuvent pas sélectionner le bouton Exporter .

  • [SE-23446] - Si les utilisateurs choisissent de ne pas inclure les notes aux états financiers, une section de notes vide est incluse lorsqu'ils impriment les états financiers.

  • [SE-23458] - Dans les tableaux dynamiques, lorsque les utilisateurs choisissent d'afficher une ligne avec le signe opposé, le signe opposé n'est pas appliqué aux colonnes de calcul personnalisées .

  • [SE-23489] - Lorsque le paramètre Décimales est défini pour afficher zéro décimale dans les paramètres du produit, les symboles décimaux ne s'affichent pas dans le fichier d'engagement.

2020-09-U4

Version : 2020 / 09 / 559

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-09-U4.

  • [SE-23544] - Les utilisateurs ne peuvent pas sélectionner "28 février 2021" comme date de fin d'année dans leurs propriétés d'engagement.

  • [SE-23592] - Lorsque les utilisateurs impriment des états financiers, la table des matières du PDF généré n'inclut pas les numéros de page.

  • [SE-23614] - Lorsque les auteurs utilisent l'option Copier à partir d'un autre produit dans le modèle de produit, le volet Liens brisés affiche des conflits qui ne peuvent pas être résolus. Le message "Ce type de lien n'est pas pris en charge actuellement" s'affiche lorsque les auteurs tentent de résoudre le conflit.

  • [SE-23534] - Les utilisateurs reçoivent plusieurs messages d'erreur lorsqu'ils utilisent un proxy pour développer SE.

  • [SE-23435] - Une erreur se produit lorsque les utilisateurs tentent de reporter un engagement.

2020-09-U5

Version : 2020 / 09 / 561

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-09-U5.

  • [SE-23697] - Dans les tableaux dynamiques, si des dérogations aux états financiers et des différences d'arrondi sont appliquées aux comptes d'un groupe, la ligne du total ne se calcule pas correctement lorsqu'elle est réduite.

  • [SE-23450] - Les annotations dans les tableaux d'analyse ne sont pas incluses dans les PDF générés lors de l'utilisation de l'option Télécharger la copie de référence.

  • [SE-23706] - Les missions créées à partir d'un report ne comprennent pas les liens de données (fx) qui ont été ajoutés ou modifiés dans le fichier d'origine.

  • [SE-23653] - Les auteurs ne sont pas en mesure de visualiser les notes d'états financiers qui comportent plus de 100 sections de contenu, telles que du texte ou des tableaux dynamiques.

  • [SE-23619] - L'option d'aperçu avant impression ne fonctionne pas comme prévu. Il y a des différences entre les sauts de page dans l'aperçu avant impression et les documents imprimés.

  • [SE-23557] - Dans les tableaux IFRS (tableaux de mouvement), les utilisateurs ne peuvent pas comptabiliser les ajustements pour les groupes qui sont affectés à des dimensions de deuxième niveau.

2020-09-U6

Version : 2020 / 09 / 566

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-09-U6.

  • [SE-24058] - Dans les états financiers, les liens de données (fx) dans les sections de texte s'affichent comme "#ValueUnavailable" si les informations d'orientation pour la section sont modifiées.

  • [SE-23422] - Dans les requêtes, lorsque les utilisateurs essaient d'ajouter des liens de données (fx) au texte de la question, le lien de données est déplacé au début du texte.

  • [SE-23999] - Dans les missions créées à partir d'un report, les utilisateurs ne sont pas en mesure d'ajouter des sauts de page aux états financiers.

  • [SE-23831] - Les formules de connecteur existantes dans les fichiers sont toujours disponibles après la désactivation de la licence Caseware Cloud Connector d'une entreprise.

2020-09-U8

Version : 2020 / 09 / 569

Un problème a été corrigé avec le déploiement de la SE 2020-09-U8.

[SE-24233] - Les documents externes qui ont été corrompus lorsqu'ils ont été téléchargés dans un engagement et qui ont été réparés depuis ne sont pas accessibles lorsque les utilisateurs utilisent l'option Télécharger la copie de référence. Les fichiers PDF sont corrompus.

2020-09-U9

Version : 2020 / 09 / 578

La SE 2020-09-U9 corrige un problème de fuite de mémoire. Par conséquent, les utilisateurs ne peuvent pas télécharger des documents externes tels que Word, Excel et PDF.
