Rustines SE 2020-12

Voici un résumé des correctifs qui ont été déployés pour la SE 2020-12.

2020-12-U1

Version : 2020 / 12 / 581

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-12-U1.

  • [SE-24714] - Dans la nouvelle page Documents, les rôles de signature ne sont pas affichés dans le bon ordre et ne fonctionnent pas comme prévu.

  • [FLAT-6164] - Une erreur se produit lorsque les utilisateurs tentent d'exporter des fichiers XBRL.

2020-12-U3

Version : 2020 / 12 / 583

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-12-U3.

  • [SE-25012] - Un problème technique qui entraîne une dégradation des performances et des interruptions intermittentes.

  • [SE-24938] - Lorsque la page des nouveaux documents est activée, les utilisateurs ne peuvent pas effectuer de report en bloc si le dossier de mission de l'année précédente comprend des types de questions à prendre en compte pour l'année suivante.

  • [SE-24965] - Dans les lettres et les mémos, le formatage appliqué aux éléments de la liste à puces avec des conditions de visibilité n'est pas sauvegardé.

2020-12-U4

Version : 2020 / 12 / 584

La SE 2020-02-U4 atténue un problème technique qui entraîne une dégradation des performances et des interruptions intermittentes.

2020-12-U5

Version : 2020 / 12 / 588

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-12-U5.

  • [SE-24382] - Les modèles de produits contenant des fichiers d'historique volumineux ne sont pas exportés comme prévu.

  • [SE-24985] - Lorsque la nouvelle page Documents est activée, les URL des liens externes ne sont pas reportées.

  • [SE-25066] - Les valeurs saisies dans les cellules d'un tableau ne sont pas sauvegardées si l'utilisateur utilise la touche Tab ou Enter pour naviguer entre les cellules.

  • [SE-25042] - Si les paramètres d'impression d'un document sont modifiés dans le modèle d'entreprise et que le document est ensuite supprimé dans les versions ultérieures du modèle de produit, une erreur se produit lorsque les auteurs d'entreprises tentent de mettre à jour le modèle d'entreprise.

2020-12-U6

Version : 2020 / 12 / 589

La SE 2020-02-U6 traite d'un changement de configuration qui s'applique à la région canadienne.

Les autres régions peuvent se référer aux notes SE 2020-12-U5 pour plus de détails sur les problèmes concernant votre région qui ont été corrigés dans cette version.

2020-12-U7

Version : 2020 / 12 / 590

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-12-U7.

  • [SE-25258] - Les utilisateurs ne peuvent pas revenir sur les soldes écrasés dans les tables dynamiques.

  • [SE-25142] - L'option None pour les schémas de signature ne fonctionne pas comme prévu. Lorsque les auteurs sélectionnent cette option pour un document dans le modèle de produit, les utilisateurs finaux peuvent toujours signer le document.

  • [SE-25281] - Un problème qui provoque l'interruption de certains engagements (au sein d'applications spécifiques - pas toutes les applications Cloud).

  • [SE-25208] - Lorsque les utilisateurs exportent un fichier XBRL, les sections de texte cachées et les tableaux dynamiques ne sont pas marqués comme étant cachés dans le fichier.

  • [SE-24955] - Dans les missions qui utilisent l'ancien modèle de requête, les contacts reçoivent des erreurs d'autorisation lorsqu'ils ouvrent des requêtes.

  • [SE-25193] - Un problème technique qui affecte les performances dans les listes de contrôle.

  • [SE-24437] - Une erreur "Échec de l'archivage" se produit lors de l'utilisation de l'option Télécharger la copie de référence si un autre utilisateur télécharge un fichier vers l'engagement en même temps.

2020-12-U9

Version : 2020 / 12 / 594

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-12-U9.

2020-12-U11

Version : 2020 / 12 / 599

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2020-12-U11.

  • [SE-25562] - Dans les modèles de produits qui supportent plusieurs langues, les liens de données (fx) dans les états financiers ne fonctionnent pas comme prévu. Lorsqu'un auteur change de langue, le lien de données s'affiche comme "#ValueUnavailable#" et l'auteur ne peut pas ouvrir la boîte de dialogue Lien de données pour modifier le lien.

  • [SE-25476] - Dans les tableaux dynamiques, les utilisateurs ne peuvent pas ramener les cellules du tableau à leurs valeurs d'origine après avoir remplacé la valeur calculée.

  • [SE-25254] - Une amélioration technique liée aux capacités de journalisation.

  • [SE-25316] - La tentative de report d'un engagement pour le cas d'utilisation suivant entraîne une erreur :

    1. Les utilisateurs ont ajouté des assertions à une procédure de liste de contrôle dans le dossier de l'année précédente

    2. L'auteur principal ou l'auteur principal a ajouté des assertions pour la même procédure dans le modèle de l'année en cours.

  • [SE-25707] - Un problème technique qui cause des problèmes de performance dans la nouvelle page Documents.
