SE 2021-03 correctifs

Voici un résumé des correctifs qui ont été déployés pour SE 2021-03.

2021-03-U1

Version : 2021 / 03 / 611

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-03-U1.

  • [SE-26547] - Dans les tableaux dynamiques, les soldes des comptes des groupes personnalisés s'affichent avec des valeurs nulles après report.

  • [SE-26495] - Dans la page Données, l'assignation de comptes à un groupe parent différent dans un groupement fiscal ne fonctionne pas comme prévu. Les utilisateurs doivent ouvrir deux fois la boîte de dialogue Modifier avant de pouvoir sélectionner un nouveau groupe parent.

  • [SE-26270] - Les préférences de formatage sélectionnées pour les titres dans les paramètres du produit sont appliquées aux titres des notes.

  • [SE-26228] - Lorsque les utilisateurs impriment les états financiers, les dernières lignes des sections de texte sont coupées dans le PDF généré.

  • [SE-26410] - Une erreur se produit lorsque les auteurs tentent d'ouvrir le modèle de produit s'ils suppriment une langue de contenu alors qu'ils ont déjà ajouté du contenu dans cette langue.

  • [SE-26425] - Amélioration des performances de la nouvelle page Documents.

  • [SE-26490] - Dans la nouvelle page Documents, l'option Télécharger le journal des signatures n'est pas disponible pour les requêtes.

  • [SE-26515] - La boîte de dialogue des erreurs de validation XBRL s'affiche lorsque la propriété showValidationErrorMessage est définie sur false.

  • [SE-26629] - Lorsque les utilisateurs exportent des fichiers XBRL, les zones, la table des matières et les colonnes de notes dans les tableaux dynamiques ne s'affichent pas comme prévu dans le fichier généré.

2021-03-U2

Version : 2021 / 03 / 615

Le déploiement de la SE 2021-03-U2 apporte des améliorations techniques liées à la stabilité.

2021-03-U3

Version : 2021 / 03 / 616

Un problème a été corrigé lors du déploiement de la SE 2021-03-U3.

[SE-26707] - Les utilisateurs ne peuvent pas importer de données depuis Xero.

2021-03-U4

Version : 2021 / 03 / 622

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-03-U4.

  • [SE-26685] - Les lignes de compte dans les ajustements ne s'affichent pas dans l'ordre dans lequel elles ont été créées.

  • [SE-26699] - Dans les missions de consolidation, les soldes ne s'affichent pas pour les lignes totales dans les tableaux dynamiques si le tableau est paramétré pour utiliser des données consolidées.

  • [SE-26703] - Dans les produits qui supportent plusieurs langues, les modifications apportées par l'utilisateur final au texte dynamique ne sont pas sauvegardées pour toutes les langues dans les engagements reportés.

  • [SE-26803] - Lorsque les utilisateurs impriment les états financiers, le tableau d'analyse des comptes ne s'affiche pas comme prévu.

  • [SE-26789] - Les utilisateurs reçoivent une erreur lorsqu'ils tentent de reporter un engagement.

  • [SE-26762] - Dans les listes de contrôle, lorsque les utilisateurs masquent manuellement les procédures après la signature du réviseur, les modifications ne sont pas affichées lorsque l'on utilise l'option Modifications depuis la création de l'engagement dans les outils de révision.

  • [SE-26795] - Lorsque les utilisateurs impriment la page Ajustements , les ajustements sont répartis sur plusieurs pages.

2021-03-U5

Version : 2021 / 03 / 623

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-03-U5.

  • [SE-26835] - Les utilisateurs ne peuvent pas modifier le document de matérialité.

  • [SE-26903] - Certaines applications Cloud ne fonctionnent pas sur les appareils intelligents (téléphones et tablettes).

  • [SE-26916] - Le volet Plan du document n'est plus disponible dans les listes de contrôle lorsque les auteurs intègrent un formulaire personnalisé.

  • [SE-26772][SE-26913] - Dans les engagements créés à partir d'un report, les tableaux dynamiques ne s'affichent pas comme prévu.

  • [SE-26906] - Les valeurs des tableaux IFRS (mouvement) ne sont pas incluses dans l'exportation XBRL.

  • [SE-26929] - Formatage incorrect des cellules numériques dans les formulaires personnalisés intégrés aux états financiers.

  • [SE-26934] - Dans les tableaux dynamiques, les valeurs inversées (réaffectées dynamiquement) dans les lignes Total ne sont pas alignées correctement.

2021-03-U6

Version : 2021 / 03 / 624

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-03-U6.

  • [SE-27046] - Une erreur se produit lorsque les utilisateurs tentent d'importer des documents PDF depuis Class Super dans les cas d'utilisation suivants :

    • Les auteurs créent l'espace réservé de destination et le déplacent ensuite dans un autre dossier du modèle de produit, puis les utilisateurs tentent d'importer les PDF.

    • Après avoir importé les documents PDF, les utilisateurs déplacent un document dans un autre dossier, puis tentent de le réimporter.

  • [SE-27069] - Dans les missions créées à partir d'un report, la mise en forme des documents ne s'affiche pas tant que les utilisateurs n'acceptent pas ou n'ignorent pas les modifications du report.

  • [SE-27015] - Les tableaux dynamiques ne s'affichent pas comme prévu. Les lignes de titre affichent une valeur de 0,0 dans les colonnes de calcul personnalisé et les valeurs négatives ne sont pas alignées correctement.

2021-03-U7

Version : 2021 / 03 / 636

Voici la liste des problèmes qui ont été corrigés avec le déploiement de la SE 2021-03-U7.

  • [SE-27419] - Lorsque les utilisateurs impriment les états financiers, les couleurs des polices ne s'affichent pas comme prévu dans le PDF généré.

  • Les questions suivantes s'appliquent aux missions créées à partir d'un report :

    • [SE-27183] |[SE-27127] - Problèmes techniques provoquant des erreurs lors de l'ouverture de missions.

    • [SE-27330] - Lorsque les utilisateurs impriment des états financiers, les lignes du tableau dynamique qu'ils ont ajoutées dans le fichier de l'année précédente sont surlignées en vert dans le PDF généré.

    • [SE-27246] - Le fichier reporté n'est temporairement disponible que pour l'utilisateur qui a effectué le report et les utilisateurs ayant un rôle de sécurité administrateur.

    • [SE-26854] - Les annotations ajoutées aux procédures de la liste de contrôle dans le fichier de l'année précédente sont dupliquées.

    • [SE-27240] - Le formatage du texte existant dans le champ Résumé des procédures de la liste de contrôle ne s'applique pas tant que les utilisateurs n'acceptent pas ou n'ignorent pas les modifications du report.

  • [SE-27456] - Une erreur se produit lorsque les utilisateurs tentent d'utiliser l'option Télécharger la copie de référence. Les requêtes ne sont pas incluses dans les PDF générés.

  • [SE-27413] - Dans les tables IFRS, une erreur se produit lorsque les utilisateurs sélectionnent une dimension de niveau primaire pour une colonne.

  • [SE-27138] - Lorsque des questions de requête incluant des liens de données (fx) sont liées à des procédures de la liste de contrôle, les liens de données ne s'affichent pas dans la liste de contrôle.

  • [SE-27194] - Une erreur "Download Reference Copy Failed" se produit lorsque les utilisateurs tentent d'utiliser l'option Download reference copy si le document Summary of Misstatements est inclus dans les PDF générés.
