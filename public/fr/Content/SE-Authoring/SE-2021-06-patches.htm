SE 2021-06 rustines

Voici un résumé des correctifs qui ont été déployés pour SE 2021-06.

2021-06-U1

Version : 2021 / 06 / 648

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-06-U1.

  • [SE-27953] - Dans les lettres, lorsque les utilisateurs copient des éléments de liste avec des conditions de visibilité, la logique de visibilité ne fonctionne pas comme prévu pour les nouveaux éléments de liste (collés).

  • [SE-27785] - Les auteurs ne peuvent pas ajouter de codes de référence de risque dans les paramètres du produit.

  • [SE-27847] - L'importation de données depuis BGL Simple Fund 360 ou Class Super ne fonctionne pas comme prévu. La boîte de dialogue d'importation est vide et invite l'utilisateur à se connecter au logiciel de comptabilité.

  • [SE-27916] - Dans les listes de contrôle, lorsque les utilisateurs sélectionnent Rejeter les suggestions de report restantes, les suggestions de report précédemment acceptées s'affichent comme rejetées. Lorsque les utilisateurs actualisent la page, les suggestions s'affichent correctement.

  • [SE-27924] - Les paramètres de visibilité dans les listes de contrôle ne fonctionnent pas comme prévu. Si les conditions de visibilité d'une procédure sont remplies, mais que les conditions relatives au groupe ou au sous-groupe de cette procédure ne le sont pas, le titre du groupe ou du sous-groupe ne s'affiche pas.

  • [SE-27927] - Un problème technique a été identifié après avoir examiné les journaux de production.

  • [SE-27928] - Problème technique provoquant des erreurs de report.

  • [SE-28018] - Un problème technique provoquant des erreurs lorsque les utilisateurs tentent de modifier les formules de calcul dans les tableaux dynamiques.

  • [SE-27835] - Dans le document "Summary of Misstatements", le montant total de l'ajustement est calculé de manière incorrecte.

2021-06-U2

Version : 2021 / 06 / 649

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-06-U2.

  • [SE-28045] - Une erreur se produit lorsque les auteurs tentent d'associer une bibliothèque de risques à un produit.

  • [SE-28036] - Une erreur se produit lorsque les auteurs tentent d'interroger l'API getTags pour récupérer les détails des étiquettes pour les groupes de risque.

  • [SE-28044] - Lorsque le personnel consulte des requêtes à partir de la page Activités, il ne peut pas ouvrir les documents PDF téléchargés par les clients dans les réponses aux requêtes.

  • [SE-28046] - Les utilisateurs reçoivent des messages "Prolonger la session" lorsque le fichier d'engagement est inactif pendant moins de 5 minutes.

  • [SE-28069] - Lorsque les utilisateurs ajoutent des risques d'une bibliothèque de risques à un engagement, le nom du groupe de risques associé ne s'affiche pas.

2021-06-U7

Version : 2021 / 06 / 658

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-06-U7.

  • [SE-28172] - Les utilisateurs de Caseware Cloud Connector ne peuvent pas insérer de données dans des fichiers Excel car le panneau Cloud Connector est vide.

  • [SE-28236] - Problème technique affectant le développement de formulaires personnalisés.

  • [SE-28128] - Dans les tableaux dynamiques, les cellules des lignes de titre qui contiennent des formules sont vides, au lieu d'être cachées, lorsque le groupe de comptes est développé.

  • [SE-28138] - Problème de traduction entraînant l'affichage en anglais du bouton MARK ALL RESPONDED dans les requêtes alors que la langue du contenu du produit n'est pas l'anglais.

  • [SE-28200] - Amélioration des performances liées aux types de réponse de la signature électronique simple dans les requêtes.

  • [SE-28246] - L'option Insérer un risque dans la bibliothèque est visible par les auteurs principaux dans le modèle de produit.

  • [SE-28118] - Une erreur se produit lorsque les utilisateurs tentent de définir des conditions de visibilité basées sur des types d'organisation spécifiques.

  • [SE-28212] - Dans les états financiers, lorsque l'option Utiliser un espacement de paragraphe cohérent entre les zones de contenu est sélectionnée, l'option d'aperçu avant impression ne fonctionne pas comme prévu - le texte en bas des pages est coupé.

  • [SE-28243] - Lorsque de nouveaux documents sont disponibles dans un engagement, le volet Documents n'affiche pas automatiquement les documents nouvellement ajoutés et les utilisateurs doivent rafraîchir la page.

2021-06-U9

Version : 2021 / 06 / 665

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-06-U9.

  • [SE-28367] - Les utilisateurs ne peuvent pas modifier les champs de texte d'une requête s'ils ouvrent le menu Plus d'actions () pour n'importe quel commentaire alors que les modifications apportées à un commentaire sont en cours d'enregistrement.

  • [SE-28375] - L'option Exporter les conditions de visibilité ne fonctionne pas comme prévu pour les documents dont les conditions de visibilité font référence à des valeurs provenant de formulaires personnalisés. Une erreur se produit et l'exportation n'est pas terminée.

  • Les questions suivantes s'appliquent aux missions créées à partir d'un report :

    • [SE-28421] - Une erreur se produit lorsque les utilisateurs tentent de reporter en bloc une liste de contrôle qui contient une section de conclusion avec des procédures.

    • [SE-28422] - Une erreur se produit lorsque les utilisateurs ouvrent des listes de contrôle. Le contenu de la liste de contrôle ne s'affiche pas.

    • [SE-28376] - Les utilisateurs ne peuvent pas ajouter de notes aux états financiers.

2021-06-U10

Version : 2021 / 06 / 670

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-06-U10.

  • Reporter

    • [SE-28456] - Une erreur se produit lorsque les utilisateurs tentent de reporter en bloc les documents de la mission.

    • [SE-28466] - Les notes des états financiers qui sont définies pour être exclues dans le fichier de l'année précédente sont visibles dans le fichier de l'année en cours jusqu'à ce que l'utilisateur rafraîchisse la page.

  • [SE-28523] - Une erreur se produit lorsque les utilisateurs tentent d'approuver une liste de contrôle si les procédures de la liste de contrôle avec un type de réponse Date n'ont pas de date valide sélectionnée.

  • [SE-28586] - Une erreur se produit lorsque les auteurs tentent de définir des conditions de visibilité basées sur le type d'organisation.
