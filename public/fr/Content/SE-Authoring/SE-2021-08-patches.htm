SE 2021-08 rustines

Voici un résumé des correctifs qui ont été déployés pour SE 2021-08.

2021-08-U1

Version : 2021 / 08 / 681

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-08-U1.

  • États financiers

    • [SE-29088] - Lorsque l'arrondi est appliqué à un compte dans un tableau dynamique, les autres tableaux dynamiques qui incluent le compte ne sont pas mis à jour avec la valeur arrondie.

    • [SE-29229] - Amélioration des performances.

    • [SE-29163] - Dans les états financiers, les zéros ne s'affichent pas comme des tirets lorsque l'option Afficher les zéros comme des tirets est sélectionnée dans les paramètres du produit.

    • [SE-29113] - Les sections d'orientation ne s'affichent pas dans les notes aux états financiers.

    • [SE-29119] - Lorsque les utilisateurs définissent des associations de notes pour les notes de niveau parent, l'association ne s'applique pas aux sous-notes correspondantes.

  • [SE-29284] - Lorsque les utilisateurs ouvrent un document PDF, les annotations ne s'affichent pas et l'option d'ajout d'annotations n'est pas disponible.

  • [SE-28920] - Dans les produits qui supportent plusieurs langues, lorsque les auteurs de cabinet marquent une langue comme inactive dans le modèle de cabinet, la langue n'est plus visible pour l'auteur de cabinet dans la liste des langues de contenu dans les paramètres du produit.

  • [SE-29161] |[SE-29329] - Les procédures de listes de contrôle dont le type de réponse est une information sur l'entité et qui sont liées au champ du type d' organisation affichent "Non attribué" comme réponse si le type d'organisation est "Société à responsabilité limitée (SARL)".

  • [SE-29226] - Les documents externes sont supprimés des missions si un utilisateur tente de mettre à jour les propriétés du document en même temps qu'un autre utilisateur remplace le document.

  • [SE-29231] - Lorsque les utilisateurs exportent des fichiers XBRL, les valeurs des tables de mouvement IFRS ne sont pas incluses dans l'exportation, à moins que l'utilisateur n'ait ouvert les états financiers lorsqu'il effectue l'exportation.

  • [SE-29244] - Les utilisateurs ne peuvent pas utiliser la barre de défilement lors de l'affichage de longs commentaires dans les requêtes.

  • [SE-29297] - Dans les produits qui supportent plusieurs langues, les noms de documents s'affichent comme "Sans titre" dans le cas d'utilisation suivant :

    1. Les utilisateurs changent la langue du contenu pour une langue autre que celle par défaut et téléchargent un fichier.

    2. Les utilisateurs passent ensuite à une autre langue de contenu.

  • [SE-29042] - Les modifications apportées à la section conclusion des listes de contrôle ne sont pas prises en compte lorsque les utilisateurs reportent une mission.

  • [SE-29050] - Dans les tableaux dynamiques, une erreur se produit lorsque les utilisateurs saisissent des valeurs non numériques dans des colonnes de type numérique. Le tableau dynamique ne peut pas être sauvegardé.

  • [SE-29084] - Les bordures des tableaux ne sont pas appliquées aux tableaux dynamiques dans les feuilles de calcul.

  • [SE-29218] - Lorsque les utilisateurs impriment des listes de contrôle, le PDF généré ne s'affiche pas comme prévu. Les signatures des procédures dans la section conclusion ne sont pas alignées correctement.

  • [SE-29313] - Dans les produits qui supportent plusieurs langues, lorsque les utilisateurs changent la langue du contenu pour une langue autre que celle par défaut, les options de réponse de la liste de sélection dans les listes de contrôle ne s'affichent pas.

  • [SE-29127] |[SE-29131] |[SE-29137] |[SE-29108] - Problèmes techniques identifiés après examen des journaux de production.

2021-08-U2

Version : 2021 / 08 / 686

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-08-U2.

  • [SE-29365] - Dans les notes aux états financiers, l'option Insérer à partir de la bibliothèque n'est pas disponible si les utilisateurs ont déjà ajouté toutes les notes cachées de niveau parent, même si des notes cachées de niveau inférieur sont disponibles dans la bibliothèque.

  • [SE-29484] - Dans les missions créées à partir d'un report, les termes du glossaire de la mission qui ont été édités par les utilisateurs finaux dans le fichier de l'année précédente ne sont pas reportés comme prévu.

  • [SE-29376] - Lorsque les utilisateurs importent des données depuis BGL Simple Fund 360, l'écran d'importation n'affiche pas l'entité BGL à partir de laquelle les données seront importées.

  • [SE-29334] - Lorsque les utilisateurs importent des données de balance de vérification, les données s'affichent comme étant déséquilibrées si l'un des soldes de compte comprend une notation exponentielle.

  • [SE-29449] - Si le contenu traduit n'est pas disponible pour la langue de contenu sélectionnée, le contenu ne s'affiche pas du tout au lieu de s'afficher dans la langue par défaut.

2021-08-U5

Version : 2021 / 08 / 690

Un problème a été corrigé lors du déploiement de la SE 2021-08-U5.

[SE-29885] - Amélioration des performances liées au développement de formulaires personnalisés.

2021-08-U9

Version : 2021 / 08 / 700

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-08-U9.

  • Tableaux dynamiques

    • [SE-29813] - Les étiquettes de diagnostic s'affichent comme partiellement cachées et ne peuvent pas être modifiées.

    • [SE-29712] - L'ajout d'annotations à une ligne de tableau dynamique ne fonctionne pas comme prévu. Lorsque les utilisateurs actualisent ou quittent le site, l'annotation n'est pas sauvegardée.

    • [SE-29894] - Les décimales ne s'affichent pas dans les colonnes de saisie.

    • [SE-29728] - Le menu Modifier la taille de la police ne s'affiche pas comme prévu. Les utilisateurs ne peuvent pas accéder au menu pour sélectionner des options.

    • [SE-29852] - Les liens de données (fx) ne fonctionnent pas comme prévu lorsque les auteurs copient des tables dynamiques.

  • [SE-29826] - Les clients ne peuvent pas télécharger les fichiers téléchargés dans les documents de requête.

  • [SE-29918] - Les données fiscales de l'année précédente ne s'affichent pas dans les formulaires personnalisés.

  • [SE-29972] - Une erreur se produit lorsque les utilisateurs tentent d'exporter des fichiers iXBRL. Les fichiers ne sont pas générés.

  • [SE-23001] - Les liens de données (fx) avec des conditions de balance de vérification ne fonctionnent pas comme prévu si le paramètre de groupe financier pour la condition est un groupe de revenu net.

  • [SE-29317] - Dans les missions de consolidation, la page Données s'affiche incorrectement comme étant déséquilibrée.

  • [SE-29474] - Dans les missions de consolidation, la page Données n'affiche pas les noms corrects pour les entités subsidiaires.

  • [SE-29637] - Lorsque les utilisateurs créent des problèmes, il y a un délai entre la saisie de l'utilisateur et l'affichage du texte saisi dans la zone de texte de la description.

  • [SE-29769] - Dans les graphiques d'analyse, les infobulles des soldes des comptes ne s'affichent pas comme prévu. Un signe égal (=) s'affiche à la fin de la valeur du solde du compte dans l'info-bulle.

  • [SE-29858] - Lorsque les utilisateurs ouvrent un document à partir de la page Documents, c'est le mauvais document qui s'ouvre s'ils ont choisi d'afficher les éléments cachés de la page.

  • [SE-30051] - Dans les produits qui prennent en charge plusieurs langues, lorsque les utilisateurs changent la langue du contenu de l'engagement pour une langue autre que celle par défaut, le contenu continue de s'afficher dans la langue par défaut.

  • [SE-28565] - Une erreur se produit lorsque les utilisateurs tentent de reporter un engagement pour le cas d'utilisation suivant :

    • L'auteur principal a supprimé une formule de données (fx) du modèle de l'année précédente.

    • L'auteur principal a ensuite utilisé la fonction d'annulation (CTRL + Z) pour annuler la suppression.

  • [SE-29529] - Lorsque les utilisateurs exportent des documents XBRL, les faits XBRL étiquetés avec des balises faux-y ne sont pas inclus dans le fichier généré.

  • [SE-29861] - Lorsque les utilisateurs impriment les états financiers, le texte dont l'alignement est justifié ne s'affiche pas comme prévu.

  • [SE-29926] - Dans les états financiers, lorsque les utilisateurs sélectionnent une zone dans le plan du document, ils ne sont pas dirigés vers le début de la zone.

2021-08-U10

Version : 2021 / 08 / 701

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-08-U10.

  • [SE-30237] - Un problème technique qui provoque des erreurs lors du report d'une mission si des rôles d'approbation identiques ont été ajoutés à la fois dans le fichier de mission de l'année précédente et dans le modèle de cabinet de l'année en cours.

  • [SE-30090] - L'onglet Risque dans les listes de contrôle ne fonctionne pas comme prévu. Les noms de certaines assertions ne sont pas affichés.

2021-08-U11

Version : 2021 / 08 / 707

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2021-08-U11.

  • [SE-30443] - Lorsque les utilisateurs impriment des documents, les tableaux dynamiques qui s'étendent sur plus d'une page ne s'affichent pas comme prévu dans le PDF généré. Après les sauts de page, les en-têtes de tableau ne sont pas formatés correctement.

  • [SE-30323] - Lorsque les utilisateurs impriment les états financiers, les notes ne s'affichent pas comme prévu dans le PDF généré si les notes au niveau de la société mère ont été supprimées par l'auteur de la société dans le modèle de société. Les notes de sous-niveau pour les notes parentales supprimées sont incluses dans le PDF.
