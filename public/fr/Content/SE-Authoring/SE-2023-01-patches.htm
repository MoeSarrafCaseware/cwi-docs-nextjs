Voici un résumé des correctifs qui ont été déployés pour SE 2023-01. Les correctifs sont énumérés dans l'ordre, le plus récent en premier.

Remarque : Note :Pour vérifier la version actuelle de votre produit, sélectionnez Information () | À propos dans votre modèle. La boîte de dialogue À propos affiche la version serveur et la version client de votre produit. Vous pouvez vérifier la version du serveur pour déterminer si vous disposez du dernier correctif serveur et la version du client pour déterminer si vous disposez du dernier correctif client. Les versions du serveur et du client sont affichées dans le format suivant :

  • Version du serveur : 2022-02 Build 123 Release
  • Version du client : release-se-2022-06-U1
