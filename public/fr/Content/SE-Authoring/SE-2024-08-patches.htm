SE 2024-08 correctifs

Voici un résumé des correctifs qui ont été déployés pour SE 2024-08. Les correctifs sont classés du plus récent au plus ancien.

Remarque : Note :Pour vérifier la version actuelle de votre produit, sélectionnez Information () | À propos dans votre modèle. La boîte de dialogue À propos affiche la version serveur et la version client de votre produit. Vous pouvez vérifier la version du serveur pour déterminer si vous disposez du dernier correctif serveur et la version du client pour déterminer si vous disposez du dernier correctif client. Les versions du serveur et du client sont affichées dans le format suivant :

  • Version du serveur : 2022-02 Build 123 Release

  • Version du client : release-se-2022-06-U1

2024-08-C3

Version : 2024-08 Build: release-se-2024-08-2024-11-05-11690422168

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2024-08-C3.

  • [SE-50310] - Problèmes d'affichage avec les bordures dynamiques des titres de tableaux.

  • [SE-50499] - Lorsque les utilisateurs suppriment une ligne dans un ajustement, la colonne Dimension pour les lignes d'ajustement ne s'affiche plus. Les utilisateurs doivent fermer la boîte de dialogue et la rouvrir pour afficher la colonne.

  • [SE-50610] - Dans les ajustements consolidés, les soldes des comptes s'affichent de manière incorrecte si :

    • Dans la mission mère, un compte d'une mission filiale est non affecté puis réaffecté.

    • Le compte fait l'objet d'un ajustement dans l'engagement subsidiaire.

  • [SE-50785] - Dans les missions de consolidation, lorsque des colonnes sont ajoutées à la page Données pour les entités subsidiaires, ces colonnes ne respectent pas les paramètres pour les nombres négatifs définis dans les paramètres de la mission.

  • [SE-50788] - Les auteurs principaux ne peuvent pas ajouter de sous-groupes aux groupes financiers parents dans le modèle de produit.

  • [SE-50798] - Une erreur se produit lorsque les utilisateurs tentent de remplacer une liste à puces par une liste numérotée dans une section de texte si la liste contient un espace réservé pour la date. La page se bloque.

  • [SE-50828]- L'indicateur de chargement ne s'affiche pas lorsque les utilisateurs impriment les états financiers au format PDF.

  • [SE-50857] - Dans les tableaux dynamiques, lorsque les utilisateurs ajoutent des colonnes de saisie, les lignes de balance de vérification affichent des soldes nuls lorsqu'elles sont réduites.

  • [SE-50872] - Lorsque les utilisateurs réorganisent les rôles de signature, les signatures existantes ne s'affichent pas comme prévu sur la page Documents jusqu'à ce que les utilisateurs rafraîchissent la page.

  • [SE-50954] - En raison de la visibilité des composants, certains groupes financiers ne s'affichent pas pour les auteurs dans les menus déroulants.
