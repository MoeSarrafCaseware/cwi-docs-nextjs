SE 2024-09 correctifs

Voici un résumé des correctifs qui ont été déployés pour SE 2024-09. Les correctifs sont classés du plus récent au plus ancien.

Remarque : Note :Pour vérifier la version actuelle de votre produit, sélectionnez Information () | À propos dans votre modèle. La boîte de dialogue À propos affiche la version serveur et la version client de votre produit. Vous pouvez vérifier la version du serveur pour déterminer si vous disposez du dernier correctif serveur et la version du client pour déterminer si vous disposez du dernier correctif client. Les versions du serveur et du client sont affichées dans le format suivant :

  • Version du serveur : 2022-02 Build 123 Release

  • Version du client : release-se-2022-06-U1

2024-09-S4

Version : 2024-09 Build: 12433555974

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2024-09-S4.

  • [SE-51808] - Les boutons Migrer les personnalisations du modèle précédent et Mettre à jour de la page Paramètres - Gérer le modèle ne s'affichent pas comme prévu.

2024-09-C2

Version : 2024-09 Build: 2024-09-C2

Voici la liste des problèmes qui ont été corrigés avec le déploiement de SE 2024-09-C2.

  • [SE-49928] - Les listes de tables des matières ne sont pas uniformément indentées au-delà du premier niveau.

  • [SE-50341] - L'option Insérer à partir de la bibliothèque pour les notes aux états financiers est visible lorsque toutes les notes sont affectées à des composants cachés.

  • [SE-51075] - L'horodatage est parfois exclu des commentaires des requêtes.

  • [SE-51294][SE-51542] - Lors de la comparaison des versions des états financiers, les modifications des valeurs des cellules sont toujours affichées sur une seule ligne, et les colonnes de texte et de tableau réarrangées ne sont pas affichées de manière intuitive, ce qui rend la compréhension difficile dans certains cas.

  • [SE-51412] - Si une version d'état financier ne parvient pas à être créée, elle est tout de même ajoutée au panneau de contrôle des versions et ne peut être ni consultée ni supprimée.

  • [SE-51538] - La tentative de création d'une version d'état financier provoque occasionnellement un plantage du navigateur Internet.

  • [SE-51672] - Le bouton Nouvelle version est disponible lors d'une comparaison d'états financiers.

  • [SE-51689] - Les lignes d'espacement ne peuvent pas être ajoutées au-dessus du total dans un tableau dynamique ; la ligne est ajoutée sous le titre à la place.

  • [SE-51720] - Après avoir reporté un document contenant un tableau dynamique, certaines modifications suggérées ne sont pas visibles tant que la modification n'est pas acceptée.

  • [SE-51748] - Le fond gris utilisé pour identifier les formules de référence des notes n'est pas visible.

  • [SE-51843] - Si un auteur principal signe une liste de contrôle dans un modèle, l'engagement devient invalide.

  • [SE-51878] - Certaines réponses aux procédures ne sont pas reportées dans le document de la liste de contrôle.

  • [SE-51966] - Les vues sauvegardées de la page Groupes financiers ne conservent pas les colonnes sélectionnées.

  • [SE-52039] - Les états financiers se chargent lentement et sont parfois interrompus, ce qui entraîne l'affichage d'un message d'erreur "This page is having a problem" dans le navigateur Internet.

  • [SE-52125] - Les documents SE Builder ne fonctionnent pas correctement après avoir été reportés lors d'un processus de report en bloc.

Pour une liste des mises à jour XBRL, voir What's new in XBRL - December 2024.
