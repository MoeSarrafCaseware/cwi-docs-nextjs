Configurer l'automapping (page de données)

Vous pouvez configurer le mappage automatique de manière à ce que les comptes soient automatiquement affectés aux groupes les mieux adaptés lorsque les utilisateurs importent les données de la balance de vérification du client.

Pour mettre en place l'automapping :

  1. Activez la fonctionnalité automap dans votre produit (voir Liste des fonctionnalités du produit (API)).

  2. La liste des classifications universelles permet de rechercher et de déterminer la classification de groupe appropriée pour le progiciel comptable.

  3. Pour chaque groupe :

    1. Select Edit () to open the Group Properties dialog.

    2. Saisissez la classification du groupe dans le champ Classification du groupe .

    3. Sélectionnez Enregistrer.

    4. Conseil : Vous pouvez également ajouter les classifications de groupe dans le fichier Excel ou CSV d'importation de groupe et l'importer.
