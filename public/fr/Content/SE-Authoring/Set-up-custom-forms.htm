Créer des formulaires personnalisés

Outre les types de documents intégrés, vous pouvez développer des formulaires personnalisés dans votre produit SE à l'aide de SE Builder. Pour en savoir plus sur le développement de formulaires personnalisés, voir Démarrer avec SE Builder.

Vous pouvez rediriger les utilisateurs vers les formulaires personnalisés de votre produit plutôt que vers les pages par défaut pour les fonctionnalités suivantes :

  • Risques

  • Contrôles

  • Matérialité

  • Résumé des anomalies

Pour commencer, sélectionnez l'icône Information () dans le menu supérieur, puis choisissez Paramètres. Sélectionnez Fonctionnalités |Formulaires personnalisés dans la boîte de dialogue Paramètres du produit qui s'ouvre.

Pour remplacer une page par défaut par un formulaire personnalisé, ajoutez le formulaire à la page Documents , puis sélectionnez-le dans le menu déroulant de la fonctionnalité.

Remarque : La section Risques n'apparaîtra pas dans les paramètres des formulaires personnalisés si elle n'est pas activée dans votre produit.

Vous pouvez également ajouter vos formulaires personnalisés à la boîte de dialogue Ajouter.

Pour ajouter un formulaire personnalisé à la boîte de dialogue Ajouter :

  1. Sélectionnez Ajouter un nouvel élément.

  2. Saisissez un nom pour le formulaire. Ce nom sera affiché aux utilisateurs dans la boîte de dialogue Ajouter.

  3. Sélectionnez le formulaire que vous souhaitez ajouter dans la liste déroulante.

  4. Sélectionnez Enregistrer.
