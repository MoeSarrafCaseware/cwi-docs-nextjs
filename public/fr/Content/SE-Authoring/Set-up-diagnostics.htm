Mise en place des diagnostics

Vous pouvez mettre en place des diagnostics dans les états financiers pour aider les utilisateurs à identifier et à localiser les divergences entre les soldes des tableaux des états financiers.

Pour configurer les diagnostics dans les états financiers, vous devez :

  1. Créez les étiquettes de diagnostic.
  2. Étiqueter les groupes et les comptes dont vous voulez que l'application compare les soldes entre eux.

Lorsque vous étiquetez des groupes ou des comptes, l'application compare automatiquement les soldes des lignes ayant la même étiquette.

La comparaison se fait en temps réel. Si les lignes étiquetées proviennent des données de la balance générale, les résultats de la comparaison s'affichent dès que les utilisateurs importent la balance générale.

Pour en savoir plus sur la manière dont les utilisateurs affichent les diagnostics, voir Afficher les écarts de solde.

To label rows in financial statements tables:

  1. Select the dynamic table where you want to label accounts or groups.

    2. Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez Autres actions () | Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Sélectionnez la ligne que vous souhaitez modifier.

    4. The row can be an account or a group.

  4. Allez dans le champ Étiquette de diagnostic en haut de la page. Vous pouvez créer une étiquette ou sélectionner une étiquette existante pour l'affecter à la ligne sélectionnée.

    • Pour créer une étiquette, saisissez le nom souhaité dans le champ. L'étiquette est créée et attribuée à la ligne sélectionnée lorsque vous appuyez sur Entrée ou sur la barre d'espacement.

      • Remarque : Les étiquettes de diagnostic ne peuvent pas comporter d'espaces.

    • Pour trouver un label existant, commencez à taper le mot-clé dans le champ pour obtenir une liste de résultats correspondants. Choose a label from the list to assign it to the selected row.

    • Pour sélectionner une étiquette existante, sélectionnez la liste déroulante et choisissez une étiquette pour l'attribuer à la ligne sélectionnée.

      • L'étiquette attribuée s'affiche en haut à la place du champ Étiquette de diagnostic .

      Conseil : Pour supprimer l'étiquette attribuée, sélectionnez Supprimer () à côté de l'étiquette. Notez que s'il s'agit d'une nouvelle étiquette (c'est-à-dire que vous ne l'avez pas sélectionnée dans la liste déroulante), ce bouton supprime également l'étiquette.

  5. Désélectionner la ligne.

  6. Sélectionnez SAVE ou SAVE & CLOSE.

Remarques importantes concernant l'établissement des diagnostics

Notez les points suivants lors de la mise en place des diagnostics,

  • L'application ne compare que les lignes ayant la même étiquette. Veillez à attribuer le même étiquette aux lignes que vous souhaitez que l'application compare entre elles.
  • L'application compare les valeurs absolues des lignes concernées. Les différences de signe ne sont pas signalées comme des écarts de solde.
  • Si vous étiquetez un groupe, l'application compare l'étiquette du groupe avec l'étiquette du groupe solde total du groupe à d'autres lignes ayant la même étiquette. Il ne compare pas le solde de chaque compte du groupe aux lignes ayant le même libellé.
  • Vous pouvez attribuer une étiquette à une ligne.
