Mise en place du rapport de contrôle

Remarque : Cette rubrique ne s'applique pas aux applications en nuage pour lesquelles le nouveau module de risque est activé.

Vous pouvez activer le rapport de contrôle pour permettre aux utilisateurs d'ajouter des contrôles qu'ils ont définis pour atténuer les risques identifiés tout au long du déroulement de la mission. Vous pouvez activer un rapport de contrôle pour prendre en charge des audits importants et complexes (si votre produit le permet).

Pour activer le rapport de contrôle dans votre produit, spécifiez la propriété com.caseware.ca.general.control-report dans la propriété form du fichier product.json. Pour en savoir plus, voir Activer ou désactiver des fonctionnalités dans votre produit.

Une fois que vous avez activé le rapport dans votre produit, vous pouvez l'ajouter à votre projet et le configurer.

Pour ajouter une liste de contrôle, sélectionnez Ajouter ( ) dans la page des documents, puis choisissez Liste de contrôle.

Pour définir les propriétés de l'engagement :

  1. Pour ajouter des informations d'orientation :

    2. Sélectionnez le champ de texte pour commencer à ajouter du contenu. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour effectuer les modifications suivantes :

  1. Si vous souhaitez ajouter des contrôles par défaut dans le rapport, sélectionnez +Contrôle.

    1. Saisissez un nom et une description pour le dossier.

    2. Sélectionnez le cycle et la zone associés au contrôle.

  2. Sélectionnez un contrôle pour l'ouvrir.

  3. Sélectionnez Editer ( ) pour modifier les détails du contrôle si nécessaire.

    1. Associez-le à un groupe financier sous GROUPES FINANCIERS.

    2. Add a document reference at REFERENCES.

    3. Associez-le à un risque en le sélectionnant dans la liste déroulante RISQUES.

    4. Associez-le à un cycle ou à un domaine en le sélectionnant dans les listes déroulantes CYCLES ou DOMAINES.

    5. Une fois que vous avez terminé, sélectionnez Enregistrer.
