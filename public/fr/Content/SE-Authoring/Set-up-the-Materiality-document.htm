Mise en place du document de matérialité

Vous pouvez activer le document de matérialité pour permettre aux utilisateurs de définir la matérialité pour les domaines financiers.

Pour activer le rapport sur la matérialité dans votre produit, spécifiez la propriété com.caseware.ca.general.enhanced-materiality dans le formulaire du fichier product.json. Pour en savoir plus, voir Activer ou désactiver des fonctionnalités dans votre produit.

Une fois que vous avez activé le document dans votre produit, vous pouvez l'ajouter à votre brouillon et le configurer.

Pour ajouter une liste de contrôle, sélectionnez Ajouter ( ) dans la page des documents, puis choisissez Liste de contrôle.
