Établir le rapport d'évaluation du risque d'anomalies significatives (RMM)

Remarque : Cette rubrique ne s'applique pas aux applications en nuage pour lesquelles le nouveau module de risque est activé.

Vous pouvez activer le rapport d'évaluation RMM pour permettre aux utilisateurs d'évaluer la RMM par domaine financier. Par exemple, ils examineront la trésorerie et décideront s'il existe ou non un risque d'anomalies significatives dans la trésorerie.

Pour activer le rapport d'évaluation des risques dans votre produit :

  • Attribuer la valeur trueà la propriété riskAssessment

  • Spécifiez la propriété com.caseware.ca.general.risk-assessment dans la propriété form du fichier product.json. Pour en savoir plus, voir Activer ou désactiver des fonctionnalités dans votre produit.

Une fois que vous avez activé le rapport dans votre produit, vous pouvez l'ajouter à votre projet et le configurer.

Pour ajouter le rapport d'évaluation des risques, sélectionnez Ajouter ( ) dans la page des documents, puis choisissez Rapport d'évaluation des risques.

Pour établir le rapport d'évaluation des risques :

  1. Dans la liste déroulante Filtres , en haut à gauche, sélectionnez le filtre que vous souhaitez appliquer au document.

  2. Sous Paramètres ( ) , modifiez les paramètres par défaut si nécessaire.

    3. Les options suivantes sont disponibles :

Paramètre

Description

Utiliser des données non consolidées

Utilisation ou non de données consolidées

Montrer l'importance relative

Indiquer ou non l'importance relative (les informations proviennent du document relatif à l'importance relative)

Conclusion

Whether or not to include the Conclusion section at the end of the report

Colonne supplémentaire

Ajout ou non d'une colonne supplémentaire à côté de la colonne Raison pour chaque domaine financier

Autoriser les évaluations dans le cadre de cycles ou de domaines

Autoriser ou non les évaluations par cycle ou par domaine.

  1. Dans la liste déroulante Grouper par , sélectionnez si vous souhaitez que les informations contenues dans le rapport soient groupées ou non.

    2. Si vous souhaitez regrouper les informations, vous pouvez choisir de les regrouper par domaines ou par cycles.

  2. Pour ajouter des informations d'orientation :

    3. Sélectionnez le champ de texte pour commencer à ajouter du contenu. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour effectuer les modifications suivantes :

  1. Renommez les en-têtes de colonne pour personnaliser les termes si nécessaire.

    2. Vous pouvez renommer les colonnes intitulées RMM, Motif et Informations complémentaires.

  2. Ajouter une conclusion.

    3. Sélectionnez le champ de texte pour commencer à ajouter du contenu. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour mettre en forme et modifier le texte.
