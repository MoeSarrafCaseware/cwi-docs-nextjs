Configurer le rapport sur les risques

Remarque : Cette rubrique ne s'applique pas aux applications SE pour lesquelles le nouveau module de risque est activé.

Le rapport sur les risques comprend les risques identifiés que les utilisateurs (par exemple, les auditeurs) ont ajoutés tout au long du déroulement de la mission.

Pour activer le rapport de risque dans votre produit, spécifiez la propriété com.caseware.ca.general.risk-report dans la propriété form du fichier product.json. Pour en savoir plus, voir Activer ou désactiver des fonctionnalités dans votre produit.

Une fois que vous avez activé le rapport dans votre produit, vous pouvez l'ajouter à votre projet et le configurer.

Pour ajouter une liste de contrôle, sélectionnez Ajouter ( ) dans la page des documents, puis choisissez Liste de contrôle.

Pour définir les propriétés de l'engagement :

  1. Sélectionnez les attributs de risque à afficher sur la page.

    2. Utilisez la liste déroulante Attributs du risque en haut à gauche.

  2. Pour ajouter des informations d'orientation :

Sélectionnez le champ de texte pour commencer à ajouter du contenu. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour effectuer les modifications suivantes :

  1. Sélectionnez Editer ( ) pour modifier les paramètres des risques existants.

    1. Modifier le nom et la description.

    2. Sélectionnez les paramètres de type () pour marquer les types de risque que vous souhaitez voir disponibles.

  2. Une fois que vous avez terminé, sélectionnez Enregistrer ou Enregistrer les paramètres de risque.
