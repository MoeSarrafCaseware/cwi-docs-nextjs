Mise en place du document "Résumé des anomalies" (Summary of Misstatements)

Vous pouvez activer le document de synthèse des inexactitudes qui résume l'impact des inexactitudes (omissions) d'informations dans les domaines des états financiers et présente une conclusion (si les inexactitudes sont considérées comme significatives ou non significatives).

Pour activer le rapport Résumé des inexactitudes dans votre produit, spécifiez la propriété com.caseware.ca.general.misstatements-summary dans la propriété form du fichier product.json. Pour en savoir plus, voir Activer ou désactiver des fonctionnalités dans votre produit.

Une fois que vous avez activé le document dans votre produit, vous pouvez l'ajouter à votre brouillon et le configurer.

Pour ajouter le résumé des anomalies, sélectionnez Ajouter ( ) sur la page des documents, puis choisissez Résumé des anomalies.

Mettre en place le document "Résumé des anomalies" :

  1. Pour ajouter des informations d'orientation :

    2. Sélectionnez le champ de texte pour commencer à ajouter du contenu. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils pour effectuer les modifications suivantes :

  1. Utilisez la case à cocher Applicable () pour spécifier quelles sections sont incluses ou exclues du document.

  2. Dans les écritures au journal réelles et proposées, sélectionnez l'option Spécifier les groupes de classification pour spécifier les groupes de balance de vérification de la classification pour les écritures au journal réelles et proposées.

    Remarque : L'option de classification des groupes d'utilisation n'est actuellement pas fonctionnelle.

  3. Sélectionnez +Nouveau groupe, puis sélectionnez le champ de texte pour afficher la liste des groupes de balances de vérification. Sélectionnez le groupe souhaité pour les actifs, les passifs, les capitaux propres, les recettes et les dépenses.

  4. Renommez les éléments sous Données clés pour personnaliser les termes si nécessaire.

    5. Une fois que vous avez renommé un élément, l'icône Revert () est disponible pour vous permettre de revenir au nom d'origine.

  5. Ajoutez les champs suivants :

    • Évaluation qualitative

    • Résumé des informations omises (description de l'omission ou de l'inexactitude, autre domaine concerné, discussion avec le client, raison de l'omission)

    • Évaluation

    • Conclusion

    • Approuver

Sélectionnez le champ de texte pour commencer à ajouter du contenu. La barre d'outils de formatage s'affiche. Utilisez la barre d'outils décrite à l'étape 1 pour mettre en forme et modifier le texte.
