Configurer la bibliothèque de notes

La bibliothèque de notes fournit un ensemble de notes qui permet aux utilisateurs de trouver et d'ajouter facilement des notes dans le document des états financiers. Notes are automatically organized in the library by Categories (note and note headings).

La bibliothèque comporte également un champ de recherche, où les utilisateurs peuvent saisir un mot clé pour filtrer les notes.

Vous pouvez mettre en place la bibliothèque de notes dans les états financiers afin de fournir aux utilisateurs finaux un ensemble de notes et d'informations prédéfinies dans votre produit.
