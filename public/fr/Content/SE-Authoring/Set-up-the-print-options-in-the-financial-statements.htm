Configurer les options d'impression dans les états financiers

Le bouton Imprimer () convertit le document au format PDF pour que les utilisateurs puissent l'imprimer ou le sauvegarder.

Pour configurer les options d'impression :

  1. Accédez à la carte du document, puis sélectionnez Paramètres () | Paramètres d'impression du document.

    2. Si vous souhaitez définir des paramètres différents pour les notes, accédez à la carte du document et sélectionnez Paramètres () | Paramètres d'impression des notes

    Si vous souhaitez définir des paramètres différents pour une zone, accédez à la zone souhaitée dans le document et sélectionnez Autres actions () | Modifier les paramètres d'impression.

    Si vous voulez avoir des paramètres différents pour une page spécifique, allez au saut de page de cette page et sélectionnez Plus d'actions () | Paramètres d'impression.

  2. Spécifiez les paramètres préférés pour l' en-tête, la taille de la page et les marges.

  3. Sous Tableaux dynamiques, sélectionnez l'option Afficher les en-têtes de colonne sur les pages suivantes si vous souhaitez que les tableaux dynamiques répètent les en-têtes de colonne sur les pages suivantes.

  4. Sous Conserver ensemble, sélectionnez l'option Conserver les en-têtes de notes avec les paragraphes et conserver les paragraphes ensemble si vous voulez vous assurer que les en-têtes de notes ne sont pas imprimés sur des pages séparées du contenu de la note et que les paragraphes ne sont pas répartis sur plusieurs pages.

    Notez que les paragraphes de plus d'une page seront toujours scindés.

  5. Sélectionnez Appliquer un fil igrane pour appliquer un filigrane dans le document.

    Vous pouvez saisir le contenu du filigrane dans le champ de texte associé.

  6. Sélectionnez Paysage pour modifier l'orientation de la page.

  7. La valeur par défaut est Portrait.

  8. Décochez la case Compter dans la numérotation des pages si vous souhaitez que la numérotation des pages commence après la page de la table des matières .

  9. Sous Numérotation des pages, sélectionnez Commencer la page à et entrez un numéro de page dans le champ de texte si vous souhaitez commencer la numérotation des pages à un numéro de page personnalisé.

    Par défaut, la numérotation des pages commence à 1 pour les états financiers et les numéros de page d'un domaine individuel sont repris à partir du domaine précédent. Vous pouvez relancer la numérotation des pages plusieurs fois dans les états financiers en modifiant les paramètres d'impression pour des zones individuelles.

  10. Dans la section Paramètres d'impression des notes:

    1. Dans les Paramètres d'impression des notes, sélectionnez Masquer les en-têtes de sous-niveau avec une seule note si vous souhaitez masquer automatiquement les en-têtes de sous-niveau des notes lorsqu'il n'y a pas de notes suivantes dans le groupe.

      Par exemple, si la note 1 comprend une sous-note 1.a et qu'il n'y a pas de notes 1.b, 1.c, etc., l'en-tête de la sous-note 1.a ne s'affiche pas dans la sortie PDF. Seul le contenu de cette note s'affiche.

    2. Sélectionnez Activer le texte d'en-tête continu si vous souhaitez saisir un suffixe personnalisé pour les en-têtes des notes qui s'étendent sur plusieurs pages.

      Par exemple, si vous saisissez le suffixe "suite", une note dont l'en-tête est "Principales méthodes comptables" s'affichera comme "Principales méthodes comptables suite" sur les pages suivantes.

      Par exemple, si vous saisissez le suffixe "suite", une note dont l'en-tête est "Principales méthodes comptables" s'affichera comme "Principales méthodes comptables suite" sur les pages suivantes.
