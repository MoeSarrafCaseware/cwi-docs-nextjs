Spécifier l'option d'affichage des comptes et des groupes

Vous pouvez prédéfinir l'affichage des groupes et des comptes dans chaque tableau des états financiers. Par exemple, vous pouvez faire en sorte qu'un tableau affiche par défaut les groupes réduits.

Pour définir l'option d'affichage des groupes et des comptes dans un tableau :

  1. Sélectionnez le tableau souhaité.

    2. Les icônes des options du tableau s'affichent à droite.

  2. Sélectionnez Autres actions () | Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Sélectionnez Paramètres ().

  4. Si votre produit possède plusieurs structures de regroupement financier, sélectionnez la structure de regroupement souhaitée dans la liste déroulante. Vous ne pouvez sélectionner qu'une seule structure de regroupement par table.

  5. Sélectionnez l'option d'affichage souhaitée pour les comptes parmi les options suivantes :

    • Exclu - Afficher uniquement les groupes dans le tableau.

    • Inclus (réduit) - Affiche les comptes et les groupes dans le tableau, les groupes étant réduits par défaut.

    • Inclus (développé) - Affiche les comptes et les groupes dans le tableau, les groupes étant développés par défaut.

  6. Sélectionnez SAVE ou SAVE & CLOSE.
