Pour modifier l'option d'affichage d'une ligne :

Une fois que vous avez ajouté une ligne de balance générale, de saisie, de dimension ou de groupe personnalisé, vous pouvez spécifier l'option de symbole de devise souhaitée pour la ligne dans les états financiers et dans la copie d'impression du document PDF.

En fonction du type de ligne, vous pouvez choisir entre trois ou deux options d'affichage :

Pour modifier l'option d'affichage d'une ligne :

  1. Sélectionnez le tableau.

  2. Sélectionnez Autres actions (), puis choisissez Modifier le tableau dynamique.

    3. Le tableau s'ouvre en mode édition.

  3. Sélectionnez la ligne souhaitée à l'aide des cases à cocher situées sur le côté gauche.

  4. Sélectionnez l'icône de l'œil ().

    5. Une liste déroulante s'ouvre et présente les options d'affichage disponibles pour la ligne sélectionnée.

    Appliquer le comportement par défaut est sélectionné par défaut.

    Remarque : Si la ligne est une ligne de saisie ou une ligne de groupe, l'option Toujours masquer cette ligne est disponible dans la liste - sinon, seules les options Appliquer le comportement par défaut et Toujours afficher cette ligne sont disponibles.

  5. Sélectionnez l'option d'affichage souhaitée.

    6. Conseil : Si vous ne souhaitez pas modifier l'option d'affichage actuelle, décochez la case de la ligne.

  6. Sélectionnez SAVE ou SAVE&CLOSE.

Notez que les lignes qui ne seront pas incluses dans l'impression du document s'affichent en gris, ce qui vous permet de les repérer facilement.
