Préciser si les données consolidées doivent être utilisées ou non

Si la consolidation des engagements est pertinente dans votre solution, vous pouvez décider d'utiliser ou non les données consolidées dans les documents relatifs aux états financiers. Par exemple, si vous devez préparer des états financiers distincts pour l'entité mère et les filiales (entités consolidées), vous choisirez de ne pas utiliser de données consolidées.

Par défaut, le document inclut les données consolidées.

Utiliser des soldes non consolidés dans les états financiers

  1. Accédez à la page Documents.

  2. Sélectionnez Settings () , puis cochez la case Use non-consolidated data (utiliser des données non consolidées ).

    3. Les données consolidées ne sont plus disponibles dans les états financiers.

Pour inclure à nouveau les données consolidées dans les états financiers, décochez la case Utiliser les données non consolidées .
