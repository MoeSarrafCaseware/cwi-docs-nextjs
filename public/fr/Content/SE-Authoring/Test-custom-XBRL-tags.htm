Tester les balises XBRL personnalisées

Remarque : Cette fonctionnalité n'est disponible que pour les distributeurs dans le cadre de tests bêta. Pour visualiser cette fonctionnalité, ouvrez un fichier d'engagement et apportez la modification suivante à votre URL : https://<nom du serveur de votre site partenaire>/<nom de l'entreprise où vous effectuez votre rédaction>/e/eng/<identifiant>/index.jsp?enable-features=customXbrlTagging;testTaxonomy

Vous pouvez tester le marquage des faits XBRL directement dans un fichier d'engagement pour vous assurer que les options disponibles pour les utilisateurs finaux lors de la création de balises personnalisées pour les faits XBRL sont correctes.

Vous pouvez également baliser les faits XBRL directement dans le modèle de produit, mais nous vous recommandons de continuer à utiliser des scripts pour baliser le contenu par défaut de votre produit. Cela garantit que les balises que vous créez sont considérées comme des balises système, ce qui optimise les performances lorsque les utilisateurs examinent les faits XBRL et exportent les documents d'instance XBRL.

Pour marquer un fait XBRL :

  1. Sélectionnez l'icône XBRL ().

  2. Dans la section XBRL REVIEW , sélectionnez la taxonomie que vous souhaitez réviser.

  3. Sélectionnez START REVIEW.

    Le document des états financiers s'ouvre en montrant la première zone qui contient des faits XBRL. Les faits marqués sont soulignés en vert et les faits non marqués sont surlignés en jaune.

  4. Sélectionnez un fait non balisé, puis sélectionnez Add XBRL Tag.

  5. Sélectionnez une vue dans la liste.

    You can also search for a specific tag using the search box.

    Lorsque vous sélectionnez un élément dans la taxonomie, les informations relatives à cet élément, y compris l'étiquette, la description et les propriétés, s'affichent dans le panneau de droite de la boîte de dialogue de recherche.

  6. Sélectionner SELECT.

  7. Dans le panneau de révision XBRL, sélectionnez les informations de contexte pour la balise parmi les options disponibles dans le menu déroulant.

    Les options de la liste déroulante proviennent des informations contenues dans votre taxonomie XBRL.
