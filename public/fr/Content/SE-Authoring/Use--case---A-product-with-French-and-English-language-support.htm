Cas d'utilisation : Un produit qui prend en charge les langues française et anglaise.

Cette rubrique décrit un cas d'utilisation pour ajouter la prise en charge de la langue française dans un modèle de produit. Vous pouvez utiliser ce cas d'utilisation comme référence lorsque vous configurez plusieurs langues dans votre modèle.

Remarque : Vous devez répéter les étapes énumérées dans ce cas d'utilisation pour chaque langue active de votre modèle.

Hypothèse :

  • Le contenu et les produits à livrer en anglais ont déjà été ajoutés et rédigés dans le modèle.

  • Vous avez ajouté le français dans le modèle et l'avez marqué comme langue active.

Prérequis : Changez la langue du contenu du modèle en français.

Étape 1 : Add the French language in the product settings

Ajoutez la langue française et effectuez d'autres modifications si nécessaire, comme le changement de la langue par défaut de l'anglais vers le français.

For detailed instructions, see Set up multiple content languages.

Étape 2 : Renommer les assertions, les phases, les cycles, les zones, les dimensions et les composants

Pour commencer, sélectionnez l'icône Information () dans le menu supérieur, puis choisissez Paramètres. La boîte de dialogue Product Settings s'ouvre.

Renommer les assertions, les phases, les cycles, les zones, les dimensions et les composants :

  1. Allez dans l'onglet Importation .

  2. Renommer chaque élément de l'onglet :

    • Dans l'onglet Assertions , saisissez l'abréviation française et le nom complet de chaque assertion.

    • Dans les onglets Phases, Cycles et Domaines , saisissez le nom français de chaque phase, cycle et domaine.

      Remarque : Les onglets Cycles et Zones ne sont disponibles dans votre produit que s'ils ont été activés dans votre fichier product.json. Pour en savoir plus, voir Activer ou désactiver des fonctionnalités dans votre produit.

    • Dans l'onglet Dimensions , saisissez le nom français de chaque catégorie de dimension et de chaque dimension

    • Dans l'onglet Composants , saisissez le nom français de chaque catégorie de composant, le nom du composant et l'ID du composant.

Étape 4 : Ajouter le contenu français

Voici l'approche recommandée pour ajouter du contenu traduit dans un produit multilingue pour les lettres/mémos, les listes de contrôle, les requêtes, les états financiers et les groupes de balances de vérification.

Lettres/mémos

Ajoutez une version française au contenu de la lettre ou du mémo, puis configurez-les pour qu'ils ne s'affichent que si la langue sélectionnée pour le contenu du modèle est le français.

Listes de contrôle et questions

Ajouter les versions françaises de chaque procédure et question d'interrogation.

Pour ajouter les procédures françaises dans une liste de contrôle :

Ajouter les références françaises faisant autorité pour une procédure de liste de contrôle :

  1. Sélectionnez + Lien de référence faisant autorité .

  2. Saisissez le texte du lien à afficher dans la procédure à la rubrique Étiquette.

  3. Saisissez l'adresse URL dans URL.

  4. Sélectionnez OK.

États financiers

Pour ajouter une version française aux sections de texte, aux en-têtes et aux pieds de page, assurez-vous que la langue du contenu du modèle est le français, puis modifiez les zones de texte, les en-têtes et les pieds de page existants afin de supprimer le contenu anglais et d'ajouter le contenu français.
