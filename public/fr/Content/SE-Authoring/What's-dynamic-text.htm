Qu'est-ce qu'un texte dynamique ?

Un texte dynamique est un texte conditionnel. Il s'agit d'un champ dans lequel un texte différent s'affiche en fonction d'une condition ou d'un ensemble de conditions. Vous pouvez inclure du texte dynamique dans divers champs de texte lors de la rédaction de listes de contrôle, de requêtes, de lettres ou de mémos et d'états financiers.

Pour savoir comment inclure du texte dynamique dans les différents champs de texte de votre modèle, voir :

  • Ajout d'un texte dynamique dans la description de la procédure dans les listes de contrôle
  • Ajouter un lien de données dans une question de requête
  • Ajouter du texte dans une lettre ou un mémo
  • Ajout d'un texte dynamique dans les sections du texte des états financiers
