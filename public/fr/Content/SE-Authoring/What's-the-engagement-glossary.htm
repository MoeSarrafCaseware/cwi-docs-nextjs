Qu'est-ce que le glossaire de l'engagement ?

Un glossaire est une liste de termes accompagnée d'une définition pour chacun d'entre eux. Dans le SE, vous pouvez définir des termes dans le glossaire de l'engagement qui seront réutilisés dans tous les documents du modèle. Les termes peuvent être des mots ou des phrases. Si vous modifiez un terme dans le glossaire, il est modifié dans toutes ses occurrences dans le produit.

Le glossaire des missions est constitué de groupes contenant des termes apparentés. Chaque terme du glossaire doit faire partie d'un groupe. Vous ne pouvez pas ajouter un terme s'il n'y a pas de groupe dans le glossaire.
