Que sont les paramètres de visibilité ?

Les paramètres de visibilité permettent une automatisation intégrée afin que seuls les documents ou contenus pertinents soient disponibles dans votre dossier d'engagement. Pour ce faire, des conditions de visibilité sont définies pour les différents éléments de l'engagement.

L'application évalue les conditions. Si les conditions sont remplies, certains éléments sont visibles (disponibles) dans le dossier d'engagement - sinon, ils sont cachés (non disponibles).

Voici quelques cas d'utilisation :

  • Masquer les documents qui ne sont pas pertinents pour l'utilisateur jusqu'à ce qu'il ait défini l'étendue de la mission.

  • Rendre une zone de texte d'un document visible uniquement si l'utilisateur fournit une réponse spécifique à une procédure. Cette fonction est utile lorsque vous souhaitez personnaliser le contenu des lettres, des mémos et d'autres documents en fonction des informations collectées dans la balance générale des comptes ou dans divers formulaires et listes de contrôle.

Pour mettre en place la visibilité, vous devez spécifier :

  • le type de condition et ses paramètres

    • Les paramètres indiquent où le système va chercher les informations (valeurs) pour évaluer la condition. Chaque type de condition a un ensemble de paramètres différents.

  • la logique de visibilité

    • Définir l'action de visibilité (si l'élément se cache ou s'affiche) lorsque les conditions sont remplies. De même, si vous avez ajouté plus d'une condition, vous devez définir si toutes les conditions ou au moins une condition doivent être vraies pour que l'action de visibilité s'applique.

Pour plus d'informations sur la mise en place d'une logique de visibilité dans l'ensemble du modèle de produit, voir les rubriques suivantes.

  • Ajouter des paramètres de visibilité dans les listes de contrôle
  • Ajouter des paramètres de visibilité dans les requêtes
  • Ajouter des paramètres de visibilité dans les lettres ou les mémos
  • Ajouter des paramètres de visibilité dans les états financiers
  • Ajouter une logique de visibilité pour les documents et les éléments de l'engagement
