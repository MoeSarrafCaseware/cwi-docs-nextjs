Flux de travail (comment finaliser le projet ?)

Avant de finaliser et de distribuer votre projet, voici quelques éléments à prendre en compte...

  • Des identifiants XBRL (eXtensible Business Reporting Language) sont disponibles dans toute l'application pour aider les marchés qui doivent soumettre leurs états financiers au format XBRL.

  • Si la consolidation des engagements est pertinente dans votre solution, notez que vous devez décider si vous voulez que les données consolidées soient affichées dans le document des états financiers ou non. Pour plus d'informations, voir Spécifier s'il faut ou non utiliser des données consolidées et Configurer les données de la colonne.

  • Si les rapports intermédiaires sont pertinents dans votre solution, vous devez.. :

    • activer les propriétés de rapports intermédiaires dans l'API du produit, voir Propriétés d'engagement pour plus d'informations.

    • paramétrer les types de colonnes dans le tableau pour qu'elles soient la période en cours, voir Paramétrer les données des colonnes.

    Vous pouvez aussi :

    • paramétrer la visibilité de manière à ce que des documents spécifiques soient disponibles dans le modèle en fonction de la période en cours. Voir Période de référence.

    • mettre en place un texte dynamique et des termes de glossaire d'engagement afin que des termes spécifiques soient disponibles dans le texte narratif (dans les lettres, les mémos ou les états financiers) sur la base de la période en cours. Ajouter des liens de données.

  • Si le report est pertinent dans votre solution et que vous avez configuré des lignes ou des colonnes d'entrée dans les tableaux dynamiques pour les données de l'exercice en cours (ou de la période en cours) et de l'exercice précédent (ou de la période précédente), l'application n'est pas automatiquement fixer les valeurs des années (ou périodes) dans les lignes ou colonnes de saisie après l'action de report.

  • Vous devez ajouter une note aux utilisateurs indiquant les lignes ou colonnes d'entrée pour lesquelles ils doivent fixer manuellement les valeurs de l'année après avoir effectué un report.

Les étapes suivantes résument le processus recommandé pour finaliser votre projet. Vous pouvez utiliser ce flux de travail pour vous guider.
