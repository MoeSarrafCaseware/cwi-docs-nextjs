Workflow (comment mettre en place des tableaux dynamiques ?)

Les tableaux font référence aux différentes annexes que vous pouvez inclure dans vos états financiers.

Pour ajouter un tableau dans une zone d'états financiers, sélectionnez Ajouter (), puis Tableau dynamique. An empty table is added in the area with a default title Dynamic Table. Vous pouvez modifier le titre par défaut en le sélectionnant et en ajoutant un nouveau titre dans le champ.

Remarque : La taille de police par défaut pour le contenu des tableaux dynamiques est de 12 et ne peut être modifiée.

Les étapes suivantes résument le processus recommandé pour finaliser votre projet. Vous pouvez utiliser ce flux de travail pour vous guider.

  • Add rows

  • Ajouter des colonnes

  • Créer des groupes personnalisés dans les tableaux

  • Exclure des lignes d'un total de groupe personnalisé

  • Ajouter une référence de note dans une section de texte

  • Permettre aux utilisateurs d'ajouter des lignes de saisie dans la zone du tableau sans accéder au mode d'édition du tableau

  • Set the default display option for accounts and groups

  • Ajouter un saut de page dans les états financiers

  • Copier le tableau pour en créer rapidement un autre avec une configuration similaire
