Automatiser les entrées de temps en fonction de l'heure dans un fichier

Que vous soyez cadre ou employé, il est essentiel de soumettre des entrées de temps exactes. Le suivi automatique de l'engagement, qui enregistre automatiquement le temps passé dans un dossier d'engagement pris en charge, est un moyen de garantir l'exactitude des données. Cela permet d'éliminer la plupart des erreurs d'écriture si vous soumettez vos heures manuellement.

Le suivi automatique de l'engagement fonctionne en enregistrant le temps que vous passez dans un fichier Working Papers (ou une application intégrée) et crée des entrées en attente pour votre approbation. Les entrées en attente fonctionnent comme des entrées de temps, mais elles doivent d'abord se voir attribuer un code de travail et être approuvées avant de pouvoir être comptabilisées comme du temps facturable ou non facturable.

To generate time entries based on time spent in the file automatically:

  1. Assurez-vous que vous avez le rôle Settings Admin ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Dans le menu déroulant Temps , sélectionnez Paramètres de temps et de dépenses.

  4. Cochez la case Suivi automatique de l'engagement , puis sélectionnez Enregistrer.

Le temps passé dans les applications et fichiers intégrés est désormais enregistré dans l'application Temps sous la forme d'une série d'entrées en attente.
