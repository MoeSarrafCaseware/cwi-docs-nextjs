Renseignements sur les données Caseware Cloud

Caseware Cloud permet de stocker facilement vos fichiers pour les utiliser avec la gamme de produits Caseware.

Où sont stockées mes données ?

Toutes les données des abonnés sont stockées dans l'environnement de production en nuage. L'utilisation de données des clients dans des environnements hors production est contrôlée au moyen de processus sécurisés de manipulation des données, qui exigent une approbation explicitement documentée des clients dont les données sont concernées et doivent se conformer aux exigences légales réglementaires pour le nettoyage des éléments de données sensibles.

Une équipe des Opérations désignée est responsable de toutes les fonctions opérationnelles relatives à l'infrastructure et au stockage avec des responsabilités attribuées qui ont été définies, documentées et communiquées.

Caseware Cloud est hébergé sur les serveurs Web d'Amazon dans le monde entier. Lors de l'abonnement aux services Caseware Cloud, Caseware Cloud (CWC) informe les clients du pays dans lequel se trouve le serveur attribué pour héberger les données des abonnés et les renseignements personnels. Vous pouvez consentir à cette attribution ou refuser un serveur ainsi attribué.

Pour des raisons de performance, nous vous installons généralement dans les pays suivants :

  • États-Unis/Nord de la Virginie si vous êtes situé aux États-Unis ou en Amérique du Sud
  • Canada/Montréal si vous êtes situé au Canada
  • Australie/Nouvelle-Galles du Sud si vous êtes situé dans la région Asie-Pacifique
  • Irlande/Leinster si vous êtes situé dans une autre région

Comment puis-je obtenir des détails sur mon utilisation actuelle des données ?

Les détails des données stockées dans votre compte Cloud sont catalogués pour votre référence. Vous pouvez consulter les quantités de données actuelles et maximales ainsi qu'un certain nombre d'autres statistiques.

Pour connaître les détails de vos données actuelles, utilisez

  1. Assurez-vous que vous disposez du personnel : Rôle d'administrateur ou droits équivalents.

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Dans le menu, sélectionnez Système | Statistiques du site.

Le volet des statistiques du site affiche les détails de votre utilisation actuelle des données.

Combien de temps les données sont-elles conservées après la désactivation ?

Les comptes Cloud désactivés conservent toutes les données hébergées au moment de la désactivation. Si un compte désactivé est rétabli, les données seront à nouveau accessibles.

Pour l'instant, il n'y a pas de date d'expiration pour les données stockées. Si vous souhaitez supprimer complètement votre compte Cloud et toutes les données qui y sont associées, vous pouvez en faire la demande en contactant sales@caseware.com. Notez qu'une fois supprimés, le compte et les données ne peuvent pas être récupérés.
