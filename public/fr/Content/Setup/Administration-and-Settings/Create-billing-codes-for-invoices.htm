Créer des codes de facturation pour les factures

Cloud est livré avec un ensemble de codes de facturation prédéfinis pour le travail fourni par votre organisation : GEN (comptabilité générale), MGM (conseil en gestion) et TAX (préparation fiscale). You also have the option to add new billing codes. Par exemple, si votre organisation effectue des travaux d'audit, vous pouvez ajouter un code de facturation spécifique (tel que AUD).

Vous avez créé un nouveau code de facturation. Vous pourrez l'ajouter aux nouvelles factures que vous créez ou en modifiant des factures préexistantes. Pour afficher la liste de tous les codes de facturation de votre organisation, sélectionnez Paramètres | Temps | Codes de facturation.
