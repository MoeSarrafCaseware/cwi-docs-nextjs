Créer des champs, des étiquettes et des flux de travail personnalisés

Bien que Cloud dispose de plusieurs outils de catégorisation intégrés pour vous aider à organiser votre personnel, vos équipes internes, vos contacts et vos clients, vous pouvez personnaliser ces catégories et les ajouter ou les modifier afin de mieux refléter les activités de votre organisation.

Création de balises

Lestags sont des mots-clés que vous pouvez attribuer à des personnes, des entités ou des fichiers dans le nuage. Les étiquettes peuvent être utilisées comme termes de recherche, ce qui permet aux utilisateurs de localiser facilement les documents et le personnel concernés. Les administrateurs peuvent créer des étiquettes à l'échelle du système qui peuvent être attribuées à toutes les catégories et des étiquettes spécialisées que les utilisateurs ne peuvent attribuer qu'au personnel, aux contacts, aux fichiersou aux entités.

Pour créer des balises :

  1. Assurez-vous d'avoir le rôle d' administrateur des paramètres ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Sélectionnez Personnalisation | Gestion des balises.

  4. Sélectionnez Ajouter pour ouvrir la boîte de dialogue Ajouter une balise.

  5. Saisissez un nom et indiquez le type de balise(Ensemble du système, Personnel, Contacts, Fichiersou Entités) pour votre nouvelle balise.

  6. Sélectionnez OK.

Vous verrez les balises créées dans la boîte de dialogue Balise () pour tous les objets du nuage pour lesquels les balises sont disponibles. Sélectionnez les balises non attribuées pour les affecter à l'objet, puis sélectionnez Enregistrer pour appliquer vos modifications.

Pour ajouter des champs personnalisés aux entités :

Cloud est livré avec un ensemble prédéfini de propriétés que les utilisateurs peuvent attribuer à chaque type d'entité lorsqu'ils ajoutent ou modifient une entité, mais vous pouvez également ajouter des groupes de propriétéspersonnalisés.

Par exemple, supposons que vous souhaitiez que les utilisateurs puissent classer les autres entités en fonction du type de service qu'elles fournissent à votre organisation, comme les fournitures de bureau, les conseils juridiquesou le recrutement de personnel. Vous pouvez ajouter un groupe de propriétés intitulé Type de service aux autres entités, puis ajouter chaque type de service fourni en tant qu'option.

Pour ajouter des champs personnalisés aux entités :

  1. Assurez-vous d'avoir le rôle d' administrateur des paramètres ou des propriétés équivalentes.

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Sélectionnez Personnalisation | Propriétés de l'entité personnalisée.

  4. Sélectionnez le type d'entité pour lequel vous souhaitez ajouter un groupe de propriétés(Tous, Clients, Autres entitésou Entités internes), puis sélectionnez Ajouter un groupe de propriétés.

  5. Saisissez un nom de groupe dans le premier champ de texte et sélectionnez Ajouter une propriété pour ajouter des champs de texte pour les propriétés.

  6. Dans les cases déroulantes situées à côté de chaque propriété, sélectionnez le type de données à fournir par l'utilisateur pour cette propriété, par exemple une URL ou un montant.

  7. Pour tout champ dont la saisie par l'utilisateur est requise pour l'enregistrement de l'entité, cochez la case Champ obligatoire .

  8. Sélectionnez Enregistrer pour créer les propriétés personnalisées de l'entité.

Le groupe de propriétés sauvegardé apparaît dans les boîtes de dialogue de création et de modification du type d'entité spécifié.

Création de flux de travail

Dans le nuage, un flux de travail est un processus qui peut être créé pour n'importe quel utilisateur, entité ou type d'objet du nuage. Les flux de travail attribuent des étapes à ces objets afin de les organiser en fonction de leur état d'avancement dans le "flux" conçu

Par exemple, vous pouvez créer un flux de travail pour les entités clients afin de classer vos clients en fonction de l'étape actuelle de votre relation commerciale (par exemple, nouveau clientou client à long terme). Voir Exemples de flux de travail pour d'autres suggestions de flux de travail.

Pour créer un nouveau flux de travail :

  1. Assurez-vous d'avoir le rôle d' administrateur .

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Sélectionnez Personnalisation | Gestion du flux de travail.

  4. En regard du type d'élément à créer, sélectionnez (Non attribué) | Créer un flux de travail.

  5. Saisissez un nom de flux de travail et sélectionnez Ajouter une étape () pour commencer à spécifier les étapes de ce flux de travail.

  6. Sélectionnez la liste déroulante des couleurs ( ) pour modifier la couleur de la phase de travail ou faites glisser l'icône ( ) pour réorganiser les phases.

  7. Cliquez sur OK.

Pour modifier un flux de travail existant :

  1. Assurez-vous d'avoir le rôle d' administrateur .

  2. Allez dans Paramètres | Personnalisation | Gestion du flux de travail.

  3. Sélectionnez Gérer les flux de travail.

  4. Dans la boîte de dialogue Gérer les flux de travail, sélectionnez une étape de flux de travail applicable et sélectionnez Modifier.

  5. Apportez vos modifications au flux de travail et sélectionnez OK.

Once you create workflows, you can assign them to items. Par exemple, vous pouvez créer un flux de travail sur le statut du personnel avec des étapes telles que nouveau, probation, temps pleinet congé temporaire. Une fois que le flux de travail des membres du personnel est en place, vous pouvez définir le statut de vos nouveaux utilisateurs.

Pour attribuer des flux de travail à des utilisateurs :

  1. Assurez-vous que vous avez le rôle d' administrateur du personnel ou des droits équivalents.

  2. Dans le menu Nuage, sélectionnez Personnel ou Contacts.

  3. Sélectionnez l'utilisateur dont vous souhaitez modifier l'étape du flux de travail.

  4. Dans le menu de l'utilisateur, sélectionnez le flux de travail actuel, puis l'option nouveau flux de travail.
