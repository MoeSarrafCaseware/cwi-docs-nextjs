Personnaliser les courriels générés par le système

Vous pouvez personnaliser les courriels générés par le système de votre entreprise, tels que le message de bienvenue pour les nouveaux utilisateurs et les notifications envoyées au personnel et aux contacts.

Personnaliser les courriels générés par le système

  1. Assurez-vous d'avoir le rôle d' administrateur des paramètres ou des privilèges équivalents.

  2. Dans le menu Cloud (), sélectionnez Paramètres.

  3. Sélectionnez Email | Settings.

  4. Personnalisez les paramètres d'impression selon vos besoins.

  5. Sélectionnez Enregistrer.
