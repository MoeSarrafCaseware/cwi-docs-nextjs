Organiser les fichiers par dossiers et étiquettes

Caseware Cloud contient de multiples façons d'organiser vos fichiers.

Organiser les fichiers dans des dossiers

Les dossiers permettent d'organiser les fichiers dans une entité.

Pour créer un dossier :

  1. Dans le menu Cloud (), sélectionnez Fichiers.

  2. Sélectionnez l'entité appropriée dans la liste déroulante des entités.

  3. Dans le volet Dossiers , sélectionnez l'emplacement où vous souhaitez créer un nouveau dossier.

  4. Sélectionnez le menu déroulant () puis Créer un nouveau dossier.

  5. Saisissez un nom et une description pour le dossier.

  6. Sélectionnez OK.

Pour déplacer des fichiers dans un dossier :

  1. Sélectionnez les fichiers que vous souhaitez déplacer dans un dossier.

  2. Dans le panneau de détails du fichier, sélectionnez l'option Plus d'actions() | Déplacer vers le dossier.

  3. Choisissez le dossier dans lequel vous souhaitez déplacer le fichier et sélectionnez Déplacer. Vous pouvez également glisser-déposer le fichier pour le déplacer dans un autre dossier.

Pour supprimer un dossier :

  1. Sélectionnez le dossier à supprimer.

  2. Sélectionnez le menu déroulant ( ) puis Supprimer.

  3. Sélectionnez Oui pour confirmer la suppression.

Si vous avez supprimé un dossier par accident, vous pouvez le restaurer. Voir Récupération de dossiers et de fichiers dans la corbeille pour plus d'informations.

Fichiers Tag

Les balises vous permettent de classer vos fichiers par catégories afin de les retrouver plus facilement.

Avant de pouvoir attribuer des balises, vous devez en créer. Pour en savoir plus sur la création de balises, voir Créer des balises.

Avant de commencer, assurez-vous que vous avez le rôle d' éditeur ou des droits équivalents. Pour en savoir plus, voir Rôles intégrés.

Pour marquer un fichier :

  1. Sélectionnez un fichier, puis sélectionnez Tag ().

  2. Sélectionnez les balises à appliquer au fichier.

  3. Sélectionnez Enregistrer.

Déplacer des fichiers en masse

Dans Caseware Cloud, vous pouvez déplacer des fichiers en masse vers une entité ou un dossier.

Pour déplacer des fichiers vers une autre entité :

  1. Sélectionnez Toutes les entités ou l'entité contenant les fichiers à déplacer dans le sélecteur d'entité.

  2. Dans le menu Cloud (), sélectionnez Fichiers.

  3. Sélectionnez les fichiers que vous souhaitez déplacer vers une autre entité.

  4. Sélectionnez More Actions () | Move to Entity.

  5. Choisissez l'entité vers laquelle vous souhaitez déplacer les fichiers et sélectionnez OK.

  6. Dans la boîte de dialogue de confirmation, sélectionnez Oui.
