Définir les taux du personnel et les codes de travail

Classez le temps et les dépenses facturables et non facturables de votre personnel en créant des codes de travail personnalisés. Par exemple, si vous souhaitez que votre personnel suive le temps passé à des tâches administratives internes, vous pouvez ajouter un code de travail de temps non facturable configuré de manière appropriée.

Create new work codes

Vous pouvez créer de nouveaux codes de travail à partir de la page Paramètres .

Pour créer un code de travail :

  1. Assurez-vous que vous avez le rôle d' administrateur des paramètres ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Sélectionnez Temps et, sous Codes de travail, sélectionnez Temps ou Frais.

  4. Sélectionnez Nouveau | Ajouter.

  5. Complétez les champs comme il convient. Le seul champ obligatoire est le code de travail.

    Remarque : La liste des codes de travail peut ne pas être triée correctement si elle contient des codes alphanumériques de longueur incohérente (par exemple, 1 ABC, 23 DEF, 456 GHI). Pour remédier à ce problème, ajoutez des zéros à chaque numéro afin de garantir une longueur cohérente (par exemple, 001 ABC, 023 DEF, 456 GHI) ou utilisez des codes numériques.

  6. Sélectionnez Enregistrer.

Votre nouveau code de travail est ajouté.

Pour importer une liste de codes de travail :

Vous pouvez également ajouter des codes de travail en masse en utilisant un bloc de texte délimité ou unfichier .csv. Vous aurez besoin d'une liste d'utilisateurs dans un fichier .csv ou un bloc de texte délimité.

Pour l'une ou l'autre méthode, il faut s'assurer que

  • Chaque ligne représente un membre du personnel

  • Chaque colonne représente une propriété, telle que le nom, l'adresse électronique et le groupe d'utilisateurs. Pour inclure un utilisateur dans plusieurs groupes, placez un tilde (~) entre chaque nom de groupe.

Dans cet exemple, un administrateur souhaite ajouter trois codes de dépenses :

Code, type, nom, taux, supplément

E900, B, Divertissement, 0,05, 0,00

E10100, NB, Equipement, 0.10, 0.15

E1000, B, Frais de consultation, 0,10, 0,00

Pour importer une liste de codes de travail :

  1. Assurez-vous que vous avez le rôle d' administrateur des paramètres ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Paramètres.

  3. Sélectionnez Temps et, sous Codes de travail, sélectionnez Temps ou Frais.

  4. Sélectionnez Nouveau | Importer.

  5. Sélectionnez Importer à partir de CSV et naviguez jusqu'à votre fichier de répertoire .csv, ou collez le texte de votre liste délimitée dans la zone de texte.

  6. Sélectionnez les options appropriées au bas de la boîte de dialogue, puis sélectionnez Suivant.

  7. Mettez en correspondance la propriété qui correspond à chaque colonne. Le numéro de code de travail est nécessaire pour terminer l'importation. Si des codes existants utilisent le même numéro de code de travail, ils seront mis à jour lors de l'importation.

    8. Si vous mappez le type de facture, il vous sera demandé d'identifier les codes facturables et non facturables dans une boîte de dialogue séparée.

  8. Sélectionnez Enregistrer.

Une boîte de dialogue confirme le nombre de codes importés ou modifiés.

Fixer les taux de rémunération du personnel :

Le personnel doit être paramétré avec les taux appropriés avant de pouvoir soumettre ses entrées de temps. Vous pouvez attribuer des taux standard et des taux de code de travail à des membres individuels du personnel. Supposons, par exemple, que vous souhaitiez appliquer des taux différents pour le travail général et le travail plus spécialisé, tel que la consultation juridique. Dans ce cas, vous attribuerez un taux standard de base et un taux plus élevé associé au code de travail pour le travail spécialisé.

Fixer les taux de rémunération du personnel :

  1. Assurez-vous que vous avez le rôle d' administrateur du personnel ou des droits équivalents.

  2. Dans le menu Nuage, sélectionnez Personnel.

  3. Sélectionnez l'utilisateur dont vous souhaitez modifier le taux. Dans le menu de l'utilisateur, sélectionnez More Actions () | Rates.

  4. Pour ajouter un taux standard, sélectionnez Taux standard | Nouveau, remplissez les champs et sélectionnez Enregistrer.

  5. Pour ajouter un taux spécifique à un code de travail, sélectionnez Taux de code de travail | Nouveau, remplissez les champs et sélectionnez Enregistrer. Vous devrez rechercher ou sélectionner les codes de travail existants de votre organisation dans le champ Code de travail.

Les taux que vous avez ajoutés ou mis à jour se trouvent dans la boîte de dialogue Taux de personnel de chaque utilisateur.
