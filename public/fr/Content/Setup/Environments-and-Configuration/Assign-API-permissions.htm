Attribuer des autorisations API (Azure AD)

La configuration des permissions API permettra à Caseware Cloud de lire les informations de base de Microsoft Azure afin d'authentifier correctement les utilisateurs.

Pour attribuer des autorisations API :

  1. Sélectionnez Azure Active Directory | App Registrations, puis sélectionnez votre application.

  2. Dans le volet de navigation de gauche, sélectionnez Autorisations API.

  3. Sélectionnez Ajouter une autorisation.

  4. Sélectionnez Microsoft Graph.

  5. Sélectionnez Autorisations déléguées.

  6. Utilisez la case à cocher pour sélectionner les autorisations suivantes :

    • Courriel

    • Openid

    • Profil

    • Utilisateur.Lire

  7. Sélectionnez Ajouter des autorisations.

  8. Sélectionnez Accorder le consentement de l'administrateur pour le répertoire par défaut.

  9. Sélectionnez Oui à la question "Souhaitez-vous accorder les autorisations demandées pour tous les comptes du répertoire par défaut ?"
