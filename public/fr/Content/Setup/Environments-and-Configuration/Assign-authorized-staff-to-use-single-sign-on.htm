Affecter le personnel autorisé pour utiliser l'authentification unique (Azure AD)

Si vous voulez vous assurer que seul le personnel autorisé au sein de votre entreprise est en mesure d'accéder à Caseware Cloud via l'authentification unique, vous pouvez assigner des utilisateurs autorisés à votre application dans Azure. Si vous souhaitez que tous les membres du personnel de votre entreprise puissent accéder à Caseware Cloud, vous pouvez sauter cette étape et procéder à l' attribution des permissions API.

Pour affecter le personnel autorisé :

  1. Sélectionnez Azure Active Directory | Enterprise Applications, puis sélectionnez votre application.

  2. Dans le volet de navigation de gauche, sélectionnez Propriétés.

  3. Réglez l' option Affectation de l'utilisateur requise sur Oui.

  4. Sélectionnez Enregistrer.

  5. Dans le volet de navigation de gauche, sélectionnez Utilisateurs et groupes.

  6. Sélectionnez Ajouter un utilisateur.

  7. Sélectionnez Utilisateurs et sélectionnez chaque utilisateur que vous souhaitez affecter à l'application.

    Remarque : Si vous disposez d'un compte Microsoft Azure Premium, vous pouvez créer des groupes d'utilisateurs, puis sélectionner le groupe pour accorder l'accès à tous ses membres. Pour plus d'informations, voir Créer un groupe de base et ajouter des membres à l'aide d'Azure Active Directory.

  8. Sélectionnez Sélectionner lorsque vous avez ajouté tous les utilisateurs qui doivent avoir accès.

  9. Sélectionnez Attribuer pour permettre à ces utilisateurs d'accéder à l'application.
