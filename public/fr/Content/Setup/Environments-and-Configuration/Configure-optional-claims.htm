Configurer les réclamations facultatives (Azure AD)

Avant de terminer le processus d'intégration SSO, vous devez modifier le manifeste pour y inclure les revendications given_name et family_name. Lorsqu'un membre du personnel utilise le SSO, ces demandes fourniront leur prénom et leur nom à Cloud.

Pour configurer les réclamations facultatives :

  1. Connectez-vous à votre portail Azure.

  2. Sélectionnez Azure Active Directory.

  3. Dans le volet de navigation de gauche, sélectionnez Enregistrements d'applications.

  4. Sélectionnez la bonne application dans la liste.

  5. Dans le volet de navigation de gauche, sélectionnez Manifeste.

  6. Modifiez la propriété optionalClaims comme suit :

    "optionalClaims" : {
    
	"idToken" : [
    
		{
    
			"nom" : "prénom",
    
			"essentiel" : vrai
    
		},
    
		{
    
			"nom" : "nom_de_famille",
    
			"essentiel" : vrai
    
		}
    
	]
    
},

    Remarque : Vous devez recopier exactement les informations ci-dessus pour configurer correctement les réclamations facultatives.

  7. Sélectionnez Enregistrer.
