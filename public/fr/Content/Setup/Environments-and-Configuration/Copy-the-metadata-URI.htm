Copier l'URI des métadonnées (Okta)

Un URI de métadonnées fournit les données nécessaires au fonctionnement du SSO. Vous l'utiliserez lorsque vous configurerez le SSO dans le Cloud.

Pour copier le point de terminaison des métadonnées :

  1. Connectez-vous au tableau de bord Okta de votre entreprise avec un compte administrateur .

  2. Développez le menu déroulant du compte et passez votre curseur sur l'URI des métadonnées. Une icône de presse-papiers s'affiche. Sélectionnez-le pour copier l'URI des métadonnées.

  3. Coller l'URI des métadonnées dans un document texte. Il se présentera sous la forme suivante :

    <compte>.okta.com

  4. Modifiez l'URI des métadonnées pour qu'il corresponde au format suivant :

    https://$<compte>.okta.com/.well-known/openid-configuration

  5. Attribuez-lui le nom de Metadata URI et enregistrez le document texte pour une utilisation ultérieure.
