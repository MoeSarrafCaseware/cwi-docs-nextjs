Créer une application API Web (ADFS)

Créez une application Web API pour le site Cloud de votre entreprise et définissez les autorisations de l'application.

• Créer une application API Web

  1. Dans la console d'administration ADFS, cliquez sur le dossier Groupes d'application , puis double-cliquez sur votre nouveau groupe d'application.

  2. In the Properties dialog, click Add Application.

  3. Dans la liste Template, sélectionnez Web API. Cliquez sur Suivant.

  4. Collez le Secret du Client que vous avez copié précédemment dans un document texte. Cliquez sur Suivant.

  5. Dans la liste des politiques de contrôle d'accès, sélectionnez celle qui s'applique le mieux à votre entreprise. Cliquez sur Suivant.

  6. Dans la liste des champs d'application autorisés, sélectionnez les champs d'application suivants :

    • toutes les demandes d'indemnisation

    • Courriel

    • Openid

    • Profil

    Cliquez sur Suivant.

  7. Cliquez sur Oui pour remplir le rapport.
