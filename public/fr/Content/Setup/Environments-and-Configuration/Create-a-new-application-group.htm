Créer un nouveau groupe d'applications (ADFS)

Pour intégrer Cloud à ADFS, vous devez d'abord créer un nouveau groupe d'applications pour Cloud.

• Créer un nouveau groupe d'applications

  1. Lancez la console d'administration de l'ADFS.

  2. Cliquez sur le dossier Groupes d'application , puis dans le menu Actions, cliquez sur Ajouter des groupes d'application.

  3. Saisissez un nom pour le formulaire. Dans la liste Template, sélectionnez Server application. Cliquez sur Suivant.

  4. Le champ Identifiant du client est rempli automatiquement.

    Important : Copiez l'identifiant de l'application et collez-le dans un document texte pour une utilisation ultérieure. Inscrivez-y le mot " secret client".

    Dans le champ Redirect URI, entrez une URL fictive (par exemple, https://www.caseware.com) et cliquez sur Add. Cliquez sur Suivant.

  5. Sélectionnez Générer un secret partagé.

    Important : Click Copy to clipboard and paste the secret to a text document for later use. Inscrivez-y le mot " secret client".

    Cliquez sur Suivant.

  6. Cliquez sur Oui pour remplir le rapport.
