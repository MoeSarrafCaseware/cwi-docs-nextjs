Créer une règle de transformation des délivrances (ADFS)

Créez une règle Issuance Transform pour faire correspondre les attributs LDAP aux créances sortantes.

• Créer une règle de transformation des délivrances

  1. Dans la console d'administration ADFS, cliquez sur le dossier Groupes d'application , puis double-cliquez sur votre nouveau groupe d'application.

  2. Dans la boîte de dialogue Propriétés, double-cliquez sur l' application API Web.

  3. Dans la boîte de dialogue Propriétés de l'API Web, cliquez sur l'onglet Règles de transformation de l'émission , puis sur Ajouter une règle.

  4. Cliquez sur le menu déroulant du modèle de règle de réclamation et sélectionnez Send LDAP Attributes as Claims (Envoyer les attributs LDAP en tant que réclamations). Cliquez sur Suivant.

  5. Saisissez un nom de règle de réclamation, puis cliquez sur le menu déroulant Attribute store et sélectionnez Active Directory.

  6. Complétez le tableau Mapping of LDAP attribute to outgoing claim types comme suit :

    Attribut LDAP Type de réclamation sortante
    E-Mail-Addresses Courriel
    Prénom "prénom",
    NOM "nom_de_famille",

  7. Cliquez sur Oui pour remplir le rapport.
