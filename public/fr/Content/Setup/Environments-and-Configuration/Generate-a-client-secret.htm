Générer un secret client (Azure AD)

Pour permettre une communication sécurisée avec votre domaine Active Directory, vous devez générer un secret client. Le secret du client est utilisé par Azure pour authentifier le Cloud.

Pour générer un secret client :

  1. Connectez-vous à votre portail Azure.

  2. Sélectionnez Azure Active Directory.

  3. Dans le volet de navigation de gauche, sélectionnez Enregistrements d'applications.

  4. Sélectionnez la bonne application dans la liste.

  5. Sélectionnez Certificats et secrets dans le volet de navigation de gauche.

  6. Sélectionnez Nouveau secret client.

  7. Saisissez une description appropriée de la clé et sélectionnez une durée.

    Remarque : Si votre clé expire, les utilisateurs ne pourront pas accéder à Cloud. Nous vous recommandons de renouveler votre clé avant son expiration afin de minimiser les perturbations. Pour renouveler votre clé, générez un nouveau secret client et ajoutez-le au champ Secret client dans Paramètres | Signature unique | Fournisseur d'identité, avec l' ID d'application et le point de terminaison des métadonnéesexistants.

  8. Sélectionnez Ajouter.

    Important : Copiez la valeur secrète du client et collez-la dans un document texte pour une utilisation ultérieure. Inscrivez-y le mot " secret client".

    Votre secret client sera définitivement caché lorsque vous quitterez le volet Certificats et Secrets. Vous devez avoir accès à votre secret client pour compléter le processus de configuration SSO. Si vous perdez l'accès à votre valeur de secret client, vous devez supprimer votre secret client et en générer un nouveau.
