Générer les informations d'identification du client (Okta)

Pour qu'Okta authentifie l'intégration avec Cloud, vous devez générer un ID client et un secret. Veillez à conserver une copie de ces informations d'identification afin de ne pas avoir à les régénérer ultérieurement.

To generate client credentials:

  1. Connectez-vous au tableau de bord Okta de votre entreprise avec un compte administrateur .

  2. Dans le menu principal, développez le menu déroulant Applications et sélectionnez Applications.

  3. Sur la page Applications, sélectionnez le nom de votre intégration d'applications.

  4. Sur la page d'intégration de l'application, sous l'onglet Général , copiez la valeur de l' ID du client et collez-la dans un document texte pour une utilisation ultérieure. Etiqueter l' ID de l'application.

  5. Dans le groupe Secrets du client, sélectionnez Générer un nouveau secret. Copiez la valeur secrète du client et collez-la dans un document texte pour une utilisation ultérieure. Inscrivez-y le mot " secret client".
