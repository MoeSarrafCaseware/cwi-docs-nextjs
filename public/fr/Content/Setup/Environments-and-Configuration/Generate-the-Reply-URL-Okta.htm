Générer l'URL de réponse (Okta)

Okta utilise une URL de réponse pour rediriger les utilisateurs vers Cloud après qu'ils se sont connectés avec Okta.

To generate the Reply URL:

  1. Assurez-vous d'avoir le rôle d' administrateur des paramètres ou des autorisations équivalentes.

  2. Dans le menu Cloud (), sélectionnez Settings | Single Sign-On | Identity Provider.

  3. Sur la page du fournisseur d'identité, remplissez les champs suivants :

    • Nom du fournisseur d'identité: Saisissez un nom significatif pour votre fournisseur d'identité. Par exemple, Okta. Ce nom sera affiché à côté de l'URL de votre réponse pour l'identifier.

      Remarque : Ce nom ne peut pas dépasser 32 caractères et ne peut contenir que des lettres, des chiffres, des traits de soulignement, des signes plus et moins.

    • Point d'arrivée des métadonnées du fournisseur d'identité: Collez le point de terminaison des métadonnées que vous avez copié précédemment dans un document texte.

    • ID du client : Collez l' identifiant de l'application que vous avez copié précédemment dans un document texte.

    • Secret du client : Collez le Secret du Client que vous avez copié précédemment dans un document texte.

    Sélectionnez Enregistrer. L'URL de votre réponse s'affiche.

  4. Sélectionnez le bouton Copier dans le presse-papiers () pour copier l'URL de votre réponse. Vous devez l'ajouter à votre application dans Azure AD.

Set your Reply URL in Okta

Après avoir généré une URL de réponse, vous devrez l'ajouter à Okta pour vous assurer que les utilisateurs peuvent se connecter avec succès.

Pour définir votre URL de réponse dans Azure :

  1. Connectez-vous au tableau de bord Okta de votre entreprise avec un compte administrateur .

  2. Dans le menu principal, développez le menu déroulant Applications et sélectionnez Applications.

  3. Sur la page Applications, sélectionnez le nom de votre intégration d'applications.

  4. Sur la page d'intégration de l'application, sous l'onglet Général , descendez jusqu'au groupe Paramètres généraux.

  5. Sélectionnez Modifier.

  6. Faites défiler la page jusqu'à la section Personnes.

  7. À côté du champ Sign-in redirect URIs , sélectionnez Add URI.

  8. Collez l'URL de la réponse dans le champ.

  9. De plus, collez l'URL de la réponse dans le champ URI de connexion de l'initiateur . Sélectionnez Enregistrer.
