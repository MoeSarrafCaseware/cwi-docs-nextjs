Générer l'URL de réponse (ADFS)

ADFS utilise une URL de réponse pour rediriger les utilisateurs vers Cloud après qu'ils se soient connectés.

Pour générer une URL de réponse dans le nuage :

  1. Assurez-vous d'avoir le rôle d' administrateur des paramètres ou des autorisations équivalentes.

  2. Dans le menu Cloud (), sélectionnez Settings | Single Sign-On | Identity Provider.

  3. Sur la page du fournisseur d'identité, remplissez les champs suivants :

    • Nom du fournisseur d'identité: Saisissez un nom significatif pour votre fournisseur d'identité. Par exemple, ADFS. Ce nom sera affiché à côté de l'URL de votre réponse pour l'identifier.

      Remarque : Ce nom ne peut pas dépasser 32 caractères et ne peut contenir que des lettres, des chiffres, des traits de soulignement, des signes plus et moins.

    • Point d'arrivée des métadonnées du fournisseur d'identité: Saisissez votre point de terminaison des métadonnées en utilisant le format suivant :

      https://your-adfs.com/adfs/.well-known/openid-configuration

      Remplacez votre-adfs.com par votre nom d'hôte ADFS.

    • ID du client : Collez le Secret du Client que vous avez copié précédemment dans un document texte.

    • Secret du client : Collez le Secret du Client que vous avez copié précédemment dans un document texte.

  4. Sélectionnez Enregistrer. L'URL de votre réponse s'affiche.

    Important : Sélectionnez l'icône Copier dans le presse-papiers () et collez l'URL de la réponse dans un document texte pour une utilisation ultérieure. Etiqueter l' URL de l'application.

Set your Reply URL in ADFS

Après avoir généré une URL de réponse, vous devrez l'ajouter à l'ADFS pour vous assurer que les utilisateurs peuvent se connecter avec succès.

Pour définir votre URL de réponse dans Azure :

  1. Dans la console d'administration ADFS, cliquez sur le dossier Groupes d'application , puis double-cliquez sur votre nouveau groupe d'application.

  2. Dans la boîte de dialogue Propriétés, double-cliquez sur l'application Serveur.

  3. Dans la boîte de dialogue Propriétés de l'application serveur, effacez le contenu du champ Redirect URI, puis collez l' URL de réponse que vous avez copiée précédemment dans un document texte. Cliquez sur Ajouter, puis sur OK.
