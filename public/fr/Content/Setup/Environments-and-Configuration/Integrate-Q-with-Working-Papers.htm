Intégrer Caseware Q à Caseware Working Papers

Si votre organisation dispose de l'application Q, vous pouvez l'intégrer à un modèle d'audit des documents de travail pour activer ses fonctions.

Intégrer Q dans les documents de travail :

  1. Dans Working Papers, sélectionnez Outils | Modèles pour lancer le menu Modèles.

  2. Sélectionnez le modèle que vous souhaitez intégrer et cliquez sur Modifier. Le modèle s'ouvre.

  3. Dans le ruban Accueil , accédez au menu Insertion et sélectionnez À partir de la bibliothèque. Dans le menu déroulant, choisissez le modèle que vous intégrez.

  4. Q doit être ajouté au modèle. Dans la bibliothèque de documents, développez les dossiers suivants pour atteindre Q :

    • Caseware Audit System Documents | Control Documents | Q - Caseware Q

  5. Sélectionnez Q et cliquez sur Ajouter.

  6. Cliquez sur Fermer.

L'application Q est intégrée à votre modèle d'audit. Vous pouvez désormais consulter l'état d'avancement de votre audit dans le nuage en sélectionnant l'application Q ()
