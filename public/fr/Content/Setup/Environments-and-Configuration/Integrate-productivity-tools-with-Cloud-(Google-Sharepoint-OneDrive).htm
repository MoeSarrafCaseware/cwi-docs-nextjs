Intégrer des outils de productivité à Caseware Cloud

Pour faciliter l'accès, vous pouvez intégrer des outils de productivité dans Caseware Cloud.

Pour intégrer votre compte Google :

Vous pouvez intégrer des comptes Google dans Caseware Cloud pour un accès rapide à Google Drive, Calendar et Viewer.

Pour intégrer votre compte Google :

  1. Sélectionnez votre icône personnelle et choisissez Mes paramètres dans le menu déroulant.

  2. Dans le menu Autres , sélectionnez Paramètres Google.

  3. Sélectionnez Sign in. Saisissez votre nom d'utilisateur et votre mot de passe dans la boîte de dialogue.

  4. Sélectionnez Autoriser pour permettre à Caseware d'accéder à toutes les applications Google pertinentes.

Une fois le compte Google intégré, vous reviendrez à votre session Cloud avec le compte lié. Votre espace de stockage Google actuel est affiché à titre de référence.

Intégrer Office 365

Vous pouvez intégrer Office 365 pour Automatiser les entrées de temps à l'aide de votre calendrier.

Pour intégrer votre compte Office 365 :

  1. Sélectionnez votre icône personnelle et choisissez Mes paramètres dans le menu déroulant.

  2. Dans le menu Autres , sélectionnez Paramètres Office 365.

  3. Sélectionnez Sign in. Saisissez votre nom d'utilisateur et votre mot de passe dans la boîte de dialogue de connexion.

  4. Sélectionnez Oui pour autoriser Caseware à accéder à toutes les applications Microsoft pertinentes.

Après avoir intégré Office 365, vous retournerez à votre session Cloud. L'adresse électronique associée à votre compte Office 365 s'affiche.

Intégrer SharePoint

Vous pouvez intégrer votre compte SharePoint avec Caseware Cloud. L'intégration vous permet de créer et de partager des documents SharePoint à partir du portail Cloud.

Pour intégrer votre compte SharePoint :

  1. Sélectionnez votre icône personnelle et choisissez Mes paramètres dans le menu déroulant.

  2. Dans le menu Autres , sélectionnez Paramètres SharePoint.

  3. Sélectionnez Sign in. Saisissez votre nom d'utilisateur et votre mot de passe dans la boîte de dialogue de connexion.

Votre compte SharePoint est maintenant connecté à Cloud.
