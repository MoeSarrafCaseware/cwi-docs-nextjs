Intégrer l'authentification unique

Cloud prend en charge l'authentification unique (SSO) avec Azure Active Directory (Azure AD), Active Directory Federation Services (ADFS) et Okta. L'utilisation du SSO avec Cloud réduit le nombre de mots de passe que les membres du personnel doivent mémoriser et aide les administrateurs à gérer les problèmes liés aux comptes (tels que la réinitialisation des mots de passe) à partir d'un seul endroit.

Notez que,

  • Le protocole d'autorisation que nous utilisons pour l'intégration SSO est OpenID Connect.

  • Time Desktop n'est pas compatible avec SSO. Une fois le SSO activé, les utilisateurs SSO ne pourront plus se connecter à Time Desktop.

Pour commencer, contactez le support Caseware pour demander que l'authentification unique soit activée pour votre entreprise.
