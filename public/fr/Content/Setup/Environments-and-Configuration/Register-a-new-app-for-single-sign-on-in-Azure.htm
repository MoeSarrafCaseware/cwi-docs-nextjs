Enregistrer une nouvelle application pour l'authentification unique (Azure AD)

Pour intégrer Cloud à Azure AD, vous devez enregistrer Cloud en tant que nouvelle application. Une fois enregistré, vous pouvez autoriser Cloud à accéder à votre domaine Active Directory.

Enregistrer une nouvelle application pour l'authentification unique (Azure AD)

  1. Connectez-vous à votre portail Azure. Si vous avez accès à plusieurs portails, sélectionnez votre compte dans le coin supérieur droit, puis le locataire de votre choix.

  2. Sélectionnez Azure Active Directory.

  3. Dans le volet de navigation de gauche, sélectionnez Enregistrements d'applications.

  4. Sélectionnez Nouvelle inscription.

  5. Sur la page Enregistrer une demande , saisissez vos données d'enregistrement :

    • Nom: Saisissez un nom approprié.

    • Types de comptes pris en charge: Sélectionnez Comptes dans ce répertoire organisationnel uniquement (Répertoire par défaut uniquement - locataire unique).

    • Redirect URI: Sélectionnez Web. Il n'est pas nécessaire de saisir une URL.

  6. Après avoir saisi vos informations, sélectionnez Enregistrer. Les détails de votre demande s'affichent.

    Important : Copiez l'identifiant de l'application et collez-le dans un document texte pour une utilisation ultérieure. Etiqueter l' ID de l'application.
