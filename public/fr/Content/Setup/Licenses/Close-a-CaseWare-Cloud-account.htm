Fermer un compte Caseware Cloud

Les organisations qui décident de fermer leur compte Cloud doivent en faire la demande à Caseware par téléphone ou par email. Deux options sont disponibles :

  • Désactiver - Fermer le compte et conserver toutes les données stockées. Le compte et les données peuvent être restaurés ultérieurement.
  • Supprimer - Fermer le compte et supprimer toutes les données stockées. Le compte et les données ne peuvent pas être récupérés.

Quelle que soit l'option choisie, la meilleure pratique consiste à télécharger toutes les données pertinentes avant la fermeture.

Dresser une liste du personnel, des contacts et des entités

Vous pouvez créer une liste du personnel, des contacts et des entités en exportant votre vue dans chaque application vers une feuille de calcul.

Pour exporter votre vue actuelle :

  1. Assurez-vous d'avoir le rôle d' administrateur du personnel ou des privilèges équivalents.

  2. Dans le menu Nuage, sélectionnez l'application correspondant à la vue que vous souhaitez exporter : Personnel (), Contacts (), Dossiers (), ou Entités ().

  3. Sélectionnez l'en-tête de l'application pour afficher le menu déroulant. Sous Gérer, sélectionnez Exporter la vue actuelle.

  4. La fenêtre d'exportation s'affiche. Pour commencer l'exportation, sélectionnez Démarrer.

  5. Lorsque l'exportation est terminée, sélectionnez Télécharger pour enregistrer le fichier.

  6. Sélectionnez Fermer.

Télécharger l'exemplaire de référence

Vous pouvez télécharger une copie de référence de votre engagement pour vos propres dossiers. La copie de référence est un fichier .zip contenant des copies PDF de tous les documents de votre engagement.

Télécharger l'exemplaire de référence

  1. Dans le menu Cloud (), sélectionnez Engagements.
  2. Dans la liste, ouvrez l'engagement concerné.
  3. Dans le menu supérieur, sélectionnez Information () | Télécharger la copie de référence.

Un fichier .zip est téléchargé sur votre ordinateur. Extrayez le fichier .zip pour accéder aux copies PDF de vos documents d'engagement.
