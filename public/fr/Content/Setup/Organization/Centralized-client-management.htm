Gestion centralisée des clients (accès anticipé)

La nouvelle application Entités est conçue pour améliorer l'efficacité administrative et créer une expérience utilisateur transparente.

Grâce à la gestion centralisée des clients, les administrateurs d'entités peuvent gérer les informations relatives aux clients de plusieurs unités commerciales à partir d'un emplacement central. Ils n'ont pas besoin de naviguer entre les différentes unités opérationnelles pour effectuer les tâches quotidiennes de gestion des clients, telles que la mise à jour des informations, le partage de l'accès avec le personnel, l'étiquetage des clients, etc.

Une navigation sans faille

La navigation au sein de plusieurs unités commerciales est désormais facilitée par l'affichage de toutes les entités sur une seule page. Vous trouverez ci-dessous des détails sur la manière dont vous pouvez utiliser les différentes nouvelles fonctionnalités.

Nombre Description
1

Onglets

Chaque onglet représente une unité d'affaires spécifique, et les clients de chaque unité d'affaires sont répertoriés dans ces onglets. Le nom de chaque onglet indique les noms des unités d'affaires. Les utilisateurs verront toutes les unités d'affaires auxquelles des entités ont été attribuées.
2

Nombre d'entités

Le nombre entre parenthèses indique le nombre d'entités auxquelles l'utilisateur a accès. Si l'utilisateur a un rôle d'administrateur, le nombre affichera le nombre total d'entités dans l'unité d'affaires.

Lorsque les utilisateurs effectuent une recherche ou utilisent un filtre, ce nombre est mis à jour pour indiquer le nombre de résultats correspondants dans chaque unité opérationnelle.
3

Défilement et tri

Les utilisateurs peuvent désormais utiliser la barre de défilement pour parcourir la liste complète des entités d'une unité commerciale. Ils n'ont plus besoin de cliquer sur différentes pages.

La liste des entités dans chaque onglet peut être triée par colonnes.
4

Menu Liste

Dans une entreprise qui compte de nombreuses unités commerciales, les administrateurs qui ont accès à toutes ces unités peuvent utiliser le menu Liste pour accéder rapidement à une unité commerciale différente, sans avoir à faire défiler la page.
5

Boutons de navigation

Les utilisateurs peuvent également utiliser les boutons de navigation en haut à droite de la page pour naviguer entre les différentes unités opérationnelles.
6

Rechercher

La fonctionnalité de recherche avancée permet aux utilisateurs de rechercher des entités dans toutes les unités commerciales auxquelles ils ont accès. Les résultats de la recherche sont affichés dans leurs unités opérationnelles respectives et le nombre d'entités est mis à jour de manière dynamique afin d'afficher le nombre exact de résultats correspondants.
7

Filtres

La nouvelle interface de filtrage comprend également un filtre pour les unités commerciales. Les utilisateurs peuvent filtrer la vue sur des unités commerciales spécifiques si nécessaire.

Les filtres ont également été améliorés pour mieux répondre aux besoins des utilisateurs.

Par conséquent, pour les champs qui peuvent avoir plusieurs valeurs (par exemple, les étiquettes), les utilisateurs peuvent filtrer sur plusieurs valeurs et les résultats incluront les entités correspondant à n'importe laquelle des valeurs sélectionnées.

Notez que tous les filtres actuellement disponibles ont été conservés.

Actions des entités

Les actions sur les entités (marquer, partager, déplacer, supprimer, etc.) peuvent être effectuées en sélectionnant une ou plusieurs entités. Les utilisateurs peuvent trouver le menu approprié pour les deux scénarios, comme le montrent les captures d'écran ci-dessous.

Importations d'entités

La nouvelle fonction d'importation est conçue pour soutenir les unités commerciales. Les administrateurs de cloud computing peuvent désormais importer une liste d'entités dans plusieurs unités commerciales à l'aide d'un seul fichier d'importation. Cette fonction permet aux utilisateurs de spécifier une unité commerciale pour chaque entité du fichier.

Comment les utilisateurs peuvent-ils accéder aux nouvelles pages ?

La phase d'accès anticipé à la nouvelle application Entities est réservée aux administrateurs de clouds. Ils peuvent trouver la nouvelle application dans le menu Cloud, parmi d'autres applications. Une fois la phase d'accès anticipé achevée, cette application remplacera l'application Entités actuelle.

Améliorations futures

Actions en bloc (option "Sélectionner toutes les entités") - Nous prenons actuellement en charge les options de sélection unique et multiple pour les entités. Dans une prochaine version, nous envisagerons d'inclure l'option permettant de sélectionner TOUTES les entités d'une unité commerciale et d'effectuer des actions en masse.

Gestion des vues à travers les unités d'affaires - Dans l'itération actuelle, toutes les vues personnalisées créées au sein d'une unité d'affaires ne sont disponibles qu'au sein de cette unité d'affaires (onglet). Par exemple, si un utilisateur a créé une vue personnalisée dans BU1, cette vue ne pourra être sélectionnée que dans cette unité commerciale. Dans une prochaine version, nous offrirons une expérience utilisateur plus cohérente pour la création, la sélection et l'utilisation de vues personnalisées dans les unités commerciales.
