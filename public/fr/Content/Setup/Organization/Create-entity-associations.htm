Créer des associations d'entités

Les associations regroupent deux ou plusieurs entités dans le nuage, offrant une vue consolidée des activités et des dossiers de chaque entité faisant partie de l'association. Les entités des associations restent indépendantes les unes des autres.

Associations possibles

Vous pouvez créer une association pour n'importe quel type d'entité, qu'elle soit interne, cliente ou autre.

Par exemple, si vous travaillez pour plusieurs filiales d'une même grande entreprise, vous pouvez représenter chaque filiale par une entité client distincte, puis créer une association pour chacune d'entre elles.

Une association peut également contenir différents types d'entités. Par exemple, si vous disposez d'une entité interne pour votre équipe fiscale, vous pouvez l'associer aux entités clientes pour lesquelles vous effectuez des travaux fiscaux.

Chaque entité ne peut faire partie que d'une seule association. La tentative d'association d'une entité avec des entités déjà présentes dans une association aboutit à une association combinée.

Par exemple, supposons qu'il existe deux associations, l'une composée de l'ensemble {A, B, C} et l'autre {D, E, F}. L'association d'une entité d'une association avec une entité de l'autre association (par exemple, l'association de l'entité C avec l'entité E) résulte en une association combinée de l'ensemble {A, B, C, D, E, F}.

Visualisation des associations

Lors de la visualisation d'une entité faisant partie d'une association, une icône apparaît à côté du nom de l'entité. Cette icône permet de basculer entre une vue associée (), qui affiche le contenu de toutes les entités associées, et une vue d'entité unique (), qui n'affiche que le contenu de l'entité sélectionnée.

Pour créer une entité :

  1. Assurez-vous que vous avez le rôle de Staff Editor ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Entités.

  3. Sélectionnez une entité que vous souhaitez associer à une autre entité et sélectionnez Autres actions () | Associer.

  4. Recherchez les entités que vous souhaitez ajouter. Elles seront affichées dans la boîte de dialogue par type d'entité.

  5. Sélectionnez Fermer.

Autorisations pour les associations

Les utilisateurs peuvent visualiser, ajouter et supprimer des entités d'une association. Il n'y a pas d'autorisations distinctes accordées pour les associations ; un utilisateur est autorisé à voir ou à modifier les entités d'une association sur la base des autorisations accordées aux entités individuelles.

  • Pour visualiser les entités d'une association, le rôle de l'utilisateur doit lui accorder une autorisation de lecture (R) sur l'attribut Summary de chaque entité. Les utilisateurs qui n'ont pas cette autorisation pour toutes les entités d'une association ne peuvent voir que le sous-ensemble d'entités pour lesquelles ils l'ont.

    Les utilisateurs ne peuvent pas voir les associations si leur rôle n'accorde cette autorisation qu'à une seule entité de l'association.

  • Pour ajouter ou supprimer des entités dans une association, le rôle de l'utilisateur doit lui accorder l'autorisation de modifier (E) l'attribut Détails de chaque entité. Les utilisateurs ne peuvent ajouter ou supprimer une entité dans une association que si l'autorisation leur est accordée pour cette entité.

    Bien que les utilisateurs puissent visualiser une entité, ils ne peuvent pas l'ajouter ou la supprimer des associations sans autorisation de modification.
