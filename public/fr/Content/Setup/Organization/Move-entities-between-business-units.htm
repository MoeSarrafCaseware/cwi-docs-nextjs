Déplacer des entités entre les centres de profit

Vous pouvez déplacer des entités entre les unités commerciales pour les organiser en fonction de caractéristiques telles que le type d'entreprise ou la région. Assurez-vous que vous avez le rôle d' accès à l' entité ou des droits équivalents sur l'entité.

Notes :

  • Les entités ne peuvent être stockées que dans une seule unité commerciale à la fois et ne doivent pas être dupliquées dans plusieurs unités commerciales.
  • Vous ne pouvez pas déplacer partiellement des entités associées ; déplacez les entités associées individuellement ou toutes ensemble.
  • Vous ne pouvez pas déplacer les entités pour lesquelles le partage d'association est activé. Désactivez le partage d'association avant de tenter le déplacement.

Déplacer des entités entre les centres de profit

  1. Dans le menu Cloud, sélectionnez Entités.

  2. Dans la liste des entités, sélectionnez les entités à déplacer.

  3. Dans le panneau de détails du fichier, sélectionnez Autres actions () | Remplacer.

  4. Dans la boîte de dialogue qui s'affiche, sélectionnez l'unité commerciale de destination dans le menu déroulant. Si nécessaire, assurez-vous que l'option permettant de déplacer les paramètres de sécurité des entités est sélectionnée.

  5. Sélectionnez Déplacer vers.

Les entités sont marquées pour être déplacées et le processus de déplacement commence. Notez qu'en fonction de la taille des entités, ce processus peut prendre un certain temps. Si les entités ou l'un des engagements qu'elles contiennent n'ont pas été déplacés, une erreur s'affiche dans le volet des détails de l'entité sous Détails du déplacement. Une fois le problème résolu, vous pouvez réessayer le déplacement.
