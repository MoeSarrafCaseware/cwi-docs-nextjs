Gérer l'accès aux applications Caseware Cloud

Lors de la configuration initiale, si votre entreprise a acheté des licences par utilisateur, les membres du personnel seront automatiquement affectés aux applications disponibles dans le Cloud. Si nécessaire, les administrateurs de l'entreprise peuvent réattribuer manuellement l'accès à l'application à des groupes ou à des utilisateurs individuels. Dans la plupart des cas, l'activation des applications pour les groupes est le moyen le plus efficace de gérer l'accès. Lorsque vous activez une application pour un groupe, chaque membre de ce groupe utilise une licence pour cette application. Par exemple, vous pouvez activer une application pour le groupe "Tout le personnel" , ce qui donne immédiatement accès à cette application à tous les membres. Vous avez toujours besoin d'une licence pour chaque membre du groupe.

Vous pouvez également gérer l'accès aux applications et les autorisations sur une base individuelle. Vous pouvez le faire si vous souhaitez activer certaines applications pour un seul utilisateur. Par exemple, si votre organisation engage des consultants, vous pouvez leur accorder un accès complet à certaines applications, un accès en lecture seule à d'autres, et restreindre entièrement leur accès à certaines applications.

Remarque : Vous ne pouvez réaffecter le personnel qu'une seule fois. Si vous avez mal affecté du personnel et que vous avez besoin d'aide pour le réaffecter, contactez notre équipe d'assistance.

Enable access to a Caseware Cloud app

Vous pouvez activer l'accès à une application Caseware Cloud pour un utilisateur individuel ou un groupe.

Après avoir activé ou désactivé l'accès à une application, les utilisateurs ou les membres du groupe concernés doivent actualiser le nuage dans leur navigateur pour que les changements soient visibles dans le menu Nuage.

Remove access to a Caseware Cloud app

Si des membres du personnel ont été automatiquement assignés à une application Caseware Cloud, vous devrez supprimer manuellement leur accès avant d'autoriser l'accès aux bons utilisateurs.

Pour supprimer l'accès à l'application pour le personnel affecté automatiquement :

  1. Dans le menu Cloud, sélectionnez Groupes.

  2. Select the group where you want to remove app access, then select Edit ().

  3. Sélectionnez Apps, puis No access pour l'application concernée.

  4. Sélectionnez Enregistrer.
