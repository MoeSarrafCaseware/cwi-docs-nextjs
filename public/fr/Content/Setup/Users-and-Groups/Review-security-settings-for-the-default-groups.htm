Examiner les paramètres de sécurité du groupe de tous les membres du personnel

Lorsque vous ajoutez un nouveau membre du personnel dans le nuage, il est automatiquement affecté au groupe Tous les membres du personnel . Les paramètres de sécurité de ce groupe déterminent le niveau d'accès de base dont bénéficient les nouveaux membres du personnel.

Remarque : Le groupe Tous les contacts est obsolète. Le groupe existant a été converti en groupe personnalisé avec tous ses membres et paramètres d'autorisation d'origine. Les nouveaux contacts ne rejoindront plus automatiquement le groupe ; cependant, vous pouvez désormais réaffecter, modifier ou supprimer le groupe si nécessaire.

Examiner les paramètres de sécurité du groupe de tous les membres du personnel

  1. Assurez-vous que vous avez le rôle d' administrateur du personnel ou des privilèges équivalents.

  2. Dans le menu Cloud, sélectionnez Groupes.

  3. Sélectionnez le groupe Tout le personnel . Le volet des détails s'affiche à droite.

Examinez les rôles que votre entreprise a attribués à ce groupe à l'échelle du système et les entités auxquelles le groupe a déjà accès. Vous pouvez modifier les rôles de sécurité attribués. Pour en savoir plus, voir Rôles intégrés.

Meilleure pratique : Pour les entreprises de grande taille, nous vous recommandons de supprimer tous les rôles de sécurité attribués au groupe Tout le personnel afin de garantir le bon fonctionnement des groupes de personnel personnalisés .
