Een zoekopdrachtdocument aanmaken

Als je informatie van klanten wilt opvragen, voeg je een zoekopdracht toe aan je opdracht.

Een query is een interactief slim document dat uit de volgende onderdelen bestaat:

  • Vragen - De verzoeken en vragen waarop klanten moeten reageren.

  • Vraagsets - Om gerelateerde queryvragen te groeperen.

  • Typen reacties - Om de verwachte reactie van de gebruiker op te geven. De volgende tabel beschrijft de beschikbare responstypen:

    Type antwoord

    Verwachte gebruikersreactie

    Tekst

    Voer de gevraagde informatie in.

    Keuzelijst

    Selecteer een van de items uit de vervolgkeuzelijst.

    Meerdere keuzelijsten

    Selecteer een van de items uit de vervolgkeuzelijst.

    Bestand uploaden

    Blader en voeg een aangevraagd bestand toe (of sleep het bestand).

    Datum

    Selecteer een label uit de lijst.
    Eenvoudige elektronische handtekening

    Let op: Dit type antwoord is alleen beschikbaar als het is ingeschakeld in je Cloud app.

    Kies of je het document handmatig wilt ondertekenen of elektronisch wilt accepteren. Merk op dat:

  • Instructies - Begeleidende informatie geven aan de klanten. Voeg bijvoorbeeld de ondersteunde bestandstypen toe voor een documentaanvraag.

Om een query toe te voegen aan de opdracht:

  1. Ga naar de pagina Documenten en selecteer vervolgens Toevoegen () | Query.

  2. Selecteer een query uit de beschikbare sjablonen en voer een naam in voor het document.

    3. Let op: Je kunt ook een lege query selecteren, maar we raden je aan om de sjablonen te gebruiken die beschikbaar zijn in je product.

  3. Selecteer Query maken.

    4. Er wordt een nieuwe query toegevoegd in de overeenkomst op basis van de geselecteerde sjabloon.

  4. Bekijk de inhoud en breng indien nodig wijzigingen aan. Zie Gebieden toevoegenvoor meer informatie.

Je kunt een query op elk moment beëindigen zolang de status niet Voltooidis. Selecteer Meer acties () | Query annuleren om een query te beëindigen. De status van de query verandert in Geannuleerd. Geannuleerde query's zijn niet langer zichtbaar in Cloud voor de toegewezen contactpersonen.

Let op: U kunt geen wijzigingen aanbrengen in de inhoud of status van een geannuleerde query. Je kunt alleen de naam van de query en de zichtbaarheidsinstellingen wijzigen. Selecteer in de Documentenpagina Meer acties () | Bewerken om de querynaam te wijzigen en de zichtbaarheidsinstellingen te bewerken.
