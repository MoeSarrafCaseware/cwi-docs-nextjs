Prerequisites for importing from desktop accounting software

Je kunt klantgegevens importeren vanuit de meeste desktop boekhoudsoftware van derden voor gebruik met verschillende Caseware Cloud apps of IDEA. Voordat u importeert, moet u de vereiste stappen voor uw software uitvoeren. Zie Gegevens importeren uit een desktop boekhoudprogrammavoor meer informatie over het importeren van klantgegevens .

Let op: Als je boekhoudpakket niet op deze pagina staat, zijn er geen vereisten. Je kunt de instructies in Gegevens importeren uit een desktop boekhoudprogramma volgen om je klantgegevens te importeren.

Access boekhoudkundige dimensies

  • Installeer Microsoft Windows SQL Server lokaal op de machine die de import uitvoert.

  • Bestand importeren: bestandsnaam.bak

    (Verkregen door Accounting Dimensions te starten en een back-up van alle gegevens uit te voeren)

  • Sla een reservekopie van je database op met het hulpprogramma Access Dimensions Administrator (ADA) dat bij het softwarepakket wordt geleverd.

  • Als de import wordt uitgevoerd op dezelfde machine waar het oorspronkelijke Access Accounting Dimensions-bestand actief is, zorg er dan voor dat de naam van het back-up databasebestand verschilt van de naam van de actieve database en dat het databasebestand in een andere map staat dan de actieve database. Anders moet de originele database van SQL-server worden verwijderd voordat deze wordt geïmporteerd.

Accpac (Advantage-serie)

  • Installeer de ODBC Engine en het stuurprogramma.

  • Installeer Pervasive SQL (Btrieve) of Microsoft SQL.

  • Bestanden importeren:

    • Profiel informatie: CSCOM.DAT / CSCOM.RECCSFSC.DAT / CSFSC.REC

    • Rekeningschema: GLAMF.DAT / GLAMF.RECGLAFS.DAT / GLAFS.RECGLABK.DAT / GLABK.RECGLABRX.DAT / GLABRX.REC

    • GL-detail: GLPOST.DAT / GLPOST.REC

    • Divisie-informatie: Bestanden met de extensies .dap, .dfe en .eap

  • Zorg ervoor dat rekeningstructuren in Accpac unieke reeksen rekeningsegmenten en verschillende totale lengtes bevatten. Working Papers kan het splitsingsnummer niet correct importeren als deze twee structuren niet goed gedefinieerd zijn. Gebruik het ACCPAC-venster Accountstructuurconfiguratie om de structuren te wijzigen.
