Ondersteunde desktop accounting software van derden

Je kunt klantgegevens importeren vanuit de meeste desktop boekhoudsoftware van derden voor gebruik met verschillende Caseware Cloud apps of IDEA. Zie Gegevens importeren uit een desktop boekhoudprogrammavoor meer informatie over het importeren van klantgegevens .

Let op: Als de ondersteunde importgegevens voor uw boekhoudsoftware zowel de proefbalans als de transacties bevatten, zal de Cloud Import Utility zowel de rekeningen als de transacties importeren. Als alleen de proefbalansgegevens worden ondersteund, importeert de Cloud Import Utility alleen de rekeningen.
