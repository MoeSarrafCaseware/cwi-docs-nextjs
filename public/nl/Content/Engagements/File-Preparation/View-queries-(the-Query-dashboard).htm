Zoekopdrachten bekijken (het zoekopdracht-dashboard)

Om je voortgang en de voortgang van contactpersonen bij query's bij te houden, kun je naar de pagina Query Dashboard gaan. Het dashboard bevat een lijst met de zichtbare query's in het engagementbestand. Het bestaat uit de volgende kolommen:

Om toegang te krijgen tot het Query Dashboard in je engagementbestand:

  1. Open het Excel-bestand.

  2. Ga naar de Pagina Documenten.

Een lijst met beschikbare queries in het engagementbestand wordt weergegeven op de pagina. Selecteer een kolom om de lijst te sorteren op status, voortgang, vervaldatum of alfabetisch.

Let op: Alleen zichtbare zoekopdrachten worden weergegeven in de lijst. Als een query verborgen is door zichtbaarheidsinstellingen, wordt deze niet weergegeven in het Query Dashboard.
