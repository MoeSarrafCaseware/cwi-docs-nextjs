Een koppeling toevoegen of verwijderen in tekstgebieden in een brief of memo

Je kunt links toevoegen naar een URL, een e-mail of een telefoonnummer.

Om een nootverwijzing toe te voegen in een tekstsectie:

Open het document en selecteer Bewerken (), selecteer vervolgens Koppeling toevoegen () en voer de volgende informatie in op het tabblad Koppelingsinfo:

  • Tekst weergeven - de geselecteerde tekst is de standaardtekst voor de link. Als je geen tekst hebt geselecteerd, voer dan de weergavetekst in dit veld in.

  • Linktype - selecteer een type uit de vervolgkeuzelijst. Op basis van je selectie toont het dialoogvenster verschillende sets velden die je moet specificeren:

    Type verbinding

    Velden

    URL

    Protocol - selecteer het linkprotocol (http, https, ftp, nieuws of andere)

    URL - voer de URL in

    Link naar anker in de tekst

    Deze functie is momenteel niet functioneel

    E-mail

    E-mailadres - voer het e-mailadres in

    Onderwerp bericht - voer een onderwerp in voor het e-mailbericht

    Berichttekst - voer de hoofdtekst van het e-mailbericht in

    TELEFOON

    Telefoonnummer - voer het telefoonnummer in

Selecteer op het tabblad Doelwaar je de koppeling wilt openen. De standaardinstelling is in een nieuw browservenster.

You can also enter more information in the Advanced tab. Merk op dat als je een link naar een bestandsdownload toevoegt, je het selectievakje Download forceren op dit tabblad kunt inschakelen zodat de bestandsdownload start zodra de gebruiker de link selecteert.

Als je klaar bent, selecteer je Opslaan.

Om een link in een tekstsectie te verwijderen, selecteer je de tekst van de link en vervolgens Link verwijderen ().
