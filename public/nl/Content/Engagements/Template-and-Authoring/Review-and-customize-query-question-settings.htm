Instellingen voor vragen van zoekopdrachten controleren en aanpassen

Als je de instellingen voor een specifieke queryvraag in het document wilt bekijken of aanpassen, selecteer je Meer acties () | Instellingen. De volgende opties zijn beschikbaar in het dialoogvenster Query-instellingen:

Instellingen

Omschrijving

Invoernotities in query toestaan

Selecteer deze optie om een tekstvak onder het antwoord toe te voegen waarin gebruikers notities kunnen invoeren. Je kunt een plaatshouder voor een notitie toevoegen in het weergegeven veld.

Meerdere rijen toestaan

Selecteer deze optie om elk type antwoord dat u toevoegt aan de queryvraag weer te geven als een kolom in een tabel. De plaatshouder voor elk type reactie wordt weergegeven als kolomkop. Je kunt + Rij toevoegen selecteren om meer hoofdgebruikers aan het document toe te voegen.

Merk op dat je maximaal drie kolommen in een tabel kunt hebben.

Reactiesets

Selecteer +Response om een antwoord toe te voegen.

Selecteer Verwijderen () om een reactie te verwijderen.

Selecteer Bewerken () om een reactie te bewerken.

Toon reacties onder procedure

Select this option to display the responses below the query.

Als je klaar bent, selecteer je Opslaan.
