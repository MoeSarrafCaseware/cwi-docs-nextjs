Vragen toevoegen aan uw klantaanvraag

Zodra uw document een groep heeft, kunt u een of meer query's toevoegen. Een query is een vraag voor de klant of een taak die moet worden uitgevoerd als onderdeel van de opdracht. Elke query bestaat uit twee delen:

  • Beschrijving - de vraag of het verzoek voor de klant.

  • Antwoord - het veld waarin de klant informatie kan invoeren of een bestand kan aanleveren.

Een query toevoegen aan een document:

Om een query aan uw document toe te voegen, selecteert u + query.

Selecteer + vraag.
