Aan de slag met Caseware Hybrid Cloud - Firm Administrator

Zorg om te beginnen dat je het volgende hebt:

  • Een actieve licentie voor Working Papers met SmartSync-technologie . Als uw bedrijf geen SmartSync heeft, neem dan contact op met uw plaatselijke distributeur voor meer informatie.

  • Een actieve Caseware Cloud site. Als uw bedrijf geen Cloud site heeft, kunt u er een activeren via MyCaseware.

Log in op de Cloud site van je bedrijf. Als bedrijfsadministrateurmoet je het volgende voltooien:

  1. Geef medewerkers toegang tot Caseware Cloud. Voor meer informatie, zie Personeel maken, Toegang beheren voor Caseware Cloud apps en Ingebouwde rollen.

  2. Entiteiten aanmaken Een entiteit vertegenwoordigt de organisatie van je klant.

  3. Maak accounts aan voor contactpersonen van klanten en wijs hen de nodige toegangsrechtentoe.

  4. Extra Caseware Suite apps inschakelen en instellen.

Voltooi de volgende processen om je Working Papers-gegevens te migreren naar de Cloud als voorbereiding op een hybride omgeving:

  • Grote gegevenssets migreren (meer dan 50 bestanden)

  • Kleine gegevenssets migreren (minder dan 50 bestanden)

  • Gegevens van derden migreren

Nadat je de gegevens van je bedrijf hebt gemigreerd, kun je beginnen met het integratieproces.

  • Grotere bedrijven kunnen de integratie opnemen als onderdeel van de Working Papers-installatie die ze hun personeelaanbieden.

  • Kleinere bedrijven kunnen de URL van de Cloud aan hun personeel geven voor handmatige integratie.
