Systeemvereisten

We ondersteunen het gebruik van Cloud-apps op machines met minimaal 8 GB RAM, maar we raden het gebruik van machines met minimaal 16 GB RAM aan voor betere prestaties.

Ondersteunde browsers

Je kunt je aanmelden bij een Caseware Cloud product vanuit elke webbrowser die TLS v1.2 of hoger gebruikt, maar we ondersteunen en adviseren het gebruik van een stabiele release (niet-beta) van een van de volgende:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

We raden een minimale schermresolutie van 1280 x 800 pixels en een minimale browserbreedte van 1024 pixels aan.

Ondersteunde MS Office-versies

Ondersteuning voor Caseware Cloud-producten:

  • Microsoft Office 2016 of hoger

  • Microsoft Office 365 (desktopversie)

Ondersteunde versies van Working Papers

Integreer Working Papers met Cloud om via een webbrowser toegang te krijgen tot de dossiergegevens van je klanten. We ondersteunen de laatste twee versies van Working Papers; Cloud-integratie is echter beschikbaar vanaf Working Papers 2017. Merk op dat voor sommige functies de meest recente versie nodig kan zijn.

Voor meer informatie, zie Caseware ondersteuningsbeleid.
