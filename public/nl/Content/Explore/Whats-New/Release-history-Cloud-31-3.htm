Nieuw - Cloud 31.3

Caseware Cloud 31.3 voegt API's toe om de externe applicaties van uw bedrijf te integreren met Cloud, zodat uw klantgegevens altijd up-to-date zijn.

API-integratie met externe toepassingen

Caseware Cloud ondersteunt nu integratie met externe applicaties, zoals CRM-systemen zoals Salesforce, MS Dynamics en meer. Dankzij deze integratie kunnen bedrijven klantgegevens eenvoudig overzetten naar de Cloud, zodat ze altijd accuraat en up-to-date zijn. Om te beginnen, maakt u een API-client aan voor de applicatie in de bedrijfsinstellingen van uw Cloud site. Zie Bedrijfsinstellingen - Integratievoor meer informatie.

API-tokens

Bedrijfsontwikkelaars kunnen API-tokens genereren om API-integraties te verifiëren. Hierdoor kunnen geïntegreerde applicaties automatisch nieuwe informatie ophalen zonder dat een ontwikkelaar handmatig een update hoeft te starten. Om een API token te genereren, moeten bedrijfsbeheerders een API client aanmaken en de client ID en secret aan de bedrijfsontwikkelaar geven. Zie Bedrijfsinstellingen - Integratievoor meer informatie.

Algemene verbeteringen en fixes

Naast de nieuwste functies bevat Caseware Cloud 31.3 diverse verbeteringen en bugfixes om uw algehele workflow te verbeteren.
