datasettype

Categoriseert de verschillende gegevensschema's die samen het Caseware Cloud datamodel vormen. Elk datasettype is gestructureerd in een set kolommen, elk met namen en eigenschappen voor gegevensopslag. Zie de datasettypes waarnaar hieronder wordt verwezen voor meer informatie.

  • Grootboek

  • Saldibalans

  • Voorraadtransacties

  • Crediteuren—openstaande saldi

  • Crediteuren—gemaakte betalingen

  • Crediteuren—facturen

  • Crediteuren—leveranciers

  • Debiteuren—ontvangen betalingen

  • Debiteuren—facturen

  • Debiteuren—klanten

  • Debiteuren—openstaande saldi

  • Beschikbare voorraad

  • Overig (synoniem van schemaloos)- Ongestructureerde transactiegegevens die geen vooraf gedefinieerd schema volgen. Dit datasettype kan gebruikt worden om gegevens te importeren die niet overeenkomen met een van de andere datasettypes. Elk gegevensbestand dat geïmporteerd wordt met dit datasettype wordt beschouwd als een eigen dataset.
