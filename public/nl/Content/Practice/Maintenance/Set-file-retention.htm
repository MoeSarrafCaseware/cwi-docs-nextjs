Bestandsbehoud instellen

Je kunt retentie instellen op bestanden om ongewenste items binnen een bepaald aantal dagen, maanden of jaren uit het systeem te verwijderen. Dit helpt ervoor te zorgen dat de dossiers van volgend jaar alleen definitieve items en rapporten bevatten.

Voordat je begint, moet je ervoor zorgen dat je de rol Editor of gelijkwaardige rechten hebt.

To set file retention period:

  1. Selecteer Activiteitenin het menu Cloud.

  2. Selecteer de juiste entiteit in het vervolgkeuzemenu Entiteit.

  3. Selecteer het bestand waarvoor je de retentie wilt instellen met behulp van het selectievakje.

  4. Selecteer Meer acties () | Retentie instellen.

  5. Selecteer de begindatum van het bewaren, hoe lang het bestand moet worden bewaard en eventuele opmerkingen.

  6. Selecteer OK om te bevestigen. Het deelvenster met de bestandsdetails wordt bijgewerkt met de informatie over de bewaarstatus.
