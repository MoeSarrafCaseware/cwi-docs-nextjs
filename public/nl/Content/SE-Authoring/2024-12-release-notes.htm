Geschiedenis van vergrendeling als PDF

#primaryauteur#hoofdauteur

Gebruikers kunnen nu de afsluitingsgeschiedenis downloaden als PDF in het referentie-exemplaar. Om deze functie in te schakelen, voegt u de volgende code toe aan de archiefarray van het product.json-bestand voor uw product:

Titel: "Vergrendeling", "uri": "index.jsp#/lockdown" } }

De referentie kopie is beschikbaar via het Informatie icoon () in het bovenste menu.

Cloud Engagements integratie met Sherlock

#primaryauteur#hoofdauteur

Voor de laatste integratie-updates met Caseware Sherlock, zie Wat is nieuw - Caseware-Sherlock.

Java-upgrades voor de Cloud Engagements-server en microservices

De Caseware Cloud Engagement server is nu geüpgraded van Java 11 naar Java 17. Bovendien zijn de microservices nu geüpgraded van Java 17 naar Java 21.

Fixes in SE 2024-12

De volgende lijst bevat een korte beschrijving van bugs die zijn opgelost in SE 2024-12.

  • [SE-43739] - Bij het toepassen van XBRL-tags op notitiekoppen wordt de selectie niet onmiddellijk gemarkeerd en de kleur en plaatsing van de markering is inconsistent met andere delen van de UX.

  • [SE-49594] - De optie om een zoekopdracht te wissen in de taxonomieviewer wist de zoekopdracht niet.

  • [SE-50284] - De aangepaste contexteditor loopt vast bij het maken van nieuwe contexten of het bewerken van bestaande contexten.

  • [SE-51097] - Tekst in het veld Inhoudsopgave beschrijving wordt niet opgeslagen als een nieuwe tekstsectie wordt toegevoegd aan de jaarrekening vóór de beschrijving.

  • [SE-51303] - Verwijderde tekstgedeelten worden niet zichtbaar verwijderd uit de jaarrekening totdat de webpagina wordt vernieuwd.

  • [SE-51482] - De velden Identifier en Naam in het dialoogvenster Documenteigenschappen zijn niet zichtbaar voor primaire auteurs, waardoor ze niet kunnen worden gewijzigd.

  • [SE-51634] - Bij het importeren van proefbalansgegevens van een bestaande opdracht kan één opdracht worden geselecteerd als bron en bestemming. Hierdoor raakt de betrokkenheid bij de bestemming uit balans.

  • [SE-51655] - Na het toewijzen van een account aan een dimensie is de dimensieselector niet zichtbaar op de gegevenspagina totdat de webpagina wordt vernieuwd.

  • [SE-51780] - Het genereren van iXBRL mislukte in bepaalde omgevingen.

  • [SE-52319] - In het dialoogvenster XBRL Context Editor wordt met de knop om context te kopiëren niet het juiste veld Toepasbaar overgenomen.
