Een pagina-einde toevoegen in de jaarrekening

Je kunt ook een pagina-einde toevoegen binnen een tabel in het document om de informatie in de tabel beter te presenteren. Je kunt bijvoorbeeld een pagina-einde invoegen in een tabel om gelijksoortige of gerelateerde informatie op dezelfde pagina te tonen in plaats van verspreid over twee pagina's.

Een pagina-einde toevoegen in de jaarrekening

  1. Selecteer de gewenste functie.

    2. Pictogrammen voor tabelopties worden rechts weergegeven.

  2. Selecteer Meer acties ():

  3. Kies Edit Dynamic Table uit de lijst met acties.

    4. De tabel wordt geopend in de bewerkingsmodus.

  4. Zoek waar je een pagina-einde wilt toevoegen.

  5. Selecteer Toevoegen () | Pagina-einde aan de linkerrand van de tabel.

  6. Selecteer OPSLAAN of OPSLAAN & SLUITEN.
