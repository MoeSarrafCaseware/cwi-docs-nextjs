Een zoekopdracht toevoegen

Je kunt Query documenten gebruiken om informatie van klanten te verzamelen. Je kunt een standaard maken voor klantverzoeken door Query sjablonen te maken. Gebruikers kunnen een querydocument toevoegen in het engagementbestand met behulp van de querysjabloon.

Beste praktijk: We raden je aan om query-documenten te maken van de sjabloon voor gebruikers in plaats van het ze zelf te laten doen. Op deze manier kun je het query document opnemen in de workflow van het engagement bestand sjabloon en er zichtbaarheidslogica op toepassen .

Een query is een interactief slim document dat uit de volgende onderdelen bestaat:

  • Vragen - De verzoeken en vragen waarop klanten moeten reageren.

  • Vraagsets - Om gerelateerde queryvragen te groeperen.

  • Typen reacties - Om de verwachte reactie van de gebruiker op te geven. De volgende tabel beschrijft de verwachte gebruiker voor elk type antwoord.

    Type antwoord

    Verwachte gebruikersreactie

    Tekst

    Voer de gevraagde informatie in.

    Keuzelijst

    Selecteer een van de items uit de vervolgkeuzelijst.

    Multi-Picklist

    Selecteer een van de items uit de vervolgkeuzelijst.

    Bestand uploaden

    Blader en voeg een aangevraagd bestand toe (of sleep het bestand).

    Datum

    Selecteer een label uit de lijst.
    Eenvoudige elektronische handtekening

    Kies of je het document handmatig wilt ondertekenen of elektronisch wilt accepteren.

    Merk op dat:

  • Instructies - Begeleidende informatie geven aan de klanten. Voeg bijvoorbeeld de ondersteunde bestandstypen toe voor een documentaanvraag.

    Let op: Als je het gedeelte Instructies leeg laat, wordt het niet weergegeven in het document wanneer de toegewezen contactpersonen het openen.

  • Linkerpaneel - Om gemakkelijk te navigeren en de items in je document te ordenen. Het toont een overzicht van beschikbare vragensets en vragen. Als je door het document bladert, bladert het linkerdeelvenster automatisch mee door de omlijnde items, waarbij het item dat je op dat moment bekijkt of bewerkt wordt gemarkeerd.

Om een querytemplate toe te voegen, selecteer je Toevoegen () op de documentenpagina en kies je Querytemplate. Er wordt een leeg document geopend met bovenaan het gedeelte Instructies .

Een zoekopdracht toevoegen

  1. Selecteer Toevoegen ( ) op de documentenpagina en kies vervolgens Controlelijst.

  2. Selecteer een bestaande query sjabloon om je query document van te maken.

  3. Voeg een naam toe voor het querydocument.

    4. Je kunt optioneel een Nummer voor het document toevoegen.

  4. Selecteer Query maken.

    5. Er wordt een leeg document geopend met de instructies van de klant bovenaan.
