Identificeer de procedure-ID

Om de procedure-ID te identificeren:

  1. Identificeer de procedure waarvan je de ID nodig hebt.

  2. Open Developer tools in uw browser en navigeer naar het tabblad Netwerk .

  3. Reageer op de procedure waarvan je de ID wilt verkrijgen. Het tabblad Netwerk wordt gevuld met de vereiste informatie over die procedure, inclusief de ID. Identificeer de ID door de eigenschappen van de procedure uit te vouwen op het tabblad Netwerk .

  4. Kopieer de ID zonder aanhalingstekens (" ") en plak deze in het specifieke veld in de Primary Author Settings.
