Analytische tests beheren in de Analytics Hub

De Analytics Hub is een centrale locatie die je kunt gebruiken om alle analysetests te bekijken en te beheren die beschikbaar zijn in je engagement. Als je een test selecteert, kun je configuratiedetails bekijken en een lijst met checklistprocedures die aan de test zijn gekoppeld. Je hebt de mogelijkheid om een analytische test te dupliceren om deze aan te passen. See Customize an analytic test for more information.

Je kunt ook Alle procedurekoppelingen verwijderen selecteren om koppelingen naar checklistprocedures voor een test te verwijderen. Merk op dat als u deze optie selecteert, de test niet langer wordt weergegeven in de Analytics Hub.
