Notebooks

Notebooks zijn interactieve documenten die live code bevatten samen met markdown tekst. Met Notebooks kun je aangepaste analyses maken, verkennen en de resultaten weergeven. Elk notitieblok bestaat uit een reeks cellen. Voer elke cel afzonderlijk in volgorde uit om de uitvoer te bekijken of voer het hele notitieboek uit.

Notebookbestanden hebben de extensie .ipynb en kunnen worden verkend, gewijzigd en uitgevoerd in JupyterLab, een open-source notebookeditor. Het Data Analytics-team heeft out-of-the box Analytics AI-tests opnieuw ontworpen om te werken met een nieuwe Notebooks-engine. Deze zijn toegankelijk via het AnalyticsAI-platform.

Let op: De eerste keer dat een gebruiker een CWI notitieblok opstart, kan enkele minuten duren. Het Data Analytics-team werkt aan het verkorten van de initiële laadtijden. Wijzigingen in notitieblokken, inclusief die in de codecellen en de markdowncellen, worden niet opgeslagen.

Raadpleeg Aan de slag met AnalyticsAI Notebooks SDK voor meer informatie.
