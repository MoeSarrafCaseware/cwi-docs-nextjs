Er zijn verbeteringen aangebracht in de widget Uitzonderingen met hoog risico:

  • Tests met nul gemarkeerde transacties worden nu uitgefilterd
  • Wanneer het testfilter wordt gebruikt voor een test met nul gemarkeerde transacties, toont de widget nu een bericht "Geen relevante gegevens"

Voor deze release zijn enkele verbeteringen aangebracht in de widget lay-outs van Envision. Zie voor meer informatie.

Gegevensimport

Excel-import

De volgende algemene verbeteringen zijn toegevoegd aan deze release.

  • Er is een link naar de vereisten voor geldige csv-bestanden toegevoegd aan de pagina Transacties importeren. Zie Vereisten transacties in csv-bestand voor meer informatie
  • Er verschijnt niet langer een bericht wanneer een gegevenslay-out succesvol is toegepast tijdens het importeren van transactiebestanden.

Wizard importeren

Deze verbeteringen zijn aangebracht in de wizard Transacties importeren:

  • Het zoekveld Gegevensbron is verplaatst naar de lijst met beschikbare aanbieders
  • Distributeurs kunnen nu de lijst met gegevensbronnen filteren door het product.json-bestand van hun app aan te passen. Zie Lijst met productfuncties (API) - Imports voor meer informatie.
  • Je kunt gegevens importeren uit desktop boekhoudpakketten met de wizard Transacties importeren. Dit komt overeen met de functionaliteit van de bestaande knop Bureauboekhoudpakketten op de pagina Gegevens importeren . Primaire auteurs kunnen deze import in hun app inschakelen vanuit het product.json-bestand. Zie Lijst met productfuncties (API) - Imports voor meer informatie.
