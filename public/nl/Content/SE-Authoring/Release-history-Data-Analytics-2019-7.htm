AnalyticsAI 2019.7

Dit benadrukt de belangrijkste wijzigingen sinds de laatste uitgave van dit product.

Updates voor het importeren van transacties

Het proces voor het importeren en mappen van .csv transactiebestanden is voor deze release verbeterd.

Je kunt nu meerdere transactiebestanden importeren. Als je dit doet, worden de bestanden verwerkt tot een enkel gegevensbestand voor analyse.

Het proces voor het in kaart brengen van geïmporteerde transactiebestanden is ook verbeterd, met onder andere betere beschrijvingen voor elk gegevenstype.

Zie Cliënttransacties importeren voor meer informatie.

Verbeteringen aan de gegevensexport

De gegevens in de Analysis Summary PDF bevatten nu:

  • Tijdreeks - Risico op waarde van transacties
  • Uitgevoerde tests
  • Lijst van beïnvloedde rekeningen
  • Bijlage A Omschrijving van uitgevoerde tests

Status van tests

De configuratiepagina toont nu de teststatussen als volgt:

  • Een voortgangsindicator wordt weergegeven voor elke test die wordt uitgevoerd.
  • Niet uitgevoerd - grijs statuspictogram.
  • Preconditie mislukt - geel statuspictogram.
  • In afwachting van uitvoering - grijs statuspictogram.
  • Mislukte run - rood statuspictogram.
  • Succesvol uitgevoerd - groen statuspictogram.
