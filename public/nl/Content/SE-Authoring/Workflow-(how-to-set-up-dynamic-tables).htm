Workflow (hoe stel ik dynamische tabellen in?)

Tabellen verwijzen naar de verschillende schema's die je kunt opnemen in je jaarrekeningdocument.

Als je een tabel wilt toevoegen in een jaarrekeninggebied, selecteer je Toevoegen () en kies je vervolgens Dynamische tabel. An empty table is added in the area with a default title Dynamic Table. Je kunt de standaardtitel wijzigen door deze te selecteren en een nieuwe titel toe te voegen in het veld.

Let op: De standaard lettergrootte voor inhoud in dynamische tabellen is 12 en kan niet worden gewijzigd.

De volgende stappen vatten de aanbevolen workflow voor het instellen van dynamische tabellen samen. Je kunt deze workflow als leidraad gebruiken.

  • Add rows

  • Kolommen toevoegen

  • Aangepaste groepen maken in tabellen

  • Rijen uitzonderen van een aangepast groepstotaal

  • Een verwijzing naar een notitie toevoegen in een tekstsectie

  • Gebruikers toestaan invoerrijen toe te voegen in het tabelgebied zonder de bewerkingsmodus van de tabel te openen

  • De optie standaardweergave instellen voor rekeningen en groepen in de jaarrekening

  • Een pagina-einde toevoegen in de jaarrekening

  • Kopieer de tabel om snel een andere te maken met dezelfde instellingen
