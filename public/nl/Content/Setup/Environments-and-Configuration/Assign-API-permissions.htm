API-machtigingen toewijzen (Azure AD)

Door de API-rechten te configureren, kan Caseware Cloud basisinformatie van Microsoft Azure lezen om gebruikers op de juiste manier te authenticeren.

API-rechten toewijzen:

  1. Selecteer Azure Active Directory | App Registrationsen selecteer vervolgens je app.

  2. Selecteer API-machtigingenin het navigatiedeelvenster aan de linkerkant.

  3. Selecteer Een toestemming toevoegen.

  4. Selecteer Microsoft Graph.

  5. Selecteer Gedelegeerde rechten.

  6. Gebruik het selectievakje om de volgende rechten te selecteren:

    • E-mail

    • Openid

    • Profiel

    • Gebruiker.lezen

  7. Selecteer Voorwaarde toevoegen.

  8. Selecteer Beheerdersrechten verlenen voor Standaardmap.

  9. Selecteer Ja als wordt gevraagd: "Wilt u toestemming geven voor de gevraagde machtigingen voor alle accounts in Standaardmap?"
