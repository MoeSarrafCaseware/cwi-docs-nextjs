De metadata-URI kopiëren (Okta)

Een metadata URI levert de nodige gegevens voor de werking van SSO. Je zult het gebruiken wanneer je SSO configureert in Cloud.

Om de metadata URI te kopiëren:

  1. Log in op het Okta-dashboard van uw bedrijf met een beheerdersaccount .

  2. Vouw het vervolgkeuzemenu voor accounts uit en beweeg je cursor over de URI voor metagegevens. Er wordt een klembordpictogram weergegeven. Selecteer deze om de metadata URI te kopiëren.

  3. Plak de URI van de metagegevens in een tekstdocument. Het zal in dit formaat zijn:

    <account>.okta.com

  4. Pas de metadata URI aan zodat deze voldoet aan de volgende opmaak:

    https://$<account>.okta.com/.well-known/openid-configuratie

  5. Geef het het label Metadata URI en sla het tekstdocument op voor later gebruik.

Volgende (Optioneel) Autorisatieservers beheren
