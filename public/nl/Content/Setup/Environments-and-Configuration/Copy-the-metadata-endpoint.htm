Kopieer het metagegevens eindpunt (Azure AD)

Een metadata-eindpunt biedt autorisatie-URL's en andere metadata die nodig zijn voor het functioneren van SSO. Je zult het gebruiken wanneer je SSO configureert in Cloud.

Kopieer het metagegevens eindpunt

  1. Meld u aan bij uw Azure-portaal.

  2. Selecteer Azure Active Directory.

  3. Selecteer App-registratiesin het linker navigatiedeelvenster.

  4. Selecteer de Import Component uit de lijst.

  5. Selecteer Eindpunten.

  6. Een lijst met overeenkomsten wordt weergegeven.

    Belangrijk: Kopieer het OpenId Connect metadata document endpoint en plak het in een tekstdocument voor later gebruik. Label het Metadata Endpoint.
