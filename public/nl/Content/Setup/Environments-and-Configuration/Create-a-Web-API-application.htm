Een Web API-toepassing aanmaken (ADFS)

Maak een Web API-applicatie voor de Cloud-site van je bedrijf en definieer de rechten van de applicatie.

Een Web API-toepassing aanmaken

  1. Klik in de ADFS Administration Console op de map Application Groups en dubbelklik vervolgens op uw nieuwe applicatiegroep.

  2. Klik in het dialoogvenster Eigenschappen op Toepassing toevoegen.

  3. Selecteer Web APIin de lijst Sjabloon. Klik op Volgende.

  4. Plak in het veld Identifier de Client Identifier die je eerder naar een tekstdocument hebt gekopieerd. Klik op Volgende.

  5. Selecteer in de lijst met toegangscontrolebeleid het beleid dat het beste van toepassing is op jouw bedrijf. Klik op Volgende.

  6. Selecteer de volgende scopes in de lijst Toegestane scopes:

    • allatclaims

    • E-mail

    • openid

    • Profiel

    Klik op Volgende.

  7. Klik op Ja om de configuratie te voltooien.
