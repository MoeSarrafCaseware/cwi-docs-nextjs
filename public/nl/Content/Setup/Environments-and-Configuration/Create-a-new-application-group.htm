Een nieuwe toepassingsgroep aanmaken (ADFS)

Om Cloud met ADFS te integreren, moet je eerst een nieuwe applicatiegroep voor Cloud maken.

Een nieuwe toepassingsgroep aanmaken

  1. Start de ADFS Administration Console.

  2. Klik op de map Application Groups en klik vervolgens in het menu Actions op Add Application Groups.

  3. Voer de naam voor de map in. Selecteer Serverapplicatiein de lijst Sjabloon. Klik op Volgende.

  4. Het veld Client Identifier wordt automatisch ingevuld.

    Belangrijk: Kopieer de geheime waarde van de client en plak deze in een tekstdocument voor later gebruik. Label het Client Identifier.

    Voer in het veld Redirect URI een tijdelijke URL in (bijvoorbeeld https://www.caseware.com) en klik op Toevoegen. Klik op Volgende.

  5. Selecteer Een gedeeld geheim genereren.

    Belangrijk: Klik op Kopieer naar klembord en plak het geheim in een tekstdocument voor later gebruik. Label het als Klantgeheim.

    Klik op Volgende.

  6. Klik op Ja om de configuratie te voltooien.
