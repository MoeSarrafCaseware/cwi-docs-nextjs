Maak een Issuance Transform-regel aan (ADFS)

Maak een Issuance Transform-regel om LDAP-kenmerken toe te wijzen aan uitgaande claims.

Maak een Issuance Transform-regel aan (ADFS)

  1. Klik in de ADFS Administration Console op de map Application Groups en dubbelklik vervolgens op uw nieuwe applicatiegroep.

  2. Dubbelklik in het dialoogvenster Eigenschappen op de Web API-toepassing.

  3. Klik in het dialoogvenster Web API Properties op het tabblad Issuance Transform Rules en klik vervolgens op Add Rule (Regel toevoegen).

  4. Klik op de vervolgkeuzelijst Claimsjabloon en selecteer LDAP-attributen verzenden als claims. Klik op Volgende.

  5. Voer een naam in voor de claimregel, klik vervolgens op het vervolgkeuzemenu Attribute store en selecteer Active Directory.

  6. Vul de tabel Mapping van LDAP-kenmerk naar uitgaande claimtypes als volgt in:

    LDAP attribuut Type uitgaande claim
    E-mailadres: E-mail
    Voornaam voornaam
    Achternaam familienaam

  7. Klik op Ja om de configuratie te voltooien.
