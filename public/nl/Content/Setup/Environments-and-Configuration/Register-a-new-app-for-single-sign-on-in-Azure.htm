Een nieuwe app registreren voor single sign-on (Azure AD)

Om Cloud te integreren met Azure AD, moet je Cloud registreren als een nieuwe app. Wanneer Cloud is geregistreerd, kunt u Cloud toegang geven tot uw Active Directory-domein.

Een nieuwe app registreren voor single sign-on (Azure AD)

  1. Meld u aan bij uw Azure-portaal. Als je toegang hebt tot meer dan één portaal, selecteer dan je account in de rechterbovenhoek en selecteer vervolgens de gewenste huurder.

  2. Selecteer Azure Active Directory.

  3. Selecteer App-registratiesin het linker navigatiedeelvenster.

  4. Selecteer NEW AREA.

  5. Op de pagina Registreer een toepassing voert u uw registratiegegevens in:

    • Name: Voer een andere naam in.

    • Ondersteunde accounttypes: Selecteer Accounts alleen in deze organisatiemap (Standaardmap alleen - Enkele huurder).

    • URI omleiden: Selecteer Web. You do not need to enter a URL.

  6. Nadat u uw gegevens hebt ingevoerd, selecteert u Registreren. Je sollicitatiegegevens worden weergegeven.

    Belangrijk: Kopieer de Geheime waarde en plak deze in een tekstdocument voor later gebruik. Label het Application ID.

Volgende Genereer een nieuw klantgeheim.
