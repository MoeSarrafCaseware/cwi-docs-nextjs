Beveiligingsarchitectuur van Caseware Cloud

Caseware neemt de beveiliging van je gegevens zeer serieus. Hieronder vind je informatie over de beveiligingsstructuur voor Cloud.

Hoe worden mijn gegevens beschermd?

Caseware Cloud ondergaat voortdurend onafhankelijke beveiligingsaudits om onze toewijding aan gegevensbeveiliging, privacy en nalevingscontroles te versterken. Onafhankelijke auditors onderzoeken ons volledige Information Security Management System (ISMS) om onze dienstverlening, activiteiten en het beheer van het Caseware Cloud-platform te controleren en te garanderen dat het voldoet aan de industrienormen. Deze certificeringen tonen aan dat Caseware Cloud zich internationaal inzet voor veiligheid en kwaliteit.

  • Caseware Cloud is ISO 27001:2013 gecertificeerd en het ISMS is in overeenstemming met best practices.

    Deze certificering is onafhankelijk van Amazon Web Services (AWS), dat zijn eigen ISO 27001-certificering heeft voor zijn datacenters, infrastructuur en activiteiten.

  • Caseware Cloud is SOC 2 Type 1 en SOC 2 Type 2 gecertificeerd.

Caseware Cloud gebruikt dezelfde encryptiesterkte als online bankieren. Encryptiesleutels worden gegenereerd en zijn eigendom van Caseware Cloud Ltd. en worden nooit gedeeld met andere partijen, waaronder Amazon. Hetzelfde coderingsniveau wordt toegepast op:

  • Gegevens overgebracht van en naar Caseware Cloud

  • Gegevens in rust

  • Back-up gegevens

Zie Cloud Security Compliancevoor meer informatie over beveiliging in Caseware Cloud.
