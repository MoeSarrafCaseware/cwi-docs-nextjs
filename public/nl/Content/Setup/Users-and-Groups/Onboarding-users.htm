Onboarding van gebruikers

Elk accountantskantoor heeft zijn eigen beveiligingsbeleid en werkwijze. Hoe bedrijfsbeheerders gebruikers aannemen, hangt af van deze twee factoren.

Grote bedrijven

De volgende stappen geven een overzicht van hoe gebruikers in grote bedrijven aan boord kunnen worden genomen.

  1. Aangepaste beveiligingsrollen aanmaken

  2. Maak groepen op basis van functionele rollen en toegangsrechten.

  3. Create user accounts.

  4. Systeembrede beveiligingsrollen toewijzen

  5. Geef contactpersonen de juiste toegangsrechten tot entiteiten.

Kleine bedrijven

De volgende stappen geven een overzicht van hoe gebruikers in kleine bedrijven aan boord kunnen worden genomen:

  1. Aangepaste beveiligingsrollen aanmaken

  2. Groepen maken op basis van functionele rollen en toegangsrechten (optioneel).

  3. Create user accounts.

  4. Systeembrede beveiligingsrollen toewijzen

    5. Let op: Als je gebruikersgroepen hebt gemaakt, wijs dan systeembrede beveiligingsrollen toe aan de personeelsgroepen in plaats van aan individuele medewerkers.
